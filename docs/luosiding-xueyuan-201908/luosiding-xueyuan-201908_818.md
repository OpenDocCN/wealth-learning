# 钉大，现在实体经济

(提问)你说我是包子么 : 钉大，现在实体经济不是很差么？那 我们投的公司还是盈利的么。。

2018-12-07

回答：估值表里有 ROE，这个为正，那指数背后公司还是赚 钱的。从全世界范围看，主要经济体的宽基指数，基本都是 每年盈利的。只不过每年盈利的增长速度会波动。大部分年 份盈利是增长的，少部分年份盈利下滑。 最近几年，盈利增 长速度最差的是 2015 年，那年虽然股市大涨，但是实际上上 市公司的整体盈利是接近负数的。之后 16-17 年上市公司盈利 增长复苏，16 年全部 A 股盈利增长大约 8%，17 年全部 A 股盈利 增长大约 18%，今年财报还没出完，预计全年盈利增长大约 9%。比 15、16 年好一些。 过去历史上 A 股上市公司的盈利平 均增速，大约是 12-14%。未来没有那么高了，预计保持在 10- 12%左右。17 年是显著高于历史平均的，今年略微低于平 均。这个盈利增速也是围绕均值上上下下的。(12 赞)
