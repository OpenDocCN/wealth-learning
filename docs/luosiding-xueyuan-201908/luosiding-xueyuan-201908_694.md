# 钉大，所谓的神奇公

(提问)乡巴佬 阿懒 * : 钉大，所谓的神奇公式究竟靠不靠谱， 中邮中证的两只基金，可以买吗

2019-01-04

回答：1.神奇公式的话，原理问题不大。 其实这种理念，就 是挑选又好又便宜的股票。只不过又好又便宜的标准，很多 指数会有区别。 比如说红利机会，用股息增长衡量好，用股 息率的高低衡量便宜。 比如说 PB-ROE 策略，用 ROE 高衡量 好，用 PB 低衡量便宜。 神奇公式的指数，目前挑选出来的股 票，以中盘股为主，类似于红利机会和 500 低波动。 以 500 低 波动为例，从 2004 年底以来，收益跟神奇公式指数也差不 多。 所以神奇公式长期收益都还可以，不过也没有说特别的 鹤立鸡群。 2.中邮的这只神奇公式指数基金的规模有点小， 目前不到 1 亿，暂时还是继续观察下了。 后面有合适的品种会 考虑的。(17 赞)
