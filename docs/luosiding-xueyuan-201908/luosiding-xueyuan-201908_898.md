# 想请教一下螺丝钉先

(提问)忘记了名字的我 : 想请教一下螺丝钉先生：房地产信托 指数基金既然是投资商业地产领域，为什么不是滞后于股市 发生上涨，而是存在一定的负相关性？还有就是如果我购买 了 000179，那么人民币的汇率是否也会影响到我的收益率？ 谢谢您的指点♥

2018-11-23

回答：房地产信托基金也就是 REITS 基金。国内还没有这类品 种，主要是美股市场有。国内能买到的也是投资美国房地产 的 QDII 基金，例如广发美国房地产，投资的就是美股的 REITS。 在美国，美股的 REITS 和美股是有一些正相关性的。 因为 REITS 也是在股票交易所里上市交易。如果是一个股票投 资者，他可以投资美股也可以投资 REITS。所以当股市火热或 者进入熊市下跌的时候，REITS 也会受到一些影响。 而美股 和 A 股相关性目前还很小，美股的 REITS 跟 A 股相关性自然也 很小。所以这两个有的时候甚至是负相关的。 000179 就是广 发美国房地产，这个基金的净值是受到人民币和美元汇率影 响的。美国房地产也是美元资产(8 赞)
