# 老师，如果定投螺丝

(提问)张不二 : 老师，如果定投螺丝钉组合，定期定额定投和 定期不定额定投，三年以后的收益会有多大差异呢？谢谢 2019-06-04

回答：之前回测过，大约能相差 2 个点的年化收益率。也就是 投 1 年下来，收益会差别 2 个点左右。 对收益影响比较大的因 素： 1.低估投资，对收益提升最明显。 2.选择优秀品种。 3\. 定期不定额，能提升 2 个点左右。 这三个技巧是我们最常用 的。(17 赞)
