# 钉大你好，根据你多

(提问)chinaren : 钉大你好，根据你多年的指数定投经验，你 觉得低估买入高估卖出，与盈利达到 30%止盈两种方法比 较，两种方法各自有何优缺点？如何把握更好？长期看哪种 收益更好？

2019-06-14

回答：1.低估买入高估卖出 好处： （1）交易频率低，交易费 用低 （2）通常高估是牛市，会有 1-3 倍不等的收益。 缺点： 等待时间长，需要耐心等待牛市到来。 2.30%收益率止盈 这 是可转债自带的止盈方法。 好处： 通常 2-3 年就可以等到一 次。 参考可转债，大部分可转债在 3 年以内触发 30%的收益。 指数基金的涨跌周期跟可转债很相似，也差不多是 2-3 年可以 遇到一次这样的机会。对投资者更友好。 缺点： （1）可能 错失大牛市的收益，在牛市初期就卖出，卖出后可能市场已 经上涨了不少，也不好找其他低估资产。 （2）交易费用高一 些。 不过第一种是可以凭借估值来有一个统一的卖出提醒 的。 第二种不同投资者开始投资时间不同。有的人已经是达 到 30%卖出了，有的人可能刚开始投。所以按收益率止盈， 只能是大家自己判断。组合能提示的，也主要是第一种卖出 的机会～(34 赞)
