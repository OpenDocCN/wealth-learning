# 想请问钉大，我们在

(提问)衣非 : 想请问钉大，我们在申购基金时申购费是交给基 金管理公司吗？为什么在第三方基金销售平台买，申购费能 够打折很多折扣，这些平台是怎么赚钱的？除了申购费、赎 回费，我们还需要交其他费用吗？谢谢

2018-11-20

回答：公募基金监管是很严格的。我们投资指数基金，主要 是几块费用 1.申购费赎回费。给到有基金销售牌照的销售平 台，例如蚂蚁天天蛋卷。 2.管理费。给到有公募基金牌照的 基金公司。 3.托管费。给到有托管资质的托管行。 4.买卖股 票的交易佣金。给到基金公司合作的券商。 5.各种税。给到 国家。 6.对指数基金来说，还有一个指数授权费，给到指数 公司。不过这个比较低。指数公司在国内也是国有性质的， 半垄断。 对指数基金来说，上面的费用基本都是透明的。 销 售平台能插手的，就是申购费赎回费。 如果销售平台比较强 势，那销售平台还可以争取到管理费。例如招商银行和蚂蚁 这样的高流量平台，甚至可以要到自己平台销售基金的 50% 以上的管理费。 而且卖基金卖的好的招商、蚂蚁、天天等， 自己本身就很有钱，甚至愿意前期不赚钱降低费用吸引投资 者。 如果单靠基金的申购费，基金销售平台是不够养活自己 的。(9 赞)
