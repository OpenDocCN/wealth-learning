# 长江电力，属基金类

(提问)紫风 : 长江电力，属基金类吗?每年都分红吗?怎么分 红。谢谢

2018-11-24

回答：长江电力是股票。学院里搜索“长江电力”能找到之前 的分析文章的。 它是股票，基本每年都分红，分红计划写在 了公司章程里。所以相对来说分红还是比较稳定的，一年一 次。分红直接现金分到股票账户，之后可以提现。 它的经营 模式很简单，就是持有水电站发电，每年的现金流很充沛， 收益很大一部分以分红的形式分出来。同时国家给水电定价 的时候，水电的长期涨幅是可以跑赢通货膨胀的。所以长江 电力的预期收益率，大概率是比“买入时的股息率+通货膨胀 收益率”要高一些。 所以最好是在股息率比较高（例如在 5% 以上）的时候再考虑投资。 另外持有这种股票，目的通常是 为了打新。持有 20-30 万市值的股票，平均每年可以打中 1 签。如果只是有几万元，那买长江电力这种个股就比较吃 亏，因为打新很难打中，缺少了一块收益～(10 赞)
