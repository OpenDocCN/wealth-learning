# 高房价之痛，华为这次真的要跑了，第一批 2700 员工刚刚搬离深圳

![](img/a0c3c74abf32d2c865a2c3db4a71b0a0.jpg "跳跃猫引导关注")

前几年，不要让华为跑的了文章火遍朋友圈，这个口号也成为深圳市政府的行动指南，给予了华为很多优惠政策。但是华为搬离深圳的决心似乎没有因此动摇，今年 7 月 1 日，华为公司正式搬迁，7 月 2 日，将会有第一批 2700 人从深圳到东莞松山湖溪流背坡村去上班，约 1500 辆车（含 70 辆大巴）。

因为这些人目前还住在深圳，所以大规模集体往返上下班会对交通构成压力，媒体称华为还特请交通部门的领导多加关注。这一波，华为的研发部门整体全部搬迁到了东莞，还包括一些其他部门的人员，只留总部在深圳。

我们都知道，华为是最舍得花研发费的，华为的研发工程师那都是高薪群体，是一个城市房价最强力的支撑群体。而华为此次真的把人都搬走了，其中最重要的原因，很显然是因为深圳的房价和土地，都太贵了。

深圳目前上演的是 8.5W 一平的千万豪宅，冒雨排队抢筹的闹剧，千万豪宅还需要抢，摇中的概率还只有 11%。华为的研发工程师的确薪水很高，但是对于这种级别的房价，这点薪水还不够看。而东莞为华为在松山湖提供了 3 万套公寓，售价 8500 一平米，唯一要求是承诺呆满 5 年，彻底解决华为员工的安居乐业问题。

而高房价同时拉升着深圳的地价，高昂的地价让深圳的华为总部难以进行任何扩张，成本高的惊人，为了扩大企业规模，这个占地问题也只能向东莞去解决，同样一亩地，东莞不仅地价便宜还给出种种优惠，实际付出的成本不到深圳的零头。

在这种情况下，华为是早晚会跑的，顶多就是把总部留在深圳做政治表态就不错了。对于政府收入来说，总部所在决定税收大头，税收估计是没啥问题，但是华为的基层员工源源不断的外迁，对深圳的房价构成了流出压力。其他的城市都在到处抢人，目的就是拉一把房价，深圳这边反而在流出人口，外流的还是相当有钱的工程师，这是一个很坏的示范效应。

各大房企全线收缩阵地

这次华为搬迁，虽然是筹划已久的事情，但是时间点也刚好很敏感，是整个国家一轮对房价的打压风暴的档口。各大房企全线收缩阵地，以 18 年全国销售冠军碧桂园为例，在今天相关媒体曝出新闻，明确叫停“全覆盖”战略，不惜一切代价回笼现金，一切工作重心围绕现金流运转，甚至不惜项目利润转负。

去年的时候，碧桂园还雄心勃勃的推行全覆盖计划，要求中国的每一个城市，甚至是发达的乡镇，都要有一个碧桂园小区，这个野心可以说是贼大，大到不敢想象。到了 18 年，风头急转直下，碧桂园陆续退出多个合作项目，然后全力消化自己主刀的项目，宁可不赚钱，也要把手头的土地迅速变成房子，然后卖出去，把现金回笼到手里。

这一切，是国家的调控大背景在起作用，不管是收紧房地产企业债，还是暂停货币化棚改，都可以说是致命一击。

中国的房价有点高了

政府这次的政策，主要打击的是三四线城市的房价，但是实际上，一线城市的房价也有点高的，之所以不直接打崩，是因为没人能承担的起中国房价泡沫刺破，直接暴跌硬着陆的巨大风险。所以只能稳住泡沫，期待用时间实现软着陆，包括我本人，都是认为如果购买房产，那么一线的最划算，其次是省会城市，但是这并不代表，一线城市的房价是合理的，这个泡沫不仅有，而且很大。

我举个例子，因为一线城市房价过高，所以同步拉升了这些城市的土地价格，工业用地、商业用地价格也都同步上涨，对工业和商业造成了严重的挤压效果，但是这些对平民百姓造成的影响是隐形的，并不能直接凸显矛盾。但是高房价造成了墓地价格的飞涨，这就造成了一场闹剧。

2008 年底，冯导的《非诚勿扰》里面，葛优碰到一个推销墓地的人，当时的推销价格是 3 万一块，看似天价，但是时间拨到 2018 年，同一块墓地目前的报价是多少呢，据媒体爆料，已经达到了 33.8 万一块，涨的比房子还快。

生前买不起房，死后买不起墓，这土地价格着实吓人，以上海为例，选一款经济型墓地，价格也要 10~20 万，更厉害的是，这个价格只是 20 年的租金而已，20 年后，你还要继续交钱。

精明的上海人就想到一个奇招，跑到周边的县城买房子，一个一室一厅的小户型，也就几十万而已，价格比墓地便宜多了，这些地方通常不限购不限贷，还能贷款买，首付更是寥寥。买了这样的一套住宅供奉先人骨灰，比买一个墓地要气派多了，而且还算一套持有房产，将来极端情况下还可以变现，不像墓地就是个死物。这些县城是有点远，但是那些墓地也很偏啊，反正是一年来祭拜一次就可以了，这点距离还是可以接受的。

买这样的房子做墓地，可以说是上对得起祖宗，下对的起子女，这个做法很快就普及开了，把上海周边的一些小城市的个别小区，给生生的买成了阴宅，一个小区里死人比活人还多，平时阴森森，一到清明节就热闹非常，简直吓人。

房子本来是修了给活人住的，结果现在给死人住，这种情况肯定不科学，属于严重的资源浪费。为什么会出现这种闹剧，第一个原因就是大城市的房价实在太贵了，顺带也把墓地的价格给拖上去了，如果墓地三四万元一个，我想没有人会采用这种极端的办法买商品房当墓地。第二个原因就是这些上海人肯定是预期房价还会继续涨，才买商品房做墓地的，相比更多人的人买了投资房直接毛坯空置，他们还算多了一个使用功能，有些毛坯直接闲置一二十年，更浪费。如果他们预期房价每年都会下跌，那么没有人会把自己买来的房子进行毛坯空置，也不会有人特地买房子当墓地用。

为什么他们敢这么预期，因为他们赌政府不敢让房价崩，他们这种思想是对的，没有任何人敢预测政府会主动让房价崩，我也不敢。中国房价将来可能早晚会崩，但是那会是很久以后，而且一定是在政府主动护盘的基础上意外崩掉的，没人会认为政府会主动刺破房价泡沫，因为他在里面获得的好处太大了，所以这个概率无限接近于 0

在这种预期房价永远涨的基础上，调控房价就变得异常困难，要打破这种预期只有发动一场酣畅淋漓的暴跌给一次风险教育，让投机客知道楼市有风险，入市需谨慎。但是这个暴跌引发的可能性后果难以预料，政府也不敢，于是就只能这么僵着了。

但是房地产这种泡沫价格，肯定是不正常的，三四线城市的房价不正常，一线的其实也不正常，华为这样赚钱的大企业，都被迫慢慢迁出深圳，这就是在给深圳的房价示警，而政府在给三四线城市房价示警，当所有的城市都有示警征兆的时候，这是对全体中国人都敲响了警钟。

而目前的稳房价，总体横盘的策略，是中央的大方针，这种策略优点是稳，对经济影响可控，缺点是会把这种高房价的痛苦延长很多年。在总体横盘的基础上，力所能及的时候，我建议能否打破一些城市的必涨预期，给予一定的风险教育，适当的让房价来一次下跌，引导资金进入实体行业。

高房价是中国人之痛，我希望这种痛，能否少持续一点时间，华为已经跑了，不要让其他的公司也跑了。
