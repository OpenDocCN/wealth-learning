# 论市场经济在国庆堵车里发挥的重要作用

![](img/dde1b3282d2d1f1dcc6b6ac59c1c10c9.jpg)喜欢我的都关注我了~

向我们昂首阔步走来的是某镇救援队，他们整装出发，正在奔赴高速堵车救援第一线。东北大鱿鱼选手一马当先，冲在队伍的最前沿，他们的鱿鱼个大、味美、价格低，是东北劳动人民的骄傲。

然后进入会场的是重庆小面、快乐小柠檬，他们凭借积极的心态参加这次盛会，他们勤奋刻苦，不分昼夜，敢于拼搏，定能在这次比赛中取得傲人的成绩。

最后压轴的队伍是鲜榨果汁，这里要重点给观众朋友们介绍一下这位鲜榨果汁，他采用新鲜水果制成，可以去热解火，当你在高速上堵到冒火的时候，特别需要一杯这样的饮品。

高速堵车不用怕，市场经济来帮你，祝广大的司机朋友们，堵车愉快。
