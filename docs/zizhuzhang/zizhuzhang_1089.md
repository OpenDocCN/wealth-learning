# 中国教育问题的根源在于大学而不在高中

![](img/52a79e3d52be9eb2a9968752e7048aee.jpg "bki-20140213163010-663826156.jpg")

紫竹张先生

让你一天比一天更聪明的财经号

![](img/755cdb1558dd0c0841944db23c8f859b.jpg "undefined")

中国的高中老师，激励学生好好学习的时候总会这么说：“好好学，考上大学你们就轻松了，想怎么玩就怎么玩。”而事实上这些老师并没有忽悠学生，他们说的都是真的。在学习上，中国高中生的勤奋和大学生的懒惰形成了鲜明的对比。打游戏、谈恋爱、考前突击刷题复习成为了中国大学生的日常生活。

"为什么我们的学校总是培养不出杰出人才?"这就是著名的"钱学森之问"，在过去，我们把这个责任推给了应试教育头上，认为中国机械填鸭式的教育是培养不出人才的主要原因，所以在高中残酷的学习选拔之后，大学就应该自由发展，放羊式教育，其实在我看来，中国教育问题的根源在于大学而不是高中。

一个“问题”学生的困惑

在高中的时候，我是一个“问题”学生，之所以获得这个称号，是因为我太爱问问题了，一下课就把老师缠住，把自己积攒的问题全部扔给老师解答，这种爱问问题的习惯，让我的学习成绩飞速的提升，同时也获得了老师的赞扬。因为高中的老师，是希望学生爱问问题的，一个能提出问题的学生，其成绩会明显比那些被动填鸭式学习的学生成绩好，不怕你问，就怕你不问。

在老师的鼓励下，我越来越爱问问题了，被老师当众称之为“问题”学生，并列为典型进行表扬，我曾经一度天真的认为，老师和学生的关系，就是老师教，学生学，然后学生提出问题，老师负责解答。

这一人生观在我上大学的时候被颠覆了，在大学里，我也曾认真学习，积攒了不少问题，等课后的时候找到了专业课的老师，结果老师并没有给我任何解答，只是甩出了一句“你回去看看书，自己好好想一想不就知道了”，然后就急匆匆的走了。

也不是所有的大学老师都是这个样子，但是十个里面会有七八个是这样，剩下的二三个会帮你解答一下问题，但是也不会尽心尽力，问一二个问题还行，多了就烦了。很多老师，上课的时候就是放个 PPT，照本宣科把书上的东西重新念一遍，下了课就赶紧走人。既然上课也就是在听复读机，那为啥不自己看呢。

这个问题，一度让我非常困惑，在大学里，师生关系到底是一种什么样的状态。后来索性就自己看书了，钻进图书馆，什么类型的书喜欢看就看什么，并不拘泥于专业书，因为兴趣被打击殆尽，从知识储备量来说，造就了今天的我，但是从大学专业教育来说，这明显是中国教育的失败。

中国大学教师的无奈

与此同时，中国的大学教师也很无奈，中国的大学老师，其素质和能力绝对是远超高中教师的，大学招聘教师动辄要求中外双博士等等让人目瞪口呆的学历要求。这些老师为什么清一色都变成了毫无责任心的甩手老师，这肯定不是老师个人素质修养的问题。

事实上，中国的大学极其不重视教学问题，大学教师的工资是和职称挂钩的，而职称和论文、科研、项目等挂钩，但是唯独不和教学挂钩，也和学生的成绩和未来的成就不挂钩。

也就是说，这个课啊，应付应付就行了，教好教坏一个样，甚至教于不教都一个样，你教出来的学生未来拿了诺贝尔奖，你也不会加薪，未来成了民工，你也不会减薪。更严重的是，大学的教育是由多个老师负责的，一个总负责人都没有，到底谁教的好，谁教的坏，更难区分了，一起吃大锅饭的唯一后果就是，大家都偷工减料，把所有精力集中在论文、科研、项目，认真教课的老师是傻老师，认真发论文做项目的老师才是好老师，这就是目前中国大学的价值观。

在这样的价值观下，再有理想和情怀的大学教师，也不得不屈从于现实，上课的时候照本宣科的念一遍书本，下课赶紧闪人做自己的“正经事”才是大学教师的日常生活，那种爱问问题的学生直接打发回去，免得其他人跟风问问题耽误自己的宝贵时间。

而对应的，中国的高中，采用教师跟班制度，班主任和主课老师一路跟班负责，一切以高考成绩说话，学生成绩好教师加薪升职，成绩差则减薪降职，这种激励制度下，高中教师当然把学生盯的死死的，碰到爱问问题的学生马上当成宝列为典型，诱导其他学生也来问问题，因为这是提升学生成绩的最有效办法。

中国大学制度的无奈

大学生教学质量差，我们可以甩锅给大学教师，大学教师不认真教学，可以甩锅给大学制度，那么大学制度如此设立，又是为什么呢？

从建国以来，国家对大学的设立目标一直没有改变，那就是为急缺人才的新中国以最快的速度培养出基础人才，支援国家的现代化建设，首先是吃饱饭，然后才能谈得上吃满汉全席。在这种目标的引导下，尽最大的努力，培养出最多的毕业生才是考核校长的重要指标。

如何培养出最多的毕业生，那就是努力扩建学校，努力要招生指标，努力招老师，扩大规模成了最有效的办法，至于培养的质量，没有任何指标能考核这一点，所以学校也就不予考核。所以，我们可以看到，中国的大学教育，就是以速成、流水化批量生产为目标的，大学宿舍就是完全参照军营进行设计的，很多大学宿舍还不如富士康的职工宿舍。

一开始的大学教师，还是认真教一下学生的，毕竟除了教学也没啥事干了，不做这个做什么呢，教的好和教的差，业内也是有一杆秤的，就算博个桃李满天下的名声也好啊。

但是随着大学扩招，大学生数量迅速饱和之后，国家的目标就改变了，基础人才已经不缺了，国家现在缺的是尖端人才，大学的任务从培育基础人才改成了培育尖端人才。这下就麻烦了，麻烦在什么地方呢，考核指标很难定。

尖端人才的一大特殊之处，就是他根本是无迹可寻的，能体现出他是尖端人才，那是很多年之后的事情了，这个考核没法定，如果没有考核的压力，国家也清楚，那下面的人一定是划水偷懒的，不可能取得自己想要的效果。于是国家改了另外一条策略，国家要人才，本质上需要的是科研成果嘛，大学教师队伍里博士那么多，你让他们给我做科研出成果不就完了嘛。

所以，大学教师的考核指标诞生了，以论文、科研、项目作为职称评定来激励这些教师多出科研成果，至于教学则不列入考核，毕竟那只是为了培育基础人才，随便教教就完了，大差不差。

但是后果大家也清楚了，大学的教学乱成一团糟，老师划水偷懒，学生睡觉梦游，到了考试的时候大家心照不宣的放大部分人及格过关，保证绝大部分学生都能拿到毕业证不伤和气和面子，天天打游戏的学渣也能顺利毕业，这就是大学教育的现实。

至于国家期望的科研成果，那就不谈了，十年来大学教师队伍里写出的论文，申请的专利，弄出的发明车载斗量，一堆人借此评上了教授，然后这些科研成果直接被扫到了垃圾堆里作废，因为他自己也清楚，这些东西就是拿出来凑数量的，科学技术届也有一个重大的 BUG，那就是质量好的论文是一篇论文，质量差的论文也是一篇论文，考核看的是数量，而不是质量，因为质量无法审核。

但是实际上，一篇能拿诺贝尔奖的论文，其含金量超过几十万篇这种为了凑数而生产的垃圾论文。

大学教育怎么改

那么既然我们知道了大学教育的问题，应该怎么改变他呢？

首先，大学教师的考核应该和科学研究进行分离，教师这个职业就是拿来教书育人的。如果以科研为目的，应该把这个人才塞进研究所做事情，过几十年你科研水平高，做出了成果，大学可以聘请你为名誉教授来开几个讲座，如果你没做出什么成果，那就继续呆在研究所里做研究吧。

做科研和做教育，要严格区分，就好像让打篮球的姚明去赛跑一样，虽然都是体育界的，但是那是二码事。

然后，一定要清洗西方快乐教育/素质教育的余毒，中国大学教育之所以如此糜烂，还有一个重要的原因就是迷信西方的素质教育，认为西方之所以出人才，是因为西方的快乐教育和素质教育，而中国之所以不出人才，是因为填鸭式教学和应试教育。既然中国的高考是应试教育无法改变，那么中国的大学生就应该进行素质教育来进行弥补，中国对素质教育的理解肤浅到什么地步呢，目前在中国大学的教育实践是这样的，老师不教学+考试不严格+学习全凭自觉=素质教育。

而我们对比下欧美的素质教育，美国的名牌大学，其教学是非常残酷的，所有人都战战兢兢的拼命学习，因为淘汰率非常高，每年拿不到毕业证的人都占据总人数的一小半，考试异常严格。我们自己鄙视以考试为目的的教育，但是外国大学却把考试当成教学的法宝，美国的名牌大学非常类似于中国的高中。

那么欧美的素质教育，上课在玩耍，下课没作业是假新闻吗？不是，是真新闻，不过只是片面的真新闻。欧美有二种高中，一种是私立中学，奉行应试教育。一种是公立中学，奉行素质教育。

优秀的私立中学，考试异常严格，训练异常残酷，很多都是寄宿制，见过中国的高考工厂毛坦厂中学嘛，就是那个调调，只不过硬件档次高了很多而已。英国的伊顿公学、斯科特中学等私立中学，聚集了全英国 7%的学生，却占据了牛津、剑桥每年录取的学生中的 50%，而未来，他们将成为英国的首相、国会议员以及社会精英人士。要想进入这些私立中学，需要孩子从 6 岁开始就进行准备，经过长年严格的学习和考试，才有希望进入，其学习难度，丝毫不亚于中国的高中。

而同时，英国和美国的公立中学，奉行快乐教育，也就是素质教育，国家仅提供基础的教育，你如果愿意学，国家给你提供了知识的来源，不会剥夺你受教育的权利，如果你只是想来混日子，国家也不会理会你。

这才是欧美所谓素质教育的核心原因，你可以统计统计，欧美的成功人士，有多少是从残酷教学的私立中学出来的，有多少是从素质教育的公立中学出来的，教育从来都是精英的权利，他们设立公立中学，提供一个基础教育，只不过是为了堵上民众的嘴而已，给你机会了，不学那是你自己的事。

而这种素质教育，被拿来当中国大学的法宝，以为靠这个就能出尖端人才，简直是滑天下之大稽。在欧美的大学生头悬梁锥刺股好好学习，天天担心自己无法毕业的时候，中国的大学生躺在素质教育的庇护下打游戏泡妹子，中外尖端人才的缺口不仅没有缩小，反而急速扩大了。

中国身为考试的老祖宗，至少在大学教育上，考试难度不能比英美名牌大学还小吧，中国高中生碾压美国高中生，但是中国大学生被美国大学生碾压，这种教育乱象也该改一下了。

考试是督促学生学习的重要手段，严格考试的教育氛围未必能培育出顶尖人才，但是不考试不学习的教育氛围，一定不可能培育的出顶尖人才，就算有个别的好苗子，也被舍友拉去天天打游戏了，毕竟人是群居动物，很容易受到周围环境的影响。

中国的强大靠大学

在美国，二级分化非常严重，商场的收银员连二位数的加减法都要靠计算器，与此同时顶尖的数理人才横扫全球。他们最大的区别就是前者是美国的公立中学毕业的，是快乐教育的产品，后者是美国私立中学和私立大学残酷教学的产品。我希望中国学习的是后者，而不是前者。

相比初中高中的基础知识，大学才是培育真正科研人才的地方，大学里的教材，应该加入最新的科学成果，大学里的教师，应该和教学成绩挂钩，大学里的学生，应该每天努力学习。

目前中国的大学生，每天混日子，毁了自己也毁了中国的未来，每个人的孩子将来都会上大学，每个人的生活和中国的国力也息息相关。

中国的大学教育真的应该改一改了，我这里真诚的呼吁大家发出自己的声音，为中国的大学教育改革推动，出一份自己的力量。
