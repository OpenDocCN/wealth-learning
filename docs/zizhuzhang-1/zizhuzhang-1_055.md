# 我兜里没钱，你让我拿什么去支持中国制造！

![](img/a0c3c74abf32d2c865a2c3db4a71b0a0.jpg "跳跃猫引导关注")

我们都知道，振兴实业是国家喊了很多年的口号，但是很可惜，怎么喊都不见效。不管银行如何定向降准，这些钱兜兜转转最后都流入了房地产，死活都不愿意跑去支持实体经济，其中的核心原因，是中国普通民众的钱包实在太瘪了，在六个钱包全空的情况下，普通民众是没办法去支持中国制造的，不是不想消费，是没钱消费。

萎缩的实体经济

在中国，实体经济投资增速连年下降，如果刨除房地产基建的部分，那实际增速下降的更严重。所谓实体经济，基本就是指盖工厂，买机器，招工人，做产品那种经济体系，也称传统经济。

实体经济投资增速逐渐减缓代表什么，代表工厂主不愿意扩大再生产了，维持原规模就心满意足了。换言之，这个行业产生的 GDP 和容纳的就业人口不会再逐渐增长了，开始停滞甚至下滑。，也就是说，中国这个世界工厂的规模膨胀到了天花板。

这下政府急了，中国经济的高速增长掩盖了一切改革阵痛，正是因为蛋糕越做越大，社会各阶层才能全部满意，一旦经济增速下滑，则社会危机可能爆发，也就是俗称的中等收入陷阱。而中国号称世界工厂，一身 GDP 基本都系于实体行业，实体涨不动那中国就涨不动，新经济和虚拟经济对中国有补充作用，但是还没发展到那个能扛起重担的规模能力。

所以，每年政府都想尽办法的给实体经济输血，想让银行贷款给实体经济，多次定向降准扶持小微企业都体现这一倾向，只要企业主贷款，就代表他在扩大再生产，不然他贷款干嘛你说对吧，只要企业扩大再生产，经济引擎就能重新恢复运转功率。

实体经济不缺钱

但是实际上，不管定向降准如何“精准”的给实体经济放款，这些企业主拿到钱之后还真的不一定就去扩大再生产了，很多时候兜兜转转，全去投资房地产了，为什么会出现这种情况，因为实体经济其实并不缺钱。

在八九十年代，在中国做实体经济堪称暴利，暴利到什么程度呢，白手起家成为亿万富翁，那不是一个梦，而是很多人都已经做到的现实，做到这种神话的老板，很多也只是小学文化而已。换句话说，只要你敢投资，你就能赚大钱。年化收益率 100%，对于八九十年代的工厂而言，那真的是稀松平常的事情，没有这么高的利润率，哪能催生出这么多富豪。

这个时候，银行的贷款全被实体经济给拿走了，银行也乐于放贷，这么多土地厂房设备做抵押呢，一投产就有现金流做保障，安全，省心，越大的工厂，银行越喜欢。

但是到了新世纪之后，工厂的收益率就逐年下滑，到了 18 年，实体经济的收益率普遍下跌到 6%左右，换言之工厂主拿自己的钱去投资，还不如直接存银行理财，至于用 8%以上的贷款成本去投资，那更是亏本买卖。

所以银行贷款无论如何都无法流入实体经济，这个是必然的，不止银行贷款流不进去，这些工厂主自己多年积累的家族财富也都拒绝再投资，工厂主关闭工厂改行炒房这种新闻，简直屡见不鲜。

所以，要想让实体经济扩大再生产，只能提升实体经济的收益率，至少要扩大到平均 10%以上，才能吸引工厂主自己再投资甚至贷款去投资。

那么，如何才能提升工厂的收益率，很简单，扩大需求，只要产品供不应求，工厂主就会涨价，只要产品涨价，收益率自然就上去了，只要收益率上去了，振兴实业轻而易举，压根不用国家费心，甚至都有可能吸引房地产的资金回流到实体经济。

穷困的老百姓没钱去买产品

现在我们把问题的根源梳理完了，只要能扩大产品销量，那么就能轻而易举的完成振兴实体经济的目标，让实体经济投资增速重新飞跃。那么如何扩大中国制造的产品销量呢，第一是靠外贸，第二是靠内需。

靠外贸拉动经济，是中国走了几十年的老路，现在这条路逐渐走到极限了，频频发生的贸易摩擦说明中国的巨大外贸逆差已经达到了外国无法容忍的地步，所以靠扩大外销来增大中国制造的产量，这个是行不通的，按现在的趋势，逆差是肯定要缩小的，不管是增大进口还是减少出口，都会对中国制造形成挤压效果。

那么只能靠扩大内需了，这条路其实也喊了很多年了，但是这条路的一个最大问题的就是，中国人没钱啊，基层群众实在太穷了，被房地产给榨干了所有财富。

先说说那些“有钱人”，当然不是指马云那种财富自由人士，而是说那些在一线城市有一套以上住房，或者持有房产市值超过 500 万元的人。这些人算有钱人吧，在好多人眼里是非常成功的人士了，那么你可以做一个抽样调查，这些人在银行有多少现金，月入多少，月供多少，多久出去旅游一次，每个月家庭能拿出多少钱来消费，这些人立马就原形毕露了，和普通的白领相比也差不多。看似家庭财富几百万，实际上银行存款常年就几万，背着百万级的房贷，月供上万，每个月实际可支配的也就几千元，根本没办法靠这些人提振内需。

这些人不是不想扩大消费，是没钱消费，家庭所有的财富高度依赖房地产，都是纸上富贵，是没办法拿出来花的，他们的确想每天买一杯鲜榨果汁，但是想了想，还是每星期买一杯勾兑果汁算了。

这还是在房地产上行期，如果来一次香港日本那样的房灾，直接腰斩再腰斩跌个 70%，我不知道有多少房奴能熬个十年二十年，直到房价回复原价的那一天，现在都不舍得消费，那个时候更不舍得。

然后看看那些刚刚掏空“六个钱包”凑上首付的年轻人，这些人是最穷的，比那些首付都凑不齐的人还穷，因为他们直接耗尽了整个家族所有的力量，付了首付之后，出门打个出租车都要心疼半天，这样的人你指望他来扩大消费？别说勾兑的果汁了，他能保证稀饭榨菜的消费就不错了，不仅自己如此，他还一口气把整个家族都拖入了勒紧裤腰带的日子。

最后，是那些连首付都凑不齐的人，这些人最大愿望就是凑够首付成为房奴，日常消费保持个最低值就好了，绝无可能大肆消费。如果你在日常生活中碰到一个买不起房子，还天天买这买那，月月出去旅游的人，你是羡慕这种人活的潇洒，还是暗地里骂一声败家子呢，我想后者肯定居多。

枯萎的中产阶层无力扩大内需

所以，除了极个别已经实现财富自由的人群，绝大部分的中国人是没有资格大肆消费的，页没资格去扩大内需支援中国制造。中国的中产阶层，都是一些纸面富贵或者连纸面富贵都没有的人，他们就算有一套，甚至二套三套房子，他们敢卖了房子去消费吗？

不敢，那么他们的房子不管市值多高，都是虚幻的，无法对内需形成刺激作用，他们其实不算有钱人，他们真正的消费能力，就来源于他们每个月的工资，而且还要扣除月供的部分。。。

整个房地产行业，已经吸干了中产阶层的精华，原本应该是内需主力的中产阶层如今已经枯萎，丧失了他应有的消费能力。

越是内需不振，越是要拉动房地产来刺激 GDP，越是拉动房地产，越是内需不振，这已经形成一个恶性循环，全民勒紧裤腰带去玩这个击鼓传花的地产游戏，拿什么去振兴中国实业？

做实体经济最佳的境界，是供需二旺，老板有钱赚，工人有钱花，而如果变成老板没钱赚，工人没钱花，只靠房地产支撑经济，这堆钢筋水泥它真的支撑得住吗？

没有人不想过上好日子，没有人不想去买好东西，但是兜里的钱都被人掏光了，你让我拿什么去支持中国制造。
