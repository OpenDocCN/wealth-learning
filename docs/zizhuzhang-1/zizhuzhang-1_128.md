# 普通人如何才能年薪百万

紫竹张先生

年薪百万是绝大多数人一生的梦想，也是成功人士的标志性门槛，但是实际上，达到年薪百万是非常困难的，从上市公司的财报中可以观察到，在中国，百万年薪只有大公司高管才有可能达到，这些人不仅需要具备强大的管理能力，而且至少有 3 亿元以上的销售业绩才行，同时，只有年收入数十亿的上市公司，才会给高管开出百万年薪。

但是对于普通人来说，成为上市公司高管是不太可能的事情，那么对于从事普通工作的人来说，有没有可能百万年薪呢？有可能！曾有人把赚钱分为几个档次，卖体力的，卖知识的，卖注意力的，卖信息的，其实这些都不完全对，因为真正能发财的，其实是卖天赋的。

所谓三百六十行，行行出状元，只要你在某一个领域达到了登峰造极的地步，你就能够获得百万年薪，而如果你没有天赋，例如你选择比较高端的卖注意力，大明星能成功，你未必能发财。对于普通人从事的普通工作，如果你能做到极致，一样能发财，我们看下面几个例子。

化妆师，是一个普通的工作，一般的年薪就 10~20 万吧，但是下面这位化妆大师，年薪能达到 100-300 万级别，因为她几乎做到了化妆的巅峰，下面是她火爆网络的视频剪辑其中之一个妆容。

化腐朽为神奇，手艺达到这个地步，这已经不是在化妆了，这是在变魔术，古代易容术也不过如此了，化妆在她手上已经成为了一门艺术，让人赏心悦目，叹为观止，她能获得高收入，一点不奇怪。

下面这里还有一个百万年薪的调音师，这种水准也是能够化腐朽为神奇的，堪称业内精英。

化妆师和调音师，都是一份普通的工作，但是普通水平，和顶尖水平，那一定是差距巨大的，顶尖选手的其薪水并不亚于上市公司高管。这就告诉我们，但凡是含有技术差别的行业，其收入都是差距巨大的，如果你能做到常人所不能做到的极致，那么你就能获得梦寐以求的高薪水。

所以，我的观点就是，你究竟是卖知识还是卖注意力还是卖信息，这都不重要，只要你的天赋在这上面，能充分发挥你的天赋潜力，你就能年薪百万，所以我称之为卖天赋。

我再举几个例子，刘翔是跑步天才，跑出了年薪千万，按照刘翔的身体素质，去打篮球也是毫无问题的，至少可以碾压普通大学生。但是刘翔的天赋在跑步上，他能把跑步给跑到了极致，所以他才能年薪千万，如果他去打篮球，还会有这么高的收入吗？不可能。同理，姚明去 NBA 打篮球，年入数千万美金，按姚明的体格，扔铅球也是碾压 99%的普通人的，但是姚明会选择去扔铅球吗？如果他去扔铅球，估计年入几万美金就不错了。

而与此同时，所谓的高薪行业，比如金融圈、软件工程师，是不是说谁进去都能年薪百万呢，当然不是，低阶收入的也是一大堆，如果你没有这行的天赋，那么进去也是金融民工和程序猿。而对于一将功成万骨枯的演艺圈，甚至游戏直播圈，普通人进去九死一生的行业，如果你有天赋，那你获得的收入是远远超过其他所有行业的。

所以片面去追求做什么赚钱，卖什么赚钱，这种思维是错误的，而是先问问，你适合做什么，让自己在最擅长的领域，把能力锻炼到极致，然后等风来。我个人认为，选择什么行业并不是你赚钱的主要因素，因为百万年薪的人群永远只占 1%，只有最精英的那批人能拿到。所以选对你天赋所在，才是你赚钱的关键。男怕入错行并不是说你选的行当没前途，而是你压根就没有所选行业的天赋。找到自己天赋所在的行业，然后全力投入进去，发挥你最大能力，成为这行业的精英人士，才是拿到百万年薪的奥妙所在。

除此之外，还有一些不能用钱来衡量的成功人士，他们从事的也是很普通的工作，但是其价值，远远不是年薪百万能来衡量的，他们做到了国宝级。比如读书读成钱学森、制药制成屠呦呦、种地种成袁隆平。

上天是公平的，给你关上一扇门，就会给你打开一扇门，每个人都一定有自己独特的天赋，那么怎么知道自己天赋在哪呢，很简单，你有哪方面异于常人，对于某一个行业或工作做的特别快特别好特别专注，而且发自内心的喜欢他，那么这个行业就是最适合你的行业，深度耕耘，用十年时间把自己的能力锻炼出来，成为这个行业的翘楚，你就必定年薪百万。

每个人都有自己独特的天赋，找到它，发挥它，普通人也能做到百万年薪。
