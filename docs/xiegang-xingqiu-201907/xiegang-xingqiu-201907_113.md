# 如何戒掉游戏？

小猿 : 如何戒掉游戏？ 之前分享过一个方法，就是在玩游 戏的时候把声音关掉。不知道大家有没有实践看看。 今天 这边再分享一种也很有效的方法，就是每次玩游戏后，都 把游戏彻底卸载删除安装包。 因为每个人都是有惰性的， 我们都知道现在一个游戏都很大，手游 300 多 M，一些电脑 游戏大的几个 G，假如每次你想玩游戏的时候，都要去下载 这么大的软件，然后安装加载，一来二去，大多数人都因 为会闲麻烦，就不在去玩这个游戏了。 我个人之前玩王者 荣耀也上瘾，经常用的也是这个方法，玩了就卸载，一年 期间，我重新下载下来不超过 5 次，现在技术水平跟不上身 边的朋友，自然也不想再去玩了。 总之，每个人都是很懒 惰的，用懒惰克服玩游戏，绝对可以戒掉游戏瘾。不信， 你试试看！ 你还有什么好办法，欢迎留言分享。
