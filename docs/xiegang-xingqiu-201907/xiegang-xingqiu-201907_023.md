# 本人在某 211，现

WHB : 本人在某 211，现在大二信息安全专业。这是昨天 (2019/6/14)绿盟科技来到我们软件学院招暑假实习生给出的一 张 PPT，我给拍了下来。负责人经理对我们说，假如你还很迷 茫，就按照 PPT 上的内容去做，这也是他们企业找人的标准。 首先最重要的是培养学习能力，你要善于思考，善于学习， 把别人要学两天的东西花一天学，还要比别人学的好才行。 这不禁让我想起期末考试的学习效率。平时甚至跟着老师讲 还听不懂的东西，反而在临近期末三天内可以预习+复习完一 本书，考试还能考得不错。事实证明在高压之下能够出效 率，毕竟挂科 Very serious！那么设想，假如平时都能已这种 效率学习，什么样的技术学不好呢。第二个是文档编写能 力。这个看起来不难，但真正做得比较好就难了。首先你得 有话可说，写出一些真正有关与你项目的东西。最为一名 211 计算机大二学生而已，这一点我反而感到尴尬，平时基本上 能百度谷歌的全部用搜索引擎来解决，没有一点自己的主见 或感谢在实验报告里。这样做的结果就是，文档可以快速完 成，但自己真正获得的知识少之又少。如在学习操作系统之 Linux 上级课时，大部分人都是复制粘贴 Linux 命令，程序，然 后在 Ubuntu 上运行截图作报告，花个十五分钟解决两节课要 学习的知识，还完成了作业。岂不美滋滋。但你知道吗，但 我最近在搞搬瓦工 vps 的时候，搭建了一个 Ubuntu 在云上。本 以为学过的知识，应该不复杂，但是真正到了上手的时候， 发现之前学过的命令，运营都忘了!这也就是给我们的启示是 平时还是得踏踏实实做事。做文档也是如此。第三个要求是 沟通能力。任何人都得于其他人沟通，如何简明扼要的说话 就变得很重要。负责人经理还说假如你技术做不下去，你还 可以走营销的路子，毕竟每一件生产出产品都得推广于市 场。之后是你的专长，我们学校给我们开设的课程是全栈式 课程，从前端到后台，从数据库到 Linux 运营，还有我们的安

全渗透。那么你到底在学校学得怎么样，这个就要你把握住 自己的专长，也是你今后的饭碗。像我而言，我在前 端，Linux 和安全渗透方面学得可以，我也不准备考研，大三 之后就得出来工作，故在大三的这一年我可能会花大量的经 历来搞安全和前端，我心之所系的是安全之路，但也得找前 端作为后路。然后谈了一下证书和实习经历。项目经理那证 书打了个比方，证书就像钞票，当你拥有的时候，马云就说 过他对钱没兴趣。但当你没有的时候，你就知道了证书的重 要性。实习经历也是如此。企业会优先考虑有证书和实习经 历的同学。最后，他也总结出他找人的准则:能力＞证书＞实 习＞其它。而你的能力如何提现呢，他说一是用证书来体现 二是拿你在校期间做的项目来体现。总之，有能力的人最好 去考相关证书。之前听到有人说什么证书不太重要，学历不 太重要，现在想想有 985 的学生很有可能又不会要 211 的学 生，自己还是得努力才能实现自己卑微的梦想。这也就是我 今天花了 4k+报考 nisp 的原因。 以上是我对于绿盟经理所说的 感想，今天要是第一次进小猿星球，看到小猿说要将自己想 写的话写出来大家一起参考，就突然一下子写了这点东西， 由于经验或经历的不足，请各位大佬多多海涵，小弟在此不 胜感激。

2019-06-15
