# 为什么我建议你远离

Brililant : 为什么我建议你远离“肥仔快乐水” 毫不夸张的说， 死宅，是我们进步的最大绊脚石。 经常会听到别人说，给我 一张床，一个 wifi 账号密码，我可以永远不出门。好的，你开 始废了。 在压力徒增、生活便捷的今天，宅的现象越来越普 遍，但是宅在不知不觉中阻碍了我们前进、发现、思考。 我 不希望你也是这样的人，如果你也死宅，那么请慢慢改变。 宅，就和周围的人没有参照和对比。 说起参照和对比，不由 得想起这几年出现的钉钉步数、微信步数排行版，这类的功 能会不由自主让我们和周围的人进行步数对比，让我们有意 识的走出房间，散散步，跑跑步，爬爬山。挺好的一个功 能。 在学校里，你的舍友考了 xx 证书，你是不是也会琢磨着 考个合适自己的 xx 证书，提高自己的市场竞争力。 在职场 上，周围优秀的同事 kpi 很高，我们是不是看着眼红的同时， 自己也会想努力一把，提高自己的 kpi，准备下一轮的升职加 薪。 过年回家，我们都会见见亲朋好友，总会听到，某某人 今年挣了几十万，城市买房了。我们听着就像打鸡血，默默 定下小目标。 宅，关闭了外面的窗户，没法了解真实的外 界。 我们经常刷新闻资讯和朋友圈，这不是真实的世界，都 是作者特意选择的图片以及文字。也许你会问，新闻怎么不 是真实的，新闻都是别人想让你看到的，负面的新闻也许你 看不到。所以“上网姿势”还是很重要的。我们唯一可以做 的，就是提升个人思维认知，信息的真假自行判断。所以我 特喜欢一句话，你越成功，看到的真相越多。 宅，让人失去 交际能力。 交际能力在职场的作用毋庸置疑。不知道大家有 没有发现，不爱说话的人，基本都是宅的。作为技术人的 我，反观我的周围，因为工作的性质，我们完成任务可以不 用说太多的话，这也导致我们养成了不爱说话的坏情况。所 以多数技术人都是相对比较宅的，如果想在职场风声水起， 升职加薪，想在公司侃侃而谈，希望大家日常多出去走走，

约约朋友，约约妹子，多给自己一个锻炼自己的机会。 之所 以写这篇文章，是给大家以及自己的建议，有则改之无则加 勉嘛。远离宅，多出去走走，看看，也许你就发现商机了， 也许你就找到女朋友了。 我还是建议大家多说话，你就算都 是废话也没关系，起码你已经话多了，慢慢提高自己的语言 质量。在星球中多发言，多和伙伴互动，多结交朋友，多发 表自己的一些日常看法，养成写日记的习惯，让更多的人可 以看到，大家也会给出有价值的意见。
