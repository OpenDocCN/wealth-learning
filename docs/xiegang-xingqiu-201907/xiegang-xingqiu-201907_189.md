# 什么叫项目管理，项

PM 小曹 : 什么叫项目管理，项目管理就是在有限的资源下， 按照规定的时间，在不超成本的情况下，圆满完成项目。其 过程分为：启动、规划、执行、监控和收尾；其中规划贯穿 整个流程，起到跟踪和监督的一个作用。 实际工作中，我们 也需要注意几点： 1，需求一定要明确，需要反复沟通，不能 遗漏任何问题，能签字确认的都要签字画押，免得后期扯 皮； 2，要做好资源规划和人员分配，作为项目经理，一定要 预估好整个工期，可以采用 peter 法、或者类比法，然后按照 整个大的工期做进度计划； 3，做好问题跟踪，有问题，一定 要记录，并实时进行跟踪，看到底解决了没有，如何解决 的，详细记录； 4，做好风险控制，前期做好风险登记册，想 好应对策略，当风险触发时不至于手忙脚乱，整个过程中一 定要实时跟踪； 5，团队士气，一定要把控好，每个人安排工 作尽量合理，团队成员之间要多沟通，多协调，团队建立从 形成、震荡、规范、成熟、解散，每个过程都不一样，一个 高效的团队能够事半功倍。 这些说的比较泛，经验都是慢慢 摸索出来的，每个人都有每个人的方式，但最终的目标都是 一样的，都是要把项目做好。 别人教不了如何做项目经理， 只能提供自己的相关经验，遇到问题具体分析，因为每个人 碰到的情况都是不一样的，只能作为参考，而不能拿来照 搬，拿来主义不怎么靠谱。
