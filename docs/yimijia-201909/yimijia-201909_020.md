# 一个人的认知水平，

谁也锤不了我 : 一个人的认知水平，如果与市场的平均水平相 当，则很难赚取到超额利润，甚至平均利润都很难赚到。

你的认知水平，决定了你未来能赚到的钱。比如你目前的认 识水平是“4 万单价”的水平，那么你买 4 万单价的盘，你就是 没赚到钱啊；

如果你的认知水平达到了“6 万单价”的水平，你自然就会在未 来收获这部分红利。

你们觉得有无道理。[玫瑰][玫瑰] 2019-09-09(3 赞)
