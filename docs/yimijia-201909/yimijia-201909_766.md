# 锤哥你好， 我现在

2019-03-25

回答：优选方案 1。广州就买小面积学位房挺好，比如天河北 鸿翔大厦小户型、比如金润大厦，还比如天府路小学的理想 青年荟；深圳的话，根据资金量大小，我觉得首选宝安，西 乡碧海片区(3 赞)
