# 这一期的回答数量虽

这一期的回答数量虽然不多，但是我个人觉得价值很大。很 多人在面试的环节碰到一些开放性问题不知道怎么回答。

又有很多同学明明会的题目在面试的时候没想起来。 还有很多同学确实遇到了不会的题目，但是相同不会的题

目，有的人却能脱颖而出。

为什么呢？因为面试本来也是一件有技巧的事情。建议大家 把几十位知友的面试经验都看看，绝对的有帮助。

晚点我也会自己总结一套。 上周出的这个问题，我本来以为很开放了，但是回答的数量

却不多。大家能来说说什么原因吗？是因为很多人没有面试

经验？

1.你在面试的时候碰到最棘手的问题是什么？相信你在面试之 后也做过了搜索总结，那么在你心中你给这个问题的标准答 案是什么？

2.你在面试别人的时候，最喜欢出哪道题？你心中对这道题有 什么标准答案？或者希望听到什么样的回答呢？
