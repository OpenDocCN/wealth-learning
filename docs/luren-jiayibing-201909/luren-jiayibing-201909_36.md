# 大家无论是在喝饮料

路人甲 : 大家无论是在喝饮料还是某些白酒啤酒的时候，会碰 到一些商家做活动：很大一部分都是在瓶盖印上一个二维 码。扫描这个二维码，可以领到一些红包。

那么请大家思考一个问题：这些商家在瓶盖印二维码领红包 的意图是什么？

我自己知道原因有 2 点，大家可以发表主题回答这个问题。如 果跟我心中想的 2 点一样，100 元红包伺候。
