# 还有一个感悟。很多

老李校长 : 还有一个感悟。

很多小伙伴尤其是那种性格特别老实的，毕业干的工作也是 特别稳定当然收入也是不怎么高的那种，这些小伙伴问我一 个共同的问题，就是我是如何走上创业之路的。

今天我就不从什么行业什么技术角度讲。 我今天从心态讲： 我对职场，对公司是没有信任感可言的。 这个感觉来自我的经历。

我在美的第二年因为和新来的经理张崧也不知道怎么得罪他 了，把我等于变相开除了。

之后我负气开了一个小破咨询公司黄了，没钱了，又跑回美 的洗衣机上班。

美的洗衣机的待遇收入和美的空调没法比。

1700 一个月的工资，1500 的费用，年终奖不到 2 万，让人没法 干，美的空调工资 2500，费用 2500，年终奖 6 万，同工不同 酬。

美的洗衣机的经历让我对这种美的这种大企业的无耻加深了 印象，把人不当人，那个时候起，我对大企业完全没信任度 可言了。

然后我跳槽不干去了海南跟一个小老板打工。

之后的联想，也是以残酷考核而出名，每周总结完成任务情 况，2 个季度完不成任务走人，精神压力很大。

这职场一路走来，都是跳槽，离职，断断续续，职业生涯像 个破屋子一样，到处漏水。

我对职场一直没有信任感，也没安全感。

这种长达十年的不安全感，使我对创业这种不安全感的事， 切换起来很平滑。

就是反正都是死，不如创业死的轰轰烈烈。

假如我职场走的很顺，我不会出来创业的，我想好好报效朝 廷，无奈朝廷总是要开除我。

创业的确是每天都有不安全的感觉，但是这种感觉很熟悉， 很多年前在职场里被人开除，被离职的时候我早领教过了。
