# 接上一条。成功也在

老李校长 : 接上一条。 成功也在挑选自己的主人。 怎么挑选，制造各种各样的困难，障碍，难关。

并且这些困难，障碍可能都是不仅仅靠吃苦能解决的。 往往有时运，眼界，毅力，厚黑，操作，人玩人的方式。 最终是哪把钥匙解开了这个谜题。 这个本事都是一个考验。 所以我从来对考研，看书，考证之类的努力不屑一顾。
