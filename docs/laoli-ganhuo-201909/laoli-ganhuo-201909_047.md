# 我表弟是我公众号的

老李校长 : 我表弟是我公众号的兼职助理，他负责老秘书，源 靖还有我的排版，还有一些杂事，兼职，自己有一份保险公 司的会计工作。我给他发小几千，不多。比不上我团队里其 他人的收入。我昨天给他说，我给你发的钱，其实就是买你 的时间。

跟他聊完，我在寻思。

月薪 1 万以下的工作大部分出卖的时间，这种工作很容易被取 代，替换。而大部分人就在这个区间，别人买你的时间，做 一些器械性重复性的但必须人完成的工作。假如我表弟这都 干不好，我立马马上可以找一个人来替换他，发的还是这个 钱。

月薪 1 万以上到 3 万，别人买的就是你的管理能力运作能力还 有操心责任心。比如给我做互推的助理，这个特别考验数据 能力，沟通能力，拓展能力，责任心。就拿这个数。

月入 3 万以上 8 万，别人买的就是你的创造力，能给新媒体供 稿写出十万 百万阅读量的人，就可以拿这个。

年入 100 万以上，其实你和别人就是合作关系了，这个收入类 似于分钱，比如整容主刀医生。

每突破一级，都不容易，得照之前我说的，一次一次的毁灭 自己，改变自己的思维方式，觉得自己以前是个傻逼的感 觉，做出以前根本想都不敢想，做都不敢做的事，比如给自 己动刀子。

我强调逆袭是革命，而不是改良，我视一切小打小闹的举措 什么考个证，换个工作，现在越来越没效用了。

如果那么有效，你们还会来找我么？早就被人发现了。 大部分的招数，都是无效的。 发财之路，得先革自己的命。
