**【灰产圈】高端社群人脉分享会圆满举办成功！**

**北京时间 2017-6-19**

由灰哥主持的【灰产圈】高端社群人脉第一期分享会  八点正式开始  

由百十位会员轮流分享自己的个人简介，资源和从事行业以及需求，

会员由 传统企业管理人员，自媒体从业者，互联网行业人士，传统制造业，广告行业，微商行业等等的行业大咖和互联网草根组成，

从年收入十几万到几十万数百万不等的，社群此次分享会的目的在于  

实现 会员之间的相互了解，资源谋合作。人脉谋共享，在 2017 年共享经济的时代背景下，实现 项目 人脉的双赢，

后期会邀请互联网个行业大咖做分享会，敬请期待，以及线下的交流会。

希望在 2017 年，由灰哥组织的灰产圈高端社群能给会员们带来思路 收入 人脉的 巨大提升。
