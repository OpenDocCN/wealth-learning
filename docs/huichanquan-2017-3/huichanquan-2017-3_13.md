**一.什么是微信”炸群“？**

微信“炸群”，是指利用特殊字符的系统特性，人为反复多次叠加，导致点开这个内容时，系统性能短时间被大量消耗，从而出现卡屏或软件崩溃。

别人发了消息过来，你点开后，程序就被弄闪退了，甚至把手机弄死机。

简单来说，就是你的手机 CPU 瞬间被大面积占用了。

微信团队在声明中表示，“炸群”并非微信系统漏洞或黑客行为，跟病毒或木马无关

**二，"炸群"的原理**

它主要是先利用代码软件或者手动在群里发布大量的垃圾内容，比如说超长消息、大量特殊字符、高帧数动画等等，然而手机的处理器在短时间内无法承受这么大信息量的处理，就会因此变得卡顿，甚至出现闪退或者崩溃现象。

![](img/692072479b2acc236f316bd518ca6368.jpg)

**三，‘炸群’的方式**

超长表情雨

超长特殊 Unicode 字符

高帧数动画表情

![](img/abd0e54c484eba35258046e9fb7ed97e.jpg)

**四，**微信炸群怎么解决?****

可暂时断开网络，重启微信，在微信消息列表中长按此群名字或左划，选择“删除该聊天”，历史群消息(包含炸群信息)会被清理;

可登录微信电脑版，对新的炸群消息进行删除或退群操作，并及时通知微信群主移除发布“炸群”信息者，以免其他用户被“炸群”信息影响。
