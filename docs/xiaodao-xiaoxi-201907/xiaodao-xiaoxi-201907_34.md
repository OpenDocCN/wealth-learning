# 荐书：《20 世纪

Fenng : 荐书：《20 世纪简史：从无线电到柏林墙》 最近看完 的书里。这本书还是值得读。过去的 20 世纪，我们知道发生 了一战、二战、冷战、全球化。但一战产生的根源，希特勒 为什么能崛起，欧洲国家的形成，苏联的崩溃，很多事情的 逻辑不是很清晰。 这本书不是百科全书，很薄。作者能把不 少事情阐述清楚，很有笔力，给了我不少启发。
