# 微信、QQ 同号的要当心了！多人已中招……

相信很多人都有 QQ 号被盗的经历 

一般通过申请找回就可以了

然而一旦你的微信和 QQ 同号

而 QQ 密码又被盗

你的微信号、以及微信上的亲朋好友

将会处于危险境地

![](img/2674019b343e0ad73d0130462f299988.png)

日前

四川省成都市武侯公安分局

公布了这样一起

典型的电信网络诈骗案件

成都市民齐某报警称

自己在 QQ 群内

**点击了一位网友发出的链接**

**仅仅几分钟后 QQ 就下线了**

自己当时并不在意并进行了重新登录

然而很快，微信号也被下线

当齐某通过手机重新设置微信密码

**才发现在微信下线的时间内**

**有几位亲朋好友**

**陆续被骗走了数额不等的款项**

嫌疑人作案过程![](img/22de6dfdeea9715b4173319ba3c1be91.png)

民警随即进行了调查取证

复盘了嫌疑人作案过程

**嫌疑人先是通过在 QQ 群内**

**发布木马病毒链接**

**随机盗取受害人 QQ 号密码**

**再尝试用 QQ 号登录微信**

当登录成功后

嫌疑人便向受害人好友群发信息

**以“帮忙借款给某朋友”的话术**

**骗好友们转账至陌生账号**

![](img/5414f3bfb79948fab4e800f662952143.png)

由于微信上大多是至亲好友

不少人没有丝毫怀疑

立即就转款了

被骗取的金额高达 17 万元

骗子的“诱骗术”![](img/22de6dfdeea9715b4173319ba3c1be91.png)

![](img/a406e03c2d9dedb3431201056b290004.png)

派出所在进行大数据比对梳理后

辗转查到嫌疑人实际地址在广东

案发 7 日后

嫌疑人在老家被抓获

随后，警方循线深挖

目前

3 名嫌疑人已被派出所刑事拘留

![](img/cc7c89b7a57be121e5b65ca664fc4a26.png)
