# 思考是一件很难的事

池建强 : 思考是一件很难的事情，人们为了避免深度思考，会 去做很多不经思考的尝试，东一榔头，西一棒槌，然后美其 名曰这些叫做试错和迭代。 其实是缺乏深度思考能力。很多 人做错了事会自我解嘲说，不试试怎么知道呢？但能够思考 的人很少做无谓尝试，确定目标和方向，勇往直前。
