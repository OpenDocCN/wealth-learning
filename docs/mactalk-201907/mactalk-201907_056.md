# 关于读书

池建强 : 关于读书 读书确实是开启心智和民智最好的方 式。能多读书的人无一例外是因为他们真心喜欢阅读，而 不是去吸收信息噪音 通常，越是好书越难读，无论是人文 的还是技术的。但是不读更难受，爱读书的人看起来天生 聪明，真相是他们不读书也会成为傻瓜。奇怪的是，这些 人没法不读书，不得不越来越聪明。这也是无可奈何的 事。 当然了，现在读书的方式跟多样化了，比如纸书、听 书、电子书、专栏（图文+音频）等等，当无论怎么变化， 都是做书的人希望让你更好的阅读。 读书的本质是不变 的。
