# 放弃「被喜欢」的需

池建强 : 放弃「被喜欢」的需要 有时候我们被一些垃圾人的 言论困扰，就是因为很多人潜意识里存在一种需求：让所有 人满意，被所有人喜欢。事实上这是不可能的，也没有必 要。有时候「Hater」的增加，反而证明我们在做 remarkable 的事情。 我们唯一能做的就是保持真实，可靠，持续进步， 每天生产有价值的东西。
