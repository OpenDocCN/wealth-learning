# 如何学习并且学有所

池建强 : 如何学习并且学有所成？ 1、间断式学习，符合记忆 曲线，而不是突击性学习。 2、通过多种方式进行学习，听老 师讲课，视频，音频，阅读，写写画画，做实验，实战。 3， 考察自己的学习成果，小测验，发布，与人沟通，获得反 馈，促进学习。 4，关联新老知识，打通认知通道，融会贯 通。 这是人类最有效的通用学习法，极客时间和极客大学正 在践行这种方式。
