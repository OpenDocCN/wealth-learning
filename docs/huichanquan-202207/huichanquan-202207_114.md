# 亲，上门做核酸，700 一次！？

今晚的故事，尸哥想从“包小姐”说起。 

这个“包小姐”，很神秘，向来都是神龙见首不见尾。

记得在一几年的某天清晨，

当我还睡眼朦胧的跑向公交站赶上班的公交时，就很突然，

发现无论是大街上还是电线杆子上，莫名出现了很多狗皮膏药般的卡片。

毫无征兆！这个“包小姐”一夜之间就火遍了大街小巷！ 

![](img/1e78522dae981c28d416169ca26ecc61.png)

**这“包小姐”何许人也？ ** 

经历过的人应该都明白，这包小姐并非是具体的**某一个“人”**， 

而是卖 y 女性的统称，就像各位“老色批”一样![](img/4634b2517cf639e7293fd98d8fe4ba21.png)

那个时候打广告的方式很局限，

因为互联网还没有现在这样发达，大多的广告都要靠电视或者广播来进行推广。

![](img/d7ef45e941fc94fa9696394825ce4069.png)

除了这些付费推广外，

最直接有效，还不用花钱的方式就是，满大街贴广告。

直到今天，这种东西也是随处可见。

![](img/f4fc747b52b82a3d8463cd40e9560c27.png)

而这种从古至今就没断绝过的违法“生意”，自然是没办法通过电视广播来推广的。

所以相关从业者，便印刷了大量的“小卡片”，满大街的撒。

姜子牙钓鱼，愿者上钩。

![](img/d53b2681cec0f68048d2527618ba5bc8.png)

再后来随着互联网的兴起，

这帮人便停掉了耗时费力的发卡片招嫖的方式，转头盯上了互联网这块大肥肉。

从最开始的 QQ 随机添加男性好友开始，

再到后来的各种社交软件平台，这些“相关从业者”可谓是把互联网玩的透透的，

不会放过任何一个有 LSP 存在的网站或 App，无孔不入。

除了一些常规的，非常直接的招嫖信息外，

他们有的还打着**伴游**的幌子，寻找目标嫖客。

![](img/65576df2160e5586f4c73323f9918e59.png)

有的则是线上陪玩陪聊，线下可约见面。

![](img/9398cac7fff5590c3c06e72d7036d986.png)

有些还专门盯上了一些司机师傅，利用下单的方式招嫖。

![](img/b47494e59215e83e2e2b4cb4cdb48914.png)

反正就是，哪个平台火，他们就去哪个平台发招嫖信息， 

哪个事件比较热门，他们就蹭哪个！

这些人的“网感”，甚至都可以吊打 80%的互联网从业者，

**人家热点追的，比我们专门追热点的，都专业！**

这不，有些人竟然开始打着“上门做核酸”的名义，开始卖 Y 嫖 C 了..... 

这些人被一些网友（LSP）亲切的称之为：**核酸援。**

**![](img/b8079e761fa9fc85358e7883bff1f2cb.png)**

如果说之前大火过的核酸媛，只是为了穿着工作服拍拍“美照”，

![](img/0f97d7f13783b4340c981b09e80c47a4.png)

那么最近兴起的“核酸援”，人家可是真枪实弹的真做，真上门... 

从价格上就能看出来，

她们做的根本不是什么正经的“核酸”，而是挂羊头卖狗肉。

![](img/ec173f2734ce75121122c3e3ec2d3abc.png)

很快，一些比较热门的 App 就被这些相关从业者霸屏了。 

几乎所有比较热门的城市，

都可以搜到“上门核酸”的从业者，头像也是个顶个的漂亮。

如果尸哥楼下采样点儿的护士能有这番姿色，我愿意天天去做![](img/5a38e3dd3d5517c6ff8f9de9ed0c7cb5.png)

![](img/c41d7590f3d563e45da4bd8a7900b581.png)

咱也不知道现在的“从业者”是越来越年轻漂亮，还是他们都是盗用她人的照片视频。

反正点进她们主页，总是会看的人热血沸腾！

不禁感叹，贵圈内卷的也如此严重了？

这种别人得花个百八十万才能娶到的人，是尸哥花个几百块就能拥有的吗？！！

![](img/adf714384f9c83cdc8ceaae0d225809c.png)

“你的微笑迷人的脸庞

美好时光我的心荡漾”

![](img/47e06d8c5531a14f8743123e1bcf4a21.png)

不只是短视频平台， 

只要有人的地方，就会有江湖。

其他一些比较热门的平台，也难免被屠，

着装还可以“私人订制”，好这口的 LSP 看后喜笑颜开。

![](img/4431e0e8067e8b2e086d3ecdbf418808.png)

白天做正经核酸，晚上普度众生，活菩萨不过如此！ 

![](img/a9e78f8d7416e5a0e8d3d77022a56cb9.png)

即便与“核酸”八竿子打不着的“从业者”， 

借着这波东风，也开始把手里的“艺人”往核酸护士的方向打造，

有流量的地方就有卖片儿的，有流量的地方更少不了招嫖的。

原来你家楼下按脚的王翠花，

摇身一变可能就成了王护士，身价立马翻倍。

![](img/3ee704d800501637fbec4f593be950bf.png)

但这次依旧很可惜，依旧是已经治理完毕后才让大家知道消息。

当把这篇文章发出来的时候，各平台已经展开了“扫核酸援”的行动。

目前如果大家再去搜索“核酸援”，除了零星的几个新注册用户外，之前那些大秀身材，看得人流鼻血的用户早已消失在了人海。

![](img/eb3cd96a934afbcc90c7b93b140613b3.png)

虽然说这种援还是比较“务实”，

比起那些看得着摸不着的“佛援”、“田媛”、“飞盘媛”实在太多，

但是真心觉得这些人，不该顶着“核酸”的名义来做这种勾当。 

人家正经的医护人员顶着 3 4 十度的高温，给大家做核酸检测，确保国民的人身健康，

而你们却以他们的名义，干着偷鸡摸狗的勾当，这着实说不过去，不灭你灭谁？！

热点可以蹭，流量可以抓，但要有道德良知。 

这番操作，很难不让人担心会把真正的医护人员的名声搞臭，

这以后谁还敢给咱做核酸，

特别是那些本来就没有报酬的义工，看到后更不敢了。

玩归玩闹归闹，别拿医护人员开玩笑！ 

今晚就到这里吧，大家周末愉快。
