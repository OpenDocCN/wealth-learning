# 微博这个公告......把大家最爱的谐音梗整没了？

昨天，微博让众多网友炸锅了。 

![](img/0992907953d92eb38ac2a48445c8d247.png)

相信玩微博的小伙伴们应该了解，微博有一套自己的**违禁词**。

![](img/19b2601c65094e3b516534d7c7f6201c.png)

当你发的内容触碰到违禁词时，微博会**限制你的微博阅读**，导致其他粉丝看不到你发布的微博内容，严重点可能会被禁言。

微博甚至狠到夹过自家的 CEO‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍...... 

![](img/eb2144434657c1e166302e5c7d4daad3.png)

![](img/61312ae5680037228345043012d942e2.png)

但是到底有哪些词语会被限制，微博并没有公布出来。

这就导致网友们会自行摸索可能被屏蔽的字眼，不得不用**谐音字、表情、缩写**等方法替代。

![](img/37b0daef7c911e85c1ea2569fe5a949e.png)

虽然会影响阅读，但大家为了发出去莫得办法。

结果吧，**微博又开始搞事了......**

![](img/4733ce001ad228c1f68948a8a72013dc.png)

昨天，微博管理员发布公告：

为营造清朗的网络空间，维护文明健康的社区生态秩序，**站方将对站内利用谐音字、变体字等“错别字”发布、传播不良信息的违规行为开展集中整治。**

![](img/1fd576073511a60490e1a956f14b6734.png)

具体内容如下：

1.加大对利用“错别字”借机传播不良信息等违规行为的排查清理力度；

2.健全平台用语管理机制，完善关键词识别模型；

3.通过建立正向激励机制、加强站内宣介等方式，引导站内用户规范使用汉字。

![](img/9f391a2aaa95ce3ff3d9f9bf8bde637a.png)

王建国这盛世……终究是楼塌了......

谐音梗真的要开始扣（jin）钱（yan）了......

燕子，啊不，谐音梗，没有你我怎么活啊......

![](img/f87042c7e8d73e855daf02dd02f73242.png)

这条公告一出，火速登上热搜第一。

有拍手叫好的，表示终于**可以举报用谐音梗网暴的黑子了。**

![](img/76292b1115d841f3bcb12a17895647c2.png)

**终于可以顺畅看中文，再也不用猜来猜去了！**

![](img/6a912f3ece3db16f1fc0560a5ee5e6b2.png)

“的得地”小警察也在狂喜，表示**不能忍受的得地不分**。

![](img/2d97d9da04ee0a6121476c0248b1cff8.png)

![](img/157760cd01eae345844f97886c238f52.png)

于是有人质疑道：语文一百分才配上网吗？

![](img/abadab258dc059f8265a4b6a6a7b5a52.png)

随后，微博官方账号@围脖侠补充道，站方整治的目标是**“通过错别字传播不良有害信息，不是不允许任何错别字的出现”**。

![](img/1a237a0d83bbd5807a9f8dacf98e5a6b.png)

也就是说，的得地没人管你，打错字没人管你，但如果故意打错字就有人管了。

那么，你们官博这么**顶风作案**，好像有点不太好吧......

![](img/11cecb7d8f94aa44a6bb14ee573fb74b.png)

对这件事，拍手叫好的人很多，但是反对的人也很多。

许多网友表示：**说到底，大家不好好用中文打字的目的，都是为了规避被平台屏蔽限流。**

![](img/274bc04f202d806eb9e5353fbe6a30fa.png)

就拿性科普博主来说，众所周知，性科普博主说讲的内容，大半部分都会被口口口口口掉。

所以，不正常打字也是实属无奈之举。

![](img/75084b6f70c07e2ec3d28da9e4e7797c.png)

**如果平台不限制发言，就不会有谐音字变体字的存在。**

有网友们总结现在微博平台做的事情：

**这就是整治在马桶里拉屎的行为。**

**你们有病，让我们吃药。**

![](img/b6124c27d962741f1f7c92fa8f2643af.png)

如果一边屏蔽一边又要整顿，那么以后说话是不是得这样了： 

勼蕶後哋焱暒妏叒婹偅炪茳煳孒

阝余 ㄋ↘ 氵冗 黑犬ィ我 辶不 乘刂 丅 イ+ ㄙ 

![](img/69e6a9d2c9c9160db6f47e304d93f7c5.png)

---.---....-.--/---.---....-.--/-..----.--...../---.--.-....-..

![](img/c3f3dad6633580ab8219d6a796da7ce1.png)

在微博上，确实会有很多通过谐音字、变体字传播色情赌博等不良信息的存在。

例如有些评论区，会莫名出现一些乱七八糟广告。

![](img/fb960ba3f83f485244c57c3693b80ffc.png)

他们的广告往往都是利用谐音字来表达，例如

**“你想看摄情内容都在这里，加扣扣 XXXXX”**

![](img/ecf381fb3be2010716c39ce56a88f055.png)

这些内容很难被平台检查到，因此微博发布的整顿，加大审查力度减少不良信息的传播。

这对于网友们来说，还是有益处的。

![](img/9af44b409c79b01d3d62716881eddc5a.png)

但是，如果只是为了整顿网络不良风气，而直接对内容**一刀切**，似乎显得有些**本末倒置**了。

毕竟，不良信息不仅仅只会以谐音字、变体字的形式出现。 

![](img/e07680adb4c6b3822a6ee89b458184f6.png)

引起网友们强烈不满的根本所在就是，正常打字发不出，而现在谐音变体字也不让发。

**网友们都认为，整治在于实质而不在于形式。**

不少网友质疑，举报黄色信息和骂人都没有成功，微博怎么能保证谐音字就能准确无误的审查到。

![](img/df636a8e64c07ee93a49694f5823be03.png)

那么微博这一举动到底是为了营造良好的网络风气，还是让大家闭嘴少说话呢？ 

![](img/bb74e82bad7ae6724a8bc060ccb3cfa5.png)

在微博发出声明时，**没有关于任何“传播不良信息”的定义，也没有具体的整顿说明，**这使得整顿的界限非常模糊。

玩谐音梗讽刺不良现状算不算传播不良信息？

这些定义又是谁来定义谁来掌控的？

这些问题都需要微博的回应。

如果因为整顿内容不清晰，而导致殃及到无辜的网民，这只会加深用户对微博的负面印象。
