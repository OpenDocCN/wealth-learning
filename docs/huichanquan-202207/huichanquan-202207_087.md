# 一个走入绝境的网投人员自白书！！

阿龙，人生:死很容易，活着好难。

有一些粉丝主动联系阿龙，他们不是为了举报或求助，只是想找个人倾述他们的悲惨经历，阿龙很理解他们，一个人面对社会的黑暗和艰险，自保已是万幸，反抗显得毫无意义。今天就分享一个粉丝的经历。

我的经历: 2020 年 3 月来柬埔寨，在戈公省博彩公司工作，做客服，公司特别好，我们工作一点压力都没有，做了一年多，公司解散了，有的同事选择回国，有的选择留在柬埔寨。我和一个 00 后同事留下来，到西港找工作，那时候西港没有这么爆乱，到了西港找了一家体育彩票公司，在金水园区，做客服，做了两个月感觉这个工作压力比较大，坚持不下去了，又离职，我们两个人离职出来，找了酒店住着，自己有些存款，没有着急找工作，每天闲着就去了赌场赌博，几天赢了几万块，让自己有点飘了，不想找工作了，结果有一天把赢得钱和自己的存款全部输光了。没有办法不得不找工作了，然后找了这里一家嗨场的服务员工作，虽然工作有点辛苦，但知道自己没有钱了，只能坚持工作只是服务客人，服务好了客人都会小费，做了一段时间，存了一些钱，想到之前自己输了那么多钱。就想去赢回来，赌博毁了自己，之后的灾难不断，每天下班就想去赌场玩，算是比较幸运，几天把我之前输得钱赢了回来。

就这期间里，公司的一个同事和我走的比较近，下班有时间经常一起玩，就是他的一个朋友把我和我同事一起陷害绑架，因为不配合他们，被打的头破血流，腿伤了一个多月才好，把我的钱都拿走了。我的同事被卖到哪里我不知道，从此联系不到，我被卖到了菩萨省，一个偏远的山区网投诈骗公司，离西港十几个小时的车程，我到了公司只能先安分的工作，每天十五六个小时工作，不用说压力有多大了，主管骂人都不把我们当人，我不甘心这样下去，诈骗都是骗中国人，我就偷偷的报警，联系大使馆和中柬义工队，结果应该是被警察局出卖了，公司知道我报了警，可想而知，一顿电击，暴打接着卖去别的公司，又把我卖回了西港，卖到皇冠，到了这个公司感觉是比较好一点的公司，以为会有好转，可是后来公司知道我报警的事情，又把我转卖到凯博中国城，到了这个公司让我更是大开眼界，在里面呆了几天，了解到很多人都是被绑架卖过来的，过着生不如死的日子。
