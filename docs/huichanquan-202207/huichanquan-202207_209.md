# 

又是穿短裤短裙的季节了，那些偷拍的狗又要出来作祟了。

如今的科技水平也比较发达，各种微型摄像层出不穷，促使有很多无辜女性被犯罪分子用尽各种龌龊手段偷拍。

![](img/53c7e88d3e07a0b92492c990686b6e99.png)

现在的互联网时代，也改变了偷拍的性质，再也不是为了一己私欲，而是通过各种色情网站不断的传播，甚至以极其低廉的价格出售，获取利益。

这些偷拍者的手段专业到让你根本发现不了，也正因为这群偷拍者，你不知不觉的就成为了非法网站的女主角，被大家肆意讨论。

![](img/3860f29625fe86db573158f2705c172c.png)

1 

之前，苏州就有一起“**情侣酒店被偷拍**”事件；

张先生和女友来到苏州旅游，在景德路一家主题酒店入住了三天

可高高兴兴回去后没多久，张先生的朋友无意间看到他们长达 40 分钟的亲热视频，全是张先生和女友酒店内的亲热的私密画面。

**更让张先生气愤的是，这个视频已经传播了一个多月，在一次次的传播下，这段视频已经被 4 万多人点击观赏。**

**![](img/ab0eaf06a08245327d0935936411d75b.png)**

![](img/e141eee61d8129b47e65854992c7b970.png)

视频底下是上百条不堪入眼的评论，想到自己的身体从头到尾被“观赏”过一遍；

张先生的女友直接崩溃，并有了数次轻生的念头。 

![](img/b0cbaa9eaf931badbf8446c6f6c0648e.png)

2

除了在常见的酒店、KTV 等私密空间纷纷中招外，就连健身房也不能幸免于难；

要知道健身房里的男男女女基本都是半裸相见，这更是激起了偷拍者的欲望。

有人专门偷拍健身房更衣室里的裸体男生和女生，**然后以高价卖出这些照片。**

在海口市的一家健身房，就有那么一起事件，一个女会员在健身房的浴室被偷拍，将偷拍的裸照刻成光碟，装在信封中放在私家车的窗上。

嫌疑人将自己的电话留下，向受害者进行勒索，只要不答应，就把照片和视频发在成人网站上供人观看。

还听说偷拍都会火？

**的确是会火，是人民群众对你的一腔怒火**。

![](img/a2bac97e2f8c86036fd1a504f3890f63.png)

虽说健身房在很多人眼中是积极有正能量的场所，但是林子大了什么鸟都有；

所以大家在锻炼身体的同时，也要注意保护好自己的隐私，不要让别人有可趁之机。 

![](img/070d1e8750dd6bbd379c3f437b69224c.png)

3

明枪易躲暗箭难防，有时即使女孩子们有着十二分的小心，也架不住坏人们推陈出新的手法。

拄拐杖的老人其实拐杖就是他的偷拍道具

![](img/b8fb26f43f0b11642c2ee67646944da5.png)

还有人在纸袋子上穿个洞来偷拍

![](img/584ad859597133cb87e7bd7ae36bfe79.png)

常见的厕所里的原有物品，马桶刷、芳香盒、挂钩，挖个洞就能成为实时摄影机。

![](img/956190d4889dc8a5292d9da997535d24.png)

另外还有口香糖&易拉罐等，放在厕所比较恶心，很少有人会去主动翻看，同样也是盗摄的重灾区。

![](img/681b263af614b0b8cdf70052da798338.png)

谁都不想让自己成为下一个偷拍镜头里的主角，更不想自己的隐私暴露在大众之下；

如果不幸遭遇到偷拍，**请勇敢拿起法律武器对抗，而不是忍气吞声，让坏人得逞。**

**曾经就在一个直播平台上看到，有的女主播为了要礼物，就去女厕所偷拍。**

**最后一个压轴的场景非宾馆莫属了。**

**这个就不用多说了吧，打开软件的国产区，大浴缸、大圆床...，有网友曝光说：尤其是民宿更猖獗。**

**而国家也一直在打击黑产。**

**前段时间也出了《关于开展摄像头等黑产集中治理的公告》，对互联网平台以及电商平台等五个平台提出了硬性要求。**

**![](img/d55a74920795bc354f4ac32c814af82d.png)**

**但黑产十分隐蔽，打击起来也非常困难。**

**所以，我们能做的就是提高防范意识，多了解偷拍手段，这样才能更好的保护自己。**

**![](img/2bedfad8818d95d84b12a91bbab836c5.png)**

**就像上面所说，摄像头藏的地方非常隐蔽，比如地上的口香糖、纸巾、马桶塞、易拉罐、衣架等。**
