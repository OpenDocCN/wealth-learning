# 因“哽咽”而道歉的曾颖被曝自杀，有些人迫不及待地准备“开席”

昨晚临睡前，刷到一个令人悲伤、愤懑的消息。旅日前媒体人曾颖自杀了。目前正在抢救当中。![](img/6cdaedc563bd24e0e500c3f51715745d.png)
![](img/4aed23281b3bdf54e246aea386c3b232.png)此前，日本首相安倍晋三在日本奈良街头发表演讲时遇刺身亡。因为在同国内媒体直播连线时哽咽落泪，曾颖在中文互联网上引发了巨大争议。准确地说，是遭遇了一轮铺天盖地的网暴。压力之下，曾颖不得不站出来向网友进行解释并道歉，同时表示，任何恐怖主义都不该被狂欢，作为一个有血有肉的人，会永远坚定自身的价值观，做一个正直善良的人。但网友并不买账。随后她的社交平台账号遭到了禁言。直到这一次她被曝不堪网暴而自杀。网络上流传的曾颖发布在朋友圈的遗书显示，她“很平静地跟大家宣布一个消息，32 岁的我，决定要离开”，并简单对自己 32 年的不算太长的人生进行了梳理，曾颖说，“这一生过得好值得，我才 30 出头，除了没有成家生子，我什么都体会过，我的一生在同龄人里，是激荡而丰富的一生，很高兴在这一生里，我能认识你。感恩！再见。”![](img/9262d8ee7f6f088294ea34db971c3a20.png)如果说，在此之前，我还尽量尝试去理解那些网暴曾颖的人，他们只是容易被某种集体性的情绪所裹挟，只是觉得自己朴素的感情受到了冒犯与伤害——从专业角度说，曾颖的表现确实有可商榷之处——观点可以不同，但人性层面终归是有交集的。但是我错了。当曾颖在日本东京自杀的消息传来，网络上瞬间充斥着什么声音呢？“去见安倍，如你所愿”，“准备开席”“气氛都烘托到这了，再活过来就不礼貌了。安倍桑也很期待与她相见呢。”“追随她最爱的安倍关白去了吗”“现在才自杀吗，有点不孝啊”“她也太爱安倍了吧”，“还有这好事”“没死就是作秀，有本事就下去陪安倍”……有公号文章说她是”**咎由自取**“。曾颖的自杀，为他们带来了一场新的狂欢。在他们眼里，“人性”这个词是有罪的。因为立场大于人性。 因为，他们可能真的没有意识到，为他人的自杀迫不及待地准备“开席”，意味着已经没有了基本的人性。我不想使用某类骂人的词汇，但我可以说，相对于正常人类而言，他们已经在“爱国”的掩护下成了一个新的变异物种。他们从来就不会爱一个有血有肉、活生生的“人”。别以为他们只是为安倍遭枪杀而幸灾乐祸。也别以为他们只是为有“哽咽‘罪行的、长期居住在日本、面目可疑的曾颖而幸灾乐祸。记得数年前，媒体上传出一桩偷渡惨剧，39 名偷渡者躲冷冻集装箱内前往英国，气温骤降至零下 25 度被活活冻死。最初传出这些偷渡者是中国人，后来被证实为越南人——但对某些人来说，后面的证实已经不重要，因为，在此之前，他们已迫不及待地表达了对遇难同胞的幸灾乐祸。原因只有一个：不好好待在自己的国家，却投奔资本主义国家，死了活该。是否比零下 25 度的冷冻集装箱还冰冷？这些年来，这样的网暴群体在不断壮大，有如黑压压的蝗虫，遮天蔽日，令人望而生畏。他们逼死了少年刘学州，也逼退了罗翔，逼退了缪晓辉，让张文宏心灰意冷，让张学友不得不发声明……而今，他们正在逼死曾颖。![](img/e17ac9bb1c71dd34edeb820de69d33f7.png)希望曾颖能抢救过来，尽管有这样的“异形“存在，这世界仍然如此美好。他们不值得你为那些高喊你“去死“的人去死。他们也不值得拯救。![](img/1356ab20a0415ba06b7c9151509dc1e8.png)抽离观点与立场，如果有人仅仅因为我为曾颖的自杀表达同情与惋惜，仅仅因为我谴责突破人性底线的网暴而对我恶语相向，我唯一能做的，就是将这样的人默默拉黑。生命，应当浪费在更美好的事物上。![](img/164c15b2b8633fd8c028fe270ca31119.png)![](img/369aa345505613643c9a9162143b3890.png "小米 6-0419.055.gif")![](http://mp.weixin.qq.com/s?__biz=Mzg5ODAwNzA5Ng==&mid=2247488051&idx=1&sn=1ad64c0293a2731910bb4823694deb98&chksm=c0687b64f71ff27226524aa2a4d77cc1119ffb9ff5938f2973505bb619c73402a8a6b2e59e60&scene=21#wechat_redirect)![](img/6ec274e574566755002d3fff72f48220.png)![](img/6d353bec5b422832db465d348fdd5885.png "小米 6-0419.052.gif")
