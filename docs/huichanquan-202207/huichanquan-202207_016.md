# 大批性感美女雨天在上海街头集体激情搓澡，洗澡媛来了？

近日，上海突降暴雨。 

街头忽然出现了又一“魔幻”场景，很多人在街头雨中搓澡。

不少人也是纷纷围观和效仿。

![](img/f194ffc7e9267f0ee1b84260d71c10ba.png)

不过，很多人看了都觉得这是“行为艺术”，就是为了“哗众取宠”。

当然，更多人吐槽这种行为很尴尬，丢人现眼。

也有人觉得，可能是被之前长时间“禁足”给憋坏了，所以出来撒撒欢，目的就是要沐浴外面的雨水，感受一下外面的空气。

![](img/62b07c04bd81d6879c65543e8fa9c63b.png)

这个事情绝对的醉……老实讲，一开始出来这个事情的时候压根就没什么热度，一直到这几天不断有人来问我们，才知道此事已经传到那么久远，乃至于很多不知情的网友都觉得上海人喜欢在户外洗澡？

然后，镜头一转，另外一位美女浑身洗澡泡沫的出现在了街头，这个姿势摆得非常妖娆，这个背景明显就是上海的又一网红地标：武康路。

![](img/36e7268132eb42bb00d0f46e71ce68fa.png)

从目前两位美女的穿着，妆容，还有拍摄的取景手法来看，我们非常的怀疑。然后镜头一转，又到了上海的另外一个地标：静安寺。

![](img/d8b666a248806e10056afee2db243ea3.png)

接下来，大批的美女出现在外滩，相互搓澡：有男有女，人数众多：字幕配的是“用更阳光积极的心态面对新生活”。

![](img/74cc8771d5d305f0ec9aa825eb938ea4.png)

我们真的是要吐了，这是哪跟哪？有哪个正常人会在雨天跑出去洗澡？还是大批量的？统一还是年轻的俊男靓女？

众多美女在雨天洗得不亦乐乎：

![](img/837998672ebabc8ce31f743feb2e4c61.png)

老实讲，我们看完是整个懵逼的，不知道到底是什么意思？这是什么脑回路需要在户外，把马路搞得一塌糊涂都是洗澡泡沫，甚至可以说污染了环境，去体现什么“正能量”？

![](img/6b4719b51b3feed81163032f9c222a29.png)

一个接着一个的美女拿着搓澡巾当众洗澡能体现什么？除了污染眼之外完全没有任何的意义！ 

![](img/add1ad434e0b4e40cb195ca30457974f.png)

若是在晴天，估计城管就会赶来。但城管不一定会驱赶他们，因为他们这是“行为艺术”。

假如是一群农民工在雨中搓澡，很可能城管就已经将他们驱赶走了。

至于这群皮肤白皙的男女，城管还真不一定会“管”。

![](img/06afcff60c26600ce107331cdf6565f0.png)

在晴天这些“行为艺术”表演者，可能也不会当众搓澡。

趁着下大雨在街头搓澡，究竟想表达什么呢？

节约用水，憋疯了出来搓澡，还是沐浴大自然的雨露？

或者，还是什么其他的意思？

总之，也不过是吸引流量罢了。

![](img/a7dc14e058ba80af9a94364292036ec9.png)

上海，这座国际大都市，在疫情期间的一些事，已经让人感到很闹心了。

这个时候，类似雨中搓澡这种行为艺术，不知道是给大上海增添光彩呢，还是又给大上海人添堵呢？
