# 司令，你能分析一下

(提问)匿名用户 : 司令，你能分析一下自杀的人心理机制吗？ 生命的本质是活下去，那为何会有的人选择自杀呢？是被现 实塑造的对于自我存在合理性的绝望？

2018-12-14

回答：生命体的自我存续是正常情况下的本能诉求，就如同

电路是用来导电的，但是短路了就会起火！ 人家那个精神结

果本来是用来连接的，就是个体之间相互连接成一个更大的

整体——一个更大的生命体，让基因能够在更大的生命体里

面得到延续，具有更强的适应性。 凡事有利就有弊。人的神

经系统维持正常的运转，需要消耗掉人体 40%的能量。这还

是在什么都不做的情况下。一旦要应付情绪或者思考这种深

加工的突发状况，人体的大量能量都被这个黑洞吸附进去

了。所以不管是处于高度的情绪冲突之中，还是处于深度思

考之中，人在其他的方面都会很迟钝。人也更容易，精疲力

歇。因为能量被抽空了，就会造成其它部分的坍塌。在生物

化学机制上，它就会变成响应不够灵敏，甚至萎靡。 如果人

的精神结构产生了一种无法解决解开，还不断的自我强化和

叠加的内在冲突。它就会加速的抽干人的能量，嗯，能量的

匮乏又会引起身体其他部分的预警。产生新的信号和情绪，

进一步消耗能量。这是一个恶性循环。 但这个恶性循环越过

临界点的时候，人体是撑不住的，需要解脱。就如同短路把

线路烧毁了一样。因为人体——生物体是由神经系统控制

的。所以，这种解脱无非两种：要么疯，要么死！(7 赞)
