# 这里有个天然的矛盾

(提问)匿名用户 : 这里有个天然的矛盾：人一方面会追求更纯 的体验，那种饥渴感会一直驱使着你，另一方面，错配又会 让能量无法正常流动，从而生出肿瘤——所有的怨，所有的 恨，所有的苦涩。

司令，纯的体验是指什么？人为什么又要不断的追求这种体 验？

2018-12-06

一种巨大的浪费——人通过这种浪费来寻求极限的突破，它

源于基因对最强适应性的探索。(8 赞)
