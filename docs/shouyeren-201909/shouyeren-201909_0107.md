# 星主 新公务员法里

(提问)蓄 : 星主 新公务员法里 职务和职级俩条线并行 代表国 家什么倾向 对以后会有什么影响

2019-06-18

回答：你是新人? 新人才这样称呼。 军衔和职务是分开的，

也存在某种对应。军衔代表等级体系。职务代表权力的行使

范围和要处理的目标问题。换句话说，任命一个职务是要去

有针对性的解决一个具体问题。这个问题或许比较复杂，但

依然是可以有边界，能够定义清楚和具有明确导向性的。 军

衔是等级，它清晰的架构起这支有组织的暴力的内部结构

——只要结构本身是完整的，成员的流失和更替不会改变整

体的组织性。所以军衔必须是按部就班，不能乱了套，不能

临时起意，不能随便授予。 但是事情讯息万变，有很多临时

性，突发性的事情要处理。如果都走组织流程，事过境迁就

耽误了。有的时候走组织流程，本身就存在冲突。所以要搞

一个临时策略。你可以任命一个上校临时当战场总指挥，这

场仗打完以后，他回去还是做他的上校。既解决了当下的问

题，又没有改变整个组织体系的完整性。 我举的是军队的例

子，是因为军队的许多东西在社会组织上面具有非常明确的

代表性。甚至代表着最先进的方向。所以全世界的企业管理

都向以色列军方和美国军方学习。 职级和职务两条线并行，

就类似于军衔和职务两条线。如此一来就可以跨区域跨领域

调配干部。调配过去的干部是去解决具体问题的，并不对当

地的干部造成压力，也不挤占他们的资源和名额。所以他们

不会有那么强的抵触。也不需要做那么多的思想工作。这是

制度设计！中枢诸公要做社会治理体系的结构性调整。(13 赞)
