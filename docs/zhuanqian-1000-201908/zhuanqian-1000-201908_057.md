# #368 《这些低

帮主 : #368 《这些低价百度网盘会员是怎么来的呢？》 大家应该经常会看到有人发广告说可以低价代充百度会员。

百度网盘超级会员官网一年 300 左右，而且几乎从不打折，而 他们代充竟然只需要 100 多，折扣非常大了。

本着好奇的心，我去搜了下，原来是利用 iOS 的退款机制实现 的，真是会做生意啊。。。

解释下：苹果商店购买的虚拟商品，比如游戏装备、网盘会 员等等，基本都是可以无理由退款的，而且退款之后，你购 买的商品仍然存在。

所以，这些黑卡商人就利用这个原理来赚差价了。 这个生意是违法的，大家不要做哦。
