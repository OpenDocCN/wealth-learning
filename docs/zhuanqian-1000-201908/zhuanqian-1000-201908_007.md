# #517 太兴的门

帮主 : #517 太兴的门店策略 上周去广州出差，路过一商场，观察了下。 请看图一的三家门店——靠得住、太兴、茶木。 你是否知道这三个品牌，其实都是太兴旗下的呢？

太兴把自己的 3 个品牌都相邻开到了一起，从外边看是 3 个不 同的门店，但是因为都所属一家集团，在经营上就会有很多 优势。

1、太兴主打港式茶点，已经很有名气了，太兴完全可以给靠 得住和茶木进行引流。

2、共享厨房，传统门店在厨房装修上投入巨大，三家门店开 在一起甚至可以共用一个厨房，极大的节约了成本。

3、共享卡位，这三家门店主打的菜品不同，太兴是主打港式 茶点的，靠得住是主打粥品的，茶木是主打快餐的，客众的 消费时间会有些许差别，这样就可以共享卡位。
