# #109《一个引流

帮主 : #109《一个引流的小技巧》 网上经常会有人求各种电子书。 如果你擅长搜索呢，就可以利用这个来引流。 下面是操作方式：

1、利用搜索技巧搜集各种绝版电子书，各种很难搜到的电子 书等等（如果很容易搜到，引流效果就不太明显了）。

2、去百度知道（或者其他平台）自问自答，说自己有这本 书，然后底部留自己的微信号。

3、把粉丝引流到微信号，大功告成。 ps：我正好有个不错的搜索课程，感兴趣的同学看看哟[奸笑] [有道云笔记](http://note.youdao.com/noteshare?id=fa7c6205a455670877b885b34391aad4&sub=F1F7DDEB10FB45ACA93189D9D40D86DF)
