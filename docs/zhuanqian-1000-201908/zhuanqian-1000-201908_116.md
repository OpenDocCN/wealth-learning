# #298《测面相小

帮主 : #298《测面相小程序，几日分销收入过万》

上个月有个刷屏朋友圈的小程序《面相研究院》，他就是把 传统面相和互联网结合，利用人的好奇和展示心理在朋友圈 迅速走红。

与此同时他还有套路很深的分销机制，有不少人通过短短的 一两天分销收入过万。

其实做出类似类似这种小程序的费用大概只需要 2000-5000 之 间，只要能躲过微信团队的审核，这种小程序其实还是有很 大赚钱空间的，除了面相，如：测姻缘、生辰八字测算、测 风水、星座运势等等也都有很大机会。

有人可能要问了，做这种内容很容易被封吧？是的，通常只 能活几天，然后就被封了，但是他们钱已经赚到手了。

而且，他们通常会注册很多个公众号和小程序，被封了一个 再换一个，真是前仆后继，子子孙孙无穷匮也了。
