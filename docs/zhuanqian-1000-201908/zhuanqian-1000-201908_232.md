# #059《她加入我

帮主 : #059《她加入我的星球一星期，赚了 20000 元》

有位朋友从事保险行业，昨天她找到我，特地对我表示感 谢。

为什么呢？

因为她加入我的星球之后，经常会把我介绍的一些福利和赚 钱思路告诉她朋友圈好友。

结果反响非常热烈，她的朋友圈好友天天催着她更新。 其实呢，卖货就是卖信任。 她通过这件事大大加强了粉丝和她之间的信任。 于是，一周多成交了好几单。

轻松多赚 20000 元！ 这个案例非常有价值，有心的同学可以尝试起来。

分享不不分享我的星球内容没关系，只要是好的有价值的内 容，粉丝都会喜欢的。
