# #323《夸夸群月

帮主 : #323《夸夸群月入过万？》

新事物真是层出不穷，最近又突然冒出来一个新项目：夸夸 群。

夸夸群是啥玩意？ 顾名思义，就是群里人员天天互夸。

现在大家压力都这么大，就算不是发自内心的夸赞，听别人 拍自己的马屁也能身心愉悦。

于是，就有精明的商家开始搞付费夸夸群了。 我去淘宝搜了下，销量高的商家月入 5 万，很夸张了。 这个项目基本上有两种玩法。

1、活人夸，雇佣高智商+会聊天的人才，群里专门夸人，然 后利润各自分成。因为是活人夸，效果好，但是成本也高， 所以这种群门票比较贵点。

2、机器夸，使用机器人自动夸人，这个夸人的效果可能没有 活人好，但是我们可以把门票弄低点，降低大家心里预期即 可。
