# #286《微商的面

帮主 : #286《微商的面膜 OEM》 想必大家还记得前几年火爆朋友圈的面膜微商~ 为什么火爆朋友圈的是面膜而非其他产品呢？ 我们可以从几个方面分析。

第一方面是朋友圈的活跃用户中女性且年轻的比较多，他们 的需求也比较大。

第二方面是面膜这个产品是一个快消品，使用频率较高。 还有一个方面非常重要，就是产品开发门槛低~

面膜的开发和生产门槛很低，大多数微商都是找 OEM 工厂代 加工，然后再进行注册、包装和销售。

所谓的 OEM 工厂就是自己没有品牌，专门给其他品牌提供产 品生产代加工的工厂。

我认识一个小伙伴就是做这个行业的。据他和我透露，一片 面膜的成本在 5 毛~2 元之间，效果和大品牌的产品相差无几。

只要注册一个公司，设计好自己的包装，然后找一个比较靠 谱的代工厂，一两个月就能生产一批面膜，面膜一盒成本控 制可以控制在 10 元之内，售价普遍在 100 元~200 元之间，利润 可以说是非常可观了。

只要给自己的代理供货时能保证他们的利润比较高，代理们 还是很愿意去卖货的~
