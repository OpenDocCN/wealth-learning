# #318《关于职业

帮主 : #318《关于职业选择》

分享一个选择职业的思路，我认识一位名牌大学博士毕业后 从事保险行业的大哥。当我知道他的学历后，我就觉得很好 奇，他学历这么高，其实可以有更好的工作选择，为什么会 选择保险这个行业呢？

我们来看一下他的分析~

当时他毕业的时候也很迷茫，虽然有很多职业机会提供给 他，但是这些职位的竞争者也都是高学历的人才，自己考上 港大博士已经是非常艰难的事情了，如果继续和相同水平的 人竞争，是很难做到行业顶尖水平的。

相比他原来的专业，保险行业从业人员的学历要低一些，而 且做的专业的并不多。如果用他的一套思维模式和方法论来 做保险行业，一定能做到行业内的最顶尖水平。

所以他就锁定了保险这个行业，然后基本上和市面上所有保 险团队的负责人沟通过，最终选择了一个最看重自己的老 大。

后面这位大哥果然在保险行业内做的炉火纯青，3 年后就拥有 了多个自己的团队。

分享这个思路给大家，希望大家在选择职业的时候也能正确 分析自己的优劣势后正确选择。
