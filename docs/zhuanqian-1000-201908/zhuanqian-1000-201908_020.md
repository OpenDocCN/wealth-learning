# 480 这个民宿看起

这个民宿看起来很棒，就是不知道现在盈利了没有。 我简单算下，有 7 个客房，假设一晚上总收入 7000 左右。 一个月是 21 万，一年是 250 万。 去掉淡季，按照一年 100~150 万算吧。 再除掉人工成本，一年纯收益 50-100 万左右。

据说投资成本是 500 万。 具体数据我也看不到，只能推测。 这个民宿风景极佳，所以客房定价高了也没关系。 住宿房价再涨一倍，利润就能再涨 1.5 倍。
