# #352《赚钱的两

帮主 : #352《赚钱的两种思路》 长话短说。

一种是打造个人 IP，塑造个人影响力，然后靠粉丝变现。 另一种是玩项目，孜孜不倦找项目，打一枪就换一个地方。 两种方法，各有利弊。

做个人 IP 虽然看起来很好，可是能做成的人不到 1%，剩下的 人都会死在半路上。

玩项目需要大量的尝试和实践，你可能前面 99 个项目都赚不 到钱，但第 100 个也许就赚到了。

两种思路我们可以都尝试下。 小步快进，充分运用 MVP 思想。
