# #108《一个借力

帮主 : #108《一个借力打力的案例》 对于有商品的人来说，吸引新客户是最难的事情。 下面分享一个例子，也许对你有用。

假设你是卖羽毛球的，那么你可以去淘宝找下羽毛球拍销量 比较高的店铺。

然后找他们合作，让他们在每个包裹里面都放你的店铺信息 和二维码，每个包裹你额外付一元给他。

当然不是每个店铺都能谈成的，那如果你谈了 20 家店铺呢？ 总有一个能成的吧。

这样吸引来的都是精准粉丝，相当划算。

上面只是举个例子，重在介绍这种思路，大家可以自行拓展 应用场景[嘿哈]
