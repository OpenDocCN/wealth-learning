# #131《揭秘星球

帮主 : #131《揭秘星球排行榜机制》

人多的星球如果进了排行榜倒可以理解，有的星球只有几百 人也经常进入排行榜。

这其中必有原因。 我观察了几天，逐渐发现了星球的排名算法。 人不在多，关键在于互动。

只有几百人的星球之所以能进入排行榜，是因为他们天天在 星球里面打卡，这样数据量就可怕了，等于他们一天能更新 几百次。

所以，自然就进入排行榜了。 进入排行榜之后，又会源源不断吸引新的粉丝加入。 形成了一个完美的循环。 如果你也准备运营星球，希望对你有用~
