# #312《春节期间

帮主 : #312《春节期间如何赚点零花钱》 马上就过节了，你家贴好春联了吗？

别看不起卖春联，超市里随便买个春联都要 20 块以上，其实 进货价只有 1、2 块钱。

有一个哥们每年春节之前都会开车去街上卖对联，每年都能 赚个几千块。

除了开车去街上卖，在小区里卖给左邻右舍也是一个方法， 如果还能帮忙贴好，那小区的邻居们当然都会在你这购买啦~

春联这种商品有个好处，即使今年卖不完也不担心，可以留 着明年继续卖~
