# #292《补班课赚

帮主 : #292《补班课赚钱》

众所周知，孩子补课的价格是越来越贵了，大家也都知道做 家教是一个还算容易实现的额外收入。

但在大部分人的心中，能做家教的人往往是某一门学科学习 的非常好的人，能开补课班的人一般都是在学校工作的老 师，因为他们往往自己有很多学生资源而且还精通一门学 科。

而我认识一个小伙伴，他虽然不是老师，从事的职业也与教 育不怎么沾边，但是他也自己开了一家小型补习学校，后来 还转型成了一名家教老师，现在已经经营了有 3、4 年左右， 每年可以赚不少钱。

下面来分享一下他是怎么做成的~

他是联通的业务员，本来是和教育丝毫不沾边的行业，但是 在划分业务区域的时候碰巧被划分到了大学，这就让他有很 多机会与大学生接触，同时也认识了很多学习很好而且讲课 能力很强的学生。

有一天他小区有个家长就问他说，能不能介绍一个学习好的 大学生给他家的孩子补课呢？

就是因为这次介绍，他突然意识到了自己手里面的大学生资 源可以为他所用，所以他产生了办补课班的想法，碰巧他家 附近有几个小学和高中，他索性就把自己的家改装成了一个 小型的教室，一边和学校里的同学沟通是否愿意做兼职补习

班老师，一边在附近区域招生。

因为他平时和同学的关系很好，所以他招起老师来非常方 便，一来二去他就找到了几个讲课能力非常强的学生，另外 他区域内的大学几乎是整个城市最好的大学（小城市好大学 还是很少的），所以他在招生的时候也不是很吃力~

后来因为经常听这些大学生给孩子讲课，他觉得讲课也没什 么难的，就想自己也尝试一下做讲师。

接下来他做了 2 件事。 1、买了一堆教作文的教辅书籍，和视频课程，一边学习一边 打磨自己的课程。 2、打磨了几节课以后就免费给班上的几个孩子上课。

没想到学生们还都蛮喜欢听，再加上他是一个比较负责人的 人，每天督促学生练习，孩子的成绩也确实在提升，现在他 还成了当地小有名气的作文老师。。。

有的人说他之所以能赚钱是因为他占了天时地利人和，即有 老师资源，家附近又有很多中小学校，招生也方便。虽然的 确有些关系，但更多原因是他善于发现并利用资源~
