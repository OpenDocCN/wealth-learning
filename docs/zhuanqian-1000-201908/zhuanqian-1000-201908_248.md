# #017《案例分享

《案例分享，一小伙 3 天赚了 5000 元》

情人节前夕，一个小伙找到了一家巧克力 DIY 店，希望和店家 合作，让店家低价提供巧克力给他，因为他走的是量，所以 低价还是能谈下来的。

为什么要找 DIY 店呢，因为这家店可以做出各种风格的巧克 力，而且可以写上各种祝福语，非常适合情人节期间男女朋 友互相赠送。

谈好合作之后，他打印了 2000 张名片，宣传推广情人节巧克 力，名片留的是自己手机号，然后去繁华地段发名片给路过 的年轻人。

接单之后，就联系店家制作巧克力。

因为是情人节期间，所以巧克力临时卖高价也没关系，恋爱 中的男女对价格是不太敏感的。

最后他一共接了 100 多单，累计赚了 5000 元左右，一共只花了

3 天时间。

这是不是空手套白狼呢！ 准确捕获节日流量，然后利用店家的资源进行整合。 虽然这次钱赚的不多，可是他有这样的思维，以后赚大钱那

是指日可待的。

一个人整合能力的大小，决定他事业的大小。 怎么样，这个案例是不是很有启发呢？

逢年过节的时候，大家也可以想想，在自己的城市能不能也 这样玩玩看？
