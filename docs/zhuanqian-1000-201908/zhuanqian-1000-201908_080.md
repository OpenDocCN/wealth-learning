# #343 推荐一个引

推荐一个引流 APP：CHAO。

这个 APP 是知乎推出的，对标小红书，相当于男性种草指 南。

我下载测试了几天，发现可以用来引流。 为什么这么说？

这个 APP 估计刚上线不久，官方现在管的不严，你可以直接 把个人信息改成微信号或者 QQ 号什么的。

然后正常发帖引流即可。 建议使用小号注册，被封了损失也不大。 好时光可能不多了，官方一旦开始严打，就没法引流了。
