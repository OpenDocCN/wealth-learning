# #304《口红机怎

帮主 : #304《口红机怎么赚钱》

前一阵口红机在抖音火了，这个口红机和娃娃机很相似，花

10 元或者 20 元玩 3 局游戏，就有机会得到心仪的口红。

经营者呢可以在后台设定过关概率，大部分都设定在每

200~400 局出一只口红。

一台口红机的硬件成本在 5000-10000 元左右。

一个还不错的点位费用大概在每月 1k-4k 之间，加上电费和其 他人工运营费用，其实成本还是蛮高的。

如果点位不好，可能一天也没几个人玩。

我曾经在一个商业街的口红机旁边观察过 1 个小时，也只有一 个人玩了一次[流泪]。

所以就有人想到把线下口红机搬到线上，这样就可以借助微 信赚钱，不仅不受时间空间的限制，还能大幅降低成本。

要知道一个开发微信小程序才需要 3000~5000 块，比线下的成 本少了几倍。

后面就和面相测试的玩法差不多了，既有代理模式也有分销 模式。

其实我们很少会看到特别新奇的赚钱方法，大部分的赚钱方 法都差不太多，关键是找到一个需求，然后想办法解决这个

需求~
