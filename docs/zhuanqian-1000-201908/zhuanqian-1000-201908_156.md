# #249《美团外卖

帮主 : #249《美团外卖红包》

现在点外卖已经是很多人的需求，所以要是每天都能领到几 块钱的话，一年也能省 1000 多呢~

简单说下玩法：

1，建个微信群，每个人点完外卖都往里面发红包

2，外卖红包经常会告诉你第几个人领最大，如果你有小号的 话，就可以估摸着时机点进去看看。

3，比如是第 7 个领取的最大，那么当你小号发现自己是第 6 个 领取的时候，这时候让大号领取，就能领到最大红包啦~ 2018-07-18(29 赞)
