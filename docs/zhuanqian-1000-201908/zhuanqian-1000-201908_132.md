# #281《PPT 做

帮主 : #281《PPT 做的好也可以借助很多平台来变现》

很多职场人士的 PPT 做的都很漂亮，有没有想过通过这个来变 现呢？ 比如可以试着做些抖音视频，每天坚持发作品和自己的技 巧，然后留下自己的微信，接下来就可以通过卖自己的原创 模板之类的进行变现了。

有小伙伴说自己不会做好看的 PPT，你也可以尝试一下其他的 工具啊，比如 PS，Excel，甚至教别人做菜，慢慢发掘一下， 总会发掘到一些自己的优势能力的~
