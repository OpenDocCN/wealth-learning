# #274《甲醛房可

帮主 : #274《甲醛房可能导致白血病-室内空气治理行业可以 了解一下》

最近关于室内空气不达标可能导致白血病的问题又推到了风 口浪尖。

大家越来越关注室内空气的问题，室内空气检测和治理的需 求也越来越多。

我有一个朋友恰巧就是做这个行业的，今天和大家简单分享 一下他的创业思路~

以下来自她的自述~

我大学毕业之后没有找到太适合自己的工作，最后就选择了 创业~

当时也了解了很多的行业，最终觉得咱们国家的室内环保， 空气质量净化治理行业还属于一片蓝海，于是就选择了在这 个行业创业。

一开始只是有这么个想法，但是完全不知道该怎么操作，于 是我就上网查了很多资料，网上有很多的加盟公司，我就一 家一家咨询，最后花了几千块加盟了一家看起来比较靠谱 的，注册了个小公司开始经营。

可能是比较幸运，业务一展开就赚钱了。 平时出去发发传单，打打电话很快就有客户成交，每个月不 仅可以维持公司的开销，还能每个月赚 2 万左右。

但是，几个月以后我发现继续这种经营模式的话，业绩也只 能维持现状，很难提升了。

这时我发现了一个规律，有新生儿的家庭和幼儿园的需求是 最大的，于是我将自己的目标人群重新定位，还给公司更了 名，专业治理幼儿园和家庭儿童房。

我还同时找了本地的研究院，升级了自己的技术，提高了治 理的标准。

国家标准规定室内居住甲醛含量是 0.10mg/m3，儿童房的标准 是 0.08mg/m3。

我直接把自己的治理标准设置为 0.08mg/m3，不论客户是不是 儿童房，我都以这个标准来治理。

还在治理后帮助幼儿园申请质量监督检验院的检查和报告， 慢慢积累了很多口碑，我的客户也愿意将我引荐给其他朋 友，就这样我所在的区域 50%以上新开的幼儿园都成为了我 的客户。

这时候我遇到了第二个瓶颈，甲醛处理其实是个低频需求， 只有在新装修的时候能够用到，而且用户只使用一次。

一个城市的幼儿园只有这么多，我很担心全部都治理完了就 没有生意了。

碰巧我的一个客户打电话问我，能不能去帮他们幼儿园消消 毒，那时候正是诺如病毒高发期，幼儿园的小朋友很多，经 常是一个小朋友生病会传染给很多小朋友，院长为这个事很 头疼。

于是我又找到了研究院，经过一番学习和调研，推出了纳米 银消毒业务，并赠送给所有新装修空气治理的客户一次。

预防疾病传播对幼儿园来说是非常需要的，所以几乎我赠送 过的幼儿园都会办消毒年卡，每周消毒一次，这样又一次拓 宽了我的业务，增长了公司的业绩。

现在我也注册了我自己的品牌，也有了一些加盟的小伙伴， 但是创业的路上依旧有出现各种各样的问题，等待我去突破
