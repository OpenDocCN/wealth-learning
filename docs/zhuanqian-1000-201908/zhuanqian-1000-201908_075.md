# #350《百度贴吧

帮主 : #350《百度贴吧引流技巧》

百度系的产品现在管的越来越严了，基本上都不让你留联系 方式。

但是，百度贴吧流量又非常高。 那该如何引流？ 我测试了一个方法，成功率挺高。 步骤如下： 1、选中一个精准的贴吧

2、发引流文案：“我有很多这方面（根据贴吧名来修改）的 资料，但是发不了链接，一发就被和谐了，需要的朋友留下 邮箱，我发给你~“

请留邮箱的同学看仔细了，是让你去贴吧发帖啊！不是让你 在这个帖子下面留邮箱。。。真是哭笑不得，一堆人在下面 留邮箱[捂脸]

这样做的好处是什么呢？

1、别人回帖留邮箱是不会被和谐的，但是作为发帖人留邮箱 就会被和谐。

2、别人每回复一次，帖子就相当于被置顶了一次，这样看到

的人就会越来越多。

3、将资料通过邮箱发送一半给他们即可，资料里面留自己的 微信号，说加微信领取另一半资料。

懂的朋友可以去测试了。 当然，建议用小号测试比较好。
