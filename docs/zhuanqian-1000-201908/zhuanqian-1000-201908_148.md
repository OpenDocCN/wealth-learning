# #258《每日坚果

帮主 : #258《每日坚果怎么就火了？》

从前年开始，有一种坚果突然就火了，他的名字叫“每日坚 果”。

他把不同品种的坚果去皮处理并和果干搭配在一起，每天一 小包，号称可以补充各种营养。

根据市场相关趋势分析来看，每日果仁在 2016 年的市场增长 率为 300%。

大家有没有想过？为啥这个坚果突然就火了呢？

我们以前吃坚果都是单个品种购买的，为啥每日坚果能想到 研发这种产品呢？

因为他洞悉了用户更本质的需求。

很多人吃坚果无非为了健康或当零食打发时间，但是以前我 们如果想吃多种坚果就要单独购买很多包，开封后不吃又容 易坏掉，还需要自己剥皮，就很麻烦。

这里我们发觉到了用户的需求:健康，方便，可当零食。

每日坚果就是发觉到了用户需求，从而进行了创新，这款产 品就应运而生啦。

发散下思维，做个知识性的产品，从用户的需求出发，比如 每日一本书，每日一签等等都是不错的思路。

比如我的朋友李伟诚做的一个日签星球就非常不错，大家也 可以尝试行动一下。
