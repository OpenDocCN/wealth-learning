# #354《关于公众

帮主 : #354《关于公众号的好消息》 500 粉丝即可开通公众号流量主了。 之前是 50000 粉丝，门槛降低了 100 倍！ 开通流量主有什么好处？ 就是你文章底部可以加个系统自动生成的广告了。

这种广告单价极高，你的读者点击一次广告，你就能赚 1~3 元 左右。

一看单价这么高，有的同学可能就会想到建个公众号用户 群，然后互相点击对方的文章，从而达到多赢的效果。

有几个注意事项，请务必了解：

1、点击率千万不要太夸张哦，尽量小于 3%，比如你公众号 文章阅读量是 1000，那么累计点广告的次数最好不要超过 30 次（包括读者点的也算）。

2、如果被查到作弊了，第一次会封禁流量主功能一个月，第 二次就是永久取消这个功能了。

3、可以适当赚点小钱，不要太过就行哈。
