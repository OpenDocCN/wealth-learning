# #113《精准获取

帮主 : #113《精准获取你的粉丝》 涨粉是永远的需求，得粉丝者得天下。 那如何获取精准粉丝呢？ 以我自己为例，给大家参考下。

我去年做了一个读书饭团《人生逆袭手册》，但是我一开始 是没有粉丝的。

所以我就去各种微课群里面加人，就这样有了最初的几百个 粉丝。

后来饭团开始裂变推广，才把基数做大了。 为什么要去微课群里加人呢？ 因为听微课的同学一般都比较爱学习，是精准粉丝。
