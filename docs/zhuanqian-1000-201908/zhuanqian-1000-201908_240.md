# #031《我观察小

《我观察小区门口一个大爷三个月了。。。》

大家都知道，小区门口经常会有摆摊的，我这个小区也不例 外。

其中有一个大爷，每天都来。 他是干什么的呢？

卖油条。 油条现炸，卖 2 元一根，每天都有很多人排队，供不应求。 他赚钱的速度完全取决于他炸油条的速度。 我每次晚上回家都会路过他的小摊子，并观察一会。 他每天下午 16.00 开始，21.00 收摊，一共出摊 5 小时。 平均每小时炸 60 根*5 小时=300 根*2 元=600 元。 成本不到 100 元，利润 500+

一个月利润就是 15000 比大多数白领强多了。。。

门口摆摊的小商贩真是不能小瞧啊

人家其实是在闷声赚大钱。 以上是我亲自观察的案例，希望能对大家有所启发。
