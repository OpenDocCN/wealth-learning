# #297《宝妈兼职

帮主 : #297《宝妈兼职社区团购店主，月赚千元》

最后一公里带来很多新的商业机会，社区团购就是其中之 一。

社区团购的主流模式是通过平台将商品覆盖至每个社区店， 再由各社区店主将商品推销至社区居民，从实际层面解决订 单、配送、备货、提货、售后、结算等环节的痛点。

我所在的社区就有一个宝妈在做，收益还不错，每个月在家 里发起团购，偶尔帮忙送送货，可以月赚千元，在家赋闲的 宝妈可以尝试选择你所在区域比较靠谱的平台，做一个社区 店主。

目前这种社区团购企业也越来越多，未来基于社区团购可能 出现一波新的千团大战。
