# #194《他教烧烤

帮主 : #194《他教烧烤月赚 60 万！》

最近发现了一个烧烤神店，味道还算不错，人也不少，不过 光是一个串店的话，其实是没什么好分享的。

可是这家串店的老板不一般，他不仅开串店，还在串店招学 员，一个月能招 120 个学员，每人学费 5000，光收学费，一个 月他就能收 60 万

这些学员白天上课，晚上实践，就是说白天教了学员，晚上 晚上这些学员又能帮他工作，一举两得。

那这些学员都是怎么收到的呢?

首先就是有些顾客吃完饭觉得很好，在店里就报名学习。 另外一个引流的方法就是快手直播和抖音小视频，老板在快 手有 30 万粉丝，每天直播烤串 大部分学员都是慕名去学习 的。
