帮主 : #269《以小博大的投资项目-域名投资》

最近在研究搞网站，在选购域名的时候越来越发现域名对品 牌至关重要。

好的域名每年可节省一大笔广告推广费用，所以这么多企业 才不惜重金购入域名。

比如： 京东花费近 3000 万买下 JD.com 小米花费 2000 万买下 MI.com

......

域名投资是以小博大的投资，从几块钱到几百块钱都可以注 册。

www.net.cn 在万网就可以注册购买~

全球有很多牛人靠投资域名积累了巨大的财富，靠的是就是 对市场的敏感度、眼光和机遇。

不过市场上还是只有少数人能赚大钱的，投资有风险，入行 需谨慎哈。
