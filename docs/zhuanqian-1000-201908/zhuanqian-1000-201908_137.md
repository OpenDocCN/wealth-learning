# #273《教人谈恋

帮主 : #273《教人谈恋爱》

之前我们说过基于相亲需求的赚钱思路，今天再分享 2 个教人 恋爱的案例。

一、国内某个男性情感在线教育的服务商。主要业务是教男 生改造朋友圈制造虚假的网络形象吸引异性关注，及教男生 追求女生的一些技巧等等。注册用户 50 多万人，日活人数 5 万 人以上，年收入百万级。

二、一个小伙被女友甩了，伤心的他在一个网站里把自己的 故事写成文章，没想到关注的人非常多。他发现原来不少人 都在为恋爱烦心，于是把这些烦恼一一记下来，为大家逐一 寻找解决方法，很快就成了网站里的大神级人物。当时，他 一篇帖子就有 15 万的点击量。

后来他尝试做线上课程，一个人收费 260 元/小时，一个月最多 的时候有 320 多个学生。 到 2014 年底，他已经有了一个 16 人的团队，赚到了人生的第 一个一百万。

这个案例发出来是希望给大家一些启迪，只要是有需求的地 方，就有赚钱的机会~ 但是不建议大家模仿，毕竟恋爱还要真诚些，忽悠妹子可不 好哈。
