# #112《水果店的

帮主 : #112《水果店的会员卡策略》 现在的理发店啊水果店啊什么的，也都纷纷玩起了会员卡。 会玩的人，能大大提升销售额。 下面这个策略就比较不错，值得借鉴。 比如一家水果店是这样搞的：

1、假设你第一次购买了 30 元的水果，这时候老板就会向你介 绍会员卡。

2、预充 200 元即可成为终身会员，永久享受购物 8.8 折，并且 这 200 元是可以消费的。

3、此外，第一次买的 30 元免费赠送！ 是不是看起来很划算的样子？ 所以，自然就会有很多人办卡。 这样做的目的是什么呢？

要知道，现在用户最宝贵的就是注意力，你让他办卡就等于 长期霸占了他的注意力，长期受益是非常大的。

而且，通过办卡的形式可以迅速回流资金，要知道现金流是 非常重要的。
