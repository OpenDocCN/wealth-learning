# #339 如何判断一

如何判断一个项目是不是传销？ 很简单。

拉人头+交会员费，符合这两条基本就是了。

然后再看他有没有清晰的商业模式，如果模式不清楚，让人 看不懂，那就更危险了。

如果再和区块链什么的扯上关系，那 99%就是传销了。 当会员费收到一定数量，就是这个项目跑路之时了。 榴莲微视、共享纸巾、某某计步 APP 等全是传销。
