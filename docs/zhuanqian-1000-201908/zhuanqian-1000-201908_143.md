# #265《爱分享爱

帮主 : #265《爱分享爱生活，就能免费吃喝玩乐》

朋友圈里有个小姐姐，每天吃吃吃，逛逛逛，有一天她告诉 我她去的很多店都是免费的，还能享受 VIP 服务。

因为她每次去逛街或者吃饭，都会在大众点评打卡，大众点 评经常有一些霸王餐活动她也会报名，经常就被选中，所以 能经常免费吃喝玩乐。

平时外出比较喜欢探店的吃货们也可以像小姐姐一样每去一 家店就拍一些照片回来，在大众点评网上传并且写点评，然 后坚持抽去免费活动哈。

长期培养这个习惯以后可以提高商业敏感度，又能免费吃喝 玩乐，一年也能省下几百几千块哦。
