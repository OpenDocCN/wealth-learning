昨日，知乎网友“**希望正义会出现**”公开发表**《上海中医药大学首席教授李 X 与他的“学术后宫”》**，全文如下：

![](img/ccf32c62bd0b4e160f60948a71a91fc9.png)

我是上海中医药大学研二的学生，出生于 1997 年。

现揭发我校导师李 x**蓄意打造“学术后宫”“科研圈小红楼”，**

****滥用职权长期打压、PUA、性骚扰门下学生，****

******弄虚作假套取科研经费等涉嫌违法犯罪的无耻行径。******

****本人会为所有言论负责，只求相关部门能彻底查处李 x，还长期以来被欺侮的学生们一个公道！****

****![](img/479e46395de55d78f25a11c75a974342.png)****

******以下是李 x 的恶劣行径： ******

******一、李 x 对我的身心迫害******

****我于 2020 年秋季入学，经过激烈的角逐拜入李 x 门下，没想到这个业内赫赫有名，曾让我景仰崇敬的导师，后来竟成为我的噩梦！****

****研一入学没多久，李 x 就**营造出自己婚姻不幸的形象**，言行间透露出对我的喜欢和关爱。****

****从礼节性**拥抱到看手相，再到背后环抱，亲吻**，一步步变本加厉。****

****我从开始的害怕到逐渐被他说服，最终与他“交往”。****

****![](img/5cdf4dd5d20361599fbf240afa194d74.png)![](img/2f37b5bb67f798991de1966cc553a75d.png)****

****与学生建立“恋爱关系”****

****在一起后，**他承诺会娶我**，**每次开房他都只出示我的身份证**，出入酒店跟我保持一定距离，****

****微信上也不会正面提到“开房”等相关字眼，**每次做都不带，并且总是无 t 内 x。****** 

****![](img/33c91006bb5c36452de51069e62d70a6.png)![](img/73564e1ad446a5949e856171622acd50.png)****

****开房打车记录及付款记录****

****后来怀孕了，他让我打掉。****

******2021 年 4 月 20 日，我独自去做了人流**，****

****![](img/cb94b47740c6065d573793e658b31224.png)****

****由于没人陪护只能选择**有痛人流**，我在意识清醒的状态下听着自己的肉被刮下来，忍过了一次不打麻醉的刮宫术。****

****![](img/b5df2aa1d201534a23de4e5d4efb4eff.png)****

****但当时的我并不怨恨李 x，反倒在他的长期洗脑下，我认为这一切都是自己的错，不怪老师。****

****由于科研任务很重，打完胎第二天就去实验室接着干活了，结果在同门师姐命令我熬夜查资料时大出血。****

****其实她们经常刁难我，我想让李 x 出面主持公道，以为至少能得到几句安慰。****

****结果他无动于衷，还嫌我幼稚、不懂事，**对他来说堕胎好像是一件很平常的事情，伤心之下我想跟他分手。******

****李 x 为了稳住我这个“性伴侣”，**开始提出要给我换实验组，甚至还打算把没有跟他睡过的学生的文章给我**，****

******并要送我出国**，以此证明他是对我好的。****

****当时我是纯粹的恋爱脑，只想向他索取情爱，结果又总被敷衍。****

****因此，我很痛苦，开始跟他闹情绪，**让他把以前堕胎的钱还给我******

****![](img/955a97df79cbb458d9c7dafdf8974c5a.png)****

****堕胎后的几个月我身心俱疲，经常失眠，就让李 x 给我开些调理的药，一开始李 x 给我开了一些中药，同时配合西药舒乐安定。****

****但有一次，我让李 x 给我开安眠药，****

****![](img/3c10bbc0fc233babc50c4ef6c044fd9b.png)****

****他给了我一瓶装在“**附子理中丸**”的中药瓶子里，但实际上是西药的药片，****

******吃半片就会连着头晕三天**，这绝对不是正常的药效，****

******我怀疑他偷换成了某种副作用很大的药物**。****

****![](img/c6d137f688a73926a0847c7754337e47.png)****

****更恐怖的是，我发现李 x 给了我这瓶药后，**跟其他人说我精神不正常**，让他们不要惹怒我，不要相信我说的任何话！****

****2021 年末，我彻底跟他提出结束，并让他不要再冒犯我，否则会让他付出应有的代价。****

****结果他又说要送我出国，培养我，称自己上有老下有小，**企图装可怜博取我的同情******

******![](img/6aee7aefd4363da2b7c616fc5490dda6.png)******

****而那时的我已经彻底不信任他。还提什么父母！****

****这件事我都不敢告诉父母，怕他们在网上看到女儿被她的正人君子般的导师欺骗，怀孕、堕胎，这每一个对他们来说都是重大的打击！****

****我不知道该如何是好，就去找同学求助，**结果发现了李 x 的“学术后宫”秘密。******

****![](img/479e46395de55d78f25a11c75a974342.png)****

******二.李琦的“学术后宫”******

****她们说这是李 x 的一贯套路。****

****每当有新入师门的女学生，李 x 都会予以特别呵护，同时以各种方式引诱她们。****

****比如，他以习惯了美国的见面礼节为借口，每次面谈结束都会跟女学生拥抱，****

****如果对方没有明显反抗，下一次就是看手相，趁机握手揩油，****

******一步步洗脑、试探**，（很多女学生都遭受过不同程度的性骚扰）诱骗学生跟他恋爱，转送一些不值钱的礼物，****

****谎称会娶对方，脚踏几只船，肆意玩弄感情，每次开房都只用学生的身份证件。****

****此时我才如梦初醒，原来他之前的一切都是**“惯犯”**行为，****

****他害怕女学生留下太多证据，所以非常小心谨慎，****

******微信上时常以“聊课题”、“喝茶”等理由叫女学生去找他。******

****那些知情后还乖乖委身于他的，会得到学业和前途上的**“特别关照”**，****

****比如**发文章、入选优青、安排工作**等；****

****那些“不听话”的（比如我），李 x 会通过营造舆论使其“社死”从而消除威胁；****

****至于那些**不愿“献身”的，会被李 x 卡课题、压论文直到毕业前才给发表，相应的奖学金、出国留学等机会基本无望。******

****男学生更是遭受李 x 的忽视和边缘化。****

****此外，李 x 作为硕博导师，不致力于指点我们的学业，却把自己比喻成上帝，经常灌输要听话，学会讨好、服从于他的思想；****

****指责我们不努力，而实际上我们承受着高强度的工作安排，每周 6 天的工作时长，****

****从晨起到深夜的每一天，最后产出的科研成果却被挪用给李 x 身边的“红人”****

****李 x**为了培养自己看重的女学生，不管有无实质贡献，都会让别人做的文章都带上该女生的名字**，****

****临床数据也是手下的学生做的，但是毕业前只给实质贡献很多的学生一篇中文文章，达到最低的毕业要求。（博主发的这部分都附有相关证据，篇幅有限，不一一列举）****

****李 x 还涉嫌通过**非法手段套取科研经费**，这部分的证据请相关部门彻查！****

****由于李 x 师德沦丧，利用自己的职权优势，降维打击学生，****

****以爱情/学术前程为饵撩拨大家**为他“献身”、争风吃醋**，导致师门氛围诡异，产生很多与科研毫不相干的恶性竞争，****

****本该求真向善的学术圈子沦为乌烟瘴气的**“学术后宫”**，沦为受李 x 肆意摆布的“**科研圈小红楼**”！****

****没几个学生敢赌上自己的前途跟黑暗对抗，因此被欺压了十多年，每一届都如此！****

****李 x 很清楚这一点，所以才有恃无恐、变本加厉。****

******我们学医十余载，满怀热忱来到这所学校，本着治病救人的愿望深造，没想到先病倒的是自己……******

****又一度愤怒失望，为什么我们这群辛苦求学的医学生，****

****在本该圣洁的医学院，沦落到受导师压迫，**需要出卖自己的身体，用苟且才能换来一个“光明前途”？**！****

****我不知道这个世界怎么了。****

****因力量悬殊，我选择将所有事情公布于网络，希望借此引起广泛的关注和监督！！！****

****![](img/33d7e9e4a64ff15897a01d8466be5844.png)****

****最后，再次恳请相关部门能彻查李 x，我相信党中央，相信国家一定会还学生们一个公道！！还高等学府一片蓝天！！****
