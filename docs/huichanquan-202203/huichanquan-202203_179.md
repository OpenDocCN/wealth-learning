# 空农宅接网线，提供给境外团伙，每根网线每日可得 2 千元好处费

> 原文：[`mp.weixin.qq.com/s?__biz=MzIyMDYwMTk0Mw==&mid=2247531021&idx=5&sn=f1b8990b45ae82bb1ab6fcd0be63af60&chksm=97cbb135a0bc3823e83daa8cd7477f6d1789aba99f8b0e8e5b6c6685e9fefe7be1de5d44020d&scene=27#wechat_redirect`](http://mp.weixin.qq.com/s?__biz=MzIyMDYwMTk0Mw==&mid=2247531021&idx=5&sn=f1b8990b45ae82bb1ab6fcd0be63af60&chksm=97cbb135a0bc3823e83daa8cd7477f6d1789aba99f8b0e8e5b6c6685e9fefe7be1de5d44020d&scene=27#wechat_redirect)

2 月 18 日，上海奉贤警方发现，柘林镇营房村一处无人居住的农宅，近期突然有人申请接入了两根网线，还安装了四门电话，这一离奇情况引起警方注意。

经调查研判，网线系不法分子提供给境外犯罪团伙使用，每根网线每日即可获得 2000 元好处费。

目前，7 名嫌犯已被依法采取刑事强制措施。
