就在前天晚上，一则《滴滴司机迷奸乘客》的消息在全网传播。滴滴方面紧急回应：已报警处理并紧急核实。![](img/7217f32895d560b46eeece3a30bd68a9.png)经过启人的核实，让这则消息快速传播的导火索，来自一条微博。一位名为“巧克力翠翠杀”的网友，在昨天下午 18:41 分**痛斥滴滴司机性侵乘客，而且并非个例，希望有关部门尽快介入调查**。![](img/3ef337cee68683b2855e3e484b110fb9.png)不过要注意的是，**她所有的信息，又都来自 GitHub 上的一篇爆料**。她只是将 GitHub 上的这篇文章搬到了国内的微博，然后用自己的话说了一遍。![](img/0ab01e7a8795041ff10bf299cf863b19.png)在 GitHub 的原文中，一位匿名爆料者洋洋洒洒写下了上千字的檄文。匿名者（性别未知）认为：**性侵乘客已经形成了一条黑色产业链**。 ![](img/8ef002f23d80178f7fe3cc0bd5eb5db2.png)滴滴司机会以各种各样的手段迷奸乘客，并在直播平台上骗取收费观看。 ![](img/5ed806eed36cc34d634833ec7e8747e7.png)这一切看似都能说通，不过看到这位爆料者提供的视频，启人发现，**里面的“剧情”并不符合常理**。 （视频点击下方观看）

视频中所谓的滴滴司机，一边开车，一边用手机直播，完全不符合安全驾驶要求。 而她迷奸女乘客的方法，**仅仅是喷三下疑似香水样的迷药**。 在喷过“迷药” 5 分钟后，视频显示，女乘客陷入了昏迷状态。随后，司机和观众互动，要求观众送礼，并对女孩实施性侵。 ![](img/809f633887acce9122afa00eba64d7a7.png)问题就出现在这里： **这种超强力的迷药根本就不存在**，即便是医用的挥发性麻药，最强力的药物，也需要 5-8ml ，而且要全部吸入。如果要受害者维持昏迷的状态，整个过程必须持续给药。 但司机的视频中，完全没有这样操作：用“药”不到 1ml，没有持续给药，**司机甚至没有在进车后开窗通风**。（知乎网友：奋斗的小蚂蚁）![](img/56d4ba2a1133fc725627da5c3dc58a72.png)迷药是假的，晕倒也是假的。 **唯一的解释就是：这根本就是角色扮演摆拍**。类似前些日子色情网站中的“美团外卖”系列，但那根本不是真正的美团外卖员。![](img/4e987b1ef032fd7112c4292649e51f25.png)在这件事情发酵之后，GitHub 上的匿名爆料者也表示，她的视频来源自某视频直播平台。 ![](img/0eb366c4bd24f9e32e9d0af2185a7a79.png)至此，这件事真相大白： 

*   视频平台确实存在。

*   里面也确实有很多“迷奸”视频。

*   但那都是演的，和滴滴、美团等公司没有任何关系。 

目前，此平台已被网友举报。**最新消息：**

12 日，网传滴滴司机性侵直播，郑州公安通报：两人系夫妻。
