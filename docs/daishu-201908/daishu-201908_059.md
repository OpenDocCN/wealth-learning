# —又来聊信息差。昨

张佳 : —又来聊信息差。 昨天发布在自己星球，同步给大家～ 聊一个信息差或者方向选择的问题。 我以前没想明白，最近勉强想通了。 举个例子：

我最开始写公众号的时候，思考了良久要写什么。最后的方 向大家也看到了：数据、产品和互联网。

出发点是，要做自己专长的事情。 实际上，当初那个选择是一个没啥前途的方向。

不信你看看我现在的这些粉丝（也就是你们），多特么精 明！想从你们身上赚点钱、搞点流量难得一匹！并且，稍微 有些东西写的不够专业、不够深度了，你们就不会赞赏、分 享甚至点赞。

这很累，并且受众不会太大，因为聪明、有思想的人很少， 很难做到规模化的刷屏。

写公众号这件事，其实是一个“降维打击”的活：你得有足够 高的水平，才能让你的粉丝读的“欲仙欲死”。如果你和粉丝 只有一点点差距，很难做大。

靠内容赚钱、赚流量是一样的事（还包括知识付费、技术分 享、资料分享），要有足够大信息差才行。

所以应该怎么选方向呢？

我的建议，尽量不要选自己专长的方向。因为“专长”这种事 情，要非常专长才行。但实际上，如果你足够专长了，肯定 有更赚钱的方法，而不是写个公众号、做个抖音号这种。

要选那种，受众非常广，并且受众是看不上的人。当然不是 道德品质上的看不起，而是能力、思维方式等方面的看不 上。

“看不上”这一点很重要，这表示了你和他们有足够大的信息 差，你可以“驾驭”他们。

然后剩下的，就是研究你的受众，用你们之间的信息差去频 繁的“折磨”他们。

到这里，按照以往的习惯我会举几个例子。但今天不行了， 我都说了“我会看不上那些受众”，说了就伤人了…
