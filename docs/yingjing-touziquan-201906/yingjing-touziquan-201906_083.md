# 85.猪周期复盘一：

禺 : 猪周期复盘一：

今天我想给大家讲一下，我是怎么发现养猪行业这个重大 投资机会的，希望对未来大家寻找新的投资机会提供借 鉴。

去年 7 月做完养鸡之后，其实我就一直在寻找新的投资机 会，我说的这种投资机会不是一般人理解的那种今天买明 天卖或者持有几天几周的那种机会，我所指的是可以在相 当长时间内重仓操作的重大投资机会。

从 2018 年下半年之后我就一直觉得大盘基本到了底部区域， 当时如果买入一些有潜质的股票埋伏进去应该会有很不错 的收益，但是就这样找了几个月都没有找到太满意的机 会，比如 5G 当时我一直觉得估值有些高，市场太热，再比 如油运，当时也是我关注的一个重点行业，也是在行业底 部区域，但是说实话行业反转还不成熟，不确定性因素很 多，还有其他一些机会，都不是那种能长期看好，让我能 下重仓买入的股票，后来发现了凯撒，游戏这个行业，竞 争很激烈，游戏新品的跟踪需要花费大量时间，凯撒作为 一家三线游戏公司有黑马潜质，但也有不确定性，这种类 型的股票可以作为仓位配置，但不属于战略性的大机会。

就这样一直耗到了 11 月初，一个特别偶然的因素我看到了 一篇研究报告，我现在反复回想，我竟然想不起来当初看 到的是哪个券商谁写的研究报告了，大意好像就是非洲猪 瘟可能会使行业拐点提前，这才是我关注猪周期的开始。

这里我就要和大家说一下重点了，分析一个行业或者股票

有没有大机会，首先要简单分析几个要素，业绩的爆发 力，估值，还有增长确定性等因素，简单分析之后如果不 符合直接就可以排除，大家在做投资筛选股票的时候也要 建立一个最简单的筛选方法：

对于业绩爆发力来说，周期股的爆发力一般比较强，而猪 周期又是其中弹性特别大，爆发力特别强的，周期低点几 个亿利润甚至亏损，周期高点业绩增长几十上百倍都可 能。

对于估值，我记得我当时刚开始关注正邦时市值不到 100 亿，头均市值不到 1000，对你没看错，当时正邦头均市值不 到 1000，预估周期高点 pe 只有 2-3 倍。

对于确定性，猪周期的核心就是判断行业景气反转的确定 性，当然周期股还得预判反转时间和空间。

至于我是如何一步步验证猪周期逻辑的，我以后有时间再 简单讲一下。

最后我希望大家要记住，做投资一定要掌握正确的投资和 分析方法，否则即使重大机会摆在你面前，你也抓不住。 更坏的情况就是把本来不是机会的机会当成机会，重仓操 作损失惨重！而且还有很重要的一点，做投资一定要学会 忍耐和等待，等到真正机会的出现，除非你说现在就是大 牛市，那不管什么股票，你闭着眼睛买就行了。
