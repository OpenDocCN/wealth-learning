# 分享一本书《稀缺》

2018-11-25(4 赞)

评论区： 纯洁的微笑 : 坐等

上官伟斌 : 1、稀缺俘获大脑，带来一点点好处，我们能够在面对迫切需求时，做得更好。但从长远的角度来看，我们的损 失更大；我们会忽视他所需要关注的事项。 稀缺让人变笨和更加冲动 孤独就是社交稀缺 2、借用：当面对资源稀缺时，不 会通过借用相应的时间或金钱来就对突发事件。从长远看，借用会进一步加剧稀缺。 3、带宽,就是心智的容量,包含认知能 力和执行控制力。稀缺会降低带宽的容量，致使我们缺乏洞察力和前瞻性，还会减弱我们的执行控制力。 4、管窥：专注 于某一事物就意味着我们会忽略其他事物 消防员在出警途中由于未系案例带导致坠车身亡，原因就是接到报警后消防员面 临时间稀缺的问题，途中他们要制定复杂的消防策略而忽视其他事项，这是专注红利的代价。 制定计划很重要，但是并不 紧急，而这类事情正是管窥心态导致我们所忽略的事情。 5、余闲就是我们在拥有很大的空间，不存在稀缺心态时的产 物，也是我们在资源丰富时进行资源管理的特定方式，余闲可以将我们从做权衡的苦差事中解脱出来。 想象着你要出差， 使用大行李箱收拾时，人们总是十分随意；而用小行李箱时，人们便会变得小心翼翼，思索再三。 余闲是应对突发事件的 利器 圣约翰中心手术室不够用的例子，手术被延期，员工因为加班而工作效率不高，重新安排手术又额外消耗医生的精 力。最终的解决方案是留下一间手术室待用。 道路占用率为 70%时达到最佳的运转状态。 面对太多等待去做的事物，人们 第一反应就是将时间排得慢慢的，然而一旦发生一些突发情况就会牵一发而动全身，导致全盘崩溃。最合理的做法就是留 出适当的闲余，不会太多，也不能太少。 NASA 火星探测器团队由于弄错力学单位而导致探测器坠毁的例子，类比软件工 程其实就是赶工期的时候缺乏测试。 亨利·福特制定了每周 40 个小时的工时规定，不但增加了工人的总产出，又减少企业的 生产成本。 但在资源充裕的时期，人们容易犯下拖延的毛病。 意志力就像肌肉，在频繁使用时会变得疲劳。如果某个人需 要持续的抵御诱惑，那么意志力就会被耗尽，想了要逃离稀缺陷阱就会变得更难。 起轮事故：二战期间，飞行员在降落 后，总会在收起侧翼时误将轮子收起。是飞行员过于疲劳吗，还是飞行员放松警惕了，都不是。真实的原因是驾驶舱中的 机轮控制杆和侧翼控制杆挨在一起，容易诱导飞行员误操作。

上官伟斌 : 老外的书陈述的观点都是有据可依的，都有相应的科学实验支撑，但同时也带来了篇幅过长的问题

上官伟斌 : 稀缺防御： 利用外界的干预手段阻止稀缺心理对现状造成实质性的破坏，如助理提醒开公时间快结束了；公路 上的减速带 拉入管子：将重要的事情接入你的管子视野中，如每月的储蓄提醒；私人健身教练会和你约时间 疏忽=默许： 设置忽略问题时等同于默认的解决办法；如支付宝每月自动充话费；自动还款； 一性性行动：将多性操作转成一次行动， 如办理公交卡，就不用每次准备零钱 安息日：权衡会消耗大量的精力，根本无法做到两全。犹太人的安息日，是应对稀缺 权衡的一种独具智慧的传统。人们在安息日不工作、不发邮件、不写字、不做饭、不开车。宁静而安详的一天，就是用来 恢复活力的。都有人都只能休息，不存在权衡问题。“安息日是上帝赐予的时间礼物” 为带宽不足的人提供便利，举个反 例：享受公众福利的人每年都要填写大师的表格，这等同于向申请财务补助的人收取一大笔费用 人们会知道自己缺乏时 间，但很少会说自己缺少带宽。尝试说“我带宽不足了，做这件轻松的事吧”
