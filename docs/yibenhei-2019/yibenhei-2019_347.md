01**看 X 网被黑**

大约两周前，我们收到一封求救信，起初我们并没当回事，因为，太 TM 搞笑了。

邮件是个小哥发的，他说自己逛 X 网时被人家黑了摄像头，一番激情操作都被录下来。对方发了邮件给他，说让打钱，否则就把视频发给他的亲朋好友，让大家都看清他衣冠禽兽的一面。他希望老师傅出手帮帮他。

我把邮件发给老师傅，老师笑的差点被烟呛死，并回了句，这也信？

原以为这件事在我们的笑声中就过去了。

可是几天后又收到这位小哥的邮件，这次还附上了勒索邮件全文。

小哥说自己非常慌，在收到邮件后，靠着 Google 一个字一个字地翻译知道了大概意思。

小哥仔细回想了一下平常逛过的 X 网。但由于自己射猎范围实在太广，具体是在 P 网还是 X 网还是哪个不知名的小站被黑的，他实在没有头绪。而且今年又刚换了安卓手机，下了好几个深度生理教学 App，还都是破解版，在手机端被黑也有可能。

“刚谈了女朋友，万一这种事被她知道了，搞不好要分。”小哥在邮件里写到。

小哥找了最好的朋友商量，朋友建议报警，但在交流之中，他也逐渐发现了一些端倪——文章里提到了比特币。虽然自己英文不行，看人体互搏只听得懂嗯嗯啊啊，但是对 BTC 和交易规则还是了解一些的。

但显然发邮件的这个人并不清楚小哥是个数字货币资深粉的事情，因为邮件里居然还像教小孩一样，告诉他要给 BTC 地址汇款应该怎么操作。

“这真的是一个黑客吗？老师傅每次出手可是连别人内裤颜色都扒得出来，但这人却连我的基本信息都不清楚。难道，邮件是群发？”

这时，远在国外的好基友也给发来了深情的慰问，他说自己也曾经收到过类似的邮件，但是内容读起来太幼稚，又没 video 附件，十有八九是骗子。劝小哥别当真。

虽然基友的安慰让小哥的内心倍感温暖，但是 96 和 69 的区别，总要亲自上阵才知道不同的。所以无论如何，他都希望老师傅能帮他这个忙，查一查。

02  **广撒网 多设套**

尽管小哥即将面临被分手的风险，但老师傅依旧不打算插手，他说，这就是个骗子，有什么好查的。

我对此感到很愤怒，老师傅也太没人性了，就算是骗子也找个证据出来，让人死心啊。

果然，自己单身就见不得别人好，于是我选择自己去联系小哥。谁让老黑我总是收到好人卡呢！（摊手.jpg）

我给小哥发了邮件，问他后续还有没有收到勒索信。他说有，在被拦截的垃圾邮件中找到了，不过是更早之前的事，邮件和之前一样，也是代发，内容也大同小异。唯一不同的是发件人地址和钱包地址。

机智的小哥还自己查询了两家网址，结果你猜怎么着？一个是日本某设计公司的网址，另一个居然是养马场的网址。

不仅如此，小哥说他的企业邮箱在二月份也被异地登陆过，地址在德国。

我很欣喜地把这些信息同步给了老师傅，问：“干他娘的一炮？”，但他并没有给我任何回应。

不得已，我只好递上准备已久的黄鹤楼。老师傅瞥了一眼，没伸手。

我。。。我只好把整盒烟都递了过去。

行，啪。你牛你有理。

老师傅比较了这几封邮件，发现除了姓名和地址，其它内容全都是一样的，也就是说对方伪造了邮件头，再通过内部关系拿到 X 网的用户信息，之后广撒网钓鱼。

“这样很不精准啊，上当的人不多吧？”我问老师傅。

**逮到一个算一个呗，又不要什么成本。**

“那你现在能看一看吗？”

老师傅说晚上试试，他一会得出门。为了证明清白，中途他还拍了照片发我。

行吧，都等那么久了，也不差这几个小时。好不容易等到晚上，老师傅终于上了线。

但我们目前掌握到的全部有效信息就只有这个邮箱地址而已。老师傅说，“无妨，发来我看看”。

老师傅看完之后，发现邮件内容几乎一致，索要的金额也都为 1000 美金，而且所有邮件都是代发。这绝对不是巧合。老师傅说对方极有可能是团伙作案。

为了验证老师傅的猜测，他找了一个专门做黑 IP 解析的朋友帮忙，发现这些 IP 地址表面上看起来分散在世界各地，但实际上同属于一个地区。曾经做过大量的垃圾短信推送，也就是类似这种勒索邮件的推送。

也就是说**这类邮件并没有任何针对性，全都是群发**，就是老师傅说的广撒网。

03 **认真你就输了**

被勒索邮件威胁的可不止小哥一个，网上这样的新闻一抓一大把，而邮件不仅有英文的，还有日文、韩文各种版本。

随着 AI 换脸面世后，这类勒索邮件的内容看起来更加可信，因为他们会提供“真正”的视频证据。

他们通过社交网站获取收信人的个人照片，并将其置换到 AV 之中。尽管看起来并不完全相像，但散播出去依然能引发腥风血雨。

在一些勒索邮件中，他们甚至能给出收信人真实的社交账号及密码。

那么这些账号和密码从哪来的？为什么只有部分人收到？

大概在 2018 年，Facebook 及领英等一大批社交网络曾发生大规模的用户数据泄露事件。也就是在那次事件中，一些黑客窃取了大量账号数据及用户隐私。

被泄露密码的用户，有些及时更改了密码，有些选择弃用，有些则无所谓。而这就是为什么有人恐慌，有人惊讶，有人无所谓的原因。

尽管账号和密码是真的，但邮件中所提到的，黑了摄像头录了视频基本是不可能存在的。所谓的获取了你的通讯录资料也不太现实，社交网络好友，或许有这个可能。

老师傅再次提醒，收到该类邮件要做到“四不”，不慌、不信、不回复、不打钱。

要是你回复了，就真的上当了。所有的诈骗，玩的都是人心。

只要你恐惧，你就会怀疑；只要你怀疑，你就会行动；只要你行动，你就会上当。

> 如果邮件里提供了链接，忍住，别冲动，手别贱。
> 
> 不要在社交媒体公布个人隐私信息。
> 
> 不要所有账户都使用同一个密码（或者过于简单的密码）。
> 
> 不定期检查自己的账户。
> 
> 最后一点，也是最重要的一点，看片别乱来。
