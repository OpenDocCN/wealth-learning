********04 ****观看即是同犯********

******鉴于我国没有影片分级制度，所以，一切有色影片都是非法的存在，甭管是成人的还是儿童的。******

******群内传播的成人影片来源，看过的都知道，此处略过。但儿童色情影片的来源必须引起重视，因为所有儿童色情的背后一定都是是犯罪。******

******任何一个国家的儿童色情都是违法的，儿童色情总是伴随着性侵或是猥亵。******

******2018 年媒体报道儿童性侵案例共 317 起，涉及受害儿童超 750 人。出于对儿童的保护，儿童性侵案件一般保密处理。所以，317 仅仅只是被曝光的案件，更多的犯罪藏在你看不见的阴暗角落里。******

******![](img/943deead8da852c48716403400f44f25.jpg)******

******在那些阳光照不进去的地方，我们能做些什么呢？******

******或许我们无法阻止儿童性侵的发生，但是我们可以尽自己所能去减少这些事件对孩子的伤害，譬如，发现传播传播儿童色情影片的组织，及时举报、报警。******

******平台不只有享受流量的权利，也需要承担监管的任务。******

******tumblr 因为儿童色情泛滥，永久禁止平台内一切带色内容的传播。虽然微信和 QQ 也关闭了漂流瓶，但这显然不够，大佬能做的远不止这么点。******
