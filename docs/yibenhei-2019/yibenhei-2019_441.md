每个人都有属于自己的人生轨迹，也许平稳、也许波折，但对于

他有着优越的家庭条件，二十几年来，一直是个不知愁滋味的少年，坐拥香车美女，过着令人羡慕的潇洒生活，谁也未曾料到，他日后会参与一起诈骗案，并在东窗事发后，利用假身份，开启了长达一年多的逃亡生活。

**从逃亡到被抓，家豪经历了骗安眠药、惊险人脸识别、肥皂雕象棋等。**

再次回忆这段不寻常的人生经历，家豪的内心、五味杂陈。

文 | 木子梨

# 

**01****断崖式下跌的人生轨迹**

家豪的父母，都是体制内人员，在他们看来，家豪也应该拥有同样稳定的人生。

本科毕业前，家豪的生活确实很稳定，但毕业后，家豪先去了香港，和朋友一起炒手机，后来又回到深圳，从事酒店供应商。

眼看家豪的工作逐渐平稳，父母迫不及待为他安排了一桩婚事，两位年轻人在双方家长的施压下，未领证就同居了五个月，最终，还是因为感情不和，而选择分手。

这段五个月的关系，在双方各自寻找律师，签署了“解除非法同居证明”等一系列材料后，彻底宣告终结，而这，也成了家豪与父母矛盾爆发的导火索，一气之下，家豪离家出走。

失去了经济来源，加上短时间的释压挥霍，家豪口袋瘪了，却依然赌气不与父母联系，此时，正巧有两个曾有金融相关经验的朋友找上门来，提议合作一把，家豪欣然同意。

三人虚构了一个项目，以此来拉投资、套资金，不到半年，就搞了 700 多万，最后，一些客户发现情况不对，报了警，见东窗事发，三人迅速瓜分了这笔钱，分散躲藏。

**一夜之间，家豪完成了转型，从纨绔子弟，变成了网络逃犯。**

在花了 800 块，从网上买了假身份证和配套的银行卡，又买了数张未经实名认证的电话卡后，家豪在成都，开始了隐姓埋名的逃亡生活。

# 

**02****备受煎熬的网逃生活**

人生之本，衣食住行。对于网逃中的家豪来说，最难的、是住和行。

家豪用买来的假身份证开房住，一方面是因为兜里有钱，另一方面，是因为酒店人员流动较大，入住不易引起怀疑，每半个月或一个月，就要换一家酒店。

使用假身份证时，要做到镇定自若，这对心理素质是一种极大的考验。

家豪把假身份证所属的个人信息，牢记于心，住宿会提前在网上预订好酒店，到店登记时，要装作若无其事，在前台询问是否有预订时，家豪理直气壮的把身份证甩到桌面上，说有预定，然后旁若无人地低头看手机。

但是让家豪有些意外的是，越是等级高的酒店，尤其是预订高价房间时，酒店前台对于证件的核查，异常宽松，有时戴个墨镜，前台丝毫不会查问证件。

唯一一次住酒店被查问，反而是在一个普通的连锁酒店，当时，前台拿着身份证，仔细打量并多次询问是否为本人，家豪一口咬定，后来，前台就没再问了。

“双方对峙，这就是一场心理战，你要是怂了，对方肯定会报警，那就挂了。”

而迟迟未被识破，也是因为买来的身份证，照片轮廓和自己是有几分相似的。当初购买的时候，卖家发来一个压缩文件，里面有数以百计的照片，总能在里面找到一个与自己最相像的照片，后面只要参考照片，剪一个差不多的发型就好。

**但因假身份而起的惊险，远不止这一回。**

还有一次，因为太过无聊，家豪上网报了一个旅行团，前往某佛学院。

因为那里是严管区，路上有警察用 PDA 查身份证，家豪假装淡定，将身份证递过去，只听“滴”的一声，顺利通过，本以为万事大吉，万万没想到，到了佛学院门口时，要排两队才能进入，而远远望去，入口处明晃晃的摆着一个人脸识别机器。

随着队伍慢慢前进，轮到自己了，家豪将买来的身份证，靠近感应区，并抬头看向摄像头，意料之中，人脸识别没有通过，此时，家豪内心异常惊慌，担心万一这系统自动报警，那就真的玩完儿了。

正当内心风声鹤唳、草木皆兵的时候，一个男人走过来，问了一句：“你是不是整过？”，白送的解药当然要吃，家豪马上回答说：“是啊，整过”，男子看了他一眼，说完“旁边等一下”，就去维持后面队伍的秩序了。

等在这儿，还是用真实身份证先混过去？

家豪站在边上，经过一番挣扎，内心的不安占了上风。他趁四下无人注意自己，慢悠悠晃到另一边队伍中，掏出沾满手心汗渍的真实身份证，这次、过了。

“当天晚上回到酒店，回想起白天使用自己真实身份证的行为，心里忍不住后怕，不停的担心警察会不会因此，而发现自己的行踪。”

结果是，他仿佛走了狗屎运一般，一切如常，什么事都没有发生。

网逃的一年多里，不管是地铁里，还是广场上，每当看到警察，或者听到警笛声，家豪都不由得心跳加速，手心直冒冷汗，表面却还要假装淡定。

期间很多次想去自首，却又迟疑。就因为这迟迟下不去的决心，家豪经常彻夜难眠，也会借酒浇愁到天亮，但即便如此，也只能短暂入睡几个小时。

睡不着觉，是一件特别痛苦的事情。想借助安眠药，却因为其有成瘾性，属于国家管制药物，不能随意购买。

怎么办？骗呗。

家豪去医院，谎称自己是从事开吊机这类高危工作的，白天需要精神高度集中，导致压力过大，晚上睡不着觉，这时，医生就会开一些，但也只有几颗。

吃完再换个医院，以类似的说辞去开药，靠着这些骗来的安眠药，家豪才能勉强维持基本的精神状态。

**本以为这样的生活会维持下去，然而，命运是最好的编剧，让他在最惊险的情况下全身而退，却在毫无防备的时候，面对迟来的被捕。**

# 

**03 ****网逃结束后的返深之行**

2018 年春节后的某天，就像之前无数个平凡的早晨一样，家豪在酒店里刚刚洗漱完毕，听见服务员在敲门，打开门，两个警察站在门口，说的第一句话是：“我们是深圳 XX 分局的。”

“在外奔逃了一年多，这一天终于还是来了，只是没想到自己会如此平静，也许是打心眼里厌恶了担惊受怕的感觉。”

**被捕后，家豪先在成都被提审，过程中，家豪最先对警察说的话，是谢谢。**

网逃期间，最怕逢年过节，一个人孤零零的漂泊在外，和以前光鲜亮丽的生活，形成了鲜明的对比，每每想到自己从普通人到逃犯的身份转变，还是很难接受。

最让家豪恐惧的是，就算自己的逃亡之路顺利，但自己的真实身份证还有几年就过期了，需要再次补办，到那时候，自己肯定不能去补办，这也就意味着，不能再做回自己了。

而买来的假身份证也会有过期的一天，到那时候，是再买一张，适应一个新的身份，还是？家豪不敢想。

“其实被抓是一件好事，警察再不来抓我，我也马上要因为巨大的心理压力，而去自首了。”

而后，在被警察押解回深圳的时候，家豪体验了一次最“特殊”的登机。

一辆车，家豪和便衣警察一行三人，提前于该趟航班所有旅客登机，坐在了最后一排，家豪手上戴着的手铐，被警察用衣服遮挡住，表面上无异于正常乘客。

不过，在航班飞行过程中，也有尴尬的事情发生。

由于家豪是该航空公司的白金卡会员，航空乘务长照例过来问候，走到家豪面前，第一句话是：“白金卡是哪位？”

带着手铐的家豪，只能笑笑。

当航班抵达深圳，所有乘客都下机后，家豪一行三人才起身离开。

**04 ****看守所里的百态人生**

进看守所的第一天，次数最多的感受，是惊吓。

进了大门，先后需要经过体检，换衣服，拍照、按十个手指的指纹、以及双手掌纹等流程，还要通过做下蹲，来检查是否有夹带，流程走完，就会被安排进舱。

看守所分为内舱和外舱，内舱用来住宿，有很多房间，外舱用来活动，洗澡、晒太阳都在外舱。

内舱每个房间都有标号，能容纳 30 个左右的犯人，没有床，大家都打地铺，睡在地板上，每个人能活动的面积很有限，人均还不到一平米，稍大的翻身都会踢到旁人。

但，不是人人如此。

大哥级别的人，拥有的“领地”就会宽敞很多，还有一些“VIP”人员，也能睡的宽松一些。

家豪靠着察言观色，左右逢源，慢慢取得了“大哥”等人的好感，为自己赢得了一个较为宽松的睡觉范围。

在看守所里，有拖地、打饭、打扫厕所等工种，每个人都有着非常明确的分工，当然，级别高的人是不需要干活的。

家豪负责的是打饭。看守所里，一周吃两次肉，家豪每次需要从餐口，接进外面送进来的盒饭，然后把盒饭里的肉，按地位依次来分，一般的人员是很难分到肉的。

看守所里是禁止讲方言的，大家都只能说普通话，这是为了防止拉帮结派、影响团结。

慢慢的，家豪适应了这里的生活，虽然辛苦，但至少心里安稳。

在看守所这样特殊的环境里，孕育着闲来无事时的特殊消遣。

白天那么长，总要找点事做，想下象棋，没有怎么办？自己做呗~

看守所里有生活用品购买的地方，每个人都有一张卡，家人充值即可使用。因为肥皂是不限量供应的，大家就买来肥皂，用肥皂雕刻成象棋，然后围在一起，来几盘厮杀。

“那象棋雕刻的特别像，仿佛就是真的，人在无聊的时候，也许真的能激发出一些潜力。”

在看守所待了五个月后，家豪得知了判决结果，而因为认罪态度积极，加之主动退还所有不法所得，取得了受害者的原谅，经过考量，家豪获得了缓刑。

缓刑的代价，就是必须 24 小时佩戴定位手环，外观像个智能手环，每天都必须打卡，打卡时，手环会自动上传当前位置，有点钉钉打卡的既视感。

（监外人员佩戴的电子手环：图片来源于网络）

同时，上交护照，不能出国，甚至连港澳通行证也一并上交了，平时只能在深圳市内活动，特殊情况需要出省时，必须打报告，经过允许才可以。

并且按照规定，平日里，家豪每月必须要做够一定小时数的义工，还需要前往司法所，听取规定课时数的法制教育课。

“不管怎样，至少不用再提心吊胆、东躲西藏了。”

**THE END**

后来，家豪才知晓，警察是如何找到自己的。

原来，警察通过对网购、外卖、打车等互联网行为进行追踪，通过大数据比对和分析，哪怕在频繁更换电话号码的情况下，依旧可以较为精准的定位到你的踪迹，所得数据，甚至能够细致到你接入的 WiFi 名称。

**科技改变生活。**

**法治社会中，也许可以一时侥幸，却无法一世侥幸。**

**天网恢恢，疏而不漏。**

**劝你善良，因为干了坏事，你真的跑不掉。**
