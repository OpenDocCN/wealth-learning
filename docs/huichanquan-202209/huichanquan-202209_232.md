# 上传到“成人网站”的女生，都是怎么被拍下来的？

如今的科技水平也比较发达，各种微型摄像层出不穷，

促使有很多无辜女性被犯罪分子，用尽各种龌龊手段偷拍。

![](img/6a923a0e70dda194385a82a58dc94af8.png)

现在的互联网，也改变了偷拍的性质，再也不是为了一己私欲。

而是通过各种色情网站不断的传播，甚至以极其低廉的价格出售，获取利益。

这些偷拍者的手段专业到让你根本发现不了，也正因为这群偷拍者，你不知不觉的就成为了非法网站的女主角，被大家肆意讨论。

![](img/6ee8090a643271ba3567a367f9bdf8ad.png)

之前，苏州就有一起“情侣酒店被偷拍”事件；

张先生和女友来到苏州旅游，在景德路一家主题酒店入住了三天

可高高兴兴回去后没多久，张先生的朋友无意间看到他们**长达 40 分钟的亲热视频，全是张先生和女友酒店内的亲热的私密画面。**

更让张先生气愤的是，这个视频已经传播了一个多月，

在一次次的传播下，这段视频已经被**4 万多人点击观赏**。

![](img/5d9e16d58a2291ea8b04a22032f2cc90.png)

![](img/58af7c7fdeddb28683efd08b46b1046f.png)

视频底下是上百条不堪入眼的评论，

想到自己的身体从头到尾被“观赏”过一遍；

**张先生的女友直接崩溃，并有了数次轻生的念头。** 

![](img/717be1eb31b2a29ab8b33fc3a19b07de.png)

除了在常见的酒店、KTV 等私密空间纷纷中招外，就连健身房也不能幸免于难；

要知道健身房里的男男女女基本都是半裸相见，这更是激起了偷拍者的欲望。

**有人专门偷拍健身房更衣室里的裸体男生和女生，然后以高价卖出这些照片。**

在海口市的一家健身房，就有那么一起事件，一个女会员在健身房的浴室被偷拍，将偷拍的裸照刻成光碟，装在信封中放在私家车的窗上。

嫌疑人将自己的电话留下，向受害者进行勒索，只要不答应，就把照片和视频发在成人网站上供人观看。

还听说偷拍都会火？的确是会火，是人民群众对你的一腔怒火。

![](img/982ac490580bd7151bb66eac19bba5af.png)

虽说健身房在很多人眼中是积极有正能量的场所，但是林子大了什么鸟都有；

所以大家在锻炼身体的同时，

也要注意保护好自己的隐私，不要让别人有可趁之机。 

![](img/61ea58cb5cb87fe7bf2566476ad17136.png)

明枪易躲暗箭难防，

有时即使有着十二分的小心，也架不住坏人们推陈出新的手法。

拄拐杖的老人其实拐杖就是他的偷拍道具

![](img/328c6dec97732a675b96170f48e8ec8d.png)

还有人在纸袋子上穿个洞来偷拍

![](img/06699e41e5e4ff12da86f0d7b0f6316b.png)

常见的厕所里的原有物品，

**马桶刷、芳香盒、挂钩，挖个洞就能成为实时摄影机。**

![](img/bed151fc7241f9c11ce733dce1657a69.png)

另外还有口香糖&易拉罐等，放在厕所比较恶心，

很少有人会去主动翻看，同样也是盗摄的重灾区。

![](img/1b51c5811fe0a88913c85b2b58ea1c66.png)

谁都不想让自己成为下一个偷拍镜头里的主角，更不想自己的隐私暴露在大众之下；

如果不幸遭遇到偷拍，请勇敢拿起法律武器对抗，而不是忍气吞声，让坏人得逞。

**最后一个压轴的场景非宾馆莫属了。**

这个就不用多说了吧，打开软件的国产区，大浴缸、大圆床...，有网友曝光说：尤其是民宿更猖獗。

国家也一直在打击黑产，也出台了《关于开展摄像头等黑产集中治理的公告》，对互联网平台以及电商平台等五个平台提出了硬性要求。 

![](img/7d69e7bf7bf5a2359e91f88dd6240246.png)

但黑产十分隐蔽，打击起来也非常困难。

所以，我们能做的就是提高防范意识，多了解偷拍手段，这样才能更好的保护自己。

如果我们在户外，可以采用侧身站。

![](img/c5463afdd227b40f0a78def43c31e9ba.png)

此方法适用于一些存在高度差的地方，比如站在电梯、楼梯等场景。

现在还有各种沙雕的打底裤。

![](img/1676f0ba475033f761fc6ac7ed459216.png)

如果我们在厕所，不要怕脏。

就像上面所说，摄像头藏的地方非常隐蔽，比如地上的口香糖、纸巾、马桶塞、易拉罐、衣架等。

如果我们在宾馆，可以玩游戏。

其实一些摄像头都需要联网，而王者里面的网络诊断可以查出该 WiFi 下有几个设备联网。

![](img/586643f16661f56737707908d9067782.png)

![](img/938a67e5cb47a061c04b59c9b726a085.png)

还有这些容易隐藏摄像头的物品：

![](img/76561ab05ce17a79cb9e943a93df5c6f.png)

另外就是宾馆的镜子，检查是不是双面镜，你是不是听过用距离法判断的。

![](img/4ed4967327b41b2ac47893438a0e9429.png)

这里要注意了，这招太老了！

在之前确实是一个判断方法，但是被曝光后不法分子也改进了，他们会再加一层玻璃，一样有距离。

![](img/e2be92bb271f4ccdee937fdd5007297c.png)

所以最靠谱的方式：

**把窗帘和灯关掉，在黑暗中直接用手机手电筒照！**

如果是双面镜，就可以直接看到对面的场景。

![](img/aa449611620f848383dc10e594da7262.png)

如果不想成为女主，就多看看吧。

但令人无奈的是，只要心怀不轨的人依旧存在，那偷拍产业链就不会停止。

最后再啰嗦一句，各位男生在外面的公厕方便时也要注意防范偷拍的设备。

有一类黑产是专门拍男生的，然后上传到一些平台上供特殊性取向的人群观看，这个有机会的话，下次咱们再细说。
