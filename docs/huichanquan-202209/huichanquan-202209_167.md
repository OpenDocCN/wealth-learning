开会也风险。

日前，在一个学学术论坛上，远程参会的北理工方岱宁院士开启了摄像头。

会议中，院士对着学生大谈年轻人如何提高自身业务水平，接着就有某年轻女子亲了其面颊多次，并还直视摄像头，报以主人翁般的微笑。

方岱宁，今年 64 岁，中科院院士，北京理工大学教授、副校长。

随后，有知情人说女主是北理工的一位女博士后。

有网友爆料：这两人可都是已婚状态。有北理工的校友在网上分享了李立明博士论文的致谢部分，在论文的最后几段，李立明特意感谢了一番自己的爱人以及公公、婆婆。

女方在大庭广众之下吻的太投入，男方一时懵逼不知所措，这才让这段忘年恋呈现在大家面前，成为网友无聊生活中一味解闷剂。

也许是孤寂难耐，也许是相互喜悦，也许是才华吸引，反正李立明和方岱宁就这样走到了一起，在北理工上演了一曲院士与博士后忘年之恋。

有朋友看了会议视频，感慨说，得到老教授的首肯，凭借老教授的资源，不说未来能够上天，起码不用爬山了，省却了相当长的奋斗时间。

对于学生而言，能够站在教授的肩膀上当然是很好的，不过途径还需要正确，否则的话倡导的反而会成为另外的非道德的情形，就对其他学生的不公和侵害。

但是不管如何，直播过程中出现这种亲昵的举动，都显露着不太正常的情况。

男教授被亲脸事件的最牛评论：人家只是在传输知识，因为知识是可以通过亲吻传播的。

昨天早上 8 点 33 分，北京理工发表了情况说明，称该校将进行核查。

八卦嘛，再聊一个。

两个月前，头部知名某券商（中*证券）召开一个线上会议，路演推介一个某投资公司的私募产品。

参会的除了客户，还有不少该证券的员工，包括颜值颇高的帅哥及美女员工。

刚开始，会议还是挺正常的，正当大家感到乏味的时候，与会人员瞬间有了精神头！

因为线上会议里竟然开始传出一些异样的声音，啪啪啪，一种只有成年才能懂、大家心照不宣的声音。

![](img/74eb9412c35bd3533f4146a052d47985.png)

啪啪之声，声声入耳。

大家都在脑补某种美妙的画面。

![](img/e1a230b176a4d8ce0d1ea6de14ed15e5.png)

参会人员信息非常透明，很快能锁定会议 C 位出道的男女主角。据了解，是该机构新入职一名美女员工，而对象则是其男同事。

对此有位接近这位女一号的同事表示，本次剧情女一号的颜值非常高，在单位有很多男同事爱慕。更难能可贵的是女一号受教育背景很好，简历很光鲜，可以说是才貌双全。

另外还有人爆料，女一号据说有男朋友，非本次男一号。

路演就是路边演出的意思，很应景，不违和啊。

男女之间的爱，是需要靠性来维持的。

**“其实，睡你和被你睡是差不多的。**

**无非是两具肉体碰撞的力，**

金融圈美女众多，这里面动作片繁多，时不时会来个爆料，人性也是体现得淋漓尽致。

多巴胺无处释放，性是最好的缓释剂。

无非添一点饭后的谈资。

当然，结果没有任何悬念，两人双双离职！

有人这样评论：

1、这两名新人员工对单位说去对外拓展客户了，实际上是在互相拓展对方的 BODY。不诚信，是职业道德的绝对死穴，单位如果以此为理由开除员工，员工很难开展开行之有效的反击。

2、开线上会议时不发言者应该静音，这是一个很简单的职场小细节，两位员工却连这点小事儿都做不好，一屋不扫何以扫天下？实在是令人对其工作能力产生质疑。

3、辩证的看，该事件证明该公司的年轻同事精力充沛，充满热情和活力，想必在工作中也是生龙活虎的状态。但是也给对公司的声誉造成了一定影响。
