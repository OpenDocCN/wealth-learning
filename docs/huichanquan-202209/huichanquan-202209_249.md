# “张家界新娘”大结局：宇宙的尽头是带货

张家界新娘事件简直毁人三观，辣人眼球。

疯狂的新娘，婚礼前夜化完妆，竟然背着新郎和泡友约会，约男人无防护**多次，还玩聚众。并拍摄下视频，说听天由命，万一怀孕了，就说是新郎的。

无良的泡友，为了炫耀自己辉煌的“战绩”，竟然把和新娘的聊天记录发到了群里，在当地引发了轩然大波;

![](img/bf49c759d896bececfb6117d138d66c8.png)

可怜的新郎，在婚礼办完之后才发现，自己花费 20 万彩礼娶到的新娘，私生活竟然如此不堪;

一场婚礼，三个主人公，上演了一场“张家界新娘婚礼前夜穿着婚纱出轨”的闹剧，也引发了全网的围观。

![](img/e78f3d17b9f75ab4ed6938dda90d6f6b.png)

下面，先稍微复盘一下“张家界新娘事件”，再告诉大家目前几位主人公的现状。

话说，前几天网上突然爆出了一段视频，视频中有聊天截图，也有一对新人的合照，乍一看很幸福唯美。

可是，把视频中的微信截图放大之后，网友们才发现，里面的内容竟然如此劲爆!

起初，是一位即将举办婚礼的张家界新娘，在微信上和自己的泡友晒婚纱照。泡友也很知情趣，马上表示“新娘很漂亮”，新娘也有些洋洋得意。

![](img/b628d639386f6dfa2d9e81737c356b0b.png)

可是画风一转，泡友的言语间就有些歪楼了。

泡友表示：“你这么美，我明天一定好好‘伺候你’，而且不想采取措施。”新娘也毫不避讳地说“我同意”。

短短几句话说明，他们的关系维持了很长一段时间了，而且对彼此很满意，所以才会一而再再而三地继续约会，言语才会如此大胆。

![](img/86e17251a4ccc849707675db5ace31d0.png)

随后，两人又聊起了婚礼前夜的一些准备事宜，泡友又突然提议“很想看你穿婚纱的样子，想不采取任何措施，看看你会不会怀孕”。

张家界新娘竟然一点都不害怕，也不觉得哪里不对，更没有把新郎放在心上，竟然直接表示“好的，我不怕，听天由命”。

话说，见过无耻的，还真没见过两个这么无耻的，对感情和婚姻完全没有任何责任心，肆意伤害着新郎，简直没有任何下限。

![](img/76b2ae8327b76fabb97b478821f26eb5.png)

毫不夸张地说，这俩就是“当代的潘金莲和西门庆”，一点羞耻心都没有了。

随后，泡友继续调戏新娘，又提议说“要不明天我再叫一个哥们，三个人一起运动一下，更刺激”。(这里说的是明天，还不是婚礼举行前夜)

新娘也没有拒绝泡友的提议，欣然同意了。这说明，这位张家界新娘已经不是第一次和泡友一起“玩三个人的运动”了。

更可气的是，两人再次提及“婚礼前夜的约会”时，都显得非常兴奋，而且一点顾忌和羞耻心都没有，简直无耻之极。

![](img/d5dc212e48fb005c36c89ea89c5ef84b.png)

几天后，张家界新娘举行了盛大的婚礼，就在婚礼举行之前，这个男泡友欣然赴约，和新娘来了一场“穿着婚纱的运动”。搜索关注“瞳瞳客栈”，一个让你欲罢不能的公众号。

在去赴这场约会之前，这位叫“小白龙”的泡友为了炫耀自己的战绩，随后把和新娘的聊天记录发到了当地群里，然后被疯狂转发，被新郎的朋友看到了。

![](img/2c147f09296fecfb189037a44604d20b.png)

随后，新郎的朋友又告知了新郎，刚刚举行完婚礼，正在享受蜜月温情的新郎才发现：自己花 20 万彩礼娶回家的姑娘，竟然这么不堪。

虽然新郎也试图找过“小白龙”放狠话，但“小白龙”却一点也不害怕，表示“不尽管来，我在家等着你”，一点害怕和羞耻心都没有。

![](img/06e62d66f5492f6ffe89ebbbdac19366.png)

“张家界新娘事件”在网上曝光之后，迅速引发了网友们的围观，聊天记录中的新娘婚纱照中，也曝光了新娘和新郎的面貌，他们也迅速在当地“红”了。

而这件事带给新郎和新娘的打击，可谓是全方位的。

![](img/b0d8c7bc44becf40bd77b107a34ea76e.png)

**一是新郎的内心备受打击。**

新郎可以说是完全疯狂了，还想找“小白龙”打架，但最终不了了之。

然而，这种无处宣泄的气愤却让人难受，于是新郎以“侵犯名誉权”把私下发布自己婚纱照的“小白龙”起诉了。

![](img/8e8e8cff868aa1b017c555eb5585f701.png)

面对新娘的背叛，新郎也大受刺激，目前已经提起了离婚诉求，后续过了离婚冷静期之后，两人将彻底分道扬镳。

虽然付给新娘一方的钱财可以追回，但这位“张家界新娘”对新郎造成的精神打击，以及对他的羞辱，将是他一生抹不去的伤痕。

![](img/e151b4a259885733213f9cf1ce0a9f73.png)

**二是对双方老人的打击也不小。**

男方的父母觉得非常丢脸，没想到儿子千挑万选的女孩，他们将来日夜相处的儿媳妇，竟然是这样一位不堪的姑娘，一家人脸面尽失。

新郎在当地是一位旅行社的经理，日常认识的人很多，他的婚礼也刚刚举行完，他的媳妇就出了这样的事儿，他也成了大家的笑柄，实在可怜。

![](img/cebe42345d71b8b7d7bcfbfa44594d35.png)

这件事除了对男方父母的打击很大，对新娘父母的打击也不小。

新娘的所作所为被网友们认为是“家教不严”，纷纷质问“这到底是谁家的姑娘，父母是怎么教育的，怎么会有这样的三观”?

![](img/99579034b825aeba973b76e3354f7b51.png)

数以万计的网友都表示“这个张家界新娘的父母是怎么教育孩子的，肯定家庭有问题，父母该出面回答一下，自己怎么能教育出这样的女儿”?

虽然新娘的父母没有露面，但估计也知道了女儿的所作所为，在伤心气愤之余，也会时刻关注网上的消息，那些谩骂之声估计他们也看到了，不知是否会反思他们的教育方式?

![](img/3a2d48cd20630e8860881eb355e7a6d1.png)

**三是对新娘的全方位曝光和打击。**

因为有婚纱照为证，新娘的容貌也被曝光，随后有人顺藤摸瓜，竟然发出了新娘结婚当天的婚礼视频，看起来很热闹。

然而，在这场热闹之前的夜里，新娘穿着婚纱和男友“做运动”的事，却在亲戚朋友以及网友们之间传开了，她的声誉也直接毁了。

![](img/e466aec0564cb1a9c92854749f900491.png)

目前，新娘躲在家里无脸见人，父母对她也很失望，新郎则追要着彩礼，还一直追着她去办离婚手续，她终于为自己的放肆付出了代价。

预料不错的话，未来的几年，这位“张家界新娘”若想在当地找个好老公，几乎是不可能了。

也可以说，这位新娘已经亲手毁了自己的未来，以及父母和全家人的声誉。

![](img/7c523f94d98e9efdc5579fc25d776e38.png)

新郎新娘和其父母都遭到了这件事的反噬，也尝到了这件事带来的恶果，可是那个叫“小白龙”的泡友却安然无恙。

除了曝光了一个网名，关于这位“小白龙”的身份、名字和工作单位，坊间一无所知，真是让人气愤。

在这件事中，除了新娘的做法让人无语之外，这位“小白龙”的引导和无底线做法，以及自己的炫耀才是造成这件事直接的主因，但人家偏偏逃过了一切惩罚。

![](img/b1a976c11e63311138eb034e5ff19b0f.png)

不过，大道有轮回，苍天饶过谁?

总有一天，这种无底线、无下限、无责任感、无良的臭男人，会受到惩罚，遭到唾弃，为自己的所作所为买单。

![](img/fa5aa06cdc640d89a5cf5ec518b1b14c.png)

就在大家同情张家界新娘事件中的新郎时，他已经走出来卖货了！

![](img/7ad5ed3d28674373907b6a903c48cc58.png)

“我都还没有走出来，你已经出来卖货了！”能利用这波流量，内心也是非常强大的了。
