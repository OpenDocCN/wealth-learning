大家好，今天我又来得罪人了。先来问大家一个问题吧。**你知道，为什么你总能收到骚短信息吗？**
明明你没和任何人泄露过自己的信息，然而总有垃圾短信“精准轰炸”。

*   有时候你刚从商场出来，就会收到大促短信提醒。
*   有时候你路过售楼中心，立刻就会收到“10 万首付轻松拥有”。

![](img/9973f32d9e282f5269fb278b03a97940.jpg)其实，今年的 3·15 晚会上，央视就给我们解答了这个问题。都是一个叫“WiFi 探针盒子”的小东西搞的鬼。
只要你的手机 WiFi 处于开启状态，路过这个探针盒子，盒子就会自动获取你手机的 Mac 地址（也就是网卡物理地址）。
![](img/511d6d0477b3a4fdd73b74fadf476609.jpg)安装盒子的人获得你手机的 Mac 地址后，会去购买其他公司的用户数据库。
（所以说，不要乱给软件权限）![](img/0a4ec5c12b98ba81f50021ce6178cb8f.jpg)数据库中一般会包含用户的：姓名、地址、IMEI 码、手机型号、Mac 地址、手机号、搜索记录、消费记录。安装盒子的人，用自己获得的 Mac 地址，匹配买来的用户数据。这样，他就可以精准知道你所在的区域，平时有什么消费习惯。![](img/e712ca7f4b26f4bda1420df86ab8a468.jpg)然后按照机器计算出的画像，对你进行精准营销。
![](img/f5d1b50b83eda580a72e47bbc8a9dd28.jpg)**用了这盒子的商家卖得很好，你不用，你就落后。**大家都用，也就进入了恶性循环状态。看得出，在这条灰色产业链中，**除了卖“WiFi 探针盒子”的公司，里面还掺杂了一些卖用户的大公司。**（有公司层面的卖数据，也有私下贿赂员工的卖数据）但是哈，启人要说但是了。央视这个曝光其实很给面子，说的非常委婉。曝光探针盒子，央视只从 WiFi 方面入手，说如果你开了 WiFi 会怎样怎样。但在实际生活中，搞灰产的会这么蠢吗？**不开 WiFi 就探测不到了吗？**太小看他们了。
事实上，在今年 3·15 晚会，国家电视台对 WiFi 探针盒子报道后，WiFi 探针盒子不但没有销声匿迹，而且功能越发强大。**甚至，打起了广告。****堂而皇之走上了台面。**启人尝试性的在淘宝搜索了一下“WiFi 探针”。
瞧，第一个就是广告。
![](img/71ae9bc9f47a47e9601aed0b9d63e4fa.jpg)话说，工信部在去年就发布公告，明确要求，全面清理此类设备相关营销推广信息，让相关设备“发不出、收不到、用不了”。启人怎么一搜就搜到了呢？而且还是个推广广告。![](img/c74e5c072a1831a156721cc2c7cd4718.jpg)来，咱们点进去瞧瞧卖家到底卖的是什么。
进去一瞧，原来卖家卖的不光是“探针盒子”。还有... 4G 流量广告机？
![](img/2592e309b92344ae9b4a1ebe4dd71942.jpg)什么意思，就是**不用连接 WiFi ，也能在 4G 流量状态下给用户推广告？**来到宝贝详情页面，老板说的非常隐晦：看不懂不要下单，看懂 5000 非常值。
![](img/254d43557c5ab3b02a278a8251636141.jpg)**按照启人的理解，这难道是国家明令禁止的“伪基站”？**
如果说“WiFi 探针”还能算灰产，那“伪基站”可以说，就是妥妥的黑产。启人赶紧就这件事和老板核实，不过，老板读取了我的聊天记录，却不做任何回复。只是自动向我抛出了一个微信，引导我去其他平台和他联系。![](img/1a0f3fb4c67f09aef8bb4e0dd028e3fa.jpg)于是我**又在淘宝上找了一个 WiFi 探针的****广告**。
这个探针就非常小了，放在优盘里。
![](img/9b5ffd30286b36ed880e27711e89a1d3.jpg)启人又去和老板骚了一圈，老板依旧没有任何回复，只是说：请添加微信。![](img/570f5a90427e717b9ba5b5575c4cdb6f.jpg)

话说，商家诱导顾客去其他平台，这事符合淘宝规定吗？

**如果不理亏，去其他平台想干嘛呢？**

事实上，这样的例子不止这两个。比如这个：2019 夏季百搭...自动拨号。![](img/e91a4f4875c64ed8ef1d4e866683c0c9.jpg)作为一个小微互联网公司出身的人，你知道启人第一反应是什么：

*   这广告应该归到服装类目下，还是通信类目下？

*   **大公司没有广告审核团队吗？这****到底算啥？**

*   我搜探针盒子，它出自动拨号。关键词优化怎么做的？

当然了，启人也不是只把目光放在了淘宝上。
**比起淘宝上有这些东西的广告，更邪门儿的是做这东西的人，还能成立合法公司。**比如，我们在搜索引擎上搜索“WiFi 探针”都是负面消息。
![](img/58d5c73b6d1b136eb8e9db9601700a7d.jpg)那就改个名字，换成“获客盒子”。一下就搜到了，**而且还有认证**。![](img/e32fee8a92869d3625f1f69be0e1322c.jpg)进去一看：他家能做霸屏闪信，就是这种强制弹出，不看不行，看后即焚的营销短信。![](img/e2d26e6c9ad05bbd9edc63c1b2ce050b.jpg)能做网页弹窗。![](img/c5cf9a7baf4e52b8421843cc2eb509a1.jpg)**甚至自称能进行运营商直接投放。**
![](img/671296b1d29251cd3fe63e52817b1751.jpg)还卖各种“广告投放”设备。
![](img/4995c28e09741b43fb369533da68b8e2.jpg)启人在企查查上，找了一下这家公司的详情。
发现，他们公司经营范围里，虽然有“电子产品”和“数据处理”两项。
但是没有“通信设备”和“通信技术”两项。![](img/7967c44ed65990c908364219a6f6fcd4.jpg)不过，有趣的是，他们申请了 5 个域名。还成功申请一个商标。![](img/1a14bfb63606499aea849245839ddd4c.jpg)启人又查了一下，原来，给他办商标的，是一家嘉兴的知识产权代理公司。![](img/80b4fb152a880aeabbe81eb66b285fa9.jpg)这家知产代理公司没有官网。
能找到的资料仅仅是这样的。![](img/dd488341a178f52d621be3f6b1e84a6e.jpg)再一查，**这家知识产权公司的商标，还是南京一家知识产权公司代理的。**![](img/531e811b05eeeb96c1e7548e141a0fd3.jpg)但不管怎么说，反正经过这么一层层的操作。
这家公司成立了。至少...能做认证了，能投广告了。
老实的讲，这世界有好人，也有坏人。有白昼，也有黑夜。
有人搞灰产、搞黑产，这事一点都不奇怪。甚至可以预测，黑产在未来一定会越来越多，越来越高端。
**但我要问：****大公司在这件事中到底扮演了什么角色。****你们有没有尽到监管和审查的义务？**我们明白互联网公司大部分都是广告公司，广告主才是最大的客户。
但不能什么饭都恰吧？技术的攻防永无止境，我们能靠的是什么？让用户一次又一次的提高意识？不对，是良心！
