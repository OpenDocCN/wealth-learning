如果很轻松，又可以很快赚钱，那该多好啊。

最近听说有种轻轻松松就能赚钱的方式，一点本金，只需要跟着计划投注就可以，每天稳定的收益？

![](img/7eaf54f9b146134724b6d364512b383a.jpg)

研究了一下，差点吓一跳，这哪是赚钱，是坑钱啊。一旦走进骗局，多少储蓄都会被清空。

来看看怎么回事吧：

**通过宣传，引起你的注意****最开始的套路都是相似的，要不就是通过短信或者海报图片的宣传，引起你的兴趣：** 

**![](img/5b7a202475477ac9aa7b36ec48f0b142.jpg)**

**要不就是假装电商平台客服或兼职招聘，以其它理由先吸引你关注公众号：**

**![](img/ad620bfac109a039607ca9516423abf3.jpg)**

**![](img/a3e1884de764e7b99043d4510797e5ef.jpg)**

****通过聊天，降低你的防备****

**接下来，可能会引导你去添加某个名字高大上的公众号。这个公众号自己不发内容，但！它会通过后台跟你聊天。**

**![](img/4d476685855c3816cb27c766c5c9dfd3.jpg)**

**由于平台持续对这类公众号进行打击，诈骗分子会要求转移到其他聊天软件，引导你加一个私人“向导”。一通复杂的玩法介绍之后，套路就开始了……**

**![](img/e5d89a7d317fc3ca4b4cd88d3d75efa9.jpg)**

****时机成熟，收网等你上当****

****简单来说，“向导”会让你登录一个类似网络赌博的平台，在账户里充值，并且按照他的指令下注，即可获得收益。****

****![](img/3d20e56c3cec5e308f1be30dbf9dfbed.jpg)****

******到这里，骗局就出现破绽了。如果听向导的话就都能赢，那这些钱从哪里来的？******

****其实，媒体早就揭露过这些团伙的手段了。事实上，这些所谓的“赌博”，结果都是可以操控的。**诈骗团伙想你赢就赢，想你输就输。**最开始是会真的有收益，但骗子会用还没达到提现额度、系统故障等等原因阻止你提现，于是为了赚更多，你的赌注会越下越多……****

****后面的故事你一定能猜到了，当你赌本下得足够多的的时候，赌局就会在适当的时机“输”。无论你是不服气再加注，还是就此离场，诈骗团伙都稳赚不赔。****

******微信公众平台持续对此类违规公众号进行打击**，但骗子变化多端，我们也要增强自身辨识能力才能防止被骗。****

******以下提示，请大家牢记******

******1、凡具有低投入、零风险、高收益、操作简单等特点的“兼职”或“赌局”，一定要提高警惕，天上没有简单的赚钱方法。******

******2、购物平台的客服不会通过私人短信或电话形式联系，请官方渠道核实信息。******

******3、不要随便打开安全性未知的第三方链接，不要轻易泄露银行信息及转账。******

******4、十赌九骗，网络赌博属于违法行为，请勿参与。******

******最后，**如果大家发现此类公众号，欢迎通过右上角“…”-“投诉”-“欺诈”进行投诉，帮助我们一同建立安全的微信环境。********
