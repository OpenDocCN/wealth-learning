# 『为什么做让自己提

『为什么做让自己提升的事你会感到难受？』

晚上好 与星友们分享丹儿的第 74 篇原创 7.7

————————————————————————

“你的梦想是什么？”这句话中国好声音的评委经常提到。星 友们的梦想是什么呢？所谓梦想，应该都是很难达到的吧！ 经常听到两种关于梦想的看法：

一种是“梦想，也就只能做梦想想吧。” 另一种是“有梦想，才有动力，我相信自己能够完成梦想。并

且我享受完成梦想的过程。”

大家觉得哪一种看法更能让梦想成真呢？我个人认为是第二 种，我为什么觉得第一种看法的人梦想不能成真呢？

【否定自己】

“梦想，也就是做梦的时候想想吧。”

为什么只能做梦的时候想想呢？

①有这一类看法的人可能是曾经努力过，但是没有结果。

★为什么因为努力了没有结果，就不继续努力了，从此自暴 自弃一蹶不振了呢？有没有思考过是不是我们的关注点产生 的问题呢？

大家有没有过这样的经历？当我们更在意结果，而不是享受 过程时，当结果和我们预想的有偏差时，我们会特别的失望

，从而否定自己，觉得自己是不是就是不行？没有能力？

举例

比如上学时候考试 觉得自己认真复习了，记了很多笔记，努力了，但是成绩一

下来，还是不理想，就很失落，觉得努力了也就这样，没什

么用，我就是这样的人，学习就是学不好。

如果我们可以深刻的思考一下，我努力的方式对不对，能不 能尝试一下别的努力方式会不会有别的结果呢？

当我们把注意力放在测试我这次用的学习方法能不能提高我 们的成绩，而不是我努力我就必须得有好成绩不然我就不学

了的时候，即使你的成绩还是不好，你也不会很失落。

你只会觉得，嗯，上次的学习方法，不行，我总结一下在研 究研究别的学习方法。当你自己努力通过不断的摸索，找到 适合自己的学习方法把成绩搞上去时，那你就厉害了！这时 候你会很有成就感，你觉得自己是有价值的，觉得自己有价 值会给你带来什么呢？

自控力的提升，也就是自律。自律的好处大家都知道，彭于 晏多好的例子，大家说他的身材像古希腊雕塑一样完美。

②还有可能是否定自己，还没有去尝试就觉得自己不行，自 己的梦想凭自己的能力根本完不成！

前者起码还看重结果努力过，后者直接连结果的不看重了， 尝试都不想尝试，你们身边有这样的人吗？遇到困难就向后 退，能躲就躲，或者直接推给别人，他们是自私，爱偷懒， 比偷奸耍滑吗？

也不全是，我想他们不愿意面对的原因就是他们认为这件事 他是做不好的，他害怕被别人知道自己的无能，所以宁愿扮 演一个自私自利的人，让别人觉得这事儿我能办好，我就是 不去做，我就是懒。

这样的人还有一个共同点就是身边总有能为他们的行为买单 的人，无论做什么都不需要他去面对，去解决，别人都会给

他处理好。这个人可能是父母，也可能是爱人。看似是为了 他好，其实他越不去思考，处理问题的能力越差。

举例

小侄子学习不好让父母苦恼

今天去表姐家玩，表姐就一直向我表达自己的忧虑，小侄子 期末考试成绩下滑很严重，还越来越叛逆，但是表姐觉得自 己已经是一位很称职的妈妈了。

为了孩子换了份清闲的工作，孩子写作业时刻陪着学，重点 内容都帮着画下来，英语单词也和他一起背。她不明白自己 的付出，孩子并没有看到成果，却还对她像仇人一样。

小侄子叛逆表现在哪些方面呢？ 今天一天和他妈妈已经动不动就大喊大叫不止 5 次了，对待其

他人情绪也比较暴躁，我能感觉到，他已经接近崩溃的边缘

了，才一个上二年级的小孩，压力就这么大，随时想爆发。

从心理学的角度分析了一下小侄子叛逆的原因： 表姐太注重他的成绩了，她觉得她为孩子放弃了那么多，这

么用心的陪伴他，唯一能让她觉得她的决定是正确的就是孩

子的成绩。孩子成绩好，她觉得一切都值得，成绩不好她就

有很强的失落感。

所以在家里她总和孩子说只要你成绩好，什么都不用你做， 甚至把考试的重点，怎样背英语单词的方法，这些学习上本 应该孩子自己做的事，她都帮忙做了。我也问过她，小侄子 有别的优点吗？她说有，但是不重要，重要的是他成绩不好 啊！

陪伴孩子没有错，但是本应该让孩子自己探索，自己解决问 题的事情，不要都替孩子解决了，那样他会觉得自己没有价 值。我在这个家里就是个学习机器。别的什么用也没有。成 绩一下话表姐的表情动作都会伤害到孩子，他不仅不会想把 成绩搞上去，还会更加的厌恶学习。

还有一点就是表姐总说，小侄子都说他聪明，就是不用功学 习，小侄子就更害怕，本来其他事情就没有价值了，如果用 功学了成绩还不好岂不是聪敏都不占了，所以只有不去用 功，才能让大家一直以为我是聪明的。

★表姐希望孩子好，这没有错，只是方法用错了。她替孩子 做的越多，越让孩子失去了探索的能力！如果一直这样发展 下去，小侄子长大了就是觉得梦想只是想想而已的那类人， 他不想去探索，因为他从小就是失去了探索，解决问题的能 力。

这个世界上，没有坏家长，很少有人希望自己的孩子不好， 但是却有很多的笨家长，其实只要你稍微学习一点教育方

法，就可以改变你和孩子的关系。

孩子一出现了问题，就想改变孩子，但是您是否深层的想 过，孩子为什么会变成这样？和您的教育方法没关系吗？孩 子来到这个世界上就是一张白纸，最后会呈现什么样的色彩 完全取决于拿笔的人，也就是你自己。

＊＊＊＊＊＊＊＊ end ＊＊＊＊＊＊＊

▼为什么做那些提升自己，帮助你完成梦想的事你会觉得难 受？就是因为你太在乎结果，你怕结果是失败自己接受不 了。还有就是你根本不想提升自己，做不喜欢的事当然会难 受。

解决的办法就是，我们要知道“学习是我必须完成的课题”， 如果是必须去做的事情快乐的做也是做，效率还更高。痛苦 的做也是做，还越做越烦。为什么不寻找学习过程中的乐趣 呢？

不要把结果看的太重，注意力放在学习过程中你会发现自己 会快乐很多❤
