# 《方向不对，努力白

《方向不对，努力白费！在前行的路上，很多人踏出的第一 步就错了！》

年后上班，我堂哥随我一起来了，他说他需要一个安静的氛 围，去找自己感兴趣的事和未来的路，而呆在家里，永远不 会有这样的环境。

堂哥是个厨师，现在要转行，当然他也有转行的底气，因为 最近他们家拆迁了。堂哥不想被迫谋生，想做自己感兴趣的 事。于是买了近万元的各类付费课程去学习，去找方向，但 家里有两小孩，还有各种杂七杂八的事，这让他始终静不下 心来，于是就跟我来到长沙。

我也觉得在老家那种环境下学习，是件挺考验自律能力的 事。因为过年回到老家后，我自己就松懈懒散了不少，晚上 看手机看到一两点，白天睡到十点多才起床，整天感觉迷迷 糊糊的。而一到长沙，我就立马充满工作和学习的动力，脑 子也清醒不少，难怪有人说人是环境的产物，此言不虚啊！

昨天晚上我和堂哥都在办公室，我看到他学的课程，居然是 一个关于视频剪辑的教程。里面讲解着这个软件里每个工具 是怎样操作，怎样应用的。

我问他，你想好了学这个要去干嘛了吗？他摇了摇头！这让 我有点诧异，这么学习，不是无本之木，无水之源吗？连目 标都没有的学习，能坚持多久？又能有多少收获？不过随即 我又释怀了，因为以前的自己也是这样。

我记得刚开始做设计时，觉得三维软件 MAYA 非常酷，那时 还没有中文版软件和教程，我硬是一个英文一个英文地去翻 译着学习，花了很多时间，结果在后来的工作上，完全没有 用到，工作只用到行业常用的 3DMAX 软件就完全足够了。

还有一段时间，我去研究发光原理，研究日光灯里荧光粉， 为什么受到灯丝发出的紫外线作用就能发出可见光……现在 想想，研究这些，对于我们照明设计师有多大的帮助呢？会 使用冰箱就好了，没必要去了解冰箱的工作原理啊，我又不 是研发人员！

虽然有人说“功不唐捐”，没有一点努力是白费的，但我觉 得，我学 MAYA、去研究发光原理，就是挺浪费的一件事， 至少从性价比上来说，是非常不划算的。

所以我更赞成“方向不对，努力白费”这句话。我们所有的努 力，应该建立在正确方向的基础上，否则必然会要做很多无 用功。而要有正确的方向，首先就要有一个明确的目标，如 果没有目标，那就先不要发力。

就像我堂哥的学习，他有那么多课程，不能随便挑一个就发 力去深入学习，而是应该先把所有课程做一个浏览，首先知 道每个课程大致讲的是什么。这么做的目的是拓展自己的视 野，大致知道遇到什么问题，学哪个课程能解决就好。

只有找到自己的目标，再去学习实现这个目标所需的技术， 锻炼所需的能力，这才是高效的学习方法。

比如你想做照明设计师，那就得先知道一个合格的设计师， 在目前的市场环境下需要懂得哪些技能。然后你才去学如何 用 PS 做夜景效果图，如何用 AE 做亮化动画，去了解常见的照 明方式和各种灯具。

而不是花大把时间把 PS 软件和 AE 的几百个命令全部学会，以 为自己牛得不行。结果发现做一个照明设计师，只要用 PS 和 AE 里的十几个甚至几个命令就完全能够胜任。

其实生活中有很多人，在努力前行的路上踏出的第一步就错 了，犯这种错也许是受自己认知和见识的影响，因为他不知 道有一些方法技巧可以让自己少走弯路，这一点我觉得还情 有可原。

但更可悲的是，有些人受惰性思维的影响，宁愿勤奋到死， 一直沉浸在“我在努力工作，努力学习”的表象中，也不愿跳 出来，花点时间思考一下自己的方向正不正确。
