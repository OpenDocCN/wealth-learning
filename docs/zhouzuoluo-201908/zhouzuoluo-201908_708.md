# 《警惕成为这样的人

《警惕成为这样的人：不仅穷，还有“才华”！》

有些人认为自己很有才华但是郁郁不得志，处于贫穷的状 态。然后，当别人做出了成绩，就认为这没什么了不起的， 换做是他，他也能做到这种程度，甚至他比别人都能做的更 好！然后，自己心安理得，仿佛世界上所有做出成绩的人， 只要给他同等条件，他都能做的一样甚至更好似的。这种 人，可以给他一个称号：有才华的穷人。

这种人，通过批判或者否定别人的成绩，认为别人的成绩有 猫腻，肯定不是自己奋斗出来的，以此作为心理安慰，于 是，这个世界，所有的成功者对于他而言，都是运气好。他 自己很能干，但就是运气不好。他都没有挣到钱，那么别人 挣到钱也是假的！

你发现了这种人心理的问题吗？是否感觉这样的人充满了负 能量，就像整个世界都欠他似的？然后他还是认为自己比别 人牛逼，无法正视自己，也无法正视他人。

那么为什么很多人有这种倾向呢？其实答案和心理学有关 系：人们对于自己、他人以及环境的归因不同：

如果是自己成功了，就认为是自己牛逼，忽略了环境的因 素； 如果是自己失败了，就认为是因为运气不好，环境太差，而 不是从自己身上找原因。 如果是他人成功了，就觉得这个人是运气好，是环境造就 的，是赶上了风口，而忽略别人本来的品质；

如果是别人失败了，就觉得是别人的问题，自己不会管理团 队，自己一意孤行等等。

对他人的成功和自己的失败归因于环境，对他人的失败和自 己的成功归因于人。这是一种与生俱来的归因倾向，是一种 自我的心理防护机制。这样的防护机制，在某种程度上可以 减轻我们受到的打击，但是它容易扭曲我们对自己和他人的 看法，导致我们存在偏见。

心理学家强调，人们在对问题的归因时，要综合内归因和外 归因，即人和环境，这样才能尽可能的减少归因偏差，也能 从失败中吸取教训，或者成功中获得激励。而带有归因偏见 时，这种人总是戴着有色眼镜看待自己和他人，这样的人， 必然会活得很累。

以后如果我们再有以上四种情况，是否可以深入的考虑个人 和环境的关系呢？比如一个人成功或者失败，究竟是什么原 因？他的特质和他所处的环境对他成功或失败的影响率各是 多少？而你自己成功和失败时，你的特质和你所处的环境对 你的影响率，又是几多呢？

如果你能够真正的正视自己和他人的成功或失败，你最起码 不会陷入到“有才华的穷人”这个误区。
