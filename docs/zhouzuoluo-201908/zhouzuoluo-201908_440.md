# 《一个人走得快，一

《一个人走得快，一群人走得远》

好有成就感啊，只要粥老师不写作业，我就可以拿着小皮鞭 去催他，哈哈！今天表现不错，补了几条作业。

他说不想做训练营，因为懒。其实我也不想做，因为做不做 我都能领工资的嘛，有时间多写写文章，多拉拉广告，能得 到打赏和奖金。所以，这事还真不是金钱驱动。

这些天看到训练营里的同学打卡写作业，而且写得非常认 真，一个劲地邀请你去点评作业，心里就觉得很有劲，好像 看到了当初的自己。

纯小白转行，半路出家，过程中不仅仅是艰辛，还有太多的 迷茫和孤独。学习毫无头绪的时候，内心真的特别渴望能有 人来指点迷津，过来拉我一把。

辛苦做训练营，就是本着这点初心。希望那些真正像学习的 人，可以有一个机会和渠道，深度接触专业的老师，得到指 导；有一个好的氛围和环境，去结识同频的朋友，让成长的 路上有个伴。

我们的作业挺花时间的，基础弱一点的同学每天甚至花五六 个小时学习，一路走来居然没有掉队，真的不容易。相比其 他很多训练营，作业简单得多，助教点评少，同学们做作业 打卡的热情度反而没那么高。

这是为什么呢？

有几个同学相似的反馈，给了我答案。她们说又想训练营快 点结束，又特别害怕结束。因为自己一个人的时候没发自 律，但现在居然坚持学习十几天了。

其实我很理解她们的心情，人都是有惰性的，但是在一个学 习热情度很高的氛围里，你看到其他同学都认真写作业了， 每天给你点赞和鼓励，班长每天催你写作业打卡，助教认真 给你点评。如果再偷懒，真的脸红。

这让我想起一句话：一个人走得快，一群人走得远。一个人 无法战胜的惰性，在一个良好的集体环境中酒可以克服。

（这不是打广告，第二期马上要满了）

我想说的是，成长的路上，尽可能融入一些好的圈子，偶尔 你想懒那么一下的时候，去寻找集体的力量帮助自己去成 长。
