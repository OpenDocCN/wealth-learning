# Day132《写作

《写作，使不被看见的东西被看见》

昨天，马云在美国西点军校一年一度的麦克唐纳品格领导力 论坛上，分享了自己关于领导力的理解。

马云说：“大部分人因为看见，所以相信；领导力是因为相 信，所以看见。领导力意味着看到别人看不到的事情。”

今天咱不谈领导力，谈谈写作。 因为我最近在持续写作和思考，马云的这句话也适用于写作

这件事。写作如同一束光，照到不被看见的事物上。

记得龙应台在一次演讲中用一句德文描述写作的时候，提 到“写作就是使不被看见的东西被看见”。

马云说：“绝大部分人都会去埋怨，这些人应该少去打交道， 而经常检查自己的人是靠谱的。你见各种各样的人，听各种 各样的事，经历各种各样的磨难，你的格局自然会高。”

在持续写作之前，我对人对事的关注和投入都是不足的，而 写作实现了对“曾经不被看见的东西被看见”。我开始试着去 写一些人和一些事，在这个过程中，我对人对事的认识和理 解获得了不小的提升。

同时，写作的过程也是自省的过程，我也发现了自己想法和 做法中不靠谱的地方。一旦漠视就是盲点，而写作照亮了这

些盲区。

可以说，写作让我生活得更认真了，也照见了一个更大的世 界。

大部分人因为看见，所以相信；有的人因为相信，所以看 见。而写作意味着使不被看见的东西被看见。随着每天的写 作，或多或少的，我越来越相信写作的力量。

在我的工作中，衡量一个人的创意是不是够厉害，多数时候 不是证明自己有多聪明，而是谁了解世界多一点，对人的认 识更深一点。

因为写作，在你的生活中，在你每天的日常里，你可以比别 人更专注认真地生活，你要大量对人对事有兴趣，并积累知 识和经验。

希望你也能拿起笔来，让写作成为一束光，去照见更大更有 趣的世界。
