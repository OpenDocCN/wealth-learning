# 《关于社交：当你变

《关于社交：当你变强的时候，你敬的酒别人不会不喝》

看了粥老师今天推送的文章，忽然想起了自己一段没出息的 糗事。

我第一份工作在 500 强企业，吃年夜饭的时候，部门领导和集 团领导坐在一桌。我的助理给领导敬酒了，其他的同事也都 给项目总敬酒，我平时就不爱社交，但觉得自己也应该去敬 一杯酒，所以我就端着酒杯小心翼翼的走了过去。

项目总就没有起身，也没有端酒杯，淡淡地说：“你跟我隔着 两个级别，你敬什么酒。”

我陪着笑脸说：“先干为敬，我干了，您随意。” 他什么也没说，也没拿杯子，当时所有人都看着我，感觉空

气都凝结了，我默默地喝完坐了回去，后来我找理由先走

了，在回家的路上越想越难过，就坐在路边哭，就是人来人

往的马路边上。

我觉得特别没面子，很生气很恼火很丧气，可工作还得继 续。

不久后的一天，有人掉进了我们影院的地下停车场的深坑， 八根肋骨断裂，我发现以后立马通知了顶头上司，但他以项 目未正式移交推脱不理（事实上，不管移交不移交，都是我 们集团的责任）。

我一边安抚当事人家属，一边派出秩序部的同事立刻封锁事 故现场，通知其他部门通过社交手段，拦截媒体的报道。

等事情处理完，项目总才得到消息，他很恼火地教训我：“为 什么不通知领导？”

我拿出了通话记录和微信截图，证明已经用多种方式通知了 两位上级领导。然后他说：“以后我直接汇报跟他回到工作。 ”（他也曾经说过，我跟他隔着两个级别，没资格跟他汇报工 作）

后来我得知，如果事故的等级升级（例如当事人死亡或事件 被媒体曝光），项目总就会被撤职。

几个月之后，我从集团离职了，有天他突然打电话给我，我 顺口就说：“***总好啊。”

他说：“叫什么总啊，叫哥就行了！” 他说了大概十几分钟，问我离职以后，是否需要他帮忙介绍

工作，可以把他当成朋友之类的话。

我到现在也不认为，我们就是朋友了。只不过我已然明白， 当自己不强的时候，即使努力地去放低自己去迎合别人，依 然得不到尊重，只有当你变强，变得有价值，别人才会把你 放在眼里，只有你变强了，你敬的酒，别人才不会不喝。 2019-05-10(38 赞)
