# 逃回了北上广深，然

逃回了北上广深，然后呢？

每次春节过后，“逃回北上广深”就是一个热点，毕竟很多人 在家过得并不“舒服”。不舒服的原因嘛，多种多样。比如被 催婚、被问工作待遇、不适应人情等。

而要不要继续回到北上广深等大城市工作，可能很多人也在 纠结，于是也有各种感叹，“留不下的大城市、回不去的故 乡”、“一线城市留不下肉体、三四线城市容不下灵魂”。

为什么要到大城市？ 显而易见，比如更好的工作机会、更高的薪资待遇、更宽容

的文化、更自由的环境。这些都是吸引我们的地方。

大城市相比小地方而言，确实“冷漠”，但也正是这份冷漠， 因为没有人关心你，也不会有人阻碍你。这样，才能让我们 摆脱很多束缚，成为自己。

那要不要到大城市？ 这个问题，我想了很久，却得到了一个再普通不过的答案：

看个人意愿。

这个世界上，没有十全十美的生活，你想要一些东西，那肯 定要放弃另外一些。

去大城市奋斗也好，留在小地方生活也罢，终归是要看你个

人意愿的。当然，留在大城市意味着辛苦，留在小地方则意 味着发展机会十分有限。

千万别想着都要，什么都想要的人，可能最后什么都得不 到。

不过，越是小地方的人，越应该到大城市去学习、工作。这 是真心的建议！

但是，不要为了来，而来 这个，其实也正是我反思的地方。

当大家说起北漂、深漂的时候，就想起来辛酸的生活和努力 的工作。辛酸的生活，这是事实；努力工作，其实只是刻板 印象。

来到了大城市，你真的是在奋斗吗？好像，不一定啊！ 这句话，很多人不爱听，但却是事实。

大城市里，沉迷于游戏的人，少吗？得过且过的人，少吗？

工作懒散、不爱学习的人，少吗？细数下来，其实并不少。

不要拿身边少数优秀的人举例子，很多人其实就是拿着不高 不低的工资，过着每天都原地踏步的日子。然后自我安慰： 我在大城市，我在奋斗，我其实很努力。不愿意回去的原 因，其实是在逃避。

披着“漂泊奋斗”的外衣，躲在自己想象的、虚无的壳子里， 逃避那些人生要面对的重大问题。这就是很多人的写照啊！

要找到自己的位置

大城市的薪资待遇确实高，但是我们看到的，不应该仅仅是 工资待遇，而是要找到自己的位置，把握住属于自己的机 会。这才是你在大城市奋斗的意义。

仅仅努力是不够的，还要考量自身条件，比如学历、专业、 自身性格、掌握技能等等，去寻找一个合适的位置再进行奋 斗。这个位置，不仅仅是岗位，还要考虑行业、公司规模 等。

没有技能就去学，不知道信息就去问。任何事情，总有解决 的办法。与其困在原地抱怨，倒不如去试着找路。

比如，大家都知道程序员的工资高。但是，只有互联网公司 的程序员工资高；传统 IT 企业中，程序员的薪资是，普遍低 于互联网公司的。你看 CS 专业的学生，如果走技术路线，通 常是首选互联网公司。

你看，大家做的都是类似的工作，但是待遇就是差别很大 所以，即便是类似的岗位，在不同行业、不同公司，因为规

模不同、岗位内部权重不同、导致薪资待遇差别相当大。

这仅仅是一个方面，还有很多其他重要的因素。都是需要我 们努力搜集信息、认真思考的。只有这样，我们才能享受到 大城市的“加持”，才是真正的奋斗。

别再纠结逃离的问题 什么样的人在想着逃离？就是觉得自己混得不好嘛。什么样

的人，又要逃回？哎，是因为发现在其他地方也混不下去

了。

觉得自己能在大城市立足的人，从来不会考虑逃离的问题； 在小地方活得舒服的人，又哪里会考虑来北上广深呢？

说白了，没有一个明确的目标，也不知道自己到底在为什么 而奋斗。活在时代的洪潮中，被舆论的声音所左右，对自己 始终没有一个残酷的思考、清醒的认知。

所以，不要纠结“逃离”、“逃回”的事情。认真思考下，自己到 底想要什么样的生活。即便是有一天，因为家庭、户口、子 女教育等原因离开的话，那就早做准备，从容的离开。

不要做随波逐流的浮萍！而是要在万千世界中，找到自己的 立足之地！
