# Day200《连接

《连接高能量的人，让你实现不断跃迁》

最近，我越来越强烈感受到社交时代已经到来，未来一切商 业模式都将离不开社交，通过社交网络与别人建立高效连 接，将成为我们生存和发展的核心能力。

今天，结合我的实战经验，与大家分享一个价值极高的话 题：连接高能量的人，推动你的人生开启精进模式，不断实 现跃迁。

不知道你有没有意识到：人与人之间的认知水平、能力层 次，以及影响力量级差距是极大的，我们可以统称为“能量等 级”不同。我想说的是，那些能量爆棚的人，并不是一下子就 达到了现在的层次，而是通过不断获取能量，不断跃迁到一 个个新的量级上的。

那么，我们如何能够让自己的能量获得提升，实现自己的能 量跃迁呢？

首先，利用时代势能，连接高能量的人。 社交时代，我们可以低成本的与更多人建立连接，尤其是那

些高能量的人。靠近高能量的人，你才能不断吸收正能量，

增强自己的能量，从而进入新的能量层级。

时间是有能量的。你要花大部分时间重点去连接高能量的 人，进入高能量的圈子，而不是停留在自己既有的圈子里。

金钱是有能量的。你可以付费去连接那些高能量的人，用投 资的心态去花钱，而不是捂着自己的那点钱，让金钱的能量 高效流动起来，从而为你服务。

你自己是有能量的。你要做一个积极主动、做事靠谱的人， 让别人感受到你的能量，你是能不断做成事、取得成绩的 人，这样你更容易吸引到高能量的人。

2019 年我给自己制定了一个行动计划：选出自己朋友圈的能 量 TOP100 的人，把更多时间、金钱和能量投注在这些人身 上。

其次，打造自品牌，放大自身能量。 社交网络可以帮助你打造自己的个人品牌，同时丰富的学习

资源可以让你在自己热爱的领域成为专家。具备个人品牌的

专家将天然具备强大的能量，会大大提升你的价值和影响

力。

这已经在许多人身上得到了验证：自品牌会成为一个能量接 收器，拥有自品牌的你会更容易获得机会，从而又将自己提 升到更高的能量层次。

最后，小结一下：连接高能量的人，并通过打造自品牌放大 个人能量，你也将成为越来越高能的人。
