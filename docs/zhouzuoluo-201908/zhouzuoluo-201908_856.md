# 《化悲观为乐观，就

《化悲观为乐观，就在你的一念之间》

悲观和乐观就在你的一念之间，如果你能接受最坏的打算， 你也能接受最好的打算，你能接受最糟糕的状况，你也能接 受最好的状况。想想你面临的最大困难或者你生活里经受的 任何个人挫折、失望。你是打算用什么样的心态来面对呢？

如果发生最糟糕的情况，什么样的反应对你最没有帮助？你 可以看一下以下两个例子，如果发生了最糟糕的情况，用最 悲观的心态对待，会是什么结果？

1、“遭遇裁员，于是在社交媒体上对前雇主和同事大肆吐槽 诋毁，令其他公司以后不太可能雇用我。”

2、“每一次投稿都遭到拒绝，从此不再写一个字，更别谈写 作了。”

当然，悲观带来的后果往往很严重，让自己认为这个世界都 对自己充满深深的恶意。然而，最悲观的反应的对立面是什 么呢？

1、“在社交媒体上对前雇主和同事大肆吐槽诋毁”变成“每天在 社交媒体上，寻找我佩服其工作成果的公司或人，说一些积 极的话。如果他们看到了，说不定就能帮我建立人际网络 呢”。

2、“从此不再写一个字”变成“每天都写点什么，记录自己，记

录世界”。

当你心中能够想到糟糕的应对方式和好的应对方式，你的心 理接受能力会更强，会使你的心理更有弹性，能接受坏的也 能接受好的，相当于个人有选择。不管你“最糟糕、最无济于 事的反应”的对立面是什么，试试挑战自己，去做一件需要付 出更多力量和决心的事情。

拿第二点写作来讲，写作初期的时候，我们坚持每天哼哧哼 哧的写作，像一只勤奋的小蜜蜂，但是没有收到什么回报， 自信心容易受到打击，容易怀疑自我。

然而，你想象一下你初期写作的对立面：我通过写作，你链 接了数万人人，帮助了更多的人，我收到了丰厚的金钱回 报，我的文章阅读量都是好几万，我创建了个人品牌，等等 等等。

你这样一想，是否更有激情坚持写作了呢？ 反正我正是如此想的。大约两周之前，也就是写作刚一个月

的时候，我感觉我的肚子里已经没货了，苦苦憋了 2 个小时， 还是毫无进展。然而，我再想，与其在这里苦苦憋不出来，

不如想象一下，未来的我有多牛逼。

这时，我又想到了投资自己这个概念，所以，立马就再翻看 李笑来老师的专栏，再把投资自己看一遍，又有了新的感 想，就这样，以一副对未来的憧憬，忽然激活了大脑，让我 顿时有了目标，有了前行的方向，一篇文章也就由此而生。

这样对未来的憧憬，给了你一个积极并有意义的具体目标。 这个目标现在成了一种你有愿望望甚至渴望实现的东西。

当前的写作困境只是现在而已，放到长远的未来，现在的困 难都不是事，我的选择是挑战自己，报以成长的心态写作， 争取做未来期待的自己！

ps:这篇有点碎碎念的样子，但也是给自己一点小小的鼓励。 我相信，未来的大牛这个坑位，为我留着呢！
