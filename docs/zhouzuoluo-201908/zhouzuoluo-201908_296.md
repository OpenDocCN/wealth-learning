# 《年轻人真的不要太

蓝子鱼 : 《年轻人真的不要太爱钱》

现在大部分的年轻人压力都特别大，工资普遍不高，却要面 对高额的房租和各种费用，所以很多人把钱看的很重，但如 果过分爱钱，这可能是一种坏毛病。

前几天遇到了一个同学，就聊起了他的工作，在这个过程中 他很得意的告诉我，他是如何高明的每天早上省下五块钱。

他目前还是大四，实习的时候住在学校，距离工作的地方有 点远，去公司的路上有地铁和公交，如果坐公交然后转地 铁，只需要半个小时，价格是 7 块钱。但如果只坐公交的话需 要转一次车，但因为刷次数所以转车是不用钱的，这样路上 要花一个半小时，价格只要两元钱，他选择了第二种只需要 花两元。

因为每天上班坐公交要一个多小时，所以他早上就要起的比 较早，晚上又很晚睡，我问他这样不困吗？他说当然很困 啊，反正实习也没什么事干，到公司就找个没人的房间打瞌 睡。他是做测试的，所以有一些用来测试的房间平时老板也 不会进去。

我这同学为了省五块钱，每天早上要多花一个小时的时间在 路上，因为早起到公司还很困，工作效率自然也很低，虽然 省下五块钱，但我却觉得他亏了。

之前看到一位博主爆料了一件事。春节将近，他帮同事抢一 张上海到贵州的火车票，一共是 749.5 元，同事转给他 749 元。 他问：“还有五毛钱呢？再说，你用微信转我，我提现到卡里

还要手续费呢。”

同事说：“不就差你几毛钱吗，真小气。” 这位博主听了很生气，直接怒怼回去：“对，我是不差这几毛

钱，那你自己抢吧！”博主当场就把辛辛苦苦抢到的票给退

了。

这个博主的做法真的太解气了，辛辛苦苦帮同事抢票，同事 还想占便宜。有时候几毛钱就能看清一个人。

前几天我三个同学合买一个东西，价格是一千块，我室友付 的钱，另外两个同学通过支付宝转给他。一千块三人平均是 333.33，第一个同学问我室友总共多少钱，我室友说一千块， 然后这个同学自己就转了 333。后来另一个同学来了，我室友 让他也转 333，但他说不行啊，这样岂不是让你亏了，然后他 转了 335。

第一个同学看似赚了三毛，但他无形之中也告诉了别人，他 就是一个爱占小便宜的人。这个同学家里非常有钱，但他却 是一个爱占小便宜的人，平时各种占便宜，现在整个班都知 道了他。

年轻人爱钱没有错，但太爱钱的人往往都混的不怎么样，人 生有很多看不见的东西远比钱重要。

不要太关心存量，而应该关心增量，与其费尽心思省几块 钱，不如把心思用在工作上，多创造点价值，多拿些工资。 2019-06-16(28 赞)
