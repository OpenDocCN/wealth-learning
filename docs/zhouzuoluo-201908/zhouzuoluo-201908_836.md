# 认知新得 自我成长

静水 : 认知新得 自我成长 四十岁，我为什么带着孩子一起逃离高考重灾区？

01

大学毕业那年，静水阴差阳错来到这个十八线小城，侥幸进 了一家光鲜的一流名企。

做过财税干过营销，当过内训师登过大学讲台。 裸辞后创办了自己的育儿工作室，业余经营自己的新媒体。 这一年，我对这个世界的认知足以颠覆我前半生的逻辑。 如果老老实实待在体制内，我可以无忧无虑地享受国企特有

的稳定与安逸，但我却永远无法看到一米之外的精彩人生。

得失永远是一对孪生兄弟，裸辞这一年，我以自己的行动和 业绩，彻底撕去了瞎折腾不认命的黑标签。

创办育儿工作室的初衷，是陪伴孩子的成长，也不丧失自己 的人生价值。

然而，这一年下来，我发现残酷的事实推翻了美好的预期。 虽然我把工作细化到了极致，并把工作室定位为高精尖。

我惊人的发现，有的孩子写作业比登天还难，父母急得吼劈

叉。

我以为我能彻底改变这一顽疾，因为我坚信我有足够的耐心 和爱心来陪伴。

并且除了作业，我还为孩子们设定了阅读与写作时间。 事实证明，一个人的力量太过单薄。遗憾的是，大部分时间

都耗在了矫正孩子的基础学习习惯和提升孩子的学习积极性

上。

02

当有个孩子告诉我：“我要的是自由、自由、自由，老师你怎 么不理解我呢？我爸妈没有上大学，照样日进十万。”

愕然之余，我继续努力感化他的内心，没想到孩子却说，你 给我打鸡血没用，再这样管我，我就自杀。

真的不敢想象，一个十岁的孩子，居然在深夜无人时砸了我 工作室的玻璃。

我没有生气，只是觉得孩子很可怜。我做了很多功课，却怎 么都燃不起他对学习的热望和动力。

当他偷偷摸摸地打游戏，手机被我没收时，仇恨的眼里欲喷 出火来，并大声质问我：“老师请你告诉我，为什么在家里我 爸可以打着游戏骂着人，我妈可以刷着屏聊着天？”

“你家都有什么藏书？” “啥书都有，荡满了灰尘，从来没有人读。”

我不忍放弃一个资质优秀的孩子，为了寻找答案，我拨通了 孩子妈妈的电话。并把孩子的原话传达一遍，欲和她探讨个 一二。

“我在公司已辛苦一天了，到家需要放松一下，他是学生，他 当然不能玩手机打游戏。”

“孩子是不是也辛苦一天了？”

挂断电话，我陷入了沉思。

我突然意识到，一个孩子的底色来自于他的父母。无论老师 多么负责任都抵不上他有一个好爸爸或者好妈妈。

也许我能做的，也只能努力把自己的孩子教育成理想的样 子。

03

谁说学历不重要？没有学历我们从乡村到城市扎根发芽就会 无比艰难。普通高校的学历在和 985、211PK 时，注定被 PASS 在外。

因为我们多数人都是普通人，就连马云、俞敏洪、刘强东如 此的牛人，不也是高考的受益者吗？

天津引进人才政策公示后，我便和张先生商议，我要关闭自 己的育儿工作室，好好在家教育自己的娃。

并且打算卖掉其中一套房子，凑个首付，带着孩子逃离高考 重灾区。

“你咋不直接去米国呢？”他表示很无语。

“要是有条件，你以为我不敢？”我不依不挠。 重大决定前的第一个回合，我们打了个平手。

接下来我就发挥自己的语言优势，动之以情、晓之以理，来 对比我俩的人生。

“我们都是资质很好的孩子，为什么只上了一个普通院校？为 什么从房奴车奴甚至孩奴，一路奴到将近四十岁？”

“起点不同，无可比性。”理工男向来惜字如金。 虽说条条大路通罗马，但很多人就在罗马脚下，我们无缘坐

落罗马脚下，但可以在罗马郊区择一隅而居。至少我们的孩

子可以轻松望见罗马。

2018 年河南高考人数 98.5 万，天津 5.5 万。我承认我们的孩子 不是天才，如果同样的努力和奋斗，我更想让她分出一点精 力和时间，做她自己。

作为过来人，我清楚个体在这个社会生存中必须遵守的法 则。学历不仅仅是块敲门砖，还蕴含着你年轻时的奋斗，你 见过的人，你走过的路，还有你未来的人际圈。

物以类聚人以群分，每个阶层的界限向来泾渭分明，除了努 力渗透，你永远无法一下子跨越至另一个阶层。

如果有捷径，我认为高考仍是底层人逆袭的唯一捷径。

当然，会有人拿出一堆真实的案例反驳我，谁谁没有读过大 学照样年薪千万，谁谁没有读过大学照样全球飞。

我不否认，如果有这种奇才，只有两种可能，要么他有个王 思聪般的爹地，要么他后期的成长一定堆积了超越常人的拼 死努力。

04

“静水啊，我当年为了你放弃大城市的繁华来到这里。十多年 过去了，我觉得在这个小城挺好的，生活得舒舒服服，有班 上着，有孩子养着，你还想要啥呢？”

“要想改变，你早干啥去了呢？”

我给出了如下回复：

一是我可以保证不给你施加任何经济压力。虽然我暂时还远 远达不到想要的财务自由。

二是面对重大选择，看似是起点高低的问题，实则是一个人 视野与眼界、见识与认知的差距。没办法那时我没有这个认 知，现在有了。

“好吧，你去折腾吧，你是一个支点，期待你能撬动地球，如 果能把我撬走，我就缴械投降，不能撬走，我就按部就班。”

费尽口舌，一家之主终于肯支持我的想法了，于是就迅速行 动。

在这里就不得不说说我的“静水人生”了。

从 2017 年元旦，静水正儿八经在微信公号上分享自己的思想 价值。

从婚姻到育儿、从理财到读书，越来越多的同频人走进我的 世界。

短短一年多的时间，粉丝从零到一、到一万、到两万、到五 万。感谢这个伟大的时代，感谢互联网创造的奇迹。

因为写作，我认识了同龄的二娃妈妈燕子，因为燕子我认识 了阿轩，一个优秀的 85 后中介。

静水交友有个原则，这个人首先得靠谱，其次得有责任心， 在我眼里，阿轩便是这种人。

在阿轩的帮助下，短短两个月时间，我完成了迁户买房两件 大事。将近四十岁这一年，静水又狠狠地折腾了一次。

浮浮沉沉十几载，我越来越认同，没有背景，你不想总有背 影，那就去一个凭本事吃饭的地方。

大城市更包容一个人的努力奋斗，也更容易接纳一个人的与 众不同。

关于选择，没有绝对的“得”，也不会有纯粹的“失”。取舍之 间，必有得失。

一个人无论身处什么样的环境，只要足够自律与平和，坚持 与努力，前方的路只会越走越宽阔。

年近四十，我带着孩子一起逃离了高考重灾区，为了孩子也 为了自己。

真诚静水和各位宝宝，一起成长，精进人生
