# 2019.02.1

《其实是你在给自己贩卖焦虑，与他人无关》

…………………………………………………………

【星球坚持每日写作第 73 天】

每日一句：时间有限，把时间留给新鲜的事。

都说我们现在这个时代，是极易贩卖焦虑的时代。我却认 为，是真的他人在贩卖焦虑？还是只是你眼里看到焦虑而 已？

昨天我分享了成长型思维和固定型思维的区别，受到很多有 同样感悟的伙伴的认同，若是认真理解了成长型思维，所谓 的贩卖焦虑，其实是不存在的，因为成长型思维的人，看到 的是机遇、是挑战、是自我成长的机会。

有句话说，你所谓迷茫，其实是你的能力配不上你的野心， 从未牺牲过，却想要获得本满钵满的，你不焦虑？谁焦虑？ 自己也经历过，焦虑到极点，那种感觉不好受，但是不能回 避，应是直面它。

所以，再有下一次，自己处于焦虑期了，我希望你可以这样 做，什么事让你焦虑不已的，就专注做这件事，做到你不焦 虑为止，真的有效果，这是有一定原理证明；再来就是不断 培养自己有成长型思维的习惯，改变自己的思考模式，眼里 看到的都是解决方法，那么再有所谓的贩卖焦虑，你看到的 完全就不一样了，你发现世界变了，不是真的世界改变了， 他不会因为任何人而改变，若真的是，那一定是你看世界的

角度变了。

以前喜欢说人定胜天，这里面难免含着年轻时的气盛，而现 在我喜欢说人定感天，努力到感动上天，世界宇宙这样的大 哥，都会为你让路。

所以，真的没必要拿着稀罕的时间去指责谁谁谁又在贩卖焦 虑。

………………………………………………………………

年轻时，拼命向上生长是唯一正确的事。
