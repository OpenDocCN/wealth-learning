# 《瑜伽练习不一定在

《瑜伽练习不一定在瑜伽垫上也在生活中》

新年好[玫瑰]今天是橙橙加入星球的 193 天，和大家分享关 于‘瑜伽’的话题」

今天早上一早去妇幼产检，排队空腹抽血，早上起得早赶去 医院，来不及在家练习瑜伽了，就感觉到浑身不舒服，排队 的过程总是一个漫长的等待。

我在想做点什么好呢？ 想起师父曾经说过：我们在垫子上练习瑜伽最终都是服务于

生活的，要把在瑜伽垫上练习的心态带到生活中入。

1.  01

    我是如何在排队的过程中练习瑜伽的

    排队的时候也在调整山式站姿，因为大着肚子，重心不稳， 骨盆容易前倾，如果不注意重心都在前脚掌，所以要有意识 的主动调整身体的中立位并且保持。

    如果细心观察：大部分排队的人都喜欢一个姿势就是稍息的 站姿，重心偏向一边，然后再交换双脚转换重心的方向。

    可能从来没有练习瑜伽的人不太清楚，这不仅仅是重心的问 题，会直接影响到脊柱侧弯和肩颈不舒服，因为身体是一个

    整体。

2.  02

当身体出现不适如何自我疗愈？

瑜伽老师有个小能耐，就是当自己的身体出现不舒服的时 候，懂得自我疗愈，比如：如何缓解腰痛？如何缓解肩颈的 疼痛？如何缓解腿部的酸痛？

我们总以为要在瑜伽垫上练习才算真正的瑜伽练习，很多瑜 伽爱好者在工作和生活中因为不良坐姿，造成身体的不适， 然后来到瑜伽垫上练习一个小时，感觉很放松，但回到自己 的工作生活中依然不注意，所以还是没有缓解到不适，那为 什么不能将垫子上练习时的锚点带到生活和工作中呢？

码字的时候，肩膀很容易紧张，但只要把瑜伽练习的锚点带 进来（锚点：双肩下沉，肩胛骨向下去找骶尾骨，头顶心向 上，颈部后侧创造空间）其实关于肩颈得调理不一定要在垫 子上去习练才有效果，平时把瑜伽的意识带到生活和工作中 就可以缓解了。

▼总结一下： 引入师父说过的一段话：

练习瑜伽的人常常为瑜伽垫子所局限住，认为垫子上的练习

才是瑜伽练习，离开垫子就是另一种生活。

瑜伽垫子上的练习是帮助练习者去探索、发现真正的自己， 从而可以与自己和谐相处。离开瑜伽垫子之后，就要把垫子 的练习感受带入生活，与自己和谐相处，与他人和谐相处。
