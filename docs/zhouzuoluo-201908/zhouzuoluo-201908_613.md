# 你终身成长道路上的

你终身成长道路上的第一个坎儿

01

在持续进步的道路上，刚开始，只要你有执行力，你的进步 不会太慢。这个时候也是你能经常的获得正反馈，能坚持下 去的时期。

但，到了中间的时候，也来到了所谓的“半途而废”阶段。 这时你觉得你进步不像之前那般快了，甚至是在原地踏步，

你很苦恼，也开始怀疑，我是不是不适合这个，我到底还要

不要坚持下去，做这个还有意义吗。

这时候就不是执行力的问题了，单纯的执行力，并不能帮你 解决这个问题。

02

你现在面对的，已经不是一个每天花功夫死磕就能解决的简 单问题了。

而是一个系统问题。 这是你的瓶颈，也是你的龙门。

这意味着，你现在的做事方式已经解决不了你当下的困境

了。这个阶段，你需要用更加全面的思考方式：

执行力对于现在的你来说只是基础要求，你还需要更进一 步：你要有掌控力。

何为掌控力呢？ 我通常把你要解决的系统问题，称之为项目。而掌控力，便

是你对这个项目进度的把握能力。

那如何把握呢？ 把握能力的体现，在于时间。

比如，你把这个项目完成，你给自己定一个交付日期。 那如和定一个合理的交付日期呢？

这就有一个前提条件： 你必须要熟悉自己的单位时间产出效率。

有一个专门针对这个的概念，叫做：时间颗粒度。 举个例子：如图二。 我这里主要是关于写作的放在上面。当然这个“时间颗粒

度”也可以用于你平常工作，学习，甚至是娱乐上。请自行举

一反三。

关于“时间颗粒度”如何计算，你可以在平时刻意的注意一 下：当你在做一些事情的时候，大概花了你多少时间。

当你对于自己的“时间颗粒度”有清晰的认识以后，你对项目 定交付日期时便能做到胸有成竹。

03

最后掏心窝子的和你讲一点。

很多人其实就是不清楚自己的能力边界。 往往给自己定一些自己就觉得能够完成，自我感觉良好的，

实际上却完成不了的目标。

然后，做着做着，发现，这个项目卡住了，推进不了了，于 是开始焦虑，混吃等死，各种拖延，最终啥也干不成。

这样的半途而废令人惋惜，你千万不要放弃。 把这些困难当做项目去完成。当你项目做得多了，你的掌控

力也越来越强。你会慢慢发现，人生，理想不就是一个个项

目组成的吗？

越走到后面，路是越宽的。
