# 《至:粥老师与星友

道亦有道，大道至简 :

《至:粥老师与星友的一封信》 努力做到:与人为善，久处不厌，的 90 后大男孩 今天是 18 年的最后一天，就是所谓的大年三十，早上收到了

亲朋好友的祝福，同时也走心的回复了一封祝福信。

亲朋好友都是我生命中必不可少的组成部分，也都是冥冥之 中安排的，也是在工作与生活上提供很多帮助的人，想到这 里不由想起今年使我进步很快的粥老师和星友们。

刚刚查看了一下，自 11 月份进来星球陆陆续续写了 32 篇文 章，共收到 109 条评论更甚者是共收到了 432 个赞，我的天啊

，这对我来说是多么值得高兴的事情。

这其中包含粥老师的打赏与写作课的引导，还有就是星友们 的鼓励和互动，当然也有我努力的成分在里面，在此跟大家 说声＂感谢＂。

目前内容质量还未达到很好的水平或者用粥老师的话说就是 还很屎，但我明显感觉到写作在生活与工作对我带来的影 响。

以前的我，每天都在学习，但能吸收的养分只有自己清楚。 现在不一样了，工作中在思考、聊天在思考、看电影在思

考、出去玩还是在思考，可以说何时何地都有了思考的习

惯。

还有我感觉明显有提高的地方就是，以前写个一千字需要两 个半小时。现在先不说质量，就速度已经比之前要提高很 多，一个小时左右就能搞定，有时候思路清晰速度还可以更 快，这些是让我很欣慰的地方。

在这里我借助社群，向粥老师及五千多名星友拜个早年，祝 大家新年快乐，新年里无论业绩，还是个人成长都有明显的 进步，变得与众不同。

也同时说几点＂温馨提示＂。

1.注意饮食饮酒 过年回到家，亲戚朋友难的一聚，几乎所有项目都是，喝

酒、唱 K、大鱼大肉的场合不断，这样就很容易引起消化系统 的疾病，大家还是要注意荤素搭配，合理膳食，饮酒切忌，

小饮怡情，大饮伤身。

2.注意休息 工作一年了，回家好好放松放松，打牌，打麻将，这都是久

坐的项目，对眼部、腰部、颈部伤害是很大的。

过年放松可以理解但要适量适度的进行，否则结果就是，本 来该放松的假期因为休息不当反而更加疲劳，收假后反而对 工作不利。

3.放松之余不忘充电 我看了很多星友的帖子都说了春节期间继续日更，这无遗起

到了很好的带头作用，我个人也很支持这个做法。

记得毛 yeye 曾说过一句话，三天不学习，赶不上 liushao 奇，华 与华的华总也说过这样一句话，要想做好一件事就要费日日 不断之功，出滴水穿石之力，我想这就是从平凡到伟大的捷 径吧。
