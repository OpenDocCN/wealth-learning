# 《与“内心潜藏的需

《与“内心潜藏的需求”做朋友》

每次复习读书笔记，看着自己以前写的思考笔记，就又会产 生新的思考。今天我再度复习了，一句话：与内心潜藏的需 求做朋友。这句话是什么意思呢？且看我分享给星友们。希 望对你们有所启示。

首先，问下你：你有过这样的体验吗？ 你必须在今晚写作 1000 字，否则会付出代价。但你却呆呆地

坐在电脑前，脑子里空空如也，一个字也不愿意敲。你一会

儿看看手机，一会儿打开各种网站看看，所有的内容你都看

过了，但你还是一遍遍地刷着手机，刷新网页。

你强烈地谴责自己，发誓再也不做这些无聊事了，要搞手头 的事情。但过了一会儿，刚写了几个字，你又开始刷新网页 了。

那么，换个想法。你仔细地聆听一下你内心的声音，你会听 见，你心中有一个部分在大喊：你整天做令人烦躁和劳累的 工作，你太需要休息和娱乐了，赶紧玩耍一会儿，放松一下 吧。

现在，你要如何做呢，真的聆听你内心那些声音，然后就放 下手头的工作，去玩游戏，追剧吗？那不和上面的体验一样 吗？

其实你要做的是，给你的内心说：“我理解你的想法，我一定

会去做的，但不是现在。因为，此时此地，我必须先把手头 的工作完成，请你也理解我好吗？"

这个时候，你会发现，那些让你分心的想法不再纠缠你了， 你也放下了不停的刷手机的动作，去做你手头上真正要做的 事情。

大家不妨试一试，有没有这样的效果，有的话，为什么会有 这样的效果？

其实，原因就是：你把自己内心的每一部分需求都当作朋友 来看待，这样它们都不会捣乱。这样的话，你不是试图控制 或压制一些内心的需求，不管好需求还是坏需求，你总能从 它们当中找到正面的信息。

反之，如果不这样做，而是一个人整天强迫自己完成这个任 务，担起那个责任，那么，这样的人就会在其的潜意识中， 埋下负面意识，随着时间的推移，这种负面的潜意识越积越 多，终将爆发。

然而，在爆发之前的那一段时间，从显意识上看，这个人似 乎很负责、很正常，但从潜意识上看，这个人的内心中会有 很多冲突。

但多数人没有这样的意识，他们对自己心中的负面需求，采 取无视或排斥或压抑或控制的态度。

比方说，有一天早上醒来，你不想上班，有人可能教过你， 要忘掉这种不好的感觉，对自己大喊几声“我很好”、“我很 棒”、“我一定能行”等口号，用这种积极暗示压下内心那个不 想起床的自己。这会起到一定效果，但最终会造成显意识与 潜意识的分裂。

你的潜意识中不想早起的意识，并没有消失，而是被压了下 去，但说不定哪一天，它会来一个大爆发。

我以前，就做过这样的事情，每一次想要早起，但起不来的 时候，我总是给自己打鸡血，我很行，我一定能起来，但我 丝毫没有注意到内心的感受，我是强迫我自己起来的，当 然，早起还是失败了。

而现在呢？我每次早起起不来，我的内心拒绝怎么办？我会 用理智，平静的告诉它，我要写昨日的复盘，还要写自己的 思考，要好好的对自己做一个检查，这样的话才能查漏补 缺，也是为了身体与心灵的健康啊！那么，请允许我起来， 迎接更好的自己吧。

其实这种做法就是接纳并安慰自己内心的那个“朋友”，让它 能够听从我的理智的指导，帮助我成为更好的自己。总的来 说，这就是一种正面的心理暗示。

使用好正面的的心理暗示，与你内心潜藏的需求做朋友，能 够使得你达到身心合一的状态，星友们不妨试试哟！
