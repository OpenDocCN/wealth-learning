# 《创始人的格局》一

《创始人的格局》

一个创始人的格局关系到公司的发展和方向。

1、 最近一个爆炸新闻在公司内部传开，听说某联合创始人股东 即将离职。刚传开不久，昨天公司人事部就发通知该股东由 于个人原因已离职。

由此事联想到对公司创始人格局的思考。现公司是由三个合 伙人共同创立。当时大老板从自己前公司找了两位同事出来 一起创业。

现在离职的这位股东，跟着大老板一起干，把公司从零干到 一。他出钱又出力，在该行业有超过十年的实战经验。

自从去年，他就被大老板发配“边疆”，调去武汉市场管理三 个门店。一个市场总监不管本土整个市场了，就相当于减 权，被边缘化。

听说现公司创始人是一个不放权的人，公司只要有超过一千 元的费用支出都需要他审批。假如公司盈利 100%，他是那种 只会给出 10%给下面的人。

他创立的品牌目前已经运营三年了，工厂的业绩也只是两三 千万。一个生产额只有两三千万的工厂根本不算多，以三年 的发展速度来看太慢了。

2、 另外一家公司的创始人，60 岁多的年纪还依然奋战在一线， 凡事亲力亲为。不放权给下面的人去干，公司发展规模也很 慢。

这位大老板每年不惜重金花千万请一位大师回来培训员工， 给员工洗脑，让员工深刻认识到自己的公司有多好多好。

殊不知他们的工厂早已破旧不堪，下雨天员工还要打着雨伞 上班。大老板有这么多钱请大师回来培训员工，却不肯花钱 修建工厂。或者花钱在提高人力资源成本或提高售后服务也 好啊。他们的售后也是大堆问题。

3、 我前公司的创始人在四十多岁的时候就开始物色到一位职业 经理人，一步步放权给他。目前这位职业经理人已经晋升为 公司的副总裁。

公司分为营销中心和工厂两个分管部。工厂老板和公司创始 人是以合伙人合作关系。两人各占一半左右的股份。

这说明什么？说明创始人自己不擅长的领域他就寻找擅长的 人合作并且达成股份制合作融为一体。大家一起合作，有钱 一起赚。

前公司越发展越趋向于年轻化，创始人知道未来的世界是年 轻人的世界。

他愿意在公司创立发展到稳步阶段时寻找职业经理人为他接 班。他好在适当的时候成功身退，既能享财富自由又能去享 受生活。

前公司经过十多年的发展，规模越来越大，越来越成功，目 前已经是行业的龙头企业，第一家上市公司。做到只要是这 个行业的人无人不知无人不晓的大品牌。

一个公司创始人的格局，他的企业在市场发展中看得见。公 司的发展见证着他做人做事的格局。
