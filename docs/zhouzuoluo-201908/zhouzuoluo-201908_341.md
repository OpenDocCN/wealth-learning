# Day186《戒骄

《戒骄戒躁，方能成事》

今天读曾国藩日记的家事和写字部分，从诸多细节中感触最 大的是对戒骄戒躁的反思。易骄易躁是大多数人都会遇到的 问题，古代如此，今天更严重了。

1、为什么富贵人家的子弟多不成器？ 曾国藩在日记中讲到了两点：一是因为有所依恃，也就是有

恃无恐，家里有权势和钱财，认为出事了都可以摆平； 二是

因为性情骄纵，打小活在别人的赞美和羡慕里，习惯于别人

捧着和顺着，于是“骄”就成为了富贵人家子弟中的通病。

而曾国藩是无比清醒的智者，认识到“得运乘时，幸致显 宦”，也就是是世上的达官显贵，绝大多数都是时运好，并非 自己的本事就真的比别人高了许多；即使有真本事，得乘时 运也是第一位的。于是，他特别重视克己自律，以及对子女 戒骄的教导。

在这一点上启示我们，要拿成为榜样为目标严于律己，不管 是在家中对子女，出门在外对朋友，还是工作场合对同事， 一定要戒骄，千万不要自以为很了不起，要知道成功往往讲 求运势，比如你遇到了一帮良师益友支持帮助你才能成事， 或者一帮可靠的同事一起合作才达成了目标。

2、为什么读了那么多书、听了那么多课，还是不成功？

曾国藩谈练字的感悟，讲到熟能生巧，学习除了有形的，还 有无形的，那些不可教的部分，要靠自己勤奋练习，日日 练，事事练，这对我们学习所有技能都有启发。

现在我们可以很方便地获得诸多知识和课程，学到五花八门 的方法和技巧，但真正学成技能，还是要靠持续的练习和践 行，过于急躁，没有耐心是不行的。

很多时候，我们只是看到了别人成功的结果，但各种艰辛、 所费的功夫、付出的代价，却被我们自动忽略了，所以我们 每当看到谁谁又成功了，谁谁又赚钱了，就变得焦躁不安， 什么都想试试，结果却一事无成。

用专注和耐心，代替浮躁之心，把一件事做扎实了，结果才 会越来越好。

最后，每天问问自己：今天，你又骄躁了吗？
