# 《为什么故宫后门的

《为什么故宫后门的北京老冰棍儿摊能爆卖？背后藏着的营 销道理》

2018 年夏天到北京学习结束后，最后一天早早便去了故宫参 观，游览完从故宫后门出来就是护城河，正好是烈日当空， 着实是热得慌。

很多中年大妈背着泡沫箱，一边走一边吆喝兜售“北京老冰棍 叻~~北京老冰棍~~”如果你朝她望一眼，她就说“来一根呗”， 这生意做得勤快又热情，然而生意却并没有很好，主动掏钱 购买的人并不多，我也没买。

后来有一对摆摊儿的夫妻，在桌子上铺开一张塑料纸，放上 些碎冰，然后把上百条冰棍儿从泡沫箱里拿出来，堆成一座 小山，冒着一股子凉气。他们放上个录音喇叭，重复地播 放“正宗北京老冰棍，一块钱一根”。没想到，很多人围上去 买，有的一个人就拿同伴拿了好几条，我买了一根。

这事我觉得挺有意思，同样的冰棍、同样的价格、同样的地 段，为什么背着泡沫箱吆喝的小贩只是零零散散卖几根，摆 上小摊的夫妻，却是“批量销售”呢？

1、视觉刺激 在桌子上堆成小山，大家都看得见到底是什么样的冰棍，而

且看着冒着的那一股冷气，连带着心里都感觉清凉。

相反呢，如果放在泡沫箱里，外地人根本不知道什么是“老冰 棍”。人都是这样，对于对陌生没有对熟悉的东西兴趣那么 大。

2、价格明确 摆摊的夫妻明确告诉大家“一块钱一根”，而是是用喇叭播放

的，证明人人都一样，不会存在买贵吃亏。

明码实价，对于外地游客其实是尤其重要的一点，很多时候 去旅游景点不太敢买东西，不一定是嫌价格高了，而是怕被 坑了。

而且，明确价格之后，不用多问一句，买完即走。

3、营造一种热销的氛围 一大堆的冰棍，一大群围着买的人，营造出一种生意火爆的

氛围。这种氛围一旦营造成功，后面的路过的人群，容易产

生从众的心理，比如我，本来是不买的，好奇为什么大家都

去买，就也买一根试试，反正只要一块钱嘛。

4、选择最好的时间节点 我在那条街站着等车已经有一会了，但是前面夫妻俩并没有

摆出老冰棍，是没有货吗？是来得晚吗？

我想不是的，是刻意等到人流量比较大，天气又比较炎热，

大家对冰棍的需求量最大的时候出摊。否则，如果冰棍儿摆 出来几分钟没人买，就得融化了。

再者，他们中午两个小时的销售量，也许就能赶上一天中其 他所有时间的销售量，从时间成本来算，性价比也是最高 的。

最后，做什么都怕有心，只要能想到的比别人多，同样的货 就能卖得比别人好，用最好的时间赚钱更多。
