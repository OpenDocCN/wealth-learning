# 《自我面对:不要拿

《自我面对:不要拿“为了生活”成为两地分居的借口》

有人说中年女性上有老下有小，为了生活，这些长期分居的 女性，哪有时间爱惜自己!

说的也有道理，可是你自己若不学着爱惜自己，难道还是指 望别人来爱惜你?

解决现实问题向外求：找方法，解决关系问题得向内求：找 感觉，你得自己静下心来思考为什么这些年一直不会爱惜自 己，你得面对且承认目前自己还不具备爱惜自己这项能力。

那么从现在开始，你要开始学习如何爱惜自己，就像学习如 何写文章一样。

1、觉察和面对自己的内心。 你可以在独处的时思考几个问题： “谈到爱惜自己，你想到了什么。” “你父母疼爱你吗，你妈妈爱惜她自己吗？” 在心灵成长的道路上，如果我们还一种执着于方法，其实表

示我们还只停留在头脑层面，如果只停留在头脑、理性层 面，又怎么去解决内心的苦痛呢!

要想解决内心的痛苦，你得先面对自己的内心，得鼓足勇气 先进入自己的内心，觉察自己的内心、表达自己的内心，否 则就是隔靴搔痒，就是毫无营养的心灵鸡汤。

2、极度独立的背后，是极度的孤独和无力。

异地恋或两地分居看起来是为了生活、被逼无奈，但深层原 因还是“不配得”的心理：觉得自己不配享受亲密关系的滋 润；这些女性可能从小到大从来没有可以谈感受、想说什么 就说什么的亲密关系。

他们的父母也许很负责任，让他们有衣穿有饭吃有学上，但 他们的父母很少去关注他们开不开心，父母灌输给他们的是 责任、义务。

他们的父母也许还经常吵架，吵架的时候谁也不让谁，吵完 之后谁也不安慰谁，甚至好几天都不说话，冷暴力。

从小生活在这种只有责任义务，没有情感流动的家庭中，孩 子感受到的是父母吵架，甚至互相伤害，幼小的心灵就对亲 密关系产生恐惧。

异地恋夫妻，各自从小到大独来独往惯了，每个人的内心都 是一座孤岛，极度渴望亲密，又极度脆弱敏感，就像两只刺 猬在寒冷的冬天相互偎依取暖，彼此又都不敢放下防卫的 刺。

3、亲密关系中的小吵小闹就是在“调情”。

对于大多数婚姻，吵架是常态，不吵是例外，不要再幻想夫 妻可以相敬如宾，那都是骗人的，爱一个人有多深，吵架的 时候对他就有多恨。

你若不离不弃，我必生死相依，吵着吵着就心软了，吵着吵 着就恨了，恨着恨着又爱上了。

当然有一种情况例外，那就是家挺暴力，家庭暴力也是经常 吵架，但还经常打人，经常伤人，即便如此，被打的一方也 愿意不离不弃，甚至生死相依，但这是病态的，这是一种施 虐和受虐的互动方式，不属于我们今天要谈论的内容，以后 我们有机会再讲。

除此之外，不管你们看起来有多嫌弃对方，吵架的时候有多 么伤害对方，只要你们还天天在一起，床头吵架床尾和，你 们就是最匹配的，你们的婚姻就是这个世界上最美满的婚姻 之一，磕磕绊绊才叫亲密关系。

4、与人亲密的能力与工作能力一样重要，一样需要学习。

之所以选择异地恋，或多或少还是因为缺乏与人亲密的能 力，解决之道是要去学习如何与人亲密，具体怎么学，可以 想想你的工作能力是如何培养出来的，你花了多少时间和心 思。

学习任何东西就怕没心，你心里没有这个东西，一旦它进入

了你的内心，“亲密关系”的能力培养起来也是蛮快的。

你心理得有它，每个人心中都有学习和工作，因为我们从小 就被管输了要好好学习，将来找个好工作。

可是父母、学校、社会从未告诉我们，如何去表达自己的感 觉，如何更好的与异性相处，如何去爱一个人，如何与异性 相处。

其实与人亲密的能力甚至比工作能力更重要，工作能力决定 着你的生存，亲密关系决定着你能否生活的更好，能否活的 更自由。

5、舒服的夫妻关系，可以让你变得更有钱，更富足。

如果你过去的新年愿望是：工作顺利、多挣钱，自己和家人 身体健康，那么新的一年，是否可以把以下愿望抛进内心的 许愿池。

“允许自己享受亲密关系，享受性。” “学会享受陪伴带来的安全感。” “放下独立的面具，慢慢学会依赖。” “既然嫁给他，就要和他在一起。” “好的婚姻就是要经常小吵小闹。” “允许自己享受亲密关系，享受性。”

因为和谐社会的基础是和谐的家庭，和谐家庭的基础是和谐 的两性关系，和谐两性关系的前提是夫妻双方都能表达爱、 感受爱、享受爱。

夫妻感情好，身心才健康，经常生病的，多半夫妻感情是有 问题的。

夫妻感情好，才是真正的富足，你见过哪对夫妻感情好，家 里穷的?

6、找谁结婚，也会“遗传”，背后有神秘的力量驱使。

其实没有一个具体方法，你照着做了，就可以解决两地分居 的烦恼，就像没有一个方法可以让你快速挣钱。

穷不过三代，富不过三代，但夫妻吵架感情不好真的可以传 递三代，不信你去体会自己的婚姻，再仔细观察父母的婚 姻，你爷爷奶奶的婚姻，再去问问你爷爷奶奶他们父母的婚 姻。

其实你要改变的不只是两地分居问题，而是背后你看不见 的，甚至是整个家族几代几百年的情感表达方式。

就像荣格提到的集体潜意识，它是我们的祖先遗传下来的潜 在记忆痕迹的仓库，它具有强大的力量。

往上推三代整个家族的情感交往模式可能像你们的夫妻关系 一样，但你可以通过学习 在你这里把它切断。

你有这种使命，你有责任和义务，为了自己，为了家庭，为

了家族。

当然你的内在也有这种潜能，因为一切的发生都是最好的安 排。

当你的婚姻发生了好的变化，你整个家族的命运也会有所不 同。

心灵成长的道路上，一直找方法，动不动就问怎么破，有时 是一种逃避，一种自欺欺人，一种偷懒。

当学生准备好了，老师就会出现，方法在实践中，方法在你 和伴侣的相处之中，方法在你和伴侣的争吵中，方法想你的 心中，

就像学习写作，只要你真心想学习，你一定能找粥老师的知 识星球，当你愿意花钱加入星球、每周花时间在星球分享， 慢慢的你就会找到写作的感觉。
