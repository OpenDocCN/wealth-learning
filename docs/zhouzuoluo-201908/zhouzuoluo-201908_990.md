# 《跨年之际，想要立

熊小帅 : 《跨年之际，想要立下 2019 年的 flag 并且执行到位？ 这里有三个思维模型，一份行动指南，可助你成之！》

★前言 今晚是跨年的一晚。明天，我们开始进入新的一年，在这跨

年交接之际，相信星友们已经开始准备立下 2019 年的 flag 了。 不过，我们总有这个坏习惯：开年立下的 flag，总是在年底被 啪啪打脸。那么，在这里，我给星友们分享三个思维模型， 一份行动指南，我称之为：【三合一疗法】，助你保持 flag 屹 立不倒，直到 2019 年底！

★正文

这三个思维模型，分别是：【HAVE -- DO -- BE】、【BE -- DO -- HAVE】、【SMART 原则】。行动指南是：【持续行 动】。星友们可以在这里停留一两分钟，思考一下，或者猜 一猜，我说的三个思维模型和一份行动指南，具体指的是什 么。你带着你自己的疑问，再来看下面的内容，你会有更深 的见解哟！

★①第一个思维模型：【HAVE -- DO -- BE】 这个思维模型，具体解释如下： 1、从目前出发:我目前有什么资源?【HAVE】

2、分析依靠目前的资源能做的事:就目前的资源情况,我能做 点什么？【DO】

3、实现:能做什么就做点什么，看最后能成什么样子。

【BE】

简单概括就是：我有什么? (HAVE) ，我能做什么? (DO)，我 成为谁? (BE) ，这是一种资源导向的限制型思维，内在包涵 的是一种由始至终的思维，用孙子兵法中的一句话来讲，叫 做：“先战而后胜”。

★②第二个思维模型：【BE -- DO -- HAVE】 这个思维模型，正与【HAVE -- DO -- BE】相反，指的是： 1、从目标出发，我要成为什么的人，完成什么样的事?

【BE】

2、找到实现目标的方法和路径:我需要做什么来实现?怎么做 能实现？【DO】

3、分析方法的可行性和所需资源:我拥有什么?我的不足是什 么?弥补缺少的资源:如何弥补我的不足?【HAVE】

简单概括就是,我要成为谁? (BE) ，我要做什么? (DO) ，我需 要什么? (HAVE)

★★★这是一种目标导向的成长型思维，也是一种以终为始 的思维，即孙子兵法所取之上策：兵者，先胜而后 战！★★★

大家一看，就知道，第二种思维模型才符合我们目标的确 立，即使没有资源，也要创建资源，没错，就是这样。然 而，现实中，大多数人想的是第一种思维模型，总是在思

考：自己有什么，可以用自己有的东西，怎么做，能不能成 事，总是想着自己现有的资源，这样就具有局限性了。

★③第三个思维模型：【SMART 原则】

【SMART 原则】，相信大家不陌生了。这里，我就不细讲 了，其实，在本文中【SMART 原则】起的作用是【DO】， 也就是如何做来完成自己的 flag。

我之前在星球里分享了一篇关于跑步目标，利用 smart 原则， 一年后，达到一次 10 公里的程度。我已经坚持两个月了，本 来预计 3 个月后，每天能坚持跑 2 公里，而现在，计划比意料 中的好：我已经每天能够轻松坚持跑 2.5 公里了。

【持续行动】是支撑你【DO】环节的关键要素。哪怕想法再 美好，计划再完美，不去行动，那也只不过是镜花水月。所 以，我们要行动起来！关于这一点，我在之前也在星球分享 过，现在把链接分享给你。
