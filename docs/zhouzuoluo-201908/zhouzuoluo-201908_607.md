# 写给初入职场的弟弟

不二 : 写给初入职场的弟弟的一封信 我有一个久未谋面的朋友，十几年前就开始做临时工。

当时工作机会太多，用工荒刚开始出现。街道上，道路旁， 电线杆上，随处可见的招聘信息。网络上扑天盖地的消息是

＂人口红利己是过去式，＂这时临时工应运而生。

那时年轻，感受不到危机。当时临时工的工资略高于正式 工。丝毫不愁工作的机会，进一个工厂面对着正式工有种优 越感。总觉得这儿工作一段时间，玩一段时间没钱了，又换 个地方工作。总觉的这样潇洒，自由！

时间就这样一天天过去，一点都没为将来着想过，也没有认 真打磨一项技能。三天打鱼 两天晒网。关键是工资也不高， 火爆网络的三和大神就是这时侯诞生的。

前段时间微信上联系我，说现在没有工作了。意思是有好工 作介绍一下。

我有点恍惚，这么多年了，丝毫没有为将来打算打算，也没 想着去学个技术啥的。一直到现在，36 岁的年龄，正值中 年，身上没任何余粮，也没有任何技术。去厂里当个普工， 人家都懒得要。人到中年不如狗，这已经是低微到了尘埃 里。

我想到了当时的情景，年轻气盛，每天下班后去小摊边烤 串，来碗田螺，来瓶啤酒。喝酒吹牛，每天都是微醺。后 来，我走出来了。他继续沉迷其中。

直到现在，当他发现过去的＂好＂日子一去不复返了。此时 后悔已来不及。现在找工作也只能找那些最垃圾的作坊，维 持着最底层的生活状态。

我问他，这些年一直东奔西跑，短则几天，长则个把月，动 不动提着行李捆着凉席跑去新工厂，累么？

他说，没办法，正式工工资太低，规矩多，不自由。 我说，哪里是什么规矩多，不自由?你这是懒散惯了，只想着

钱多事少。不想多付出一点。做事完全没有规划，年纪轻轻

就开始养老了。

弟弟，我想说的是，你现在正值年轻，你看不到这么远。你 肯定也迷茫，只是不知道该怎么办？就像我们过去，该吃 吃，该喝喝，偶尔也会在夜里规划一下将来。但狐朋狗友一 叫，又坐在了酒桌前。今朝有酒今朝醉，什么狗屁未来，全 部抛到九霄云外去了。就这样日复一日，时间不知不觉就溜 走了！

人会长大，年龄会增长，现在回望过去，多好的年华啊，不 管体能还是头脑最强大的时候，白白浪费了。你终究会为这 一切后悔。你终究会拼命地抢时间，弥补过去的遗憾。你终 究会玩命的学习，来找到那一丝丝不可能的曙光。

你现在刚刚毕业，有一个相对稳定的工作。但那是小城市， 生活节奏慢，根本没有热火朝天的气氛。我听你说过，平时 工作非常清闲，有时候正常工作日没事干，也休息，跑到附 近热闹地方玩耍，如同当年的我们。

其实，你唱 KTV，你上酒桌，这些没有任何意义。你身边的

人有能量场么？你做这些娱乐时有目的性么？若没有，那毫 无价值。你只是自己感觉很舒服。但完了之后，你真的快乐 么？

适当放松一下，有其必要性。但那也只能是为了更高效的对 待工作学习。现在的你，玩乐至死是最好的概括。大块的时 间这么宝贵，思考停止，身体停止。你是否会在夜深人静的 时候，感觉到虚度光阴呢？

我常去野外徒步爬山，我明白，当一个人实在走不动的时 候，停下来休息，那种快乐才是极致的体验。一个人，只有 累到极致，才能感觉到极致的舒服。这种满足感，是靠玩乐 永远无法体验的。

社会竞争这么激烈，高层次的人都在努力工作，不断提升自 己。你在这个年龄更应该玩命学习。当你做好规划，每天完 成自己的小目标时，你会感觉内心充满力量。坚持下去，整 个人会慢慢散发出能量场。

是不是完全没方向感？不知道该怎么提升？ 其实我们都是社会底层的人，你身边的亲戚朋友都是底层，

眼睛早己混沌，看不到那么远。无法指点你的将来。你应该

多接触一些高能量场的人，看看他们平时是怎么工作生活

的。人在迷茫的时候，阅读是最好的方法，书是他人的人生

经验的浓缩，只要多阅读，量变引起质变，一点一滴改变。

还有，可以加入一些付费群，别在乎这些小钱，就算这些群

没有实际价值，至少里面的人都是鸡血一样的人，体验体验

那个气氛，打打鸡血也是价值。看他们聊天多了，万一哪天

突然有灵感了呢？

听君一席话 胜读十年书，老祖宗的教悔，还是有道理的。

其实，努力主要还是看自己，一个老师手上的学生也有好有 差呢？你需要跟强者在一起，他们能有所成就，肯定有与众 不同之处。你无法融入强者圈，至少要观察他们，学习他 们。人生需要不断给自己打鸡血。

再有名的品牌都要持续投广告呢！
