# 《学会提问，让你每

《学会提问，让你每次都问有所得》

学会提问在现在这个社会真的是个大学问，经常会有一些莫 名其妙的人发一堆不知所云的话，说是问题吧，看着也不 是。说是闲聊，它还一直干扰你。

今天来和大家说说如何提问才能每次都问有所得 第一，遇到事情先自己想办法解决。 现在搜索引擎这么便捷优化，想知道的东西随时都可以找

到。凡是能自己解决的事情，就不要麻烦别人了。

人与人之间的关系都是有情感账户的，你每麻烦一次，就是 在取一笔钱。

第二，自己解决不了时，正确的向别人求助。 对于陌生人，放低姿态，谦虚谨慎，先介绍自己，表达清楚

自己的问题，等待回复。不论怎样，注意礼貌。

对于熟人，也要看看人家的时间和精力，不能勉强，道德绑 架。

第三，提问的内容要具体，不要太大。 这阵子总会有人问我“考研应该怎么准备”，这么大的一个话

题，你没有具体的科目，没有具体的时间段，我怎么知道怎 么回复？

知乎贴吧那么多资料，为什么不看完了带着自己的具体问题 来说呢？

没有思考过的东西，就去问别人是极其不负责的行为。 第四，提问要适可而止。 我微信上有个女生，三天两头问我问题，今天问一个人走孤

单怎么办啊，明天问想出去玩没钱啊……

就是因为我之前耐心解答了她的几个问题，她就一直有问题 就找我。现在让我看到她的消息就赶紧略过。

问问题要有度，这个度掌握不好，就会让人反感。
