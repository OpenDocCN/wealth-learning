# 《有关自律思考》前

听听 : 《有关自律思考》

前阵子朋友建了一个读书学习群，我们约定要做日更的，结 果我失约了。

朋友 a 拉了一个读书打卡群，将一些朋友拉进来抱团学习，a 找到我说，让我每天一起做一些读书分享。我没多想就答应 了下来，觉得以这样方式来促进互相交流学习是一件很不错 事情，而我也正需求这样互相监督的方式来促进学习。但是 没坚持几天，我就放弃了。

为什么？ 今天读到自律时候，我才“恍然大悟”。除了我给自己找的“借 口”：群里没有形成读书打卡的氛围，21 人的读书群除了我和 他每天傻傻的叨叨叨分享，并没有其他人参与进来。其实更 重要的原因是由于外部反馈的枯竭。

而外部反馈其实是有关自律最本质的东西。也就是说我们追 求自律渴望自律，根本痛点是在于外部系统的评价，也即通 过自律方式把自己不好的地方转化为好的，积极向上的，从 而获得外部认同。

比如回到朋友 a 组织的读书社群，我希望分享能得到群里小伙 伴的正面反馈，想获得他们的认同，在这过程中如果没有人 关注，那么这个读书分享的行为就会受到影响。

同样的，回到星球上其实也有相似性。我们加入星球的原 因，除了粥老师外，还想找到一群志同道合的小伙伴，互相 学习互相鼓励互相支持，抱团一起向上生长。只是要注意，

一旦没有及时得到外部反馈，比如写的不好，没有点赞没有 认可，也请不要就放弃。

自律的动力应源源来自内在系统，而外部反馈则是起到锦上 添花的作用。
