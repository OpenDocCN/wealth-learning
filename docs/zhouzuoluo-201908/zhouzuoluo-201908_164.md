# 大家好，我是六弦，

六弦 : 大家好，我是六弦，今天发表第九篇原创。

【中年人是如何一步步走向崩溃的？】 直接谈人从强到崩，或许较为困难，那我们就从物讲起吧！ 现代人的家中都有电脑，基本上也都遇上过电脑死机，系统

崩溃，那么电脑是如何一步步走向崩溃的呢？

大家都知道，电脑刚出厂的时候，系统空间是非常宽敞的， 运行起来是非常快速的。

可用着用着，空间越来越小、速度越来越慢，为什么？ 答案很简单，因为系统中储存的东西太多。 这些东西可以分为三类：

一、我需要的文件；

二、我不需要的垃圾文件；

三、我厌恶的病毒。

当这些东西，越积越多，不能得到及时清理，而系统也不能 够及时升级扩容时，电脑只能以蓝屏来表达它的愤怒了。

人又何尝不是如此呢？ 我们刚出生的时候，是单纯的、宽广的、毫无压力和焦虑

的，这时系统的运行速度是非常之快的；

到了青年期刚刚毕业时，我们还算轻松，只在事业上犯些头 痛，不过没关系，我们还很年轻，系统运行还能一切正常；

三十岁过后，大部分人开始焦虑了———和当初想象的人生 差别好大啊！ 为什么我们脱了鞋子往前赶还是难以在事业上超越呢？为什 么老婆不再娇羞可爱、贤良淑德了呢？为什么我的孩子总是 笨得出奇呢？为什么老爸老妈干啥都不如以前了呢？

......

这个时候，我们的肩膀上同时扛起了事业和家庭，虽然系统 略卡，比之前慢了许多，但总还凑合着用......

很快，我们终于步入了四十岁的不惑之年，可照常迷惑不堪

———不知道老婆为什么越来越能叨叨，不明白孩子为什么

那么多开销，不懂得父母为什么要不断的检查吃药，不清楚

老板为什么越来越刻薄。

钱呐！你为什么总是花的太多，挣的太少？ 在老板、老婆、孩子、父母、隔壁老王、优秀同学的多重压

力之下，你终于蓝屏——崩溃……

这样看来，中年人的崩溃完全来源于生活压力。 可是，哪个人不是这样子过来的呢？为什么偏偏就你崩溃了

呢？为什么人家的电脑运行的还是非常完美呢？

原来，中年人的崩溃并非主要来缘于生活上的压力，而是因 为缺乏持续不断地清理和升级。

清理什么？

清理不需要的垃圾文件和病毒啊！ 前面说过，当电脑存储的东西太多，就会使得系统越来越

慢，越来越卡。

我们的人生若不及时清理掉那些不值得记忆的琐碎和生活中

的不良习惯，必然也会导致大脑反应迟钝、身体素质降低。

我们要擅于忘记，忘掉曾经的苦痛、烦恼，那些令人痛苦的 事情，真的不值得回忆。大脑空间是有限的，装满了忧伤， 必然装不下快乐。只有把那些忧伤都及时清理干净，我们才 能时刻保持大脑轻松、心情愉悦。

不良习惯就接近于电脑中的病毒了，什么抽烟、喝酒、赌 博、好色、懒惰、托延，这些病毒表面无碍，却在无声无息 中消耗着人的健康和时间。 健康差了，不但不再能持续赚钱，还要转过来花钱，成为家 庭中的另一个负担，早知如此，何必当初呢？

为一场酒喝上整个下午，为一个异性消耗半月时光，这种人 的时间太不值钱了，所以才有了后来的崩溃。如果把这些时 间都用在升级系统上了呢？ 前面有讲，除了时常地清理垃圾和病毒，还要持续地升级扩 容，才能让空间更广，系统运行起来更快。

人的系统自然也需要升级。 想要升级就要不断学习、持续思考。 而这总需要大量的时间。

当我们把病毒清除干净后，便有了更多的闲时去学习和思 考。

当电脑没有了垃圾和病毒，系统也不断更新升级，自然能够 常时间完美的运行下去；

当人擅于遗忘痛苦，并可以做到克己自律，还能坚持每天学 习和思考，自然就不会崩溃啦！

愿你我的人生皆不见蓝屏，一路到底顺水顺风。
