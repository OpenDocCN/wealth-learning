# 《粥左罗：我会更加

《粥左罗：我会更加爱惜我的文字》

接上篇分享，简单说一下我对写字人的看法。 很多人说我写东西负责任啊三观正啊有正能量啊。 除了我自己的性格和喜好，还有个原因，我的很多亲人都是

我公众号的忠实读者。

前几天我大伯家的姐姐来北京出差，我们一起吃饭，她说我 写公众号上的文章她每一篇都看。这个我是第一次知道，因 为我们基本没怎么联络。

但是我知道，我爸会一直看，我姐会看，我女朋友的姐姐还 会读我的文章给她上小学的儿子听。

做产品的人，你要问他：你自己做的产品，你敢给你爸妈用 么？

写字的人，你要问他：你写的东西，可以给爸妈看么？可以 读给你未来的孩子听么？

为什么要这样问，因为贩毒的人不吸毒。 写自己信的东西，写不怕自己的亲人看的东西，我觉得这就

是个很好的标准。

网上有很多人写的东西，恐怕自己都不信，也更不愿意让自 己的亲人成为读者。

这个世界上的人，其实就两类人。你的亲人，和别人的亲 人。

所以，即使你的亲人不是你的读者，你也要把读者即别人的 亲人当做亲人看，也即是，写字的人要对读者负责，你不能 乱写，瞎写，因为你写的每一行字，都有可能改变一部分人 的想法行为，所以你写的字应该是能够让这个世界变得更美 好的。
