# 《没有边界感的人最

丁丁 : 《没有边界感的人最讨厌》 女儿现在四岁半，个子比较小，但头发很长，不扎辫子的时

候几乎已是长发及腰。

每次视频的时候，奶奶都会说：“哎呀，头发怎么那么长了， 该剪短了，不能叫头发那么长。”女儿每次都直接回答：“我 喜欢长头发。”

上次来北京的时候，见面又说头发的事，女儿都嫌烦。然后 奶奶还郑重其事的给我支招说：“你就趁她睡觉的时候，用剪 刀直接给她剪了。”我不好直接反驳，只能笑着说：“那您孙 女醒了还不得和我干仗？她自己愿意留着长头发。”

这次回老家，有亲戚一见到我女儿，也是立马来了一句：“你 这头发也太长了吧？该剪了该剪了。”心里忍不住奔跑过一万 匹草泥马。

我女儿的头发，无论是梳头还是洗头，一直都是我打理。梳 头或者洗头，她从小就很配合，从未让我感到一丝不便。她 喜欢留着长发，我也喜欢看她长发飘飘的样子，我们两个人 在这件事上高度默契，却不知为何，总引来旁人的操心。

我们的生活中，也经常会遇到类似的情况。这样的人，往往 没有边界感，不经允许地乱闯入别人的领地，强行干涉他人 的生活方式，恐怕是在生活中令人最唯恐避之不及的一类 人。

这样的人，最令人讨厌。

1、没有边界感的人，缺乏对他人的尊重。

在这类人眼中，可能根本就没有尊重他人的意识。特别是作 为长辈的一方，更倾向于把小辈当做附属品，而非独立的人 对待。

相信很多人小时候都有过这样的经历：妈妈不经你同意，就 直接把你的东西送给了别人。

我小时候，我妈曾经偷偷地把我很喜欢的一对扎头发的皮筋 送给我的一个表姐，之所以偷偷地送，是因为我已经明确表 示了不同意。几十年过去了，我还对这事记忆犹新，记得自 己那天泪流满面的情景。

即使长大后，有时也难以逃脱这样的事。 没要孩子之前，我也全职上班。每年婆婆来北京住的那段日

子，都会趁我上班的时候把她儿子的衣服从衣柜里搬出来重

新整理一遍。从来不和我说。

我一向比较心细，婆婆只要动一下东西我就知道。我不反对 婆婆在家里做整理，但是这种不经允许地乱动，让我有种深 深地被入侵感。

更可怕的是，有时候这种没有边界的入侵行为，是包裹在“为 你好”的漂亮外衣之内，反而显得你不领情，当事人还满脸委 屈。

因为没把对他人的尊重放在心中，所以才会那么理直气壮。

2、没有边界感的人，喜欢对人指手画脚，高高在上。

这类人，往往倾向于认为自己掌握的知识或者技能更正确， 看到别人和自己做的不一样，总忍不住指手画脚。

比如我自己母乳喂养的事。 我因为是全职妈妈，母乳喂养比较方便，一直到女儿 42 个月

（3 岁半）的时候，才因为需要服用抗生素而停止哺乳。

从女儿一周岁开始，就会有人听到我还在母乳喂养而惊呼劝 告：“怎么还在喂奶？可以停了喂奶粉了。”我心中满是不 屑，不过脸上还笑盈盈的：“我家穷，国产奶粉不敢喝，进口 奶粉喝不起，还是只能母乳了。”

到了女儿两周岁后，听说我还在喂母乳，有些人的脸色都会 变两变：“唉呀，怎么还在喂奶？这个时候奶就没营养了，不 要喂了，不要喂了。”你倒是给我拿出母乳没营养的证据来！

到了女儿三周岁，就有人表情异常地说：“这么大还在喂奶？ 听说这个时候奶都有毒了。”我差点没背过气去。所谓的信口 雌黄是不是就是这个样子？

是否母乳喂养，喂养到几岁，是件再私人不过的事，个人完 全可以根据自己的实际情况决定。那些对我指手画脚的人， 仗着自己年长一点，经验多一点（好吧，其实还有些人根本 就没有喂养经验），跑过来指手画脚，真是除了让人厌烦外 起不了任何作用。

3、做个有边界感的人

我相信，没有人想做让人讨厌的人。

我曾经有段时间，也喜欢对人指手画脚，老是给别人提建 议，还沾沾自喜不自知。

直到有一天，我突然意识到，本质上，这是一种越界。所以 现在，除非别人亲口提出，我绝不随便给建议。

别人是不是喜欢你，你决定不了；但做个有边界的人，至少 能让人不讨厌你。
