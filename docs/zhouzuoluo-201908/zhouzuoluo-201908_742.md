# 《高小羊：如果在做

《高小羊：如果在做和不做之间为难，立即去做》

今天得知了一件事，让我惊呆了。 我前公司一个同事，发微信跟我说，年后忙成狗，分分钟想

走人。

又一次，跟我吐槽工作忙这件事。 我惊讶的不是她吐槽的这件事。 而是，她喊了大半年的离职，竟然还没有行动，还死吊在那

里。

从 18 年的 8 月份，她就告诉说国庆回来后就提出离职。结果， 国庆回来后，反倒是我，在假期想好了以后的路，提出了离 职，她却没有，然后一直拖，拖到现在。

中间有好几次发微信给我说，等拿了本月工资就走，到后来 等拿了年底奖金就走。

我没问她现在年底奖金到底有没有领到，我觉得我已经没有 多大的耐心去问了。我一问，估计她说等看四月份调薪幅度 决定要不要走，或者等到劳动节回来后再看…

这就是明显的对现状不满，却又不敢做出行动来改变现状， 而是一味地向别人倒“苦水”，寻求安慰自己的犹豫寡断，来

试图解释自己有这样那样不能离开的原因。

结果原因一堆一堆的，自己心里又不舒服的做着当前的工 作，每天都在这样的状态下工作，能做好工作，过好生活 吗？

她的工作我是了解的，每天从早上上班，到晚上下班，一直 在拍照修图。每天加班到晚上八九点，周末还要加班，全都 是无偿。公司三个部门的产品要拍照修图修，全部由她一个 人来做，之前上级答应她要给招聘新人来分担工作量，到目 前为止还没有到位。这期间她除了拍照修图外，还要设计产 品包装，这个最费时间，如果要设计一个包装，至少要两三 天。两三天过后，就会堆积大量的产品要拍，要修。

她的工作永远是个死循环。 她是有硬技能的，就是设计。她平时也报了一些线上课程在

学，大学学的也是这块，她本人也是想从事这方面的工作。

当然她平时的工作也有涉及到这块，但是按照工作量来说，

拍照修图占了一大半。

她不止一次跟我提到说要去深圳从事这方面，然后每次在谈 话后面还要加上：担心深圳的房租吃饭水电费…

她之所以不敢做出决定，就是不舍和胆怯。

1，不舍目前公司的稳定的薪水，怕到深圳找不到工作，无法 生存。（她本人陈述的）

2，不舍东莞这里的低生活费用，怕到深圳生活费用太高，无 法生存。（她本人陈述）

3，思想上的巨人，行动上的矮子。怕做出改变，适应了一天 天无所谓的工作，熬一天是一天。东莞有熟悉的朋友，有熟 悉的环境。到了一个陌生的地方，一切都是陌生的，怕适应 不下去，怕融入不了其中，无法生存。（我说的）

假如一个人脑海中产生一个想法，证明是有一些客观原因导 致他产生了这个想法。如果他不采取行动，让这个想法变成 具体的行动，或者让这个想法彻底消灭，它会一直盘绕在脑 子里，时刻都在影响他目前做的事。

这个时候，他要立刻做出决策，是去还是留，早点定下来， 不然被他耗掉的是一去不返的时间，和永远不会改变的现 状。

浪费时间最可耻，浪费时间就是在浪费生命。 我最痛恨浪费时间的人和事。 如果一件事你不确定做不做，建议你立刻去做。 比如，我晚上有时候想去跑步，但是又有那么一点纠结，这

个时候我肯定会去。

因为我知道，如果我今晚不去，我就会一直耗着时间东摸西 摸，最后啥也做不成，然后肯定会后悔，因为我脑子里想去 跑步的比例要稍微大一点。

我这个同事，明显想离职比不想离职的比例要大很多，却被 一些未知的问题和胆怯的心理阻碍着，一直不敢往前走。

一件事你在纠结做还是不做的时候，请你立刻去做。

浪费什么都不能浪费时间。
