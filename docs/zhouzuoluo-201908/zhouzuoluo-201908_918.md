# 《被动善良是一种选

智多星 : 《被动善良是一种选择，主动善良是一种能力》 张小龙在演讲中，讲到微信的原动力是①坚持做一个好的，

与时俱进的工具。②让创造者体现价值。

我觉得这背后的驱动不只是价值观，更是一种善良的能力。 每个创业者都可以说我是为某个群体，或为全人类，这个社

会创造价值，解决需求，然后这个产品创造出来，能赚钱

了，收获越来越多的流量，这时候能不能守住所谓的初心，

就看你有没有主动善良的能力。

1、我知道这是错的，所以我不做 罗辑思维的公司原则一直坚持，“不作恶、不造假、不行

贿”。

这一点也深深影响了小马宋，他的公司对外数据绝不造假， 战略规划绝不含糊。如果对客户没有帮助，他会主动解约， 因为他认为诚信是合作的基础，不能昧着良心赚钱。“我需要 钱，但我觉得这样挺好”。

有原则地做事，这是一种社会信用资产。别人能不能相信 你，你的交际圈有多高的质量，取决于你的底线在哪。

罗振宇本来担心找张泉灵合作，她那么忙，会不会三心二 意，后来就有人跟他说，像张泉灵这种级别的社会人物，最 注重的就是信用资产，如果她答应了做事，就会全心投入。

积极培养自己的信用资产，做一个主动善良的人，不赚小钱 和快钱，挖更大的宝，开更宽的流。

有些人宣扬知识付费，是为了制造焦虑，收割韭菜，但有些 人是真心实意做这件事，他希望能尽他的力量创最大的价 值，虽然这个外壳是商业形式，但没有法律和人可以要求做 事不收回报，而且商业往往能比慈善创造更大的价值。

有人觉得善良是对的，是不用做但你选择做的事；有人觉得 善良是有用的，能帮你赚钱的。每个人的驱动力都决定他跋 涉的轨迹和终点。

张一鸣洞悉用户心理，让用户沉迷产品，不能自拔；张小龙 尊重用户需求，让用户用完即走，提高效率。

一种套路的“善良”，把玩人性；一种主动的善良，引导人 性，分别决定了微信和今日头条两条路子、两种走向。

2、我坚信这是对的，所以我要做 秋叶大叔说，迪士尼这么成功，是因为他们的思维不同，用

笨方法做大事。

每个游客都能在迪士尼，体会到入梦的愉悦，这背后是每个 员工的全情付出。一般人会想，迪士尼肯定投了一大笔培训 费吧？员工的素质要求很高吧？

实际不是这样的，迪士尼的管理很独特:

①告诉员工，“把每一天的工作当成一次演出，只要进入乐

园，面对游客，你就是电影世界的某个角色”

比如，来招聘的人被选上，HR 不会说你应聘成功，而是告诉 你“恭喜，入选 xx 角色”。

整个管理层用了电影导演的思维，指挥每天工作的进行，让 员工和游客同时入梦

②让自己的员工获得好的内部服务体验。

“你对待合作伙伴和同事的方式，最终会关系到那些同事在你 不在的时候对待你的顾客以及与其互动的方式。”

在迪士尼，每位员工都能感受到公司的尊重和信任，在这种 团结友好的工作氛围里，他们更愿意相信是为自己工作，而 不是被动服务于人。

迪士尼至今都是全球最大的造梦工厂，每年有 1700 万的游客 为其购票，这就说明他们的笨方法才是真聪明。

而后继者只学到皮毛，用精明的房地产商思维，造出一个乐 园，就马上让周边地皮涨价，“除了制造焦虑，大家只关心赚 了多少钱”

思维和思维的不同，决定一个人是做正确的事，还是只做有 钱的事。

有钱的事，谁都愿意做，但正确的事，还需要你足够善良。 顺丰是中国快递的龙头企业，它能做大做强，谁都不会怀

疑，因为处处都是细节，拿最前线的快递员来说，顺丰给他

们家的感觉，既有实际的红包在手，也有换位思考的贴心服

务。老板把员工当兄弟，员工拿实事来回报。

很多人说，王卫很有头脑，把企业管理得这么好，但其实人 家是拿真心去换，该讲利润的时候讲利润，该善良的时候真 善良，既有眼界和胸怀，更有主动善良的能力。

3、每个人都有作恶的机会和能力，选择不作恶是做人底线。 我最近看到一段话，“权势是相对的，每个人都有权势。即便

在家里，相对幼儿和老人，也有权势。所以，每个人都有机

会作恶”

不管是以单位构成的社会，还是以个人构成的家庭，只要有 人的地方，就会有权势、压制和不公，有制度上的漏洞和人 性的阴暗。

所以，不要觉得作恶离我们的生活很远，总会有某个时刻， 老天爷想作弄你一次，在你松懈软弱的时候，把诱惑的钥匙 放在你面前，让你能轻易打开罪恶的大门。

又或者，在法律的死角和畸形的社会习俗里，用相对的权 势，做相对的恶，虽然不会有人谴责，不会有制度约束，但 有些观念本身就是错的，不道德的。

我们每天和人、和环境的相处都是一次修炼和考验。 你给老人让座，不乱扔垃圾，可能是舆论的压力，但如果出

于自愿，那就是主动善良的能力；

你对下属一视同仁，和颜悦色，可能是制度的要求，但如果 出于自觉，那就是主动善良的能力；

你对工作忠于职守，不急功近利，可能是环境的约束，但如

果出于本心，那就是主动善良的能力；

你对家人不提那么多要求，不用言语暴力和动手制压，可能 是一时的忍耐，但如果出于真心，那就是主动善良的能力。

主动善良不只是一种价值观，更是做人的基本要求。 红黄蓝的虐童事件、食堂里的发霉食物、造假的狂犬疫苗、

花总的走投无路、杨永信的人间地狱、被保姆殴打到全身是

血的老太太...这些都是触目惊心的社会现象。

有些人依然无助，变成可怜和不幸的牺牲品；有些人仍在社 会上八面玲珑，恃强凌弱。

人间很值得，是我们还需努力。多一分善意和自觉，多一点 真诚和尊重，对人对事对自己，都是负责和心安。
