# 大家好，我是吉水，

大家好，我是吉水，这是我的第 31 次分享希望大家多多指教

《去尝试吧，别让恐惧困住你前进的脚步》 一直以来，我都是一个偏内向的人，慢热、胆怯，在身处的

集体当中很少有存在感。

直到大二时的一次演讲比赛，让我开启了自我认知与改变的 道路。

从不敢参加任何比赛的我，那一次突然逼着自己一定要去尝 试一下，尝试着把自己“抛到台前”看看究竟会怎样？

结果没像之前担心的那样，稿子写不好，站在台上双腿发 抖、讲不出话。

反而，在台上的时候我无比的享受那个舞台，享受观众和评 委老师投来的目光，享受展示自己的过程。以至于整场演讲 完成的非常流畅，并顺利的拿到了单人组一等奖。

那一次经历给予了我足够的信心，并支持着我在许多方面做 出了尝试和改变。

但并不足以彻底扫清恐惧。

【[爱心]01 跳出恐惧，勇于尝试】 最近，新一届的省师范生技能大赛已经开始校内选拔了。我

在一个星期前就已经接到了通知，但迟迟没有报名。因为又 陷入了恐惧中。

尽管我们在学校已经上过了实训课，尽管我已经通过了教师 资格证考试，但还是不敢报名。担心讲不好，担心拿不到名 次，担心被老师批评。

就像很多人想考研，但是怕考不上；想每天跑步，但是怕长 肌肉；想谈恋爱，但是担心分手……

但身处恐惧中的我们，都忽略了一点：我们还没有开始尝 试。不试一试怎么知道自己不行呢？

读《刻意学习》时，作者 Scalers 说：很多人都喜欢使用复句， 先描绘一个美好场景，然后用一个“但是”转折成一地鸡毛。

试问有什么比我们啥都不做更可怕呢？ 去尝试，会有两种可能，第一种是成功了，这时你会更加有

信心；第二种是失败了，但你一定收获了经验和过程。而如

果不去尝试，就只能原地踏步。

把时间线拉长，如果我们一直在做出尝试和改变，五年十年 之后就会产生一定的质变。到那时，再回看现在的失败和挫 折，你一定会觉得云淡风轻。

跳出恐惧之后，我抓住最后机会，填写了报名信息。

【[爱心]02 拥抱恐惧，方能获得自由】 之前的若干次尝试和改变，或成功或失败，都增强了我的信

心和毅力。但那些经历并没有帮我扫除恐惧，这一次师范生

技能大赛到来时，我还是不敢果断的报名。

同理，相信即便这一次比赛取得了优异成绩，下一次我依然 会害怕。

恐惧或许会一直萦绕，就像每一次比赛、上台、考试等，参 加重要场合时的紧张心情一样，无法彻底扫清。

既然无法扫清，那就不如去主动接受它。只要我们有所选 择，有所控制，掌握主动权，不为恐惧和紧张所困，并加以 刻意的练习和尝试，这种恐惧感便会慢慢减少。

否则，如果我们一心想要扫清恐惧，那我们的眼里便都是恐 惧，而看不到任何希望和机遇。这时，我们便会被恐惧困 住。

结语：接受恐惧，跳出恐惧，去尝试吧，别让恐惧困住了你 前进的脚步。要知道，最可怕的是你什么都不去做。
