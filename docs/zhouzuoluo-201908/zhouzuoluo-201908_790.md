# 《祝大家新年快乐心

《祝大家新年快乐心想事成！附达成目标的方法》

回望过去的一年，最大的收获就是真正走向了新媒体之路， 这个行业跨度，是有点让亲戚朋友们惊讶的。

我开了六年的服装店，甚至直到今天还有老顾客问我“啥时候 再开店卖衣服”。我有点哭笑不得地答她，应该不会了。

我过去开店只想着赚钱，对物质生活考虑得比较多，对人生 的思考其实非常少，精神世界匮乏。

偶然看到粥老师的文章，才忽然有了学新媒体的初心——多 赚点钱（那还是因为感觉实体行业越来越不行了）。

在我最初的印象里，要想靠新媒体多赚点钱，普通小编当然 不行，必须得是牛逼的新媒体人，比如说像粥老师写文章一 流的。

身边常常有朋友会说“怎么可能呢？”连我亲妈都吐槽“你野心 太大了吧？”

其实我也怀疑过自己：第一、我好多年不太阅读和写作，基 础差起步晚；第二、就业的环境差，我该如何真正走向这条 路？；第三、没有任何资源，我该如何学习如何精进？

思考之后，我给自己定了非常明确的目标——找新媒体行业 相关的工作锻炼自己。

为了找到工作，那必须还得有一点“混饭吃”的基础。比如说 公众号排版、简单的文案、基础的写作能力（不论好赖，至 少能限时完成一篇文章吧）。

我当时电商运营工作，下班后学习课程，做了一个自己的公 众号。

温馨提示：如果你也是从零开始，想找一个新媒体行业的工 作，最好的办法就是做一个自己的公众号，这是一个展现自 己实力非常好的方法，亲测有效。

定了目标，找工作先不要太在意薪资待遇，而是先考虑是否 符合对达成目标有帮助。2017 年底，我的老领导邀请继续跟 他工作，薪资从 3800 直接涨到 6100，我直接就拒绝了，而是 找了一份底薪只有 3000 的文案工作。

我觉得这是一条特别重要的原则：坚持把目标放在第一位， 其他条件置后。

后来文案工作的薪资涨得算很高了，但当地一家专做新媒体 的公司挖我，但试用期的底薪没有文案工资高，福利待遇也 差一些，我还是决定跳槽。

我觉得专业的同事在一起，可以互相学习，了解更多新媒体 知识。只可惜，我后来失望了，因为他们既不够专业也不够 努力……

这都不重要，互联网时代知识是不存在地域差的，只要你愿 意，可以在线上挑选到很好的课程去学习，比如说粥老师的 公众号运营课、写作课。

截止到今日，距离我的目标其实还有很长的路要走，但我很 开心已经走在一条正确的道路上了，并且是目前对我来说最 好的一条道。

新年伊始，一定很多人会给祝福你，心想事成。希望我的经 验能给你一些启发，要“心想事成”，首先就是要“瞄准”目标， 有时可能需要你舍弃一些旁的东西，坚定不移地朝着目标去 走，你才可能真的“心想事成”。
