# 大家好，我是阿楠。

林 :

大家好，我是阿楠。2019 年，为了提高写作能力与思考能

力，我决定每天写一篇文章，希望大家监督并指正，这是我

的第 34 篇文章。

《做事前请先问自己：我真的力所能及吗？》

今天文章的主题是——做事前请先问自己：我真的力所能及 吗？

1.力不能及却去办，容易好心办坏事 之所以想写这个题目，源于前两天我好心办了件坏事。 我到一个区县办事，回来的路上坐公交。忽然呼啦啦上来一

群大妈，她们吱吱喳喳地聊天。由于声音太大，我听到她们

是在讨论坐的这班车对不对。过了一会儿，她们似乎确定

了，就各自找座位坐下，其中有一位就坐在我身后。

公交车缓缓往前驶，过了个红绿灯后就向左转到了另一条路 上。

我身后的大妈突然哎呀了一声，然后喃喃自语道：“这车怎么 向这边转了，难道我坐错了？不对呀，我上次明明坐的是这 路车呀。”说完还四处张望，显得十分着急。

我看到车辆一刻不停往前驶，虽然对这片区域也不熟，但想 着我有手机导航，还是帮不认路的老大妈一个忙吧。于是我 拿出手机，把公交线路的站点图调了出来，转身对大妈

问：“您要去哪儿？我帮您看一下。”

设想中，她会说出一个公交站名，然后我对照着手机上查到 的公交站点图，看有没有她要去的站，就能知道她是否坐错 车了。

谁知她并没有直接回答我公交站名，而是说了一个大型商场 的名字，这一下子把我整懵了。我迅速扫了一眼手机的公交 站点图上，并没有以她说的这个大型商场命名的公交站。于 是不确定地对她说：“好像这车没有这一站，要不您问问司机 弄清楚，要不你们就先赶快下车，免得往回走要很远。”

大妈是个急性子，她听我这么一说，并没有问司机，一招呼 她的朋友们，一群大妈又呼拉拉地在最近的一站下车了。

正当我觉得自己做了件好事时，转折发生了。这时，旁边的 一位大姐告诉我，大妈说的那地方，应该在某某站下车走几 步就到，她们没坐错车。刚才这位大姐也想告诉她们来着， 但大妈们风风火火走得太快，没来得及张嘴她们就下车了， 只好作罢。

车行驶了一会儿，果然如这位大姐所言，在大妈们说的大型 商场旁停住了，只不过站名的确不是这个商场的名字。我目 瞪口呆，脑海里回想起大妈们在寒风中下车到路边重新问 人、等车的情景，才知道自己给人家说错了，好心办了坏 事。

想想也是，我只是来办事的，对这片地区本来就不熟。在力 不能及的情况下，要帮人指路，看似好心，其实却容易给人 误导。所以下次有别人问路，先问问自己是否对地址绝对地 熟悉。如果你自己也不确定，你可以说不知道或不清楚，但 不要猜测着给人指一条路，万一那是错的，就可能更耽误别

人时间。

2.力不能及充胖子，容易轻诺寡言

《道德经》有一句充满哲理的话，曰：“夫轻诺必寡信，多易 必多难。”

意思是，那些轻易发出诺言的，必定很少能够兑现，把事情 看得太容易，势必遭受很多困难。

生活中，我们常会碰到朋友需要帮忙。很多人有时即使力有 不及，但碍于朋友情面，一般会答应帮忙。又或者没仔细考 虑，不管三七二十一先答应下来再说。但落到现实情况时， 有很多事情往往不像表面看起来那么简单的，否则朋友也不 至于要求人帮忙。所以当我们真的去做时，发现自己也实在 完成不了，只好跟朋友摊牌说做不了了。

这样做，可能比一开始不答应更容易失去友情。因为如果你 开始不答应，朋友可以尽快去找真正能帮忙的人，说不定事 情已经解决了。但你答应下来了，他放弃了其他机会而满怀 希望地等着你，你到头来又说办不成，一是耽误事儿，二是 你在朋友的心中也会贴上轻诺寡言、办事不靠谱的标签。

《论语》当中，孔子和弟子子路也有一段很有意思的对话。 子路问：“老师，如果您率领军队，会找谁共事呢？” 孔子曰：“暴虎冯河，死而无悔者，吾不与也。必也临事而

惧，好谋而成者也。”

这句话什么意思呢？意思是：赤手空拳和老虎搏斗，不用船 只去渡河，这样一介武夫，死了都不后悔的人，我是不和他

共事的。我所共事的是那种面临任务便恐惧谨慎，善于谋略 而又能完成任务的人。

从这句可以看出，在孔子的心中，遇事谨慎、深思熟虑的 人，才是值得信任的。而做事不经大脑、好呈匹夫之勇的 人，不能与之共事。

这和我的一条人生经验不谋而合。遇事托朋友帮忙，如果对 方问都没具体问，就一口一个喊着哥们没问题，答应得异常 爽快的，十有八九是不靠谱的。如果对方没有马上答应，而 是问了一些具体的问题和情况，然后再说我要想一想改天给 答复的，如果他最终答应帮忙，多半是能将事办成的。

所以，下次无论是别人找你帮忙，还是你找别人帮忙，都在 心底想一想“力所能及”这四个字。量力而行，方能长守信 誉。

3.力不能及不自知，容易影响整个集体效率，甚至付出沉重代 价

生活中，还有一些人力不能及而不自知，自以为自己能力很 强大，遇事爱毛遂自荐，这就很令周围的人痛苦了。

我有个开公司的朋友，他有个下属就很令他头疼。这下属说 他没能力吧，也有点能力，这些年来干出过一些业绩。但缺 点也不少，办事的效率不太高，且有点粗心，业务能力跟真 正的精英比起来还有挺大的差距。

偏偏这下属挺爱表现的，公司有什么新项目都爱往前冲。我 这朋友有时吧，也不太好打击下属的工作积极性，就把任务 给他了。但给他了之后，往往要安排一堆人为这下属的拖延 症和粗心善后。

这下属就是典型的力不能及不自知，拖累了整个集体的工作 效率。

有时，力不能及而不自知，付出的甚至是生命的代价。 有个成语叫“纸上谈兵”，说的是战国时期赵国的将领赵括。

赵括是名将赵奢之子，年轻时学兵法，谈起兵事来父亲也说

不倒他。虽然没有实战经验，但自信心膨胀的赵括认为自己

熟读兵书，论领军打仗的本事肯定已是天下无敌了。后来，

赵括接替廉颇为赵将，在长平之战中，死搬兵书上的条文，

完全改变了廉颇之前的作战方案，结果被秦军大败，四十多

万赵军尽皆被秦军坑杀，他自己也被秦军箭射身亡。

这场历史上大名鼎鼎的“长平之战”告诉我们，力不能及而不 自知，不仅会害了自己性命，还害了四十万士兵的性命，教 训不可谓不惨痛。

不要以为今天是和平年代，力不能及办点事关系不到生命。 我有一次住出租屋里灯坏了，叫了一个邻居帮我修，他半懂 不懂的，用螺丝刀一弄电路，差点没把我屋子里的电线全弄 起火了。搞得我赶紧把他请下来，找了专业的人士来维修。

同样，我们在各种工作岗位上，有时就是一次自认为懂的尝 试或粗心，少弄一枚螺丝，多按一个开关，就可能会酿成危 及生命的大祸。

因此请记住，下次做事前先问问自己：我真的力所能及吗？
