# 最近看的《跃迁》一

最近看的《跃迁》一书中，学到一个观点：爬问题树而不是 爬知识树。

我们由小白到专业过程里，建立自己完备的知识体系是非常 必要的。

那这就绕不开“学习”这个坎了。 如何学习？

如果把相关领域的知识全部学完，那我们一辈子也学不完， 一辈子也达不到专业。

我们要善于寻找当下的问题，根据问题来寻找需要的知识输 入。

就是所谓的爬问题树： 如图 1。

根据你当下遇到的问题，去查询所需要的知识。 这样建立的知识体系，是对你的生活成长真正有用的。 而不是看着十分庞大，但是对你却丝毫没作用的花架子：百

度里屯着那么多知识，没有查不到，只有想不到，但你用得

着么？

但是，寻找问题，也不是一件容易的事。

很多时候，我们甚至都不知道问题出在哪儿？ 没关系，一步一步来，提问可不是一个简单的活儿，这是我

们需要一生去打磨的技艺。

至少，我可以给你提供几个思考问题的方向： 描述一下你最近碰到的糟心事儿。 描述一下你未完成的小目标，为什么完不成？ 描述一下，你想做又不敢做的事情，问问自己：你怕什么？ 回忆你上次想自我改变的时候，你想改变什么，为什么现在

依然没有改变？

等等，这些方向，我想已经够你挖掘出一大堆问题了。 不用担心，挑自己感兴趣的，能对生活产生正向影响的问题

开始解决。

这样做，即搭建了对自己有用的知识体系，还满足了自我价 值实现，又兼顾了生活。

以上。
