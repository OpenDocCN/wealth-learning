# 《真正聪明的人，懂

《真正聪明的人，懂得欣赏别人的优点》

训练营做到第二期，有一些发现。就是一部分人，会非常难 去欣赏别人。

比如说：总觉得别人的作业好像还没有自己好，为什么被老 师选为精选作业，而自己却没有。这个一定是老师偏心，一 定是加精太容易了。

一遍质疑老师偏心，一遍吐槽同学的作业优质度不够，还一 边要求减少加精的作业。

哎呀妈呀！可真够忙的。

1、偏心？不存在的 一个妈生的孩子，有可能偏心长得好看更乖巧的那个。这几

百人个人，谁也不认识谁，谁也没见过谁，谁也不多给老师

一毛钱，偏心真的不知道从何说起。

2、别人写的优质度不够？自己够？ 读@孤独大脑老喻的一篇文章，有提到一个现场，就是人们

常常容易陷入一种错误的认知，那就是“我一定比别人好”。

比如说看到别人搞砸了一件事，心里就想：让我来做就肯定 不会搞砸啊，那个傻逼；看到人家开垮了一家店，心里就

想：让我来开就肯定生意兴隆啊，那个败家仔。看到人家写 了一篇十万加文章，心里就想：哎呀！马马虎虎嘛，这种文 章我也可以写呀；看到人家写的作业别加精了，心里就想： 靠！这老师什么眼光，写得还没有我好呢。

看低别人，高估自己，其实这都是蛮正常的现在，正因为如 果才需要时刻提醒自己，不要陷入这样的认知偏差中。

老师给了一些同学加精，那么一定有他比较好的部分，比如 说作业特别认真，或者某一个点比较好。人家写了十万加， 自己没有，那么可能是标题差了那么一点点、选题差了那么 一点点、转发朋友圈的动力差了一点点。

多去发掘别人好的地方，而不是发现了别人不如自己的一个 点，就吐槽：哎呀！那个傻逼……哎呀！老天太不公平 了……

3、减少加精的数量？这就扯淡了 这就有点好想说，我周围都是高楼大厦，偏偏自己家住的是

平房，就希望政府给人家贴个违规建筑，拆了还不补钱哈。

狐狸吃不到葡萄说葡萄酸不可怕，最怕它把葡萄藤都给拔 了，你们都别想吃。

4、真正聪明的人，懂得去学习别人的优点 我们之前都没有做过训练营，为什么能做得好呢？学习！大

量购买优质的社群，尽力去发掘人家的优点，然后收集起来

分析和学习。

过程中我会问一些同样参加社群学习的人，你觉得这个营的

运营怎么样？不止一个人给我的答案是：其实没什么运营 啦，这个就是自运营。

其实稍微有点脑子就想明白，怎么可能有靠着“自运营”就一 直优秀的社群呢？这里面一定有很多让人难以察觉到运营技 巧。真正的高手，不就懂得去发掘这些优点吗？

总结： 欣赏别人其实挺难的，我也常常会想：哦！他呀，没什么了

不起嘛。但是，只有学会去发现别人的优点，静下心来学习

别人的长处，才能帮助自己更快的进步。共勉共勉！
