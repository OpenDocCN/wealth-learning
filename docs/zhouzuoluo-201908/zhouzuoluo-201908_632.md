# 《北大硕士去送外卖

《北大硕士去送外卖的背后：一条路上的双面人生》

张根在文章中说：“我辞掉了坐写字楼的工作，去做了一名外 卖员。穿上蓝色制服的时候，我对着玻璃里面的自己拍了张 照片，心想：我现在已经来到了你告诉我的底层，你们还能 把我怎么样。”

是啊！中产阶级拿他毫无办法。 他可以矫情一下说“中产阶级的生活太焦虑”，然后翘着腿涮

着火锅，跟他的精英朋友感叹“现在还是很焦虑”；他可以在

月入只有 5718 块的时候，给在国外读书的女朋友转 5000 块买

打折的 Tory Burch，并调侃她“愿意嫁一个外卖员吗”。

是啊！底层更加拿他毫无办法。

他可以把送外卖的经历包装成一篇爆文，约稿和采访的人马 上蜂拥而至；他可以随时甩手不干，然后掏出漂亮的履历找 一份光鲜的工作，继续打着领带套着名牌西装，穿梭在各大 写字楼里。

一个人在奋斗的路上感到了辛苦，忽然想说要“淡泊明志，随 遇而安，闲庭信步，看雪望云”，人家过的是自己愿意的生 活，旁人也无权干涉。

但是只要他高兴，就能随时离开，并且完全有能力重新选 择。

就好像当富翁过腻了打渔的生活，可以去伦敦喂鸽子，可以 去看阿拉斯加的正跃出水面的鳕鱼，可以去逗梅里雪山爬上 树尖的金丝猴；可以欣赏西藏的一直盘旋云端的山鹰；可以 去尼泊尔当一个背包客，一起端起酒杯坐在火堆旁.....

当然渔夫也可以厌烦打渔，也可以选择喂海鸥；你厌烦了眼 前的苟且，也可以说走就走去寻找一下“诗和远方”。
