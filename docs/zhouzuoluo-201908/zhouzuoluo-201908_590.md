# 中年女人如何投资自

中年女人如何投资自己的人生？

01

这几天，静水在看吴军的《见识》，刚看到前两章，感觉受 益匪浅，对其中的观点特别认同，他在书中说到，年轻人最 重要的投资，除了投资自己很重要以外，比投资自己更重要 的投资，是找一个好的配偶。

作为中年女人，大部分人已经错过了第二个更好的投资机 会，那最好的投资就是投资自己了，如何投资自己？

我身边不少女人，但凡孩子慢慢脱手后，我发现对外在越发 在乎，脸上没动过刀子的不多，拉皮，隆鼻，隆胸，当然这 不能说不好，爱自己，就从形象开始，一点都没毛病，但我 吃惊地发现，她们剩下的日子就是重复，年复一年日复一 日，如果有个稳定的收入还好，没有稳定的收入的就无比焦 虑，在我看来，她们失去了一个极为重要的投资，就是投资 自己的脑袋。

困住一个女人的，从来不是年龄，而是脑袋，在静水的星球 上有个大姐，就是 ZIPPO 大姐，奔五的年龄，我感觉她很年 轻，对新鲜事物一直都保有极大的好奇心，在星球上分享的 感悟思想价值不凡。当然，她的这种精进精神，也给了她回 报，她创业多年，积累了厚实的家底。

02

谈生活品质，多数得有这个大前提，那就是有钱，古人云， 仓廪实而知礼节，说的就是基础的物质不满足，谈礼节有点 不现实。

无论你是 80 后还是 60 后，从现在开始，别哔哔，别做祥林 嫂，靠近优秀的圈子，学习优秀的思维方式，想办法想丰富 自己的口袋，然后不断投资丰富自己的脑袋，良性循环的人 生就会慢慢地质变。

从一个阶层跨越过去，真的不是一件容易的事情，我们需要 理性而客观地认识这个问题，所以，别抱怨你出身寒微，也 别抱怨你起点低，无论怎样，从一楼到二楼，还是有极大可 能的。

就像静水，一个农村黄毛丫头，通过高考，接受过高等教 育，和那些城市出身的孩子比起来，什么都算不上，但和自 己比起来，已经跨越了几阶楼梯，所以，我一直坚持这种思 想，不要和别人比，就和自己比，不把自己逼上绝路，也不 让自己荒废人生，保持不紧不慢的前进速度，量变终会引起 质变。

中年女人的时间的确很少很少，但无论有多难，一定抽出来 时间，来丰富自己的头脑。比如：你可以精进自己的文字功 底，也可以精进自己的厨艺，也可以精进自己的茶艺，根据 你喜欢的领域，工作之余，再创造一份价值，坚持下去，你 试试，绝对大不同，五年足以改变一个人。

03

有的姐妹告诉我，自己一看书就犯困，我直觉就是你的内在 动力不足，或者你从未想到坚持做一件事能为你带来的收益

或者改变，就像写作，你每天坚持五百字，你试试，用不了 一年，你的文字功底就会大进步，语言组织能力和思维表达 能力大不同。

周末，静水因读到《见识》，和大家友情分享一点想法，真 诚静水愿意和各位一起努力，不断精进自己的人生。艾特静 水 jingshuimm
