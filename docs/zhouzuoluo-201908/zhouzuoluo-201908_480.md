# 《你的写作给读者提

《你的写作给读者提供“新知”了吗？》

晚上好[玫瑰]今天是橙橙加入星球的 251 天 与星友们分享第 137 篇原创！

昨天写作训练营的课程，粥老师分享了好文章的 5 个标准，其 中第一个就是：“提供新知，创造价值”。

今天有同学在群里展开讨论：

①我们平时写文章，该怎样判断是否提供了新知？

②我们都知道文章要提供了新的知识、新的视角，才能受到 欢迎。但是，实际写作的时候，经常发现“自嗨”的情景，觉 得自己提供了新的价值，但是实际反馈并不好。

以下我整理了粥老师和助教们关于“新知”的分享，还有我 自己的思考以及获得新知的实践方法。

01

粥左罗关于提供新知的阐述：

写作不是自嗨，而是为了创造价值，想要创造价值，绕不开 的就是，你得提供新知。

新知，新很重要，你所表达的，若是人人都知道的，那读者

不必读你这篇文章。阅读是读者对自身的自我完善，他需要 从你的文字中汲取到，他身上本来没有的东西。

新知，知代表的是知识、认知，你要提供新的知识、新的认 知，但知也不仅局限于新知识、新认知，它也可以是新方 法、新资料、新故事、新视角等。

案例分析：

《湖畔大学梁宁：成就高的那批人，全都有同一种天分》

这篇文章为什么好，就是因为里面提出了很多人不曾有过的 新知。 比如：人人都想知道自己的天分，这篇文章提出：“人最重要 的天分是快乐”（惊讶：从来没有听说过快乐也是天分）

即要找到那件能让你一直不厌其烦地做下去的事，你不厌其 烦的地方，就是你的天分所在。

这就是新知，通过阅读这篇文章，吸收到了自己不曾有的东 西。

02

来自助教倾词的解答：

提供新知的前提在于能够博览群书，知道哪些已经成了旧 知。只有在广泛吸收的基础上，才能推陈出新。

那些我们一写出来就是 80%的人都看过的，其实不是因为你 没有把握新知，而是你没有见过新知。而且，写文章，尤其 是新手，最初写的阶段不能盲目追求提供新知。毕竟，你能 吸收知识写好就不错了。

到成长到一定的阶段，有过定量的输出后，就会有更多新的 感受，新的观点，新的名词，这些都是新知。

读的书多了以后，你能判断出你的水平和普通大众水平之间 的差别。

03

来自助教小墨老师的解答：

对于新知的判断： 第一，如果被提及率特别高，在别人文章里已经看到特别

多，泛滥的，就不算新知，这个叫老生常谈。

第二，以个人为中心写作，只要有什么是你之前不知道的， 现在知道的，并且触动你的，就算新知。

其实同一个观点，换一种说法，就成新知了，比如王阳明的 知行合一，韩寒换成“听过很多道理，依然过不好这一生”， 讲的是同一个东西，换个说法，加上个人化的体会，不就成 新知了

我比较建议以个人为中心的写作，因为每个人都有不同的成 长阶段，这个成长阶段感悟到的东西，对你非常重要，可能 等到你进入下一个成长阶段，根本就不值一提了。但是你不 能否认它的价值。

换个角度来说，一定有人正在经历你的成长阶段，他们还没 找到出路，那么你的思考成果对他们来说就是新知。

04

总结自己的写作是否有提供新知：

我在写作练习中，提供新知的写作也是我目前遇到地一块大 石头，但有的时候文章写着写着也觉得自己能提供一些新 知，或许这种新知就像小墨老师提出的第二种：以个人中心 的写作，来提供的新知吧！

①我还没有达到博览群书的阶段，我的知识面有限，正在成 长的初级阶段，我最想培养的是写作的整体水平，能系统的 掌握写作的基本要求。所以没有花更多的时间去研究某个 点，比如如何提供新知。

②在学习写作之前，只有输入没有思考和输出，而现在的写 作练习更多的是练习我的思考和输出。

很多知识点虽然对于你们来说是老生常谈，但对我来说经过 我的思考整理出来的东西就是新知，而我分享出来，是对我 输出的一种练习，我没有太过在意对于读者来说是否是新 知，但确实经过思考整理出来的知识点。

05

获得新知的实践方法：

①当然是博览群书，见多识广，跟上时代的脚步，能够敏锐 德发现热点。

②找个三五篇优质的文章一拆解，分析它的新知是哪里获得 的？如何转换成自己的知识点传达给读者？学会总结前人的

经验也能让我们少走很多弯路。

③当获得新知的时间成本较大时，可以先尝试以个人中心的 写作套路，去提供新知，再慢慢的积攒自己的知识库，依然 可以写出有价值的文章。
