# 《你总是给自己留太

《你总是给自己留太多退路，所以你一直没有进步》

如果一个人总有很多退路，那他无论做什么都很难拼尽全 力。他总会想着只要尝试一下就好，即使做不好还有别的路 可以走。因为缺少破釜沉舟的勇气与魄力，所以也很难取得 大的进步。

最近，乒协公布“史上最严”教练组考核办法。中国乒协备战 东京奥运会公布了 2019 年国家乒乓球队教练组名单及男女一 队教练组考核及奖惩办法。根据该考核办法，如男女队任何 一个教练组考核不及格，作为中国乒协备战东京奥运会战略 体系总指挥的乒协主席刘国梁将自罚全年薪酬，誓与团队共 进退。

说实话，虽然国乒成绩确实强硬，要求高一点无可厚非，但 这种近乎变态的考核办还是让人震惊。刘国梁自断后路的治 军手腕，真的让人不得不佩服。而当一支队伍或一个人孤注 一掷时，势必会激发出最大的能量。

而反观自己，总是给自己留太多退路。今天没读的书就明天 读吧；今天没练的字就明天练吧；今天没跑的步就明天跑 吧……我们总是一次又一次轻易地原谅了自己，所以一直都 没取得大的进步。

但是，二十几岁的年纪，我觉得不能就这么和自己妥协了。 不都说不逼自己一把永远不知道自己有多优秀吗，要想得到 更多，就要付出与之匹配的勇气与决心。总是差不多的思

想，只配得到一个差不多的人生。

提高对自身的要求，才有可能遇见更好的自己。千万不要在 最好的年纪就选择得过且过。
