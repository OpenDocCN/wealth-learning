# 谁说月薪 3000 的

静水 : 谁说月薪 3000 的人生一定穷？

01

“静水，昨天看到你发布信息，要征集真实素材，我想说说我 的故事。我今年四十岁，离异，月入 3000，儿子抚养权判给 了前夫，系四线城市某行政单位二级机构小公务员。昨天是 我四十岁的生日，没收到任何祝福，心里很不是滋味。我想 不明白的是，我都四十岁了，省吃俭用，为什么还这么穷。“

“我的同事，收入也不比我多，她们的朋友圈时常冒着香气， 晒旅行、泡温泉、做护肤 SPA 的九宫格。我这 3000 块钱，每 个月支付儿子 1000 元抚养费，父母身体也不好，哎，这都是 命啊。一样的收入，我的生活过得一塌糊涂，毫无品质可 言，无望过好这一生，真心难过。“

这段长长的留言来自河北一位读者，“我都四十岁了，省吃俭 用，为什么还这么穷“这句话很扎心。

人到中年，上有老下有小，下岗失业，婚姻危机，身心不 适……欺人的小鬼龇牙咧嘴扑面而来，毫无提防。哪怕对于 一个有着生活重担的失婚女人也不放过。

是的，必须承认是金钱限制了你的想象力。但穷为何不思 变，没有人愿意为你糟糕的人生买单。

谁不想飞黄腾达一步登天，谁不想腰缠万贯元气满满，谁不 想智慧打底坐实人生。可惜你爹不是李刚，你也不是王思 聪。

那是不是月入 3000，就真的意味着无望过好这一生？

NO，上帝从不辜负每一位用心热爱生活的人。 02

中国人的幸福观里有个奇怪的现象，也许是人性使然。TA 明 明正在享受当下的幸福，但若听到同层次的人比 TA 还幸福 时，TA 的幸福感瞬间被打折，

鸟语花香的季节，心向自由，来一场说走就走的旅行，想想 就醉了。再看看卡上可怜的余额，世界那么大，我却无能去 看看，实现财务自由简直就是他妈的做白日梦。这是多少身 处底层的人和冷酷现实 PK 后的无奈心声。

讲真，幸福的生活的确离不开钱。就像那些动不动就笔伐男 人无用，实则源于对男人期望破灭而愤怒的人一样，那些唯 金钱身外之物论者，多半来自金钱匮贬时的无能和自慰。

羞于谈钱的人要么自卑得要命，要么虚伪得要命，总之四个 字自欺欺人。

请记得钱永远都是为人服务的，沦为金钱的奴隶就失去了挣 钱的意义。但生活中从不乏这种人，不切实际地与人攀比， 自己气自己，一生都活在无休无止的欲望里。到头来发现钱 是挣不完的，飘渺的光环也没有那么重要。

了解我的人都知道，静水和张先生出身寒微，白手起家，是 不折不扣的穷 N 代。这么多年静水就像一名脚踏防滑鞋，双手 拿着探险杖的拓荒者，不断跋涉在寻求宝藏的路上。

03

入职第二年，我俩卡上余额共有 8000 大洋，我所在城市的商 品房，当年的单价也就一千出头，对应的楼盘还是高档小 区。因为太想在这个举目无亲的地方，有个属于自己的窝， 在众人的质疑声中我们做了买房的决定。同事瞪大眼睛质问 我是不是开玩笑的，翻译后就是你也敢买房太不可思议了。

没有多想，我凑齐一万块就交了定金，总房款下来才 155104.2 元，这个数字足以令我铭记一辈子。就这样我们成了单位里 第一对按揭买房的小夫妻。没有钱又刚需怎么办，静水认为 顺势而为也许是最好的选择，买房这件事开启了我研究家庭 理财的意识。

当然，生活还未起步，就为自己套上了枷锁，一个字，难， 两个字真难，毫无选择，唯有负重前行。坚持办法总是比问 题多的人生哲学，让张先生一味苦笑我近乎无知的勇敢。

简单铺完地砖，安装上坐便，入手两张简易床，我们就开始 了从出租屋到新房的“乔迁之旅“。

十多年过去了，事实证明这是人生中最明智的选择。如今半 年的薪水已经买不起这个城市的一个卫生间。很庆幸自己的 勇敢，首套刚需还个差不多时，入手了第二套商品房。如今 180 多平的宝贝足以成为我追求诗和远方的灵魂栖息地。两套 房子坐实了原始积累的基础，心中就会更踏实一些。

人，你要敢于打破传统，才能获得新生。不少财富的积累， 很多时候是踩上了时代的红利，和汗水并不对等，当然你得 有敢于选择的能力。

无论对于穷人还是富人，上天有一项恩赐特别公平，那就是

时间。不多不少，每人每天都有 24 小时。

不要小看这八小时之外的收获，你去坚持做一件事情，积累 的力量远远超出你的想象。

04

阅读与写作，是我与自己内心对话最好的方式。无论人生中 遭受到何种委屈，键盘上那种噼里啪啦的释放，会将我的灵 魂按摩得神采飞扬。

情怀之外，生存才是硬道理。婚后不久，大宝出生，养育孩 子的支出、房贷的刚性支出，孝敬双方老人的支出，挣钱的 速度似乎总是赶不上花钱的速度。这一度曾让我惆怅无比。

易经里有句话，穷则思变，变则通，通则久，几乎适用于各 个领域，更适用于脱贫这件事上。

八小时之外，财经专业出身的我，本职从事财税工作，最好 的兼职手段也就是代理记账或者去培训机构做代课老师了。 十年前，一个小时八十元的课酬对于我这个小白来说，堪称 高薪了。

那一年，靠着不断努力，我和张先生还清了亲戚朋友的刚需 房首付借款。记住，当时首付款都交不起。

辞去代课老师的兼职后，我又接了两家代理记账，一个月晚 上去两三次，当月的账面就整理得差不多了，这样一个月又 多了两份收入。也许你会说一个女人家何必这么辛苦呢。

借用上海黑社会老大杜月笙的一段话送给你——你原来是一 条鲤鱼，修行了 500 年跳了龙门变成龙了；而我呢原来是条泥 鳅，先修炼了 500 年变成了鲤鱼，然后再修炼 500 年才跳了龙

门。倘若我们俩一起失败后，那你还是一条鲤鱼而我可就变 成泥鳅啦，你说我做事情怎么能不谨慎呢？

别人的人生里，永远有你不懂的深情，我就是那个在以不同 方式兜售自己时间和价值的人。所以，你只需做好你自己， 莫在乎那些廉价的评论，但行前程，无问西东。

05

十年前我和张先生的月薪加起来还不到 5000 大洋，即便债务 缠身，我仍坚持做一件事情。每个月进行基金定投 300 大洋， 你千万别小看这三百块，资本市场上不是单纯以金额论收 获，还有来自积累的力量。

2007 年上半年第一波牛市来临，我赎回了三年的基金定投将 近两万块，加上提取的住房公积金，提前还了银行的部分贷 款。2007 年下半年进入熊市，继续定投，坚持到 2014 年春天 牛市再度来临，又是七年的积累，小有成就。

因为我入手的几乎全是偏股型基金，和大盘同步。切记做人 不可太贪心，4200 多点时，我赎回了全部的定投基金和分批 次买入的基金。贪心的玩家八月份以后只能哭了。放空后接 着继续定投，到现在又坚持了将近四年。

为提升小家庭的抗风险能力，我为每个家庭成员均投保了最 基本的商业险。八小时之外我热衷学习阶梯理财、基金定投 理财、也斗胆尝试过房产理财。因为实在看不懂中国的股 市，理财道路上唯独对炒股无趣涉猎。

跌跌撞撞的前半生，一副原始的烂牌慢慢地被我理出了头 绪。把拧巴的日子过顺溜是需要一点点耐心的。

N 年前张先生时不时会抱怨我说，小马拉大车的生活，拖着他 前进，真心累。现在小马成长起来了，车载能力自然就提升 了，偶尔加载一些额外的负担就不那么容易恐惧，日子自然 就好过多了。

漂亮的樱花 其实没有任何一项努力是白费的，就像地下的暗流，你也许

一时看不清流向，但坚持不懈地滋润生命，一旦机会来临，

定能碰撞出激流。

就像我写作，如果不是 N 年前在搜狐上写过几十万字的网文， 新媒体时代到来一百次，拎起笔墨我也很难这么从容。

折腾的灵魂似乎一直在路上，很多时候，在静水看来，理的 不仅仅是财，而是一种生活态度。

社会越开放，自我提升渠道就越多，赚钱的渠道也就越多。 收入少不可怕，不用心生活才可怕。掰着指头算计着省钱， 不如甩开膀子挣钱，然后合理规划再让钱生钱才是硬道理。

人到中年，谈钱色变，人生就不免有几分悲哀。请记得提前 用智慧淬炼和打底自己的人生，修炼自己的核心竞争力。做 一个能挣会花，掌控自己人生的女人。你会发现越是到重要 关卡，这种能力越刚需。

当今已经进入了互联网飞速发展的时代，当很多人感叹钱越 来越难赚的时候，互联网上单价一元的社群课，有人卖到了 一千万。

流量为王的时代，套用李笑来老师一句话——不会赚钱，是 一种病，得治。多么深刻的领悟！

静水告诉你，不是没有路子，是你没有找到圈子。举个简单 的例子，若你是单位的文秘，你可以在任何平台上开个号， 分享你的实用干货；若你喜欢做美食，你可以做成有品质的 独立包装，在圈子里兜售价值和情怀；若你有一套独特的记 忆方法，你也可以传授给很多人；哪怕你是家庭主妇，你可 以用心做上一两道拿手菜，在下厨房 APP 上分享，会得到意 想不到的收获。

总之，还是那句话，办法总是比问题多。不怕做不到，就怕 你连想都不敢想，心之所愿无所不及。谁有都不如自己有， 谁会都不如自己会，相信自己，你可以！
