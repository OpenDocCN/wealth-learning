# 为什么我把老公拉进

颖雪晓晴 : 为什么我把老公拉进星球？ 昨天我和老公一直聊到深夜两点，因为最近我们俩的共同语

言明显变少，这半年，我从县城来到省会，加入了粥老师的

星球，买了许多书，也接触了很多优秀的同龄人，可以说，

我已经不再是县城的那个只知道吃饭带娃的自己，我开始萌

生了进化认知，改变思维，开创副业，提升自己等很多很多

想法。

可以说，我现在到达了张辉老师提到的“发现自己无知而焦 虑”的状态，知道自己什么都不会，知道自己需要学的太多， 知道外面的世界很精彩，知道自己也可以活得与以前不一 样，所以我经常为时间不够用而焦躁。可是我老公还处于樊 登老师提到的“无知而快乐”的状态。他还是按部就班地拿着 死工资，一提到增加收入，就会想到求老板加薪，还没有被 动收入，时间复利，认知进化，黄金思维等概念。

所以我特别希望他跟我一起改变，去除曾经的落后的保守的 残缺的思想，一起打造新的人生。

可是我无论怎么给他说，他总是理解不了我的意思，昨晚， 我让他加入星球，他非得说，这个星球可以干什么，可以教 他什么？他现在以为的还是我既然花钱了，你就得教我一项 知识，否则我钱就白花了。而我现在的认识是，我花钱了， 你可以不教我具体的知识，你能打开一个世界的缺口让我 看，让我的认识，我的头脑有一点改变，让我发现到这个世 界比我想得更美好，更丰富，我的钱就没有白花。

从我老公身上，我就发现了很多人的影子，许多人不是能力

不够，而是思想有问题。我昨天问我老公，你大学毕业后， 有没有看过书？他说，看过。我问他，看什么书？他说，设 计规范，设计理论等等。我指的是提升自己的书，而不是为 了工作不得不看的书，照他的理解，人家买个玩具，看看玩 具说明书都可以叫看书了。

我问他，你渴望改变不？他说，渴望。然后我给他推荐一些 电影，一些书，让他看。可是，他又说我给他塞东西。我 说，咱们可以玩个游戏，在朋友圈，让认识我们两个人的评 论一下，在思想上格局上，谁更成熟一些？他又说，我不能 以偏概全，在他建筑领域，他比我强多了。

通过昨晚我和他的聊天，我就发现，他最大的问题是，认识 不到自己的问题，别人一说他，他立刻像刺猬一样，反击。 而真正想要变得优秀的人，会像久经干旱的小草一样，如饥 似渴地吸收甘霖。我始终坚信，男人的尊严不是女人白给 的，承认自己弱，承认自己暂时不优秀，没什么。

我想很多夫妻之间，都会遇见这样的问题，真正同频的，共 同成长的太少了，因为当一个人走得远了，走得快了，想要 去拉那个慢的人，那个落后的人根本不相信你说的，根本不 把手伸给你。所以，久而久之，两个人，越走越远。

我想起好朋友说的一段话，有两个井底青蛙，一个跳出去 了，告诉另一个，外面的世界很精彩，快出来吧，另一个不 但不出去，还说，井底下的水比外面的好喝，你下来吧。问 题是他根本没喝过外面的水。

纷纷洒洒，写了这么多，是希望对大家有一点用，不知道星 球里有没有夫妻？我昨晚成功把我老公拉进来了，希望他在 这里可以得到真正的改变。

希望大家支持我们哦！
