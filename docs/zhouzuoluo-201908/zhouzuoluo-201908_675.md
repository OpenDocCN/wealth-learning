# 《延迟满足，才是穷

《延迟满足，才是穷人最难迈过的坎！》

（一）你想改变，生活未必允许 表哥辞工在家学习淘宝卖货，40 多岁的他从来没网上购物

过，接触互联网少，学起来自然非常吃力，这几天经常在微

信上问我各种问题。虽然我质疑他的学习决心，担心他坚持

不了，但也挺乐意跟他聊。可是姑却打个电话来，让我劝表

哥出去找工作！

姑说表哥这些天，天天钻在房间里上网，都没心思去外边工 作了，这让她不安。要我跟表哥说，网上卖货这事一时半会 不会有效果，挣不到钱的，只能当个副业，还是要去打工才 行的。

接着她又吐苦水，说表哥压力不小，今年得还两万多的车 贷，房子借了人家的钱，今年也要还了，小孩也大了，不出 去打工挣钱怎么行？

可怜天下父母心，表哥这都 40 多了，六十多的姑却还为他操 心着。我答应劝劝表哥，要挂电话时，姑又跟我说了一 句：“别告诉他是我说的哦！”

这之前，虽然我对表哥突然辞工来接触网络很诧异，但这种 想要努力跟上时代，力求上进的态度我很喜欢。只是姑的这 通电话，让我意识到一个普通人既使想要改变现状，想去谋 求更好的工作和未来，其所处的环境和生活给的压力也未必

允许。

（二）即时满足，成了我们的习惯 我做夜景照明设计的，以前在广东找工作时，去了一家公

司，包吃住的。入住的第一天晚上，寝室的人告诉我，他们

已经两个月没有发工资了。听了这句，我第二天就走人了，

我不能忍受连按时发工资都做不到的公司。

可能是多年上班都没有遇到拖欠工资的情况，我一直觉得按 时发工资是天经地义的事，付出了一份努力，就得有一份回 报，而且这回报来得越快越好，最好不要压两个月工资，反 正拖欠工资是绝对不行的。我想不少职场人都有我这种想法 吧？

但也许正是因为这种工作一个月，就有一个月工资发的认知 深入骨髓，一旦遇到三、五个月没有收入时，我们就会非常 焦虑烦躁，难以忍受。

我记得刚开引航灯照明设计工作室时，虽然有充分的心理和 资金上的准备，但连续三四个月入不敷出的状态，着实让人 难受，每天如坐针毡，度日如年，一度怀疑自己的决定是否 明智，是不是太冲动，失眠还挺严重。

这种状态除了面对未知的恐惧，我觉得也受到上班时习惯 了“即时满足”的原因有关。所谓即时满足，就是无需多长时 间的等待，就能获得想要的结果。而习惯了即时满足的人， 最大的缺点就是对一件需要长时间积累的事往往缺乏耐心， 容易急躁，在等待中自控力极其薄弱，遇到挫折容易心烦意 乱。

这也是为什么很多人转行失败，做一件事难以成功，无法取 得较大成就的原因，因为我们在前进时无法获得即时满足， 自己的努力一时半会看不到回报，心态就变得不稳定，就开 始质疑自己，思想动摇，因而难以坚持。

（三）延迟满足，才能获得更大的收获 与即时满足相对的就是“延迟满足”，延迟满足能让我们甘愿

为更有价值的长远结果，而放弃即时满足的抉择倾向。

比如有人为了将来更美好的生活，选择在年轻时拼命学习与 工作。有人为了身体健康，不暴食，不抽烟喝酒。有人为了 达成学业，不看电视，不参加那些没有意义的聚会等等。

（当然，我觉得如果只是为了缓解压力，而去消遣娱乐一下 倒也无所谓。）

总之，延迟满足往往能让我们获得更长远或更大的收获。因 为取得大回报，往往需要时间的沉淀，短时间内往往只能获 得小回报。

当然短时间，也有能获得相对的大回报，只是这种回报未必 长久。要么是你运气好，要么是你抓住了趋势。比如自媒体 这一波红利，很多人就挣到了钱。但这波红利过了，大多数 也被淘汰出来了。真正留下来的，要么是每天拼命学习成长 的，要么是以前有文字功底，或者在纸媒、新媒体工作过， 有很多年积累的。

（四）延迟满足，穷人最难迈过的坎！ 前些天滴滴亏损几百亿的消息满天飞，那为什么亏了这么

多，还能做这么大？还有投资者继续投钱？无非就是觉得滴

滴拥有更广阔的前景，将来能获得几十，甚至上百倍的回 报。

很多大老板，往往也不怎么在乎公司几个月的盈亏，因为他 们看的是一年甚至几年的收益情况。

这些都是投资者、大老板能延迟满足的表现。只有普通打工 者，才会盯着每个月的工资钱。

那为什么普通的打工者很难延迟满足呢？我觉得我表哥现在 的处境，就可以很好地解释这个问题。因为他会受到了内、 外双重压力的影响。

从内部，也就是从心态上讲，多年来“上一个月班，下个月就 能拿到工资”的这种思维习惯，会影响他，一旦遇到几个月没 有收入的情况，很可能原来的信心满满，信誓旦旦就会偃旗 息鼓，原来的坚持也会在悄无声息中放弃。（当然，如果只 是当作一门副业来做，那又另当别论了。）

从外部来讲，就像我姑说的，房子、车子、孩子、人情世 故，这些生活开销，往往也逼迫着你尽快挣到钱，不会留多 少时间让你好好去学习、成长。

这也可以解释，为什么大部分成功的企业家，小时候的家庭 环境往往相对还算优越。因为他们从小不太为温饱发愁，有 安全感，有自信，能够延迟满足，看问题也相对更长远。

而小时候家里特别穷的，一直处在为生计发愁，希望获得即 时满足的状态，眼光也就没那么长远。长大后即便挣到钱， 也往往是赶紧给自己买大房子，买豪车，弥补小时候缺失的 安全感，走出自卑，找回自信后才会发展自己的事业。

事实上，不谈外部压力，从内部能够突破这种思维壁垒，让 自己延迟满足，长时间面对一件短时间内没有收益的事，对 一个普通打工者来说都是非常困难的，难以坚持的。

（五）延迟满足与即时满足相结合，才能产生持久的驱动力 即时满足可以解释很多现象，比如游戏通过不断地各种升

级，让你不停地获得满足感，成就感，让你沉迷其中。短视

频不断地让你笑成猪叫，感觉开心，让你欲罢不能。总之，

当一件事能让你不断地获得即时满足时，你就更容易把他坚

持下去。

延迟满足最大的缺点是难以坚持，而这正是即时满足的优 点。如果将两者相结合，那就能产生持久的驱动力。

比如，如果把写作这个事，当作一个长期事业来做，对于很 多人来说应该是一件比较痛苦，很难坚持的事吧？但如果你 写的东西，有很多人点赞、留言并表示认同，特别是写自媒 体文章如果爆文了，带来了几百上千的收益，这时你是不是 自信心爆棚？写作更有动力了？不会觉得写作是一件痛苦的 事，反而充满期待？（如果一直不爆文，那就不叫即时满足 了）

所以说，当我们要做一件需要长时间学习积累，才能完成的 事时，就不能一直盯着那个终极目标，而应该注意在每前进 一小步时，就要让自己获得即时满足，这种满足可以是自我 鼓励，可以是吃一顿好的犒劳自己，甚至去和朋友吹吹牛， 谈谈自己的梦想，我觉得也未尝不可。

即时满足不仅可以自己创造，也来自于周围人的夸奖肯定。 只不过一个普通人说要做改变，大多数情况下，只会被周围

人质疑，难以获得肯定。这也能解释为什么在一个相互监督 鼓励的学习氛围下，你的成长，一定比自已一个人学习要快 得多。

而对于像咱们这种不富裕的穷人而言，更应该意识到，延迟 满足，才能帮助我们实现有更多更大回报的长远目标，而那 些不在长远目标路径上的即时满足，往往都是耗费你的精 力，容易把你带偏的。

总的来说，如果你搞清了即时满足与延迟满足的作用，就能 很好地审视自己在工作生活中的一些表现，进而矫正自己的 心态，端正自己的行为，做出更好的决策，希望对你有所启 发。
