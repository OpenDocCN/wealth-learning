# 《與其無效社交，不

闪电焦 : 《與其無效社交，不如高效獨處》

2019 年，我最慶幸的是：終於找到了自己那把鑰匙——心安 理得獨處一室。

週末兩天，通常來講我得值班一天，另一天選擇待在房間裡 頭，跟自己好好相處。包括平時下班后的時間，我也特別珍 惜和期待，一個人安靜聽歌、看書、手繪、思考、寫作的狀 態。

一進家門放下背包，我會先換上最舒服的家居服，洗把臉， 摘除路上聽歌的耳機。踏進我的小房間，馬上置身于一種特 別的時空，沒有時間的滴答聲，我可以忘我地做自己的事 情。

或者在客廳走動，窩進沙發裡一動不動看半本書，然後保持 同樣的姿勢閉眼休息十幾二十分鐘，或者是半個多小時？不 用理會，不用計時，不用著急。

有句話在很久以前看過沒有很深的感觸，但今天再見時我卻 為之喝彩，有可能是因為我現在能夠深諳其中的奧妙：低质 量的社交，不如高质量的独处。

但是我有另外一個想法：與其無效社交，不如高效獨處。 我們部門的人關係處得很好，幾個人經常在下班後相約出去

吃火鍋，週末也會組隊去爬白雲山，或者幾個人租一輛車跑

去佛山覓食，這些活動中，除了打羽毛球我會馬上報名，甚

至主動建了微信群確認第二天場次和時間，其它活動我從來

沒有參與過。

晚上九點多，在他們一瓶接一瓶地喝啤酒大聲聊天，甚至興 奮地決定去唱 K 時，我一個人在客廳的瑜伽墊上做運動。 白天他們爬白雲山并開心地跟小孩子玩泡泡機時，我坐在床 上攤開所有的彩色鉛筆繪畫，怡然自得。

我從來不用擔心是否會讓他們覺得自己“不合群”，火鍋我並 不是不吃，但是太晚的進餐和聊天對我來說成本太高，本可 以在下班前从食堂輕鬆吃一餐飯的我，為什麼要挨個等著一 起下班，去一個火鍋店門口等一兩個小時吃頓飯呢？而且日 常的這種聚餐消耗不止時間，還有精力，一整個晚上甚至都 不夠，因為喝太多酒等原因熬夜還要搭上第二天的精神。

白雲山我僅去過一次，就知道自己大概再也不會去第二次 了。因為山太低，人太多，它吸引的目標人群是去“散步”放 鬆的，於我一個規矩鍛煉健身多年的人來說太過簡單。當我 第一回走到山頂時我是極其失望的，以為還不到半山腰的時 候居然已經到頭了。

但是，打羽毛球或者一起去游泳館的話，如果方便的話我一 定第一個報名，或者主動成為組織者。因為我喜歡這幾項運 動，如果去參加了就可以代替平時的運動量，而這種變換著 的運動方式是新鮮的令人感到興奮的。

如此多次以後，他們也就知道了：吃飯喝酒散步不用找 Megan 她不會去的，但是打球游泳你約她一定會有興趣。 從此我也不用費心像第一次那樣跟他們解釋“我為什麼不一起 去玩呢，大家都去了”。

人生还有太多重要的事情，不用故意放棄自己的寶貴時間和 精力消耗在不必要的社交上。
