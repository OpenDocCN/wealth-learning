# 《凌晨两点的首都机

《凌晨两点的首都机场，那些揽客的“黑车司机”告诉我两个 道理》

上个周末跑了一趟深圳，差点就舍不得回来了；但还是在星 期天晚上十点半起飞离开深圳，在凌晨一点半的时候落地北 京。

在深圳出发前，我看某平台有新用户立减 50 的优惠，加上我 怕到北京难打着车回家，咬咬牙预定了接机服务。事实证 明，这是个明智之举，虽然还是比打车贵了小几十块，但至 少不用排几十号人才能打到车。

当我飞抵北京，从航站楼走出来的路上，陆续有揽客的“黑车 司机”过来搭讪，问我是不是回市区，要不要用车；我没有理 会他们，径直走到机场外边等我预定的接机专车。

在等车的时候，我想起了那些凌晨两点还在机场揽客的司机 们，他们就像我曾经在街边发传单一样，基本是被拒绝的， 至于被拒绝的原因，我得出的结论是：信任成本过高。

一个你不认识的人突然走过来问你要不要打他的车，即使你 有用车需求，心里肯定是戒备的。

首先，他是一个个体，没有打车平台做背书，缺乏对我的说 服力；其次，一个主动搭讪的“黑车司机”这个角色难以让人 信任，且有潜在风险；再者，我选择很多，有各种打车平台 为我们提供服务，机场外还有正规公司运营的的士，我没有

理由去选择一个不确定性极高的“黑车”。

这就好比以前在街上发传单的我，通常路人是懒得理会的， 但如果看到一场路边的促销展会，很多人都会注意一下甚至 去凑个热闹，我觉得其中也有信任成本的原因：

一个活动展台等于一个平台，这显然比在街上游荡的单个人 员更让人信服；而主动给你发传单的小哥和黑车司机主动跟 你搭讪一样，带有一种“侵犯”的感觉，更加会让人有戒备心 理，但促销展会和打车平台是“被动吸引”你，所以你会更容 易信任；最后，不管是我需要买一个产品，还是游泳键身， 或是打车住宿，我都有更多的选择、更有保障的渠道，没有 必要去选择信任成本很高的途径。

既然揽客的黑车司机和传单小哥都很难获取用户，为什么他 们还要去做这样吃力不讨好的事情呢？

除了生活所需，更重要的是因为有市场。 对于揽客的黑车司机来说，机场航站楼每小时出来那么多

人，他总能逮一个人做成一单生意，这样一百来块就到手

了；而雇佣人去街上发传单的“广告主们”，并不一定都指望 你做成几单生意，能起到一定的品牌曝光也是不错的，而且

街上也是来来往往的人，总是有一定概率拉到意向客户的。

有句话叫“在中国再怎么人尽皆知的事情，也起码有好几亿人 不知道”，比如淘宝，也还有几亿人不知道这个产物，这说明 再怎么饱和的市场里面也是有市场的，毕竟市场的规模这么 大。

在深圳的那两天，我和我朋友聊起以前上班的那家“内容作 坊”，她很不理解为什么这么粗制滥造且批量生产的内容还有

这么多人看。我和她说，市面上起码一大半的自媒体公司是 搞这种“批发”的走量模式，因为看内容的人实在是太多了， 总是有人看的。

就像如今的打车服务如此普及，营销渠道四通八达，还是有 凌晨两点在机场揽客的黑车司机，还是有在大街上被无数人 拒绝的传单小哥；就算劣质内容人人嗤之以鼻，还是有人会 关注且看得津津有味。

以上，就是凌晨两点的首都机场，那些揽客的“黑车司机”告 诉我两个道理。关于信任成本和“总有市场”。
