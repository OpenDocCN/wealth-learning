# 《自己眼中的自己与

————————————————————————

今天看到一个多芬的真美广告，引发了我的思考，好的广告 不一定要请一线明星，只要它的内容是个好故事，同样可以 起到良好的宣传作用。这个广告播出以后，引发大量女性的 共鸣。有一本书上是这样形容多芬广告的成功的，看了广告 以后大量的女性冲进店里购买多芬的香皂。

是什么样的广告，在没有请明星的情况下能引起这么大的反 响呢？下面和大家简单描述一下。

一，【多芬真美广告】 这只广告长达六分钟，相当于一个小短片。刚开始只是有一

个人隔着布在画画，另一个人在说话，后来随着故事的深入

看明白了，那个画画的人是 FBI 的画家，专门通过受害者的叙

述画嫌疑人画像。

从这个线索得知，这个画家能够精准的通过别人叙述样貌画 出一个人的样子。当叙述人形完自己的容貌后，又出现了一 位陌生人只见过画像里的人一面，通过这一面之缘形容刚才

那个人的样子。

当画了很多张画以后，画家把同一个人的两张画像挂在一 起，当被画的那个人看到对比图时都哭了。为什么呢？因为 她们在陌生人的眼里比她们自己认为的漂亮很多，而且更像 她们自己。

她们这才意识到，自己对自己是多么的苛刻，我们平时总是 喜欢看自己的缺点，从而忽视很多美好的地方。

我今天出去剪头发，和理发师说剪的显脸小一点，托尼老师 就说为什么你每次来都要求把你的脸显得小一点呢？

我说“因为我脸太大了呀”，托尼老师就很惊讶“是谁告诉你的 呢？你的脸一点也不大呀？”

我一想确实好像也没人说过，但是我就觉得自己脸大。正好 想到多芬的这个广告，对自己太苛刻了，总注重自己的缺 点。就有种想法想买一块多芬的香皂，提醒自己要多留意自 己美得地方。

★虽然这个广告很短，但是它的故事却很好。它引起了大多 数女性的共鸣。虽然它没有宣传它的产品，但它传递的那种 肯定自己感觉，买了它的产品就能获得重复的体验。所以这 销量能不好吗？

二，引发的思考

平时总想改变自己，看到自己这里做的也不好，那里做的也 不好。都需要改变。但是别人口中描述的自己就比自己想象 的美好很多。

所以不要总盯着自己的缺点看啦，其实你并没有自己想象的 那么糟糕，善于发现自己的优势

可能行动起来效果更好呢。我发现我设定的目标全都是改变 自身做的不好的地方。

明天要思考一下自己的优势有哪些，也要安排一些提升自己 优势的时间。

＊＊＊＊＊＊＊＊ end ＊＊＊＊＊＊＊

▼别对自己太苛刻，但也不要太放纵自己，掌握好那个度其 实是最难的，但，正是因为这样寻找平衡的过程把变得更有 趣呀～每天都在探索全新的自己，多有意思。

人类最有意思的地方就是会改变，你今天看我这个样子，可 能过一年两年就会变得完全不一样，更有意思的是，我们的 潜能是无限的，连我们自己都不知道自己会发展成什么样！

只要生命一天没有结束，谁也不能给你的一生下定义。
