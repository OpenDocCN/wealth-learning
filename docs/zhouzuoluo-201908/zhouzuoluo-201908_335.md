# 第 28 次分享《教是

《教是最好的学》

生活中很多技能，即使手把手教，也有绝大多数人都没学 会，比如——正确使用筷子。

这是来自李笑来老师的例子，借此表达欠缺学习能力，是终 生原地踏步的根本原因。老师随后给出了老外写“如何使用筷 子”的链接。

看到这个例子后，我深深感到震撼，虽然我使用筷子的方式 是正确的，但是不知为何正确。而哥哥的方式是错误的，他 被妈妈念了很多年，依旧没有改变。

所以，会正确使用，别说教别人，自己的孩子也不一定能教 明白。由于欠缺学习能力，大多数人得一辈子忍受使用错误 的方式。

为了学会如何教别人，我点开老师推荐老外的文章，打开居 然是全英文的，一下子把自己吓着。

虽然懵，但我内心萌生一个想法，想用英文教我的小孩使用 筷子，将来说不定还能教老外。

于是，我琢磨使用筷子的关键点。

1，两根筷子中，下面那根一直是处于静止状态的；

2，张开、夹上的动作，其实来自于上面那一根。

3，大多数人败在无名指的用力上：

下面那根筷子，跟手一共有三个接触点：两端作为支点； 大 拇指的根部在中间压住筷子，无名指是“反向用力顶住”。

弄清楚后，我用左手练习，很快就会使用了。 接着，我开始读老外的文章，查完不认识的单词，整理出文

章重点，再敲到 word 文档上。最后拿着筷子，录成英文视

频，发送给闺蜜。

闺蜜回复：“傻样”。我知道，这是她对我的鼓励和赞赏。 这个过程一点都不容易，短短时间我体会到笨拙、紧张和刺

激，进而收获满满的成就感。

第二天我教会哥哥左手使用筷子，正好检验我是真正掌握了 这个“技能”，这让我无比雀跃。

对了，我和哥哥都是右撇子，之所以练左手，是因为左手是 零基础，右手是负基础。许多时候，负基础需要纠正偏差， 才能有效领会正确技能。

学习“如何学习”，在笨拙面前不要放弃。每一次“失败”，每一 次“成功”，都可以让学习能力提高一点点。

持续践行，品尝到学习能力得到提升的快感时，会停不下 来。
