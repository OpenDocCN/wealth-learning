# 《职场中，你是“正

花小生 : 《职场中，你是“正确型人才”，还是“优异型人 才”？》

在工作中，我经常会碰到两种人： 一种做事勤勤恳恳，有求必应，乖巧听话，是上司眼里的“实

干家”；

另一种按自己的思维办事，不走寻常路，是同事眼里的“独行

侠”。

最近我读了万维钢老师的一篇文章，才明白到，第一种人属 于“正确型人才”，而另一种呢，是“优异型人才”。

这两个词是万老师自创的，我觉得实在太对了。 两者之间有什么不同呢？我从 3 个方面跟你聊一下。

01

正确型人才专注做对， 优异型人才专注做好

如果说在什么地方工作，最要守规矩，我相信应该是体制 内。

每周要固定时间学习行为规范，上报工作要层层请示，执行 公务要严格遵守流程，写公文要按照模板。

有一个同事比我大五岁，已经在体制工作七年了。

他做事特别较真，尤其是在走流程方面。比如让他盖个公 章，有时领导不在单位，或者工作比较急，要立马盖，他就 是不给，说要领导签字才行。

我跟他说，已经电话跟领导汇报过了，他还是一样死脑筋。 我跟他急，他就拿出一个“老干部”的姿态教育我：你不知道

单位工作规范么？

所谓“正确型人才”，是指这种做什么事都要符合规范的 人，TA 们只关注有没有做对，形象来说，等于被工作所“驯 化”。

而我遇过另一个同事，他做事的方式，常常出乎我意料。 比如上级下发一项工作，按道理要经过领导审批才能开展。

但 ta 没有这么做，ta 自己先掂量一下，然后下基层调研一番，

最后形成一个报告给领导看。有好几次，他都跟领导说：这

个工作我们不适合开展。因为有以下理由…

而领导看完，又很神奇地被 ta 说动了。结果 ta 没干几年，就升 职到别的地方去了。

这种“优异型人才”虽然不按常理出牌，但非常注重工作效 率，考虑的是怎么把事做好，不浪费人力物力。

在我看来，正确型人才和优异型人才各有各的优点，在乎你 的选择。但不同选择的人，注定差距会越来越大。

02

正确型人才抓短板， 优异型人才扬长板

关于“木桶理论”，你应该知道：一只木桶能盛多少水，并不 取决于最长的那块木板，而是取决于最短的那块木板。

很多人在实际工作中，当看到自己的短板，比如办公软件不 熟悉，说话表达不流畅，人际关系不友好等等，都会焦虑不 已，想方设法去学习各种课程弥补缺陷。

当学了之后又发现，虽然有一点改善，但影响不大，而且自 己似乎没什么擅长的技能，然后陷入新一轮的焦虑。

我们以为职业竞争力往往取决于自己的短板，殊不知，真正 应该关注的，是自己的长板。

与“木桶理论”相对的，是“长板理论”：木桶盛水量的多少，不 是取决短板，而是取决于长板。在短板相同的情况下，把长 板发展更长，就有机会装上更多的水。

只要把木桶倾斜，长板的优势就凸显出来。 网球天王费德勒就是把长板发挥到极致的人。在职业生涯前

期，他的反手是众所周知的弱项，每次比赛，对手都会重

点“照顾”他的反手。

但他从来没有掩饰自己的短板，曾经好几次在公开场合承 认，自己的反手是最弱的。

他没有把过多精力放在提升反手上，而是精磨自己的发球和 正手，把这两项技术练到数一数二。

结果，他凭借自己的发球和正手，站到了历史之巅。

一个“优异型人才”的优异之处，在于承认自己短板的同时， 认识到一个道理：当你在一项技术足够强大时，弱点是可以 忽略的。

03

正确型人才讲专业，优异型人才讲跨界

今天我看到 Spenser 写的一篇文章，里面讲到很多互联网从业 者，正在面临“被裁”的困局。

原因可能让你很惊讶，不是因为他们不专业，没有职场竞争 力，而是恰恰相反，太专业了。

腾讯的一个离职员工说道：每个人手上分到一小块工作，然 后在未来的很长时间里，不断重复着这个工作，成为这个小 模块的“专家”。

Spenser 说，这种精细化分工虽然效率最高，但容易导致个人 的能力拓展受到阻碍，从而过早陷入职场发展的瓶颈。

正确型人才往往会在一个领域深耕很久，但越是深耕，越难 摆脱专业的束缚。

Spenser 在文章中举了奇葩说的邱晨，也是我很喜欢的大咖之 一。她做过记者、编辑、设计师，现在是米果文化 COO，很 多 title，跨界做得很成功。

她的跨界，基于两个底层武器：说话+画画。

在两个底层能力的帮助下，她可以延伸出多个职业角色，但 无论哪个角色，都是原有技能的拓展，所以她能做到在几个 职业之间自如切换。

在如今这个时代，最值钱的不是专业人才，而是可以能力迁 移的通才。

很喜欢一句话：“正确型”被定义，“优异型”定义定义。 每一个有所成就的人，应当是“优异型”人才。
