# 大家好，我是 sma

smallNIfan* : 大家好，我是 smallnifans。今天是我的第三十五 篇分享，全文 3359 字。

《我们为什么会陷入间歇性努力的怪圈里》 读书时，我们不知愁是个什么滋味。工作后后万般烦恼涌上

心头。我们想着不能像以前一样颓废了，不然一辈子可真的

完了。

但我们似乎找不到方向，陷入了一个怪圈--间歇性努力，我们 沉下去，挣扎着浮上来，再沉下去，如此往复。

我们是如何陷入这个怪圈的，又有什么办法可以让我们走出 或者改变这种糟糕的局面了。

不断的失败，不断的放弃，不断的想卷土重来的心态让我开 始思考。

1.我们这代人所谓的辛苦，其实不值一提

《穷富翁大作战》邀请了一些社会精英，如上市公司 CEO、 富二代、律师等，让他们在一周的时间里，去体验流浪汉、 环卫工人等底层生活。几乎所有精英在体验之前，都持有一 种盲目的自信：认为自己即便是落入底层，也可以通过能力 和智慧改变命运。

然而现实的残酷处境告诉他们 “真正身处底层时，他们除了生 存之外没得选择。”

田北辰的父亲是香港一代裤王，他自哈佛大学毕业后，就创 立了自己的服装王国。到今天，他已经坐拥全球七百多家分 店，在香港商界可以说是无人不识。节目里，田北辰变身为 时新只有 25 元的扫街清洁工，每天做九小时厌恶性工作，每 日生活费只有五十元，住在不足十五尺的“豪华笼屋”里。

还不到两天，信奉“只要有斗志，弱者也能变强者”的田北辰 就感叹道：

“体力上很累，第二天开始就有点得过且过，如果要做一个月 或者半年，我未必有斗志。”

节目接近尾声时，几天的体验让田北辰有了深刻体会： 这个社会正在极严厉地惩罚读不成书的人。

是啊，这个社会对于读不成书的人惩罚太严重了。 幸好我们没有生存在那样的条件下，否则那边是永无天日的

人死循环。

爷爷说，我们这代人是生活在蜜罐里的一代人。从小父母便 尽其可能给予我们最好的条件，这也导致我们这代人在步入 工作岗位之后没有怎么经受过风吹雨打。

“工作回家后总觉得自己很了不起，假如这个时候有家人要求

自己做些什么事，一般都会拒绝。”

“工作之余看了几页书，便觉得自己实在太勤奋了，得休息一 天。好吃好喝和，好好犒劳自己。

工作回到家，只要一件事情不如意，便会大发雷霆。

......

相比于田北辰所体验底层人民的几天绝望生活，我们这点苦 又算的了什么。

年轻的时候多吃点苦，到老时才不会有遗憾，就怕你自己碌 碌无为，还要安慰自己平凡可贵”。

2.间歇性努力的罪魁祸首

（1）所谓的辛苦，其实不过如此 知乎上有一个热度很高的问题： “有没有一件事，能够帮我迅速扭转颓废的状态？” 题主描述了自己现在的生活状态，硕士毕业不到三年，在一

家上市公司工作，结婚两年，孩子 1 岁，看起来是一个成家立

业的准精英，但他却对自己很不满意。

工作上，每天都是数不清的待办事项，经常加班加点，但业

绩却始终平平；

家庭里，要不停地在老婆、孩子、老妈、岳母之间“周旋”， 每天回家就跟打仗一样；

生活上，总想着能抽出时间来提升自己，但列了一堆学习计 划、健身计划，最后全都变成了熬夜、抖音和看剧。

他知道，自己过得太颓废，也想改变自己，但是总是找不到 出口，感觉所有都是混乱的，每天浑浑噩噩，恨自己，但又 控制不住堕落的自己。

从题主的文字中我们可以看出满满的无奈，也惊叹如题主一 般在外人眼中“如此优秀”的人也会想尽办法拖延享受，“做的 少，玩得多”，陷入间歇性努力的循环里。

但是只要认真分析，思考，其实结果就显而易见了。

我们自己放大了所谓的辛苦，相比于最底层的劳动人民，每 天除了想着吃，穿，住，工作就耗尽了所有时间和精力，逆 风翻盘毫无可能。而我们生活在如此“优渥”的条件下，我们 真的是没有时间了吗？好好想想，其实只不过你放大了所谓 的辛苦，懒惰而已。

（2）放弃只不过是人类的天性 读书时从生物课堂知道了动物趋利避害的天性。 坏消息总比好消息传播的快而且广，这是人类超越其它物种

逐步进化成人，创造文明的重要原因之一。

人类对于对自身有害处的外在因素总是极度排斥，想尽办法 解决。

因为不想洗衣而发明的洗衣机。因为害怕联系不到远方的亲 人朋友而发明的电话。因为害怕出行不便而发明的汽车。如 此等等。

可是你发现了吗？显而易见的外在因素都被人类解决的差不 多了，那，那些内在的不容易发现的坏处了？

它像温水一样，而我们只不过是那只倒霉的“青蛙”。

2008 年高考后，有个叫徐孟南的安徽考生，为制造考 0 分的轰 动效应，故意选择在答题卷上写下这样的信息：

“我的名字叫徐孟南，我的考号是……我是蒙城二中的……”

当媒体的聚光灯和叛逆的神光环快速褪去，家境贫困、兄妹 四人徐孟南面对的，终将是一个农家少年粗粝且辛苦的人 生。

没有文凭和学历的他，辗转多个工厂和车间，组装广告箱， 制造窨井盖，生产卫浴品……干遍了脏活儿累活儿，也在生 活的磨砺中渐渐明白自己犯了什么错。

2012 年，徐孟南创办了名为“高考 0 分生”的网站，讲述那些高 考 0 分学生的故事，更呼吁后来者不要重蹈他的覆辙，不要因 为逞一时之快而留一生之恨。

如果徐孟南早一点看清远方，看见高考前的所作所为，即时 改正。那他的人生可能大不一样。

我知道徐孟南的列子选的太过局面，大家可能会说他只不过 是个特例而已，这种列子没有说服力。我想说的例子虽然是 个特例，但是其中蕴含了我们看不清远方，看不清自己所作 所为对自己以后有多么大影响的这个道理。

大多数人没有高瞻远瞩的眼光。即便有，也会岁月的磨砺中 慢慢磨灭自己的意志，放弃在烟火尘嚣里。

“因为学习不好，去工厂耗尽一生的厂哥厂妹”。 “打了一辈子牌，临走是后悔的赌徒”。 “沉溺于抖音，手游，玩乐的我们。”

（3）心情，最大的影响者。 大家发现了吗？每次我们立下一系列目标的时候是什么时

候？每次我们放弃又是在什么时候？

5 月份我开始放纵自己。

回想起开始制定学习计划和开始放纵自己的原因，我发现了 谜底。

心情高涨我们开始制定一系列的学习计划，而心情低落我们 开始选择逃避，开始找借口来放纵自己。

所以心情低落时，请再坚持一下。挺住你就赢了。 这里我有自己总结出的一个小窍门，有奇效，关键在于你要

在放弃时多坚持一下。

我发现每次在感觉累了想要找借口放纵自己的时候，马上开 始写作，马上开始写作，马上开始写作。

对，就是这么简单。如果这时候你打开游戏或者抖音有极大 的可能，你见天计划泡汤了。

开始计划之后，你会发现心慢慢会静下来，但是在此过程中 还会有“瘙痒”等症状，不要理它，慢慢的它们就会散去。

哪怕你想放纵，在完成计划之后。那感觉多爽，你需要自己 体会。

3.一生的敌人看似很多，其实只不过你自己

人生短短几十年，事事和别人比，那么事事你都会不如意。 有的人出生就在罗马，有的人带着天赋出生，有的人一生享

受年华。

在这个焦虑的年代，焦虑几乎是主旋律。

《摩拜单车创始人胡玮玮套现 15 亿，你的同龄人正在抛弃 你》

《毕业三年，我在北京有了一套房》 听隔壁阿姨说他儿子和我差不多大，年入百万。

......

我开始焦虑了。

“为什么我这么屌丝”。 “同样九年义务教育，他就比我优秀那么多”。 “生活太他妈苦了，我真的改变得了吗？

”

......

我从一开始的焦虑，慢慢到想要放弃。我觉得他们都是带着

天赋出身，而我只不过是连汗水都洒不出来的失败者。

经历多了，慢慢悟到了一个道理--其实最大的敌人只不过是我 们自己。

和别人比什么，我们想要得到什么的过程，不就是一步一步 超越自己的过程吗？

不和别人比，我们只和自己比。 做事问心无愧，不在意他人说法。学习慢慢超越以前的自

己，哪怕进步一点点。生活发现美好的东西，每天给自己一

点正能量散播正能量。俗话说得好“赠人玫瑰，手有余香”.其

实帮助别人，也就是帮助自己。

开始写作，老想着赚大钱。但现在想通了，别人几年十几年 的积累，你几个月就超越了，合理不？显然不合理。

现在每天哪怕没有发表文章，只要每天比昨天的自己进步一 点点，问心无愧就好无遗憾了。

一辈子或贫穷或富贵，在努力过后。一切都不重要了，重要 的是那个敢于和命运抗争的自己。

一辈子很长，一辈子也很短。我们需要用一段不自由的时光 换取余下生活的自由。
