# 《对于职场妈妈来说

《对于职场妈妈来说孩子重要还是事业重要？》

晚上好，今天是橙橙加入星球的 200 天，和大家分享关于‘职 场妈妈’的话题」

这是多少职场妈妈探究的问题。 我的答案一直是：都很重要！

理由是： 1）孩子是妈妈的心肝宝贝，打生下来那天起就注定和妈妈一

起成长。

2）事业是女人经济独立和人格独立的寄托，也是在这个社会 实现自我价值的载体，还有一个特别待遇是：职场妈妈在婆 婆面前会比全职妈妈更讨喜欢受尊重。

1.  01

    因为上班而把孩子寄养在邻居家

    同小区的宝妈在离职 3 年后还是决定去上班，把 2 岁的女儿送 到邻居家寄养，每天晚上下班才接回来自己家，有时候因为 就是工作太忙或者公司应酬，忙的顾不上孩子，好几次都很 晚才来接孩子。

    今天中午我带着橙哥也在这个邻居家里玩，看着被寄养的女

    宝宝总是走到门口拿个小板凳想要开门去找自己的妈妈，毕 竟不是在自己家，家里的妈妈也不是自己的妈妈，所以她觉 得特别孤单。她估计永远想不通，为什么妈妈要把她送去别 人家，只是为了不丢失工作吗？我看得出来，她一点都不开 心。

    很多职场妈妈还口口声声地说：“我这么忙为了啥?还不是为 了孩子将来能过得更好嘛！

2.  02

孩子真正需要的是什么

到底怎么样才是为了孩子好呢?孩子需要的最重要的东西又是 什么呢?是妈妈用辛苦忙碌赚来的钱买米的高级玩具、锦衣玉 食还是妈妈温暖的怀抱和亲切的爱抚呢?

对于 3~6 岁的孩子来说，还处于依恋敏感期，这种依恋对象， 通常是妈妈。从每天陪孩子睡觉最能体会孩子需要妈妈的安 抚才能安心入眠。

没有经历温暖和恒长依恋的孩子，长大后难以形成与他人健 康的亲密关系。这个时期妈妈的爱对于孩子的心理健康来 说，就像吃饱穿暖对于身体的健康一样重要。

邻居对你的孩子再好，也无法像对自己亲生孩子的一样对你 的宝贝投入那么多的关注，更不要奢望有亲密的身体接触 了。孩子缺少营养会影响身体发育，缺少母爱同样也会影响 孩子的心理成长。

心理学大师约翰包尔比认为：孩子如果过早离开父母独立， 这种对孩子心灵损害的程度等同于成年人失去亲人时所经历

的痛他们自己被父母所抛弃或者定是自己做错了什么事情而 受到了这样的惩罚，从而沉浸在极度的自卑中。

孩子最需要得是妈妈得陪伴和关爱！

03

选择能承担的，也要承担所选择的

★那些在平衡事业和家庭之间陷入困惑的女人，一定要记住 姚晨说的这段话： “因为努力工作，我才有了选择的权利；因为当妈妈，我才了 解了生命的意义。”

两者之间，不论你做出哪种选择，你都会有得有失。所以对 于我们女人来说唯一可以做得就是尽量均衡事业和孩子，虽 然很难，但也要努力做到！

还记得姚晨说过：不必后悔自己的选择，因为人生千万条 路，你要选择能承担的，也要承担你所选择的。

既然我们选择生下宝宝就不要抱怨，尽最大努力抚养好，既 然选择投身事业也要安顿好孩子，便可安心工作。其实只要 一切都提前安排好，职场妈妈是可以在家庭和工作中游刃有 余的！
