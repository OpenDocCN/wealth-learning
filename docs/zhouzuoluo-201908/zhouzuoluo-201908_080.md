# 《普通人为什么要会

《普通人为什么要会写作？》

即使你现在觉得“写作”两个字跟你压根无关，我还是建议你 看完这条帖子，因为我不是作为职业写作者来讲道理，而是 站在一个普通人的立场，分享一些身边的故事，相信会对你 的成长有所启发。

我现在负责写作训练营的运营工作，算是公司里最“不职业的 写作者”，不需要像写手一样每天写稿，看起来“写作能力”对 我来说是最没用的。但前些天招聘的时候，我打开邮件，还 没看简历，就 pass 掉了一小半人。然后我跟粥老师聊天说：来 我们公司干活，即使不是做写手，没有写作能力根本不行 呀。”

粥老师回答很干脆：对！不会写作啥也干不了。 这话还真是。 比如说我们做社群运营，经常需要写发给学员的话术、编辑

一些注意事项、通知、各种海报文案、甚至招生软文。我常

常想当然地以为这些好简单，事实上大部分人写不好，不是

有明显的逻辑漏洞就是措辞分寸不当。

回想起刚刚跟粥老师一起工作的时候，他叫我写一段申请授 权转载资格的话术（在公众号转发他人原创文章，需要申请 授权），100 字不到，我都得思索半天还不合格。

我私下请教了一个老师，重新调整了几遍发给粥老师，结果 还是不合格。最后他亲自写了一段，我领教了什么才是会 写。后来工作中，大量重复性的沟通，我都会先编辑好话术 来提高沟通效率， 不知不觉地变得能快速写好各种话术。

有人可能会说：“你们在新媒体公司或者做社群才需要写这 些。其实不是的，因为不管你做什么工作，都需要沟通。

我的大姑子在房企做 HR，她说很多时候跟员工谈话，心里明 明有很多话却讲不出来、很多理所当然的道理却讲不明白， 憋得内伤。她来询问我怎么解决，我就建议她学习写作，因 为讲的过程就是一个人组织语言的过程，底层逻辑就是写 作。

她报名了第四期写作训练营，前几天家里有聚会我就给她带 了粥老师的《学会写作》，同桌的同龄人就很好奇地 问：“嘿！不都是小孩子才学写作吗？成年人还学这个有什么 用？”

我就问她：“你平时做什么工作？” 她回答：“当老师。”

我说：“老师的很棒啊，天天跟学生们在一起，除了教学之外 还需要做很多的沟通吧，累不累啊？”

话到这里她便停不下来，跟我分享她工作过程中遇到的各种 沟通问题，比如口都讲干了有的学生就是不听。我又问她怎 么说的：“你要好好读书啊，作业一定要写啊！不写作业是不 行的，你知道吗……”

我只好笑着说：“写作认知里面有一条叫做用户思维，如果你

有时间了解一下，应该会对有帮助…….”

最后她拉着我问了很多，其实都是与相关的内容。生活中， 我也常常会被问“工作总结怎么写”、“朋友圈文案怎么写”、 “我要发个招聘怎么说”、“我要求行业大佬办事如何措辞”等 等。

正因为这些，让我觉得写作技能对每一个人其实都很重要， 有时候觉得自己不需要，可能是因为还没弄明白，如果自己 拥有了这项技能到底有多爽。

最后，根据前面三期学员的学习反馈，我跟粥老师反复斟 酌，调整了训练营课程大纲，加入了写作认知中最重要的底 层逻辑（用户思维、读者思维）以及具体的实战应用（个人 品牌、写作变现）。第四期训练营我也尽可能多分享一些工 作邮件、简历的写法，提高过稿率的方法。
