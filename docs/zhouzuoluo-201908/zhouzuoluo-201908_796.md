# 《用心做事的人，总

《用心做事的人，总不会被亏待》

临近农历新年，有几个行业火爆异常。无论你什么时间去， 都要排好长时间的队。比如理发店，比如洗车行…

下班打算把车清洗一番，让它也干干净净的迎接新年。排了 好久的队，终于轮到我。由于我后面还有好多人等着刷车， 我本以为刷车工人为了节省时间，可能不会刷的太仔细。但 事实证明，我真的是太狭隘了。

毫不夸张的说，这是我见过刷车最仔细的一家洗车行了。如 果客人不多这么仔细我还可以理解，但今天这么对人还在后 面排队，刷车工人都没有为了多赚钱而草率刷刷了事，真的 让我印象深刻。

甚至有同行认为他家傻，不会做生意。一年到头旺季就这么 几天，不趁着这几天多赚点，实在说不过去。他家店今天可 能确实会少刷几台车，但长远来看，他得到的是更多忠实的 客户。到底是聪明还是傻一目了然。

其实这个世界没有傻子，你是敷衍应付还是用心做事，别人 肯定能看出来。一次不能，两次三次也会看出来。而用心做 事的人，终究不会被亏待。

前几天，新东方年会上一首改变的沙漠骆驼红遍网络。其中 让我印象最深的一句台词是:干活的干不过写 PPT 的。这句歌 词引发了众多的共鸣。

在职场上，我们确实会遇到一些不公。辛辛苦苦干的工作， 有成绩了功劳都是别人的，出问题了锅都是自己的。但我觉 得没必要争一时的是非对错，只要用心做好每一件事，时间 久了，肯定会有人看到。

如果领导一直看不到，那我们在用心做事的过程中都已经提 升了自己的能力，既然在这得不到应有的对待，那就果断跳 槽，去追寻更广阔的天空。这是公司的损失，不是个人的损 失。

而且领导之所以能当上领导，肯定有过人之处。那些人在用 心做事，那些人在偷奸耍滑，他们心里真的有数。包括新东 方年会 diss 公司的几名员工，都得到了丰厚的奖励，这就是对 用心做事的人最大的肯定。

但行好事、莫问前程。我相信只要用心做事的人，都不会有 差的前程。
