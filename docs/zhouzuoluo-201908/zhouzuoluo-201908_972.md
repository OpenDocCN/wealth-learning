# 2018 年，我自己

2018 年，我自己写了很多，但直接分享的好文不多，我总觉 得要自己原创才对得起大家，最近我琢磨了一下，这样并不 好，2019 年，我建了个新标签“粥左罗的私人阅读清单”，除 了我每天继续写，我要坚持每天分享一篇我当天读到的好 文。

这件事只适合在星球里做，粥左罗笔记做不了，因为那个账 号定位原因，我们只能推荐个人成长、认知提升、职业发展 相关的文章，“粥左罗的私人阅读清单”想要分享的不局限任 何定位，所以我的分享会是五花八门，但有一条，一定是我 当天阅读到的我认为值得一看的。

一个人“吃”什么，就会“长”出什么，我写原创是给你们展示我 长出了什么，我分享我每天的优质阅读，是展示我在吃什 么，所以 2019 年，我每天都要分享一条我吃的东西。

大家平时太忙，没时间天天看的话，就每次进来时点这个标 签，就能看到我所有的好文推荐。

希望 2019 年的这个栏目大家会喜欢。
