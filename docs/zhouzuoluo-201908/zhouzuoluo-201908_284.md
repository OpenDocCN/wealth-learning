# 《我从粥左罗身上学

《我从粥左罗身上学到的 3 点管理才华》

你们认为粥左罗会管理么？ 反正我一直不知道他会不会，因为他除了偶尔给工作目标之

外，关心重要节点之外，几乎不太管员工，也就是他自己说

的“佛系”。

我第一期的做训练营的时候经常想，这老板也太心忒大了 吧，我们都没有做训练营的经验，他居然放心让我一个人去 安排所有的事情，工作细节也几乎不过问。

很幸运的是，我们一期做得挺成功，二期做得还更好了，最 近我跟不少运营同行交流，她们也都反复称赞我们做得太棒 了，加上昨天听说了一些社群管理失败，我才忽然发现自己 老板的 3 点管理才华。

1、足够的信任和支持 起初我们是不打算做写作训练营的，首先从人手和经验来说

我们不足，其次从投入产出的性价比来说不如直接卖广告。

但我觉得我们已经有了很好的写作课程，可惜大部分学员买

了课程无法坚持、不太懂得如何科学训练，以至于收效甚

微，如果有一个好的学员氛围再加上老师指点，肯定更容易

坚持下去，进步更快更明显。

我直接就把策划书交给粥老师，我知道他心中肯定有疑虑和 怀疑（这点后来结营他自己也说了）但当时他啥也没多问， 立马就同意了。经过一轮的电话沟通，我就开始紧密筹备。 整个过程中我们也有意见不一致、法彼此说服的时候，他都 尊重了我的意见，而不是像一般的老板说：“我是老板，你就 听我的。”

这一点我在带助理、带团队的时候，也尽可能放手让对方先 去试，她们提出的一些方案和活动，我们也一起去尝试，当 然不够理想的时候我会再调整，对整个结果负责。

2、舍得给钱不仅仅计较 没有什么比给钱更直接的激励了，我认为聪明的老板都懂得

如何给钱，笨的老板都还在抠抠缩缩。

为什么这样说呢？ 举个例子，最近听说一些运营官对自己的团队成员非常抠

门，包括义务服务的班长，发红包不让抢，也不给其他的物

质和精神奖励。这也导致部分班长们心生怨气，服务自然做

得就很被动很吃力。

那么我们呢？粥老师在一期训练营发的红包就有一万三千 多，二期我也经常给自己团队成员发红包，虽然金额比不上 老板，代表的也是一份对大家付出的感激；当有的学员对社 群有贡献的时候，比如说及时整理好直播音频、文档、做出 思维导图，我基本上会给专属的红包。非常认真负责的助教 老师，我甚至会给他们一一寄礼物。

所以，平时就做一个大方的老板，才可能有愿意付出的员

工。另外，我的团队成员，经常会自己掏钱在社群发红包， 甚至学员完成打卡也会红包奖励，像班长们本身就是义务服 务的，所以我也感谢他们无私的付出。我觉得这始于老板， 最后形成了一个非常正向的循环。

3、私下批评、公开赞美 做完两期训练营，我原本希望的温柔人设是立不住了，淘气

的班长们给我取了个外号“文一刀”，学员也跟着各种求刀。

奇怪的是，为什么大家还愿意下期继续复训，继续虐呢？有 的时候团队工作不到位或者有些小失误，我基本上都是立刻 用私下沟通，言辞往往犀利，但在公开场合尽量不提，表扬 肯定要比批评多。

这是我从粥老师身上学到的第三点：私下批评、公开赞美。 私下里我挨的批评不少，可以说他骂人直接又扎心。但是公

开场合却不会有过激的言辞，基本上都是表扬和感谢，给人

足够的尊重。

做为员工不能玻璃心，得挨得了批评；做为管理者要懂得维 护员工的尊严，尽可能私下批评。
