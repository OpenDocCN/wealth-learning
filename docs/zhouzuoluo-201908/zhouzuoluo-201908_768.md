# 《文章拆解与点评练

《文章拆解与点评练习 002：终于，我们「逃回北上广深」 --

粥左罗》

突然来拆解与点评粥老师的文章，我冒着极大的勇气。不 过，拆文时的我，是一名思考着，拆文者，我要做的是从拆 文中获得认知与思考。当然，这篇文章，就有我想要的东 西。

拆解的文章，我以 word 文档 + 批注的方式写到了文件里，大 家可以下载看看。我花了 2 个小时，总结出了粥老师这篇文章 3 点启发：

1、 切忌偏安一隅、追求小地方的稳定。这世界，唯一不变的 就是变化。

2、 作为年轻人，有抱负要去大城市施展，这样有更大概率成 功。 3、学会借势很重要，借势成功可以将个人努力放大 N 倍。

虽然花了 2 个小时，但是从中得到的思想深度是其他碎片文章 得不到的。因为我结合自己的经验，投入了大量的思考，这 些都是我无形的、宝贵的财富。未来，我会继续寻找好文章 拆解，而不去看那些流水一般的信息流文章，没啥意义，徒 劳浪费时间。

我现在已经明确的知道：我要提升的是思想深度、认知深 度。那些所谓鸡汤，统统泼掉。只留那种能够经过时间检验 的文章。粥老师的文章正是如此，时间的冲刷，不仅不会让 粥老师写的文字失去意义，反而如同酿酒一般，越来越醇

香。

好啦，感慨发完了，以下是我对粥老师的《终于，我们「逃 回北上广深」 》这篇文章的拆解与点评。看看星友们能否从 中获得启发！
