# 《读〈坚毅背后的秘

《读〈坚毅背后的秘密〉对成长型思维有了新的认知》

一、成长型思维从“我不行”到“我可以” 主人公讲述了自己从一开始不会唱歌甚至讨厌唱歌的自卑， 到后面不断努力学习之后，建立自信获得成就感的经历。

做一件事情你一开始不喜欢其实没关系，只要你在这件事情 上努力去做之后，让你重新建立自信获得成就感，你真的就 会喜欢去做。

越是喜欢，就愿意花时间去练习，花的时间越多，进步也越 多，进步越多就越有成就感，这就是一种正向循环。

喜欢就是建立在“我可以”这种积极的信念上的，如果对自己 没信心，不相信自己可以做好，那么就很难坚持到拥有成就 感的那一刻，体验不到成就感，我们就不可能喜欢上这件事 情，有时候我们真正爱上的不是事情本身，而是做这件事情 良好的自我感受。

二、自我效能和成长型思维 我们之所以会喜欢一件事情，并且能坚持下去，就是因为这 件事情给我们带来了持续进步的体验，良好的自我感受，以 及他人的赞美。

不管任何领域做出成绩之前，我们肯定会经历艰难而又痛苦 的初学期，只有熟练掌握知识和技能之后，我们才能体验到 质变带来的成功体验。在这段时间内，积极的自我信念非常

重要，我们必须坚定自己可以成功，否则就很容易产生挫败 感而放弃，这种积极的方式就是自我效能。

拥有成长型思维模式的人，他们相信能力可以通过后天的努 力来提升的。相对应的固定型思维的人他们觉得能力是天生 的。

三、如何训练成长型思维

1.改变归因模式 当我们做不好的时候，如果我们把表现归因为那些自己无法 改变和提高的事情，那么我们就会认为是自己的问题，是自 己的能力不够，这种归因方式是固定型思维。

如果我们把自己的归因为自己可以通过努力和改变来提高自 己的能力，我们就不会因为自己做得不好来否定自己，而是 让自己处于加倍努力提高自己的水平，这种方式才是成长型 思维。

2.学会学习 在知识的学习和技能的发展上，天赋、努力和方法扮演着非 常重要的角色。天赋和方法是可以互补的，而正确的方法是 可以通过后天的学习来掌握的，努力就是加速器。

3.积极的面对错误和失败 对犯错和失败持有消极的心态，想尽办法逃避失败，这就是 典型的固定型思维。

想要改变对错误的态度，我们要放弃不能犯错的想法，我们 需要赋予犯错的积极意义，这边有一种方法是建立犯错档 案，把自己犯错记录写下来作为学习的素材，思考自己如何 才能收获和成长。

两种不同的思维方式导致的两种人生肯定是不一样的，我们 从现在开始培养自己的成长型思维。
