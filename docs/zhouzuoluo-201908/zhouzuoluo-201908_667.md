# Day114《世界

《世界越复杂多变，我们越要简单至诚》

江南春说：“价值决定市值，你能够做到多大，取决于你能帮 客户做到多大，这永远是品牌传播企业所依赖的价值观和使 命感。”

在我看来，我们在成为行家的路上，同样需要聚焦核心价 值，那就是尽可能给更多人带去积极的改变，真正能够帮助 人和成就人。

记得，年前看罗振宇《知识就是力量》节目的某一期，他提 到了三条最想分享的人生建议，如今回想起来还是非常清 晰，因为他所讲的三条，本就与我内心的想法产生了共振。

1、寻找不变，回到不变的规律和事物 放在行家之路上，不变的就是为别人带来价值的意愿和能

力。我们做定位，做传播，不断学习和精进，都是为了更好

地去服务更多、更对的人。

还有就是不变的人性。如果想把事情做漂亮了，要时时刻刻 把人放在心上，你需要被鼓励被认可，就要学会鼓励他人和 认可他人；你需要获得成就，就要学会主动去成就他人。

用一句话说就是：利他才是最大的利己。这是不变的人性。

2、回到简单，复杂的事情简单做，简单的事情重复做

时间堆积出来的力量是无法被超越的，关键在于持续去做。 我们在行家之路上，不需要盯着大事去做，而是能把简单的 小事做到极致，就能获得别人无法超越的优势。

在我加入的剽悍江湖社群中，就有不少老铁，就是通过持续 践行，把看似简单的事情做到了极致，从而成就了他们在江 湖的个人品牌和影响力。

3、保持真诚 没有什么道路可以通往真诚，真诚本身就是道路。 你的真诚，别人是很容易感受到的。可能我们的能力不是最

强的，但做人做事是最真诚和靠谱的，那就能被越来越的人

感知到，我们的个人品牌就出来了。

世界越复杂多变，我们越要简单至诚。在成为行家的路上， 更是如此。
