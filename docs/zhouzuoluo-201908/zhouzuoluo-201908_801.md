# 《真正的朋友，是那

丁丁 : 《真正的朋友，是那个身怀六甲还坚持开车来机场接你 的人》

赵赵是我读研究生时期最好的朋友。 她长我两岁，但比我小一届，也不是一个系的。我在广告

系，她学的是录音工程。我们在一个宿舍楼，她住在我对面

的隔壁。我是活泼外向的文科生，她是典型的工科女生，每

天就宿舍- 实验室- 教室三点一线。

有时候我都好奇，两个性格差那么多的人，是怎么成为好朋 友的。

想想原因之一，可能是我俩都喜欢吃，把吃视为生命重要的 一部分。特别在我研三的那段日子，刚刚经历过一次失恋， 常常在宿舍颓废睡到中午，几乎每天都是赵赵从实验室给我 打电话把我叫起床去食堂吃午饭。

不过我想，更重要的，应该是她那种什么都不过问的性格， 让我觉得很舒服。那次失恋对我打击很重，我不仅颓废，甚 至有些堕落。当同学们都在忙着毕业论文和找工作的时候， 我却天天在玩夜不归宿。

宿舍里当时住了一个本科生，每次看见我都问，“姐姐，你昨 天晚上去哪里了呀？怎么没回宿舍呀？”烦的我每次看到她就 想躲。我从来都不习惯被人盘问。

这让我更喜欢与赵赵相处。她每天只是准时叫我去吃午饭和 晚饭，吃饭的时候和我聊聊老师或者同学的八卦，却从不问

我的事。

后来我毕业了，在离学校不算太远的地方租了房子，留在了 北京。

赵赵保持了她一贯的吃货本质，常常过来找我一起吃喝。有 一年我过生日，男朋友给买了个生日蛋糕，赵赵自己差不多 吃完了一整个六寸蛋糕，这也成为多年后我们聊起往事的笑 柄。

毕业后，赵赵离开帝都，去了魔都念博士。那个时候我过得 随性非凡，好几次从帝都飞魔，找赵赵吃喝。还记得赵赵在 宿舍给我展示她那副巨大的婚纱照，我至今好奇最后她是怎 么把照片寄回去的。

博士毕业后，赵赵随老公去了珠海，在珠海某大学当老师， 这应该是很适合她的职业。从此以后，我们见面的次数就少 了很多。

幸好学校差不多每年都有带学生出来实习考察的机会，有几 年都是来帝都。每次来帝都，赵赵都愿意住我家，她说不喜 欢住宾馆的感觉。

最后一次来，是我怀孕的那年。我整个孕期都没有食欲，基 本吃不下饭。赵赵来的这次，因为我妈妈住着照顾我，所以 没住我家，但是还是过来看了看我，顺便蹭饭。把我妈做的 一桌好吃的一扫而光，然后舒服地在沙发上睡了个午觉。

后来联系真的渐渐少了，特别是我生完孩子后。我也不便过 多和她联系和她联系，毕竟她结婚，毕竟她结婚很多年，从 念博士那会就想要孩子，不知为何一直要不上。

今年过年前打算带孩子去珠海长隆玩，也想顺便见见赵赵。 微信先联系一下，才知道她终于怀孕了，已经七个月了，真 是不容易。想要孩子要了十多年了，真心为她感到高兴。

听说我要去珠海，她一个劲地说开车来接我，把我吓一跳。 一个孕妇，挺着个大肚子，开车一个多小时来机场接人，别 人我不清楚，就赵赵要孩子的费劲劲儿，我可不敢让她来 接。赵赵说，她产检都是自己开车去的，还说，自己又不是 娇贵的人，但我依然是不敢同意她来接。

但是，真的很感动。 真正的朋友，就是这样的吧。
