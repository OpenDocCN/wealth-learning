# 【拿死工资的人，正

【拿死工资的人，正在慢慢拖垮自己】

开篇想问大家一个问题： 在深圳，如果像下面这样的生活，你愿意过吗？ “设想你是个刚毕业的大学生，来深圳找工作，很快收到一家

公司的 offer，每个月拿着死工资 10000 块，到了年底还有年终

奖。你勤勤恳恳，辛勤工作，然后升职加薪当了个小主管，

算不上大富大贵，但衣食无忧，过得也很体面。”

你是不是觉得生活很舒服，过得很安逸？然后时不时还能和 朋友去个酒吧，小资一下？

如果这样的生活，你愿意过，那我实在是太心疼你了.. 这种安逸生活的背后，其实隐藏着巨大危机：你全部的收

入，都来自于出卖自己的劳动力赚钱，那你有没有想过，如

果你无法再提供劳动力，你靠什么生活？

1

拿死工资并不是稳定，而是危机四伏

我认识一个小学同学 A 君，从小学习成绩就特好，当年高考， 全深圳都可以排名前 500 的那种，大学考到了重本线的中山大 学。

按理说，中大毕业，毕业后找的工作也不会差到哪里去，由 于 A 君 IT 特别强，所以刚大学毕业那年，正当我们所有人都认 为他会从事 IT 行业，当一个工程师的时候，他却做出了一个 让我们意想不到的决定——当公务员。

原因是 A 君从小是个乖乖仔，父母在深圳也是做了几十年的公 务员，他只好听父母的话，在某机关单位干着，一干就是干 了 3 年多，一直规规矩矩呆着。

年薪不高，大概一年 10 万块左右，单位包吃，家里老早在深 圳就有了房子，只能说吃穿不愁，去年结了婚，生了个女 儿。

老婆说想出个国旅游，手里存款不是很多的他，只好做了 N 套 攻略，算了 N 次账，只想算出最具性价比的旅游方式。

女儿喝奶粉，老婆说要喝进口的，他只能一个月花个好几千 买奶粉。

我问他，你就没想过跳出体制，做些自己想做的事情吗？固 定工资没了，你怎么办？

他说，公务员铁饭碗啊，怎么可能说没就没，当初就是因为 看爸妈都是公务员，还干的好好的，才选择走公务员这条路 的。而且，我在单位表现也不差呀，

他真的是个乐观主义者。 如果换做是我，我估计乐观不起来。因为每个人在人生的一

定阶段结婚生子，时不时还会生一次病，可我身边绝大多数

人面临的情况是：生病了，怀孕了，没法工作，死工资没

了，收入骤降为零，只能吃存款。

稳定，只是一个表象，背后隐藏的，才是最让人担忧的种种 你看不到的危机。

只拿死工资，是最不靠谱的人生经营。

2

避免死工资，最主要的是优化收入结构 我曾经在深圳的公交车上看过这样一个现象。

早高峰的时候，一位农民工大叔走进了公交上，原本就已经 挤到不行的人们立马变得更加拥挤，大家很自觉地在大叔的 四周留下了一点点空间，形成一个小包围圈。

大叔的脸上，顿时显得十分尴尬... 我还听到旁边有个穿的挺漂亮的上班族女生小小声的说了

句：离我远点，别碰到我...

拜托，这位女生，你怎么就好意思嘲笑别人农民工大叔呢？ 不要嘲笑人家农民工大叔衣服脏，似乎没什么出息的样子。 其实，你一个拿着死工资的上班族和这位农民工大叔并没有 本质上的区别...

那也许你会问，我怎么才能避免死工资呢？ 关键是优化你的财务结构。

举个例子。

深圳最新的平均工资月收入是 9561 元。这 9000 块钱，如果下 面有两种使用办法，你会怎么去组合使用？

第一种，一份稳定的白领工资，自己做事也比较努力，工资 扣除五险一金啊个税啊等等之后，到手就是差不多也有个一 个月也有个 8000 块钱。

第二种，月入的 9000 块钱分别来自三种不同途径，一，你有 一份写字楼的工作，平时清闲得很，压力不大，到手工资 5000 块；二，除了本职工作，你还做了一些小型的理财，这 些收入，加起来，1000 块钱；此外，你还会做一些兼职，哪 怕周末去开滴滴，去做一些兼职，这些收入，加起来，3000 块。然后你还可以存起来 1000，到手也是 8000 块钱。

同样是月入 9000，收入渠道不同，财务结构也就不同。 哪一种财务结构好呢？

答案是第二种。

为什么我建议你选择第二种财务结构？因为工资是线性增长 的，而房价却是指数级增长的。

之前我发关于深圳房价的文章，很多人都在抱怨深圳房价太 高，根本买不起。当然了，你靠死工资增长的速度，敌得过 通货膨胀的速度吗？

如果你在 2008 年有 100 万，那绝对是一笔巨款，南山区基本所 有房子你随便挑，但在 2018 年，100 万只够给南山区一套房子 的首付..

倘若你选择第二种财务结构的话，我不能保证你短时间能赚 大钱，但一定能让你在用钱上，不会像我那位公务员朋友一 样，捉襟见肘。

3

优化收入结构的目的，是让你的现金流更持续 从而实现财务自由

上个段落我教了你如何优化收入结构，也许你会问，这样下 去会得到一个什么结果？我讲一个我朋友殷桃的故事吧。

殷桃的本职工作，是一个会计。在好几年前，她和你我一 样，曾经都是拿着死工资过活的人。但很快，她慢慢意识 到，拿死工资，每个月很多时候想花钱，却发现根本没有闲 钱可以花。买个稍微名牌一点的包包，喝个下午茶，也要思 前想后。

后面，她下定决心说道：我一定不会再过这样只靠死工资的 日子！

她下定决心要优化自己的工资结构。于是她开始使用记账的

app，每个月都会留下一点工资存起来。工作 3 年后，她终于

有了一笔可观的存款。她知道自己是个会计，升职加薪的机 会并不大，于是她把自己辛苦存了三年的钱，拿出三分之 二，和男朋友一起合伙投钱搞了一家手游店。当然，她出小 头那份，男朋友出大头。

剩下三分之一的存款，她就拿去银行买了点基本的理财产 品，没有啥风险，就当搞点小钱。

毕竟做会计，平日的工作不算太忙，做了那么久基本一个早 上就能把日常的工作搞定了。于是平日空出来的时间，她就 学着搞搞抖音，拍拍小视频。她喜欢做饭，于是就在抖音上 教人家做饭。

后来，手游店每个月都有分红，抖音流量现在又是红利期， 她每一期的做饭视频都十几万人点击，所以粉丝蹭蹭上涨， 很快就有广告商找上门和她商量广告合作的事宜。

今年，她和男朋友顺利结了婚，前两天她告诉我，她怀孕 了，然后就直接向公司提出离职。

“没关系，拍拍抖音，再加上手游店的分红，够生活了。”

看她如此有底气的提离职，自己又实现了财务自由，我真的 替她感到开心。

* * *

讲了这么多，最后我想说的是， 不要再沉迷于死工资了，勇敢跳出来，优化自己的财务结构

实现财务自由，才是决定你下半生过的怎么样。
