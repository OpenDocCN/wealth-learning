# 《初中没毕业的他，

《初中没毕业的他，在上海买房买车了！——我却一点也不 羡慕！》

春节走亲访友是少不了的，昨天是我妈这边的亲戚到我家 来，一年到头难得见几次面，总有些话要聊的。晚上亲戚们 回去后，老爸跟我说，你们这些兄弟姐妹中，我最佩服的就 是你表弟小钟，他初中没毕业，现在却在上海那边买房买车 了。

我表弟确实是初中没毕业，就出去打工，打工几年后，跟着 他一个在上海搞会展设计的堂哥学设计，学了三年之后，他 就自已在上海找了份工作，由于扎实肯干，很快得到了老板 的认可。在这行多年，工资也水涨船高，加上提成，平均下 来，每月能有两三万的收入。

后来他在上海谈了个江西的女朋友，两人结婚生娃后，就在 上海周边城市买了个 140 万的房子，去年又添了一部 15 万的 车。

以这么低的出身和学历，取得现在的成绩，的确值得肯定。 在我爸眼中，这也许就是一个成功的人生，我已经听到好几 次他在别人面前表扬表弟了。

但是我却一点都不羡慕，不谈每个月近两万的房贷车贷和养 娃的压力，最主要的是我觉得他这种做法会限制自己的成 长，让自己失去更大的发展机会。因为有房贷车贷的压力， 生活都是捉襟见肘的。他肯定不敢轻易跳槽，更不敢有自己

创业的念头，在每天的精打细算中，他也会要放弃一些需要 花钱花时间去尝试，去学习成长的机会。不像我，看到一个 可能让自己成长的机会，只要价格合适，不用太纠结就会花 钱去学。

而且他的这个设计工作是有天花板的，在前期，他的收入也 许比新人高不少，但五年、八年之后，他与新人之间的能力 和经验差距就不那么明显了，我认为这是会展设计和我所从 事的照明设计共有的特点，在前期你的成长很快，在后期往 往会停滞不前。

反观我自己，虽然我现在收入不咋地，但我一直都在尝试新 事物，一直在寻找新的收入来源。而每次获得新收入时，我 的自信心就会增强一些，我不再担心有新人会超越我，因为 我有更大的视野，更多的见识，以及更多的出路。

我有信心在本职工作以外还能养活自己。而拥有这种信心之 后，我的心态也更豁达了，我敢于向别人分享自己的经验， 不再害怕“教会徒弟，饿死师傅”！我时常想，如果我一心钻 研照明设计这份本职工作，我的收入一定比现在高不少。但 我从不后悔，因为我更向往一个有想象力的未来。

说了这么多，我是想说我表弟的选择是错误的吗？不是！我 是想说每个人都有自己的认知系统。一件事，在你的认知系 统里，你觉得是不可取的，不正确的。而在别人的认知系 统，可能会认为再正确不过了。

先说我爸，我爸那个年纪的人吃过不少苦，他小时候兄弟姐 妹多，吃不饱饭，成家后做生意又亏了本，为了还债出去打 工，刚存了点钱，又遇到我和我妹几乎同时得大病，又欠下 一屁股债，继续打工还债。一个人的人生经历和阅历就决定 他的认知，也许在我爸的认知里，一个人在大城市有房有

车，那就是成功的象征。什么个人成长、工作前途，未来趋 势，那都不如摆在眼前的房子车子来得实在。

再说我表弟，我表弟买房时，儿子已经五岁了，这个时候他 不得不考虑儿子以后上学的事。而且这时他手上也存了点 钱，加上工作也处于平稳期，在房产公司工作的小姨又刚好 替他看中一套好房子，试问他这时不买房还要等到什么时 候？所以在我表弟的认知里，买房是件百分百正确的事。

而在我的认知里，我觉得成长更重要，我可以为更美好的未 来牺牲现在的利益，我可以为未来活得更轻松，而在现在更 拼命。当然，我的认知也在改变。比如以前看 C-C-TV“国家- 品牌-计划”的广告时，总觉得这是经过严格筛选出来的，后来 被约谈的报道出来后，才知自己被误导，对央视广告的信任 度也降低了不少。又比如没孩子时，觉得买不买房都无所 谓，有了孩子后，也开始考虑将来要上学的事，总觉得应该 买房给她一个安稳的家。

也就是说，我们的认知是在时刻改变的，而我们做的任何决 定，往往也是我们当时的认知系统认为最正确，最有利的， 只是在别人的认知系统里却未必能获得认可。

明白了这点，也许你就能更加设身处地的站在别人的角度考 虑问题，也多了一种透过现象看本质的方法，那就是从自己 与他人的认知系统里，去找问题的症结所在。这样会让你在 面对生活中的很多矛盾时，更加坦然从容地去面对了。
