# 《成长，骗自己容易

《成长，骗自己容易，骗别人很难》

承认自己的“LOW”，是一件挺难的事情，但这是你不得不承 认的事实——第 145 天的星里浣

你会给自己贴上“中等偏上”的标签吗？ 我会，但我其实并没有中等偏上的能力。

昨天看了 @粥左罗笔记 的推文 这篇文章讲的是人才的 6 个层 次，看完以后，我有点不想承认结果，因为我发现自己所处 的层次是倒数第二的“四流人才”，而我打开这篇文章之前的 心理预期是“怎么着也得是个中等偏上的层次吧。”

我一直在心里给自己的标签就是“中等偏上”，尽管我在很长 一段时间都表现得非常“LOW”。

昨天这个“四流人才”的结果，让我重新意识到自己身上这 个“中等偏上”标签背后的问题：这其实是一种自我麻痹的欺 骗。

不想承认自己的不足，认为自己是“中等偏上”，应该获得更 高的评价，这样的想法，骗得了自己，却很难骗过别人，因 为你是什么样子，你有什么能力，外界是能直观的看到的， 但你自己却很难去面对。

为什么会出现这样的现象呢？经过对自己的剖析，我发现主

要有 3 个原因：

1-恐惧带来的逃避 做手术要打麻药，因为人怕痛，这是对疼痛的逃避；我给自

己贴上“中等偏上”的标签，是因为我怕“LOW”，想要逃离平 庸。

2-嫉妒带来的忽视 我刚毕业那会，因为找不到工作，只好做月薪 2600 的游乐场

安全员，但我有不少同学就能一个月拿七八千甚至过万，还

有已经准备买房买车继承家业的那种存在。

当时我很嫉妒他们，觉得他们怎么就这么命好，怎么就这么 会做人，怎么就能有一份不错的工作。为了寻求心理平衡， 我就选择性忽视了他们的闪光点，把他们“过得好”的原因都 归于“命好”、“会做人”、“工作不错”。同时，我也选择性忽视 了自己的缺点，觉得自己也算是个“中等偏上”的人，只是运 气没那么好罢了。

3-理想带来的美化 电影《被嫌弃的松子的一生》里有这样一句台词：每个人小

时候，都认为自己的未来是闪闪发光的。

因为我们天生就抱有成为主角的理想，所以很多时候不想承 认自己的差劲，就会给自己的所作所为进行美化。而我美化 自己的手段就是贴上“中等偏上”的标签，以一个“比大部分人

好”的理由来心安理得的接受不想努力的自己，

4-成长，骗自己容易，骗别人很难 我一直都给自己贴着“中等偏上”的标签，因为我恐惧平庸、

嫉妒优秀、想要一个理想的自己。我曾经在 @粥左罗笔记 推

送的文章 下留言道：

“是我太自负了还是怎么了，我怎么觉得我在第八个认知层 级。也就是我能发现自己的优点和缺点，看得到这世界的良 善与丑陋，并且想得开，也做的起。”

但今天看来，那时候的自以为是，其实是先入为主给自己贴 上了“中等偏上”的标签。
