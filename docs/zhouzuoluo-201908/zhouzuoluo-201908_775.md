# 《在农村待了三天发

《在农村待了三天发现：毁掉一个人太容易了》

昨天粥老师写了篇文章《终于，我们逃回北上广了》每次他 写北上广的话题我都想抗争一下，想说生活在小城市其实也 可以怎样怎样……

但是却不得不承认，大多数人很难逃脱环境的影响，这几乎 是一种宿命般的无从逃避。

在广州农村和亲戚们吃饭，饭桌上有两位年轻的母亲分别带 着一岁、两岁的孩子，吃饭时孩子有些吵闹，她们就把手机 拿出来立在饭桌边上，一边给看短视频一边喂饭，孩子马上 停止了哭闹，边吃边笑。

我委婉地提出，吃饭时看视频对孩子不太好，我四岁的儿子 插嘴说：“这样会看坏眼睛”。

我不好直言，看坏眼睛是其一、没礼貌是其二、破坏专注力 才是对孩子最严重的伤害。

抖音、短视频只有十几秒钟或者几分钟，都是最简单直接的 刺激，看完马上就得到一种快乐的感觉。而人的快乐是有阈 值的，没有付出就得到快乐，会让快乐的阈值变得越来越 高，人就不会愿意付出去得到更深层次的快乐，比如说—— 学习。

学习是一种需要付出，延迟才会得到的快乐，更深层次的快

乐。比起刷抖音看视频，要来得慢得多，难得多。当人习惯 了简单的快乐，还会愿意去做更难的事，去得到快乐吗？不 会了，他们更渴望即刻的快乐，而不是延迟满足——专注力 已经遭到破坏了。

专注力对于一个孩子，是非常重要的。如果没办法专注，几 乎不可能搞好学习。但凡那些有所成就的人，都是能持续专 注，学习能力非常强的人。

如果说以前农村的孩子只是物质条件差，那么现在，他们输 给了父母的无知。这么说可能有些伤人，因为为人父母就没 有不为孩子好的，他们希望孩子能开心快活。

就比如我妈，小时候总是对我要求严格，晚上先做完作业才 能看电视，并且不准超过 8:30，她的主要理由是害怕伤眼睛。 但有一阵她帮我带孩子，就会刷抖音给他看，一条搞笑的抖 音反反复复刷好几十次。

她不懂得，这样会伤害孩子的专注力，造成更加深远的影 响。只是觉得，孩子看了好得好开心啊！

还有就是大人们的娱乐，几乎都是打麻将打字牌。三五岁的 孩子跟着身边只能打滚玩手机，七八岁的孩子就会帮着切 牌，甚至“凑角”一起打牌。

小时候我常常抗议我妈，太爱打麻将了。她说：“我都已经几 十岁了，打牌不过是混日子，你别学我……”

从小耳濡目染、怎么可能不受影响。我的玩伴和同学，就没 几个到了假期不熬夜打牌的，他们说：不打牌干什么玩？

我不打牌，因为曾眼睁睁伯伯因为打牌不够过瘾而走向赌

博，最后倾家荡产，让堂哥堂弟早早辍学，年满十六就离家 务工。

现在，我的堂兄弟们，娱乐不是抖音就是打牌，他们的孩子 们也是如此。看着他们其乐融融，我居然感到有一丝悲凉， 曾经拿过奥数三等奖的堂哥，一辈子就只能这样了……

一个人要逃离环境太难了，如果可以选择，一定要为自己为 孩子，挑选和优化成长的环境。
