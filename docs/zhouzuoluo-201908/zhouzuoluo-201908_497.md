# 《以诚待人，你的人

《以诚待人，你的人生路越走越宽》

诚信，是一个人的立身之本。

01 以诚待人，更应以诚待己。 前天跟表哥聊天，聊到他最近职场上一些事。他进入公司已

有 4 年，这中间换了 3 个老板，员工走了一拨来一拨，只有他 一直没走。

前几年，公司管理不太严格，但他工作很讲逻辑，每次调研 数据别人看完就丢弃了，只有他汇总整理保存。

他每次做 PPT 都很认真，尽管做了没人看。也许，喜欢整理资 料，把它们分类保存，就这样在他的电脑里保存着。

几年后，公司调来一个经理，开始大范围整顿。这个经理需 要通过历史数据，分析原因为何公司亏损，可是其他人都说 没有数据。

这下他的 PPT 派上了用场。按年分类，按用途分类，各种格 式，清清楚楚，明明白白。

其实，他当初的想法是，就算没人看，也要把它做好，起码 不能骗自己。

因为他很清楚，环境散漫不是自己懒惰的借口，个人成长提 高不能丢。

以诚待人，更应以诚待己。 这就是，历经多年他深受 BOSS 器重的原因所在。

02 诚不欺人人自欺 表哥还说过一个事。经理刚来不了解情况，有的同事就趁机

编谎话逃避责任。事后被发现，搞得面红耳赤尴尬极了。

而表哥，却有一说一，不怕挨批评，反而很坦然。当发现问 题及时解决问题，而不是说谎话推卸责任。

诚信待人，你将收获成功。 自欺欺人，只会自取其辱。 03 以诚待人，路越走越宽

像表哥这样，从来不欺骗人。凡事，把话说在明处，人就喜 欢跟他打交道。

工作上，同事都想跟他搭档，因为他干活靠谱，常常跟别人 抢着干。他说自己吃点亏不要紧，但是不能欺骗同事。

下班后，朋友请他帮忙做视频剪辑，这个肯定是要收费的， 不能一直白帮忙。但是，他把钱赚在明处，摆在桌面上，价

格公道活又好，找他的人越来越多。

最近，听说又接一个网站维护的活儿，老板也喜欢他这样的 人。

你看，我表哥就是一个活样本。不是世界太阴险，而是社会 缺乏诚信。

以诚待人者，人亦诚而应。

诚信待人，以诚守信，一次次的所见所闻，让我明白，唯有 以诚待人，人生路才越走越宽。

以上希望对你有用。
