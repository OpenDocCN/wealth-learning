# 《燃点，只点燃自己

《燃点，只点燃自己是不够的》

我在经历第四家创业公司，看着老板在燃——野蛮生长第 65 天的彦初

今天公司团建，一起看《燃点》。看之前，我领导一直说想 看戴威，看他的反差。

看完后，我领导仍对戴威卷土重来抱有希望，我简单梳理我 的感悟。

01# 创业，是你点燃了自己心里的火 我的实习工作，毕业后第一份工作，都是知名外企，我很清

楚，我不适合待在大公司，所以，我离开，我进入一个又一

个初创公司。

我的老板们，都很燃。他们点燃自己的创业梦，像影片中的 草根企业家，安传东一样。

散场后同事们一起走，说安传东之所以会创业失败的原因， 有说是因为他项目不好，有说是因为他人性格不好，有说他 分析问题不好，也有说他就应该先去大公司打工。

批评别人总是容易的，安传东的项目被张颖说烂，全片也只 拍他的失败，他甚至贯穿整部纪录片，为什么？

因为，他创业的那团火，始终燃着，不管多难。

但是不够。

02# 创业，是你想把这团火烧得更旺 老罗被嘲笑，戴威跌落神坛，但是他们的火想燃遍世界。 重新定义个人电脑，让世界没有陌生的角落，他们的梦想，

不是常人所能理解的，所以也更容易被嘲笑，被否定。

影片想让观众觉得创业难，但其实生活中什么不难？各有各 的难处。

成功的那 7%创业公司，不敢说全部，但应该大部分的眼光是 看得更远的，而不局限当前的项目。

所以傅盛看到了机器人行业，马薇薇知道生产内容同样不 够。

03# 创业，是你需要点燃更多人陪你 傅盛说他没法跟人交流自己内心关于创业的焦虑，家人不

行，同事也不行。

这种焦虑，会一直伴随他们。 有一次和我上任老板一起吃宵夜，她尽情地喝，喝醉了尽情

地喊。

当时另一个股东，也是老板的闺蜜就说，你看，这就是创业 的压力。

后来这个闺蜜也离开公司了，一个个股东相继离开。老板曾 经点燃了他们，陪她一起闯，也看着他们熄灭。

这种经历，我无法感同身受，但我看着，思考着，创业，究 竟是怎么一回事？

《燃点》为了让观众对创业者感到共鸣共情，刻画创业艰辛 的一面更多，这部纪录片，很难，也很有意义。

但我认为，创业不是只有难处，还有很多让创始人，让团队 感到成就感的部分。影片没有记录到这些，又怎么算纪录片 呢？

创业成就感和满足感不是鸡汤，它们也可以是燃点。
