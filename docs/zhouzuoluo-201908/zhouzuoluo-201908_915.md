# 《要是...就好了

《要是...就好了，这是不可能的；但你可以“要是...就能”》

往者不可谏，来者犹可追——第 153 天的星里浣 有一种思维模式是：“要是怎样怎样，就好了。”

要是我生在一个有钱的家庭，就好了；要是我以前好好学 习，就好了；要是我当年上了一个好大学，就好了；要是有 后悔药，就好了......

今天晚上，突然就想到了几年前和朋友聊天的那个晚上，至 少说了几十句“要是...就...”那时候特别愤青，看整个世界都觉 得不爽，自己没有拥有的都是因为：当年没有“要是...就...”现 在想想，那时候的想法真的太幼稚了。如果世事都可以“要 是”，那每个人不都活成了一样，没有追求了么，那人活着还 有什么意思呢。

这背后的想法，就是想要一个不用负责任的人生，想要一份 不会踩坑的特权；那是那样，你不觉得是在做梦吗？

虽然我们没法让过去“要是”，但是我们可以对未来“要是”。 你想想，要是你现在努力赚钱，以后不就有钱了吗？要是你

现在好好学习，以后不就能学会更多吗？要是你现在好好上

学，好好工作，以后不就能去到更好的学校，找到更好的工

作吗？

我们总喜欢回望过去，看到那些失败的决定，却忘了把握当 下，想想以后的路要怎么走。因为没有看到前方的路，思想 便停在以前的维谷中绕不出来。

我记得小时候奶奶经常对我说，你要好好读书，以后才能有 出息。那时候我只是含糊不清应着。现在我突然明白了，要 是我以前好好读书，现在真的会比现在有出息。但，往者不 可谏，我现在也无法回到过去劝说那个“不听话”的小孩；不 过，来者犹可追，我还能从现在出发，去追上未来那个“有出 息”的自己。

要是...就好了，这是不可能的；但你可以“要是...就能”。共 勉！
