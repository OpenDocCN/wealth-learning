# 《学习时，警惕这种

《学习时，警惕这种思维方式：这不就是 XXX 吗？》

开篇给大家讲个小故事：

有一次，我请邻居家的大爷出去吃饭。我说，我们去吃点新 奇的东西。于是，我带大爷来到一家西餐厅，这家店里是卖 披萨的。进了餐厅，我给大爷点了一份披萨。等到热乎乎的 披萨上桌时，大爷顿时不开心了，他说道：这不就是大饼盖 肉吗？有什么新奇的？

没错，披萨和大饼盖肉确实有一定的相似之处，但是，对我 们来说，发现披萨和大饼盖肉的相似之处，本来不需要学习 能力，大脑立马就能用旧知反应出来。

而如果我们能刻意辨析二者的区别，才是学习能力的体现。 比如说：披萨跟大饼盖肉的原料都有哪些不同？做法各有什 么特色？什么样才算披萨？什么情况才算大饼盖肉？

为什么大饼盖肉才卖十块？披萨能卖 30？是什么让披萨卖到 了全球？而大饼盖肉只存在于中国北方某些地区？这些问题 都是在界定边界，也就是二者的区别，到底在哪里。

我们更多的是要看新知与旧知的不同之处。看到新东西，就 拿大脑已熟悉的东西附加解释，这就是在附会旧知。这本质 上是一种思维模式，叫做用“已知”去附会“新知”，以为自己看 透了，读懂了，理解了。

这里还有几个例子：

1、某个概念，不就是我从另外一本书看到的吗？

2、书里讲的东西不都是老生常谈吗？

3、尝新食物，人们的反应：这不就是 XXXX 吗？

那么，我们为什么会习惯于“附会式学习”？ 因为，一方面，人们在遇到陌生事物的时候，会感到不适，

在心理学上称之为失调，这时，如果把陌生事物强行理解为

我已经了解了的事物，就能够快速解决那种不适，为此，人

们宁愿忽视这二者之间的差别。

另一方面，人们从小就被训练，“归纳中心思想”，强化了人 们附会的习惯。所以越是学历高，越是博学的人，越容易止 步于附会，附会的思维模式，是我们提升理解力的障碍。

要检验你是否在附会旧知，很简单：你把你学习的新知，按 照自己的理解写下来，然后对着你写的文字，扪心自问：如 果我没有读原文那段话，我写下的那些文字的内容，我是知 道还是不知道，如果答案是，在此之前，我知道，那你很有 可能就是在用旧知附会新知了。

那么，我们怎么破除这种思维方式，提升我们的理解能力 呢？很简单：费曼技巧！

这是我在星球上第三次提及费曼技巧。我之前介绍了费曼技 巧的作用，没有说为什么要用它。其实，它就是用来解决我 们理解不够的工具。学习的七层模型，最高层就是教给别 人。会用费曼技巧，不仅能帮助我们加深对知识的理解，更 加能让我们将知识教给别人！
