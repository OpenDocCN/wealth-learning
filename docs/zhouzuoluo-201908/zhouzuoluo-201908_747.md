# 《人生那么多不甘心

《人生那么多不甘心的事，你怎能甘心被社会抛弃？》

和朋友在群里聊天，聊到 2019 年开始实行的“六项专项附加扣 除和扣缴”的事情，让我很不解的是一个工作了 3 年多的人， 怎么可以全然不知漠不关心？

我把“六项专项附加扣除和扣缴”的相关文件说明分享到我们 的群里，但是我的一个朋友一句“看不懂”，让我陷入沉思。

是什么让一个同样工作几年的人全然不关心这些政策？难道 父母不会老吗？不需要赡养？还是从不打算买房住？亦或是 孩子永远不会长大，不需要上学？

后来解释说到要达到税收起征点 5000 元后才有用，她说“反正 我的工资两三千，达不到的。”让我回想起那篇曾经火遍社交 圈的文章《时代抛弃你的时候，不会说一句再见》。

曾经，我和她一起说过不甘心在省内上大学，然后我坚定自 己的选择来了北京，她却被家长束缚留在省内。

曾经，我和她一起说过不甘心在小地方工作拿着微薄的工 资，后来她还是选择回到了家，过着不甘心的生活，直到现 在斗志全无。

曾经，我们一起说过不甘心，说过将来要靠自己的能力买 房，可是现在我却深深感觉到，我们在渐行渐远。

我替她感到无奈惋惜的同时，想说，能阻碍自己进步、成

长、向更好的方向发展的人，只有自己，只有自己，只有自 己！重要的事情说三遍不嫌多。

当一个人失去了向上生长的斗志，我理解即将向人生的低谷 奔去，这只会是一个恶性循环。就像航海路上看不到灯塔， 只会迷失方向。

最可悲的即使知道自己不甘心拿两三千的工资，但怎么就甘 心拿着这样的工资？怎么就甘心被社会抛弃？（这句话有点 绕口，需要慢慢体会）

我想，没有大胆想过未来的人，不配拥有美好的未来。这一 点都不鸡汤，我问过一些在北京买了房的人，类似你是怎么 能在北京买房的？他们都是有坚定信念在北京定居下来的 人。这，是不甘心。

在北漂的话题讨论中，从来不缺说过几年还是要回家的人， 也从不缺说实习或者毕业我就要去北京的人。这，就是不甘 心。但是社会抛弃你的时候也不会说再见！
