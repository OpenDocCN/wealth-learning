# 《高小羊：鲁莽的人

《高小羊：鲁莽的人，更容易成功》

人生总有很多左右为难的事，如果你在做与不做之间纠结， 那么，不要反复分析推演，立即去做。莽撞的人反而更容易 赢。

因为如果不做，这件事永远是停在脑中的“假想”，由于没有 真实的反馈，诱惑会越来越大，最终肯定让你后悔，而去 做，就进入了一个尝试，反馈，修正，推进的循环，最终至 少有一半概率能做成，不后悔。

我从这个鲁莽定律中获益颇多。记得刚毕业那会，工资不 高，够维持基本生活费用。当时体重偏重，就想改变一下自 己的形象，在纠结要不要报一个健身房时，看到这条定律， 感觉脑子被雷劈到了，立即刷了卡里一大半的钱报了健身 房。

后来发现我并没有因为办了卡而吃不上饭，交不了房租，因 为当你开始运动时，就会在其他花销上比如吃饭，省下一半 的费用，因为你不会再沉迷外卖，快餐和高热量食物，而自 己挤时间动手做饭是一件非常低成本的事。

当我们在选择做还是不做时，就会有比较，而选择的前提是 你要把要比较的东西，放在秤的两端去对比，最终选出对自 己有利的一面。

在比较的过程中，你不仅要看到那些能看到的东西，同时也

要看到那些看不到的东西。比如：

选择不去运动，你看到的是痛苦，汗水，看不到的是，运动 后身体素质的提高，体型的改变，性格的磨练。

选择不去学英语，你看到的是交了学费后可能学不好，看不 到的学成后工作能力的提升，工作前景的延伸。

选择不离职，你看到的是找新工作的麻烦，看不到的新工作 带来的圈子的改变，视野的拓展。

又比如，选择走国道，看见的是省点油费，看不见的是轮胎 的磨损。

花好几个小时在各大购物网站比对价格，看到的是省点钱， 看不见的是时间偷偷从指缝中溜走……
