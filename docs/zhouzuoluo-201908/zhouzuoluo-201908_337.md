# 《不知道这个方法，

《不知道这个方法，你永远只能不停的工作，挣得钱却永远

不够花》

晚上好 与星友们分享丹儿的第 42 篇原创 6.3

————————————————————————

好朋友六月中旬结婚，打电话过来让我回哈尔滨参加她的婚 礼，祝福的话还没有说出口，她就和我抱怨，结婚收的礼钱 可能还不够还因为举办婚礼所借下的贷款。人家结婚是幸福 生活的开始，她结个婚却欠了一大笔债等等……

【为什么现在越来越多的人不想结婚？】 现在越来越多的人不想结婚了，你身边有不想结婚的人吗？

有因为婚礼要办什么样的？婚纱穿定制的，还是租的？彩礼

给多少合适？男方结婚一定要买房买车？这样的问题而闹翻

的情侣吗？

这些问题就造成很多人不想结婚，没提结婚前两个人的感情 是多么好，但一遇到这些现实的问题却不能牵手好好的过一 生了。

归根结底这些问题的根源其实还是一个字“钱”，虽然钱不是 万能的，但是没有钱确实也存在很多现实的问题。就像我马 上要结婚的闺蜜，她老公人其实很上进，很踏实对她也很 好，家务活也全包，但她为什么还是对人家不满意呢？还是 因为没钱！

【没钱的状态就是这么现实】 这么说也不是因为她有多么在意钱，如果真的在乎钱，就不

会和她老公结婚了，她感到难受的只是因为没钱她未来的生

活会有多么的困难，可能有的人会觉得她想的太多了，不要

要求太高，知足者长乐这些，但是你不在那个角度永远体会

不了她的心情。

我也经常开导她，但是她最常说的话就是“你不是我，你不理 解我的处境。你爸妈有钱，你可以依靠他们，我爸妈也没 钱，我对象也没钱，我只能依靠我自己。”

婚后她和他老公，每个月需要还车贷，房贷，和她老公的信 用卡账单。这些贷款每个月需要还 8000 多，他们的工资一共 15000。除去应酬吃喝，是没有积蓄的。

如果我的朋友怀孕了，她还需要继续工作，因为她不工作他 老公的工资只够还贷款。养活不了她，以后孩子还会有教育 方面，吃穿方面的费用，他们的工资是不够的，这还没有算

老人的赡养费，因为他们两家家庭都一般，父母经济上帮不 上什么忙，他们的压力就很大。这种家庭其实在社会上还是 有挺多的。

他们的生活都很拮据，挣的钱刚好够花，他们一刻也不能停 下工作，因为一停下工作就没有工资，无法偿还账单。长期 下去我们就会越来越焦虑，慢慢被账单支出压的喘不过气。

【生活拮据的根本原因是什么？】 他们现在的现状大家有好的解决办法吗？可能有的人会说，

可以努力工作提升工资赚钱呀？你们现在的工资比刚出身社

会时赚的多吧，钱攒下来了吗？

可能有的人比较节约还是攒下来了，但是大多数还是挣多少 花多少，不仅如此还有信用卡，花呗等，如果大家挣得钱都 够花，就不会有那么多人去马云微博底下评论，马爸爸，给 我还花呗了

包括我自己也是，挣多少花多少，挣得多了生活质量也提高 了，买的东西也贵了，就攒不下钱，但是我学习了《富爸爸 穷爸爸》以后对财务管理有了一定的了解。做了一些改变， 上个月我的财务报表显示，我终于有积蓄了，省了一大笔 钱。

★让我们生活拮据的原因是，我们的收入和支出是平衡的，

所以无论挣多少钱都不够花，而我们的支出花在哪了呢？

买入的东西可以分为两类，资产与负债。

①资产是买入以后能让你的钱生钱的东西。比如说能让我们 进步的东西，我花了 299 元，进入知识星球，写了四十多篇文 章累计打赏 259 元，才一个多月哈，星球有一年的有效期这样 下去，很快就能赚回来花的钱还会不断的收益。

这只是金钱方面的，还有事业方面的，我因为坚持写文章不 断的输入，增强的自己知识储备，输出时增强了我的逻辑思 维能力，和表达能力，不仅是能写我也能说，所以很快在单 位受到了领导的赏识，要提拔我做新店的店长。因为表达能 力强了嘛，还学习了心理学所以销售能力也强了，工资上涨 的很快。

还有一方面是副业方面的，因为总看书，我就做了个代理， 我把看过的书都写了总结有的发在朋友圈，还有很多朋友找 我买，带来了额外的收入。

这就是支出用在资产项给我们带来的好处。

②负债是买入以后没有任何价值的东西。 比如说当季流行的衣服鞋子，很容易过时，明年就不会穿

了，买完穿几次就没什么用了。如果要买最好买质量好一

点，以后拿出来穿也大大方方的，一件有一件的价值。

还有网红安利的各种东西，各种打折商品。觉得自己都用的 上，买回来用过几次呢？

★我们生活拮据这个问题的根源就是我们的消费观不正确， 不要赶潮流去买那些你不需要的东西，把钱花在资产项上， 才能慢慢改变我们生活拮据的现状。让你的收入大于支出， 才能把剩余的钱更多的投入资产项，当资产项的能产生的现 金流比工资高时，我们就不用在为了账单不停的工作了。

当然有了正确的消费观后，也不能彻底改变我们拮据的现 状，最重要的一点是要怎样找到资产项，从而使它产生比工 资高的现金流。

【那么怎样做才能找到资产项呢？】 注重积累，趁着我们还年轻没太大压力的时候学会积累，才

能使我们中年时上有老下有小不处于焦虑的状态，到那时，

你想学习都没有时间了。

积累分为三方面，一方面是自我提升的积累，一方面是钱的 积累，一方面是人脉的积累。

①自我提升的积累：自己不断的进步，能提升自己的认知能 力，更容易找到机会。并且抓住机会。

②钱的积累：当你看准机会，没有钱时也是不行的。所以年 轻时我们一定要有正确的消费观，有自己的积蓄。

③人脉的积累：这个人脉是指比我们优秀的人，从他们身上 我们能学到很多的东西，而且他有机会时还可能会带你玩。

我最近就得到了一个机会如果能抓住可以少奋斗十年。可是 我以前的积累不够，没有百十来万的积蓄呀，就错过了这次 机会。如果我以前注重人脉的积累可能，遇到机会时也会有 些人愿意借给我这么多钱，我也能抓住机会。

不过也没关系，这个机会没抓住，以后还会别的机会这个事 情只是让我了解到积累的重要性。

✔玩过现金流游戏吗？有两个圈，一个穷人圈，一个富人 圈，穷人圈里一直走不出来的人就是自顾着自己手里那点工 资，没有向外看寻找机会也不注重积累，放身边有人跳出穷 人圈时每人愿意帮助他，而那些跳出穷人圈的人，都很注重 积累，人脉资金，通过资产项购买获得积蓄，看到机会时身 边人也愿意帮助他。所以积累是很重要的。

▼总结一下，让我们苦恼的原因大多数是因为钱，怎样让我 们的生活变得不拮据呢？确立正确的消费观，让收入＞支 出，购买资产项。注重积累，自我提升的积累，人脉的积

累，积蓄的积累。这样我们才能在机会来临时抓住机会。
