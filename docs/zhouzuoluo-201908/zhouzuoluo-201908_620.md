# 《高小羊：凡是能用

《高小羊：凡是能用钱买的东西，其实都是便宜的》

我在之前的一篇文章解释了什么是“财富自由”。 财富自由，从构成来看，是财富和自由的组合。财富是工

具，是手段，是用来帮我们最终获得自由。

自由，换一个说法，就是我们可以随心所欲的生活，任意掌 控我们的节奏。换句话说，我的时间，只能由我自己来操 控，那自由，就是时间。

用李笑来的话说，财富自由就是我们再也不用为了生活，而 要出售我们的时间。

那时间和钱那个贵重？ 回答这个问题之前，先想一个问题。

为什么会存在外卖，家政这些东西？因为你如果不点外卖， 就要自己做饭，但是你又不想自己做，因为你没时间，你认 为用来做一顿饭需要的时间的价值远远不如去处理一件事情 的价值大。

所以你用钱去点外卖，买的是别人花时间给你做好的外卖， 给你送来的服务。也就是你用钱买了别人的时间。

用 a 买 b。

那肯定是 a 便宜，b 贵。 因为这样才划算，不然谁愿意用一个贵的东西去买一个便宜 的东西。 所以用你的钱去买别人的时间，而目的是为了省下自己的时 间。

那就是你的钱便宜，你的时间贵。 所以金钱没有时间贵。 反过来说，时间比金钱贵。 我们把这个问题引申到其他方面。

比如现在很热门的知识付费，用钱去买别人的知识，其实就 是在捡便宜。

当然前提是你要有慧眼识别它是否值得你去买，因为现在是 内容创业时期，门槛并不高，人人都热衷出入其中，创造出 来的内容有的是粗制滥造，有的是精工细作。

我在写作之前，也曾买了不少这方面的书籍和课程来看。看 完学完之后，你就能一眼看出是垃圾还是宝藏。

如果有幸捡到宝藏，那恭喜下自己，捡到便宜了。 因为我们是站在前人的经验里来成长。 前人已经帮我们过滤了不必要踩的坑，不必要绕的圈，并传

授了他成长的经验。

这样的结果就是帮我们省下了踩坑绕圈所要浪费的时间，同

时助我们能快速到达目的地。

所以，能用钱买的东西，其实都是便宜的。
