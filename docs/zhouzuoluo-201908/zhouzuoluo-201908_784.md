# 《越没有私心，得到

《越没有私心，得到越多》

人到底是不是自私的？——野蛮生长第 89 天的彦初 我妈妈做手工活，有些收尾工作需要花钱让街坊邻里帮忙完

成。

昨天我 10 岁的表弟来家里，我妈就叫他帮忙，他很乐意。 我妈妈跟他说，你每完成一条绳子，就能赚五毛钱。 表弟的回答，让我意想不到。他头都没抬起来，一边忙活一

边奶声奶气地说：“我不要钱，我们是一家人，不谈钱的。”

我妈当时听到肯定觉得很贴心，当天晚上就带我表弟出去超 市给他买了他最喜欢的玩具。

我的表弟不求我妈妈给他什么，但他却意外收到了他喜欢的 东西。

有些时候，你会发现，一个人没有私心，反而得到更多。 历史上有位大臣的职场生涯，就是这样的。他是张廷玉，清

朝康乾盛世时期三朝元老，是雍正皇帝的心腹。他在辅佐雍

正皇帝的十三年间，不求名利，只求把事情做好。

而他越是没有私心，雍正皇帝就给他越多。

他不求加薪，皇帝就越主动地给，生怕他生活得不好。他不 求功名，皇帝就赐他百年之后可配享太庙，这可是上等的待 遇。

可是这样的名臣，最终还是不得善终，是因为他在八十岁的 时候，太过于求得一个安享晚年。

乾隆可没有像他爹一样欣赏张廷玉，张廷玉越想退休，乾隆 皇帝就越不让，他越想配享太庙，皇帝就越觉得他不配。最 后，张廷玉一生的英明，落得个名誉扫地，

做人吧，有些时候，还真的不能要得太多，往往在不要些什 么的时候，那样东西正慢慢地向你靠近。

比如粥左罗老师的闭关的那一年，他没想要什么出名，或者 什么钱财，他只想着把技能练好，最终，都有了。
