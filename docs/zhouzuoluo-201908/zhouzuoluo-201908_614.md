# 《关于选择，我们手

方小华 : 《关于选择，我们手握权利但从没有回头路》

今天无意间在知乎回答了问题，竟收到 200 多点赞，几十条评 论，对于新手来说还是很惊讶的，于是对于自己的答案又做 了更多的思考。

题目:最后悔的一件事。

答案: 10 年高中毕业，没有听父母的话复读，上了大专也没有

认真学习，花了十年的时间去寻找心底的答案，原来成长的

代价是在这里。希望未来越来越好。

其中有两条评论是这样的:同样承受着同样的代价。没有好好 读书导致现在寸步难行，后悔也来不及了，只愿未来不辜负 自己。是的，这世界许多事后悔都来不及了，我们承受着自 己选择后的结果，但事物却又都是两面性的，没有回头路我 们却有重新选择努力的权利，我想这个回答之所以有一群人 产生回应一定是我们的现状有着或多或少的相似和痛点，明 白后悔了、明白以前的也改变不了，但却并不是所有人也就 能明白了假如现在依然不开始努力，未来也一定还是会辜负 了自己。

对于选择，我们手握权利。却很多人不珍惜自己的选择权， 在结果面前却又抱怨。何老师在采访中曾说过一段话:

如果你真的觉得很难， 你坚持不了。 那你就要放弃， 但是你放弃了。 就不要抱怨说，

我为什么没有得到。 所以我觉得人生就是这样， 要得到就必须要付出。 要付出你还要学会坚持。

不管当下的生活如何，珍惜你手上的选择权，不负自己不负 时光。
