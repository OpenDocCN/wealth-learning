# 大家好，我是争青泊

大家好，我是争青泊

--遇强则强。争做优秀 80 后，与自己死磕到底。

这是我在星球连续写作的第 37 篇

《持续输入，更会有效解决你的“固执”》

很多人都会在某件事上，被人说：“你真是好固执啊”。而自 己却还把这种“固执”，解读为是一种“执着”，做事情坚持己 见。

我曾经就被身边的朋友贴上过这样的标签，随着年龄与阅历 在增长，慢慢的对一些原来坚持的，逐渐有了多种解决它的 办法。可是，一个人的固执真就会随着年龄、阅历的增长而 自然的在改善、变好吗？还真不是。

举个例子，他是我的长辈，他的职业是传统出租车司机，给 人打工挣着辛苦的“卖腕子”钱。虽然说车不是自己的，但他 很享受这种“起早不贪黑”的工作模式。

可自从滴滴出行刚开始风行起来一直到现在的日趋成熟，“滴 滴出行”已然是时代的趋势，更成为了年轻人出行打车的首 选。它不仅是一种社会需求，更是一种新时代人们的底层诉 求。

而我的这位长辈，一直就看“衰”它。认为这样一种出行打车 方式。一来，没有保障，不像原来的出租车公司，早晚班轮 班驾驶，挣多挣少心里有数。二来，不安全，司机什么样的

人都有，从业素质、驾驶技术都没法评判。滴滴确实前段时 间出现过恶劣事件，他得知后，就更加相信“这种方式”肯定 不行。

以前我还就着这个问题，跟他辩了几次，但最终我还是失败 了。无论你怎么讲，他都听不进去，表现得异常固执。我 想，最主要的还是，他本人不会使用微信、不会使用各种 APP，智能电话到他手里，也只有拨和打两种功能。说不好 听点，他已经被社会淘汰了。但他所担心的是，滴滴会抢去 他的“市场”，让收入越来越难赚。

一个人的认知水平越低，想法就变得越单一，也越抗拒改变 和接纳，因此，就会表现的越“固执”，和他的年龄无关，但 和阅历少、经历少、见识少有关。就像汽车在没有发明前， 人类只想要更快的马车一样。

如果，要把“固执”分类，我的这位长辈所以后者，是被社会 淘汰而形成的固执。而像我们这样的年轻人，更要警惕“早期 固执”，就是时代在进步，而我们却在原地踏步，学的少、看 的少、逃避实践的累积，过着自嗨的生活。

所以，我们只能多读书、多了解、多思考，当我们的经历多 了，输入的多了，认知提升了，我们就会离“固执”远些，才 能有方法、有目标的去“执着”地拥抱未来。
