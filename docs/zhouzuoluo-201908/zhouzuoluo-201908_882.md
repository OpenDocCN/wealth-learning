# 《你是“有用之人”

《你是“有用之人”还是“无用之人”》

学习的目的，是为了改善我们的思维——第 160 天的星里浣 记得我们小学的数学课本里，常常会出现这样的题目：一个

水池，注满水需要一定时间，然后一根管子注水，一根管子

放水，注水管的流量比放水管稍大，问注满这个水池需要多

少时间。

我当时对这种题目嗤之以鼻，认为只有脑子进水的人才会这 样做；谁会没事往水池里一边注水一边放水啊。现在我才明 白，脑子里进水的人，是我。

我只看到了一边注水一边放水的“无用行为”，其实背后的计 算方法，思考方式，才是我们应该看到的“有用知识”。

还有，我以前经常想：加减乘除足以应付日常生活，为什么 还要学函数方程？我又不上天入地，不搞发明创造，物理化 学又有什么用？

这就是典型的“思维固化”，这种思想远比阶层固化可怕。也 就是“无用之人”的思考方式。

你觉得函数方程没有用，所以只能过加减乘除的生活；你觉 得只有需要上天入地，搞发明创造的人才去学物理化学，所 以你也不可能有上天入地的机会，搞发明创造的可能。

孔子说：“唯上智与下愚，不移也！”

这个世界，阶层的固化，其实是智力的固化。智慧永远是向 上的，无论身处哪个层级，都会想突破，去改变。那些不想 上，也不愿下的，其实是最愚昧的；因为即使维持现状，也 需要付出全部身心去对抗现实。

有人觉得认识的字没用，学到的知识没用，这不就和那句“听 过很多道理，却依然过不好这一生”一样吗？

不是知识没用，而是你没用；不是道理的问题，而是你的问 题。

学习的目的，是为了改善我们的思维。向上攀登很不容易， 但维持现状才是最难的。别让自己成为“无用之人”，你要知 道，到处都有我们需要的“有用知识”。
