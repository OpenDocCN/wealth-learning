# 《都挺好：从这部剧

《都挺好：从这部剧中你要懂得的 3 个道理》

没有人能不受伤走一生，如何抚平伤口重点看你，而不是伤 害你的人。——坚持日更第 134 天的彦初

46 集的《都挺好》看完了，剧里出现四五次“都挺好”都是假 话，但最后的结局却真的都挺好。

苏明玉最后放弃事业，放弃栽培她十几年的师父，选择了伤 害了她十几年的原生家庭，选择陪伴生病的父亲，重拾家的 温暖，这份温暖，等了十几年。

我无法想象如果我是苏明玉，在这样的原生家庭长大，我会 有怎样的人生，我会如何和父母相处。

看到最后苏明玉看着妈妈抱起小明玉的镜头，我甚至在想， 这是真实的回忆，还是明玉美化了和妈妈的记忆。她期待的 母爱，已无法重新再来。

剧看完了，情感总结结束，还是得来思考值得我们学习的地 方。

01# 不要用他人的过错，伤害自己 苏母重男轻女，二哥有样学样也欺负妹妹，妹妹十几年无法

释怀，每次家人见面只会加深矛盾。

《小偷家族》里女主对小女孩说，你的爸妈打你，不是你的 错。多少虐童父母会让小孩形成无法逆转的伤害，不是生 理，更是心理。

你也许这辈子都无法原谅打你的父母，无法原谅出轨的伴 侣，但你不能因为他们就不再相信亲情和爱情，不仅对其他 长辈和未来的伴侣不公平，更对你自己不公平。

02# 不要轻易对他人做出评价 苏家全家人都说苏明玉性格不好，心狠手辣，二哥更称她是

毒水母，不管她做什么事情，都觉得是在陷害他，离间他和

他家人的感情。

而外人更觉得苏明玉冷漠，不近人情，眼里只有利益，没有 情感。

你轻易地负面评价一个人，但你不知道她过去承受过什么， 甚至是亲人，都不可能做到完全了解。

03# 你能否发现改变你命运的人？ 今天老板放一段剧情给我们看，大学生苏明玉遇到恩师老

蒙，他对她的一番开解，从此让明玉跟他打拼十几年，重建

自己的精神架构，脱离原生家庭的束缚。

如果是你，在派传单的过程中遇到一个大叔对你罗里吧嗦， 会不会觉得他是有病吧。

回想我过去遇到的领导们，有真心教我做人做事的，但我不 愿继续学习，也许我错过了我的贵人。

但错过的，你可能无法追回，你能做的，就是珍惜当下。如 果你不讨厌你的领导，请努力向他学习。如果你讨厌，也要 让自己学习，多学点总归是为了自己，让自己成长起来，遇 到贵人你才不会错过。

看完点个赞呗嘻嘻——很高兴和战友们一起野蛮成长，我是 彦初。

让自己每天进步一点，365 天后你就是你最大的惊喜——让优 秀成为习惯！
