# 《6 个法则教你处处

《6 个法则教你处处令人喜欢》

都说销售有三寸不烂之舌，我很喜欢这个职业。说句不要脸 的话，本人有一种神奇的能力让客户在首次见面就喜欢我， 无论男女老少，这点对个人销售业绩很有帮助。

今天阅读《人性的弱点》文中讲到“如何让别人喜欢你”几项 甚是赞同。如果你想使人喜欢你，就要牢记这个原则：真诚 地赞赏他人，让他人感到自己很重要。

小结：

1\. 微笑是拉进陌生人距离的秘密武器。

2.人类的行为有个极为重要的法则：时时让别人感到重要。

3.人类最殷切的需求是：渴望得到他人的肯定。

4.现实生活中有些人之所以会出现交际障碍，是因为他们不懂 得或忘记了一个重要的原则：让他人感到自己重要。
