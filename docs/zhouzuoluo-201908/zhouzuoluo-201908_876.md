# 人到中年的你已经没

人到中年的你已经没有资本谈迷茫

最近在某订阅号上看到一篇文章，题目是《人到中年，在撕 裂中前行》。看完的感觉犹如喝了酒和醋的混合物之感，抽 风的难过、霹雳的辛辣、又有淡定的沉香。这段话有人称之 为“醒世恒言”，我看不尽然。

年复一年、日复一日的平淡流年，你是否已经处于波澜不惊 的泥沙状态，还是依然保持着高昂奋进的向日葵状态？你是 否也在撕裂中前行？你是否也曾经在选择面前呐喊和彷徨？ 生命里你最在乎的东西到底是什么？

这篇文章作者笔触犀利、写得非常现实，几乎是很大一部分 人三十五岁以后人生的真实写照。一边是马云、一遍是星 云；一边是位子，一边是孩子；一边是超人，一边是老人； 一边是上流，一边是逐流；一边是同学会，一边是追悼会； 一边是养老，一边是怕老；一边是庭院，一边是田园；一边 是“在路上”，一边是“故乡”，这些矛盾这些纠结想必大家都经 历过。

看完马云的演讲你会热血沸腾，试图重新点燃心中曾经的梦 想；看完星云大师的劝世恒言，又叹息万般皆身外，何必苦 相争；生完孩子，就会威胁位子，一曲《爱拼才会赢》让我 们心潮澎湃，一曲《心升明月》又让我们心生倦意……钦佩 此文章作者的视界不一般，能用如此贴切的语言总结得如此 深刻而到位。

NO.2

撕裂的欲望给我们带来诸多苦恼，我在叩问自己的内心，为 什么当下的中年人会卷入如此的状态？我们到底想要什么样 的生活？人到中年，对于一个负责的人生来讲，你完全可以 不用那么撕裂。

工作固然琐碎，生活固然平淡，还是有人把工作做到了极 致，把日子过出了诗意。用心和不用心，主动和被动的结果 绝对有所不同。心态决定了你的未来走向，要做的事情那么 多，你哪有那么多精力低吟伤感。

你会发现靠近干净的圈子、优秀的人，就是快速成长的一种 捷径。投资大神巴菲特给年轻人两个建议：“接近成功人士， 让他的想法影响你；走出去学习，让世界影响你。没有贫穷 的口袋，只有贫穷的脑袋”。还真的不是毒鸡汤，我们纵使没 有机会接触到成功人士，但可以接近比我们有优秀的人，没 有机会走出去，可以借助互联网时代的新型学习模式。

某日，我翻着这么多年来自己保存的工作记录，密密麻麻记 录了我的很多回忆；翻着一张张积累下来的照片，心中涌起 无限的感慨。时光如白驹过隙，确实有怕老之嫌啊。但是唯 有坚持努力，不负时光的流逝，让我有感于脚是踏在大地上 的安心和欣慰。

佛说，生容易，活容易，生活不容易；一点不假，如何将不 容易的生活过出高山流水的高逼格呢？那就是付出，一个舍 得付出的人，注定是一个快乐的人！你付出爱也会得到爱， 你付出快乐也会得到快乐！

NO.3

个人的行为源自于内心深处的思想导向，也许为了某种目 的，你可以装一天装两天，但是让你永久装下去你是装不出 来的。

曾遇到一个我最钦佩的老师，当时教大三的计量经济学这门 课。班上有个冒牌学霸，据说出身官宦，来头不凡，可以说 只要谈宏观经济他似乎无所不知无所不晓，把那些不学无术 的老师，忽悠得跟着他的节奏原地打转。这也可以说是他的 一种气场和能力吧，唯独计量经济学课，这位老师会和他一 直探讨，直到他靠自己那些忽悠的功底再也答不出为止。

他说过的那句话，我一直都记得，他说：“同学们，我们这些 学经济专业的学生，大学四年毕业后，能记住几个概念，诸 如恩格尔系数、无差异曲线就不错了，因为很少有人真正为 弄懂一个概念，奔着学以致用的目的而努力的。我的课堂容 不得你们马虎，忽悠在我这里过不了关，这样至少你们能真 正学到一门叫计量经济学的专业课程”。这样的老师，能遇到 是福分，他传授给我们的不仅仅是知识，还有做人的原则。

最近的朋友圈文章，都在谈层次，谈人生级别，诸如越优秀 的圈子越有趣，越优秀的圈子越懂得生活，层次越低的人越 爱抢红包，层次越低的人越不饶人……看上去似乎是毒鸡 汤，你用心观察后发现还真不是。

那些感叹时光流逝又无比恐慌的人，往往是那些人到中年， 创造的资本没有赶上人生时间表形象进度的一族。抛去那些 因先天出身而坐拥资源的优势外，那些同年毕业的人，有的 人在为坚持达成人生目标而安排的档期满满，有的人仍然拿

着不死不活的薪水掰着时钟等下班，这样的人生不撕裂都说 不过去。

NO.4

在我看来，少钱缺爱的中年人生最可怜；沉迷于游戏的男 人、沉迷于麻将的女人，念叨着那么多“要不是”的人最可 恶；当你爱的人需要你时，无能于给予和付出的人最可悲。

别傻了，你的人生从来没有经历过马云的激情，如何坐享星 云的恬淡；没有爱拼才会赢的打拼，何谈心生明月的矫情， 没有在路上的拼搏，何谈回故乡的骄荣。

看着越来越多的人因为没有技能，下岗失业，日子过得捉襟 见肘，我越来越在乎我拥有的工作；看着越来越多的人因为 利益导致婚姻破裂，法庭上针锋相对锱铢必较，甚至母子父 子反目成仇，我越来越在乎我的幸福家庭；看着越来越多的 曾经朋友，因为纷争明争暗斗、弄得你死我活，我越来越在 乎我的朋友对我的真诚！

放下攀比，放下抱怨，工作时全力以赴，人生踩着台阶一步 一步向上走；开启问答赞良好沟通模式，远离暴力沟通模 式，沉下心陪孩子陶醉的读一本书，讲一个她爱听的故事； 理解父母的唠叨，抽时间陪他们涉足一趟远行，让他们的人 生也不仅仅辽阔于空间的想象。

珍惜有人爱、有事做、有所期待的人生吧，努力做好该在乎 的事情，珍惜该在乎的人，提升水准、积累量级，丰盈自己 的内心，人到中年的你已经没有资本谈迷茫。
