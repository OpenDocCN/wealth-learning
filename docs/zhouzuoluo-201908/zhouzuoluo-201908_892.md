# 《定下初心，写出来

《定下初心，写出来的东西就不差》

用初心这个词不是我的本意，但也没有更好的词语来替代它 了。毕竟如果用张小龙的源动力，语句就不大通顺了。所以 这里用了初心，但还是要说下，初心确实很难找。

1.想写作的由来 今天晚上，与一个公众号号主聊了很多，聊了自己当初是怎

么进到公众号这个圈子的，发现一路上自己还是很幸运的。

觉得在这个圈子有用的，就会投身去做，当发现别人推荐的

新的公众号时，我就会里面关注，并关注他发的内容是否适

合我，长此以往，我有了一定的关注圈，让我能更好的成

长。当初在培训时，也没有什么钱，但也还是愿意为知识付

费，写作不易需要鼓励。

现在关注的号多了，认知提升了以后，自己也想着输出一点 东西，但一直找不到方向。想着自己写文章是为了什么？本 身语文就差，还想进入这个圈子，分一杯羹么？一开始是希 望能够赚点外快，但人们凭什么对你的文字买单？觉得目的 不单纯的我迟迟下不去写作的手。

但机缘巧合，我工作的公司是以公众号原创为主的。每天耳 濡目染下，也明白了，公众号其实就是用文字去感染他人， 推送一些别人没有发现的点。（创作不易，没有鼓励，真的 很难做下去。我是真的希望能通过自己的努力，来获得他人 的认可。但什么样的主题，观众会买单呢？）

2.定下主题

突破第一层想写的难关后，我又遇到了问题...写什么样的内 容？

我发现大多数公号主都有自己主题，例如：提高他人认知、 娱乐热点、新闻等等。当然还有很多吸粉的玩法，现在的我 刚起步，也希望踏踏实实做好内容，而套路暂时不在考虑范 围内。与一个公号朋友聊了发现，我们都是在弥补人们的信 息差、认知差，希望大家看到我们的见解，能够有所启发。 我自己特别喜欢看一些认知性的文章，就希望在自己吸收 后，转换成我看待问题的方式来说给你们听。而且我会定一 个前提，对我自己有一定作用，当输出的东西对自己有用以 后，就一定会有那么一个相似的人，能有共鸣，所以我希望 那个与我相似的你，也能感受到一点慰藉，那我就心满意足 了。

这就是我的写作的初心，秉着这个源动力去写东西，我想一 定不会错。蔡康永老师说过自己写书的故事，他说：写的文 字大多数是写给自己的，用来在自己不想做或是不确定时， 说服自己的。是否是鸡汤他根本不在意，分享出来若被用 到，它就会被称为鸡汤，能够带给别人养分；若别人看了嗤 之以鼻，表示这东西对他没用，就会为他开心，因为他没有 这样的烦恼。

3.灵魂拷问 初心或者说源动力这个东西真的很难找，毕竟我找了这么久

也就只是找到在写文章希望能坚持下去的东西。其他事或者

说人生上的事，我也和大家一样，完全未知，甚至有点害

怕。

比如我常常会问自己，你真的适合编程么？你编程真的是为 了能写出好的程序来么？还是只是为了打工，赚一口饭吃？ 其实这一思考也试用很多方面。读书又是为了什么？为了更 多知识，还是为了逃避工作上的苦难。社交是为了什么？为 了交到更多朋友，还是为了躲避孤独。

这都是我们应该在脑海里转呀转的东西，逃避不了的。（虽 然开心时不会想这些，但安静下来就一定会想）。其实我们 想这些，不是让自己变得很焦虑、很浮躁！而是我们真的忙 的焦头烂额、不知所措的时候，可以安慰着自己说，我想要 过什么的生活，就必须战胜这些困境。脑海里有了这些想 法，才能更欣然的接受。

若当我们真正想明白了，也就不会太在意他人的评价了，或 是急着去攀比了。因为每个人本就不一样，又何来评价，何 来比较呢？我们只要过好自己当下，努力进步，就已经是最 好的状态了。

史蒂芬 · 斯皮尔伯格 27 岁因为《大白鲨》出名，成为当时炙 手可热的导演，当时《时代》杂志就用他做封面，做了一期 对他的称赞，当时制片在片场拿给他看，但斯皮尔伯格并没 有把杂志接过，而是丢在一边。说道：我现在如果相信他们 对我的称赞，我接下来就会相信他们对我的攻击。

我们每个人都需要训练自己不要轻易的相信他人对你的赞 美，你才有你能力不轻易的被别人攻击。

希望你也能找到那个初心，无论是哪方面的。可能是工作上 的、生活上的，这样就是完成了自己的一件大事，能让自己 开心好一阵子呢。

另外：今天是我 23 岁生日，能够想明白这些问题，我为自己 感到高兴的。即使自己在写这篇的时候，抓耳挠腮，难产到 了凌晨三点，才写下了初稿，还是有种莫名的喜悦，因为自 己从初中以后，就再也没有这种逼自己一把的心态了。

当然高手的策略是持续的、稳定的。也希望自己能持续、稳 定的写下每篇文章
