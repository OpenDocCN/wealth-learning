# 豆瓣评分 9.0 的电

温暖 : 豆瓣评分 9.0 的电影《怦然心动》：哪有什么一见钟 情，不过是两个原生家庭的歪打正着

这几年看烦了国产玛丽苏，越来越想念一些好影片，所以重 温了曾惊艳过自己的纯爱电影《怦然心动》。

这个故事发生在 20 世纪 60 年代的美国，女孩朱莉和男孩布莱 斯，两颗不同的灵魂骤然相逢。

初相遇。 布莱斯说：我这辈子只希望朱莉.贝克离我远点。

朱莉说：从我见到布莱斯.罗斯基第一眼，我就砰然心动了。

初初见你 人群中独自美丽 你仿佛有一种魔力 那一刻我竟然无法言语

——《我想我是真的爱你》

湖蓝色眼睛，栗色长发，古灵精怪的朱莉对男孩布莱斯一见 钟情，原因是他有一双让朱莉如痴如醉的双眸。可惜这份儿 爱情，起初只是她一个人的怦然心动。

“一副绝妙的画绝非拼凑而成，一头牛只是一头牛，草地也只 是长满了青草和花朵，阳光照进大树的枝丫，亦不过缕缕薄

光，但你若能将它们衔接拼凑，却能收获意想不到的成果。”

整体大于部分之和，这是朱莉父亲教给朱莉的道理，也是朱 莉反思自己对布莱斯感情的起点。

没有旧式琼瑶剧里天崩地裂、山河破碎只为一人痴醉的狗血 剧情，这部剧之所以能获得豆瓣 9.0 高分，可能是因为描摹出 了爱情的神韵：未必曲折离奇，只在身边细碎的光阴里氤 氲。

朱莉，用中国古语形容，当是钟灵毓秀，集天地灵气于一身 的女子。又与中国女性的温柔、婉约不同，她的形象是美式 的：自信、简单、对于内在美有着最本能的追寻。

国内知名心理学家武志红认为性格是“内在父母与内在小孩的 关系模式”。 “我曾谈到性格是‘内在父母与内在小孩的关系模式’，其实这 只是基本说法。完整的说法是，性格浓缩着我们童年的一切 人际关系。尽管主要是我们与父母的关系模式，但也有我们 与兄弟姐妹、我们与爷爷奶奶、我们与姥姥姥爷、我们与其 他亲人乃至我们与宠物的关系模式。自信是内在的小孩对于 获得内在的父母的爱充满信心；自卑是内在的小孩对获得内 在的父母的爱没有信心。”

朱莉之所以自信快乐地做大家眼中的“异类”，主要是因为她 来自一个温暖和谐的家庭。尽快家境拮据，穷到连吸尘器都 要每周去邻居家借，但是父母非常相爱，更重要的是他们也 深深地爱着她。剧中有一幕尤其动人：父母罕见因为被布莱 斯父亲嘲笑家里的草坪而发生了朱莉印象里第一次分歧。但 是吵完架后，他们分别去安慰朱莉，告诉她这不是你的错， 我们是如此爱你。显然，他们都没有把朱莉当成不懂事的小

女孩，而是把她当做一个拥有独立灵魂的人平等相待。

尤其是父亲，他不会因为女儿想捍卫梧桐树而觉得她不懂 事，而是尽可能保留女儿心中的纯真美好。

“不就是一棵树吗？” “不，那不仅仅是一棵树，希望你能将树梢的美好时光永存心 底。”

所以沐浴在爱中长大的朱莉，如同一棵健康的小树，自然接 受风雨彩虹，内心丰盈美好，天然去追寻生命中的美好，而 且勇于反思。

其实有些不明白，朱莉为什么会因为一双忽闪忽闪的眼睛就 爱上布莱斯。他是如此懦弱，因为不敢与朱莉说实话，扔掉 了朱莉送的鸡蛋；没有和朱莉一起捍卫梧桐树；甚至在好友 嘲笑朱莉和叔叔一样是弱智的时候，尽管内心不认同，仍然 去附和好友。他伤了朱莉的心，也让朱莉反思自己六年的爱 是否值得给这样一个人。朱莉慢慢发现布莱斯的内心是如此 空洞，甚至连自己真正的想法都不敢捍卫。

她决定不再爱他…… 2

“有的人浅薄，有的人金玉其外败絮其中，总有一天，你会遇 见一个彩虹般绚烂的人，遇见这个人之后，你会觉得其他人 只是浮云而已。”

最初布莱斯的眼里，朱莉像个没开化的“野人”，喜欢爬树， 喜欢养鸡，不像同龄的小女孩一样可爱柔顺。后来在外公切

特的帮助下，他才逐渐放下对朱莉的成见，去认识这个有着 彩虹般绚烂内心的女孩。

和朱莉不同，布莱斯家境殷实，家里的每个人都看似衣着光 鲜，但事实上这个家本质上“四分五裂”。父亲“外表干净整 洁，内心似乎装着什么腐烂掉的东西”，一直在挑剔朱莉家， 挑剔朱莉给的鸡蛋，挑剔朱莉的父亲不务正业，甚至恶意揣 测朱莉的哥哥们偷盗。母亲更多在扮演和事老的作用，但事 实上，两个人都对布莱斯的内心成长不够关注，否则为什么 第一个察觉到朱莉和布莱斯有关联的人是外公切特呢？

这样的家庭培养出的布莱斯尽管外表英俊，看似是个小绅 士，但内心不够自信，对事物的评价并不是依靠内在的评价 系统，而是习惯人云亦云。

他不够诚实，很多时候表现的像个懦夫。对于“彩虹女孩”朱 莉视而不见，甚至避之如蛇蝎，而对只会谈论头发分叉、野 营之类琐事的雪莉充满好感；欣赏不了梧桐树的美，不敢和 朱莉一起捍卫梧桐树；听从父亲的意见，不吃朱莉给的鸡 蛋，却没有勇气直接拒绝朱莉，而是扔掉那些鸡蛋；明明知 道好友加利特的话不对，内心明明想要反驳，反而附和，并 和好友一起讥讽朱莉同样弱智。

影片前三分之二部分的布莱斯是如此不招人喜欢，以至于忍 不住想给朱莉邮寄一瓶眼药水来提高她看男人的眼光。

心理学认为一个人外部的人际关系，是他的内在关系模式的 展现。按照这个道理，一个对自己太苛刻的人，很难做到宽 以待人。相反，对自己苛刻的人，更可能的选择，是挑剔别 人。所以布莱斯渐渐明白父亲与其是在挑剔朱莉一家，不如 说是挑剔那个放弃了梦想，成为了看《花花公子》的油腻中 年的自己。

他也渐渐了解到朱莉的可贵之处，她是如此真实而又独一无 二，毕竟有哪个上初中的女孩会谈论永动机呢？

外公切特在布莱斯的内心的成长中扮演了重要角色，他告诉 布莱斯“人的性格塑造于小的时候，我不希望你就这么迷失自 我。做人要诚实，一时的隐忍难挨，或许能让你免于一世的 痛苦煎熬。”让布莱斯一步步意识到好友加利特和父亲越界 了，也勇于战胜怯懦的自己，承认已经爱上朱莉，并选择主 动做出努力挽回朱莉的心。

“我不管他们怎么想的，我喜欢朱莉.贝克”。 3

爱情之所以玄妙，可能就在于它的天时地利。

于千万人之中遇见你所要遇见的人,于千万年之中,时间的无涯 的荒野里,没有早一步,也没有晚一步,刚巧赶上了。

如果，布莱斯再晚一些成长；如果朱莉彻底对布莱斯死心。 这个故事或许就是另一番结局。还好，没有早一步，也没有 晚一步，他们刚刚好同时真正走进彼此心房。

结局时。 布莱斯说：当我站在那儿，我意识到这些年我们从未开始交 流过，但那天我们可以开始交流了。 朱莉说：我意识到我们会说很久很久。

说句实话，这个故事看了好多遍，中间也隔了几年。起初， 看到的是青梅竹马，两小无猜的爱恋。但直到辜负过别人， 也被人辜负过，才渐渐品出一番别样滋味：心动是一刹那的

风起云涌，背后却是两颗灵魂所代表原生家庭的骤然相逢。

朱莉的父母非常相爱，且尊重关爱她的内心成长，所以朱莉 成为了一个内心充满阳光，又勇敢聪慧的女孩。布莱斯的父 亲自私挑剔，常常忽视儿女的真正感受，所以布莱斯成为了 一个不敢追寻真实自我的怯懦男孩。开始时，他们是如此不 相配，等两人真正成长了，才有了后来的灵魂契合。

我们每个人都是带着原生家庭的烙印，与另一半相逢相恋。 甚至恋爱、婚姻模式都是对原生家庭的复制。

“恋爱是亲子关系的复制，如果童年幸福，更有可能复制幸 福。如果童年痛苦，更有可能复制痛苦。”婚恋的幸福，和美 貌、才华、成就等等外在条件无关，却和原生家庭有着千丝 万缕的纠缠。否则，一代才女张爱玲为何遇见胡兰成这样的 渣男还低到尘埃里。“五十年才出一个的美人”林青霞，等秦 汉的婚姻承诺等到肝肠寸断，最后心灰意冷转而选择草草嫁 人。

羡慕那些拥有健康原生家庭的人，早早就拿到了幸福人生最 重要的筹码。如果不幸如你，如我，面对一个千疮百孔的原 生家庭（在中国，这是大概率事件），也没必要把婚恋中的 不幸福都一股脑儿甩锅给原生家庭，而后止步于自怨自艾。 毕竟，我们早已不是孩童，完全有逆风翻盘的能力。

“悲观也没用。谁都想生在好人家，可无法选择父母。发给你 什么样的牌，你就只能尽量打好它。——东野圭吾。”

愿天下人，有情或无情，皆能在尘世勇敢前行，直到心有所 依，爱有所归，无怨，无悔。
