# 《你以为你收拾的是

《你以为你收拾的是家，其实是收拾自己的心情》

晚上好[玫瑰]今天是橙橙加入星球的 254 天 与星友们分享第 140 篇原创！

正文：

1.  01

    杂乱无序的小家会让你心情很糟糕

    忙碌的工作让我们总是没有闲情和时间收拾自己的小窝，特 别是家里有小孩又上班的职场女性，顾得上事业顾不上家 务，顾着孩子就顾不上收拾。

    慢慢的家里越来越乱，很多东西越来越难找到，看着杂乱无 序的家，每天回到家的心情自然很糟糕。

2.  02

收拾家是一个自我审视的过程

为什么说收拾家其实是在收拾自己的心情，当你把家里的物 品都规整好，有秩序的摆放好平时要用的东西，你的家就能 腾出更多的空间给自己。

不要觉得收拾家务是在浪费时间，你在整理物品让其有固定 归属地的时候，是方便自己可以找到，所以整理物品也是在 整理自己的时间和精力，让自己更高效和有序。

规整的过程也并非是无聊无趣的，反而是审视自己的过程， 提醒别光顾着忙碌而忘记了生活，这也是一个让自己感动的 过程。

03

腾出空间给自己一个居心地

家的主角应该是人而非物，当把物品都归置好，人便有自己 的居心地。

居心地是个美妙的地方，当你想生气发火的时候；当你情绪 低落的时候；当你想要安静的时候，这个地方就可以收留 你，没有干扰；没有杂物；也没有需要你分心收拾的小角 落。

因为你在收拾家的时候就是在给自己腾出一块居心地，用于 收拾自己的心情。
