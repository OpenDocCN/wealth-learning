# [大家好，我是懂先

[大家好，我是懂先生。这是我的第七十八次分享，很开心遇 见这么优秀的你们。在星球的一年，我希望实现三个愿望， 如果有一天实现了，我会把我的故事说给你听。]

《几岁的孩子都知道慢慢来，你怎么就那么心急？》 坚持日更一段时间后，却越来越焦虑。主要是觉得自己非但

没有取得任何进步，甚至还在退步。写一篇分享的时间越来

越长，质量还不能让自己满意。

我甚至在怀疑自己的坚持是否有意义。我不知道我的努力换 来的到底是成长还是自我感动。因此最近特别丧，情绪特别 差，大概又到了每个月几天的心理周期。

晚饭后，绞尽脑汁还是没有关于今天分享的思路。于是决定 和朋友去体育馆打会羽毛球，缓解一下最近焦虑的情绪。但 由于好久都没有进行剧烈运动，一会的功夫我就累得满头大 汗，于是坐在场地边休息。

恰巧遇见几个小朋友在教练的陪同下进行发球练习。小朋友 对待每一个球都认认真真，不一会就打完了一大箱球，然后 自己把球捡回来再进行练习。

看着小朋友脸上认真的模样，真的十分敬佩。按理说这个年 纪应该是最淘气的，但他们却能耐下性子用心练好每一个 球，真值得称赞。

而且他们都是把羽毛球当成一种爱好发展，就要付出这么多 辛苦，那专业的运动员岂不是要付出比他们多无数倍努力。 最后我们看到的得到金牌的林丹谌龙，其实都是从几岁的时 候这样一个一个球练起的，没有任何捷径可走，想要走快反 而会摔倒。

而我的日更，可能都算不上写作，才写了几天就想要结果， 就要达到和写了很久的星友一样的高度，现实吗？如果真的 那么容易，那对坚持写了很久的人都不公平。

但我却太急了，急于得到认可、急于靠写字赚钱、急于取得 成功，却因此忽略了写作应该是静下心来一点一点积累的。 欲速则不达，越着急，越容易把自己束缚起来，越不能达到 自己想要的高度。

所以，我要及时从急躁的情绪中跳出，不能总想着通过短暂 的日更就要达到怎样的高度。做好自己该做的，结果该是什 么就是什么。

如果心态波动了，就想想那些一晚就耐心打了几框球的小伙 伴，他们都是我学习的榜样。慢慢来，才能做到每一步都坚 定踏实，步子大了，会到扯蛋的。
