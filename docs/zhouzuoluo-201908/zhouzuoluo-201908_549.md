# 《查理芒格的决策方

Branden : 《查理芒格的决策方法--附思维导图》

“只要得到严格的遵守，光是“避免犯傻的芒格体系”就能帮助 你超越许多在你之上的人，无论他们多么聪明。”——穷查理 宝典

众所周知，查理芒格具有别具一格的思维方式，他独特的决 策和思维方式一直备受推崇。下面是摘录的《穷查理宝典》 这本书中关于查理芒格的决策方法中的“投资原则检查清 单”，文末附带整理的思维导图，和大家一起分享。

---投资原则检查清单---

■风险——所有投资评估应该从测量风险（尤其是信用风险） 开始

●测算合适的安全边际

●避免和道德品质有问题的人交易

●坚持为预定的风险要求合适的补偿

●永远记住通货膨胀和利率的风险

●避免不犯大错

■独立——唯有在童话中，皇帝才会被告知自己没穿衣服

●客观和理性的态度需要独立思考

●记住，你是对是错，并不取决于别人同意你还是反对你——

唯一重要的是你的分析和

判断是都正确

●随大流智慧让你往平均值靠近（只能获得中等的业绩）

■准备——唯一获胜的方法是工作、工作、工作，并希望拥有 一点洞察力

●通过广泛阅读把自己培养成一个终生自学者；培养好奇心， 每天努力使自己再聪明一点点

●比求胜的意愿更重要的是做好准备的意愿（愿力）

●熟练掌握各大学科的思维模型

如果你想变得聪明，你必须不停追问自己的问题是：为什

么、为什么、为什么

■谦虚——承认自己的无知是智慧的开端

●只在自己明确的能力圈内做事

●辨认和核查否定性的证据

●抵制追求虚假的精确和错误的确定性的欲望

●最重要的是，别愚弄你自己，而且要记住，你是最容易愚弄

你自己的人

*理解复利的魔力和获得明它的困难是理解许多事情的核心和 灵魂*

■严格分析——使用科学方法和有效的检查清单能够最大限度 地减少错误和疏忽

●区分价值和价格，过程和行动，财富和规模

●记住浅显的好过掌握深奥的

●成为商业分析家，而不是市场、宏观经济或者证券分析家

●考虑总体的风险和效益，永远关注潜在的二阶效应和更高层

次从影响

●要朝前想、往后想——反过来想，总是反过来想

■配置——正确地配置资本四投资者最重要的工作

●记住，最好的用途总是由第二好的用途衡量出来的（机会成 本）

●好主意特别少——当时机有利于你时，狠狠下赌注吧（配置 资本）——不断积蓄势能，随时准备出击

●别“爱上”投资项目，要依情况而定，照机会而行

■耐心——克制人类爱动的偏好

●复利是世界上第八大奇迹（爱因斯坦），不到必要的时候，

别去打断它

●避免多余的交易税和摩擦成本，永远别为了行动而行动

●幸运来临时要保持清醒头脑

●享受结果，也享受过程，因为你活在过程里

■决心——当合适的时机出现时，要坚决地采取行动

●当别人贪婪时，要害怕；当别人害怕时，要贪婪

●机会来临的次数不多，所以它来临时，要抓住它

●机会只眷顾有准备的人

■改变——在生活中要学会改变和接收无法消除的复杂性

●认识和适应你身边世界的真实本质，别指望它来适应你

●不断挑战和更新你“最爱的观念”（察觉自己的念头）

●正视现实，尤其是你不喜欢它的时候——尤其是你不喜欢它

的时候

■专注——别把事情搞复杂，记住你原来要做的事情

●记住，声誉和正直是你拥有最有价值的东西，而且可能瞬间

化为乌有

●避免妄自尊大和厌倦无聊的情绪

●别因为过于关注细节而忽略了显而易见的东西

●千万排除不需要的信息：千里之堤毁于蚁穴

●直面你的大问题，别把它们藏起来

共勉~
