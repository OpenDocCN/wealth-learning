# 《这是我学过激励别

《这是我学过激励别人最有效的方法》

下午好 与星友们分享丹儿的第 51 篇原创 6.14

———————————————————————

别人一件事完成的特别好时，赞扬对方，你还在说“你真聪明 吗？”无论是亲子关系，还是亲密关系，或是与朋友之间的相 处中，都会遇到需要赞扬对方的时候。

很多人会说“你真聪明，这件事办的真好”， 而少数人会说“你通过什么样的方法做到了这件事，你的思考

能力值得我学习，真棒”

大家看出这两种表达方式上有什么不同了吗？ 第一种赞扬的方式：表扬了对方，没有说为什么表扬他，只

是肯定了结果。

这样赞扬别人，对方不知道你赞扬他的原因，只是觉得，你 夸他聪明，他以为做好这件事是智商的原因。也不会去思 考，他为什么能做好这件事。

其实并不是因为他聪明，才能把这件事做好。每个人的智商 相差无几，不存在谁比谁聪明多少。

第二种赞扬的方式：表扬了对方，并说明原因，让对方知 道，他是因为去思考了，并积极的去做了，这件事才办好 了。

我们肯定的是事情的过程，而不是结果。这样表达明确的让 对方知道他这件事做好的原因。以后遇到困难也能运用到之 前的方法。

这种表达方式，既夸奖了对方，还起到了激励的作用。让她 在面对其他困难时，也能多思考解决的方法。

举个自身的例子

①早上朋友让我帮忙转发店里的招聘，但是内容一大篇，发 朋友圈内容就被隐藏了。只有第一行，转发就是帮助他招 人，所以我在评论区总结了一下主要内容，帮他发出去了。

我朋友让很多人帮他转发，我们是一个朋友圈，有很多共同 好友，大多数都只是转发，有个朋友看到我是这样发的，就 给我评论，说“我同桌真聪明”

这种赞扬的方式，就属于第一种，赞扬时只肯定了结果，没 有肯定过程。如果是以前，看到评论，我也很开心，觉得我

是挺聪明的，就要开始飘了

没有人不喜欢被人夸聪明的，我也经常被别人这样夸奖，但 是当我遇到一些我觉得很困难的事情时，一做不好，我就会 放弃，因为我觉得事情做不好就证明我不是一个聪明的人 了，为了聪明这个虚名，我不能允许我做自己不擅长的事。

长期下去就会，停滞不前啦～

②第二个案例是有关于一位特别积极，爱思考的星友听听 的，我有一篇文章，关于心理学的。听听很积极的参与讨 论，在评论下方还提到一个词“信息熵”

“信息熵”我不太了解，所以就去百度搜索了一下，了解了意 思后给听听的评论做出了回复。

听听的评论是这样的“丹对“信息熵”认真的了解和思考，这样 的学习态度真的很赞，很值得我学习。丹对“信息熵”理解很 赞，也给了我更清晰的思路，谢谢丹。”

听听的回答就是标准的，第二种赞扬方式，赞扬你并且说了 原因。听到这样的赞赏就很好的激励了我。

让我了解到这件事我做的好，是因为我的思考，和查阅资 料。以后遇到困难，我会更加愿意去解决它。

＊＊＊＊＊＊＊ end ＊＊＊＊＊＊＊

▼最好的激励方式是，表扬他，并且告诉原因，肯定对方解 决问题的过程，而非结果。

第一个案例中，我这样转发也不是因为我聪明，而是我比别 人更细心了一些，因为转发的目的就是招人内容都被隐藏了 就没有转发的必要了。所以我又把内容总结了发到评论里。

细心是学习《扫除道》以后悟到的，扫除道的精神就是『凡 事彻底』，而每件小事都能做到细心，把它做好，就能做到 凡事彻底。大家想了解关于细心更全面的内容吗？感觉还能 写一大篇 ，想了解的留言哦。
