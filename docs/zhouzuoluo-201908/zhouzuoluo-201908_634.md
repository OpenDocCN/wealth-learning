# 《高手眼里的“人脉

老王 : 《高手眼里的“人脉”，竟是这样的》

“人脉”是我这段时间，一直想写的一个话题。前几天动手试 一试，但在写了 1 千多字后，感觉实在是没啥可写的，索性也 就放弃写长文。

现在呢！我把我想说的话，以最精炼的文字表达出来，文章 短没关系，对读者有价值就好。

(1)、那些能帮到你的人，不是你的人脉，只有那些你能帮到 的人，才是你的人脉。

(2)、不同的人生阶段，会认识不同的人；或者说，当你具备 不同的价值时，你也会认识不同级别的人。

(3)、人脉的本质是价值交换，在你为别人提供不了相对应的 价值时，你和他们之间是没有连接的。

(4)、如果想要积累人脉，你能做的就是不断积累自己的价 值，并不断输出自己的价值，当你能够帮到，越来越多厉害 的人，你的人脉就会越来越广，质量也会越来越高。

(5)、任何天赋都可以产生收入，关键是要有价值，找到那个 能把自己天赋发挥价值的那个位置。

(6)、人脉，就是我们和这个世界的连接方式 (7)、每个人，都会忽视人脉中的“弱联系”，而恰恰是这些不

经常联系的圈外人，才会知道我们所不知道的东西；而我们

经常联系的熟人，他们知道的，我们也知道，他们做的，我 们也在做，互相之间没有信息差。
