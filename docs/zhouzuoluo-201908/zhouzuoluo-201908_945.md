# 《时间管理的问题和

《时间管理的问题和解决方法——下》

今天继续和大家来聊一下时间管理的问题，还有下面的 4 个方 法。

1、养成某种工作习惯，让你的身体适应起来 我们都想养成写作的习惯，那么最好让这个习惯在固定的时

间做固定的事情。比如，我下班到家以后 8-9 点是我最适合写 作的时间，那么我每天到这个时间点的时候，我就只干写作

这件事情，别的事情我都不去做。我们早上去上班，都是有 一个固定的时间对吧，到了这个时间点，我们就要投入到我 们的工作当中去了。写作也是一样，到了晚上 8-9 点，我就要 写作了，这样做的话，会更加高效，更加投入，还让我的身 体适应了我的工作，能做到生活和工作两不误。

2、学会说不 作为成年人，最重要的是效率，而不是面子。有的时候，朋

友让我们帮忙做一件特别没有意义的事情，又会浪费我们很

多时间，这时候我们要敢于说不，而不是在意面子，感觉不

太好意思，拒绝了朋友。我们要把时间花在值得事情上，而

不是做那些对自己来说没有任何意义的事情。

3、善于借用别人的力量来做事情 如果事情很复杂，需要借助别人的力量，可以花钱请别人来

帮助你。有的时候，我们遇到不会做得事情，我们可以咨询 这方面的专家，买这方面的课程，还可以买这方面的书来帮 助自己解决问题，而不是傻傻的干着急。虽然你钱少了，但 是节省了你大量的时间去做更多的事情，从而效率也提升了 好几倍。

4、拖延症和懒癌很严重怎么办？ 第一点：从小事情做起

逐步体会到提前完成事情带给你的成就感。比如，你下班回

家马上去整理东西，洗头洗澡，这样你就会有一整块的事情

可以来写作、看书，来做你想要做得任何事情。而不是躺在

沙发上玩抖音、聊天，到了 9.00 以后，我才想起来晚饭没吃， 去吃点东西，到了 12.00 以后洗头洗澡也还没洗，想一想就算 了吧，还有读书、写作都还没完成，满满的愧疚感。所以， 拖延症的第一步，从小事情做起。

第二点：专注 不要同一时间段做很多事情，专注的完成一件事情比多任务 处理效果会更好。毕竟我们的精力是有限的，不可能同一时 间段完成很多事情，所以还是需要一件一件的去完成。

第三点：把大事分解 把大事情分解成一件一件的小事情，逐步完成，用小事情带 动自己运转起来，这样才能把大事情做完。

懒的问题是没办法的，想要取得好的成绩就一定要勤奋，不 然只能平凡的做个普通人。

懒其实是自己潜意识当中的不想改变现状的一个问题，如果 我们想要改变想要过上更好的生活，你肯定会想办法克服 的。

以上就是关于时间管理的问题和解决方法，希望对大家有帮 助，从而去行动，遇见更好的自己。
