# 《人生不是只有一条

《人生不是只有一条路，走不下去的时候就换一条》

好多高校的考研成绩相继公布，几家欢喜几家愁。大学同学 阿辉又落榜了，这已经是他第五次考研失败了。

阿辉在我们这种二本大学是大神一般的存在，各种奖学金拿 到手软。早早他就定下了考研的目标，并为之努力。但不知 是目标定的太高、还是运气太差、亦或是努力方向不对，总 之每次都差了一口气，当年读研的同学都毕业了，他却还没 收到录取通知书。

当然，我绝没有嘲笑阿辉，每一个梦想都值得尊重。 但我总觉得他现在已经有点偏执，这五年，为了考研，他没

有任何收入，全靠父母养着。而他的父母都是农民，本以为

供到大学毕业阿辉就可以赚钱了。可没想到阿辉毕业了还要

靠他们养着，他们还不忍心说，只能一把年纪还要打工养儿

子。

其实考研是好事，想通过自己的努力得到更高的学历，改变 自己的命运，非常值得鼓励。但人生并不是只有考研，不能 一味死磕。如果发现这条路走不过去，可以转个弯换条路试 试，也许换条路也能看到不错的风景。

坚持是优秀的品质，但也要学会变通，如果方向就是错的， 那越坚持则错的越远。

黄渤在成为影帝之前，唱过歌跳过舞、经过商下过海，但当 他发现一个职业不适合自己时，马上换下一个，最后终于找 到了适合自己的事业--演员。如果他现在还在唱歌，我估计已 经快要饿死了。

我们的人生有无限种可能，趁着年轻要勇敢去尝试去试错， 遇到一条走不过去的路时，就换一条。一直在一条过不去的 路上纠结，即浪费了时间又消磨了意志。没必要在一条路上 死磕，换一条路，同样可以看到更好的风景。

[我是懂先生，这是我的第一百零五次分享，很开心遇见这么 优秀的你们。在星球的一年，我希望实现三个愿望，如果有 一天实现了，我会把我的故事说给你听。]
