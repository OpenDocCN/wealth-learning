# 【职场上混得不好的

【职场上混得不好的人，都有这 3 种弱者思维 - 今日头条】

作者 | 孙瑞希 前几天，朋友带着儿子沐晨（化名）来工作室拜访我。小伙子参加工作三年多，几年不见，变化很

大。

刚进门时他特别沮丧，低着脑袋嗫嚅地说：“老师，我都工作三年了，感觉自己的人生特别失败。” 我说：“工作三年，职业生涯才刚刚开始，何谈失败呢？” 但是在接下来的对话中，我发现如果跳出咨询师的视角以亲人朋友的视角去看他的话，他的确过得

不怎么成功，准确地说还有那么一点点失败。

考高中时，沐晨分数只够自费线，他妈妈费了九牛二虎之力，四处托关系，终于把他弄到一所市属 重点中学的实验班，希望他能在这个环境里取得好成绩。但他却在第一学期考试就被淘汰出局，进 入了普通班。

在这所市属重点中学的普通班，只要成绩能保持在中游，考一个二本院校问题也不大。但沐晨成绩 居于末等，高考那年，他正常发挥，去了一所大专院校。

由于学校一般，专业一般，他在大学时又混了三年，所以毕业后就只能对付着找一份工作。 低薪的重复性工作，让沐晨感到特别乏味。毕业三年，还在父母那里蹭吃蹭喝蹭零花钱。他说：“看

着工资卡那么丁点儿钱，每个月就有 30 来天不想上班。“他接着抱怨道：我家小区快递小哥都比我赚

得多。”

我说：“那你也可以去做啊？” 他说：“那不行，风里来雨里去，我吃不了那个苦。” 我问他：“你还可以换个工资高点的工作呀？” 他说：“工资高的条件也高，我什么也不会，做不了那个。” 我说：“不会可以去学呀？”

他说：“一想到学习就头疼。”

……

做生涯咨询的这些年，我接触了不少人，我发现那些在职场上混得好的和混得差的人，最大的差距 不是学历上的，不是家庭上的，而是思维方式上的。

有人在职业生涯早期，认真地对待单调重复的工作，慢慢打磨出了一技之长，以后越跳越高，越走 越好。而有些人，却在单调重复中消磨了自己的意志。

所以经过若干年的叠加，我们看到强者越来越强，弱者越来越弱。而弱者之所以弱，究其根本是由 于掉入了“弱者思维”的陷阱。从我经手的生涯咨询个案来看，职场人经常会有的弱者思维有以下 3 种。

01

错误的归因方式

我有个老乡南静，去年公司注册了微信公众号，将小编的工作划给她来做，她做得一塌糊涂，没少 挨老板数落。有次同乡聚会，她说自己修炼了一年，还是个小白，问我怎样才能快速提升写作和运 营技能。

我给她推荐了几个做得不错的公众号，还有一些新媒体类课程。其中有一家做得不错的新媒体，创 始人是我们的同乡，我着重介绍了一下。我说他现在在北京，团队规模也有几十号人了，他创业第 三年的时候，就已经年入千万了，做得特别牛。

其他几个同乡得知这个新媒体大咖年龄比自己还小好几岁时，都纷纷感叹：长江后浪推前浪，年轻 人是我们的好榜样。

南静一边划拉着手机一边问我：“他是哪个大学毕业的？” 我说“XX 大学。”（我们省的 985 大学） 南静说：“怪不得吗，学校这么好，混得当然要好一点了。” 接下来她又问：“他爸妈是干啥的？”

我说：“农民。” 南静瞪大眼睛：“那他怎么可能有资本创业？他创业之前是做什么的？” 我说：“在政府机关工作过，在央企工作过，也在 X 和岛工作过。”

南静终于松了一口气说：“我就知道，这要是没点资本，怎么可能在北京创业？你看看他的工作履 历，多光鲜那？哎，谁能有他那么命好哇！我要是有这样的工作经历，我也出来创业！”

这话比较噎人，我就不再接话了。 有些人总有这样固有的的思维模式：

他能取得成功？因为他名校毕业啊；他能去创业？因为他原生家庭好啊；他创业走得很顺？因为他 有贵人扶持啊。我没考上名校？因为我高考那年题太难啊；我没去创业？因为我爹妈没有钱啊；我 创业失败？因为我遇到一群猪队友啊。

你看，弱者总是习惯找外因。别人的成功，永远是因为有个好爹好妈、好领导、好学校、好机缘， 却忽视了那些比你厉害的人，总是在你看不见的地方努力奔跑，与自己死磕。自己的失败，永远是 出身不好，没钱、没资源、没人脉、没机会。

当我们守着这种弱者思维不放，就永远会为自己的不成功、不努力找借口，日子久了，也就失去了 努力的动力。

02

牢骚上的巨人，行动上的矮子

前面提到的小伙子沐晨，如果他能够踏踏实实地工作几年，慢慢打磨一门手艺，也会有不错的前 途。但年纪轻轻的他，除了满腹牢骚和啃老之外，没有采取任何实际行动。

家里家外，他总是在痛骂小公司没前途，领导蠢，队友猪。要是有人劝他：“实在不顺心，就换个单 位吧！”

他总有各种理由驳斥：“哪儿那么容易换那？换个新环境，不得学习新东西吗？别和我提学习，一提 学习我就脑子疼。”

他妈妈劝他，要是不想换单位，那就利用业余时间搞点副业也行。他又反驳道：“我哪有那个时间 呀？每天上班累得要死，回家什么都不想干。”

其实他的工作并没有饱和到那个程度。下班回家吃完晚饭就坐在沙发上玩儿手机，经常玩到后半 夜。

一晃毕业三年，经过这三年的社会历练，一些同学已经开始崭露头角了，有的是业务骨干，有的做 到了主管。随着时间的推移，同学们之间的差距会更大。

对于同学们身上的积极变化，他不以为然：“我就是不爱学习，我要是能像他们似的多看几页书，我 也能行。”

纸上谈兵终觉浅，绝知此事要躬行。现实生活中，总有那么一群弱者，他们认为别人做的事都是简 单的，自己也能做，只是没去做罢了。一味地发牢骚而没有采取行动，最终的结局也不过是牢骚满 腹的愤青而已。

03

急于兑现回报

前段时间，我回北京，有个学友约我见面。她说她也是做人力资源出身的，现在面临职业转型。她 问我：“国内做生涯培训的有哪些权威机构？如果我去参加培训了，也顺利地结业了，能不能去做生 涯咨询这份工作，实现商业变现？”

我对她说：“职业生涯这个事儿，快速变现不大现实。无论你在多么权威的机构接受的培训，你学到 的都仅仅是生涯领域的知识，把知识转化为技能，这需要你在生涯领域做大量公益咨询积累经验， 然后才能到付费咨询阶段。”

她又问我：“那你是如何实现变现的？” 我说：“我从 2005 年考注册人力资源管理师起就学习了职业生涯规划的知识，2009 年和北大纵横管理

咨询集团合作的人力资源咨询项目中，也有职业生涯的模块，那时候我就已经开始在实际工作中运

用生涯规划的知识了。之后又进行了系统的学习、认证以及辅修心理学知识，一路走下来 14 年。所 以，我在这个领域的变现不是快速的，而是顺其自然、水到渠成的。”

她听了比较吃惊，当她得知生涯咨询师的认证费用不菲，又需要做这么多铺垫才能有所斩获时，她 犹豫了。后来她又说，实在不行我就开个管理咨询公司吧。我没再接话，只是希望能够快速和她结 束这段对话。

春种、秋收、夏长、冬藏，这是大自然的普遍规律，刚播下种子，就想要看到果实，这样的人得有 多么肤浅。只看眼前，不看长远，这是最大的短视。急于兑现回报的人哪有美好的未来？

实际上，生活中的弱者思维还有很多，今天说的是我在生涯咨询个案中总结的职场人经常遇到的 3 种。弱者思维最可怕的是，它模糊了我们的视线，使我们看不清真实的世界。

一个人的思维方式，决定了他的格局，格局决定了视野，视野决定了见识。你在山脚，永远不知道 山顶的人能看到什么样的风景。

剔除弱者思维，做生活的强者！共勉！
