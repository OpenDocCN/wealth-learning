# 嗨，大家下午好，我

短尾猫 : 嗨，大家下午好，我是短尾猫，这是第 44 天第 42 次分 享，感谢和你们在一起的每一天 。[玫瑰][玫瑰][玫瑰]

《坚持不易，且行且珍惜》 为什么坚持如此不易，而放弃却如此容易？ 我们曾费劲心思的去坚持某一件事情，只为了达到自己所

想，但最终却都前功尽弃，无果而终。

我们也曾找过朋友互相激励，找各种办法想让自己坚持下 去，可是每每都会在不经意间，因为一点小事放弃，然后不 了了之。

01

坚持真的是一件很困难的事情，而且一个人坚持更是难上加 难。比如跑步，比如写作，比如健身，而且很多时候，信誓 旦旦的说自己要怎样怎样，但真的到了瓶颈期的时候，你会 发现，自己用意志坚持真的很难。

最近因为感冒 ，自己开始变得懒散，不再每天坚持自己， 找各种的理由来自我安慰，搪塞自己。

《奇葩大会》导师高晓松有一段很精彩的演讲，大意是说， 我们早晚都会被生活打败，当你坚持到下半场的某个时刻， 换人名额用完，体力耗光，会显得很绝望。所以还有力气的 时候，一定要狠踹生活，因为生活绝不会考虑你胆小懦弱， 什么都没干而饶了你。

其实很多的时候，人生就像是在汪洋大海里面游泳，看不到 彼岸，便生出慌乱和绝望，但你一旦选择放弃不游了，可能 就会永远沉下去。

有时候也像打麻将，哪怕你抓到一手烂牌，也要好好谋划怎 么打，上家做清一色，下家吊小七对，你能放给对家一个点 炮，其实也是赚了。

也许挺过来了，人生就会豁然开朗，而挺不过来，时间也会 教会你怎么与它们握手言和。

02

有人说坚持就像是一种修行，一个人在漫漫的人生旅途上， 默默忍受的不仅仅是孤独和寂寞，还有嘲讽和讥笑……

也有人说坚持好像一种执着，一只展翅高飞的雄鹰迎风展开 的不仅仅是强健的双翅，还有梦想和信念……

坚持的路上，同行的人会越来越少，或许他们会在下一个路 口与你告别，或许他们会在下一个站台离你而去，也或许他 们会在起点就开始止步了。

始终还记得，在《阿甘正传》里的一段对话：“你以后想成为 什么样的人？什么意思？难道我以后就不能成为我自己了 吗？”

一个智商只有 75 的低能儿，通过自身的不懈努力，从要靠金 属支架走路到飞奔如风成为大学橄榄球明星，从籍籍无名成 为街知巷闻的越战英雄、乒乓球外交大使，甚至是拥有十几 条渔船的公司股东。

大概只有从没有放弃过的阿甘有资格说出这样的话了吧。

03

坚持下去的意义也许是让曾经的你善待以后的你，也许是是 过去的你保护了未来的你。

它不只是为了一个结果，它是想让你看到你比你想象中更有 力量。

当然你也可以选择放弃，选择在平庸的人生里安稳度日碌碌 无为，可有一天当黯然的你被别人光彩夺目的人生照亮，刺 伤双眼，无处可逃时，你也许会后悔曾经的自己没有抓住那 个耀眼的机会。

其实，努力并不是没有用，努力并不是没让你成长，努力只 是在让你自己扎根。即便再小的事情，只要可以坚持做下 去，那就注定成为英雄。

有时候你会有时光里旷日持久的思考，有时候你会在日常的 生活里去较真或者敷衍，这也都不过是为了对付今后更难走 的路。

即便陷入困境之中，也并非总是孤立无援。有的人是靠倔 强，有的人是靠信仰，还有一些人不纠缠于是否生死有命， 在劫难逃，只凭一分一秒地熬，一步一步地走，就生生把那 些黑暗的日子慢慢熬成了光明。

伤心有时，沮丧有时，贫贱有时，失意也有时。当你回头看 自己，观望每一个在艰难时刻笨拙地向上爬着，始终不肯放 弃的自己，你会心存感激。

别轻易放弃，因为你的坚持不易，且行且珍惜。
