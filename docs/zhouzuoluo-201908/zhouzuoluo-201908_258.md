# 《耐下性子来 一定

《耐下性子来 一定有办法》

老哥一向沉稳，遇事不慌不忙。有时候我看到一些书，遇见 一些事请教他时他总会给我有用的解决方法。不过今天他遇 见了一件小事，终于有一次是我，不急不躁帮他搞定。

其实事情不大，但是我在帮他之前就坚定：一定会有办法 的，只要耐心看一下了解清楚。

事情是这样的。他在某宝买了一双几百块钱的鞋子，到货之 后试穿觉得太小所以选择换货，在订单后方选择了“换货”。

后台有快递上门收件的选项，他勾选了这项服务然后寄出鞋 子。在等待鞋子到达他们仓库过程中，店里客服跟他说没有 他要的尺码了，所以只能退货退款。

老哥二话不说把“换货”改成“退货”，与此同时令他想不到的 是：之前换货的快递单号消失不见。

如今，订单显示“待发货”状态。老哥之前没留单号，导致没 有任何可追寻的信息。他致电韵达快递，告知自己在几月几 号哪个快递点寄出的快递，是什么物品等等，人家一句“没有 快递单号”很难查到等，之后几经转电话，未果。

今天刚好去家庭日聚餐，他打完几个电话后跟我们发牢骚。 做一个最坏的打算，就是这几百块钱就这么打水漂了。

我跟他说，你在把“换货”改为“退货”的时候，应该得先想下会

有什么不同结果。我之前也没这么试过，不过应该会有办法 解决的。

他继续看着手机，说之前没有备注下单号，可能真麻烦了， 致电快递客服回复他拿身份证去当时寄件点找找看；联系某 宝客服，却都是智能客服回答设定的问题根本无法沟通。他 怎么可能有时间去弄这个事，几百块钱就这样不明不白没 了。关健还不是自己出了什么问题，看得出不太舒心。

我一边在百度搜查了一下类似问题，有类似找不到订单的情 况，但是没有同样的问题出现。我让他手机打开某宝给我看 下，找到订单搜寻不到一分钟，帮他找到他的订单号。

他二话不说给我发了一百块钱红包，说看来关键时刻还是家 里人靠谱。

其实很简单，在订单周围找到一些按钮，搜寻几处以后进 到“所有订单信息”页面，找到他前几天发货的记录，包括寄 件时间、送达时间和单号。

虽是一个小事，也给我带来一些想法。 一是老哥吐槽快递客服不负责任，转了好几个人也是不了了

之。我跟他说其实不怪他们，你想一下，他们每天要接到多

少客户的各种咨询电话，所以他们必定有一个“流程模式”，

第一件事当然是要你的快递单号，你没有单号的情况下，他

们第一步都开展不了。

你手上拥有产品物流信息，事情来龙去脉只有自己清楚，对 于他们来说却一概不知。没有单号查询不了，也就是你连让 他们了解你房间情况的钥匙都没有，他们就需要你自己想办 法去获得钥匙给他们。这是我作为服务行业的一员看到的；

而你站在自己角度，只认为他们推脱责任，不给力不帮忙。

再者，他们只是打一份工，除非特别的使命或者知道有特别 的结果奖励，不然凭什么要求他们“额外”增加工作量，只因 为一个咨询电话呢？

最后，回归到这件我能协助上的事来。在让他拿给我他手机 之前我已经有很强的预感能够解决，先自己找找百度再结合 一些某宝的智能客服回答，果然轻易搞定。因为我的心态很 平和，耐下性子仔细寻找一下相关联的页面，就找到了。

越对未来有信心，越对当下有耐心。
