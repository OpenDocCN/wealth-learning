# 《利他，才能合作共

《利他，才能合作共赢》

从昨晚开始，粥老师的写作课推出了一个活动，分销自己的 海报给朋友，可以活动 50%的佣金，很多 kol（关键意见领 袖）参与了进来，转发海报到朋友圈+走心文案。

我之前已经分享过几次课程了，几乎就没有购买了，忽然看 到星友王威写到一条“如果扫描他的二维码购买课程，他就发 66 元红包，祝写作一路顺利”。我就复制过来放在我的朋友圈 文案下边，结果又有几位朋友购买。

然后我推荐这个方法给其他人，几乎都卖出了课程，这事带 给我一些思考。

1、互利共赢 同样是在朋友圈推荐课程，凭什么要买你推荐的？凭什么要

马上买。

第一、你推荐的课程质量足够好（这点大家都一样） 第二、你足够让人信任，相信你的推荐就是最好的（拉开一 点差距） 第三、跟你买得到更多的实惠（在课程质量一样的情况下， 懂得让利，就是提供更多的实惠

2、同样一个思维，不同的人不同的反应

当我把这个方法分享给不同的人，得到的是不同的反映。

@匿名 他说如果我发红包给别人，那我赚的钱就少了，我不 愿意。

我只能答他，如果你不愿意把佣金分一部分给你的朋友，朋 友可能跟其他人买，你就很可能一点都赚不到。

这是一种自想自己的思维，他不能明白“舍得舍得，有舍才有 得”，在中国的古话里，早就有了利他思维的智慧。

@星里浣 这家伙是对自己最恨的，把红包数额扩大了，自己几乎没有 收益。在这次课程转化效果当然非常不错，但是如果在商业 合作中，其实这种做法并不可取。

合作讲究的是互利共赢，自己付出了时间成本，如果完全没 有利益，从成本的角度来说，其实是亏损的。

@一个悦己 她让我看到了一个词“优秀”。

首先，她把我的文案根据自己的情况进行了优化（很多人只 会完全复制） 其次，她接受了奖励政策，并再次优化升级，做了梯度营 销。比如说，今天购买我发 66 红包，明天发 55，后天 44，以 此类推…..这样一来刺激了消费的紧迫感，迫使观望的人马上 下单，有一种早买早便宜的感觉。

我更换了新的文案之后，果然又有人找我购买课程。

3、凡事有方法，永远值得去学习

同样的一份课程，不同的人，不同的方法，销量是不同的。 人品靠平时的累积，方法就是靠学习和迭代。

有的人只会抱怨，把问题归咎到别处，比如说课程贵啊、 转化难啊、朋友不感兴趣啊。事实上这对解决问题毫无用 处。而聪明的人，懂得学习别人好的方法，不仅仅搬过来使 用，还想办法升级迭代。

这一次简单的活动，@一个悦己让我看到了她的优秀和能 力，即使你觉得自己已经尽力了，也还是值得再去学习，再 去迭代优化，不断地进步。
