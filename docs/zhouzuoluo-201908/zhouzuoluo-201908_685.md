# #自我成长 # 静水 : #自我成长 #认知新得 到底什么才是成功？

昨天大越越打完狂犬疫苗，晚上开始发烧，张先生第一直觉 就是，她不听妈妈的话，穿的太少感冒了，大吼她一顿。

我提醒他，好好说话控制情绪，你等我问下防疫站的朋友看 看咋回事。

果不其然，属于正常的反应，张先生也不好意思了。 哪知，孩子痛哭流涕，泣不成声，她问我，为什么妹妹无论

犯什么错，一家人都包容她，哄着她，我哪里错了？

我意识到青春期的孩子，有异常敏感的一面。我把这件事的 利弊从头到尾给她分析一遍。

最后，我决定在她的下铺陪她，防止半夜再起烧，她表面上 说，不必了，其实她在赌气，一个孩子特别渴望父母的关 爱。

我说妈妈妹妹都想你了呢，于是厚着脸皮在她的下铺睡下 了。

哪知，她的话题打开了，她说她有很多困惑，她问我什么叫 成功？我心中的成功是什么标准？

我告诉她，成功从无定论，也没有绝对的标准，就比如我， 这个阶段的成功，就是把你们姐妹俩照顾好，身体健康，学 业有成，有足够的能力把控自己的人生。

我自己呢，提前给自己打好基础，经济上自给自足，身体健 康，尽量不给孩子添麻烦。

还有，保持与时俱进的思想，因为成长型思维会让一个人变 得越来越优秀。

靠近内心有光的人，和有品质的人做朋友，在自己的精力允 许范围内，多去一些地方走走。

这就是成功吧。

我们宿舍当年共六个女生，目前职场上混的最好的，已经是 银行地市分行行长，最次的是中学老师，不，也许在世俗眼 里，最次的是我，一个全职妈妈，一个码字工人，但我觉得 这是我最喜欢的状态，并且也没有停止创造价值，我觉得这 就是成功。

虽和赚大钱开豪车住洋房的硬件成功标准，相差甚远，但我 很开心，很充实，感觉生命很饱满。

大越越似乎有所悟。母女俩好久没这么畅谈过了。 连着写了两天和教育女儿有关的真实故事，我感觉教育真的

不是一朝一夕的事情，需要时时用心。

真诚静水期待与各位姐妹一起精进成长，照顾好我生的人和 生我的人，最终的还要好好爱自己。

此时，我坐在医院病房內的椅子上，陪着年近古稀的父亲打 吊瓶。

没有健康，幸福就会打折，当然不幸福就很难谈什么成功。 静水觉得，每个人在自己对应的年龄段，好好爱自己，和谐

与这个世界相处，就是成功。
