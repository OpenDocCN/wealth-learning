# Day 1《不要怂

《不要怂，就是干！》

今天是我在粥老师星球里分享的第一篇。 有一些个人感悟，想要跟星友们分享。

1.为什么下定决心开始写了？ 一个人认识到一件事情重要性，需要时间的累积，经历的推

动和某一个契机的顿悟。

我很早之前尝试过写作，又很快放弃。2016 年的时候，我兴 冲冲开了一个个人公众号，写了几篇就写不下去。理由很简 单：自己没什么可写的，纯矫情没内涵。 第一次写作草草结束了，甚至除了自己的妹妹，从来没有给 其他人看过。这个时期的我对写作这件事一无所感。

后来，报名了几个写作课，但是纯粹就是花钱凑热闹，买了 放一边。 甚至加入了粥老师的知识星球，除了第一天再也没有打开 过。三个多月的遗忘。

直到有一天，朋友圈无意间看到粥老师写作课的文章，看了 不到 5 分钟，报名了。 接着里面听起了课，一连听了三篇。接下来几天一有空就 听，做思维导图。

为什么会这样？ 我静静想了十几分钟，总结出以下四点：

①虽然放弃了写作，但是却在心里种下了种子，一直在发 芽。

②这段时间处于人生低谷期，想了很多，参加了很多课程， 对于改变人生有了那么一点儿看法。

③在这些课程中，反复出现在我面前的就是：目标，行动， 方法论这些关键词。

④粥老师的写作第一讲：写作认识，起到了临门一脚，真正 拨开我心中的疑惑之迷雾，开始重视起写作，并且生出一股 股强烈的冲动，想要重新开始写作，并且持续下去。

这个写作课对我来说出现的刚刚好。

2.自己对于写作的新认知？

①写作需要公开。 公开的写作好处太多啦，这点粥老师的写作认知课就专门讲 过为什么一定要公开写作。 现在的我已经不是三年前很害羞的我，可以大大方方公开写 作：公众号发朋友圈，社群做分享。因为我知道了，只有公 开写作了，我的写作之路不会秘密又悄悄地消亡。

②要建立自己的写作方法论体系。 要创造出自己的苹果树，让它持续稳定地开花结果。这一 点，我认为粥老师的写作课很优秀，它有一整套通俗易懂的 理论体系和思考路径。 能够付费请到一位前列的前辈传授经验和方法论，站在巨人 的肩膀上，很值，没毛病！ 当然，近期会主攻粥老师的写作课，学以致用。等后面了， 按照我的性格，会去汲取其他前辈们的写作课精华，融汇应

用成为自己的。 这大概就是做主题课题研究，集中精神做主题阅读。所谓书 读百遍，其义自见。只有这个体系形成时，感受到的快乐会 是最顶峰。

③让写作使自己去主动分享，教会别人。最大效率地学习新 知识，更新迭代。

学习金字塔图展示了：学了并且教给别人的学习效率是只做 阅读听讲的十几倍！

写作是性价比最高的学习路径了！
