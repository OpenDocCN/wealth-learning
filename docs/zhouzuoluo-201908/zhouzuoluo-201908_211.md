# 「你的内功，能抵得

LeO : 「你的内功，能抵得上 400 个原子弹」

最近看了一部豆瓣 9.6 分的神剧，切尔诺贝利（不过在豆瓣上 面已经被封了）。

故事讲的是一件真实的事件，1986 年，前苏联切尔诺贝利核 电站发生爆炸，这是人类有史以来遭遇的最严重的核灾难。

8 吨多强辐射物质泄漏，辐射量相当于 400 颗美国投放在日本 广岛的原子弹。

当年有记者问专家，这个地方的生态多少年之后才有可能修 复？

专家回答：至少两万年。

那到现在才过去 30 多年，现在切尔诺贝利还是个“鬼城”么？ 2017 年，有人公布了切尔诺贝利最新的航拍照片。 照片上显示，在废弃的建筑物周围，长满了绿油油的植物，

切尔诺贝利隔离区森林覆盖，郁郁葱葱。

隔离区里甚至还变成了野生动物天堂，到处可以看到山猫、 猫头鹰、大白鹭、天鹅……

还有一只野狼从隔离区溜了出来，经过研究发现，辐射对它 的影响已经很少了，和普通的野狼没什么两样。

我们都低估了生态自身的修复能力，就像我们常常低估自己 身体里的自我修复能力。

我们自身的免疫力，其实连癌症都能战胜。发现这一点，居 然是因为一次细菌感染。

咱们一起穿越回 100 多年前的 1890 年。

有一天，美国医生威廉姆·科利（William Coley）在复习老病 例的时候，一份特殊的病例，引起了他的注意。

这个病例是这么记载的：一个面部癌症的病人，手术切了癌 症组织，但是很快又复发了。更不幸的是，病人的伤口还严 重感染了。

所有的人都觉得这个病人活不了多久了。但是接下来发生的 事，出乎所有人的意料。

这个病人的伤口每感染一次，癌症组织就都缩小一点。反反 复复地感染，癌症居然越来越小。5 个月后这个病人的癌症没 了，他出院了。

科利医生觉得这不可能呀，所以找遍了整个纽约城，终于找 到了这个病人。让病人到医院检查发现，果然他的癌症没 了。

科利想，难道是因为细菌感染激发了他的免疫力，反而杀死 了癌细胞？

有了这个想法，科利就尝试用细菌毒素来治疗癌症。治了上

百例，确实还真的有人得到了缓解。

可惜，他不知道这种疗法的原理，研究也没有严格的设计和 对照，所以后来的医生没法重复，这个方法就慢慢淡出了人 们的视野。

还好，他的女儿建立了美国癌症研究所（CRI），支持癌症研 究和治疗，尤其是免疫疗法。

免疫疗法，就是恢复或者增强免疫细胞的识别能力和战斗 力，发挥自身潜能，用内力来战胜癌细胞的治疗方法。

其实解决问题的方法和能力，很多时候我们其实都已经拥有 了，只是我们不相信自己，不相信我们的潜力。

我们的精力其实还有很大的潜力没被挖掘，我们人体的整个 系统，就像大自然一样，有着极强的生命力、修复能力。

我们需要做的事情，可能是更多地了解自己，并且做出一些 调整，培养一些习惯，让自己的生命力得到更好的开发。 2019-07-02(11 赞)
