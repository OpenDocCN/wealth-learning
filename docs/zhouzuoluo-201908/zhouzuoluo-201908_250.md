# 《高小羊：在自卑中

《高小羊：在自卑中度过的 27 年》

每当和闺蜜说起小时候的事，我总会掉眼泪。就算是一个人 回忆起往事，都能哭出声。

这是我 27 年的人生中，最隐晦的事情，很多时候我不愿意想 起，想原谅父亲，也想放过自己，想和过去的自己彻底切割 开。

在一个家庭中，16 年的学生时代，我一直期待父亲能关心一 次我的成绩，我的错题，能鼓励我，能与我谈谈心，但是直 至大学毕业时，也没能实现。

而聪明的弟弟，在他 16 年的学生时代里，从错题，成绩，甚 至是高考志愿，父亲是花了了不少心血和注意力在上面。

我不愿意把它当成是对我的冷落，我更愿意当成是父亲时间 有限，肯定是优先照顾最好的那一个。

打下这些文字时，我眼睛又浸满了泪水，但是心态比以前更 好了，我知道唯有一直努力变得比过去的自己更好，才能彻 底忘记过去。

我是自卑的，但是同时我又是顽强的，毕竟我高考时最后的 成绩比分数线仅仅多出了 3 份，我又找到了人生的目标，这都 是我在自卑的间隙，延伸出的赌气式的报复，才有的结果。

我想我以后再也不会想起这段往事，我会刻意去淡化它，直 至它消失。
