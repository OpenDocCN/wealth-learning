# 《健身七年成功蜕变

《健身七年成功蜕变，我只走了这五步》

先给大家看一张图，在文末，是我在 2014 年读大学时和 2019 年现在的照片对比图。在此说明两张照片都没有刻意 P 过，视 觉上身高不同可能是拍照时镜头的拿取高度不同。

没错，2019 年我特别骄傲的一件事：坚持健身多年，终于减 肥成功。回过头望，说实话我自己都认为是一个奇迹，在几 年前我从不敢想象自己有一天会穿 S 码的衣服。两个月前新买 的标准中码裙子太宽松时，我高兴得差点跳起来。

很多人问我是怎么坚持的、怎么做到的、怎么运动的，结合 我最近看的一本书整理出来，其实只需要分五步，层层递 进，我们都可以创造奇迹。

第一步：我意识到自己对现状感到不满意。为了改变现状， 我开始采取行动。

我们家六个兄弟姐妹，五个女孩一个男孩。女孩们都遗传到 妈妈和姥姥的“大象腿”基因，衣服穿中码，裤子穿大码，无 一例外都是梨型身材；男孩随爸爸有着令姐姐妹妹羡慕不来 的“筷子腿”。

无言忍受了多年的大粗腿，当然还有轻微的“虎背熊腰”，有 一天我决定要健身减肥，告别这样的身材。

第二步：期待的结果没有出现，我意识到，仅仅只是采取行 动还远远不够。采取的行动还必须以解决困难作为目标导

向。我问自己：我如何使结果变得更好呢？我如何不这么艰 难，而是更有方法的、有效用地进行健身呢？

我开始练习慢跑，每周跑四到五次，每次三四十分钟。期待 的效果没有很快出现，困难倒是先到一步。

这样开始慢跑大概不到两三个月，我的左边小腿拉伤，首次 运动受伤的经历令我印象深刻，走路拉扯到小腿都隐隐作 痛，在此期间我喷过云南白药，敷过药膏等都没有效果，持 续半年之久不能运动，直到某天我不经意跟美国外教聊天时 说到，他教我一个拉伸方法，我半信半疑试了一周，就完全 痊愈了。

再一次踏上运动的道路，这次我更讲究方法和防护。 第三步：学到的健身知识对我起到了一定的帮助。但是，如

果拿自己跟其他成功人士进行对比，我就觉得他们好像不费

吹灰之力就达成了目标。他们拥有富有影响力的、能为他们

开启方便之门的朋友。

换取一些其他的运动方式继续进行，依然坚持每天运动至少 一小时，每周四到五次，每次健身前热身十分钟，健身后充 分拉伸二十分钟。

不仅如此，饮食方面也十分重要需要配合起来，多吃高蛋白 的食物，少吃淀粉类的食物特别是晚饭。这时候我才慢慢懂 得“健身先健脑”。

第四步：在我与自身和他人打交道时，仔细思考一下自己的 世界观。

当我在减肥时，我会发现身边越来越多想要和正在努力减肥

的人群，但是他们的方式和目的却千差万别。

有的人只从饮食方面做功课。节食，晚上不吃饭，或者少吃 多餐，但是不运动。

有的人只运动不注意调整饮食。想喝奶茶想吃披萨随便点， 口中振振有词：吃完才有力气减肥！

有的人一边运动一边控制饮食。高强度运动却低能量摄入， 或者是少量运动少量进食。

这些都是不对的，我庆幸自己遇到一名当健身教练的师兄， 他根据我的体脂测评提供运动方案和饮食搭配建议，然后我 跟着这份指导一直往前推动。

第五步：通过改变对自身的认知，我们可以带来巨大的变 化。

健身四五年的时候身材依然没有很明显的改善，我知道原 因。很有可能是没有去健身房，也没有请私教，缺乏专业的 训练方式和指导。

因为我喜欢自己一个人随心所欲的锻炼，不想去健身房参与 一堆人的流汗氛围，事实上直到今天我也没去过专业健身 房。所以我一直心平气和接受自己没有很明显的改变，接受 自己没能很快减肥成功。

不过，我一直在改变不同的运动方式去刺激肌肉，尝试过慢 跑，慢跑结合快跑，无氧哑铃，健身操，HIIT，郑多燕减肥 操，天鹅臂和腿系列，椭圆机等等。

我坚信每一种运动方式都有它的独特之处，也有它的功能所

在。不同于以前，我不再抱着要减肥要去掉多少斤的目的， 而是开始将运动作为每日必行之事，只是形式多样性可以自 己选择，这样之后所产生的效果是不同的。在运动中得以放 下手中的杂事，获得一片宁静专心出汗，释放多巴胺的同时 也收获很多能量。

后来我发现，自己再也离不开运动，便也再也没有所谓的“坚 持”说法，它更像是一天必行之事，如醒来刷牙，口渴喝水， 肚饿进食一样。

我们的自我认识和愿望，最终会成为自我实现的预言。不单 是健身，人生中很多事，皆可以此类推，演绎成说。
