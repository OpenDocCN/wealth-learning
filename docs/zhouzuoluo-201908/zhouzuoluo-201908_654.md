# 《关于逆商的几点思

《关于逆商的几点思考》

今天看完粥老师写褚老逆商的文章，有几点很受触动。

1.你要学会放下面子

粥老师说 ：“你的面子不值钱，尤其是你实力不够的时候。 ”我非常认同这句话，以前我非常在意别人的眼光，就是因为 要面子，而现在每天被拒绝很多次，我都没觉得特别丢脸。 反而看到了自己的成长，放下面子去努力赚钱。

我还记得小时候因为别人说我一句，我有时候还会想哭，心 理想他怎么能当众说我呢？会觉得特别丢脸，恨不得有个地 洞爬进去。

后面因为销售这份工作就是要放下面子，每天被客户拒绝是 很正常的事情，我就慢慢的放下了面子。

2.事来则应，事去则静 遇到问题，大家都会选择去抱怨，抱怨老板的问题，抱怨朋

友的问题，抱怨同事的问题，却从来不考虑是自己的问题。

刚实习那会，我就是一个抱怨狂，我抱怨这份工作，我抱怨 老板，我抱怨同事。抱怨的结果让我越来越讨厌这份工作， 自己心情也会越来越糟糕。因为抱怨是最无能的事情，抱怨 之后，问题还是没有解决，只是让自己越来越讨厌目前的状

况而已。

后来我经历了一些事情，我发现自己越来越不喜欢抱怨，现 在很少去抱怨。遇到问题，我想到的是如何去解决。

不爱抱怨之后，生活当中的负能量也越来越少，反而多了正 能量，让自己越来越开心。

3.保持乐观，保持信心 我奶奶是一个很乐观的人，我受她影响，我遇到事情也是比

较乐观。我一直相信，只要我去努力，我就可以改变。

遇到困难，我的心态确实还可以。我认为事情发生了 ，只能 去解决，不然还想怎样。 乐观的人遇到事情不会很愁，也不会晚上睡不着，就想着肯 定会过去。

信心很重要，是你会不会取得成功的关键因素。不知曾几何 时，现在做一件事情对自己越来越有信心了，我一直觉得别 人可以，我也是可以的。不给自己设限，按照自己想要的去 努力。

4.拥抱困境，成长即财富 毕业那年对于我来说经历了很多，承受了我那个年龄不该承

受的东西。

正是因为这次经历，我好像与过去的自己告别了，我变得越 来越努力，我调整心态，克服困难。

换成以前的我早就抱怨命运不公平，现在回想起来，我还是

感谢这次经历，让我成长。

实习到工作这几年，我逆商方面提高了一些，跟我这些经历 都是有关系的。

困难并不可怕，关键看你自己怎样去做，选择逃避还是面 对，这才是最关键的。
