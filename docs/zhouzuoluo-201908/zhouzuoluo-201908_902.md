# #来自静水的干货分 静水 : #来自静水的干货分享 改变自己真的没有那么难

01

无论从哪个维度来讲，人都是分阶层的。贵族、精英、土

豪、白领、工薪、穷人都有自己的层级和圈子。圈子与圈子

会因某种机缘产生交集，上升和坠落都有可能。只是上层圈

子跌入下层很容易，下层圈子想向上跨越往往比较艰难。

除了原始的家庭背景外，决定一个人最终能在哪个阶层活跃 的关键，主要来自于此人对其周围世界的认知水平和自身努 力程度。

要想跨越式成长和蜕变，固有的思维如果不打破重新构建， 很难有质的改变。这也是越来越多的人坚信阶层固化并安于 现状停滞不前的充分且必要理由。

一个精致女人为何手持价值八万的 LV 包包，有你不懂的深 情；一个落魄女人为自己寻找种种理由开脱，却有你分分钟 厌恶的矫情。这就是认知水平和视野层次不同所致。

02

改变自己到底有多难？

看一个社会的发展方向，就多留意那些精英的动向。比如关 注互联网行业自然多关注马云马化腾；关注房地产行业就多 关注潘石屹许家印；关注知识付费自然多关注李笑来；关注

未来教育的发展方向，自然多关注俞敏洪…..

精英是时代的领跑者，对未来方向的认知多半高屋建瓴，向 这些优秀的人学习，终究不会错。

赵薇花 1 个亿在香港为女儿买学区房，这几天成了媒体热炒的 焦点。其实我们理应更多地把目光停留在关注赵薇的选择 上，而不是聚焦 1 亿元这个数字上。

一个演员被称为人生赢家，评价维度往往不是单纯靠演技。 这其中会包括事业婚姻家庭育儿甚至在圈里的口碑。北影毕 业的女星多了去了，真正玩转投资理财的恐怕不多。不得不 说赵薇是个智慧的女人。

03

在我看来她能适时投资让赚到的钱生钱，也能让更多的钱花 到最有价值的地方才是她的高明之处。一个舍得为教育投资 的人，注定拥有精品人生，傲娇地站立在常人无可企及的高 处。

但凡注重个人能力提升和家庭教育的人，哪怕普通得如草 芥，日子一般都不会过差。你必须承认当下社会，穷人上升 的主要通道还是靠读书。所以那些动不动就鞭策高考制度的 普通人，你就闭嘴吧。没有高考，穷人的上升通道才真的关 闭了呢。

也许你会大放厥词，以某人小学不毕业却成为了千万富翁的 个例来和我对质。我告诉你即便有这么一小簇圣人，后天的 成长和历练，也绝对和其本人自我教育及提升有着密不可分 的关系。

穷人之所以一直是穷人，是因为穷人的贫穷思维一直都在， 穷人的认知理念一直都没有进步，穷人的圈子一直都不会主 动向外延伸和渗透。

04

君是三年前我教过的学生，和她产生交集，源于她慕名而来 跟我学习基础会计。目的是为了考下会计从业资格证，方便 就业。那时的她，青春靓丽，肤如凝脂，一头飘逸的秀发， 走到哪里都不缺少回头率。只是后来，课程没有结束，不知 为何她选择了放弃。

通过微信联系上那天，隔着时空我能感觉到她的兴奋和喜 悦。失联的这三年我们的人生都发生了很大的变化。从外地 回来后，她第一时间赶到了我的工作室。

抬头望见她的那一刻，我内心不由得颤抖了一下。短短三 年，整个人变化惊人。憔悴的脸蛋，长了不少痘痘，一头长 发侧露出不少银丝。她说很后悔没有听我的话，坚持到底。 如果那时能考个会计证也不至于这么落魄。

在宁波那么大的都市，能去的地方也只有工厂的流水线作业 间。每天十三四个小时的高强度作业，熬夜熬得实在受不 了。但实在不想回家，更不甘心面对犹如一潭死水的婚姻， 又放不下自己三岁不到的儿子。

05

我抛出了我的观点，起点低不可怕，不用心才可怕。我给她 讲述了我月子里用过的两个月嫂魏姐和张姐的故事（详见今 日推荐）。也和她分享了我本人以及注会学友海玲的励志人 生故事。

其实，处在社会最底层的人，不是没有上升通道。而是压根 就没有相信过“通过自己的认知提升、努力奋斗、个人规划即 便不能改变命运，也可以改善人生”不是传说也不是童话的可 行性和真实性。

多年前，面对职业天花板和职场冷暴力的双重逆境，我为自 己规划了一条路。拿下注会，走执业化道路，即便后来没有 成功。回想起挑灯夜战的那些岁月，依旧会被自己的努力和 坚持感动。

大国企的财务，业务事项细化到极致。每个财税人就像一颗 螺丝钉，经年累月地固守着属于自己的那个小小洞口。很难 接触到全盘业务。改变自己的出路在哪里呢？如果有一天， 连这颗螺丝钉都不被需要了，又能去哪里呢？波澜不惊的职 场生活，时常让我萌发出一种居安思危的忧患意识。

06

我琢磨着一个务实的非会计专业的财经人，先从攻克中级会 计师入手吧，这个证书至少有点含金量。第一年，我和小我 六岁的小师妹一起备考，每每学到会计实务的长期股权投资 我就会傻眼。反复听张志凤老师和郭建华老师的讲解，依旧 一塌糊涂。小师妹三科全过，我只过了一科，难过归难过， 但我不会放弃。

第二年，提前十个月，我开启备考模式，分解目标，每天无 论有多累都要确保完成。很多时候刚刚躺下，突然想起来还 有个事项没有搞懂。我会立即爬起来，去翻书。张先生说我 走火入魔了。亲爱的大宝那时才五岁，时常用她的童书圈成 一个长筒，对着我喊话，妈妈陪陪我吧。百感交集五味杂 陈，心里会泛起阵阵酸涩。

但是坚持到出成绩的那一天，工作人员微笑着恭喜我拿下了 全区双科总成绩第一。那一刻我的眼泪差点没有掉下来。礼 貌性地表示了感谢，就急匆匆地走出了人事局职称科的大 门，再一次翻开那个烫金的小本本，热泪盈眶。

07

因为这段艰苦卓绝的跋涉和努力，我才拥有了更多选择的权 利。开办 YY 会计公益课程、踏上高校财税课堂的讲台、公司 内部培训师的讲台……我想尝试的事情都去做了，并且做到 了。

备考注册会计师的茫茫人群中我遇到了海玲。一种相见恨晚 的情绪，促使我们如约相见在她家门口附近的小餐馆。她映 入我眼帘的那一刻，直觉这个其貌不扬的女孩，背后有太多 故事。

安静柔和的眼神，略显沧桑，却很坚定，浓浓的书生气使我 不敢相信她已经参加工作多年的事实。海玲说，虽然和我同 岁，但是自己只有中专学历，毕业后什么样的工作都干过。 但成为 CPA 梦想的小火苗从未熄灭。

08

因为通过这个天下第一考，也就意味着拥有更多更高档次的 选择。见到她时，她已经苦苦坚持了七年，就在本市这个师 范学院里蹭教室蹭课。已经过了五科专业课，还有两门综合 即将上岸。在这七年里哭了很多次，但从没有放弃。为了省 钱，她在淘宝上和人拼课程。

某日终于听到了她拿下 CPA 小本本的好消息。这不仅仅是一

本证书，而是一个人对自己人生的认知、把控、奋斗、蜕变 和成长的历程。海玲目前在一家上市地产公司任主管会计， 收入不菲。这分明就是丑小鸭逆袭白天鹅的真人版。

我告诉君，海玲从最底层至少跨越到了白领阶层。尽管离精 英阶层还有那么一段距离，但坚持下去也为期不远。只要自 己不被自己打败，一切皆有可能。

改变自己到底难不难？取决于你的认知理念。看准一个目 标，制定好详细的计划，勇敢迈出第一步，然后反反复复的 练习。成功哪有什么捷径，不过是苦尽甘来的结果。

我问君会不会诸如 WORD、EXCEL 之类的办公软件，她摇了 摇头。我说真的很抱歉，一个办公文员的职位我现在都无法 帮你推荐。这么多年，你的时间和精力都花到哪里去了。

我请君在一个很有特色的小餐馆共享了午餐。期间我们聊到 了婚姻家庭孩子。我为困惑的她推荐了那本我读了 N 遍的《亲 密关系》和《正念的奇迹》经典书籍。

平凡的人生犹如一棵会开花的树，只要用心经营，就会结出 果实。也许我们一辈子努力的终点，都到达不了理想的阶 层。成不了贵族、精英、土豪，但至少可以通过我们的奋斗 成为白领。不至于在社会最底层的泥沼里做爬不出来的穷忙 蚁族。

改变自己其实没有那么难，只要你愿意，什么时候开始都不 算晚。Those that don’t kill you will only make you stronger!

谨以此文献给那些所有身处困惑又不甘于向生活俯首称臣的 人儿。记住，心在哪里，收获就在哪里。
