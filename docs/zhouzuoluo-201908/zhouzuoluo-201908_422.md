# 《我看到了那条完美

章子 : 《我看到了那条完美的人生曲线》 这篇文章我想聊聊在心里藏了很久，一直想说出来，关于

《人生轨迹三部曲》的话题。

貌似话题很大，却是一个值得探讨的问题，它能点醒很多

人。为什么想到这个内容呢？

前几天，我听了一个大学教授精彩的讲话，“人生何需迷茫， 全凭蓄意设计”，他说:人生的成长是必然规律，就如我们学完 了加减法，要升级学乘除法一样自然，关键是得注入灵魂， 把握好方向。不断地的成长递进，我们的心智才会成熟。

一个人从成长到成才再到成熟，这个过程就像一条完整的三 部曲它承载着全部的奋斗史和他付出努力呈现的价值。优美 或者难看、欢笑或者痛苦。

在三部曲的人生轨迹里，有人思维开阔总是引领者，有人百 般努力却始终做着追随者。

怎样才能解除魔障走好自己的人生三部曲呢？ 一.成长:奋进一部曲 成长并不轻松美好，它是开拓向前的奋进过程。我朋友程程

的女儿，在一家教育机构工作了三年，昨天在朋友圈突然宣

布不干了，我很惊讶，这个工作是她一直骄傲为荣的，当初

她为了进这个机构竟放弃了异地相恋的男友。

这个女孩工作很努力，个人素质也高，艺术类的证书就有好

几个，为什么轻易就辞职呢？我百思不得其解，难道想个人 创业吗？我带着这个问题去了程程家，等我与女孩聊过才知 道，并不是要自己创业辞职，而是干的与反馈的有了不平 衡，看不到自己的价值和前景，心灰意冷了。

她说:“我大班、小班带着几十个孩子，每天累的跟个瘫狗似 的，有什么用？评奖、考评总在靠后，更别说升职了，有什 么意思？”我问:“你的同事朋友都这样的不平等吗？”“不是， 他们大部分能说、会写还会做，比我素质全面，我就是想停 下来进行刻意学习，提高自己。”

哦，挺清醒的，也有目标那就祝福她吧！随着时代的发展社 会的进步，成长的内涵边界不断的向外延展，单向的努力已 经不符合人才的要求了，这种新时代的要求像潮流一样迅 猛，很多的人都有很强的危机感。

除了企业内部的激烈竞争外，新生力量多技能的冲击，都在 刷新着新的职业要求，除此以外，智能科技的进入也逼迫着 人们开始重新考虑，自己成长内涵的升级，迫使自己开发潜 能，突出自己的亮点，把自己打造成时代需要的复合型人 才。

什么是复合型的人才？ 所谓复合型人才，就是具有宽阔的专业知识和突出的的专业

技能，有广泛的文化素养，具有多种跨界相关的能力和发展

潜力，以及和谐发展的个性和有创造性思维的人才，俗称“一 专多能”。

看了对人才的要求让人惊出一身冷汗，我们在奋发努力的成 长，而成长的内容也在不断迭代、更新，我们这部成长圆舞 曲跳得很艰辛。

然而，再艰辛也要坚持下去！我们都听过司马迁写《史记》 的故事。司马迁在少年时代就涉猎群书，立志要继承父亲的 事业，当他决定要写一部真实的社会现状，作为资料留给后 人时，就做好了要吃百般苦的心理准备。

为了实现这个愿望，他开始刻意要求自己亲自去收集真实的 故事和现实的史料，一点一点的整理、编写历史真貌。他为 了获得有真实内容的资料，跋山涉水、苦灯熬夜。正当他准 备好了丰富的资料，想接着书写时，因受“李陵事件”的牵 连，被捕入狱遭受了宫刑。

这是一个人的奇耻大辱，但是司马迁没有为此沉沦更没有放 弃《史记》的编写工作。可是就在他快要完成编写的时候， 第二个更大的不幸又一次打击了司马迁。

他成箱的写好的手稿因人不慎，被一把大火烧掉了，这是他 的心血是他的命啊！司马迁真的快被重重的打击击垮了，他 欲哭无泪，叫天天不应，叫地地不语，几天的沮丧之后，他 很快振作起来，凭着记忆开始重写《史记》，终于在他吃尽 苦头之后，《史记》完稿了，才有了流传百世的经典之作。

这个感人的故事让我深深地崇拜这个不畏艰辛，为了心中的 梦想不屈不挠去奋斗的人，每次读《史记》我都是倍加珍惜 和珍爱，司马迁的书籍和精神，都是为后人留下的大笔宝贵 财富。

与司马迁遭受的苦难相比，我们的那点艰辛算什么呢？从司 马迁的身上我们明白了:当一个目标确定下来，我们就一定要 坚持下去，这才是追求成长的内涵所在。

时代需要人才也造就人才，如今，我们赶上了这个经济、科

技大发展的时代，就要适应它，保护它，让自己的成长，再 强劲一点，再扩大一点。

二.成才:创新二部曲 你已经成才了也不意味着可以放心无忧，你得不断创新，不

断刷新自己的产品来，满足市场的要求。因为技术的变化节

奏太快，你的技术、设备有随时被淘汰的危险。所以学习和

成长仍在激发着我们的情绪。

那就是:“重修一个决定生死的能力”，这个能力叫:创新。 创新很重要，它不仅是企业的核心能力也是个人的核心能

力，创新思维让乔布斯创建的苹果 Apple 从 1976 年到 2016 年兴 盛了三十多年，直到被锐意进取、创新能力更强的华为碾

压。

今年四月华为自主创新的 pro 上市，我从视屏上看到世界各地 排大队买 pro 的盛况，当我也兴冲冲加入到手机换新队伍时， 看到华为专柜门庭若市，而紧紧相邻的苹果专卖空无一人， 偌大的差距透出的炎凉让人不寒而栗。

不是人无情，而是品无意。 真实的现状告诉我们，学习和进取一刻也不能停，尽管已经

成才但是成才二部曲仍要振作精神不能放慢成长的节奏。

我们要抓紧建立重组思维，学会拆解方法，进行多元组合和 链接新技能，让自己有效的控制好成才路上的风险，稳稳 的“活下去”。

记住创新中的个人品牌三要求:1.新元素 2.价值增量 3.能实现。

只有胜利的度过创新二部曲，恭喜你顺利到达另一个高度的 境界:成熟三部曲。

三.成熟:平静三部曲 到了这个层次，人不再年轻，年龄也到了“知天命”的阶段，

这时你的经济资源、人脉资源达到了相对的高度，你的阅

历、思想和认知已经变得深厚和宽广，你的胸怀如大海一般

容纳的下任何的波动，你的无欲将你变得静如柔水，你的看

淡将你变得大智若愚。

看看马云，看看柳传志，看看王蒙，再看看曹德旺。他们这 些拼尽全力、努力奋进跳着二部曲过来的人，那个不是活通 透了的人呢，他们在自己人生成长的轨迹上最大限度的开发 了自己的潜能，他们实现了自己的的人生价值，他们在真切 的践行过程中做到了

数树深红出浅黄 繁星闪亮领头王 没有了遗憾，没有了烦恼，就像一个人历尽艰险奋力爬到最

高的山顶，看到了世界无比的大，自己只是那个“无比大”中 的“无比小”。

成熟的人:内心安详，不再荒凉。 最后，我想说只要精心设计过自己的人生，清晰的明确自己

要走完整，这人生的三部曲的人，并且无论遇到多么诱惑和

艰难，都没忘记初心仍然沿着成长这条轨迹，不放弃不抛弃

的走到底，你一定会有个完美的，让自己骄傲的人生。
