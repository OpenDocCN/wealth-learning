# Day196《真诚

《真诚是变现最好的套路》

变现模块就是赚钱模块，大方赚钱，让自己的价值可以变 现，一方面增强信心，一方面获得回报。

但是，普遍的人性期望就是赚到，而不是被赚到——也就是 吃亏。当我们想去变现时，意味着别人需要付费给我们，让 用户心甘情愿付费并不是一件容易的事，目前朋友圈不少人 的变现姿势不对，让用户觉得被别人赚走自己的钱而产生了 抵触心理……

那么，到底如何更好更得体地变现，我认为真诚是最好的套 路。

1、一对多的真诚：在朋友圈讲故事 一旦让用户进行理性思考，其实就是开启了防御机制。于是

各种段位的销售文案利用理性的漏洞展开了全方位的进攻，

其中最容易突破的就是人的情感，这在营销圈被称之为“情感

营销”。

而最常见的情感营销手段就是讲故事。故事天然具有进入用 户心智的优势，因为听故事是人的天性。

在朋友圈可以讲你自己的故事： 你是如何一步步打造自己的 产品和服务的，花费了多少时间和心血，要让用户感觉赚

到，而不是被赚到。

还可以讲你的用户故事：TA 是谁，有什么问题，经历了什 么，发生了什么改变，通过你获得了什么。在朋友圈持续 讲，文案尽可能朴实，千万别失真了。

这样的故事天然具备传播性，可以被动吸引用户来找你。

2、一对一的真诚：与用户进行深度沟通 如果平时从来都不关心用户，也很少跟用户聊天，凭什么你

一在朋友圈发布产品，大家就去纷纷购买？

用户没有感知到你的价值，不管是对你的人，还是对你的产 品。了解和信任是通过平时来建立的。

每天都有计划地跟用户进行一对一交流，听一听他们的想 法，问一问他们的顾虑，用户是能够感受到你的真诚的—— 这不同于强推产品，只有想赚他钱的时候才想起他。

在交流的过程中，还可以了解他的预期，及时提醒你的价 值，遇到合适的人还可以主动帮你传播。

最后，再探讨一个问题：真诚和套路的界限在哪里？有句话 说得好： 没有什么可以通往真诚，真诚本身就是道路。所以 啊，真诚自在人心，总有人会感知到你的真诚，只要你是真 心的。
