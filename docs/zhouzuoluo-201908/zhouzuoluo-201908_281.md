# 33 《你只

《你只有足够优秀，才会被世界温柔相待》

不知道大家有看过奇葩说吗？我印象最深的就是肖骁。 第一季他一出现，屏幕上写的是，有妖孽，从远方来，很多

人骂娘炮，蛇精男。

第二季，第三季，也有很多人骂他。 第四季发现他不仅金句频出，还获得了冠军。他辩论时，屏

幕上写高光时刻，高晓松和蔡康永对他也赞不绝口。

一个人足够优秀，就会感觉非常公平。 穷富翁大作战中，那些有钱人体验清洁工或者照顾孩子的单

身妈妈。

一位身家过亿的富翁说，自己一定可以改变现状。 等到他做清洁工时发现，每天晚上两点才睡。还没钱租房

子，只能找纸箱子在大街上过一夜。

第二天找工作时，都被拒绝了。理由是没有固定住所。 关键是，你都没钱吃饭，怎么会有住的地方呢？ 这非常不公平，但是因为没钱租房子，所以没法反抗。

好不容易找到了送外卖的，因为送货慢，不到半小时被开 了。

而睡觉用的纸壳被人捡走了，当时他都快哭了。 穷人生存都是问题，每天工作 12 小时多，哪有时间精进自己

啊。

你只有越强大，社会才公平。 我学历不高，刚入社会时，第一份工作是市场推广。 从早上八点，到各个店铺推广我们的产品，一直到晚上九点

半下班。

一个多月后，我非常迷茫，我想让自己变得有价值，可以帮 助更多人。

但怎么帮助啊，做什么努力啊，一概不知。 经过一年时间探索，我学习了新媒体写作。渐渐地上稿。 我成功到深圳工作就是跟文章有关，HR 喜欢我的文章，录用

了我。

要知道，我之前应聘从没成功过。 来公司后，同事也对我很好，有人还夸我。 我真心觉得你越强大，世界越公平。我就是个例子。

我是柒月，希望和你们一起长成自己喜欢的样子[爱心][月亮] 2019-06-19(24 赞)
