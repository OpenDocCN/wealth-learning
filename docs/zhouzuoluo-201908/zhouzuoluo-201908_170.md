# 『任何人能做任何事

『任何人能做任何事』

晚上好 与星友们分享丹儿的第 76 篇原创 7.9

————————————————————————

“任何人可以做到任何事”，这句话你认同吗？是不是在想“怎 么可能？人与人之间都是有差距的，同一件事有的人做起来 就很容易，可有的人就做不好。怎么可能能做到任何事呢？”

以前我也是不认同的，但是现在的我是认同这个观点的，下 面分享一下不认同时我的想法。

【不认同时的想法】 以前看到别人取得好的成就，我的内心活动： 他好厉害呀，我就不行。没那天赋，也没那运气。差距就在

那，努力了也没人家做的好，自身能力就在那。

有了这种想法以后我就心安理得的不去思考，想怎么能达成 自己的目标，不去行动，完成自己的目标。

举一个小例子

以前有一个同事，她怀孕以后，就一直成交大卡，30 万，50 万的好几张，后来又成交了 100 万，和 1000 万的卡。大家都说 她是运气好，她的孩子给她带来了运气。

我当时也觉得她是因为孩子运气变好了。现在想想还挺迷信 的呢

看到朋友取得了优异的成绩为什么会认为别人是因为运气， 而不是凭实力呢？因为只有这样想，我做不到她的成绩，我 就有借口了呀。我运气不好嘛。

现在想想我朋友也是积累了四年，才开始厚积薄发，也可能 是因为怀孕，让她有了压力，才更加努力的办卡。一张卡是 运气，后面那么多的大卡都说明这不是她的运气，而是实 力。

我经常看到她下班了还没走，接待会员，怀孕了也没有休 假，没有一点孕妇该有的特权。和没怀孕一样该做什么做什 么。但是以前的我把这些细节都忽略了，因为我不想承认她 是靠自己的努力获得的成就，不想承认自己的无能。

业绩不高就从别人身上找原因，顾客没钱，没质量。其实都 是我自己的原因，能进我们店的顾客都是有钱的顾客，只是 我没有开发出来，没有一个顾客一进来就办大卡的，都是一

点一点建立信任感，顾客才愿意在你这里花钱。

★自己平时没有努力背专业知识，也没有用心维护顾客，业 绩上不来，还总在想，我已经很努力了，业绩上不来我也没 办法。我就这样，不停的抱怨。

那时候如果有人和我说“任何人能做到任何事” 我肯定会说，那你去做吧，你先做到再说，我是做不到。因

为我缺乏的是改变自己的勇气。

【认同时的想法】 现在我为什么认同“任何人能做到任何事”这个观点了呢？

因为我原来那么不愿意改变自己，那么没有能力的人，现在 通过自己的改变做到了一些我以前认为我永远做不到的事， 我更加的相信人的潜能真的是无穷的。

只要你想，并且为之努力，就可能会办到。就算办不到，没 关系，换个方法，下次就好啦。

促使我改变的，就是我寻找到了“改变自己的勇气。”

【改变自己的勇气】

通过环境改变自己

很多人认为改变应该是由内而外的，其实自己想改变靠自己 是很难得，几十年的习惯那么好改呢

我一直想改变，但是每次遇到丁点挫折就会给自己找退路， 歇歇吧，歇歇就放弃了。

外界的影响就是我的前领导，还有早起群，知识星球。 她真的是改变了我的人生轨迹，她本身就很积极，已经那么

优秀了还不断的提升自己。这样的人以前我的圈子里是很少

的。看到她的样子我就想去学习，她也很乐意帮助我，先引

导我自己思考，帮我分析我行动力慢的根源，增强我的行动

力。

开始行动以后就想认识更多的积极的人，暖暖是知识星球里 认识的，但是她写了一篇文章“为什么我要早起呢？因为我要 赢得人生啊。”

后来我主动加她好友，她还把通过那篇文章加她的人建了个 早起群，群里的小伙伴很多四点就起床了，我也受到了刺 激，大家都好优秀啊，关键是这么优秀还这么努力，这些都 变成了我学习的外在驱动力。

通过学习思考模式就会发生改变，行动力慢的原因也找到了 解决方法，行动力一上来，只要一行动并且坚持下来，就会 形成习惯。各个方面也会得到明显的提升。

＊＊＊＊＊＊＊＊ end ＊＊＊＊＊＊＊

▼希望大家都能寻找到“改变自己的勇气”，去思考去行动， 试试再说。万一做到了呢？没做到也没关系，科学家的实验 成功还经历了多少次失败呢？失败了没关系，下次换个方法 试试。

今天的分享就到这里了，希望我的分享能让你受益。
