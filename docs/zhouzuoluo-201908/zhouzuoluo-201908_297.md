# 加入星球的第二十一

张金丽 : 加入星球的第二十一天 分享的第二十五篇文章

《九个字：揭露你一事无成的真实原因》 慧慧说这幅玫瑰图是她用时最长的一幅画，因为女儿一直在

旁边捣乱。

可能对于你们来说，画的不怎么样，连初级水平都达不到， 但我却深深的震惊了。

我和她一同买的彩色铅笔，我的就刚买的那天画了一只小白 兔，从此再无进展。

我问慧慧为什么我没有她进步快。 慧慧说了九个字：没时间，觉得难，不用心。 1.没时间 今天应做的事没有做，明天再早也是耽误了。

从开始决定画画那天开始，我就告诉自己每天一幅画。

有的时候白天工作特别忙，忙到没有时间坐下来，中午回家 吃饭的时候就不午休了，利用午休时间画画。

有时候连午休时间也没有，那就晚上就等哄完孩子，洗刷好 开始画。

有一次，23：58 分，距离一天结束还有 2 分钟，我刚坐在书桌 前开始画画。

本来自己画的就不好，加上心里特别焦躁，画出来装在玻璃 杯的红豆布丁。红豆没有诱惑，布丁没有粘稠，玻璃杯没有 光泽。

比例尺没有掌控好，杯子短粗，没有一点精致的美感。 看着日期变成了第二天，太过着急的没有等待签字笔墨迹完

全干掉就开始用橡皮擦初稿，所以画纸上有很多污迹。

那幅画虽然很失败，但我从坚持画那副画中得到了几点经 验。

①时间安排还可以更合理。既然墨迹干需要时间，那我可以 先画出初稿再去洗刷。洗刷好去擦除墨迹上颜色，时间至少 可以节约十几分钟，并且画稿上也不会存在污迹了。

②选好绘画工具笔。工欲善其事必先利其器，这是最基本的 道理。绘画的签字笔不能过粗，太粗的签字笔不仅难干，最 重要的是不可控性增加了。还有就是轮廓太过鲜明，反而让 事物显得失真。

③参照物准确固定。起笔点的参照物我用的杯盖，画了一部 分又改成了杯底。这样就导致，两者结合处像是拼接图形。

所以，不要因为没时间就停止画画，真理是从实践中产生 的，你只有不停地画才能收获灵感和技巧。

高中学过的卖油翁的故事你还记得不？卖油翁能从铜眼中把

油穿过去，你以为是运气吗？

熟能生巧尔。而时间就是保证巧的前提条件。所以不要再说 没时间了，没时间就去挤，一辈子不挤就一辈子不会成功。

2.觉得难 任何一幅画，从铅笔画出第一个线条开始就已经离成稿一步

之遥了。

不要总是被眼前的这幅恢宏的巨著给吓倒，把一幅画拆分成 若干个方块，一步一步来，不要慌。

我一般分为三步：

①首先你可以在画纸上平分为四个方块，把你要画的画也分 成四个方块。找好各个方块之间的相对位置，免得画出来的 成品像是剪切拼接的。

②接下来就是画初稿了，最简单的那种，把各种元素简化成 正方形长方形三角形圆形。然后再根据你要绘画的事物特征 修改边缘、添加内部条纹等。

③涂色时一定要多涂几层，就像涂眼影，从浅到深，涂出层 次感和立体感。在背光区和重叠区涂重些，画出阴影感。

在不违背人性道德和法律下，任何别人能做到的事情，你也 都能做到。绘画如此，其他尽然。

正所谓关关难过，但关关皆过。

3.不用心

我们是不是经常听过这样一个理论：“一万小时理论，在任何 领悟，经过一万个小时的刻意练习，你都会成为该领域的专 家。”

我要说的是，再刻意，不用心也是没用的。 艺术虽然高于生活，但归根到底是源于生活。绘画即使是天

马行空的想象，多多少少也有现实事物的影子，在你想要画

一个事物之前，你可以先找到这个事物，用心的看看它，和

它聊聊天。

今天我画的是玫瑰，我们家没有玫瑰，但是有月季，月季花 和玫瑰同属一个科，有些特征是相似的。

我走到月季旁，发现花朵有很多种形态：有抱着枝头的，有 绚烂绽放的，有含苞待放的，有暗自积蓄的……

再看叶片。有深浅不一的，有枯萎泛黄的，有生病夹杂黑色 斑点的，地上也有化作春泥的……

叶片的边缘有锯齿状，叶子的纹路几乎是对称的…… 所以虽然我的画现在不好看，但是只要我保持有时间、不畏

难、用心察。终有一天我会成为一名画家的。

是啊，我知道，终有一天，会有一名新锐画家横空出现：慧 慧。
