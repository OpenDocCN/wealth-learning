# 高效能写作者的高效

高效能写作者的高效习惯

01

当今时代的碎片化时间

在朋友圈里，经常有这样的事：有些人立下 flag 要死磕一年， 到头来却选择了删除动态，最终一事无成。

也许说的就是你。 你的原因无外乎这样：

上班后你要兼顾工作和生活，你的时间会变得支零破碎，想 做一点正经事情都抽不出整块的时间。

这个问题也是很多职场人共同的苦恼。 如何解决这一困境？

在工作和生活中，寻求大块时间，来专门做这类需要专注静 心的事情？

我觉得这不现实。 没有微博微信的时代，我们的工作时间早就被电话，邮

件，QQ，会议打断成了碎片，而微博微信这样的移动媒体只 不过加剧了这一趋势而已。

所以，真正的问题应该是：

有没有办法利用碎片化时间完成系统的工作？ 也许你认为一个项目，就一定需要大块的时间来完成。 事实上我们仔细回忆写方案的过程，有两种情况，第一种情

况是下笔有如神，给你一个安静的时间段，你可以一蹴而

就；第二种情况，你写了好几个开头，都无法深入下去，只

能换个时间再写。

为什么会有这两种不同的结果？ 其实答案很简单，写起来顺畅的方案往往是因为你原来就这

个问题有很多的积累，所以给你一个安静的时间段就能下笔

千言。

假如你对要表达的问题缺少积累，给你再多的时间也无法写 好。

明白这一点，你就会意识到，有些工作的确需要大块的时 间，但如果不是提前做够铺垫，这些大块的时间并不会有效 率。

02

三个掌控碎片时间的高效写作习惯

【1】一有灵感马上记录，否则很容易忘记。 方式有很多，记在便签里，印象笔记里，有道云笔记里，这

些都是可以的。 重要的是要养成快速记录的习惯。

【2】要马上抽时间把灵感变成提纲 这句话该如何理解呢，换个方式说： 灵感，是你脑海里突然冒出来的一个点。 然后，你赶紧记录下来。 等手头空闲后，你再围绕这个点，进行联想：纵向深入，横

向拓展。

这样就出来了，一篇大致的提纲。这个提纲，就是一篇很好 的文章结构，文章大纲。

有了这个提纲，接下来，你就会有意识的在你今天的工作生 活中，刻意的注意与你的提纲有关的素材，或者是刷碎片化 文章偶遇的金句，或者想法。

这其实叫做“孕妇效应”， 孕妇效应就是当你怀孕了就更容易 发现孕妇，你开了奔驰就更容易看到奔驰，你拎个 LV 就发现 满大街都是 LV。

你列的提纲，其实就是一篇文章的框架。 比如我这篇文章的提纲就是：01 碎片化时间，02 掌控碎片化

时间写作的高效习惯，03 写作思维论

我在碎片时间列好提纲后，我只需要在晚上拿出一段时间， 对这三个框架进行展开、补充、联想、举例、金句提炼等等

工作，完成之后就是一篇文章。

【3】平时有意识积累各种写作框架 互联网时代下的写作，比拼的是文笔吗？ 并不是，因为互联网的大部分用户是达不到能欣赏高雅文章

的水平的。

所以，互联网时代的文章比拼的是真诚，有用，简洁，流 畅，有说服力。

而这样的文章，往往有很明显的结构框架。 很多朋友不会写作，一方面可能是语感，词汇需要积累，但

重要的是他积累的写作框架不够，这样到了要做碎片化写作

时，他很难在很短时间内完成写作构思。

没有写作构思串联后续的思路，就很难利用碎片化时间去搜 集素材，最终完成写作。

幸运的是，当今很多书籍都有讲写作框架，比如：三段式写 作框架，金字塔原理，黄金思维圈，SCQA 法则等等。

只要你留心，持续输入一段时间，你便能积累不少框架了。

03

最后掏心窝子和你说一点

进行写作输出，其实就是改变你的思考模式。

学会一件事的最好方法就是教别人。 所以，你输出结构化文章的过程，其实也是你进行结构化思

考的过程。

这个世界是复杂的，动态的一个系统。 要想混的更好，赚更多的钱，你就必须要用结构化的系统思

维去更加了解世界。

当你对这个世界的认知程度上去了，你的结果肯定不会差。
