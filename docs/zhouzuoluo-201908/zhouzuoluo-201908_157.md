# 硬币真的只有两面吗？

硬币不是只有两面，还有“第三面”。但大多数人，往往只是 看到了它的两面。

都知道，硬币作为一种不可或缺的商业交易媒介，有正面和 反面。日常生活中，购物时找零，是获得硬币的主要方式。 硬币币值低，很多时候，我们放弃找零，用其他小商品替 代，或者直接扔到购物袋，回家后扔到某个角落，或者干脆 不要了。

在这样的情况下，不说注意到硬币正反面，就连硬币本身， 都被我们忽略了。

就算我们注意到了硬币正面与反面，往往也就止步于此。在 使用硬币时，手指的触觉告诉我们，硬币有两面；翻转一下 硬币，视觉告诉我们，硬币有两面。

于是，我们就真的以为，硬币，只有两面。好与坏，白与 黑，成与败…不知不觉，就形成了这种二元思维定式，简单 粗暴地将任何事物划分为对立的两面。

这种二元思维，限制了我们认知世界的能力，让我们看不到 更多的可能性，进而阻碍我们的成长。

思维决定行动，行动决定结果。打破二元思维，我们才能看 到世界的多元化。

首先，必须时刻提醒自己，世界是多元的，不是简单的只有 两面。就像，这世界，不是只有白色和黑色，还有更多五彩 缤纷的色彩。

其次，保持好奇心，打破常规地行动，形成新的思维习惯。 比如，只需要一个简单的动作，就能看到硬币的“第三面”， 甚至更多面。食指和拇指捏住硬币，硬币平面与视线平行， 看到的硬币是一条线；在桌面上，将硬币快速旋转，看到硬 币是一个球形物；将硬币切割成两半，看到了硬币的里面。

硬币不止两面，打破思维的枷锁，采取打破常规的行动，看 到更加丰富多彩的世界。
