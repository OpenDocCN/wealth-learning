# 第 14 篇分享《在知

《在知识星球收获安全感》

李笑来老师的文章，提到动物双眼位置不同对思考的影响。 说的是，低级动物的双眼长在两侧，他们可以看到上下左右

前后的区域，没有视觉盲区，这是极为安全的配置。

但是他们的局限却是双眼无法集中在一个位置，没有办法长 期、仔细观察任何一个点，从而无法进入深入思考。

为何要进入深入思考，因为思考，帮助我们分析过去，筹备 未来，而不是鼠目寸光，只求当下利益。

例如挤破头才得到的铁饭碗，却不得不下岗。 高级动物进化后，眼睛长在了正面，开始有盲区，等同于舍

弃了部分安全感。

舍弃的部分，他们是会想办法补上的，例如与其他动物协同 合作，进入有效社交。

所以，我们可以试着放下一点点安全感，学会深入思考，与 他人社交，回顾过去，活在当下，展望未来。

李笑来老师提出：在生活中找几个小领域试试“放弃全世 角”，比如关掉朋友圈，或者只看几个朋友的朋友圈。

对于这个提议，我很有感触，自从进入知识星球，我的注意

力集中在这里了，每日学习和输出，很少有时间打开朋友 圈。

知识星球开始升级的头几天，我有一种终于放假的爽感，时 间长了心里有点慌，但终归还是踏实的，因为我知道这里今 日恢复使用。

近期没有持续更新文章，但都在阅读和做笔记，这是知识星 球带给我的影响。

确信自己在成长，这就是安全感。 2019-04-11(7 赞)
