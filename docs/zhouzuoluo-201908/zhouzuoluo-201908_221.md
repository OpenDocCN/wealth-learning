# 《想要跃迁？先不断

《想要跃迁？先不断挑战自己的极限》

分享一个本人的失败案例给大家，并引起思考和提出解决方 案。这个案例发生在昨天，关于时间管理。

昨晚开营仪式，本来七点整下班的我准备去食堂，按照惯例 喊了我老板一声去不去吃饭。没想到，不喊不知道，一喊毁 所有！

本可以顺利下班的我，喊了老板之后，她抬起头准备回复 我，但忽然之间好像想到了什么。然后小跑到我面前：跟不 跟姐去吃饭？

我说这不说了嘛，走呗。 她伸长了脖子，说“不是，出去吃。” 再三拒绝不了的情况下，我被老板拉去和其他部门的火锅局

吃吃喝喝四个小时，期间我不断跟上训练营的进度，时不时

回应学员，后来实在没办法了躲去厕所待了半个多钟跟营里

同学互动闷出一身汗。

令我奔溃的是，他们继而转第二场“烧烤局”喝酒，直到半夜 三点才睡下。

合算一起，总共浪费了八个小时，这八个小时的缺失毁了我 接下来很多事情安排。

周五晚上的星球日更输出，停断一天 周五当天的健身习惯，停断一天 与好友约好的周六上午去滑雪，不能拒绝；早安打卡七点不 能怠慢；导致睡眠时间只有四个小时 因不能及时和小伙伴们更多互动，失去介绍自己链接同学的 机会，第一天打卡分数奇低。 导致今天精力不够，精神状态不好，无法运动 下午抽空补觉一个半小时后八点开始开营迎接助教老师，嘉 宾和主讲老师，一一查找班级里二十位同学的打卡记录和表 现，点评后再一一私信，反馈整理情况。

等到大家都说晚安的时候，我抬起头来发现将近十二点了， 我在书桌上飞快按着手机四个小时，未曾停下来跟室友说一 句话。

我意识到，时间的安排有连锁反应，像诺骨牌一样，一块倒 下满盘皆输。时间管理道路上忽然杀出一个程咬金，傻得我 措手不及狼狈不堪。就像在雪地里上坡，跌了一个踉跄。

但是有一点我是坚定且自信的：这是我主动找的“极限挑 战”。当成为管理员时，我不仅仅做好自己就可以，还需要密 切关注团队每一个成员的情况，及时询问提供帮助。作为一 名班长兼学员，要花出数倍时间精力，还有耐心和方法。

如果我没有参与这个角色，我永远不会有这种“时间窘迫 感”，也不会触及自己的“时间管理极限”，更不会有今天的感 悟。

换另一个思考角度，昨晚与我一起吃饭喝酒的所有人，他们 无一例外都很“尽兴”，因为他们本来就没有任何安排，更何 况是周末前的“周五晚上”，其中一个醉的最早的人指着我说 的一句话，令人难以忘怀，他说：今朝有酒今朝醉。

这不是一句随口而出的时代名言，这是他的一种生活态度。 他们不在意时间的流逝，他们只在意每周一上班喊累，等着 周五到来开心迎接周末去喝酒。

我们都知道，进步已经难得，可能是比其他人自律，可能是 比其他人有计划，比其他人有追求等等。

但当你已经走在进步的队伍当中，想要实现跃迁的时候，就 要尝试加大强度，挑战自己的极限。

就在上个月，我主动报名作为训练营班长，同时也对自己的 时间管理发出挑战。 就在昨晚，时间管理挑战失败，然而却得到了更大的进步。

接下来的 22 天，期待跃迁，期待自己时间管理的成功。 今天，你跃迁了吗？试试挑战自己的极限吧。
