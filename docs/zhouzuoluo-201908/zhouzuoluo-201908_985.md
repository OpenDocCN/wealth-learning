# 《粥左罗：不要只看

《粥左罗：不要只看一个人说什么，还要看一个人做什么》

看雷晓宇写李安，对比了侯孝贤。 侯孝贤长了一张刀削斧刻的脸，李安张了一幅菩萨相，有时

候还卖萌。但是拍起电影来，李安比侯孝贤狠的多，侯孝贤

至少让《刺客聂隐娘》的隐娘活下来，说是“一个人，没有同 类”，却还给她一个魔镜少年，李安多狠，不给玉娇龙路，让 她纵身一跃，那个孤绝才是真的“一个人，没有同类”。

不怕怒目金刚，就怕眯眼菩萨，李安拍电影可怕，一个《色 戒》让你看到绝望，一切都虚无，爱情荒谬，友情虚伪，亲 情荒芜，国家分裂，革命都不纯....

所以一个人，我们不要看他说什么，要紧的是看他做什么。 你比如说，职场里，有的人老实巴交，突然有一天你触犯了

他的利益，你觉得他无比可怕，他必须要弄你。有的人看起

来张牙舞爪，别人欺负他，也就笑呵呵过去了，还装作潇洒

无谓。

你再比如说，有些老板，天天让你觉得待你如兄弟，恨不得 要把公司分给你了，可是过年了，年终奖都他么没有。有的 老板平时一脸严肃，不苟言笑，你天天加班，也没见他关心 你、夸你，你都以为这个人真不仗义，可以一发奖金时，你 看到短信上的数字，才知道，他妈的说再多有屁用，这才是 好老板。

职场里，生活中，社会上，我们跟各种各样的人打交道，知 人知面不知心，但一个人过去做过什么，现在在做什么，曾 经待人怎样，做事什么风格，这些都是实实在在的，骗不了 人的。

所以，多看一个人做什么，少听一个人讲什么，听的听听就 行。

看帖如有收获，点赞是种态度——
