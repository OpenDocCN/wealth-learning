# 《中国合伙人必备技

《中国合伙人必备技能：关系能力》

橙橙：第 160 篇原创 时间：2019 05 17

——————————————————

正文：

还记得电影《中国合伙人》王阳结婚的时候给观众的三句忠 告：

“不要跟丈母娘打麻将” “不要跟想法比自己多的女人上床” “不要跟最好的朋友合伙开公司”

王阳、成东青、孟晓骏他们三个从上学开始就是最好的同学 朋友，机缘巧合合伙创办了新梦想英语培训，磕磕碰碰走过 了 20 几年，中间因为公司的发展方向意见不和，闹得鸡飞狗 跳，甚至拆伙。

是什么导致最好的朋友成为合伙人之后既反面成仇呢？值得 深思！

1.  01

    关系的 4 个阶段

    昨天看了梁宁关于《一个人最关键的 3 个底层能力》第三部 分“关系能力”感触特别深刻。

    夫妻之间，合伙人之间，包括朋友同事之间等，我们每天都 需要处理各种关系，怎样才算是好关系，在此之前一直困扰 着我。

    梁宁说：把一个组织运营成共同体，让同伴增强自己，才算 是一种优质的关系。

    她还提出的关系的四个阶段，分别是理想阶段、冲突阶段、 协同创办阶段、整合阶段，这个模型适用于与各种不同的关 系模式。

    昨天下午，一起打拼的几个合伙人在群里商讨关于要不要购 置一大批统一 LOGO 的导师班班服，但这个讨论并没有那么 顺利，交谈中总是夹杂着一些让人难受憋气的味道，每次提 出一些反对意见的时候都感觉特别艰难，然后就僵在那没有 一个最终的解决方案。

    好像聊着聊着所有的注意力就转移到人性上面和性格问题 了，一旦触动到对方的痛点，那就是一发不可收拾的冷战 期，这就是可怕的女合伙。

    关系止步于冲突阶段，如果没有一个正确的方式去突破，那 么始终是个隐患，在往后的日子里终会爆发的。

2.  02

    关系的演变与突破

    我们都希望合伙人之间是最理想阶段的关系，但那是假象， 因为这个理想的关系还没有进入到真正的关系中。

    ★梁宁说：“真正的关系是从冲突开始的”。

    但合伙人之间因为观念或想法不一致发生冲突的时候，会出 现指责或者内疚两种状态，如果相互之间没有处理好冲突就 会进入下以下三种状态：冷漠、超越、分离。

    电影《中国合伙人》中孟晓骏想要上市却遭到拥有一票否决 权的大股东成东青的极力反对，让这三个合伙人之间矛盾不 断升级，互相暴露出各自暗黑的过去，闹得不欢而散。

    还有一种状态是真诚相对，大家都静下来好好处理冲突。这 个时候就需要合伙人具备关系处理的能力了。

    梁宁给我们提出了很好的方法论：替换，即把合伙人相处过 程中出现的“问题”这个词，全部替换成差距。

    合伙人之间，因为不同的背景和不同的经历，随着不同的阶 段，思维也会有所差异，但不影响一起为了同一个梦想而努 力奋斗，前提是各取所长，还要允许差异的存在。

    我认为，团队之间的协同合作应该是开放型的讨论，不必害 怕自己犯错误，更不必担心自己的短板遭到团队人的嫌弃， 大家一起想办法去弥补差距与预期之间的距离，把差距缩小 一点。

    不管是不是好朋友合伙开公司，都需要处理好合伙人之间的 关系，让关系进入到一个协同合作的阶段。

    03

    坦诚相对，且行且珍惜

    除以之外，彼此打开心扉，坦诚相对也是合伙人之间非常关

    键的相处方式。当真正开始了解和理解对方的时候就是关系 升级的时候。

    团队注定不是一个人的想法，要顾全大局为他人着想，还要 透彻心扉。

    孟晓骏狠心拒绝成东青大方送出的别墅，却感动于成东青以 他的名义捐赠命名的实验室，这就是相互之间的的真心相 对，因为这一次，成东青终于懂得孟晓骏真正想要的是什么 了。

    电视剧是完美的结局，可现实是残酷的，很多合伙人之间说 散了就真的散了，可能永远都没法做回朋友了，合伙人之间 具备关系能力，可以让团队合作的路走得更加长远，让我们 且行且珍惜。
