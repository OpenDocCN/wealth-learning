# 如何优质地，大量地

如何优质地，大量地输入？

首先，何为优质的输入？

（1）：你当下的目标是什么？ 比如：

你要通过写作投稿变现，

那你要投什么样的稿类？

投干货类文稿。

那什么样的干货？ 时间管理小技巧。

干货类文章结构应该怎么写？ 那你需要看哪些书？ 读哪些文章？

用一个个问题， 把目标拆解， 清晰。

具有强目标性的输入， 就是优质输入。

（2）：你当下遇到的问题是什么？

你发现自己工作效率低下？ 你发现自己学习能力不行？ 你发现自己专业能力有待提高？

这些都是你遭遇的困境。 怎么办？

你通过搜索引擎， 你通过请教前辈， 你通过查阅书籍， 你通过知识付费。

为了寻求破局之道， 无论任何方式， 无论任何手段， 地去输入。

这就是优质输入。

最后，什么是大量的输入？ 大量并不是指数量，

而是时间比，

质量比。

你的可支配时间中，

80%花在输入。

在你的总输入质量中，

80%为优质输入。

满足这两个条件， 则为大量输入。

最后最后再掏心窝子说一点： 有输入才有输出。

输入质量决定输出质量。

输出的质量决定时间的质量。 时间的质量，决定你人生的质量。
