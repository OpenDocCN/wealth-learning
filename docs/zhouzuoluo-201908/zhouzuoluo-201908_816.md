# 《过年回家，第一次

《过年回家，第一次相亲经历》

年底了，很多在外漂泊的人也都陆陆续续返回家乡了，除了 一家人都能高高兴兴的团聚在一块，对于我们这样的大龄青 年，又都到了适婚的年龄，那也必定面临家里七大姑八大姨 紧锣密鼓式的“逼婚”。

虽然无奈，但不得不承认，这现实赤裸裸的，除了舍身成仁 又能如何呢？

昨天，我也终于迎来了第一次相亲，尽管是提早有所准备， 但还是有些不知所措。

早上还未起床，就被老爸从床上扯了起来，催促我赶紧洗 漱，我就纳闷儿啥事儿这么着急，还把他给高兴成这样。

老爸说托人给我介绍了一姑娘，那边有信儿了，今天就可以 过去看看，都已经谈好了，时间是中午 12 点。

我心想，这也太突然了吧，我这刚到家屁股还没坐热呢，起 码也修整一下，不然提前打个招呼，也好有点心理准备啊， 但既然这事儿双方都已经谈好了的，不去貌似不合乎礼仪， 面子上过不去啊！

于是我问老爸：“那个女娃儿长个啥样子撒？有么有照片？拿 来咱看一哈！”

没想到老爸说：“咋来的照片呢？咱过去看一哈不就知道了。

”

“那，女娃娃长个啥样？哪咋的人？” “啥都不知道撒……咱们今天全家出动，你还怕个撒……”，

老爸如是回答到。

听到这里，我简直内心崩溃掉了，我要去相亲的对象我都不 知道长什么样，家住何方，工作，背景等一概不知啊。

天呢，这就好比是要我写一篇自拟题目的作文，文体不限， 真有点儿为难啊！

匆匆忙忙的洗漱完毕，简单的吃一点东西，我们一家三口， 老爸老妈还有我，开始出发前往指定的地点。

后来隐约了解到，相亲对象好像和我们有点半搭不搭的亲戚 关系，今天我们去的是我的表叔家里，女孩家里算是表叔的 选房亲戚吧。

还好，这样就不至于太过于陌生，显得尴尬了，我一边安慰 自个儿，一边想，好歹咱也算是在江湖上闯荡过的人，神马 事儿没遇到过啊，就当去拜访客户了。

在车上老爸就就交待我，一定要有礼貌，见了长辈要问候， 男的无论是谁一定要递上一根儿烟，然后别忘了待会儿买瓶 酒，礼品必不可少。

在农村尤其是逢年过节，不管是去哪，一定要备上礼物，然 后见面问候最好能递根儿烟，这算是常识，别人会觉得非常 受用，即使陌生人，对你的第一映像也会非常不错的。

表叔家是一座四合院，阳光照进去暖洋洋的，我们刚进门就 看到一群人在院子里有说有笑，惬意地晒着太阳。

看到我们进来所有人都起身而来，对于我这个多年不怎么回 家的人来说，很多人都是不认识的，老妈在一旁一边介绍， 我一边问候着，这个是叔，那个是婶婶，这个该叫姥爷，我 一一向长辈们问过好之后，大家便又坐下继续侃侃而谈。

这其中，我注意到一个女孩，年龄大概和我差不多，只是不 确定是否她就是这次的我要相亲的人，咋个也没个人给我说 撒，我也是纳闷儿……

不过我环顾一圈，貌似除了她再没其他人了！

女孩儿一身白色羽绒服，脚底踩着可爱的高跟鞋，看着还算 精神，长发披肩，皮肤很白，显然是经过了细致的打扮，虽 然感觉有点淡淡的高冷，但长相耐看，瞧着倒也舒服。

这个时候，旁边和一位婶婶聊天的老妈说： “这个是今天过来相亲的姑娘，这位呢是女孩的母亲。” 我上前到女孩前问了声好，估计是没听到，女孩没有做声，

继续和一旁的小朋友打闹着。

我又问了女孩的母亲：“婶婶好”，女孩母亲对我笑了笑，没 再说什么。

虽然气氛有点尴尬，我倒是不在乎，只是觉得，就算女方看 不上我吧，但也不至于不搭理人吧。

想想这种事情，我要是不主动一点，估计就要冷场了。 我上前主动问了声好，估计是没听到吧，女孩没有做声，不

过貌似也没正眼看我。

于是我又鼓起勇气上前，这才和女孩搭了话，我们进了隔壁 的房间，安静，也适合我们年轻人聊些私人的话题。

不过呢，还是我主动挑起话题，先吐槽吐槽相亲这件事情， 缓解下气氛，不过发现人家没啥反应，又聊了聊工作，还有 学校之类的，不过基本都是我在问她在答，中间正眼瞧了我 两次，对我微笑了一次……

这中间，虽然不知道女方是如何想的，但是女孩对于我不冷 不热的态度，以及我极力营造气氛的尴尬，还是觉得人家根 本无心恋战。

要了女孩微信号后，我提议出去坐会，晒晒太阳，然后我们 各自很默契的晒太阳，女孩时不时拿出手机玩玩。

过了一会，女孩也不知道跑哪去玩了，女孩的父母也找不到 人了，既然人家无意，我们何须竭力讨好，只得打道回府， 本来想打个招呼再走，但对方已经不见踪影。

尽管我不想有这种女孩都很物质的想法，但现实是，十八线 的小地方，没房没车没存款，或许人家真的不想正眼看你一 眼，哪怕是最起码的尊重！

总结一下，相亲时候，姿态别放低，正常就好，不要刻意讨 好，会很尴尬的。

第一眼缘很重要，觉得可以就深聊，要不然也是浪费时间， 更别期待能交到什么朋友了。

作为男方，一定要主动，脸皮要厚，千别期望女方会主动一 点。

最后一条，能自己谈，千万别相亲，每个女孩都渴望能遇到 白马王子，不过现实是，你喜欢的人家看不上你，喜欢你 的，你看不上人家，既然如此，要么你往死了爱我，要么你 能养的起我！
