# 《凡事都要做最坏的

《凡事都要做最坏的打算——是成年人应该具备的品质》

1

最近一个刚结婚同事跟我说，最近烦死了，可能得了婚后抑 郁症。她说，结婚和我想象不一样，我老公也没有用早餐把 我叫醒，而且回到家中衣服乱扔，各种小细节我都看不惯。 你说怎么办呀？

我说这些都是正常的事情，你结婚之前不知道吗？她说我觉 得结婚后会很美好，我压根就没想这些，不然我也不能结 婚。我说你只想婚后美好的事情，难道没有做最坏的打算 吗？她说什么？我说虽然你们现在很相爱，但是你没有想过 某一天你们没有爱了，离婚了你能不能接受？她直接愣了， 过了三秒钟，喝了口水，说我从来没想过会有那么一天。

很多人结婚都是冲着一辈子去的，婚前看到的都是对方的优 点，结婚后，由于各种原因男人、女人都会有多多少少的变 化，女人会觉得男人变懒了，男人会觉得女人不修边幅了等 等。

其实这些都是很微小的事情，但是婚前你要做最坏的打算， 就算是有一天离婚了，我也可以接受，因为婚姻有很多不确 定性，我们可以肯定的是当下是爱的，未来我们只能说期 望，万一，不爱了，我也可以接受，这样的你会坦然很对， 会看开很多，也会为了婚姻去做最好的自己。

2

前几天回老家，爸爸说我看到了一个创业项目挺好的，我说 啥？爸爸说开网店，卖花生油，我看人家谁谁卖的特别好， 我说你只看到人家的好，刚开始不好的时候你看见了吗？他 说大不了就是不赚钱，我说不会赔钱吗？压货卖不出去之类 的？爸爸说，不会，我们少进货，卖不了可以退，风险很低 的，你完全不用担心。爸爸之所以说让我别担心，是因为爸 爸做过很多生意，每次都是热情满满跟我们说肯定能赚钱， 但是结果往往不尽人意。

在我小的时候爸爸卖西瓜，从农户地里拉回来之后去市场上 卖，有一次到了晚上 10:30 爸爸还没回来，电话还打不通，我 和妈妈在家就着急了，妈妈就去大门口等，我就在家接着打 电话，妈妈一会进来问我打通没，一会自言自语的说，不会 开车遇到什么事了吧？吓得我也是提心吊胆。到了 11:30 我在 家里就听到了爸爸的车声。

我开心的出去看爸爸，妈妈问爸爸怎么连个电话也不打，不 知道我们在家着急，爸爸说我去了很远的地方卖，路上听不 见电话，妈妈说卖完了？爸爸说，你看不见？还剩一大半 呢？这时候爸爸和妈妈吵起来了，妈妈问爸爸你没想过你要 这么多货卖不完怎么办？这么热的天，放一晚上我们也不敢 放呀，爸爸说我没想到，要不然我能进这么多，他们俩越吵 越凶，我根本不敢说话，从那以后我就知道了，做什么事都 要想最坏的结果，这样才不至于接受不了，引起不必要的麻 烦。

3

马上就要高考了，现在孩子们都在紧张的备战高考，期望考

出好成绩，考上如愿的大学。

面对压力，有些孩子出现了生理的反用，胃痛，没精神。周 末回家见了表妹我问她，你怎么样？她说，没压力是假的， 但是我知道就算我考的不好，我也能接受，我现在只要尽力 就行，最重要的是不后悔么，我看开了。

愿我们在生活中无论遇见什么事情都能做最坏的打算，接受 不好的结果。 凡事都要做最坏的打算——是成年人应该具备的品质。
