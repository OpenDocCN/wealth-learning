# ♥猴哥回归第一天《

悟空 : ♥猴哥回归第一天

《逃离体制内？NO！我只想优雅的离开》 大家好，今天想来聊聊关于逃离体制内的一点想法。 我，体制内老赖了，这么多年工作浑浑噩噩，心里总想着有

一天可以逃离体制内，但是依然赖着不走。为什么？

因为那些逃离体制内、创业发达的仅是凤毛麟角。那些人的 经历是很励志！但不具可复制性，或者说，不具有普适性。 我早已过了喝碗鸡汤说走就走的年纪，拖儿带女一家老小也 不允许我头脑发热。

但是对于成功，我们要虔诚，即使几率很小。仔细思考后， 我发现，那些成功的人之所以能够成功，原因或许就是高效 的自产出能力，说白了，我创业不是依托平台、靠别人赏饭 吃；而是就算没有平台，我也能带着娃、遛着狗，同时还能 把钱给赚了。

一个逃离体制内的朋友，这么和我说:在民企当高管，每天都 是失望。环境、人员、社交，都不是你想的那样。在体制内 待久了，素质不说高尚吧，起码不低俗，但是体制外就不一 样了，什么人都有，你上班得穿软猬甲，否则天天被扎。这 个朋友就是我说的，只能依托平台，靠别人赏饭吃，看似位 及高管，可并不舒坦。

说回我自己。我有高效的自产出能力么？不过是半桶水晃罢 了。能做和做得好、做得无可替代是两回事。

这两年我涉猎了很多行业，尝试了很多业态，事情做得越 多，越有一种感觉:答应一时爽，完成愁断肠。在没有把握的 时候给自己加戏，只能咬紧牙关挺过去！如何挺？不睡呗！

粥左罗说:越是看不见的地方，越需要信仰。是的，我熬的不 是夜，而是意志！熬过去了，就是成长，熬不过去就是坎 坷，好在，我很幸运，一直在成长。

那些事，在别人看来我都做的极好，但我心虚。依我以前的 经验，有活就接，硬着头皮完成，下一次就顺畅了。可，从 今年开始，我有了新的感受:有些事情无论完成多少次，你仍 然得硬着头皮。

五月天阿信说:当你在完成一件很难的事时，不是痛苦而是享 受，那这便是你的天命所在。说得真好，特别贴切！这种感 觉我也曾经有过。

事隔一年，感谢训练营，让我再次拿起笔。当我把素材准备 好，拎逻辑线时，这种不是痛苦而是享受的感觉又来了。

现在，我坚信，文字输出就是我的天命。 那么问题来了，我离高效产出还有多远？应该不近。我一定

还有很长的路要走，也许还会误入歧途，这时候谁来为我托

底？体制！

体制内和体制外最大的区别，在于时间的红利。体制外，你 的时间是老板的，体制内，你的时间属于自己。

小时候，我妈每每给我读《基督山伯爵》，我都惊叹于爱德 蒙的传奇经历。现在想来，时间才是关键。

你无法想象，时间会怎样塑造一个人。

当年明月在广东顺德海关工作时，完成《明朝那些事儿》 刘慈欣在山西娘子关电厂任职时，完成《三体》 爱因斯坦在瑞士伯尔尼专利局供职时，提出相对论。 少林寺的扫地僧，那是不想出世，若是出世，试问江湖豪杰

谁是对手？

以前，我总和刚毕业的大学生说:如果你还有梦想，那就别考 公务员。

现在，他要再问我，我会说:如果你想精进一项新技能，来体 制内吧。

在体制内想着体制外，在有人托底的时候不荒废时间成长自 己，成长好了，再优雅的 say bye，这才是那些毒鸡汤没有告 诉你的事。
