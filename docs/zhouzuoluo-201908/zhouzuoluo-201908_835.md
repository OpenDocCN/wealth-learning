# 《承认别人的优秀，

《承认别人的优秀，真的没那么难》

有那么一种人，总是见不得别人好。见不得别人学校好，见 不得别人工作好，见不得别人穿名牌买包包，见不得别人各 地旅游吃吃喝喝。明明别人比他优秀很多，却能给别人的优 秀找出各种阴谋论，然后加以否定。

看到别人朋友圈跑步打卡，他们会说是炫耀；看到别人学英 语，他们会说是崇洋媚外；看到别人坚持写作，他们会说是 自不量力。总之，就是不能承认别人比自己优秀。其实这是 一种病，名叫嫉妒。

最近，有一朋友升职了。在他这个年纪能达到这样的级别， 不说没有，但也是凤毛麟角。作为朋友，我特别为他高兴。 但同时我也听到了许多不同的声音，那怕不是当事人，我都 十分气愤。

很多人认为这么年轻就当上了领导，肯定是有黑幕。有的说 他岳父是高官、有的说他亲爹是商人给他花钱买官，甚至有 的说某大领导是他舅。听到这些说法，我真的觉得可笑。

我和他相识多年，对他的情况也很了解。他和妻子都是农村 家庭，根本没什么高官亲戚。他能走到今天，真的是靠自己 的奋斗换来的。

最忙的时候几个月没和孩子打过照面，每天到家的时候孩子 已经睡了，走的时候孩子还没醒。为了工作饮食也十分不规

律，年纪轻轻就有严重的胃病。一头白发根本不像三十几 岁。

但当他达到今天的位置后，很多人都忽略了他的努力。他们 觉得自己不能这么年轻就达到这样的高度，所以当别人可以 的时候他们本能的认为这不可能，肯定有黑幕。说白了，就 是不肯承认别人的优秀。

而且在当今社会，这并不是个别现象，而是很多人都存在这 种畸形心理，嫉妒心太强，总觉得别人的优秀都是有缺陷 的，是伪优秀，只有自己最优秀。但这种自欺欺人的想法对 自身的成长没有任何益处，反而会让自己停滞不前。

必须承认，好多身边人都比我们优秀。我们没必要总是怀着 恶意质疑别人的优秀，而是要承认别人的优秀，然后把优秀 的人当成自己的榜样，按照榜样的标准努力提升自己各方面 的技能，这才是我们应该有的心态。

一个人如果嫉妒心强永远不肯承认别人的优秀，那估计他也 不会有大的作为。那真不应该是我们的人生追求。
