# 《高小羊：成就高的

《高小羊：成就高的那批人，全都有同一种能力》

熟悉我的人都知道，我对时间的管理和规划很严格，每天都 会抽出时间来规划当天的安排。市面上有很多教别人管理时 间的书籍和 APP，所以很多人买书去学。针对这一现象，我 有不同的看法。

我认为：别在学时间管理了，你该学的是管理你的人生。 因为真正忙碌的人，对自己人生有规划的人，都是自发的去

管理并合理分配他的时间的。他的时间太宝贵了，他从来不

会闲到去买一堆书回来，学习怎么规划时间。他们都是先有

人生规划，然后自然而然就有了时间规划。他的人生规划迫

使他不得不详细分配他的每一分钟，他的时间规划是用来辅

助他完成人生规划的。

那市场上为什么会有时间管理课程，时间管理软件，时间管 理书籍等这些东西？因为这些课程，书，软件，就是给那些 很闲，手里有大把时间空出来的人设计的。

为什么你会很闲？而且闲到有时间去学习时间管理？因为你 看到别人的规划表后，迫切希望通过学习一些时间管理技 巧，来管理自己每天除了工作外多出来的刷网页，追剧的时 间。

但是我保证你就算去学了时间管理，你还是跟以前，改变不 了什么。因为你没有具体的目标支撑时间表，你没有具体要

去执行的事，因为你对你的人生没有规划。就好比，你要好 好规划下班后的 5 小时，但问题是你要用这 5 小时来干什么？ 干什么就是说你的人生有一个目标。

所以不要在学习时间管理了，你应该要该学习的是人生管 理。
