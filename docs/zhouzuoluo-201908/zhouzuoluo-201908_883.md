# 《打败成长过程中消

《打败成长过程中消极情绪的诀窍》

晚上好[玫瑰]今天是橙橙加入星球的 161 天，和大家分享关于

「消极情绪」的话题。

1.  01

    什么是消极情绪

    在每天奔波的工作中和看似平淡的日常生活中，别人的一句 话或者自己犯下的一个小小的错误都会让你情绪变得低落， 甚至有一些消极的想法，消极通常是一个无意识行为，而且 来无踪去无影，就算很小很小的事情可能也会让我们突然之 间变得消极。

    比如你很用心做的方案，同事或者领导看过之后觉得不咋地 也不认可，你也会消极的怀疑自己放弃自己。

    比如你费尽心思做了一桌子菜给爱人享用，不但没有得到好 评还各种挑剔，你一定想发誓再也不进厨房了。

    ★当期待实现值比内心期待值小的时候，由于内心欲望未得 到满足，人的情绪就会出现失望、压抑等消极情绪体验。

2.  02

消极的情绪有什么危害？

1，消极的情绪会影响我们对生活的满意度

2，消极的情绪也会影响到身边的人

3，消极的情绪会影响我们的能力的正常发挥。

4，消极的情绪往往会使我们轻易的放弃一直坚持的信念。

03 既然消极的情绪有这么多的危害性，我们应该及时打败它， 那什么才是治疗消极情绪的的秘诀呢？

★八个字：清醒认知，积极行动。 昨天在瑜伽群里分享自我练习的感受，师父说：我的呼吸太

刻意了，我应该好好倾听肚子的宝宝需要练什么，而不是我

脑子要练什么？师父的几句话在我脑子里转了好半天，立马

我的情绪就很低落，不仅仅是因为师父的教诲，更多的是我

发现自己的努力练习走向了错误的方向，我明明知道孕妇呼

吸本来就有点困难，宝宝也需要更多的氧气供着，但练习的

时候却没有很好的训练呼吸，反而偏于体式的练习。

整个人昨晚就变得有点消极甚至有想要放弃孕期练习的念 头，后来转念一想，不能这么轻易的放弃了，虽然很难，需 要坚持探索，但我和肚子里的宝宝都需要一定的练习让我们 更好的成长，今天早上，把瑜伽垫搬到阳台，虽然有点冷但 空气好啊，静坐冥想继续训练自己的呼吸，想办法让呼均匀 顺畅还要柔和自然。

▼为此，我自己总结了几个打败消极情绪的步骤：

① 调整心态，清醒的认识到导致消极情绪的原因所在。

我之所以会因为师父的几句提醒而变得情绪低落是因为我很 在意师父对我的评价，我也很珍惜我自己努力的成果，但我 却不愿意接受自己走错了方向，所以我应该学会清醒的认识 到这些原因，然后调整好心态从新探索练习的方向。

② 从新规划自己的学习计划，让消极的情绪扼杀在摇篮里。 不要等着别人告诉你怎么做，你要主动安排自己的学习计 划，不断提升。一时的练习困境多半出于陌生，但坚持专业 性的系统学习和探索能让你用更全面的视角看待孕期的练 习，所以今天早上我换了个练习的场地和练习的方式我就专 注练习呼吸，不着急体式练习。当我与新的练习计划我便不 会再这种消极情绪中不能自拔。

③ 走出去，多和过来人沟通交流，用积极的行动打败消极 情绪。 不要觉得不好意气去请教别人，艾扬格大师说过：我们不仅 仅是瑜伽老师也是一名不断成长的学生，要放低姿态多和有 类似经验的人沟通交流，自我封闭只能让自己陷入消极状 态。人际的新鲜感往往会激发新的活力。“两耳不闻窗外 事”的做派早已不符合这个时代。每天分享完练习感受可以主 动和群里的师兄师姐沟通孕期练习的问题，用这种主动积极 的行动来抹灭消极的情绪吧。

④ 依赖是个恶魔，你需要自己掌控自己的情绪和成长之路。 大多数人都会把希望寄托在别人身上，包括我在内，我的练 习总是依赖着师父的指引，那么就容易对自己的练习失去掌 控，这种依靠带来的短时安全感，会让我们变得厌倦成长， 厌烦历练，懒于提升自己，不敢突破自己，所以，及早放弃 任何形式的依靠，才能让自己的根基越扎越深，摆脱消极情 绪的干扰。

总结一下： 调整好自己的情绪一直是人类永恒的课题，面对不同的原因 引起的消极情绪，其实都有一个突破口，那就是自己跟自己 较劲，其实就像我们转移小孩子的注意力一样，我们同样需 要这种转移自我消极情绪的能力，让我们在生活和工作中以 更多积极的情绪去面对所有的困难。
