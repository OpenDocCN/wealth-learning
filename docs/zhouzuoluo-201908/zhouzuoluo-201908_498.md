# 《敢比会更重要》我

《敢比会更重要》

我以前是那种不喜欢分享，不喜欢主动去表现的一个人，自 从写作还有销售工作的原因，我发现自己慢慢改变了。

今天在面试过程当中，面试官给我们看公司的企业文化，看 完之后就问大家谁想分享，我马上举手分享了自己的看法。

这要是换成以前的我，压根就不会去分享，我不会展示自 己，就默默的当一个观众而已。

张萌萌姐有一句名言，敢比会更重要，我一直牢牢的把这句 话记在心中，今天也付出了行动，这一次小小的举动，也可 以让我比以前更加自信一些，更加认可自己。

生活和学习当中，其实有很多机会。我们都可以去尝试，去 把握。不要一直在台下做一个观众，当有一天你去做主角的 时候，会发现原来自己也可以这么优秀。

我们不要给自己设限，不要觉得自己不行。很多事情不是天 生就会的，你今天做不到不等于以后你也做不到。

勇敢的去尝试，去行动，真的会让自己收获很多。不但可以 获得认同感，还会更加自信。

很多人问过我，你是如何从自卑走向自信的？我就是去做了 一件件曾经不敢做得小事情，如今我勇敢的踏出了这一步，

我做到了。

因为我内心真正的认可了自己，所以我不再自卑，我慢慢的 越来越自信。

希望大家多去尝试那些你曾经不敢做得事情，当你真的去行 动了会发现那个更优秀的自己。
