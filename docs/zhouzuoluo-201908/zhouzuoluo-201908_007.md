# 熟门熟路方能助己写

章子 : 熟门熟路方能助己写作顺利

今天读了粥老师加法推荐的两篇文章。其中，《在赢之前， 你要先学会输几次》的作者打赏，就有 57 个人。雕爷写的，

《真别怀疑了，新消费滔天巨浪来了》10 万+。

两篇文章，这么多人爱看，是因为写的内容，都与大众的生 活，品牌熟悉度相关；与市场竞争内情相关；与商业世界最 终结局有关。读这样的文章既解人痛痒，撩人探秘，又指人 遥望，确实是好文章。

第 2 篇文章的一个引人点，在网名雕爷。看到评论区一致的好 评，产生了好奇，去百度了一下。

雕爷—孟醒，连续创业者，互联网知名人士，商学院讲师， 新锐作，阿芙精油和雕爷牛腩，三件空气净化，薛蟠烤串等 品牌的创始人，厉害了我的哥。

难怪，雕爷讲起市场风云变幻，如数家珍呢，原来他跨界， 就如走地平线那么容易啊，由此，我有了个深刻的醒悟：

熟悉才能触及本质，底气才会讲明事理 俗话说，隔行如隔山，你不熟悉内里，怎能讲出真理，你不

摸清行情，怎能定调真价？他们的文章讲的是消费市场，在

消费的大潮里，我们是水滴，而二位作者是弄潮儿，他们怎

样避风浪，怎样借潮力，只有他们勇闯大浪的人，亲身尝试

和实践过，才有真切感受，才能说服大众。

他们闯浪成功的底气，成就他们的气场，他们熟门熟路的自 信雄壮他们的风格。这一点告诉我们：想成为一个优秀的写 作者，光学写作技巧是远远不够的。而选择线路，熟悉路 况，才是我们首要突破的课题。

仔细的阅读之后，压榨出了 3 点：

1，让自己稀缺 决定一个人的价值，不是勤奋，而是稀缺性。为啥你的单

位，舍不得让你走，因为你的能力有稀缺性；为啥雕爷一年

没写文章，那么多人还保留他的公众号，因为雕爷摆事论理

的文章太稀缺。

为啥那些有需求痛点，有广阔市场的产品，被淹没在洪流 里？就像被挤在竞跑的人群中，而跑也跑不出来？就是稀缺 性被孙悟空一根毫毛变出的同类，稀释了。

怎样能保持稀缺性呢？总结一句话就是，加速快跑，不断变 化。

2，保证有守得住的秘密。 你有一招，如果很浅显，门槛很低，很快会被别人效仿。因

为你的这一招，没有保密性，让人一复制就易主了。以前美

国仗着 4G 手机，霸道全世界，就因为安卓的秘密上着保险，

但是现在被华为解密了，鸿蒙取代安卓，美国慌了。

所以守不住秘密就面临着淘汰。 最好的保守秘密是独有的，主观气质和超前认知，因为这个

东西别人即使看明白了，也很难模仿。比如，一个叫张辉的 人，从来不向老板提加薪的需求。他却一路升职加薪，现在 工资涨了 50 倍。

他与别人不同的是，习惯用心观察发现流程和工作中的缺 陷，主动向老板建议可行的，自己设计的新方案，并把自己 看到的更远的未来前景，讲给老板听。这就是他独有的超前 认知和主观气质。别人学不去的秘密。

3，大而强的资本、资源。 前一段，看了个笑话，王健林给儿子，首次创业的启动资金 5

个亿，儿子赚了 40 亿，翻 8 倍。而我爸给我两元钱，我买了一

双手套，去工地搬砖，一天赚了 180 元，翻了 90 倍，事实证

明，我们能力是有的，只是启动的钱太少。

这虽然是个笑话，但却告诉我们，大而强的资本、资源占有 天生的优势。

无论做任何事情，想取得成功。必须熟悉行业。学会技巧。 保持自己的稀缺性和独有的主观气质、超前认知。当然写作 也不例外。
