# 《高小羊：你曾经达

《高小羊：你曾经达到的高度，日后也能达到》

我过去 5 年管理过的团队中，我发现很多员工都存在一种普遍 现象，他们想法悲观，做事态度悲观。

对于未发生的事，他说，说不定明天就发生了，永远战战兢 兢，杞人忧天。

对于已发生的事，他感觉好像天都要塌下来了，似乎无法去 补救这件事，恐惧担忧笼罩着他。

而最不可思议的是，一个原本一开始想法乐观，做事乐观的 同事，一段时间后，也变得很悲观。经过了解后，我发现原 来他近期和一位消极的同事走得很近。

人总会过度关注那些消极悲观的信息，而经常忽略掉积极乐 观的信息，因为人总擅长保护自己，关注负面信息可使得自 己周围的环境是安全的，这种心理机制，在心理学上，被称 为为选择性注意。

这种机制一定程度上能保护自己的舒适区，但永远故步自 封，看不清事实，从而失去事物辨别能力和执行能力。

上个月和部门一同事讨论产品推广手段，分析市场后，我建 议她加大备货量，使库存充足后，然后再按照我说的方法一 步步做下去，可以将这款产品推起来。

她听到我给她的备货建议数量后，直摇头，一脸惊恐样：要 是卖不掉咋办？要是中途出问题咋办？要是突然过季咋办？ 然而最后呈现出的结果是，前面她所担心的问题一个都未发 生。

大概半个月前，同样是这位员工，她负责的另一款产品，因 为一客户个例投诉，产品被系统调查，大概进行了为期一周 的调查后，发现产品没问题，解封后可以正常销售。然而调 查期间，因为停售，产品排名直线下降。

所以要通过一系列措施提升排名，恢复之前的销量。她尝试 了几天后，发现排名还是没有太大变化，脑子里的负面情绪 又跑出来了，天天念叨说排名上不去了，销量也恢复不到以 前了。

第一件事，她担心的也不无道理，可以理解她做事小心谨 慎。但我所采取的措施是基于大量的数据分析和市场分析， 以及稳定的产品质量，这些担心就无意义。

如果你永远不想冲破圈层，只想待在自己设定好的安全区 内，不想承担风险，做着手头简单工作，不用思考，舒舒服 服，那你的业绩和职业永远得不到提升。

第二件事，在事情解决后，并且你所负责的产品证实没任何 问题时，你要坚信你是可以恢复之前的销量的，因为停售时 间并不长，你所要做的是正确的推广手段加上时间积累效 应，以及你所要相信的是你曾经能达到的高度，现在依然能 达到，所以你的销量肯定是可以恢复的。

墨菲定律的其中有一条内容就是，如果你担心某种情况发 生，那么它就更有可能发生。你觉得销量不会恢复，那它就 不会恢复，因为你已经从心理上给它判了死刑。

you are what you believe(你相信什么，你就能成为什么）。这 很重要，因为先相信后看见，你看见它会失败，那它就会失 败，你觉得你会出错，那你就会出错。相反，你觉得你会成 功，很大概率上你一定会成功。

所以我们在看待问题时，在行动上，要向好的方面去靠拢。

1，相信自己曾经达到的高度，日后你依然能达到。

2，优化你的交际圈，接触积极主动的人，近朱者赤，近墨者 黑，负面情绪最容易传播，它会通过一个微小的链接，传染 开来，影响你整个人生状态。

这一点我自认为做的还不错，我是一个随时都像打了鸡血的 人，不论是工作，还是生活，健身，我都能保持一个积极乐 观的心态，而我身边的朋友，对待生活工作很积极乐观，对 万事万物都充满兴趣，正因为如此，在这几方面，我取得不 了不错的成绩。

正如张辉所说：企业也罢，人生也好，都是理念的产物，我 们最终的结果，更多取决于我们相信什么。先相信，后看 见。我们要检视我们内心的理念，去培养它，打磨它，真正 相信它，在生活工作中践行它。
