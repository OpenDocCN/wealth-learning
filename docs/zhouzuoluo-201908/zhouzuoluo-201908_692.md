# Day108《打造

《打造个人品牌的三个锦囊》

精读江南春《抢占心智》第二天。 江南春说：做成一个企业，需要找到差异化定位，抓住时间

窗口，采取饱和攻击，从而占据消费者心智，引爆主流人

群。

我想，同样的，想要成功打造个人品牌，成为领域的行家， 也是需要找到差异化定位，为目标人群创造高价值，并获得 他们的强感知和高认可。

下面，从江南春的书里挖出三个锦囊，送给你。

1、打造差异化的三大运营原则 社群运营大 V 剽悍一只猫说：运营的核心原则是“信简喜”三个

字。

信是信任和信心，在定位理论中叫做“信任状”；简是简单高 效，围绕定位的传播心法是用简单易懂的一句话打入用户心 智；喜是惊喜，给别人提供大惊喜强感知，这本身就是差异 化价值的生动体现。

2、做人得体，做事靠谱

江南春说：人心比流量重要。

与其说占领心智，不如说赢得人心。赢得人心需要做到“做人 得体，做事靠谱”。

做人得体，就是言行让人觉得舒服；做事靠谱，就是办事让 人觉得踏实。整体来看，就是给人好感，让人信赖。

正所谓，你的一言一行，都代表了你的个人品牌。建立个人 品牌，成为行家，“做人得体，做事靠谱”非常非常非常重 要。

3、把注意力放在人身上 驱动一件事情落地出成果的，是人。人也是成事与否的最大

变量。

我们衡量一件事是否成功，也要看通过这件事，是不是让人 变得更好了。

同样的，2019 年在成为行家的路上，能做成多少事情，要看 我能真正帮到多少人，成就多少人。

于是，我选择了一个看起来低效的一对一业务模式——个人 学习私教。

优点在于高互动性，高粘性，能够建立深度关系，积累口碑 势能。缺点也很明显：占用个人时间和精力较多，增长的天 花板很明显。

其实，正是因为如此，这也成为了一种差异化。因为聪明人 看不上，普通人做不了。

在我看来，这种做法不是商业模式的最优解，却是价值观最 直接的体现。

我知道，我所书写的，成为行家之路上的故事，肯定不是我 一个人孤独前行的故事，而是一群人互相成就，结成成长共 同体的故事。
