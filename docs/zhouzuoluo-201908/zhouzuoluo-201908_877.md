# 《幸福简单一点，莫

《幸福简单一点，莫比较》

有些时候，我觉得人心太浮躁了。 打开朋友圈，看到的都是晒买车、买房、结婚、生小孩。晒

聚餐、美食、旅游。晒开店、晒公司。总之朋友圈里的人都

过得很好。

他们都很钱，工作不错生意很好，赚钱很多。有车有房有老 婆孩子，家庭幸福美满，朋友同事天天聚餐开心快活，旅游 美景美食，享受生活。

现实生活中真的是这样吗？他们当中很多人是上班族，天天 挤坐地铁公交上班，那种无奈又苦逼的生活，只有他们自己 知道。上班被领导骂被客户为难，同事之间人际关系的各种 烦心事。

做生意的遇到各种困难坎坷的，难道没有吗？夫妻生活亲子 关系，婆媳关系，家庭一地鸡毛的琐事烦心事，难道没有 吗？都有。

我们都是现实生活中的本身，参与者。生活中有酸甜苦辣， 喜怒哀乐。为什么朋友圈总是看到那么多美好的照片？因为 人啊都希望别人看到自己好的一面。朋友圈只能分享自己生 活中的喜事好事。所以，我都不会相信朋友圈他们真的活得 那么好。

不要被朋友圈那些经过筛选的生活照片所蒙蔽。你看到别人 个个都过得那么好，自己没有他们那么好。没有一个好的工 作，没有像他们那样的生活，没有他们那样成功，其实都是 假的。

幸福的真谛到底是什么？我觉得是满足。切忌不要攀比，一 攀比，心里会很不满意自己，不满意自己的工作，收入，伴 侣，生活状态等等。

过好自己的生活最重要。我一直认为幸福的生活一定是简单 的。如果你物质要求过高，便是对自己的要求更高，想要的 更多。为了想要的更多，必然要承受更大的压力，风险，责 任。牺牲很多陪伴家人和小孩的时间。

我认为幸福的生活不需要特别特别多的钱。而且赚钱多少还 要取决于你的个人能力，资源，家庭背景。不是你想赚多少 就能赚多少钱。

所以，想要过一个幸福的人生。首先，学会满足，不攀比。 别人过得再好或不好，和自己有什么关系啦。我过得好与不 好，也不需要放上朋友圈去晒去获得别人的点赞和羡慕。我 内心不需要这些，自己过得好不好自己知道。

我个人幸福的人生有几个要素：

1、做一份自己喜欢的职业/工作，有成就感，有价值，有收 获。相对自由，我有话语权，我对自己的绩效负责，发挥我 的能力潜力等。

所以，我宁愿放弃在公司工作带给我的稳定和保障，也不愿 意在公司里做一颗螺丝钉。我不需要大公司的光环，我需要 靠自己还能活得好好的。

2、生活简单一点，规律一点，上班工作，学习，运动。

3、多些陪伴家人孩子，有空的时候带他们多出去走走，除了 工作以外的时间，还要把一部分时间留给家人。

4、有几个知己朋友，能够有难同当、有福同享，能够懂你支 持你的就够了。有烦心找他们出来聊一聊，聚餐喝酒。有困 难能帮助自己的朋友就够了。

5、有空多去学习提升自己。

6、有空多去运动运动，增强体质。

7、有空多去旅行，去世界各地走走看看。 这样的生活既简单又丰富。人要活得开心一点，烦心事少一

点。所以，我不喜欢那种虚伪、耍手段心机、攀比、价值观

追求不同的人做朋友。遇到这样的人我都会避而远之。

过好自己的生活最重要，把自己经营好，做一份自己喜欢的 职业，和自己喜欢的人待在一起。不要太过于无休止地追求 赚钱，赚更多的钱，实质上有一种攀比心在作祟。

生活得简单一点幸福一点，不需要那么多物质要求。物质上 简单一点，精神上丰富一点。我所认为的幸福人生就是这么 简单。
