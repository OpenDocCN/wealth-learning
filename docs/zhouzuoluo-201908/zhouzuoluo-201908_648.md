# 《读〈刻意学习〉对

《读〈刻意学习〉对学习系统：如何才能每天进步有了新的 认知》

我们在做一件事情的时候，都会有这样的困惑。我每天练习 写作，为什么没有进步呢？我每天阅读，为什么没有收获 呢？

很多人都会遇到这样的情况，我坚持那么久了，现在没有收 获，我要不要继续坚持下去呢？

看了这本书我明白行动系统解决每天“做”的问题，学习系统 解决每天进步的问题。很多时候，学习系统很容易忽略，主 要是因为多数人无法持续行动，也就是很多人坚持之后看不 到希望就放弃了。

下面分享一下学习系统

1.领域基础知识体系 在基础知识的学习中，作者说了一个“投石填河” 的场景。“投

石填河”就是在河里建一座坝，需要往河水里扔石头。河水很

急，最开始把石头扔进去，可能会被冲走，可能会沉下去，

如果我们不继续扔，是没办法拉住水流的。这个投入一旦开

始，就必须持续下去，直到能开始看到一部分石子在水面出

现。

在我们知识的学习中，最基础的一阶段也像水里扔石头，只

有等到石头堆积露出水面的时候才会慢慢有反馈。

我们是会经历一个沉默期，这个沉默期中，所有的投入和努 力是没有任何反馈。

人和人的区别就是在这里，有的人坚持下来了，有的人因为 没有反馈就放弃了。

在任何的学习领域，我们每个人都会经历沉默期，你熬过了 这个期间后面就会有所成绩，你放弃了，前面所有的努力都 白费了。 还没有成绩之前，我们要不断的坚持下去，最终肯定会有所 收获的。

2.模块间的关联类比 每一个领域中都有不同的模块，我们要找到不同模块之间的

关联，进行相关的分析。

3.构建全面连接的体系网络 我们要把知识点全部连接起来，构成一个知识体系。 输出是最好的学习方法，可以把一个个知识点串联起来。 比如我们上学的时候，很多数学的知识点都可以相互通用

的，这就是把知识点都连接起来，我们就不用害怕考前会突

然忘记一些知识点，因为知识点相互连接的，有了自己的体

系就不会忘记。

以上就是学习系统，希望对大家有帮助。
