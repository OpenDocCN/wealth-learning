# 《职场中，沟通的路

《职场中，沟通的路径很重要》

最近需要做大量的沟通工作，有几点感触很深。 1、沟通的路径非常重要 先举个例子，之前我卖衣服的时候，经常有些顾客会挑选颜

色、款式、试穿、搭配、洗涤问题……一两个小时过去了，

终于决定购买了，再看价格——太贵了，走了。

顾客有可能是闲的，我们有些自己开店的同行时间非常紧， 还是会如此操作。比如说去进货，考虑的顺序明明是价格、 款式、颜色、面料…….那么沟通的路径也应该按照这个逻辑 重点。

但很多人不是，把一些细节看了个遍，最后又觉得价格实在 超出自己接受范围。你做一家中档的店铺，根本没必要去看 进价 500 块的裙子，即使美上了天，你拿回来就赚不到钱。

回到职场也是如此，我犯过好几次错误。比如我要找一个设 计师，确定对方能力 OK 后，把需要做的物料发过去，再说明 了要求和交稿时间。结果对方做一张图要 2500 块，完全超出 我的预算，我只能迅速道歉撤退。

沟通的路径应该是，先沟通最重要的点，即决定这次是否合 作的关键因素，再考虑其他细节。

比如说，粥老师的写作课要找公众号合作推广，有的小编完 全不顾你跟他说的合作形式，一开始就会一个劲的问我细 节，比如怎么提现、怎么操作、能不能修改软文……等他确 认好一切细节之后，又倒回头问合作模式，最后给你回“这个 方式合作不了”

真是思想有多远，就可以 gun 多远。

2、获取关键信息的能力很重要 我记得我打过最长的一个电话是一个小时，反而是获取信息

最少的。然后我跟粥老师反馈的时候，才发现关键的信息严

重缺失，我被问得说不出话来。

为什么会这样？整个沟通的过程，本来我有很多想问的，很 多次都被对方打断、抢话，或者把我的问题敷衍过去，到最 后我被对方说得自己都懵掉了，一心想快点结束通话。几次 说出结束语，对方依然没有挂断的意思。

结果就是浪费了大家的时间。后来我就总结经验了，每次如 果要语音通话，先用笔记本把我要沟通的重点记下来，不管 对方怎么扯篇，我都要先获取我需要的信息，说完事就果断 的说：“OK 了，那我挂了哦”。

事实证明，这个办法非常有效。昨天我用一个小时，分别打 了三个电话，从事件开始到达成合作意向、确定合作关键细 节、汇报老板，全部都搞定了。

在比较繁琐的沟通中，说得多不重要，把握关键信息的能力 才是最重要的。

3、聪明人的沟通路径最短

同样的一个问题，我都会提前设计好沟通的路径，话术模 版。聪明的人呢，往往按照沟通路径，一个回合就理解了， 也回复明白了。

有一种人需要反反复复的解释，比如你说第一个点的时候， 她自己跳到第二个点、第三个点，最后弄不明白了，又跳回 头去问，然后抓住一些细节不放，完全没有自己的沟通路 径。

最后，沟通能力在职场中是一项非常重要的能力，需要自己 不断地总结经验，才能提高沟通的效率。
