# 《文案天后李欣频的

《文案天后李欣频的创意课，教你改变思维经营人生》

李欣频老师的书一直都想找来拜读一下，这两天泡图书馆正 好有机会，薄薄的一本，前后总计 3 个小时就可以读完。

这本书解开了我的一些人生困惑，更重要的是提供了的一些 观点让我们以更新颖的态度去经营人生。

01\. 想象你未来的样子，用未来的你指导现在的你 不论现在你在何处，把人生倒过去五年、十年，你可能因为

一次失恋而要死要活，也可能因为一次考试失败而郁郁寡

欢，如果我们能以现在的身份回到过去那个时间节点，安慰

一下曾经那个正在受伤、难过、沮丧的自己，你会对自己说

什么呢？

2005 年，凤凰卫视主持人梁冬被挖去做百度副总裁之初，因 为没有经验也比较惶恐，他想了一个办法就是“装作”自己是 副总裁。这个办法帮他克服了初期的无所适从，以一个想象 出来的成熟行事的副总裁来指导新人期的副总裁，经过一段 过渡期就把这份工作做的愈加得心应手。

这个观点对自己有什么启示呢？我一直都一个成为写作者的 梦想，以前思维封闭胆小怕事不敢想，现在胆子大了一点敢 想了，但是受限于思维惯性，行动力和意志力都跟不上来， 在梦想和现实之间拉扯苦不堪言。

如果把自己想象成一个自己特别喜欢的公号作者，我会怎么 做呢？首先要开一个公号，每天高质量的输入与输出，不断 打磨自己的写作风格，研究各类用户增长方法，通过传播正 能量的价值观来影响受众。

爱因斯坦说，想象力就是一切，它是未来生命的预览。

02\. 给自己设限是作茧自缚，相信即看见 有一个成语叫做“画地为牢”，我们的思想就像是“画地”的那条

线，所有受局限的思想把我们紧紧地按在了现实的牢笼里动

弹不得。

很多孩子毕业时都会填同学录，其中有一项是”你的愿望“是 什么？大部分人填的都是”旅行、环游世界“，但是在公共认 知上，学生如果没有父母的资助是没有独立的经济能力去旅 行的。

网络时代给了我们无限展示的可能，有一位台湾学生把自己 做的 3M 动画作品放到了网上，获得了一家企业的青睐，并且 企业赞助该学生去国外展演，这名学生不仅可以免费出国， 还赚了一笔代言费。

自己以前对体制内的工作特别头疼，觉得凡事一听到“体制 内”就意味着呆板、僵化、人浮于事，现在想明白了，这是在 给自己画地为牢、作茧自缚，因为在更广阔的世界里我了解 到有体制内人成了作家、KOL、摄影师、烘培能手……

如果你真的想做一件事情，你可以找到一百万种方法。 如果你不想做一件事情，你可以找到一百万个理由。

你相信什么，想到了什么，就会看见什么。

03\. 站在创作的源头，拒绝被喂养别人的价值观和成功经验 应试教育好像不太教学生这个，但是我们终究要迈入社会，

不管以什么方式切入只有自己才能找到唯一的人生之路，否

则命运只能被别人掌控。这个世界不存在什么“标准答案”， 别人的看法只能作为一个参考，拿来主义在这里是行不通

的。

总是在等着你喜欢的人发观点，你去看他写的文章，读他推 荐的书，你也成为不了另外一个“他”。最好的办法是拿起笔 来，开始创作，享受颅内高潮的快感。

如果你喜欢看电影，把自己想象成导演，这部电影该怎么 拍，把自己想象成编剧，这个本子该怎么写，这些都是很好 的锻炼方法。

如果你喜欢看奇葩说，拿到辩题以后，把自己想象成辩手， 这个辩题该怎么打，对方辩友该怎么接。大脑需要不断的被 多元、多层次的刺激和练习方式，久而久之这种创作的快感 会让你欲罢不能。

04\. 打破惯性思维，抽丝剥茧找到自己的人生模式 现在的你是由过去的思维方式造就的，而未来的你藏于深深

地现在的你之中。要想未来有所改变，现在即刻必须清除从

小到大家庭、学校包括社会灌输给你的惯性思维。

比如父亲一直教育我“不要给别人添麻烦”，好的一面影响是 成长过程比较独立，坏的一面是变得比较封闭，什么事情都 想自己一个人完成，这本身是不现实的。在后来的工作中，

我发现给别人添麻烦也是一种交流的方式，大家彼此帮助完 成能量的交换。

再比如，长辈们都会嘱咐小孩子要听话乖巧，很多年轻的家 长也有样学样，教导听话乖巧的孩子才是好孩子，殊不知这 极大地遏制了生命的创造力。为什么要听话要乖呢，因为好 管理，大人省事省心。只要听话就可以受到表扬。遵循这种 教导模式的孩子长大以后往往不知道该如何去把握重要节点 的选择，只能听从别人的建议，或者凭直觉选择。

检视我们的大脑，还有没有类似的这种“惯性思维”，列出清 单，一点点把它们清除掉，看看人生有没有一点改变。

创意是一种对生命主动、积极、自主的态度，如果你也觉得 现在的生活有点单调，不妨试试上面的方法。
