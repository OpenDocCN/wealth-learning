# 《减肥用对方法有多

《减肥用对方法有多重要？》

晚上好 与星友们分享丹儿的第 65 篇原创 6.28

————————————————————————

以前女性减肥居多，近几年男性也加入减肥的大军。女生减 肥是为了更漂亮，而男生为颜值的也有，大多是为了健康。 最常见的减肥方法有：节食，吃减肥药，运动，吸脂手术 等。大家都用过哪种减肥方法呢？你认为哪种减肥方法最可 行呢？

下面我们来分析一下，从长远方面考虑哪种减肥方法，最可 行。

节食减肥 每天只是一点点，体重掉的是很快，但是长期节食很容易营

养会良，无法从食物中摄入身体所需养分，慢慢体抗力会下

降容易生病

当食物摄入过少时胃也容易出现问题。

吃减肥药

减肥药市面上太多了，包括丹儿的朋友所在的门店也卖过， 而且还很贵 8000 多一个疗程，即使是这样价格较贵说副作用 很小的减肥药，其实内部人都知道现在虽然能让你瘦，但是 过几年副作用会很大，心脏和肝胆都会受损。

8000 多的减肥药尚且如此，就别提 100，200 的减肥药副作用 会有多大了。

吸脂手术 我们公司除了旗下有美容院以外，还有一个高端的医美医

院，吸脂手术刚好前两天培训自体脂肪移植的时候提到过。

吸脂手术风险是比较大的，抽脂手术每次抽脂容量有限，容 量越大，风险就越高，当抽脂容量达到一万毫升以上，并发 症随之提高，死亡率也会上升。

而且手术前会打麻药，血管收缩药，肾上腺素等，研究表 明，手术时注入体内的药物有九成会被人体所吸收，这些药 物的副作用也很大。

运动减肥

这种减肥方式，看起来比以上三种安全一些。

不仅可以减肥，还可以锻炼身体，提升身体免疫力等。

但是运动减肥有一个缺点，见效慢。即使很多人想通过运动 减肥，坚持下来的也不多。

为什么呢？说到跑步减肥，经常听到别人说，那些常年跑步 的膝盖都不好。越跑腿越粗！跑步完以后我更饿了，吃的更 多。不仅肥没减下去反而胖了。

我以前也是这样认为的，跑步带来的坏处这么多，还那么 累，我还是靠意念减肥吧，我相信自己，我能瘦的 ！

直到听了《》这本书的作者张展晖老师的直播，我才了解到 我们认为跑步的那些坏处原来是我们健身方式的方法有误。

听了张展晖老师的课，最受益的是了解到，关于健身的一些 常见误区，庆幸还没开始锻炼，在不知道正确方法的情况下 锻炼很容易受伤。我们运动时常见的误区有哪些呢？

①认为锻炼时间越长越好，精力越充沛。 如果刚开始运动的人，不建议强度太大，随着运动的强度增

高，心率会增加，导致身体不适，不仅可能会受伤，还会对

运动产生恐惧，坚持不下来。因为运动让你觉得难受了，人 天生就不喜欢做那些让自己难受的事。

正确的方法是根据自身调整锻炼时间，最重要的是循序渐 进，一般 20 分钟最佳。20 分钟可以让我们运动后精力更旺 盛，如果时间过长，反而会损耗我们的精力。

②认为锻炼出肌肉，就会越健康 其实并不是这样的，我们锻炼出腹肌，臀肌，肱二头肌，并

不能说明我们的身体就是健康的。

我们真正需要锻炼的是心肌，心是造血的嘛，血液供应全 身。大家都知道一个人的心脏不行了才能确认死亡。一个好 的心脏是什么样的呢？

心率平稳，该快的时候可以快起来，保持呼吸均匀。比如跑 步时有没有发现，跑两步就开始喘，就是你的心率没有跟上 运动的速度，反而停下来扑通扑通一直跳。

③认为跑步，加上控制饮食会瘦 其实跑步之后，我们会更饿，因为消耗的多，这时候你在少

吃，饮食跟不上，可能会瘦，但是我们的健康会瘦到影响。

健康的方式是运动后科学的饮食，补充运动所消耗的能量， 可以吃些不容易发胖的，蛋白质，瘦肉等，少吃主食。

④跑步不需要热身，和正确姿势怎么跑都行。 知道为什么那么多人跑步腿会粗，膝盖会受伤吗？ 都是因为跑步时没有用到正确的姿势，身体给的压力太大，

错误的姿势保持一个小时，重复很多次，身体肯定会出现问

题的。

跑步前的热身动作，可以加速身体血液循环，如果不做热身 直接跑，身体没有适应的过程，做好热身动作也可以防止我 们受伤哦。

⑤锻炼需要每天坚持，不然没效果 我们的身体天天运动，也会消耗精力的哦，最好的方式是每

隔一天锻炼一次，让身体有个恢复期。

▼听了精力管理的直播，除了对科学锻炼有了一定的了解，

收获最大的是，一件困难的事，只要用对方法，循序渐进， 每个人都可以做的很好。

就像樊登老师以前从来不跑步，很讨厌跑步，现在居然能跑 6 公里了，只用了 3 个月的时间。为什么他能做到以前他觉得不 可能的事呢？就是用对了方法。

不仅是跑步，任何事情都是，别人做的好一定是有原因的， 他找到了困难事情里的方法。如果我们遇到困难时也能找到 这个方法，那么我们也有可能变成优秀的人。

准备在看看《》，制定锻炼计划了，身体是革命的本钱，只 有身体好了，才能做我们想做的事情呀
