# 《人应该支配习惯，

《人应该支配习惯，而决不能让习惯支配人》

这是程先生的太太 22 次分享，希望我能给你正能量。 奥维德说，没有什么比习惯的力量更强大。我是认同的，因

为我真正见过习惯的力量。

1、你还没成功是因为你坏习惯太多 请回顾的一下，你的一天是怎么样度过的，特别是在周末，

假期这样没有约束力的日子。

你可能不曾见过周末的日出，因为一觉睡到中午 12 点，别人 是吃午饭，你是刚睡醒。也许你根本就还没醒，又或者你想 再赖赖床，因为你觉得起床了也没什么事。

然后你拿起了手机，在床上刷朋友圈，看到别人的周末很是 丰富多彩，便决定约上几个好友组个局。好不容易刷朋友 圈，刷微博刷到了聚会的开始，你才从床上起来，紧赶慢赶 的出发，到了发现你是最后一个到的人。看到桌上有吃的， 你随便拿起来就放进嘴里。这一玩，就是天黑。 上面的假设 属于虚构，如有雷同纯属巧合。

但是在你的野心和你的生活之间，你有多少的坏习惯阻碍了 你的野心？

许多人习惯熬夜刷微博，刷朋友圈，习惯迟到，习惯吃垃圾

食品，习惯认为自己没成功是因为这个社会太不公平。

我也有坏习惯，有些改过来，有些还没改过来。 我习惯吃零食，辣条曾经是我的最爱。所以我得接受我身材

肥胖，并不是因为别人太瘦，才显得我很胖。

我习惯熬夜，所以我得接受我面色蜡黄，毫无精力，并不是 因为别人年轻漂亮，才显得我人老珠黄。

2、改掉坏习惯，养成好习惯 习惯有好有坏，生活不好很多时候是因为坏习惯太多。养成

好习惯，生活自然就会变好。

我已经戒掉零食三年。我不会因为心情不好，或者出于某种 冲动就随便吃零食。因为我已经养成了不吃零食的习惯，饿 了我会选择吃主食，而不是零食。

我也很少熬夜，程先生说是在她的呵护下我变得更美了。其 实是因为在他的呵护下，我很少失眠熬夜了，脸色自然也好 了很多。

查尔斯-杜希格曾说：我们每天的活动，有 40%是习惯的产 物，而不是自己主动的决定。

习惯其实是没有什么逻辑的，是不经思考的，是我们长期进 行某一活动后形成的结果。

习惯吃零食，身材大概率可能就不会比习惯吃蔬菜肉类等营 养食物的人好。

习惯熬夜，人看起来大概率也会比习惯早起早睡的人老一 些，疲惫一些。

习惯迟到，大概率给人的感觉也没有习惯准时的人靠谱。 坏习惯和好习惯造成的结果是截然相反的，不管坏习惯能满

足当下的任何快感，从长远来看，都不会有好结果。反而好

习惯虽然当下的执行可能比较困难或者痛苦，但把时间拉

长，取得的成效确实不可估量的。

其实这是很简单的数学问题。

坏习惯我们取值 0.99，好习惯我们取值 1.01，它们的值刚开始 的是差别不大，但是习惯是每天都在发生的事情。

1.01³⁶⁵=37.8

0.99³⁶⁵ = 0.03

一年的时间，差别就很明显的了，更明显的是，我们的习惯 通常是伴随着我们一生的，结果当然毋庸置疑。

3、好习惯不是一天养成 罗马非一日之功，好习惯非一日养成。 在我们养成养好习惯的时候得先明白哪些是坏习惯。熬夜对

有些人来说是坏习怪，对有些人可能就不是了。人家晚上 3:00

没睡可能是在学习不是坏习惯，你 3:00 没睡在刷朋友圈可能就

是坏习惯了。

其次，养成好习惯是一个日复一日地过程。好比说一年内我 要做 365 个俯卧撑，你一天两小时内完成不是养成好习惯，只 是完成任务而已。但是你每天能坚持做一个俯卧撑，坚持一 年，两年甚至一辈子，这才是好习惯。我们需要尊重习惯的 养成过程。

再者，别急功近利地想一次就改掉所有的坏习惯，一次改掉 一个就好了。短时间内想改掉太多的坏习惯，往往意味着要 挑战自身的人性，结果往往都会失败。有时候慢即是快。100 改掉一个坏习惯就已经很不容易了。

大家都很熟悉《钢铁是怎么样练成的》著名名言。 人最宝贵的是生命，生命对于每个人只有一次，人的一生应

该这样度过：当回忆往事的时候，他不会因为虚度年华而悔

恨，也不会因为碌碌无为而羞愧；在临死的时候，他能够

说：‘我的生命和全部精力都献给了世界上最壮丽的事业—— 为人类的解放事业而斗争’。

但还有一句是：人应该支配习惯，而决不能让习惯支配人。 一个人的习惯优劣，很大程度上决定了他的一生。做习惯的

奴隶还是做习惯的主人，是我们的自由选择。是时候改变自

己的坏习惯了，别让它毁了你的一生之后，才追悔莫及。
