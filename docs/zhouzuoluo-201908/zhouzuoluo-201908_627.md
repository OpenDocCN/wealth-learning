# 《“土拨鼠之日”的

《“土拨鼠之日”的 3 个阶段，你属于哪一个阶段呢？》

我们总觉得看不到未来，也总忽视了当下——第 212 天的星里 浣

前两天看了一部电影《土拨鼠之日》，电影前面看得我很羡 慕，直到后面我才懂得珍惜。

电影的主人公菲尔是一名小有名气但非常自我的气象播报 员。在一年一度的“土拨鼠日”（一个用土拨鼠能否看到自己 影子预测冬天还有多久过去的节日）来临之时，菲尔需要和 两名同事（Rita&Larry）前往一个小镇进行报道。当菲尔例行 公事般完成报道准备返程时，却因一场暴风雪不得不返回镇 上过夜，等到菲尔第二天醒来时，日期却依然停留在“土拨鼠 日”，所有的人和事物，都在一直重复......

对于菲尔意外获得的这种“日复一日”的能力，我是挺羡慕 的。因为你不用担心明天到来，也不用为未来负责，反正日 子都会照样过。

这样“无所谓的轻松”，真的太有诱惑力。 电影中，菲尔一共经历了三个阶段，你也能从中看到自己的

三个阶段。

第一阶段的菲尔:无所谓，反正有大把时间

刚开始，菲尔很抓狂，但他很快接受了现实，在“有大把时间 可供挥霍”的状态下，菲尔进入了对一切都无所谓的状态，他 丢掉所有“束缚”，用“时间漏洞”欺骗感情，盗取财物，只管纵 情当下，成了一个没心没肺的“享乐汉”。

我也曾是菲尔，拥有大把时间，却对一切都无所谓，自甘堕 落、不求上进，只想混沌度日，不想担责任。

那时候自己现状很糟糕，却用“时间会让人过上不好不坏的生 活”这样的话来麻痹自己;拿着三千不到的月薪，却想着早点下 班回家玩游戏;从来不去学习成长，觉得只要养活自己就可 以。

那时候我过的就是日复一日的生活，每天都不希望明天到 来，只想沉浸在当下的懒散中，不用去管未来是什么模样。

第二阶段的菲尔:用今天的努力去改变昨天的问题 在对“放纵生活”失去兴趣后，菲尔开始重新思考起当下的意

义来，他决定利用自己的“超能力”去改变已经出现的问题。

比如在“昨天”遭遇情感危机的年轻情侣、吃牛排差点噎死的 中年男子、将要在街边孤独死去的流浪老人、马上会从树上 摔下来的调皮男孩，曾被他伤害忽视过的 Rita&Larry 等等，这 些“昨天”的问题，菲尔都在“今天”用自己那“重复时间的超能 力”去一一改变（尽管第二天还是会重复）。在这个过程中， 菲尔的生活开始变得充实，他也找到了新的人生信条——做 力所能及的改变。

想一想我们的昨天，是不是也出现过各种问题，那些问题你

有在今天通过努力去弥补吗？你是不是觉得昨天的问题已经 过去，而今天的问题无法解决，所以把期望都寄托在未来希 望时间把你拯救？

粥老师提过一个“时间困境”的概念：

“如果我现在是傻逼，那么我现在再努力学习，也还是傻逼， 因为我现在傻逼是由我的过去决定的，我现在努力学习，是 为了一年后不再是傻逼。”

电影中，菲尔就是用自己获得的“时间超能力”在今天解决昨 天的问题，或者说当下的尝试是为了在明天把今天的问题解 决掉。

第三阶段的菲尔:珍惜当下，面对未来 菲尔在一次次“行善积德”的过程中完成了心灵的蜕变，懂得

了珍惜，学会了承担责任，追求美好。也许是他通过了上帝

的考验，最终困扰菲尔不知道多久的“土拨鼠之日诅咒”终于 被破解：他成功进入了新的一天！

在电影的结局，菲尔和女主角 Rita 奔跑在雪地里，踩下了一起 前进的脚步，此时的菲尔，不仅收获了一个全新的自我，一 份全新的感情，还有将要继续的全新生活以及未来。

如果没有明天，今天你会在干嘛? 如果你陷入“土拨鼠之日”，你打算如何度过？ 你会好好珍惜当下，面对未来吗？

第三阶段的菲尔，就是此时此刻的你。
