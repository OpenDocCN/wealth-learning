# 《怎样不把自己最坏

《怎样不把自己最坏的一面展现给最亲近的人》

在我和女朋友的相处中，总是会出现这样情况，双方明明都 在爱着对方，却总是会无意间伤害对方。

恋爱中的人总是特别容易互相伤害，伤害完之后却又丈二和 尚摸不着头脑，她怎么又生气了?明明刚才还好好的，一句话 可能就会让对方流下眼泪。

在我们的沟通中总是充满着暴力，暴力沟通自然是最低效的 交往方式。

生活中，工作中，无时无刻不在进行着这种低效率的沟通， 却往往有苦难言，不好的情绪伴随我们一整天，此时我们的 大脑并不完全理智，再去做决定或者做其他事情，大概率也 是不完全理智的，生活只会变得越来越糟。

为什么会出现暴力沟通? 1.道德批判

对他人的评价实际上反映了我们的需要和价值观。别人做

事，我们用自己的价值观来评判他们，彰显自己，很可能会

招来敌意。

恋爱中的道德批判，你已经两个小时没有回复我的消息了， 你好久没有给我买礼物了，你一定是不爱我了！

不回消息、不买礼物，可能有其他原因，并不是不爱。可以 说事实，但不能评判，比如，你已经两小时没有回复我的消 息了，我今天遇到一件事情，心情不好，可不可以陪我聊会 天?

你好久没有给我买礼物了，我看上一个东西，特别喜欢，那 个东西要是你送给我，我一定会特别开心的，给我买一个呗?

工作中的道德批判，小王连续三次上班迟到，老板对他说， 小王，你工作态度太不认真了，有点责任心好不好?

迟到并不说明工作态度不好，没有责任心。两者关系并不 大。可以这样说，小王，你已经连续三次迟到了，已经破坏 了公司的规则，我作为领导不好给大家交代，你下次来早一 点呗。

道德批判会莫名的伤害到别人的感受，对于沟通是不利的。

2.进行比较 如果我们想过更惨的生活，和别人比较就完了呗，和马云比 较，和大佬们比较，相信我，一定可以的。

拿我们和更优秀的人比较，很多时候会变相贬低我们，比较 也是一种批判。

比如别人家的男朋友从来没有让我失望过！

3.回避责任 我们对自己的思想、情感和行动负有责任，“我每天都得打扫

房间，每天都得加班，因为我不得不做”，这种表达淡化了我

们的责任。

一旦我们意识不到我们是自己的主人，我们就成了危险人 物。我们无法强迫他人按照自己的意愿做事。

将我们不得不做的事情改为主动选择要做，这样就避免了回 避责任。

我每天打扫房间，因为我想让爱人回来不用那么劳累。我选 择加班，因为我想保住工作的同时升职加薪。

4.强人所难 自己不喜欢的东西就不要强加在别人身上，强人所难就是告

诉别人，如果你不这样做，就会收到惩罚，这样做没有一点

温度，我们体会不到一点爱。

以上就是我们在沟通中出问题的原因，认清这几点，然后行 动，一定可以更好的沟通。

那么怎样非暴力沟通呢？ 1.区分观察与评论 小王迟到三次。这是观察，描述了事实

小王总是迟到，工作不认真。这是评论，只迟到三次，却用

了“总是”，还由此评论工作不认真。

不带评论的观察是人类智力的最高形式。面对一件事，我选 择摆事实，不加任何评论。

2.体会和表达感受

在恋爱中，在工作中，不会表达自己的感受，那是很悲观 的。当我们感到沮丧或者伤心时，一定要完全说出来，这是 特别重要又很难的一个步骤。

有时候碍于面子，我就说不出来，总感觉说出来会显得自己 很卑微，不说出来，对方是不知道我是怎么想的，当然也不 会照顾我的感受。

示弱才有利于解决冲突。我与人交流的一个小细节就是，都 会向别人请教一个小问题，请教就是示弱，给别人足够多的 话语权，示弱了，他们对我的印象特别深刻。

表达内心的感受，不仅可以促进亲情，还可以改善工作。 3.找到感受的根源 当我们有某种感受时，大多是因为我们的需求没有得到满

足。

我们生气，可能是上司没有尊重我们。 我们郁闷，可能是女朋友不关心我们。

关系出问题，找到情绪的根源，找出哪些需求没有得到满 足。

4.请求帮助 将自己的的需求表达出来，提出具体的请求，请求对方怎么

做。这里是请求，不是要求，这真的是让我屡试不爽。

提出简单直白的请求，即使是个钢铁直男也知道该怎么做了?

举个例子: 女孩想让男孩给她更多的关心，可以这样试试:

宝贝，我们每天聊天的时间平均只有十分钟，我是一个害怕 孤独的人，每天和你聊天我都特别开心，让我感到特别有安 全感，你已经做得很好了，不和你聊天，或者有时候发消息 要过几个小时才回复，我就特别担心你，也特别孤独，我希 望你每天能多和我聊聊天，忙的时候就和我说，我能理解 的，也希望你可以多关心一下我，每一次你的关心都会让我 格外心动。

这是我用上面的方法写的，比我以前的沟通方式好了很多。 以上就是我读《非暴力沟通》的读书笔记以及一些我的体
