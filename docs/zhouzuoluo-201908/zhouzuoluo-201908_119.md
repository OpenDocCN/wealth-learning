# Day220《财富

《财富的理想国长什么样子？》

关于财富的 117 条法则，你感兴趣吗？翻开这本《财富的理想 国》，我开始去寻宝，估计我可以玩好几天。

今天先提炼 10 条精华清单与你分享：

001 人们最想用钱买到的东西有：安全感、舒适感、身份地 位、影响力、自由、受人欢迎等。我第一时间想到的是，钱 越多，就可以发越多红包了，朋友们开心，我也开心。

002 作者对很多有钱人做过研究后，发现他们都会遵循一些基 本原则。这些原则不是多么高深的知识，其中一些原则和行 为，我们其实都知道，却没有变得有钱——因为没有好好实 践。

003 赚钱需要聪明地勤奋，懒人是赚不到钱的。我终于明白我 为什么不够有钱了，一个字：懒。

004 作者对财富的定义很有意思：拥有足够的财富，无须再担 心是否拥有足够的财富。有一种不再看自己存款数字的感 觉，反正数不过来了。这跟李笑来给财富自由的定义一个逻 辑：再也不需要为赚钱出售自己的时间了。想想就很美好。

005 一定要有财富目标。如果没目标，又怎么可能向着目标前 进。在《财务自由之路 2》中，建议一开始就写下三年后的梦 想收入，不妨把这数字往你敢想的尽可能大的范围写。这个

数字本身不重要，更重要的是，这个过程中你会变成怎样的 人。

006 一切都不会自动变得更好，除非你做些什么。只是知道一 些东西确实不够，去实践才能带来真正的改变。当你有了坚 信的财富法则后，要经常问自己：我怎么才能立即行动起 来？

007 用对了钱，钱可以生钱。不要总是把钱花光，要用投资的 思维让钱为你工作。靠辛苦赚钱，永远不会成为有钱人。

008 价值的内涵远超价格，一个东西的价值等于其他人愿意为 它付出的代价。要花时间研究价值和价格的区别，并用上你 所领悟的道理。

009 赚钱是个可以学会的技能，需要具备成为有钱人所需的决 心和动力，愿意学习并认真实践。在赚钱这件事上，有改变 和行动的意愿是最重要的。

010 无论你的计划是什么，重要的是有计划，财富不会偏爱盲 目之人。计划可以让你知道做什么以及所做之事是否有助于 抵达你的目标。

每天告诉自己：不要懒惰，不要等待别人的施舍。一切有价 值的东西，都要自己去赢得。这不止于财富。
