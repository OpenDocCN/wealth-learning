# 《凌晨五点，我家小

《凌晨五点，我家小区爆炸了！》

我们总说生命无常，可没有亲身经历过，很难感同身受。而 今天凌晨三点，真的让我体会到了生命的脆弱。

咣！一声巨响惊醒了睡梦中的我，第一反应就是地震了，于 是和家人披上羽绒服就往楼下跑。跑到楼下后，看到天上有 浓烟混混，听到聚集的人说是 7 号楼的住户家里发生了爆炸， 火势严重。

于是走到视野开阔且安全的区域观看爆炸情况。看着熊熊烈 火真的让人崩溃，消防员持续利用高压水枪对火源进行喷 射，但很长时间火势都没有得到控制，看得我们都无比心 急。我们不知道还有没有人困在楼上，如果有，早一秒把火 扑灭，对他们而言就多了一分生的希望。

最后，在消防官兵的不懈努力下，火源终于被浇灭。但看着 大火后的残局，真的让人无比痛心。好几间房屋都已面目全 非，震碎的窗户不计其数，一地的玻璃碎渣诉说着这栋楼受 过的冲击。最让人关心的伤亡情况，暂时还没有官方回应。 但据小区群众说，好几人都受伤住院，受伤程度轻重不一。

官方初步调查结果显示，此次爆炸是由于某用户燃气使用不 当引起的。但他不仅害了自己，还给其他家庭带去了灾难。 对于被牵连的家庭，真的是祸从天降。他们没有范任何错， 但却要为别人的错误付出了沉重的代价。确实不公平，但也 只能接受。

明天和意外不知道那个先来，但我们总是心存侥幸，对于一 些小概率的意外总是毫不在意。但概率是针对整个群体，具 体到个人。就只有 0%和 100%。

就像我重来没想过离我家这么近会发生爆炸，但它确实发生 了。爆炸房屋两侧的六号楼八号楼，都有多处玻璃振碎甚至 房屋出现裂纹，而我家只住在五号楼，差一点就要被波及 了。

早上好多朋友都打电话询问我情况，我都开玩笑说我命大没 那么容易出事。但其实我心里是后怕的，估计被波及的家庭 也没想过有一天回和煤气爆炸沾边，但有些事就是不讲道理 的发生了。

我们总说余生漫长，今天没做完的事留给明天做，今年没有 完成的计划留给明年完成。但有些事错过了真的就没有机会 了，我们说的明天可能永远都不会再来。只希望我们都活在 当下，做好眼前事，珍惜眼前人。抓紧自己能抓住的，那些 不可控的意外就顺其自然，人这一辈子，总有许多事是掌控 不了的，强求无益。

希望因爆炸受伤的邻居早日康复，也希望我们的人生永远没 有意外。
