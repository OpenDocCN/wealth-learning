# 《品牌最大的优势是

《品牌最大的优势是能提供确定性，你最大的优势是什 么？》

昨天到广州火车站候车，我又一次选择了的味千拉面。倒不 是因为它多好吃，是因为我能确定这家店的味道，它几乎是 恒定不变的，选择它不用冒险。

记得读大学时，得先到武汉中转，有一次跟同学一起停留， 就在火车站附近吃热干面。当时正常的热干面价格是 1.5 元， 但是在车站卖到 8 元一碗，而且巨难吃，第一口下去就想干 呕。

第二次途径武汉我们就学聪明了，不吃面吃饭，结果吃的肉 丸子有一股酸馊味，看着凶巴巴的快餐店主，我们也不敢言 语，发誓再也不吃火车站附近的东西。

火车站附近的东西又贵又难吃，想必你也有过类似的体会， 不仅仅武汉火车站如此。可是后来，慢慢的情况就变了。

比如说广州火车站，附近开了真功夫、KFC，一定会选择这 样的连锁店。至少，不会把吃饭变成一次冒险。

我发现很多火车站附近，只要有连锁店出现，一定是生意最 好的。比如说厦门、上海，包括武汉现在的火车站，都逐渐 被这样的连锁店占据，越来越少那些老旧窄小的小店了。

论价格连锁店并不占优势，论味道，假如把这些店铺放在普

通的街道中，又实在是乏善可陈，不见得更加吸引顾客。在 广州一些老字号聚集的美食街，这些新式的连锁餐厅甚至根 本无立足之地。

但是在火车站这里人流量特别大的地方，它们反而发挥了自 己的优势，干就是品牌所累积的确定感。无论店开在那里， 他们提供的口味基本上是恒定的，可以预期的。

这让我联想到个人，确定感会不会成为一种优势？比如说老 板交给你一件事，是可以预期的，并且是持续稳定的。

回顾以往的经历，我认为一个人能给“确定感”就是一种优 势。

比如说开店时有的员工偶尔会业绩很好，有时会差到跌破最 低纪录；有的员工可能会经常性的临时请假，你永远不知道 她今天是不是能准点来；还有的心情好对顾客就好，情绪一 上来就把直接怼人……

一个品牌的优势与价值在于能提供确定感，其实在职场中， 也应该提供这种“确定性”，让稳定和可控成为自己的优势。 2019-02-10(73 赞)
