# 【有一种牛逼，叫死

云 : 【有一种牛逼，叫死磕到底】

大家好，我是名字有点普通的云同学，这是我第 41 次的分 享，我要成为更好的自己[玫瑰]

今天第一次观看得到的例会直播，着实被震惊了，这已经是 第 114 次例会，也就代表着两年前，他们已开始尝试内部对外 透明化，实在超前。

让我印象深刻是罗胖分享的案例，让我对“死磕到底”这个词 有了更深的了解。

上周是得到大学夏季开学典礼，有八名学员自由报名上台分 享，其中有几位受到了上海沪杭校区的马徐骏老师的培训。

这培训过程令人发指，首先，一周内要脱稿，然后手抄十 遍，正着抄完反着炒，要熟练烂透了才行。

热身结束了，就得上正菜。这不，还得再练一百遍，怎么 练？

三个人分站镜子前，念着自己的稿，说是不许相互干扰，但 耳朵里听的就是别人的声音。这马老师也站一旁，手里就捻 着个红球，随时一扔。谁的眼睛要是瞟了下红球，得了，这 一遍就不算数，重来。

这样的魔鬼训练，简直就是把悟空往死里打，让他蜕变成为 超级赛亚人的过程，一般动漫里看得多，这现实还是未曾见 闻。什么是死磕到底？我想没有比他们更为生动形象的答

案。

无独有偶，得到的技术大牛，快刀青衣也因为演讲，与死磕 上演了一段不得不说的故事。得到三周年时，快刀青衣被要 求上台做一次长达 18 分钟的演讲分享。

作为砧板上的肉，逃是逃不了了，咋办。小白一名的他只能 准备，准备，再准备。

怎么准备呢？给自己准备逐字稿，一份连标点符号的停顿和 掌声间隙都要设计在内的逐字稿。准备到什么时候？听说直 到在上台的前一刻，仍旧在迭代更新。改了多少版本？起码 从他的分享图里，大版本我就看到了不下十份。

好吧，稿子也算准备好了，自然还得背，怎么背？每天提早 40 分钟上班到公司背两次，晚上睡前再背一次。期间两次公 司培训，来个当众背稿，最后两天彩排直接上台带妆背稿。 什么？上台不知道该怎么动？那就站着别动好了。

不得不承认，这种精心准备的状态基本和当年高考无差，区 别只在于一个是小白的临时抱佛脚，一个是十二年沉淀后的 爆发。

最终的结果大家都知道，快刀青衣可谓一讲成名，不忘词， 不超时，表现一级棒。

我也终于明白，为什么得到大学的校训，第一个词是“具 体”。因为他们干的就是落地的活、死磕到底的事，你没法不 具体，不具体就出不了活，干不成事。

你看阿，这人的本身其实都不牛逼，都是因为干着牛逼的事 才让自己变得不断牛逼。

正如快刀青衣在复盘最后提到的苟且红利 :“只要你在别人苟 且的地方都多认真一点点，就会马上显得与众不同。”

因为，有一种牛逼，叫做死磕到底。 看帖如有收获，点赞是种态度[玫瑰]
