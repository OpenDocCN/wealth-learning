# Day93《查理·

《查理·芒格：我的剑，传给能挥舞它的人》

查理·芒格是我心中的智者，今天我又翻出了他的一篇经典演 讲来细细品味，又有了一番新的启发。

01 如果你想获得某样东西，那就让自己配得上它 这是芒格小时候就懂得的一个道理，他称之为黄金法则。我

们往往都嫌好东西不够多，但我们是否配得上这些东西？比

如别人的信任，更大的财富，更高的社会地位等等。

如果你真正明白了这个道理，那你在生活中能赢得更多的东 西，当然不限于金钱。

02 必须坚持终生学习 芒格说他见到的那些生活越来越好的人，不是最聪明的，甚

至不是最勤奋的，但他们都是学习机器。每天晚上睡觉的时

候，都要比那天早晨聪明一点点。

如果你早早养成了每天学习的习惯，必将终生受益。

03 只有学习了学习的方法后，才能有所进步

既然提到了终生学习，那不得不谈一下如何去学习。有个大 牛说过一句很牛的话，他说只有当人类“发明了发明的方 法”之后，人类社会才能快速地发展。同样的道理，我们学会 了学习的方法后，才能学有所成。

如果你对学习方法有兴趣 ，欢迎找我来交流，2019 年我给自 己划定的研究课题是：高效学习。

04 不断学习各学科最重要的知识点，并不断实践它们 知识无穷，最值得我们学习的却是有限的，那就是重要学科

的重要理论。当然掌握了这些道理后，还必须通过实践去使

用它们，就像钢琴演奏家那样，如果不持续练习，就不可能

谈得很好。

如果你掌握了跨学科思维，并有了正确的实践，那你的生活 会更有乐趣，你能做更多的事情，你变得更有建设性，当然 也更富有，哪怕你的天分一般。

05 要想更好解决问题，要学会逆向思维 关于逆向思维，芒格提到了一句非常著名的话：“如果知道我

会死在哪里，我将永远不去那个地方。”

比如，你想让生活变得更美好，你应该考虑的问题不是“我要 怎样才能让生活变好？与之相反，你应该问自己；“我怎样会 让生活变糟？”你应该找到对生活无益甚至有害的事情，然后 避免去做它。比如损害个人健康，侵犯他人利益等。

生活中存在诸多复杂问题，逆向思维能够帮我们解决正面思

考无法处理的问题。

06 摆脱自私及偏见，摆脱嫉妒、怨恨和自怜 芒格谈到了一种叫做“自我服务偏好”的心理机制，根植于人

的潜意识，导致人们做傻事，连著名作曲家莫扎特都中招了

——他总是透支收入导致大部分时间过得很悲惨。

嫉妒、怨恨和自怜就属于“自我服务偏好”，都是错误的思维 方式。一方面我们个人要尽可能摆脱，同时能够理解和宽容 别人的这种“自我服务偏好”。

如果你想说服别人，那你最好诉诸利益，而非诉诸理性。

07 做你们最感兴趣的事，然后努力配得上它和你的合伙人 芒格说他可以强迫自己把许多事情做得相当好，但无法将他

没有强烈兴趣的事情做得非常出色。我们每个人应该都差不

多，所以要想办法去做那些有强烈兴趣的事情。

当你做事情的时候，也要配得上你的合伙人，他可能是你的 老板，你的同事，总之在一起做事是为了把事做好，起码要 非常勤奋，要配得起他们。

“我的剑，传给能挥舞它的人。”希望你能获得智者传授的智 慧之剑，度过愉快的一生。
