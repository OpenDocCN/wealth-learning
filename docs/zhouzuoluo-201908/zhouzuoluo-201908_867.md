# 《如果时光可以倒流

老王 : 《如果时光可以倒流，你想对七年前的自己说些什 么？》

【大家好，我是老王。一个在个人成长领域探索、践行的自 媒体人，今天是新生第 32 篇文章。】

今天，我想写一写，现在的我如果遇见七年前的自己，我会 说些什么。

走出校园，步入社会刚好七年了。在这七年中，我失去了很 多，也收获了很多。我的认知水平，从一个咿咿呀呀不会说 话的婴儿，成长为一个满脸皱纹的老人。

李笑来老师说:七年就是一辈子。这话真的没错，你人生的每 一个七年，就是一次生命的轮回。

01、一辈子真的很短，只争朝夕 在我们小的时候，每天盼着星期、盼着放假、盼着过年，各

种等待，以为时间过的真慢，一年的时间太长。长大后，成

家立业、结婚生子，才认识到时间的短暂、宝贵。

每个人平均从 20 岁步入社会，第一个黄金七年是 20 到 27，第 二个黄金七年是 27 到 34，第三个黄金七年是 34 到 41。40 岁以 后，你的人生差不多就都定型了，像夕阳一样，开始走向了 下陂路。

你看，这样算的话，你的一生就清晰多了。什么改变命运、 什么抓住发财致富的机会，很大可能会出现在这三个黄金七

年中。

所以呢，请珍惜你的每一天，属于你的机会很可能会出现在 现在的某一天。如果你还像过去浑浑噩噩、稀里糊涂的去度 过，那你这一生真的就只能是那样了。

02、能一辈子坚持做一件事，真的很难 在过去的七年中，我炒过股票、买过比特币、开过实体店，

但都以失败收场。现在想想，原本可以挣钱的事，之所以变

的亏欠，是因为你没有很耐心的去做一件事。

大部分人以为，很努力的抓住一次机会就可以致富、改变命 运，其实，那是你的励志故事看多了。

那些在某个行业做的很牛逼的人，不是因为他们抓住了某次 机会。而是因为他们在那个行业里，已经沉淀、积累了很多 年，只是在某个时间点，社会发展的趋势，放大了他努力的 结果。

罗胖说过一句很接地气的话:种一棵大树，最好的时间是七年 前，其次是现在。所以呢!你要很耐心的去，用每一天为这棵 小树苗浇水、修枝剪叶。

你要相信时间，相信持续、稳定积累的力量，功夫到了，收 获是迟早的事。

03、对待金钱 年轻的时候，以为挣下的钱就是用来花的，花完还可以再挣

那！但是从大部分普通人的结果来看，我们的能力真的太有

限，时间太有限。

如果，你不去为你的将来提早做打算，挣的钱都花在那些没 有收益的事情上，那你以后的日子一定会过的很艰辛。

每月的薪水，除了家庭必要的开销，你要用很小的一部分投 资到你的学习、成长、能给你带来收益的事情上。

你要知道，你现在的精打细算，都是在为你的下一代打基 础。你要为他们做一个好的榜样，为他们提供一个好的接受 教育的环境，为他们保障基本的生活物质需求。

每个人的未来，都取决于他现在所过的每一天，现在所做的 每一件事。让我们，在未来遇到更好的自己。
