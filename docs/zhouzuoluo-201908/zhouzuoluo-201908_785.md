# 《读＂一生的计划＂

道亦有道，大道至简 :

《读＂一生的计划＂，有感》 努力做到:与人为善，久处不厌，的 90 后大男孩

最近看了一本书，名字叫《一生的计划》，是美国作家，格 莱恩.布兰德写的，号称是全球销量 100 万册，我看后觉得很有 意义所以分享给大家。

其中主要内容就是经营自己的人生总时间，然后按照四个维 度去分析，最后达到人生的平衡度。

那么什么是平衡度呢？ 又有哪四个维度呢？

我们先说一下平衡度，向死而生这本书大家都知道吧，是创 新工厂创始人李开复先生所写，他就是个工作狂人，每天大 量工作和不规律的作息导致患了癌症，这是一个多么可怕的 疾病，最后慢慢康复了以后有感写了这本书。

试想一下就算你在事业上得到了很大的成功但英年早逝，这 难道说不是最大的遗憾，还包含着不甘与无奈吧！所以人生 需要平衡，只有平衡才能让我们更好是享受生活与幸福。

第一个，＂精神生活＂ 什么是精神生活呢？

自身修养及崇高的追求，为什么有些人要说这样一句话，我 跟谁谁合不来，因为三观不同，这三观就是你对正确精神生 活的追求，还包含了儒学的，仁义礼智信这些古人留下来的 宝贵文化都可以是你精神生活的追求。

精神生活是一个人思想的境界，往往越大成的人其精神境界 越高，无论我们的生活境地是怎样的，都要孜孜不倦的去提 高精神生活。

第二个，＂财务生活＂ 这一条理解为你的财务能力，其中有关于理财方法的几条。 1.不要透支消费

2.不要冲动消费

3.建立预算，买一个账本

4.每月发工资先付钱给自己

5.正确的支付你每个月的账单及顺序

6.投资

只有好的财务基础才能让我们过上富足的生活，爱情与面包 你该怎么选，我相信现在大部分的人都选择面包，这不是大 众肤浅了而是更加的理性了。

因为，有面包同时也会有爱情，但有爱情并不一定同时带来 面包，所以想让你的父母、爱人、子女、这些最亲近的人过 上好的生活，那么就请努力提高这一条吧！

第三个，＂教育的平衡＂ 从我们出生以来就在接受教育，有的人念到初中，有的人念

到高中，有的人念到大学，还有的人念到博士、硕士。

但这紧紧是学校里面的学习，很多大学生毕业以后就停止学 习了，到了社会就很难适应，这就是没有持续学习的结果， 其实学习是没有止境的，要懂得学习带来的力量，更要明白 要做一个终身学习者。

第四个，＂娱乐目标＂ 这一条的核心是多元化发展兴趣爱好与健康，生活中少不了

娱乐，更少不了健康。

前两天看超级演说家第三季，其中乐嘉战队有一个小女孩， 因为从小父母离异，他很自卑。

但是一次意外体验让他学会了叠杯这个体育竞技，自从学会 了这个就有了目标，然而让人想不到的是，竟然只用了五年 时间就得了女子三乘三的世界冠军。

更让人想不到的是，五年之中她学习滑板，竟然也得了一个 亚洲比赛的亚军，这就是热爱带来的力量吧。

以上就是这本书的主体内容了，通过学习这本书我更加宽泛 的，而不是之前单一维度理解成功的概念了，只有达到了各 方面平衡才能幸福的过好一生。
