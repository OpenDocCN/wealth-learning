# 《上一秒的自己已死

文曲鑫 : 《上一秒的自己已死，下一秒才是要关心的》 今天去面试，发现优秀的人实在太多。简直被碾压了。

面试完后，面试结果自己心里都有数了。 按照我以往的性格，恨不得找个地缝钻进去，先逃避这个结

果再说。

现在慢慢在修炼自己的心智，不畏惧失败。 我会用到一个比较好的自我暗示技巧：

“上一秒的自己已死，下一秒才是要关心的”

这个暗示，可以很好地让我从失意的状态尽快走出来。

当你遭遇不顺心的事情时，不妨用“上一秒的自己已死，下一 秒才是要关心的”来暗示自己。

希望，对你管理情绪有用。
