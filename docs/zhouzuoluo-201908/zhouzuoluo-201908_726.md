# 《粥左罗：沉没成本

《粥左罗：沉没成本到底是不是成本？其实，是不是都不重 要》

上一篇分享我讲了：你为一件事放弃的最大代价，即成本。 那么沉没成本是不是成本？ 当然不是了。沉没成本是你已经花出去并无法再收回来的，

所以你不用再为它放弃什么，它当然也就不是成本了。凡说

成本的时候，都是面相未来的，而不是过去。

比如你买了一个知识付费课，当你付完 199 元之后，它就沉没 了，不是一块一块沉没的，它是瞬间全部沉没的。

提到沉没成本，其实是为了一个词：放弃。 人生的艺术是取舍的艺术，取舍这两个字，舍比取难得多。 之所以难，就是因为我们很多时候不懂沉没成本。我们理解

沉没成本就是为了更好的“学会放弃”。

一个课买了之后，你听还是不听，取决于什么？ 完全取决于它对你的价值。

完全不取决于它是你 199 买的，还是 1999 买的。你付完钱之 后，一个样。唯一不同就是，他们带给你的价值不同。

如果你买了个 1999 的课，你发现根本没多少价值，你应该果 断放弃，再也不要听。

很多人，就觉得哎呀太贵了，钱都花上了，硬着头皮听完 吧。

比如你会在发现电影不好看之后，起身就离开么？大多数人 会想，来都来了，钱都花了，看完吧。

比如你文章写了一大半了，发现这个选题不行，你会放弃 么？大多数人会想，写都写了，写完发了得了。

比如你在餐厅点了很多好吃的，发现吃饱了但还有两个菜

（不好打包），你会放弃么？大多数人会想，这么贵点的，

太浪费了，于是让自己撑得难受也会继续再吃一些。

比如你买了一件衣服，但其实很不适合你，穿上很给你拉 分，你会放弃么？大多数人会想，买都买了，至少穿个 10 次 8 次的再扔吧。

所以，对大多数人来说，明白沉没成本到底是不是成本，根 本不重要，因为重要的是执行。很多人明白了，也不去执 行。

我在尽可能的要求自己从各个方面要求自己通过理解沉没成 本来学会放弃，因为很多时候，“不放弃”成了最大的成本。 2019-02-21(59 赞)
