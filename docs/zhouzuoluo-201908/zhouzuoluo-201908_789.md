# 《我在恨嫁的年纪，

简小姐 : 《我在恨嫁的年纪，报读了国际 MBA》

（2019.1.31）报名过去三天了，我也慢慢适应自己报读

了国际 MBA（打算出国留学一年）。

报名那几天心路跌宕起伏，报名后会怀疑自己是冲动决

定，但也慢慢坚定着。MBA、留学，这些之前只是想想 ，现 在认真地遇见，并与自己的未来关联。

留学读 MBA 一事刚在脑子里萌生并强烈，还是无法那么 坚定，问了好友们，电话姐姐，都很支持，这让我感到很意 外也很欢喜。

这是一个重要的决定，当我决定读国际 MBA 时，我知道 它将会影响我的所有。而这之前，我在想着未来几年的工作 怎么进行，我的生活会有哪些日常，期待遇见哪些人……然 而选择了后，再看之前的日记和备忘录，有些计划已不再是 计划，方向改变了，所有都大调整。

报名前几天考虑了很多，睡不着，半夜起来翻阅资料， 早上早早醒来继续查阅，下午完全没心情参加朋友聚会。也 会考虑年龄问题，二八快二九了，是想着三十前把自己嫁出 去的，缘分未到也时常焦虑着，终有原因吧；归根到底是想 向内求，工作多年，很多方面浅尝辄止，很想沉淀下自己； 当下正处职场生涯发展期，选择了国际方向业务，一年的留 学怎么做好衔接，思绪渐远……最后，报名机会在眼前，还 是不想错过，给自己人生一个机会吧。

报考过程，一波三折，但有很多感动。 准备材料阶段，顾问老师发来的准备材料清单中有毕业

时的成绩单，懵了，毕业时没有发什么成绩单，上哪儿找？ “联系院校的教务处老师导出，或去人力资源局……”现在学 校已经放假了，再说毕业后就没和老师联系过，六年多了， 老师们还在学校任教吗？我毕业后换了省份、城市，之前所 有的联系方式都没了，有的只是躺在 QQ 里的老师 Q 号了，不

太相信能解决问题，但还是 QQ 留言了专业课老师、班主任、 系主任 ……估计是因好久没联系了，久久没回复……后来专 业课老师终于回应，告诉我学校的教务系统换了两次，16 年 以前的信息都查不了了……已是晚十点，后来问到了班主任 的电话，打过去未接……自己百度了方法，试图找到所需， 然而深夜了也无果，这是第一次绝望。我来回度步，很疲 惫，难道就此错过了吗……

第二天上午，班主任联系上了，她告诉了我现任教务秘

书的电话，秘书老师爽快答应中午去学校帮忙导成绩单，果

然中午时发了我。一切奇迹般地实现着，犹如神助，无比感

动，感动于老师们的热心帮助，不管我们毕业多久，老师永

远在那里，服务着我们，祝老师和学校越来越好。

交费环节也是好事多磨，一定要这周末交学费，机构老

师周一就放假过年了。一切都太突然，手头只有几千元周

转， 尝试过问姐姐，无果；尝试过向老师申请缓期交费，也

无果……最后只能找爸妈了，一如平日视频， 半天憋说不出 正题，还是妈妈看出端倪，问我有什么心事，“emmm...妈， 我想向您借钱……”“做什么？”“读书……”“哦，多少?”……爸 妈很爽快的答应了，我说明紧急原由，正是中午，爸爸好像 没吃饭就去银行汇款了……我以为爸妈会问很多问题，以为 妈妈会唠叨为什么这阶段不是想着嫁人，都快三十了还读书 等等的，就是担心他们可能会这样，所以打算报名后再告诉

他们。然而他们并没有，妈妈轻松答应下，我知道她很难 过，我心疼地安扶她不要伤心不要失望，好想此刻能抱抱 她；爸爸对我总是很支持，默默付出，晚上和他视频，相视 而笑，就只是笑，我却笑出了眼泪，虽看不出，内心却也难 过无言，真的出国一年会很想爸妈呢，也不知道以后会发生 什么……许久后还是爸爸风趣地开始了话题……最敬天下父 母心，总是我们儿女的坚强后盾，此份恩情，无以回报。

终于在晚六点半时交完了学费，虽然材料环节还有坎

（由于姓氏多音，学信网的英文学历翻译件还要改 ……）， 但坚持过来了，就可以再坚持下去，遇到问题，再多些韧

性，终会好的。

报名的过程，顾问老师也帮了很大忙，周末晚十二点到

家，第二天要搬办公室，要来回跑，每每失落时总会鼓励

我。

生活中总会有一些大的选择，在一开始时超出我们的承

受范围，但选择了后，发现选择没有我们想象的不可逾越。

选择如此，困难也是。

之后的这几天，自己逛留学论坛，阅读留学生们的分

享，慢慢的不再一无所知，也慢慢的培养走出去的勇气；自

己也开始发奋上课，争取开学时能跟得上老师们的节奏，以

及同学们的步伐。

若心之所向，就勇敢选择；选择了，加油就好；带上信

念，一路向前。
