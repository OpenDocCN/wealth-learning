# 大家好我是吉水，这

吉水 : 大家好我是吉水，这是我的第 23 篇分享，希望大家多多 指教

《在不明真相的情况下，闭嘴真是美德呀》 前些天，刘润老师写了一篇文章，题目是《亲爱的自媒体朋

友，咱别写假新闻骗人了行不》。

在文章中，刘润老师经过实际考证，对一些自媒传播的虚假 新闻进行了还原。并呼吁自媒体朋友：要说真话，要对读者 负责，不要写假新闻骗人了。

对于自媒体写作者来说，在不做调查、不明真相的情况下， 不写文章忽悠人应该是基本的行业良心。

我觉得，对读者来说，这一点同样重要。在不明真相的情况 下，不喷、不传播是一种美德。

信息爆炸的时代，我们每天都会接触到各种各样的热点新闻 和分析文章。在这种背景下，很多读者根本不会思考，基本 上是看到什么就是什么。别人喷什么他跟着喷什么。

比如前段时间的“跳桥事件”一出，很多媒体就开始暗自揣 测，将矛头指向了男孩的母亲。于是，我们就看到那群把文 章浏览量刷到“10 万＋”的读者，也将矛头指向了男孩母亲。

他们从来没去思考过文章的真实性，没去了解过当时的情 况，更不明白最不希望这件事发生的人就是男孩的母亲。

再比如，当年刘翔在奥运会上因伤退赛，很多人喷他无能， 喷他丢人。但这些人却从来没去翻过刘翔的参赛数据。

他们不知道：刘翔是唯一一个在直线跑道上真正赢过黑人运 动员的人（这是连白人运动员都没做到的）；他们不知道刘 翔参加了 48 届世界大赛，拿了 36 次冠军，6 次亚君。

你若问他们：“什么都不知道你怎么就喷他？” 这些人一定会说：“别人都这样喷他的！” 多么可怕的一句“别人都这样喷他的！”。这些人随口就说出

来了，却不知道我们的运动员承受了怎样的压力和绝望。

网络高度发达的时代，我们很难要求每一个新闻报道者能够 做到实事求是。但我们可以从自身做起，保持理智和清醒， 辨别事实与虚假，在不明真相的情况下，不喷、不传播。

否则，我们和那些加工虚假新闻的人有什么区别呢？
