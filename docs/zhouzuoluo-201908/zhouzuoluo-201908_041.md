# 《有关心性》上星期

听听 : 《有关心性》

上星期部门来了一个新人。 观察了一个星期，早上晨会时候老大问我们新人情况如何。 负责带新人的导师说：目前观察还好，态度积极，就是有一

个缺点学习能力不是很强，上手慢，容易犯错。

老大继续问：有对他说明具体的工作要求和工作内容吗？ 导师：有的。 老大：对一个刚毕业的应届毕业生来说，新的环境和新的工

作内容，他不熟悉会出错这是可以理解和接受的。

这里面不应该全是新人的问题，作为他的导师，我们更要去

反省，是不是给他足够多的指导和帮助了。要从我们导师体

系上去看这个问题，并帮助他一起去找解决方案。

比如容易出错的工作是哪些，每次出错的原因是什么，和这

个工作相关的一个工作流程是不是有潜在问题......

我一旁听着，莫名想到了陈春花老师说的：磨练心性。 陈春花老师说过：对于管理者来说，我们最大的挑战是如何

管住自己心性，而不是怎么管别人。

这是什么意思呢？

比如我们新来的同事，学习能力接受能力不是很强，那么作 为他的上司，要接受他这一点，然后找到解决方案，而不应 只是“抱怨” 他的学习能力问题。

这就是磨练心性。 管理的本身是朝向自己，不是朝向别人。

在管理过程中，对自己提要求，不断磨练自己心性，才是管

理的本质。
