# 【她在心底种下一片

甜颜 : 【她在心底种下一片花海，日日收割香甜】 我干妈是个神奇的老太太。

你永远在她脸上看不到一丝一毫的急躁和张扬，

当她的生活徐徐在你面前展开，

你才会发现这个不动声色的女人，

在自己的心底种了一大片郁郁葱葱的花田，

每天都在收割，

自产自销的丰盈和馨香。

她的退休生活， 是精彩人生的另一程开启。

老太太骑着单车， 和一帮子年轻人一起环游法国， 骑行夏威夷。

去俄罗斯写生， 去东三省摄影， 去沙漠徒步。

打打太极，就受邀新加坡去表演。 剪剪窗花，就剪成了市级剪纸协会的会长。

我知道， 这时候就该有杠精会跳出来要讲了， “你别站着说话不腰疼， 那得有钱有闲好哇！”

其实， 活的充满乐趣，

并不需要天天行走在路上。

大多时候， 老太太浇灌心灵花田的营养， 惠而不费。

旅行时，她把好看的画面存进手机， 有时间了再把它们用油画颜料落在纸上。

她用 APP 为自己录音，讲冯骥才的作品，再配上自己画的 画。

她在阳台上种了六株玉米， 下午阳光最好的时候，泡一杯茶，去听它们拔节的声音，记 录它们成长的变化。

她开通了简书，嬉笑怒骂，隐秘地 书写自己的心情。

她在美篇上放了自己的旅行日记，看的人多了，甚至赚到打 赏。

大多数人坐公交车只是呆呆的坐着吧， 她会观察身边有趣的人， 在手机备忘录里为他们画一幅速写。

在 2018 年的最后一天， 骑上单车特意跑到城市的边缘， “我想和今年的落日告别。”

就是这样， 她把别人没时间看，看不到，不屑于看的点点滴滴，

统统都捡拾起来， 汇成一片灿烂的花海。

啊啊啊！好羡慕！ 好想给自己也建一片花海， 马上马上，立刻立刻！

站在十字路口， 我絮絮叨叨地和她诉说着我的焦躁， 全然不知已经走到车流前。

她有力地把我拽回一步，

“慢点”。

嗯…… 慢点。

今天的阳光那么好， 给心底的花晒晒太阳吧。

大家好，这是我进入星球的第三天，被粥佐罗写作课所鼓励 了，他说要动笔要动笔，所以今天鼓起勇气发了第一篇，不 晓得符不符合星球发文规范。 有任何建议欢迎给我提，谢谢。 希望在星球能认识更多同频的朋友[愉快]
