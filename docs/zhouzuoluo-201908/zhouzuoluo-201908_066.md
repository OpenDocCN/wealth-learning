# 《认知高级的人，都

《认知高级的人，都具有空杯心态》

[太阳]挑战日更分享第 2 天

猎豹移动的创始人傅盛曾经写过一系列关于认知升级的文 章，他把人按认知做了一个分级：

第一类人——不知道自己不知道 这类人以为自己什么都知道，往往自以为是，占了 95%；

第二类人——知道自己不知道 这类人相对高明一点，有敬畏之心，开始有空杯心态，并开 始丰富自己的认知，占了 4%；

第三类人——知道自己知道 这类人抓住了事情的本质和规律，提升了自己的认知；这类 人占了 0.9%；

第四类人——不知道自己知道 这类人永远保持空杯心态，达到了认知的最高境界，这种人 只占 1%，也就是说一千个人里才有一个人。

95%的人处于第一种状态，这也就是为什么碌碌无为的是大 多数。

[玫瑰]那么，什么是空杯心态呢？ 古时候一个佛学造诣很深的人，听说某个寺庙里有位德高望

重的老禅师，便去拜访。 老禅师的徒弟接待他时，他态度傲慢，心想：我是佛学造诣 很深的人，你算老几?

后来老禅师十分恭敬地接待了他，并为他沏茶。可在倒水 时，明明杯子已经满了，老禅师还不停地倒。

他不解地问：“大师，为什么杯子已经满了，还要往里倒?”大 师说：“是啊，既然已满了，干嘛还倒呢?”

禅师的意思是，既然你已经很有学问了，干嘛还要到我这里 求教?这就是“空杯心态”的起源。

智者往往虚怀若谷，愚蠢的人才会认为自己了解一切。 空杯心态象征的意义是，做事的前提是先要有好心态。如果

想学到更多学问，先要把自己想象成“一个空着的杯子”，而

不是骄傲自满。

当一只玻璃杯装满牛奶的时候，人们会说：这是牛奶； 当它装满醋的时候，人们会说这是醋； 当它装满茶水的时候，人们会说：这是茶。 只有当被子空置时，人们看到它，才会说：这是一只杯子。

装满东西的杯子很难装下其他东西，而一个空的杯子，可以 装水、装饮料、装任何能放的东西。空，代表着无限可能。

做人要有空杯心态，就要不断给自己清零。你只有不断删除 那些过时的知识和经验，才能不断接受新的东西。否则，你 内存有限的大脑和心灵会被一些无用的垃圾塞满而死机。

时刻保持空杯的心态，不给自己设限，才容易让自己有更多

的成长空间，持续开发出更好的自己。

愿你能做真正拥有空杯心态的那个极少数。[玫瑰] 2019-08-08(15 赞)
