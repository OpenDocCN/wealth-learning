# 《阿甘正传》带给我

牧心微笔记 : 《阿甘正传》带给我的新知:底层算法与智商并 非正相关

这几个月刷了几部经典电影，其中好多都是二刷，突然发 现，几年前看过的电影，现在再看，有了更多更深刻的感 悟。

其中让我印象最深刻的，当属《阿甘正传》。 我们都知道阿甘的智商只有 75 ，差点上了特殊学校，在他妈

妈的坚持下，让他进去了普通学校。他的一生不会算计，不

会寻求捷径，只是在做每一件他认准的事情。

上学的时候第一次看，除了感到阿甘的励志，其他感受并不 多，很大部分是在于，觉得一个智商只有 75 的人，和我一个 正常智商的人，并没有多少交集。

但现在却醍醐灌顶般的醒悟。有个词叫达克效应，意思是 说，愚蠢的人是发现不了自己的愚蠢的，因为发现自己的愚 蠢需要很高的智商。从这个理论来看的话，阿甘其实也挺聪 明的，至少他对自己的愚笨自知，而我们就不是。

我是最近才发现自己很蠢的，大概说明了在不断驯化自己脑 子的过程中，我的智商大概有了那么一丁点的提高，所以， 脑子一定要常用。

那么阿甘带给我的最大的震撼是什么呢？ 就是一种最简单的底层算法。和《肖申克的救赎》中的安迪

一样，他们都是在这种最简单的算法的基础之上，获得了成 功。

阿甘不一定知道有“算法”这个概念的，但他却是这么执行的

——认准目标，马上行动，持续发力。

阿甘是这样，安迪也是这样，如果你有心，会发现很多的成 功人士都是这样。

对于阿甘来说，目标来源于很多方面，比如妈妈期待他能在 部队好好干，最终他获得了全国性的荣誉，战友临死前一直 说想和他合伙开一个捕虾公司，他也去做了，收获财富，还 和另外一位上将共同投资了水果公司（应该是苹果集团）。 他突然想跑步，于是就跑遍了全美，直到突然停下来，说， 我累了。我想回家。

每一个阶段，阿甘都会聚焦不同的目标，并为了这个目标行 动和发力，而且是持续发力。

这个算法太酷了，因为他不需要多大智商，不需要我们有多 聪明，只要我们能够掌握这个逻辑并且去做就可以了。

现在我觉得，阿甘离我太近了，因为我原来曾经取得过的一 点点微不足道的小成绩，以前都觉得是因为自己有一点点聪 明，但是复盘起来却发现，所有这些自以为是的小聪明带来 的成功，用的全是最笨的办法，比如想考研就把专业书背个 五六遍，每天学习到晚上十一点，想面试成功就多刷题，多 说多练，说上几百道，充分准备。

之前有位大咖说，想出书，那就先读上 1000 本！ 这也遵循了这种底层的简单算法。

认准目标，马上行动，持续发力。

这就是阿甘正传带给我的新知，直到今天才想明白，看来我 是足够笨了。
