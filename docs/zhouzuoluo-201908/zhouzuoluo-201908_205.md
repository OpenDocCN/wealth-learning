# 【和优秀的人对标，

【和优秀的人对标，你才知道自己几斤几两】

上个月我去北京学习了一个课程，给老师做课程助理，搭档 是一个今年刚刚考入北大的研究生。

我去之前没做什么准备，甚至忘了带笔记本，老师和我们沟 通的时候，北大的她直接掏出了笔记本电脑，老师一边说， 她一边做记录，等到老师说完，她完整的拿出了要点记录， 而我只有脑海中的记忆。

同样的，在我们刚刚见面的时候，她很热情，上来给我一个 大大的拥抱，并且一直对我嘘寒问暖，主动找话题和我沟 通，生怕冷场。

在后面的沟通中，我才知道她本科就是中国政法大学的。我 以为本科的差距，只是我们学习成绩上的差别，但是当我真 正见到她的时候，我才知道我和优秀的人差的是良好的习惯 和积极的思维方式。

今天我去给一个刚刚高考完的学妹研究，报考专业，她问了 我一个很重要的问题：到底应该去自己踮着脚才能够到的学 校学习一个不喜欢的专业好呢还是去一个普通的学校学喜欢 的专业好呢？

我告诉她，如果有机会一定要去更好的环境，因为你看到的 是好的环境，背后其实是良好环境带给你的一系列正向影 响，你会耳濡目染的改变许多。

那你不怎么优秀的时候，一定要努力挤进一个优秀的圈子， 接触到更好的资源，拥有更开阔的视野，你才知道努力的方 向在哪里，这个事情越早明白越好。
