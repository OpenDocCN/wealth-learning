# 完美的未来不是未来

完美的未来不是未来

粥老师在谈完成比完美重要的时候举了一个自己的例子，他 去年开始健身的时候，规划得很完美。要有专业的鞋子、裤 子，要有健身包，花了很长时间准备。然后一开始就买了私 教课，请专业的教练陪练，结果练了三节课后就彻底放弃不 去了。

反而他今年随意开始健身，想几点去就几点去，想练啥就练 啥，减肥之路慢慢步入了正轨。

我觉得这也可以理解为完美的未来不是未来，怎么说？因为 这是按照理想的状态构建一个完美的未来，当实际执行的时 候很难实现，人很快就会选择放弃，重新规划新的未来。但 照样犯同样的错误，再加之错误归因，再三再四以后，时间 耗尽，最后失去了机会，人生和未来也都没了。

最后讲个我最近在推行的一个项目，在一个巡检机制的设计 上我和领导有了分歧。

我的想法是要求经理级以上的管理者参与进来，才能够使得 这个项目有效落地，产出实质性的成果。因为项目落地的关

键在于经理级以上管理者的重视，不仅体现在口头、会议、 邮件上，更要体现在他们的行动上。

领导说我的想法她能理解，也很认同，但是太完美了反而不 太可能实现，这点她是基于在公司三年的工作经验做出的判 断。

她建议我先设置基层管理者参与进来（这是我们可以管控的 层级），虽然不完美，效果也可能不太理想，但是当下抓住 时机先动起来，再依据未来的变化做调整修正，这才是未 来。
