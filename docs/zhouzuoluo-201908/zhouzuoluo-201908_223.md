# 《高小羊：积极主动

《高小羊：积极主动，乐观做事的重要性》

2 个月前，我新入职了一家公司，开启了写作和上班的新模 式。

这家公司试用期 3 个月，但是如果能提前完成业绩要求，可以 提前转正。我四月份在磨合期，五月份业绩达标，提前一个 月完成业绩，同时提拔我为组长。

和我同天入职的另外一个女生，眼见六月份也要快结束了， 还没达到业绩的一半，按照公司要求，如果三个月不达标， 会解除劳动合同，所以她很有可能待不到七月份。

我们都是在这行业做了有一段时间的，因为这家公司只招有 经验的，所以从年长判断，我们的能力应该差不多。

但是再后来的沟通中，我发现在一个行业做的同样的时间的 人，能力未必是一样的。

她的问题主要在于：产品推广手段不成系统，今天搞一下这 个方法，明天搞一下这个方法，而且对这些方法的理解也基 于表面，没逻辑分析能力和判断这个方法是否可执行的能 力。

这些都是要日常工作中刻意注重去训练。 她最致命的问题是个人心态，态度，思维的问题。

公司随机分配产品去推广，每个新来的人都是从零开始的， 大概每个人分配七款产品，我接收这些产品以来，从头至尾 去分析，然后列出一份推广清单，一步步做下去，把产品翻 新后，结果自然不错。

她对待产品时是悲观的，甚至说她这个人就是悲观的，总觉 得所有问题是对方的问题，自己的推广的问题，总有借口来 贵归咎于是产品的问题。

比如，给她分的七款产品，她能把每一款产品都数落的什么 也不是，什么质量差啊，市场不受欢迎啊，受众小啊等等。

总之，这些产品不能做，所以就给人说：推广不起来是有原 因的，因为这不是我的问题，是产品问题，市场问题。

总之，在她眼里，所有的产品都有问题。 带着这样的心态，就算是给你一个顶级产品，你也能找出这

个产品数不清的负面评价。

不乐观，爱找理由，认为什么都有问题，不相信，首先你从 心态上就已经给这个产品判了死刑，能推广起来才怪。

我发现这类人还挺多的，他们总是悲观想事，做事。比如， 我闺蜜就是这一类人。我告诉她说我开始给人写文章了，他 说对方是骗子吧？

我说去报个健身房吧，她说练了没用咋办？等等，很多案 例。

我们每一个人都是信念的产物，你相信什么，你就能看见什

么。

你不相信什么，你质疑它，你找它刺，你也就收获不到成 果。
