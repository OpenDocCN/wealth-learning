# 《高小羊：做一个为

《高小羊：做一个为自己带节奏的人》

前天收到一位读者消息，咨询我是如何保持自律的？ 我当时的回复是，一心一意想要变得比过去的自己更好。 的确，这是驱使我每日能够坚持运动，坚持学习，坚持写作

的动力。但变得比过去的自己更好，似乎不能回答这位读者

的问题，因为这句话听起来更像是保持自律后的结果。那驱

使我保持自律的原因是什么？也就是说，我为什么会自律？

这两天一直在脑海中搜寻答案，试图找到一个准确的词来回 答这个问题。昨天下午无意中翻到了作家连岳之前写的一篇 文章，我突然找到了答案：节奏。

与其说我在过一种自律的生活，倒不如说我在过一种有节奏 的生活。

连岳在文章里面是这样解释“节奏”二字：寻找节奏，就是用 自己想做的事引领自己，用这件事引领你的生活，你一定会 把这件事维护好，失去它，生活就会散架，成为毫无主题的 零件，如打乱的拼图，不知所云。

引领我生活的这件事，一开始是健身，后来多了读书和写作 这两项。我以前是非常懒散的，到现在也算不上一个非常自 律的人，因为我间接性会产生不想去健身，写作，读书，甚 至不想上班的念头。

但一般到最后我都会去做，因为我知道如果不去做，我今天 的生活就会散架，这些事就是车轮上的一条条轴，要是突然 断掉一条，那轮子也就不能安全的转动，所以它就会失去原 有的稳定频率。

所以我将这个逻辑总结为：节奏是原因，它促使一个人的生 活有规律并且自律，这是过程，然后自律的生活使得这个人 变得比过去的自己更好，这是结果。

这个逻辑的起源在于节奏，你得找到能够引领你生活的事 情，有时候可能不太好找，所以你需要有一个引子。

我的引子是健身，通过健身这条引子，引出了读书和写作， 这 3 条加上工作以及下厨，大概 5，6 件事组成了我每天的生 活。这就是我的节奏。

寻找引子时，这个引子最好是能够带给你痛苦，是反人性， 甚至是要你投入大量时间和精力，并且它不太容易达到终 点。

恰恰健身这件事是不容易的，我认为它涵盖了优秀的人所要 必备良好的品质，比如，忍受独孤，承受痛苦，克己自律， 不放弃，持续投入，主动学习等。

和每天下班挺尸的人相比，去健身房锻炼的人可以称之为普 通人中的佼佼者，因为他能把下班后的自己从床上拔起来， 从手机品屏幕上拉下来，扔进健身房，去忍受独孤，忍受大 汗淋漓，忍受无味的机械反复训练。

同时他也是一个有责任心的人，因为他关注自己健康，就是 对家人和自己负责，注重身材，就是对自己外在形象负责。

找到自己的引子后，你会发现每天起床的瞬间，格外的有动 力有期待，因为这些引子就是在充满压力的生活里，多出来 的能让你为之发狂，为之兴奋的事。

假设没有这些事情，我大概会过上另一种生活：下班回家磨 磨蹭蹭，洗澡，睡觉。而现在同样的时间内，我可以做到下 班后健身，洗澡，读书，写作，然后睡觉。

大下班后健身，写文章看书，这种节奏适合我，但未必适合 你。但道理一样，你只需找到能够引领你生活的节奏就好， 用它们来填充你工作之余的生活，这就是变得比过去的自己 更好的结果。

其实但凡需要跨越时间壁垒的，都是有节奏的事，找准一 个，坚持下去。比如我一个朋友每天在朋友圈打卡素描。

耐心，韧性，坚持，相信，持续，投入，都是自律的代名 词。
