# Day68《日更 1

《日更 100 天，我只为了达到这个小目标》

持续日更写了 60 多篇了，主要是为了整理自己的思路，保持 写东西的手感，同时希望能输出一些对大家有启发的干货， 不过没有刻意去收集反馈，这些文章对外的价值到底如何， 我的认知还是一片混沌。

在我看来，好文章没有固定的标准，但今后持续写东西，理 应给自己选择一个判断标准。除了阅读量、点赞量、互动反 馈这些，我一直在寻找一个标准，更加接近基本原则的东 西。

直到我看何帆在他的专栏讲写作时，提到了一个基础写作的 目标，我如获至宝。

这也成了我接下来写作的基本目标：用简单的话把一件小事 说清楚。

不装不端，不炫技不媚俗，力求简洁清晰。很赞。 关于如何说清楚，何帆给出了三招： 1．每一段的第一句话必须是这一段的中心思想；

2．每一段后面的几句话必须要有逻辑关系，不管是递进关 系、并列关系还是对比关系，总之一定要有逻辑关系；

3．上一段的第一句话和下一段的第一句话之间一定要有逻辑 关系，不管是递进关系、并列关系还是对比关系，总之一定 要有逻辑关系。

按照这个规则，文章的结构是非常清晰的，先搭起这个结构 然后再填内容就相对容易一些。

恰好今天在看一些写作方法的文章，下面拿“注意力写作”中 的用户思维来演绎下“何帆三招”。

注意力时代，要学会注意力写作。如今是个注意力稀缺的时 代，新媒体写作关键在于抓住读者注意力，对应的写作手法 可称之为“注意力写作”。

注意力写作首先要有用户思维，懂读者诉求。能够洞悉读者 的阅读诉求，去满足读者诉求，让他爽，你的文章才会有市 场。在写作之前，可以先问自己一个问题：别人为什么要看 你的文章？

不同读者的诉求千差万别，但有两个核心诉求是一样的，这 根植于底层的人性。一是需要满足好奇心；二是满足自我表 达的需要。我们不难发现，好的注意力写作，就是反复激发 读者好奇心，接着满足读者好奇心的过程，所以我们要学会 设计自己的文章，来制造好奇。

人性有一个共同的弱点，就是过多关注自己，而较少关注别 人。所以写作的时候，要考虑读者能否接受，而不是只顾着 自己写得爽。心理学上有个鸡尾酒效应，说在一个各种声音 混杂的鸡尾酒会上，无论现场多么吵闹，只要有人喊你的名 字，你往往都能听到。这给我们的启示就是，人往往更关注 自我，对熟悉的东西更敏感。

站在这个角度来看，对一个写作者来说，他能深刻理解多少 人，就会有多少超级读者。

注意力写作还需要有产品思维和社交思维，这个留着明后天 接着用“何帆三招”来说。

发现了没？哪怕何帆三招这个技巧我看懂了，但也没能比较 到位地演绎，尽管我很努力了。如果你能“用简单的话把一件 小事说清楚”，那你已经超过 90%的写作者。

对尚未做到的我来说，有意识地大量练习少不了，接下来的 日更方向和目标由此得来。
