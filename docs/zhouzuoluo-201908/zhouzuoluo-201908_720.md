# 《最不需要天赋的执

《最不需要天赋的执行力》

晚上好[玫瑰]今天是橙橙加入星球的 195 天，和大家分享关 于‘执行力’的话题」

最近被执行力相关的文章扎心了，我们总是嫌弃自己不够聪 明，不够勤快，不够气质动人，所以在公司里不受老板待 见，领导也总是没几个好脸色。

记得 14 年我在公司上班的时候，我的领导总是说我做事慢悠 悠的，我尝试过提高工作的速度，但是发现错漏百出，还不 如慢悠悠的稳。当时的脑子里没有“执行力”这种概念，所以 活该做最底层员工，拿最低的工资。

慢慢的接触一些领导和管理类的书籍会经常提到“执行力”特 别是一些职场类的公众号，昨天一篇推文《牛逼的人都有超 强的执行力》细读之后，对这个执行力越来越感兴趣。

一名普通的员工，如果没有什么特长，长得还一般般，又不 算特别聪明伶俐的，如果还没有一点拿的出来的“执行力”我 想在现在这个竞争激烈的社会很难混下去吧。

1.  01

    我所理解的执行力？

    首先，它并不是一个与生俱来的能力，所以并非天赋，它可

    以刻意练习培养出这种能力。

    其次，“执行力”不仅体现在工作上，生活也同样需要执行 力。

    用一句最通俗易懂的解释执行力：就是去做，并且还要做 好！

2.  02

弱爆的执行力带给我的困扰

案例一：这几天沈总出差，家里的碗和锅没有人洗，我是典 型的吃完饭拖延洗碗的人，不洗都堆在洗手盆里了，到最后 还得自己洗，并且放久了的碗更油不好洗。

案例二：每周五要更新一家珠宝公司的公众号，按理说我要 提前计划好，周四就开始准备素材，写好预览，周五检查几 次就可以推送了，但周四总喜欢推到周五，周五一忙起来基 本就忘了，耽误了准点推送的时辰。

案例三：昨天接了个排版任务，若白天能计划好时间抽一两 个小时立马完成任务，也不至于晚上熬夜到一点多才睡，因 为晚上有晚上要处理的事情，对于孕妇来说，这是很不负责 任的睡眠时间。

以上所有的事情都证明了我执行能力极其弱，并且给我带来 的无法挽回的拖延后果。

03

如何培养执行力？

第一步：直接做没有借口 不要给自己留退路，不要给自己找借口，今日事今日毕，并 且预早不预迟。

第二步：确立目标做好计划 有目标，有方向，做好计划也是执行力的重要组成部分，一 个具体可行的计划，是提升执行力的重要保证。

第三步：立即行动不拖延 明日复明日，明日何其多？有了借口，有了计划，重要的就 是立即行动。永远不要说，等什么什么以后，我就开始。

比如运动，当你说，等过完年，春天等夏天，那是没有意义 的，想运动想要健康，就立即让自己马上行动起来，这才是 执行力。

第四步：从身边小事做起 从小事着手，能看到成果，从小事做起，细节决定成败，不 断的培养自己的自信心，有助于提升执行力。

比如早睡，决定早睡了就狠心抛下手机，别拿到床上刷了， 刷着刷着又凌晨了，比如吃完饭就立马洗碗，别堆在厨房喂 蚊子。

哎！不说了，说多了都是泪，我要去洗碗了，不然明天一觉 醒来还得自己洗！
