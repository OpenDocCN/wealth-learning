# 大家好，这是我加入

大家好，这是我加入星球的第 128 次分享，请大家多多指教

《“鸡汤的嘴，骗人的鬼”背后，藏着什么不为人知的秘 密？》

粥老师新一期的分享写的特别有意思，他对我们习以为常的 鸡汤，提出了新的认知：

所谓鸡汤，就是包装那些你觉得你也可以的人和事。 鸡汤的害在于：不全面，讲的人选择什么说，什么不说；以

偏概全；有目的性的错误归因。

任何一个领域都有不少大咖，出来分享自己的成就时你羡慕 的不行，大家都讲牛逼的事迹，不讲苦逼的经历，讲容易的 事，不讲难的事，当然——你太苦了谁学，你太难了谁去 做。

但是，大家有没有深入的思考一下，为什么会有鸡汤文热销 这么多年的情况出现呢？

粥老师所说的“鸡汤的嘴，骗人的鬼”的背后，隐藏着什么不 为人知的秘密呢？

相信大家，应该都有过这样的经历吧，就是看几篇文章或者 演讲视频的时候呢，情绪高涨、热泪盈眶，好像里面每一句 话都说到了自己心坎里，简直就是遇到了人生导师啊。

然后呢？就发现没有然后了……看完也就看完了，对自己的 生活啊、工作啊没有任何积极影响。这是咋回事呢？

这事儿还得从我们的大脑说起。 人类大脑对外界的信息，会有两种处理模式，一种是“产生认

同”，另一种是“转变思维”。

这就导致了我们的精神需求，其实可以分成两种： 一种是来自母性世界的安抚 一种是来自父性世界的指引。

那么，看上去满满正能量的心灵鸡汤，是属于母性的安抚， 还是来自父性的指引呢？

显然，它应该属于母亲的安抚。为什么？ 因为所谓妈妈式的安抚，通常是，保护你的安全，让你不要

冒险，不必去解决问题，不必理会烦恼，别人不肯定你，妈

妈永远肯定你。

这种妈妈式的安抚通常有两种模式，第一种是阿 Q 模式，就是 像鲁迅笔下的阿 Q 一样，擅长自我安慰。

举个例子，有这样一篇短文，曾经在朋友圈疯转，说的是从 前有个亿万富翁，来到一个海岛度假，看到一个渔夫在晒太 阳，就问他为什么不勤奋工作，渔夫说，勤奋工作是为了什 么？

亿万富翁说，勤奋工作就有钱了，有了钱就可以买更多的 船，捕更多的鱼，赚更多的钱，然后就可以在假期时出来晒 太阳，渔夫说，我现在就可以晒太阳。

这就是一篇典型的鸡汤文，这篇文章强调的是亿万富翁多愚 蠢，这么辛苦，最后和穷渔夫得到的是同一个东西

但实际呢，虽然这时候亿万富翁想要的和渔夫是一样的，可 他拥有财富，他拥有财务自由，他拥有人生的选择权，穷渔 夫除了晒太阳，什么都没有。

但是这样的故事却会让 80%的人看起来很爽，因为按照二八 理论，我们社会上的 80%的人都是屌丝，在心态上和穷渔夫 没什么区别，所以在面对挫败的时候，我们都有被安抚的需 要，都喜欢看这种段子。

再举个例子，比如有一篇文章叫做《其实他只是不那么爱你 而已》

文章的主旨是，如果你们婚姻出问题，就是因为他不够爱你 而已，你没有错，都是别人的问题。所以呢，你最好在结婚 的时候，找一个非常爱你的人，只要对方非常爱你，你的婚 姻就有安全感了。

这种鸡汤的逻辑显然是不经推敲的，因为如果按照这个逻 辑，我们一辈子只需要吃一顿饱饭就可以了，只要吃得非常 非常饱，以后就不必吃了，这显然是以偏概全。

感情基础当然是婚姻稳定的一个因素，但如果你过分以偏概 全，那就会出现这样的逻辑：

如果他爱我，就会来找我道歉——哪怕是我的错——否则他

就是不爱我，既然，一个人不爱我，我为什么要付出呢？

这个逻辑的无敌之处就在于，出了问题就是你的错，我没问 题，我不需要做任何事情。

同样是火爆过朋友圈的文章：

《致某人：我凭什么要帮你？！》和《不是我太高调，而是 你玻璃心》

作者在文章里举了一大堆例子，然后用概括性的语言和标 签，进行评价，得出一些看起来好像很有道理的金句，

比如说“你弱你有理啊”、“我用正版的还要考虑你用盗版的心 情？”之类的，让你看了以后觉得酣畅淋漓啊，因为他说的那 些人让你觉得好像在哪见过。

再一想，你身边可能就有这种让你特别讨厌的人，你想对这 些人说点什么，但你没说出口。

现在突然有个人跳出来帮你说了，而且说得还比你好，那你 肯定就会毫不犹豫地分享到朋友圈。

大家这时可以回想一下，大学里的很多演讲，不管请谁来， 现场都座无虚席，台下总是掌声雷动，但回去以后，怎么着 也想不起来都说了啥有用的东西。

这些演讲里，经常能听到这种话：

“我去过全国几十所高校演讲，发现就我们这的学生最有秩 序！”

或者“谁说大学生不应该谈恋爱？我认为，大学不谈恋爱，人 生就不完整！”

这些话只要一说出来就特别受欢迎，因为说出了学生的心声 啊。

还有种模式呢，叫丑小鸭模式 这种文章通常是一种心态，就是其实你是个白富美，只是你

不知道而已

比如几乎所有的韩剧或者偶像剧，都是这么演的， 虽然你又 普通又平凡，但总会有一个或几个高富帅对你不离不弃

很多人也会相信，你只要做一点点儿小事，比如，你在公司 的地面上看到空水瓶捡起来，丢进了垃圾桶里，老板会在无 意间发现，然后你明天就可以因为这个举动受到赏识，面试 成功或者升职加薪。

还有一个最可怕的鸡汤段子，荼毒了不知道多少中国儿童乃 至中国胎儿，那就是不要让孩子输在起跑线上。

这种鸡汤逻辑就是：幸福就是流水线，你只要按标准做完了 一切，你就会有拥有好的成绩，人生就是赛跑，就是考试。

做落后者是一种很可怕的事情，只要你学习好了，你就会拥 有一切。

很多人都信了这个逻辑，到了中年的时候，爆发了中年危 机，或者出轨或者精神迷茫

很多人不明白，我有了房有了车，有了好伴侣和好孩子，还

有好事业，可为啥我就是觉得生命毫无意义？

很多女人说，所有人都认可我是一个好妻子，为什么他会找 一个那么差劲的小三？

因为人生幸福的指标太多了，并不是考试好或者事业成功或 者嫁个好男人，人生就一定会幸福。

你看，心灵鸡汤的根基，是来自母亲世界的安抚，并没有什 么用。

其实呢，当我们长大的时候，离开了妈妈为我们营造的温室 时，我们就进入了父亲的世界

父亲的世界代表的是外部世界的逻辑，也就是理性的真实世 界。

父亲的倾向是，我们要解决问题，不要做鸵鸟逃避问题。 比如说，曾经有个大学生问某个名人，他说， 我和我女朋友毕业留在北京工作，可我们俩买不起房子，现

在租一个房子住着

我们的朋友挺多，老叫我们出去吃饭，后来我们就不好意思 去了，因为老吃人家的饭，而我在北京的薪水很低，没钱请 人家吃饭。我感觉一无所有，你说我现在该如何是好？

这位名人号称心灵鸡汤界的教主，随口就回答说，

“第一，有多少同学想要留京没有留下啊，可是你留下了，你 还有一份正式工作

第二，你有相濡以沫的女朋友

第三，这么多朋友愿意请你吃饭，说明你人缘不错人品好 啊，凭什么说你一无所有呢？”

大学生一听，你这么一说，我还挺高兴的。 但这个回答就是一个典型的鸡汤式的回答，它不过是转移了

重点，没有解决问题。

如果是父亲的视角，则可能会思考，你为什么穷，是不是职 业规划有问题，比如他从事了一个自己不擅长的领域，或者 是不是理财出了问题，如何才能提升财商等等。

有人就曾说心灵鸡汤都是光给汤，不给勺子。 那么，父性世界思维就是一把勺子，可以指导现实世界解决

问题的方法

所以当下很多人“反鸡汤”，呼唤更多的“父性的力量”，正是因 为遵循规律，更理性、更讲逻辑，才会让我们认识到这个世 界不以我们的意志为转移。

心灵鸡汤是可以给我们，提供点情感慰藉和希望，但不能长 期服用，不然它会误导你脱离真实的世界。
