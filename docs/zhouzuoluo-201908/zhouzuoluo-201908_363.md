# 《这是我用过最有效

丹儿 : 《这是我用过最有效提高睡眠质量的方法》

晚上好 与星友们分享丹儿的第 38 篇原创 5.29

—————————————————————————

昨天听完《睡眠革命》，了解到睡眠是有睡眠周期的，今早 醒来，特意算了一下，正好睡了五个睡眠周期，感觉不错[偷 笑]

准备要睡觉了吗？还是想在玩一会手机，你是否已经失眠很 久了呢？不知道从什么时候开始，习惯了晚睡，习惯了睡前 看手机到眼睛干涩也不想睡…

“为什么还不睡呢？就是想在等等，等什么呢？不知道”[捂脸]

这句流行语总听到身边的人在说，虽然不知道等什么，但是 我知道产生这样想法不愿意睡的原因是什么！

★因为手机呀！手机屏幕的蓝光，会阻碍我们分泌褪黑素， 褪黑素是什么呢？就是帮助我们促进睡眠，产生困意的物 质。睡前一直看手机，即使再累也会让我们不想睡觉。所以 就有了前面那句话。

【关于睡眠问题我们可以分为三类人】 晚起星人：晚上不想睡，白天不想起。

晚上睡太晚，严重影响白天的精神状态，工作生活效率都不

高。

失眠星人：晚上想睡睡不着的人，吃安眠药也没有太大效 果。 这类人对生活的影响更严重，长期失眠对我们的身体损耗很 大，性情也会变得暴躁易怒。有什么比睡不着觉更让人郁闷 的呢？

早起星人：这类人早睡早起，有良好的生物钟，生活和工作 的效率都很高，也更容易获得幸福感。

★从上述几类人中可以看出，早起星人他们的睡眠质量是最 好的。因为他们作息规律，不会失眠星人那样睡不着，也不 会像晚起星人那样睡不够。

【那么，我们怎样能养成早起星人那样的作息习惯呢？】

★想想我们的祖先原始人，他们有失眠的问题吗？没有的， 大家知道原始人睡前会有怎样的活动吗？在影片里应该看到 过，一大堆人坐在一起，中间有篝火，暖色的，气氛很温 馨，大家听着老的原始人讲故事，慢慢篝火变小，气温慢慢 降下来，大家也都打着哈欠，回去睡觉，第二天也没有闹 钟，清晨第一缕阳光把他们唤醒，想想这场景多美好❤

★原始人的睡前环境就是这样的，所以我们想有好的睡眠， 需要一个舒适安全的睡眠环境。下面给大家讲一些改善睡眠 的小方法，希望能帮助到大家。本人已经试过，睡眠质量好 了之后，整个人都感觉精力充沛。

【晚上入睡前需要做些什么？】

①留出 90 分钟的时间，让你的节奏变慢，卧室的光线可以调

暗一点。

②不要看电子产品，它们会阻碍你分泌褪黑素。

③卧室和卫生间的灯不要选择白炽灯，要选择暖光灯，因为 白炽灯会让我们变得精神，和白天的日光很像。

④睡前三个小时吃完晚饭，因为晚上肠道是不蠕动的，如果 睡前一个小时吃饭，吃的东西就会顶在肠胃里，会影响我们 的睡眠。

⑤睡前少喝水，因为喝的多了半夜容易起夜，也会影响睡 眠。

⑥睡前检查门窗，电器，这样会增强你的安全感。

⑦拿出纸笔，写下今天已经做完的事，和未做完的事，把未 做完的事安排一个时间做。这样就可以没有顾虑的休息了。

⑧选择舒适的床上用品，床垫不要买太硬，也不要买太软， 并不是越贵越好！

作者就是卖床垫出身的，买床垫时可以侧 躺上去让你的朋友帮你拍照，如果后背和颈部成一条直线就 可以，身体凹陷进入说明床垫太软，身体全在外面说明床垫 太硬。都不利于睡眠。

床垫十年需要更换一次，因为会产生大量的螨虫，螨虫吃人 体的皮屑，它们并不会伤害我们，但是它们的粪便会伤害我 们，引起红斑过敏，同理床单也需要及时清洗。

⑨睡姿应选择侧卧，仰卧会让我们睡觉时用嘴呼吸，容易打

鼾，打鼾有一定的微笑，会容易闭气。趴卧，对我们颈椎腰 椎都有损害，所以睡姿侧卧是最好的。

那么往哪个方向侧卧呢？你擅长用那只手就朝她的相反方向 侧卧。因为让你的有利手在外面，会让你有安全感。

⑩睡前可以选择你认为有效的睡眠工具，比如听书，看书， 听音乐等。一切能让你觉得安全入睡的东西。也可以做一些 简单的拉抻瑜伽。

★以上就是通过睡前的准备提升睡眠质量的方法。但是想让 一天的状态都很好，光是晚上睡眠质量高也是不行的哦！早 起也需要仪式感[调皮]

【早起的仪式感，提升一天的状态】

①和晚上一样起床后也需要留出 90 分钟的时间，让自己慢慢

苏醒，迎接新的一天。(这九十分钟包括上班路上的时间)

②醒来以后第一件事拉开窗帘(不要马上看手机)因为手机上会 有一些信息，资讯，当你还没有清醒的时候回复一些东西会 影响一整天的状态。

③在床上做做拉伸，伸伸懒腰，侧躺让双手抱着双腿，把昨 晚的废气排除。然后就可以起床的，顺时针按按脸促进面部 的血液循环。一会变热，以后在揉揉耳朵，因为耳朵上的穴 位很多，最后拍打大椎穴，我们的身体就被唤醒成功啦。还 可以梳梳头，疏通头部经络。

④开始洗漱，吃早餐，去厕所排除昨晚的废物。

⑤去上班，这时候就可以看手机了，我们的头脑这时很清 醒，可以看看资讯，学习一些东西了。

★以上就是关于早上起床的准备工作，做好这些我们的一天 状态都会很好。你想想吃完早餐悠闲的听着书去上班，比着 急忙慌起来边走边吃早餐的感觉是不好很多。

【关于午睡】

①午睡有利于在白天帮我们调整状态，可以睡半个小时，也 可以睡一个周期，身体会帮助我们修复。

②午睡不论能不能睡着都可以平躺一会，因为我们白天和地 球总是垂直的关系，平躺以后就会和它平行，有利于我们身 体的恢复。

③怕只睡半个小时醒不过来怎么办？可以喝一杯咖啡，半个 小时之后有效的那种，睡了半个小时咖啡就来劲了。正好开 始工作。

★以上就是关于午睡，白天的状态保持良好，午睡也是很重 要的。

【对失眠患者想说的话】 很多失眠患者会服用安眠药，其实它只是能比正常人提前 22

分钟入睡，吃多了造成猝死的机率会比正常人高很多。

所以与其吃安眠药，还不如睡前留出 90 分钟的时间完成睡眠 仪式。

还有含咖啡因的东西也要少接触，它会让我们感到兴奋。

＊＊＊＊＊ end ＊＊＊＊＊

▼以上就是今天关于怎样通过培养好的睡眠习惯，提升睡眠 质量的方法。睡眠质量高了，人体得到很好的修复，工作效 率也会变高。养成良好的作者习惯我们的生活也会变得很美 好。

最重要的一点是睡前不要玩手机哈，作者做过一个调察，每 个人平均每天打开手机的次数是 80 次，这代表着我们做事时 会有 80 次被打断，它会严重影响我们的状态，总被打断我们 是无法集中注意力做一件事的。
