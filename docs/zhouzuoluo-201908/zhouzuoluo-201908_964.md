# [大家好，我是懂先

[大家好，我是懂先生。这是我的第六十四次分享，很开心遇 见这么优秀的你们。在星球的一年，我希望实现三个愿望， 如果有一天实现了，我会把我的故事说给你听。]

《工作三年还发错邮件，你被怼真的一点都不冤！》 看着别人毕业一年升职加薪，毕业三年月薪几万，毕业五年

买车买房。你以为随着工龄的增长，你也可以达到这样的高

度。可殊不知，让一个人值钱的，重来不是工作时间，而是

能力。

今天和单位领导发生了不快。起因是早上让我发个邮件，材 料弄好后我就按要求发了过去。可下班的时候领导给我打电 话斥责为什么邮件还没发，说实话他的语气让我很不爽。所 以我的态度也不好，告诉他邮件早上就发了，不相信可以查 看邮箱。

可领导看完后又给我打了电话，让我看看把邮件发到哪了。 此时我还在心理暗骂领导是猪脑子，连个邮件都看不明白。 可看过后我恨不得抽自己一巴掌，我的收信邮箱打错了，导 致很重要的一个材料没有及时报送。

虽然最近工作很忙，加上身体出了点状况，但这都不应该是 工作失误的借口。工作三年多，早已经不是新人，应该清楚 职场只看结果，不相信任何借口。

而今天犯了如此低级的错误，真的非常自责。倒不是因为造

成了多严重的后果，而是真的对自己失望，对工作三年还能 范这种低级错误的失望。领导怼我真的一点都不冤枉，我拿 着现在的薪水，过着现在的生活真的也不冤，这和我的能力 是完全匹配的。

好多刚毕业的大学生，面试时张嘴就要求底薪过万、周末双 休、节日福利、带薪休假…其实有追求确实是好事，但不结 合自身实际的盲目追求则是异想天开。HR 给你的工资取决于 你的能力，而不是你的想象。

不用总想着单位能给自己多少钱，先让自己努力值钱。等你 优秀了，肯定会有人愿意给你更多的钱。

金立手机前不久破产了，好多员工因此下岗。但其总裁卢伟 冰日前高调宣布加盟小米公司，还没来得及因为下岗失落， 就被下一家大公司盯上了。所以，优秀的人永远不会缺市 场，如果连一份糊口的工作都找不到。只能说明你还不够优 秀。

好多体制内的人离开体制后都大展拳脚，走上了人生巅峰， 我虽然羡慕，但不会冲动。离开体制变得牛逼的人在离开之 前就有了牛逼的能力，绝不是因为离开体制就牛逼了，而不 牛逼的人即使离开也不会突然变得牛逼。

现在的我连个邮件都能发错，即使离开体制也不见得能过得 更好，所以在见不到光的日子里，还是要积极蓄能。只有自 己强大了，才有选择的权利，即使将来继续选择留在体制， 那也是因为我主动选择的这个职业，而不是我离开就会失业 而不得不留在这里。希望有一天，我有那个能力。
