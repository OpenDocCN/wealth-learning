# 「人这一生长跑，赢

「人这一生长跑，赢在找到自己的“加油站”」

生命中真正的乐趣，是当你沉潜于某一事物，完全忘我的刹 那。它是一种内心的平静，已超越了贫穷，也超越了物质的 享受。

我忽然意识到，这些乐趣，不就是人这一生长跑里面的一个 个加油站吗？让你停下来休息，让你充满能量，让你继续向 前奔跑。

01 学习路上的加油站 昨天上午七点起床，万恶的周一需要处理繁多事情，中间没 多少时间休息，一整天里只喝了三四次水，下班回到家时接 近八点。疲惫不堪坐在沙发上，打开粥老师的课程加餐—— 直播间分享干货。

不听不知道，一听吓一跳。我发现自己听到一半的时候已经 慢慢恢复体力，本来晕沉的脑袋逐渐清晰起来，听完直播课 后在一个小时内高效完成作业。开始回想这个过程，便迫不 及待想记录下来。

如果有一项学习，能令你在身心俱疲时仍念念不忘，一旦进 入学习后便慢慢吸收能量而神采奕奕，恭喜你，已经找到自 己的学习加油站。

02 职场生涯的加油站 今天下午接待的客人令我感到一种由心而发的成就感。从见 面时的不苟言笑，到逐渐打开心扉说出自己的需求和顾虑，

再者需要我的帮助提供专业建议，甚至让我自然转换称呼 由“您”到“你”，离开时基本达成一致可以签订合同。这是我在 目前这份职业中找到的最大乐趣。

无论多忙，工作再多，一旦开始与客人聊天我就进入了忘我 状态。不再需要抱着手机回复信息，不再脑子里飞速运转先 完成哪些重要且紧急的事情，我乐意跟他们坐下来，慢条斯 理地说话，清晰明了地介绍产品，观察他们的表情和语气从 冷漠到热情，听他们的故事，再到对我的专业表示认可和信 任，最后诚挚的希望与我达成合作才握手结束。

有时一不小心谈到下班时间都过了一个小时，回去办公室要 继续做未完成的文件工作，等着第二天第三天客人的口头确 认即可马上汇报领导，心里还是美滋滋的。我想，这是工作 中的加油站。

03 人际交往的加油站 如今自己想要独处的时间较多，不过每两个月都会安排一天 去姐姐家。虽然单程需要坐公车一个多小时，一天花在路上 时间接近三小时，但一旦进了姐姐家的门，到离开之前我都 能一直沉浸在放松的、愉悦的氛围里面。

姐姐会做好吃的饭菜等待我，买很多种水果放在冰箱，午饭 过后可以盘腿坐在地上冲上功夫茶，不用刻意找话题却从未 停止过说话，虽然说着话却非常轻松，不用经过思考，不用 刻意斟酌，只需自由地接收对方的分享和温暖。

姐姐一直陪伴在我身边，从离开家里后四年大学和三年工 作，还有以后很长的路，她就是我的加油站、温暖的港湾。

杨绛先生说：有些人之所以不断成长，就绝对是有一种坚持 下去的力量。

人这一生长跑，不是拼爆发力，不是拼一蹴而就，赢在找到 自己的加油站，不断补充能量，不断重新出发。
