# 为了避免混日子，请

为了避免混日子，请坚持每日复盘

01

给你们讲几个我在网上看到的段子：

【老师，我也不知道怎么回事，最近的状态就是不好，什么 都不想做，连看连续剧、刷淘宝、玩游戏我都不想做。可只 是呆着的话，又感觉特别不爽，老师，你说我这是怎么了 啊！

老师，你能否帮我解决一个问题？这个事情我已经想了好久 了，但就是不知道怎么开始，怎么办啊？

你看哈，我开始做得挺好的，可中间被一个事情打断了，我 就没有再继续推进，再想启动也动不了了，这是怎么了？

老师，我觉得自己太失败了，做事情总是做不好，这也不 行、那也不行。我也知道这就是所谓的自我攻击、自己捅自 己刀子，但我就是感觉自己做不好啊……

】

这似乎是我们很多时候的缩影，想做，也知道该做，但是就 是躺床上没有动。雄心万丈，赖在床上。然后打开朋友圈， 秀一波“积极废人”的日常。

这个时候，我们发现自己卡住了，动不了了。 为什么卡住了？

如果我叫你，现在啥也别做，去把冰箱里的饮料给自己开一 罐来。我想你会完成的非常出色。

那我说，你给我+！@#￥%……&*，我就给你一万块。我想 你会非常愿意接受这个买卖。

但这时候，你发现，尽管你非常想完成这笔交易，但是你却 根本动不了吧。

你卡住了。原因是什么？ 原因是你根本不知道我要什么，当然你就没法做了啊，于是

你卡住了。

同样，当你化身为“积极废人”时，你内心明明不想这样，你 渴望想要改变！

但是你根本不知道该怎么办！ 你完全不知道，你当下应该解决的，具体的问题是什么！

02

那么如何解决“积极废人”？

每天坚持复盘。

为什么要复盘？ 复盘的价值在于：如实的记录，从中反省，并发现问题，达 到解决问题的目的。

其实成长就是不断的发现问题，解决问题的过程。

如何复盘？ 在一开始的时候，不用写太复杂，大概记录自己今天的时间

走向和产生的想法以及总结。

例子：

【

起床八点，

上班，下班，下午六点，

到家，晚上八点，

玩手机，打游戏，晚上十二点，

睡觉！

想法：新出的一款华为手机不错，可惜没钱买！ 突然想找对象，可是自己还不够优秀。

总结：本来想花几个小时来提升一下自己，但最后还是打游 戏了】

记录时间开销，总结你本来用于自我增值时间都去哪了。然 后，这就会促使你发现问题（老想打游戏，不想学习）。

知道了问题，你就会尝试解决问题。上百度，查知乎。 而你不断的发现新问题，然后解决问题的过程，其实就是你

逐渐走出舒适区，开始自我进步的过程。

而产生的想法，有时是你的欲望，有时也是你的胡思乱想， 这些都是你私有的宝贵精神财富。

这会让你不断的审视自己的欲望，接纳自己的欲望，承认自 己的真正的喜厌好恶，最终找到自己人生到底为什么而活。

只要你坚持下去，积极废人这个词，将慢慢离你远去。 03 说说我吧 可以看文末图片，我到现在也就坚持了 83 天而已（其中还有
