# 《焦虑是成长的开始

辉 : 《焦虑是成长的开始》

“焦虑”已经成为这个时代的标签。几乎每个人，都会觉得自 己好焦虑。

其实焦虑在每个时代都存在，而且它的存在是不增不减的。 只是每个时代对待焦虑心态不一样。

80 年代的时候，很多人面对的是如何解决温饱。他们对于焦 虑绝口不提，并不代表他们的没有焦虑，相反的他们的焦虑 感比起现在的人更加剧烈。只是他们更专注于如何生存。

在佛教说，一切烦恼只是瞬间的念想，接纳 ta 自然所有的烦恼 不再是烦恼。烦恼反而是智慧的修行。

回到“焦虑”现在很多企业家，创业者常说的一词，我们也跟 着焦虑。因为企业家，创业者，是我们向往崇拜的对象，所 以我们觉得他们说的特别对。“这是一个焦虑的时代”

这里不是说他们错了，因为现在时代还有一个标签”吐槽”， 在这个快节奏的生活下，吐槽成为了一种宣泄方式。这些成 功者和我们一样都凡人，也有着一普通人一样生活方式。

但是企业家们，创业者们说完焦虑之后，是接纳焦虑的。焦 虑让他们意识到危机的存在，责任重担，使他们更加专注， 更进一步的深度思考，如何面对当下。

焦虑成为这些人成长的动力，这是为什么他们越来越优秀， 企业也越做越好。

再看整个国家，整个社会都是在一种焦虑的状态下进步的。 经济越来越好，生活质量越来越高。

焦虑是每个时代必然存在的，也陪伴着我们创造一个更美好 的未来。

假设这个时代不存在焦虑，整个社会变成什么样？ 一个无忧无虑的生活，一个不用思考的人生，一个没有纷争

的国家。我们对未来还有什么可期待的……

想想这是多么恐怖的世界，一个世界只有一种声音，一个生 活只有一种状态，多么无聊的是事情。

回到生活中，我们一直努力工作。突然给自己放一个月长 假，会发现没有工作的忙碌和焦虑，心里总觉得空唠唠的， 并没有想象中的美好。

焦虑是我们无法避免的，才有生活的积极和成长，同样我也 主动选择了焦虑，缔造了我们不一样！

所以说生活需要不一样的体验，才叫生活；人生需要困境和 坎坷才活的精彩。
