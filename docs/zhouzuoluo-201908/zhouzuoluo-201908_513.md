# 一次心的第十五次思

一次心的第十五次思考分享

《996 并不重要，真正重要的是成长》 这两天 966 工作模式引起了人们很大的议论

一小部分原因是因为马云的影响力，真正的原因却是人们对

这个工作模式早有自己的看法。

有人赞成，就像马云所说的，能够 996 是种幸福，想要成长， 想要成功，就要付出比常人更多的努力

有人反对，因为觉得生活不应只有朝九晚五的工作，还应有 自由的时间，陪伴家人，精进自己。

我觉得每种制度的存在，都会有其存在的意义，但是再好的 制度，也不适用于每个公司、每个人

如果每个公司都能像阿里那样，把加班当做员工复盘、学 习、反省错误的时间，那很好，那就加班吧，反正员工回去 自主学习肯定不如在公司，有那么好的交流平台学习的效果 好。 但是有很多公司高举着：“没有长时间的工作，就没有成

长”的旗号来压榨员工的价值，甚至还流传着一种畸形的企业

文化：老板、高管还没走，员工就不能走、我很好奇，效率

的工作不应该比形式的工作更重要嘛？如果工作都做完了，

公司也没提供好的交流平台，帮助员工成长，那还留在公司

干嘛？假装很上进？还是留在公司玩蛇？还不如回去做对自

己有用的事情，陪陪家人。

如果每个人都觉得自己的工作效率很高，没必要加班加点

了，下班也会主动学习，精进自己，提升自己的竞争力。那 很好，那就回去吧，反正你都那么自律了，清楚自己该干什 么了，那在哪里都能学习，环境对你的影响不是很大了。 但很多人下班后，到处浪，浪完回家来个葛优躺，看着剧刷 着抖音，学习？复盘？别闹了好吧。 所以一个本来不错的工作模式，只是被不良的公司，不上进 的人玩坏罢了。

工作环境比钱更重要 很多人叫嚣着：加班可以，但是钱要给够 的确，阿里的福利很好，不会差加班费。但是并不是每家公 司都是阿里，规模那么大，资金那么充裕。 而且我觉得在前期的学习阶段，其实工资的高低不是那么的 重要，能够维持基本的生活，够钱买几本书就好了。相比于 工资，好的公司、好的老板、好的同事、好的工作氛围，才 是最重要的。我们需要的是沉下心来，努力工作，保持学 习，精进自己，自身的竞争力就会提升，钱就会作为附属 品、衍生品主动地钻进我们的口袋。

做一行，爱一行。既来之，则安之。 马云说：如果不热爱，八小时都嫌长。如果热爱，12 小时也 不算什么 的确，像我们以前上体育课，总觉得时间过得很快，感觉才 打了一会的球，怎么就下课了呢？而上政治课，我们就会很 痛苦，感觉度日如年，简直是煎熬啊。为什么？不就是因为 我们喜欢体育课嘛，每次体育课我们都身心愉悦，恨不得时 间慢一点，再慢一点，一分钟变成五分钟才好。而政治课我 们不喜欢，多呆一秒都感觉是煎熬。 所以我们既然选择了这一行，就不要抱怨与后悔，尝试着去 热爱我们自己的选择。才不会觉得上班很痛苦，才不会有过 多的抱怨。 选择了这一行，选择了出人头地的道理，就不要抱怨辛苦，

抱怨自己付出那么大的代价。

如果真的不喜欢、不热爱当前的工作，那就做出改变呗。改 变，从来都不会晚。改变，只是需要一点勇气，一点决心， 一点相信。 很多人会说生活所迫，无法脱离当前的岗位。做出改变，前 期的确会很辛苦，但其实任何人都不是轻轻松松的就能成功 的，粥老师早年出来工作时，住在巴掌大的房子里，不一样 坚持自己的梦想，努力生长走到今天嘛。不热爱当前的工 作，想改变，却没改变的决心和勇气，是很可悲的。你不努 力，找的所有理由都是借口。

所以我觉得 996 的工作形式，不是公司与员工的矛盾，而是公 司、是员工自身的矛盾。自身没做好，就不要怪制度的不 好。并没没有最好的制度，只有最合适自己的制度，保持生 长，精进自己。

马云说：不为 996 辩解，但像奋斗者致敬。 世界永远不变的，就是一直在变。 最有价值的投资，永远是投资自己。
