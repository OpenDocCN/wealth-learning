# 大芳的第 36 次分享

大芳的第 36 次分享

《为什么你的文章越写越烂了》

最近一段时间状态不是那么美好，每天都熬夜到很晚，在星 球的分享也不够用心，到今天已经完全不知道该写点什么 了，那就索性做个反思吧。

今天是我的第 36 次分享了，要说没有收获肯定是不可能的， 但远远低于我的预期。究其原因，还是没有踏出舒适区。

我写过的分享基本在“新认知思考”和“我在看什么”两个主题范 围内，一个月的时间了，除了前天的一篇影评，其余全部在 属于我的舒适范围内。

文章写成了流水账，只是进行自己想法的一个表达，既不去 搜索相关素材填充文章，也不曾引用名人名句证明观点，这 样的文字写起来很快，并且很爽，不自觉就陷入自嗨状态。

因此写了一个月了，竟然感觉不到明显的进步，甚至有越来 越差的趋势，就像今天这一篇，依然属于上述情况。

我陷入了自我欺骗的陷阱里，以为自己每天坚持写就很棒， 其实《刻意练习》早就告诉过我们，“一万小时定律”是没用 的。

不知道有没有朋友跟我有类似的情况，如果有，可以再往下

翻翻，如果没有，可能下面的内容不能给你提供什么价值 了，抱歉。

那我该怎么挣脱出这个舒适区呢？我今天认真的考虑了一 下，然后给自己立了以下几个 flag，希望可以对相似情况的朋 友有点启发。

1.每周必须写一篇影评，作为对新形式的挑战，这是在形式上 首先跳出舒适区。

2.观察我自己的更新时间，大多数都在零点以后，这在一定程 度上助长了我的拖延症。熬夜真的不是个好习惯，在这里劝 一下跟我一样的朋友们。 所以从下次分享开始，必须在零点以前完成。请看到的朋友 们监督我[嘿哈][嘿哈]

3.每周写一篇财经周报（附有自己分析），挑选五到十条资讯 进行分析以及分享。这一点很重要，我要多说明一下。

①由于我本人对这些比较感兴趣一点，工作原因也更关注这 一方面，因此写这个，可以加深对行业资讯的理解，还可以 进行信息整合能力的锻炼；

②训练自己归纳总结的能力，还有深入思考的能力。财经资 讯是非常繁杂的，要从中找到比较重要比较有意思的，本来 就很锻炼归纳总结的能力，并且要针对列出来的资讯写自己 的理解，对深入思考算是一个强制练习了；

③最重要的一点是，星球内大家对于这一块涉及的比较少， 那么我写这些就比较有优越感一些，也可能比较吸睛一点，

从这个角度出发，写财经周报对我自己有很强的鼓励效果。 我认为优越感带来的自信真的特别重要。

其实我觉得这一条适用性特别强，不管哪个行业都会关注相 应的新闻动态，但由于自己还没有实施，以上效果目前也只 在分析阶段，所以等我坚持一段时间再跟大家分享它的效果 吧。

4.以后的每篇文章至少引用一句名言或者一个名人事迹。这个 算是最低标准了，我对自己还是太心软～

目前想到的就是这些了，如果有更好的建议评论区见～ 又是一篇没有价值的分享，刷到的朋友见谅[抱拳][抱拳] 爱生活，爱自己，爱你～

我是大芳，感恩遇见～
