# 《最好的自己应该是

《最好的自己应该是什么样子的？》

晚上好 与星友们分享丹儿的第 91 篇原创 7.26

————————————————————————

今天学习了怎样培养自己坚毅的品质，就从写一篇最好的自 己应该是什么样的开始！为什么要写这样一篇文章呢？其实 是在寻找内心的驱动力！帮助我坚持下去的动力！

一，首先思考了一下最好的自己应该是什么样的？

①当一个幸福的人 这个幸福并不是我要完成什么目标我才会变得幸福，而是无

论处于什么环境下，我都能感到幸福。

曾经看到过一句话“大部分人并不是因为成功而变得幸福，而 是因为幸福变得成功。”

②做一个谦卑的人

为什么要做一个谦卑的人呢？因为谦卑可以培养我的感恩之 心，当无论遇到什么事都充满感恩之心时，就会变得幸福很 多。

③成为一个有毅力的人。 就是做事能坚持下去，而不是三天打鱼，两天晒网。

④做一个有耐心的人

一件事能够做好，有耐心是非常重要的，我觉得我有点浮 躁，缺乏耐心。什么事儿都差不多得了，自己能看的过去就 行。

二，那么我应该怎样做呢？

①这三个品质想要达到都不是那么简单的，需要养成良好的 习惯。

而好的习惯养成刚好昨天学习过，就是先从每天完成 1％开 始。

②当一个幸福的人，需要养成什么习惯呢？

这个可以和谦卑一起说。因为要变成一个谦卑的人更能感觉 到幸福。

★停止做那些让自己感觉骄傲，让自己感觉比别人强的想 法。

具体的方案，可以在于别人交谈时注意倾听，避免总是想谈 自己。刚开始的时候会很难，就从与别人聊天时头五句话不 谈自己开始。

★找找批评，看看哪里还有需要改进的地方 可以让好友给一点批评意见，好久每人批评我了，总是盲目

自信。

★写一封感谢信给帮助过我的人。 那些帮助过我，让我改变的人有很多，最想给领导写一封，

因为是她让我的人生轨迹发生了改变，帮助我确立了人生的

方向。

★培养积极情绪

每天记录帮助别人做过的三件好事。

今天坐地铁帮一位宝妈推了婴儿车，（平时只会帮她按一下 电梯门）

还帮助一位不认路的路人查了导航，帮助她确认了路线，

（平时不知道的路只会礼貌的说我也不清楚你问问别人吧，

今天我特意不闲麻烦，问了她要去的地方帮忙查了导航。）

看到别人扔垃圾没扔进去就走了，我把垃圾捡起来扔进了垃 圾桶。（平时看到这样的情况，也因为懒当没看见。）

虽然都是举手之劳的小事，但是平常嫌麻烦也不愿意去做， 做了之后发现，也浪费不了多少时间，而且做了好事之后心 情一整天都很好。

还发生了一些好事在我的身上。所以决定把这个习惯坚持下 去。能做到的事情就不要嫌麻烦。

③成为一个有毅力的人需要养成什么样的习惯呢？

★每天做三件难事 做三件平时不愿意做，不愿意动脑的事，长时间的学会克服

困难，毅力自然就提升了。

★坚持每天记录

做过的事情都记录下来，是第多少天，多少件事。用数字的 形式记录，更容易坚持下来。

★收集有毅力的人的故事 讲给朋友听，也可以讲给自己听，因为真实的事迹会刺激我

也想成为那样的人。

★寻找有毅力的小伙伴 多和有毅力的小伙伴接触，自己也会受到影响。

④如何养成有耐心的习惯

★排队

★洗碗

★延迟决策

★欣赏艺术品（这里记录一本书《洞察》）

＊＊＊＊＊＊＊＊ end ＊＊＊＊＊＊＊

▼想要成为我心目中最好的自己，并不容易，但我打算从每 天微小的习惯开始慢慢改变，我相信一个好习惯的养成会有 连锁反应，就像我开始每天写作，我就要每天学习，每天学 习我的思维就在改变，思维改变了行动就会发生改变。

我只是养成了一个好的习惯就带来了这一连串好的习惯，所 以我相信日积月累想要变成我心目中最好的自己也并不是什 么难事！

以前也没想过能日更九十多篇文章，得到 600 多块钱的打赏， 不还是做到了。所以说人的潜能真是无限。只要我们愿意去 改变自己。哪怕是每天改变一点点。

自己看完写的这篇文章都感觉充满了力量，我得马上行动起 来，做点什么了，星友们要不要写一篇？
