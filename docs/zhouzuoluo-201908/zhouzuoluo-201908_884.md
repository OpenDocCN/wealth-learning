# 《年终奖最重要的不

《年终奖最重要的不是讲格局，而是讲信用》

最近越来越多写年终奖的文章，大多说“年终奖见一家公司的 格局”，到底什么是格局？

关于格局，有人说眼光长远，有人说懂得待员工好，有的人 说大度大方，每个人的理解都不一样，所以很难有一个统一 的标准。

而且，“讲格局”是对企业的单向要求，“讲信用”是双向的，任 何一方破坏，损失的都是双方法的利益。

市场经济中，只有双方互相约束互相受益，才能长久进行下 去，所以，讲信用是比格局更加一条基础的要求。

在职场中，企业常常希望员工要“靠谱”，比较普遍被认可的 说法就是“凡事有交代，事件件有着落，事事有回音”，讲的 是一种契约精神，降低企业和员工之间的信用成本。

人是很复杂的，所以契约很重要。白纸黑字签名的契约，靠 法律去维护，那么口头约定，就需要靠“信用”去维系。

讲信用不仅仅关乎于道德层面，更是对法律文契的一种补 充。比如说年终奖，一般合同里会说明“工作满****，员工享 受公司年终奖”，由于公司经营的状况未可预知，所以一般不 会规定发多少。

这里企业可能会提前口头承诺，给一个大概的预期数字。法 律留出的一些空间，处理得当，其实对双发都大有裨益。

比如说承诺给员工 10 万左右的年终奖，企业真正做到了，会 让员工觉得可靠可信，更加愿意踏踏实实跟着干。

反过来，企业也就有权威去要求员工，按时按量完成任务。
