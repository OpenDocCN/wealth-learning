# 《向工作半年工资就

小玩子 : 《向工作半年工资就多个 0 的 95 后取经》

白岩松说：“一个人的战争，这种时候你的内心已经兵荒马乱 天翻地覆了，可是在别人看来你只是比别人沉默了一点，没 人会觉得奇怪。这种战争，注定单枪匹马。”

昨天写了 2019 年目标和上半年计划。 发给给朋友看。

朋友马上给我推荐了一个已经在上海开过民宿的 95 后女孩。

忙碌的周一，和 95 后微信没聊上几句，约好见面再聊。翻了 她的朋友圈，说不出的兴奋，那种你要做的事情，有个学习 对象更形象具体化的出现在你身边，并且发现自己成长空间 真是无限大，这个世界太有趣了。

这里也借用 95 后妹子一句话告诫自己：对于任何人，我会欣 赏、会称赞，但不会崇拜。对于自己，不论发生什么，我也 不以现在的见识去定义未来的自己。

记得第一个鼓励我的星友@昂扬的羊 前几天加了我微信，我非常感谢他的鼓励。 他说：还是你自己坚持下来了，谢谢你自己。

感谢自己到了上海。 感谢自己积极主动。 感谢自己一直坚持。

现在能体会到连岳说的：行动着的人会发现，行动会引来机 会与资源。而悲观将杀死行动，最终也令你失去机会与资

源，甚至机会来找你，你都没有能力把握。

未来无限可能。 我很期待。

——连续第 34 天的小玩子 2019-01-15(22 赞)
