# 《来到知识星球，不

《来到知识星球，不是为了被点赞》

从在知识星球发表的第一篇文章获得第一个赞开始，我发现 自己每发一篇文章，就会频繁地查看是否有新的点赞。

迄今为止，我在知识星球一共发了 10 篇文章，其中有 3 篇被选 为精华，7 篇获得了打赏。星球有近 5000 的星友，每天发表的 文章少则几十多则上百，综合这样的背景来看，自己写的至 少还不算差。

可是，为什么我还是会那么频繁地查看看是否有人点赞呢？ 别人随手的一个赞，对我来说真的那么重要吗？这些到底是 不是我真正在乎的东西？我在乎的到底是什么？

点赞是一种认可。对于一个全职妈妈来说，在这样一个比生 活圈子广阔的多的平台中获得认可，是件很重要的事，是件 非常能体现价值感的事。然而，我之所以感到不安，是隐隐 感觉自己对被认可这件事给予了太多的关注，而这本不是我 来到知识星球的目的。就像走入了一座美丽的花园，被缤纷 多彩的花朵吸引了注意力，而忘了自己的目的是要穿过这座 花园，走进一片更广袤的森林。

于是，我静下心来，细细地整理好思绪，厘清自己来到知识 星球的目的：我来到这里，是为了从众多星友的分享中获得 更多认知、思考以及对新事物的关注，同时分享出自己的思 考。

如此看来，所谓的被点赞与被打赏，不过是这个重要过程中 的副产品。正确对待地态度应该是：点赞数多，证明选题的 关注度高或者视角新颖，但肯定仍有不足，无需沾沾自喜； 点赞数少，不一定是文章写的不好，也可能是选题关注度不 高，但是自己想写的，肯定是有感而发，是对自己来说重要 的事，所以也不必垂头丧气。

感谢第一性原理，让我现在每次迷茫的时候，懂得回到原点 去看问题，生活也因此轻松很多。
