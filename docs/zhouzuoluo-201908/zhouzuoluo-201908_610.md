# 《改变现状的动力越

《改变现状的动力越大，代表你对现状越不满足！》

今天刚读了粥老师的逆袭者不矫情：那些悄悄变厉害的人， 都在偷偷对自己下狠手。有以下几点思考，希望有一个能够 点击中大家。

1.当你想改变的时候什么都是动力 就来说说最近最火的电视剧《都挺好》吧，明玉出生在一个

重男轻女的家庭里，哥哥们什么都是好的，大哥二哥房间里

都有电脑，明玉没有，吃饭的时候妈妈给大哥二哥鸡腿吃，

明玉没有，大哥二哥上学，家里卖房支持，明玉上学不能跟

随自己心愿考清华，只能上免费的师范学校，明玉在家里没

有享受到公平对待，有人说，厚积薄发往往是最厉害的。

明玉在 18 岁的时候跟家里闹翻了，从此自己打工养活自己， 虽然一切有利的条件她都没有，但是明玉有一颗想改变现状 的心，她知道，只能靠自己，即使没有好的条件，但是明玉 一直在努力改变，因为明玉知道想摆脱原生家庭的残害，有 一个好的办法就是，你变得优秀，来冲击这些不好的过往， 而不是被打败，明玉就是这么做的，当然她成功了。

经常有人一边抱怨，又不想改变，要我说，你还没被逼到绝 境，当你在最低的时候，无路可退的时候，你只能往前走， 而不是矫情的想着想哪，本质就是你不想改变，只是在发牢 骚。那怎么能叫你想改变而没有办法呢？

2.危机感有多重，你就会有多努力

危机感对职场女人尤其不厚道，我们都知道，现在已婚未 育，已婚一胎，适婚年龄，的女性求职很艰难，相比未婚刚 毕业的小姑娘来说，毕竟刚毕业的年轻小姑娘很多，不可替 代的工作有很少，我们今天不深入聊这些。

前几天孙俪在微博上说，他的助理，刚生完孩子就开始给她 发工作安排，抛出一张助理在病房里吃东西的照片，孙俪 说，自己并没有给助理这么大的压力。微博低下的评论说， 要不要这么拼，职场妈妈太了不起了。其实那都是危机感造 成的，同事说，我现在真的害怕领导找我谈话，我知道我自 己因为孩子请了不少假，但是我也是没办法，为了生计，我 不能辞职在家带孩子，我说，姐你工作都带回家做，那么努 力，不至于的，同事说，我是担心我再不努力，我就真的要 回家带孩子了。

当我们自身条件不够好的时候，我们为了变好，拼命的努 力，因为只有这一条路可以走，当我们因为某些原因没有竞 争力的时候，我们也只有拼命努力，才不会被淘汰。

所以，你想要什么，自己想清楚，遵循内心。
