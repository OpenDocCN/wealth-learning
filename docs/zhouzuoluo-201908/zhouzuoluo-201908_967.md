# 2019 年，我定义

真正爱家庭教育 : 2019 年，我定义为“做自己年”。 什么是做自己呢，我们很多时候在为生活奔波，为别人而忙

碌，独独忘了自己。所以时常地要抱一抱帅气/美丽的自己。 这不是自恋，而是左手握右手的鼓励。

2019 年，我决定只做三件事。 一、打造一颗外部大脑。

碎片化时间和碎片化的信息让我们正处在一个快速割裂同时 又快速整合的世界之中，时间、注意力被互联网切成一块一 块，甚至小到 10 秒一段。

想要更好地适应这个社会，我们该怎么做？把自己也割成碎 片化吗？

不，相信那样，某疯人院会多一个病友。 我的选择是打造一颗外部大脑，把碎片化的信息经过初步处

理后，存储在外部大脑里，把碎片化的注意力拿回来做最重

要的事情。当你想要信息的时候，就去外部大脑提取。

那么如何打造呢——利用工具（印象笔记、滴答清单、搜索 引擎）去记录和总结日常的事物。

如果把人脑比作电脑，就是：把【存储】的功能交给工具； 把【CPU】的功能保留给自己的大脑。

这样大脑就不需要费力去记忆几分钟度娘就能搞定的一大堆 资料。

如每日的工作任务按照紧急重要象限法排成 1.2.3.4...记在清单 里，然后设置闹钟，到点了就开始，一开始就全心全意地完 成这一件事，有必要的话，可以用番茄工作法提升专注力。

二、让有趣的事情更有意义，让有意义的事情更有趣。 如何定义【有趣的事情】？简单来说，你做了之后会有很愉

悦的体验。别人了解了大概率会开怀一笑。

那么【有趣的事情】一定有意义吗？ 答案是：不一定。

比如说，今天你坐在街边，看人来人往，想象在这些人曾经 发生的故事，猜测他们来自哪儿，又去哪儿。这事我曾经干 过，而且是看了一整天。

那你说有意义吗？ 相比你今天花时间去健身房，去图书馆看书来说，并没有很

大的意义。所以这句话的前半句是【让有趣的事情更有意

义】。

2019 年我会选择怎么做？ 我会在路边看人来人往的同时，用笔记录下自己在他们身上

的想象，甚至我会主动去搭讪，尝试了解他们的故事。

发现让有趣的事情更有意义的方法了吗？就是去主动发生故 事，而不是被动地看。不仅仅是【输入】，更要有【输 出】。

后半句【让有意义的事情更有趣】，有意义的事情很多，继 续拿上面第二个例子——去健身房：去健身对身体好，如果 每天都坚持去，更有意义。

但是这件事情有趣吗？ 不，至少对我来说是不有趣。

为什么？因为平时都很忙，一想到健身房，就想到流汗，一 想到流汗，就想到洗澡，继而想到洗衣服...这得花掉多少时间 啊，我还不如舒舒服服地窝在沙发里看看书。在此，我自爆 个黑历史：2017 年在某健身房办了一张会员，至今没有去 过。这也是现在健身房持续盈利的原因：每天坚持的人很 少。

那么如何让有意义的事情更有趣？

【梦想】。有句话说，有梦想的人谁都了不起。 如果你想要成为一个健美先生，然后整天想象自己站在世界

舞台上闪闪发光。每天想每天想，我相信不出 3 天，你就会出 现在健身房里。

再说个例子：你正在追求一个妹子，同时呢，这个妹子喜欢 有胸肌、有人鱼线的男人、恰好你啥都没有。你会怎么做？

——每天跑 2000 米、卧推加 10 组。

你看这个过程中也有梦想的动力，想恋爱啊，想象未来的美 好啊。

恋爱的人是特别奇怪的一群人，在荷尔蒙的催动下，他们会 去做一些平时不愿意做的事情，而且还乐此不彼。

综上所述，做自己的第二条，尽量让有趣的事情变得更有意 义；让有意义的事情更有趣，乐意去做。

第三点 ：演讲 罗胖曾经讲过：当下的社会最重要的能力是表达能力——第

一写作，第二演讲。其实两者逻辑框架是一致的。真正的演

讲者同时也是一个出色的作者。

演讲不是讲给别人听的吗？为什么也算“做自己”呢？ 我的导师山立老师曾经说过：我讲我所讲，我听我所听，我

讲给自己听。事实上也是这样，一直引领着全公司的人前进

就是山立老师，因为他几乎全年都在舞台上讲。把自己讲空

了，就装新的知识，然后继续讲。不断地去讲的过程就是不

断提炼的过程，不断输出的过程。

不知道你们有没有类似的体验，在我工作过程中，我发现很 多学员第一次上台去演讲，哆哆嗦嗦，结结巴巴，一个观点 反复来反复去就是讲不清楚。

但是呢，他一次次地上去，最后第五次的时候，可能身体还

是紧张地颤抖，但语言流畅多了，逻辑也清晰多了。

这就是不断强化的过程，也是刻意练习的结果。所以 2019 年，我要主动地去刻意练习：演讲的能力。写作也一样。

以上就是我 2019 年要做的三件事了。如果全部完成，想想还 是有点小激动呢。最后以罗胖的跨年演讲的一句话作为结 束，送给大家：对未来最大的慷慨，是把一切献给现在。
