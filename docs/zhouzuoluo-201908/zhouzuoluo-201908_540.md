# Day138《影响

《影响力法则背后，归根结底是这一个关键词》

在今天《说服》一书的阅读中，作者围绕“承诺和一致性”这 条经典影响力法则阐述了具体的多个应用场景，在其背后， 我挖掘出了一个关键词：确定性。

1、承诺，本质上就是提供确定性 大家对承诺的预期是什么？当然是兑现承诺，不然就是失信

于人。一次次兑现承诺，才能建立起信任。无论是企业还是

个人，无论是工作还是生活，这一点都是相通的。

要承诺，要发誓，是要求对方给予自己确定性；主动承诺， 是给别人提供一种确定性。人是生活在不确定性的世界里 的，人的基因也承载了对确定性的先天性依赖。确定性是人 普遍的刚需。

2、一致性，是确定性的一种表现 研究发现，随着年纪增长，人会越来越看重一致性。其实不

止个人如此，公司也是如此。所以，你会发现人越年长越倾

向于保守，职位越高的人越保守，公司越大越保守。

这里的保守是指“一致性”的表现程度和对“确定性”的依赖程 度。在大多数情况下这是一种美德，如果一个人反复无常、

说风就是雨，那么不难想象跟这样的人打交道有多糟心。

而且在工作中，老板最看重的素质往往也是员工是否能够“交 付确定性”。确定性背后是信任的持续性和稳定性，当老板有 重要任务时，他不一定交给能力最强的，他往往交给确定性 最强的。因为越重要的任务，不出错越是第一要求。

我在服务客户的过程中，也越来越理解到持续提供“确定 性”是多么的重要。

3、确定性，是影响力武器中核武器 确定性，意味着信任和依赖，是给予别人安全感和舒适感。

你能持续给别人确定性，你当然越来越有影响力。

那些知名品牌就意味着确定性，个人品牌同样如此。职场 中，生活中，家庭中，对老板或同事，对朋友或客户，对家 人或孩子，都是如此。

想要建立和不断扩大影响力，那就持续地给这个世界提供确 定性吧。
