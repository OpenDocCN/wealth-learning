# 《有些事虽然不赚钱

老王 : 《有些事虽然不赚钱，但会让你更值钱！》 今天，我想和大家聊一聊写作这个话题。

写作，是一件特别消耗时间、消耗大脑的事情，在这个漫长 的过程中，有时候会感到痛苦。即便如此，我还是义无反顾 的在做这件事情。

是什么原因让我这么“执迷不悟”呢？ 01、写作会让你学到“通识的智慧” 写作，是我们每一个普通人都应该掌握的技能，不论你是从

事哪个行业的工作。

可能你会觉着，写作是为了挣钱，我没打算靠写作挣钱，特 么学啥写作？ 错!错！错。赚钱只是写作附加福利，写作的真 正收获是让你学习那些“通识的智慧”。

什么是“通识的智慧” ，简单讲，就是掌握一个领域的知识、 经验，可以运用到很多个领域。

有哪些通识的智慧，是我们普通人必须要学习的？ 02、深度思考能力 上午在和同事聊到信用卡、花呗的话题时，这位同事吐槽到:

国家真是个骗子，用各种方法诱惑老百姓透支消费。

真的是这样吗？不对!不对！

国家为啥会鼓励老百姓消费？因为只有老百姓消费，才能带 动企业增长；企业增长了，才会给老百姓提供更多的就业岗 位，才能增加老百姓的劳动报酬；老百姓收入提高了，才会 更多的消费；企业增长了，老百姓的收入提高了，国家才会 获得更多的税收，才会有钱去搞科研、搞国防。这是一个良 性的正循环过程。

你看！任何事情，都不是孤立的一个点，而是与很多看似不 相干的点，互相交织在一起的。只有你去深度的思考，挖掘 表象背后隐藏的东西，才会看到别人看不到的东西。

03、理解人性 我是一个比较喜欢阅读的人，在最近几个月里，陆续的阅读

了心理学、经济学、商学院、写作课有关的书籍。

在阅读过程中，我惊喜的发现，这些书籍中讲到了一个共同 的核心知识——人性，这个偶然的发现让我欣喜若狂。是 的，没错，只要你能理解人性，你就抓住了 20%最核心的东 西。

在持续写作的半年里，我感觉到了自己的认知水平、深度思 考能力提升了很多，这是一件特别值得庆贺的事情。

于是呢！在遇到一位比较要好的朋友时，我迫不及待的把这 件事情分享他，希望他也开始练习写作。然而，最终的结果 却是让我碰了一鼻子灰。

这位朋友用嘲讽的语气说:哎呀，行啦！咱又不是文化人，整 那没用的干啥？

后来回到家里，我就琢磨，这么有价值的一件事，到他眼里 怎么就成了“没用的事了”

其实呢！这是一种很常见的心理活动——习惯性防卫。 当我们遇到与过去，不一致的新观点时，我们的第一反应不

是思考对方的观点是否合理，而是去寻找更多的借口反驳对

方的观点。

而我自己呢！也犯了一个很重要的错误——以偏概全。 我把自己的感受错当全世界的感受，把自己的观察错当全世

界的观察，把自己的看法错当全世界的看法……

一切都从自己出发，全然不知道别人可能和自己不同，别人 所处的世界一样可能和自己不同，甚至方方面面都有很大的 不同。

在我们身边，无时不刻的上演着，各种各样的人性。在写 作、思考的过程中，你会认识到人性是如何产生的？如何利 用人性？

在这个快速变化的时代，永远不变的是人性。所以，理解、 抓住人性，你就成功了一大半。
