# 《陪着≠陪伴》

晚上好 今天是橙橙加入星球的 211 天，和大家分享关于‘陪 伴’的话题」

我们家其实我陪着孩子的时间远比爸爸陪孩子的时光要长很 多倍，可三岁的儿子更喜欢爸爸陪他玩。而我，只是他饿 了，困了，累着寻找安慰的对象。

1.  原先我一起很困扰为什么爸爸陪他的时间少，但是他更喜欢 爸爸陪他玩？

    看了很多关于高质量的陪伴孩子的文章，我才慢慢找到答 案：

    原因①：妈妈在带孩子的时候更多的是照料，比如要给他穿 衣服，做饭，哄他洗澡和睡觉，这些照料已经占用太多的时 间了，没有多余的时间真正的陪伴他，其实孩子只想你陪他 玩，陪他进入属于孩子的世界玩耍。

    原因②：爸爸因为工作的忙碌和养家糊口的压力，往往没有 时间照料孩子，所以这个重担无形中就交给妈妈或者爷爷奶 奶了，但是只要爸爸一有时间基本就是专心的陪孩子玩恐龙 的游戏，或者陪着他一起拼图和讲故事，而平时孩子遇到不

    会玩的的游戏也是会等待爸爸回来一起玩，所以孩子就特别 喜欢爸爸的陪伴。

    1.  02

        什么才是真正的陪伴？

        真正的陪伴不是时间的长短决定的，而是陪伴的质量去衡 量，而高质量的陪伴又来自于你是否用心。

        孩子是敏感的动物，他能感受到你是否全心全意，如果你在 走神或者玩手机，他会非常疑惑的冒出一句“妈妈，你在干什 么呢？不要玩手机啦”他眼里一定很讨厌此刻的手机阻碍着妈 妈陪伴自己。

    2.  03

        那么如何做到高质量的陪伴呢？

        ① 保持愉快的心情去享受陪伴的时光 孩子的世界都是童话世界，快乐多过痛苦，大人陪伴孩子的

        时候尽量避免情绪不佳的时候，保持愉快的心情陪伴，孩子

        能感受到，孩子的童年也美好

        ② 狠心放下手机和工作全心全意的陪伴 高质量的陪伴不需要很长时间，但是在陪伴的这个时间段里

        尽量避免工作和触碰手机，要让孩子觉得你是全心全意的陪

        着他，而当你需要离开去处理的工作的时候孩子同样会理解

        你给予你时间和空间的，因为爱是相互的。

        ③ 放下家长的角色，以好朋友的平等心态陪伴孩子玩耍。

        在陪孩子玩的时候，你往往不是那个高高在上的家长：讲故 事就要有讲故事的语音语调；玩游戏就要进入角色的状态； 那么孩子才会觉得自己的爸爸妈妈特别好玩，他会很珍惜这 段亲子相处的时间。

        ▼总结一下： 智慧妈妈其实不太好当，但我很清楚，要足够细心，细心得 观察和感知，孩子最需要的是什么，洞察孩子的心灵，在他 成长的关键时期给予他足够多的关爱，而高质量得陪伴，就 是一种关爱。
