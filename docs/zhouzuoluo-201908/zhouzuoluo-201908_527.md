# 《粥左罗：把自己浸

《粥左罗：把自己浸泡在什么环境中，你就会接近于长成什 么样子》

前几天跟朋友吃饭聊天，聊到我姐。 我说特别感谢姐姐，我从很小的时候，她就往家里买书，也

不管我爱不爱看，司汤达的红与黑，羊皮卷，人性的弱点，

巴尔扎克等等，我到现在经常能回忆起当时的场景，在我卧

室的窗台上那本厚厚的《红与黑》，记忆里它一直扔在窗台

上，我每次拿起来翻翻看，都摸到上面落了一层灰，每次看

一会儿，就觉得好长啊，算了吧，偶尔注意到它，又拿起来

看看。好了好几年，应该也是自己在长大，竟然开始看得下

去了，于是花了一段时间把它读完了。

我读高中时，我姐带我去过好几次新华书店，让我挑书，记 得当时给我买奥巴马那些人的演讲中英文对照版，让我回去 背，给我买好多字帖练字。

我们那边农村，这种都是不寻常的做法。大人只会说，你要 好好学习，但不会真的有行动。

我女朋友有个亲姐姐，她说她姐姐就没这样的意识，还觉得 一本书好几十块钱好贵啊，不如买别的划算。

所以，特别感谢我姐，她有意无意的让我跟书接触，身边总 是有书，而不只是教材和教辅。而她初中没毕业就去工厂打 工，赚的钱很少，买衣服啥的都舍不得，但给我买书，可以

很大方，我想买就可以买。那时候都是在书店原价买。

我读高中时，还有个拜把子兄弟一样的同学，关系老铁了， 他是个潇洒哥，不爱听讲，爱发誓考名牌，爱看企业家故 事，爱买各种闲书，虽然他最后只考了个大专，但在他的影 响下，我成长很快。他闲着没事，就带我去书店，他家庭条 件稍好，很早家里就有电脑，在农村孩子里算是比我这样的 懂得多，他会跟我说牛人都读啥书，每次买了牛逼的书，就 笑嘻嘻的拿着书过来说“看，这个太牛逼了”，读高三时，我 在他家住，他卧室里全是书，各种书。这些都对我有比较大 的影响。

上学时，还有个印象深的事情。我有个舅老爷，爱好文学， 经常自己写诗，有一天我大爷就拿了一本诗集去我家，说你 舅老爷出书了，给我和你爸一人一本，我们农村人特别稀 奇，觉得一个人能出书，太了不起了。我当时好几天都拿着 看，我爸说，你将来要是能出本书多好啊。我读大学时，还 真的买了很多教写作的书籍教程，不过当时学了一段时间， 觉得自己实在没天赋，就放弃了。

我女朋友，跟我在一块之前，是个绝对没有读书习惯的人， 但是跟我待在一块久了，我看的书老是给她看，没事也经常 读给她听，现在她已经比很多人都更爱读书了。

人都会变的，把自己浸泡在什么环境中，你就会接近于长成 什么样子。

因此，前几天聊这个的时候，我马上让女朋友干了一件事， 让她在网上买点书给她小侄子寄回去，她姐姐没这个意识， 我们可以多帮忙。我说你也别管他爱读啥书，他自己可能也 不知道，你挑一些好书，仍在家里，他这本看不下去，不一 定那本看不下去，总有些好书再好他也不喜欢，也总有些

书，读着读着就喜欢了上瘾了。关键，你要给他创造这样一 个环境。有时候你不能说小孩只爱打游戏，不爱读书，事实 是，农村里，大多数家庭了，没有读书环境和氛围。妈妈打 牌打麻将，爸爸用电脑下棋，闲着没事大家就看快手抖音， 看电视剧，那小孩子抱着手机打游戏，也是正常的。

同样，咱们知识星球也有很大的这种意义，其实这里是一种 圈子氛围，是一种学习环境，你每天闲着没事进来待个十几 二十分钟，你就会有看到，嗯，很多人都在学习，很多人都 在上进，都在读书写东西，你不自觉的就会受感染，再不爱 学习不上进的人在这种环境里浸泡久了，也会变，把自己浸 泡在什么环境中，你就会接近于长成什么样子。
