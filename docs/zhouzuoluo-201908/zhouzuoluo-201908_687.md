# 《说说让很多人菊花

向上生长 : 《说说让很多人菊花一紧的事》 今天一大早看了粥左罗的一篇《义气归义气，不要让义气负 重太多》，我自省，最近我就是那个让很多人菊花一紧的 人。

今天我想说说借钱这回事。 年底了老公公司面临破产，做工程的随着地产经济的浮动， 一下子陷入三角债的困境。我老公就是如此。失败的原因复 杂，自己的原因，行业的原因，大环境的原因，总之失败 了。我作为夫妻也成了共同债务人，一夜之间债台高筑。

都说夫妻本是同林鸟，大难临头各自飞，我这个时候选择了 共同面对，不是我高尚，是我飞了债务也飞不了，再说这个 时候我怎能对老公落井下石，雪上加霜，孩子长大了也会抱 怨我对父亲的无情。此时我选择了义气，颇有我大山东梁山 好汉的气概。我说老公你往前走去挣钱，走踏实喽，一步一 个脚印挣钱还钱，我善后，我借钱把实在于情于理不能拖延 的还上。

就这样我开始了还债借钱，借钱还债之路。因为目前挣钱还 钱实在是速度跟不上。

这不，昨天一个朋友催我还债，是啊，必须还，因为在最困 难的时候是这个朋友拔刀相助度过了当时的难关，现在人家 家里用钱无论如何必须还。信誉重要面子搁一边，找同学， 找同事，找朋友，最后不行高利贷也必须还。但是借钱是何 其困难。理由是何其说不出口。

给朋友留言，因为借钱最担心钱没借出来，还丢了友谊，我

小小心翼翼。

第一条留言:不要因为我说的事情有压力，不要因为我的事情 妨碍咱们多年同学友谊，不要因为我的事情你反而不好意 思，咱们万事互相理解，可以的话我就说一下我的难处。实 在是难开口。我想；你这么聪明已经猜到了。

第二条留言，我目前家里发生了难以跨过的坎，我想在你不 影响你的高质量生活，不影响你的孩子正常开销，不影响你 一切的情况下，能不能有宽裕出来的钱借给我一点，让我度 过一下眼前的难关。确实是我已经走头无路了，需要朋友的 支持帮助。

第三条:我再强调一下，行不行我都感谢，陪我聊会天我也非 常感谢。别因为我的请求，从此不再联系，这是我最担心 的，我害怕失去朋友，事情是事情，友谊是友谊，你千万不 要有什么想法，无论怎样，我都非常感谢。

还好朋友回复，谁家还没个难事，我刚刚离婚，存款现金全 部已经给了前妻，店里有这几天营业现金，明天早上存进银 行，给你转。我万分感谢。

其实在我落难的日子，有各种各样的因为借钱，状态百出。 第一种，留言发出后，立刻发现被屏蔽，不再接听电话，不 再回复留言，这种情况一开始让我手足无措，不知道如何面 对。后来想也许是不知道怎么拒绝。

第二种，留言发出后，立刻回复俩字，不行，你赶紧想其他 办法。这种直接的拒绝不知道让我多么痛快和感激。不会耽 误我的时间。

第三种，我的留言发出后，立刻回复，哎呀你有难处了，我

怎么也得帮忙，等我信息，我想想办法。从此石沉大海，永 无消息。

第四种，是我生命中的怪胎一样让我感激之情的存在。看到 我就说，看你情绪低落，咋了有难处不跟姐妹说几个意思， 多了没有你先用着。过年后约我出来书店看会书，有意无意 问一句，孩子学费筹齐了吗？看你也不好意思张嘴，我明天 转给你，日子先往前走。我的眼泪不知道让这个怪胎捣鼓出 来几回。我笑着，你疯了，不怕我还不起你，她说不怕，到 时候我就赖在你家，你给我做饭吃。

第五种，留言发出后，直接就是支付宝微信转账记录，就这 么多，你先用着。再无其他。

无论哪种类型，都是我生命中的存在。其实借钱也好，被借 钱也罢，都是一个真诚，借的真诚一点，拒绝的真诚一点， 还的真诚一点。你不借给是本分，借给人是情分，无论哪 份，都是分寸。

可能三五个月我还在借钱还钱，还钱借钱的困境里，但是我 相信，只要真诚一点，困境总会过去。误会总会解开。算给 自己的鼓励吧。
