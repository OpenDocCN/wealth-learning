# 《如何建立自己的知

《如何建立自己的知识体系》

喜欢看书的人都渴望建立自己的知识体系。那么，我们今天 来探讨一下如何建立好自己的知识体系？

1、只学自己触动的 我们初学者很希望拥有自己的知识体系，但是一开始学习的

时候都是一个个知识点，无法形成知识体系。

具体怎么做呢？就是在阅读时，我们要寻找触动点。 对自己触动的地方画上标记，可以在备忘录当中写下自己触

动的思考。看完一篇文章或者读完一本书，问问自己哪个知

识点触动到。

这个点可以是一句话，一个理论，只要这个点自己真的能掌 握，转化成行动，这本书就超值了。

因为我们看完一本书，不可能把所有的内容都掌握，触动你 的那个点掌握就已经很棒了。

2、触动是最好的筛选器 我们平时就会看很多文章，如果不是触动你的点，只会当时

觉得这篇文章很好，过一段时间你早就忘记了。

正是因为触动我们的地方，才是我们真正想要去改变的那个 点。通过触动，我们可以用来读书、反思，建立自己的知识 体系是非常高效的。

我平常触动到的点，会用写作表达出来，通过这个点你跟自 己的实际情况结合，你就知道自己下一步需要怎么做。比 如，我昨天看完粥老师那篇《废掉一个人最隐蔽的方式，是 让他忙到没时间成长》，就是其中的几个点触动到我，结合 自己写的一篇分享，这样我以后找工作，肯定会根据文章中 触动我的点去做。

3、仅仅触动还不够 触动虽然很好用，但还不够，因为不是所有的触动都有效。 有时候触动不一定会紧密连接。所以，为了留住它，让它成

为体系的一部分，就想办法和它产生连接。

连接有以下两种 第一用自己的语言把它重新解释出来，逼迫自己原有的知识

系统对新知识做出反应

写作就是一种很好的方法，就是把作者的知识点用自己的话 来表达清楚，转化成自己的知识体系。

第二在生活中能够经常练习和使用，实践才是产生连接的最 好方法

学习的最终目的不是为了知道，而是真实的改变。运用那些 知识转化成行动才是对自己最有帮助的。

以上就是建立知识体系的方法，希望对大家有帮助。
