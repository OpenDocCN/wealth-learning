# 《训练营开营第 11

《训练营开营第 11 天：趁年轻，去做一件让自己骄傲的事》

星友们好久不见，这条帖子想写很久了，因为最近在做写作 训练营挺忙，硬是给拖了一周才写。但从明天开始，无论有 多忙，我要恢复日更啦。

我不是为了写而写，这段时间实有太多太多的收获，忍不住 要分享一下。

大家都知道，我和粥老师都没做过训练营，毫无经验；我们 的小团队总共就只有四个人，人力有限。

但这几天我问了很多训练营里的朋友，当然也包括一些潜伏 其中、训练有素的专业运营。她们给我的回答基本都是：真 的很赞，很用心。

包括昨天新世相曾经的编辑，现在傅盛的助理潜进训练营看 了一圈学员的作业，当即答应我进训练营做助教。

为什么她同意来，一方面想学习我们的运营，另一方面她想 在我们的训练营里挑选优秀学员约稿。

今天正式邀请她进群之后，她回头对我说：文文，你太有运 营经验了。

我回她说：你信么？这是我第一次做训练营，一边学一边 做。

她说：

我们的学员打卡率稳定在 70%-80%，要知道，粥老师布置的 作业，每天至少得一个小时才能完成，有的同学甚至拿出 5-6 个小时来写作业，大家居然坚持了十几天，真的超乎我的想 象。

而且留言区非常的活跃，学员互相鼓励和打气，社群气氛非 常有爱。前几天就有几位学员跟我说：好害怕训练营会结束 啊，到时候可能会哭啊。

其实不止是学员，我看着大家为了精进自己的写作能力付 出，积极热情地对别人付出，内心有一种莫名的温暖和感 动，让人充满了力量。

实际上，在训练营的开始的前后几天，我都是 2 点多睡觉，6 点多起床继续工作。因为我的内心忐忑不安，一遍一遍地修 改流程，细分运营团队的工作职责，除了小团队会议，还要 一对一私聊，以确保她们接受到的信息是无误的。

在开营仪式的那一天，我们提前预演，我在书店坐了整整一 天，不知道为什么书店的服务员来问了我两次：你没事吧， 要不要给你倒杯水（后来她们说我坐了十个小时没怎么动， 脸色可怕）。

我问粥老师，进入训练营的感觉怎么样（看到没，大咖都是 万事俱备才进场的）。

他回答我：就俩个字，牛逼。 处女座老板，轻易不夸人，让我心里瞬间感觉，值了。
