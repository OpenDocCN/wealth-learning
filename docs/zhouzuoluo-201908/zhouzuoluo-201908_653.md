# 《年轻，才不是你的

《年轻，才不是你的遮羞布》

很多人都喜欢说年轻没有失败，把范的错归咎于自己年轻不 懂事，要求别人原谅自己因年轻做错的事，年轻成了最大的 挡箭牌。但年轻真的就应该一直被原谅吗？

单位有一同事，年纪轻轻就做到了领导岗位，属于年少有为 型的。但在为人处世方面，已经不能用情商低形容她，基本 可以说没有情商。

下属熬夜做好的方案，她可以说成是自己熬夜做的；比她父 母还大的下属和她汇报工作时，她能做到破口大骂；开大会 时，大领导话没说完，她可以把话接过来把她认为很重要但 实际没那么重要的事先说完……

如果只是我自己不喜欢她，可能是我有偏见。但几乎全单位 都不喜欢她，我觉得还是很说明问题的。

大领导为此和她提过好几次。提醒她注意工作方式方法，搞 好上下级关系。但她每次都说自己还年轻，不擅长人际关 系，希望大家多多包涵……

年轻人可以犯错。一次两次三次都可以，但每次犯错后必须 从中吸取教训，同样的错误避免下次再犯。而不是说因为年 轻，就可以同样的错误一直犯下去。年轻，才不是无能的遮 羞布。

每个人，都应该承担起自己身上的担子。不能因为年轻，就 逃避自己的责任。一个只会拿年轻当遮羞布的人，永远不会 真正长大。
