# 结束了第三期的写作

结束了第三期的写作课，但是昨天和几个朋友聊天，悟出了 点感悟，和大家分享下。

《互联网时代 -- 没有什么能够阻挡，你接触世界的渴望》 这是一个最好的时代，移动互联网，让沟通无处不在，让沟

通触手可及，让沟通穿越一切。只要你敢想，只要你敢做，

只要你敢行动，没什么能够阻挡，你接触世界的渴望。

整个七月，过的特别充实。工作上，自己编写的新需求，基 本没遇到什么特别大的困难，作为后台的我，没有成为拖慢 工期的那块木板；生活上，参加了一个 21 天写作训练营，获 得了技能上的增强；最最重要的还是圈子上的提升。

圈子上提升的认知，还是源于对世界的渴望。移动互联网的 高速发展，对社会的进步真的是太大了。国内有优秀的微 信、微博；国外有 Facebook，Twitter，让每个人都能更好的去 接触这个世界，了解这个世界，扩宽自己的眼界，扩展自己 的人脉，扩大自己的交流圈。

感谢这飞速的移动互联网，在进入七月之前，我的圈子，除 了自己学生时代认识的同学到现在是各方各面的优秀人才之 外，非 IT 技术的朋友真的太少太少。

所以，当在写作课上组成的微信群里，看到各行各业的渴望 向上生长的小伙伴们时，我忽然打开了新世界一样。形形色 色的人都有，而且还鼓励大家互相认识，所以就这样，我的

圈子，在这几天，有了很大的扩张。

作为一名 IT 从业者，尽管知道现在互联网的发达，但是没能 真正去利用好他，这次我算是用对地方了。通过和他们持续 的学习，持续的交流，感觉自己的世界真的是太小。

难道我的世界除了写程序，就是写程序了吗？在此之前，答 案就是如此。从现在开始，这个答案要变了，我的世界，除 了程序，还有心理学、社会学、销售学、运营、医学、创业 等等。

当你充满了对其他知识的渴望，你越发觉得互联网真的是太 强大了。我分享了我在写公众号用的 markdown 语法，没想到 对很多小伙伴来说，竟然是一个未发掘的神器，纷纷问我经 验之道。

同样的，他们分享了他们擅长的如何更好的看一本书，如何 获取优秀的资源，如何沟通，每次几个时间段的讨论，都是 我扩大自己眼界的时间，从中获取到的知识面，是我之前未 曾拥有的。

前几天和一个学弟聊天，我们互相交换了自己经常关注的公 众号、知识星球，没想到又让我震惊到。他的圈子、和我的 圈子，还是有很大的不同。当晚我就写下感慨，感谢互联网 的存在，让我走出了自己的小圈子，去接触了更大的圈子， 拥抱了更多的美好事物。

今天，一个朋友在星球说了，**城市会限制你的发展，但绝 不是你退缩的理由**。这不就是在说我吗？偏居一隅，但是 不会阻挡我对外界的渴望。还有朋友，分享了很多工具，很 多硬核的课程，提供了很多运营、增长相关的知识给我，我 还有什么理由不努力，不发奋呢。

在写这篇文章的时候，刚好看到帅张发的最新的文章，里面 有句话我很喜欢，也很应景，**所以这些年，我特别感谢微 信，特别感谢龙哥，没有微信公众平台，没有龙哥做出微 信，像我这种没啥能力的一小程序员，很难到现在这种阶 段，说微信改变了我命运都不为过。** 这就是微信的厉害之 处。

这个七月，是我一段学习的结束，也是我一个新认知的开 始。未来的路，只因有你们，没有什么能够阻挡，我接触世 界的渴望！！！
