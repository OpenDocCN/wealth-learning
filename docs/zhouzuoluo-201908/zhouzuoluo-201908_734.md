# 《我害怕粥左罗再给

《我害怕粥左罗再给我涨工资》

前几天一个同学跟我说：“粥老师说，你现在的工资很高 哦！”

我有些诧异，问她如何得知，她说问了粥老师，亲口告诉她 的。我明白她的潜台词，是在质疑我给她的答案。

几个月前她问我：“粥老师给的工资一定很高吧？” 我回答她：“还行，不算高。” 那时，是真的不算高。很多朋友建议我，一定要开口多要一

点，反正粥老师很大方。我最后按照以往工作能拿到的工资

+福利，报了一个不低的数字，但也不算很高。

后来每次发工资，几乎都超过我的预期，几个月下来，已经 到了一个另人羡慕的数字。 除了开心，我其实有一丝丝恐慌。拿薪水从来就是按能力 的，到底是因为我的能力已经那么强，还是老板给了偏高的 薪水？

我常常在想一个问题，抛开在成长上的帮助不论，粥老师给 出的高薪水，可以轻松在行业内找到很优秀的员工。那么反 过来，我是不是能够很轻松找到一位，给出粥老师一样薪资 待遇的老板？

答案是很难很难！

过去，我几乎从来不会害怕换工作，因为我清楚的知道，自 己可以换到更好的（不单从薪资方面）。现在呢？粥老师已 经提前给出了溢价，提前购买了我的潜力部分，我需要更努 力一点才配得上那份薪水。

所以，对于那种舍得给钱的老板，倘若自己成长的速度不 够，其实是件非常危险的事情。唯有更加努力，才能回报。 2019-02-20(63 赞)
