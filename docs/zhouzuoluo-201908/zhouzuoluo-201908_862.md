# 《万能写作法真是万

《万能写作法真是万能的吗？练好了这几点才行》

我们想学写作的人，一听说“万能写作法”一定两眼放光，恨 不得一下就想学会这个方法。因为大家正缺这个特别诱人的 方法呢！

但是它真的是想象的那样，明白了这个道理就如遇上了捷 径，抄上了小道吗？否! 它是有先决条件的。

我们看一下万能写作法的核心价值是: 条理清晰的表达逻辑。 那我们先弄清楚什么是条理清晰:

1.指叙说事物时讲究规矩性、章法和法则，即讲道理、讲规

则。

2.文章结构上要求脉络清楚，层次分明、秩序井然。所以我们 说:

一，如果没有先学会叙说事理时主次分明，而是面面俱到， 眉毛胡子一把抓，想一想谁愿意化时间看一堆凌乱的堆放物 呢？所以，我们应该先学会沟通和说话的技巧:

1.说话时言简意赅。比如，我们和面徐徐倒水，水和面恰到好 处的面块揉劲儿正好，但是猛到水一超量，面就稀软无劲儿 不成块儿了。

2\. 抓住重点一针见血，就如放箭那样直击要害，无需乱箭投 射，（目的又不是草船借箭）抓住重点讲清就行了。

二，如果会抓论点，关键得用对案例来论证论点，读的人才 能明白你要表达的思想。比如，我想告诉大家，“带着感情做 出来的菜才好吃”这个主题，可是我在文章中却列举了菜的配 方和制作的方法，全然没有一处表达内心情感的描写，肯定 是偏离了主题。

三，当有了观点，也选取了有力的事例来证明，可是，在文 章结构的安排上却出现了混乱或是层次的前后没了秩序。那 万能写作也发挥不了作用。

比如，参观上海的世博会，每个国家的展馆前都是人满为 患，怎么不出现混乱呢？因他们管理的方法是门前排放曲形 围栏，人们单行排队前行，所以再多的人也不会出现混乱。

写文章也是这样得用思考来管理，段与段的衔接一定是连接 紧密，如果上下搭不上，还得安排一段来承上启下，层也很 重要，尤其是细节描写的地方，如果乱了就像有了病，肯定 影响全篇的紧凑感。

写作是讲究逻辑关系的，也就是“依赖关系”，“条理清晰的表 达逻辑”指的就是逻辑的依赖关系。逻辑又分演绎逻辑，归纳 逻辑、形式逻辑等。逻辑关系的含义这么丰富，所以我们要 想写好文章，还得好好学一学逻辑关系。

总之，万能写作法不是听的，得好好练! 必须在有规则、有章 法的大量训练之后，产生“熟能生巧”的巧，这个巧才能使万 能写作法真正发挥它的作用。

我是章子，很高兴与大家一起努力地向上生长。
