# 《麦基公式:教你如

《麦基公式:教你如何写好故事》

什么样的故事是一个好的故事？好莱坞编剧编剧教父罗伯特· 麦基是当今世界公认的最会教人讲故事的人，他提出了一套 故事理论，其中最核心的精髓就是六个字:冲突颠覆生活。

一、什么是冲突颠覆生活？ 冲突就是戏剧化，这种戏剧化就是怎么做都对，但又怎么做

都不对。比如现在有个女强人，向来觉得男人不可靠。突然

有一天，冲突出现了，她爱上了一个男人。但这还不够，这

个冲突必须具备两面性，所以还得加一个条件，这个男人是

个有夫之妇。女主角如果面对这个感情，就违背道德，如果

逃避就违背内心。这种矛盾就是戏剧化。

生活就是让我们有情感上的共鸣，让我们大多数人产生代入 感，不自觉的把感情放进去。比如这样一个故事:有一家人， 住在一栋能看见海景的房子里，但是有一天突然楼上装了一 个广告牌，把海景挡住了，你看，这就是冲突。但不够，你 不会有情感上的共鸣，不会有代入感，也就是没有生活的元 素。

如果故事这样改动:一家五口挤在这个房子里，房子还欠着贷 款，全靠男主一个人还，为了省钱，家里人不舍得乱花钱， 日子过得非常拮据，也经常吵架。但是不管有多大的矛盾， 只要自家人一起站着看着海景，一切烦恼烟消云散。这个海 景就是他们生活的寄托。突然有一天，楼上多了一块广告 牌，不仅仅挡住了海景，还让这一个背负沉重压力的家庭唯

一的生活寄托被瓦解。这时，很多人就会产生共情。

二、“冲突颠覆生活”的原理 搞懂了含义，那这六个字背后的原理是什么呢？其实很简

单，那就是制造真空。真空其实很好理解，就是在我们追美

剧的时候，追不完会难受，巴不得一晚上看完全集，在听单

田芳评书的时候，最怕听到“请听下回分解”。这就是因为他 们在我们脑子里制造了一个真空。

这里说的真空是逻辑真空和情感真空。逻辑真空就是你在看 故事的时候，你无法猜透故事的线索，不知道情节接着会怎 么发展，就造成了逻辑真空，就好像电影《调音师》，各种 反转，但你就是猜不到后面的情节。

情感真空就是结局可能你已经知道了，但其中是如何发展的 这个过程会让你产生这种真空感。

真空不同于好奇心，真空比好奇心更难满足，只有制造真空 的人才能去填满它，解铃还须系铃人。所以冲突颠覆生活”背 后的原理，其实就是你在听众的意识里制造了一个真空，之 后，你再亲自把真空给填满。

三、好故事的三个阶段 了解了冲突颠覆生活的公式以及背后的原理以后，接下来就

要知道写好故事的三个阶段。

第一个阶段，叫生活遭遇产量，也就是主角遭遇变量，打破 一开始的生活，让自己面对一个充满不确定的世界，往往让 主角跟开场形成反差。

第二阶段，叫意外打断努力，在主角不断努力向着正常的情 节发展过程中，也不断遇到意外，这种努力被一次次打断。 正因为有这些意外，你才有看下去的欲望

第三阶段，叫情感满足，最后故事往往都会给观众一个交 代，最终会让我们满足，把真空填满。

虽然公式只有六个字，看起来很简单，但真正用起来却很 难，需要我们的不断练习以及不断地思考。
