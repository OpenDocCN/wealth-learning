# 《如何度过闲暇时光

《如何度过闲暇时光居然和我们以后的成就有关？》

晚上好 与星友们分享丹儿的第 93 篇原创 7.29

————————————————————————

“未来不仅属于受过教育的人，更属于那些善于利用闲暇的 人”今天看到这句话引发了我的思考！星友们闲暇时间是怎样 度过的呢？是看剧，玩手机？还是陪伴家人或是做点自己爱 好的事情呢？今天丹儿想和大家聊聊闲暇时间的重要性…

每个人的性格喜好都不同，所以闲暇时间我们会做的事也就 不同，先来和大家聊聊我自己的闲暇时间会做些什么吧！我 的闲暇时间分为任凭自己松散放纵的闲暇时光，和让自己心 灵更加集中更加有力量的闲暇时光。

一，任凭自己松散放纵的闲暇时光

①看剧

②刷抖音

③吃各种零食

感受：刚开始感觉还可以，可是时间越长越厌烦，但是又放 不下手机。感觉有瘾，控制不了自己。

二，让自己心灵更加集中，更加有力量的闲暇时光

①听樊登读书会

②做指甲

③看书

④扫除

⑤皮肤管理

⑥旅游

感受：闲暇时间做这些事感觉真是太棒了，做完以后觉得时 间没有浪费，觉得自己受益了在进步。

尤其是我每次感觉上瘾，只要打开樊登读书会，听完一本书 我就能放下手机了，突然发现有很多事等着我去做呢，我得 立马行动起来。

感受每次在研究新的款式时，因为喜欢会投入进去，还会思 考怎样做能更好。我发现通过思考加动手做成功的事，每次

都能获得成就感，很开心的感觉。

★星友们是任凭自己松散放纵的闲暇时光多？还是让自己心 灵更加集中，更加有力量的闲暇时光呢？做这些事情时是否 也有和丹儿相同的感受呢？欢迎在评论区留言交流哦！

那么问题来了？为什么明知道任凭自己松散放纵的闲暇时光 不好，却放不下手机，走不出来呢？

三，娱乐设施的设计 现在很多的软件，电视剧，娱乐节目等，它们都很注重数

据，也就是你能在它们上面花费多长时间！因为占用你的时

间多了，就可以获利，赢取利润！

那么怎样提升数据，占用你的时间呢？这就需要一点设计 了。能想出好的设计方案，就可以占用你的时间，绑住你。 比如说抖音，它的设计就是只给你推送你喜欢看的，让你不 知不觉沉浸其中。看它的人多了，它就可以收取商家的推广 费。

对于这个设计我们也没办法，粥老师在他的文章里也提到 过，有时候刷抖音不知不觉已经半夜了。

所以我们放不下手机的原因是因为各种娱乐设施的设计。

但，既然我们感到开心，沉浸其中为什么还会感到厌烦呢？

我想大概是因为做这些事没有让你感到成长，获得成就感 把！

举个樊登读书会的例子 有工作人员向樊登提议，要不要设计点东西让樊登读书会的

用户也进去 app 以后就停不下来，我们好赚取广告费宣传等。

樊登说“为什么要这么做？生活中那么多事需要去做，你让他 们一直看 app 干嘛！樊登读书会创立的初心就是让读者听完一 本书就放下手机去行动，去做他们应该做的事，能改变自己 的事儿”

樊登这番话说的特别好，这才是为我们成长所设计的 app。现 在自己也在创业，希望以后我的每一位顾客在我这里消费也 能有这种感觉。

★娱乐设施不是为了让你成长而设计的，而是为了绑住你而 设计的，就会让我们感到痛苦，厌烦。或许我们可以把意志 力与娱乐之间的抗争当做一个游戏，挑战一下自己。

＊＊＊＊＊＊＊＊ end ＊＊＊＊＊＊＊

▼在闲暇的时候是让自己的心灵变得更加的集中更佳的有力 量，还是任凭自己松散放纵，这个将决定一个人这一生当中 的成就。
