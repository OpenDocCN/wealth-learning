# 《》这几天又回归文

这几天又回归文文姐的怀抱了，第一次听到她的声音，完全 不是像传说的“文一刀”，而是很温柔的小声音，一次聊天， 我把自己的收获简单写写吧。

01.学习要广泛，不设限 文文姐一直说我是个爱学习的宝宝，我以前就是什么都学，

但是学的都没用，现在我更喜欢学习，是因为我认识了更多

人，看到了更多可能，所以知道自己应该去学什么。

文文姐也是说不要把自己的视野局限在自己生活的地区，因 为互联网的存在，我们每个人都可以多多去改变，自己通过 就是付费学习，慢慢精进的。

学什么都可以，知道为什么学更重要。 我现在就在想为什么毕业后人与人之间的差距越来越大，是

意识到终生学习，知道不学习就会被淘汰，所以学吧，没有

坏处，时间越长，时间的复利就越大。

02.学会提问，直接点 我还是像以前一样，遇到厉害的人就条件性反射的，离人家

远一点，害怕暴露自己的无知，怕耽误别人的时间，不知道

怎么沟通。

这次文文姐告诉我，多和助教老师沟通，他很厉害的，我就 在想我应该了解一下，我听了他的直播，看了他分享的资 料，还是觉得自己准备的不充分，又去看课程了解了一些情 况，针对自己的不足，列好了一些问题。

我问文文姐这样做好不？她告诉我直接去问，分享完了，可 以多感谢，多表扬，发个小红包，然后和人家自然地联系 上，慢慢问问题，不要太复杂，直接点，但是问题一定要自 己思考过，有价值。

怕和厉害的人沟通，多半是觉得实力有差距，但是我们可以 通过展现自己的思考去问，人家都是喜欢这样的人的，要尝 试。

03.在实践中才能学会东西 可能是在学校呆的时间比较长了，我学习东西，首先会想到

学书本上的理论知识，理论知识透了，才会在实践中去行

动。但文文姐告诉我在实践中才能更好学到，光看书是远远

不够的。

今天早上又下意识说了，先去看书，说完立马就反应到自己 的问题，然后在小群里和小伙伴们沟通了几句，发了个红 包，效果还是蛮不错的。
