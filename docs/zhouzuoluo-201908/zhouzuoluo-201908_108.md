# 大家好，我是猫猫

大家好，我是猫猫，这是我在星球的第 3 次分享。（全文大 约 2000 字）

————————————————————

❤ 《我有酒，你要来听我的故事吗？》

7 月 20 号的晚上，老徐在自己的财富观分享群里组织了一个 故事分享的活动，我当时有事情在外面没能参加，直到一周 后的今天，才拿出来细细品味。

我本来想写自己的感悟，但是列了框架以后发现不是自己想 要的，所以我想了想，决定和大家一样，厚着脸皮写下我自 己的故事。如果你希望看到一个学渣逆袭的励志故事，那么 对不起，要让你失望了。这里只有一个平淡无奇的小角色的 经历。

一、佛系的大学时期 高中的时候我的学习成绩并不好，经常排年级倒数。高考的

时候我只考了 425 分，像很多刚毕业的高中生一样，我对自己

的未来一片空白，既不知道自己喜欢什么，也不知道自己未

来想要做什么。

那个时候金融、财会、建筑、工程、电气之类的专业很热 门，不知道自己喜欢什么的我就听从了父亲的建议，选择了 金融专业，花了昂贵的学费去了西安一所财经类三本院校。

上大学之后的自己依然是个乖乖女学渣，不逃课不挂科，但 就是不学习。自己的专业谈不上喜欢，也谈不上不喜欢，每 天就那样得过且过，佛系而且随缘。同寝室的舍友铆足了劲 考证，而我沉迷玩耍，大学四年出来也只考过了一个英语四 级。考研的想法在我脑海里只出现了一秒就被我扼杀了。

我清楚地知道自己的专业是一个社会上非常赚钱的行业，同 时也非常清楚地明白如果想要在金融行业真正赚到钱，需要 有强大的学历做背书，以及足够的能力。金融专业的学生在 只有本科学历的情况下，出路基本只有销售一条，但是我不 感兴趣。怀揣着对未来的迷茫和恐慌，我就这样毕业了。

二、迷茫并且焦虑的工作时期 毕业的时候因为想去大城市，所以去了郑州，因为在大学的

时候做过四年的图书馆学生助理管理员，从而在一开始的时

候选择了书店的同类职位，做了一年以后又转战电商线上运

营。

在这中间，我曾一度陷入非常巨大的迷茫和恐慌，不知道未 来的方向是什么，也不知道自己以后究竟可以做些什么。我 们就像是在人生的跑道上，我看着我的同学和好友一个一个 接连找到了人生的方向并飞速前进，只有我一个人原地踏步 拖泥带水，那段时间整个人非常焦虑，工作的时候经常走 神，不工作的时候也经常发呆，脑子一片空白，陷入了巨大 的绝望中。

我对很多事情都很容易产生兴趣，也对很多事情都会三分钟 热度，我在闲暇时间疯狂地浏览招聘网站，想要找寻自己未 来的人生方向，每次都无功而返，除了看到大多数职位眼令 人花缭乱的专业要求和遥不可及的工资数额之外，毫无收获

可言。

那时候的自己心浮气躁，急于求成，不懂得静下心来慢慢前 进，以至于每天都很苦恼，不知道该从何努力。

三、真人版伤仲永 我在小的时候非常喜欢看书，学习成绩也非常好，据我爸妈

说，我每天什么都不干，就是学习。长大以后外界的诱惑越

来越多，也许是手机太好玩了，贪玩的我注意力不再在学习

上面，逐渐走向平凡人的行列，小时候和我一样很优秀的同

学，现在已然成为清华硕士。

小时候学过一篇课文叫做《伤仲永》，大概意思就是有一个 名叫仲永的孩子，小的时候特别优秀，长大以后泯为众人。 我就是那个仲永，在时间的洪流以及外界的诱惑中慢慢迷失 了自己。

四、接触到知识付费 我大概是在 18 年初接触到了知识付费，当时正是知识付费迅

速发展的一个阶段，我第一次接触到知识付费，感觉很新

鲜，看什么都想买，什么都想学，觉得自己什么都缺。当时

我处于一个知识付费冲动消费阶段，看见好的都买下来，但

之后全部落灰，这中间不乏一些知名的课程，买来之后坚持

下来的寥寥无几。

一年以后，时间来到了 2019 年，我看着那些因为一时冲动买 下来结果却囤积落灰的课程，开始反思：这些真的都是我需 要的吗？现阶段我最想学的课程是什么呢？我买课的目的是 为了什么呢？

我花了大约半年的时间来想这个问题，直到我遇见了粥左罗 写作训练营。

写作是我从接触知识付费领域开始就非常想要学习的技能， 但是那个时候虽然关注了粥老师很久，也在很早的时候加入 了他的星球，但是因为看的次数太少，从而一直对这个人不 甚了解，当时的想法就是看到有一个写作训练营挺对我的胃 口，就毫不犹豫的买下了。（不要打我，我现在是粥老师的 铁粉！！！）

事实证明，我的选择是对的。 我的性格属于比较慢热的类型，和人接触比较小心翼翼，比

较不爱在群内说话，一开始就是每天闷头写作业，也不与人

交流，存在感比较弱。直到最后一周苦于排名上不去，在小

伙伴那里拿到了秘籍之后开始了我的逆袭之路。

我开始花费大量时间看作业刷积分冲排名，每天积极地在群 里出现，和小伙伴们交流玩耍，结交到了非常多的小伙伴， 认识了很多优秀的助教老师们，终于到了结营的时候拿到了 我想要的排名。出于对这里的热爱，我再次加入了第三期。

我以前从来没有想过我能在这里链接到非常多志同道合的小 伙伴，也没有想过慢热的我如今可以在群里和小伙伴们谈天 说地，成为了我曾经很羡慕的那种性格的人。是训练营改变 了我，现在的我已经不再焦虑，因为我找到了自己热爱的事 情。

在迷茫和焦虑的时候，不要害怕，放手去做就完事了。 只有自己才可以打破过去的自己，谁都不能帮助你。 你还年轻，未来还有无限可能，所以不必焦虑。
