# 《你要明白，没有人

《你要明白，没有人值得你以命相陪》

我们的一生总会有许多无奈。有时心里在哭，脸上却要笑； 明明上司的决策很傻逼，却要点头称道；明明很讨厌喝酒， 却要在酒桌上硬着头皮喝。我们也讨厌死了这样的自己，可 有时真的没有办法。

但是，所有的忍让退步都要有一个限度。我们不能为了别人 满意，无底线的委曲求全，搭上了自己的健康甚至生命。我 们要明白，这个世界上没有人值得我们以命相陪。

正逢春节长假，亲戚朋友间的聚会一场接一场。即便不想喝 酒，也总要被同桌人各种劝说，最后说的自己都不好意思不 喝，多少都要喝点以示敬意。虽然无法理解，但这就是很多 人身边的酒桌文化。

前几天看到一则新闻，王先生大年初一和朋友聚会，虽然明 知道吃了头孢不应喝酒，但架不住好友的劝说，还是小酌了 几杯。结果饭还没吃完，王先生身体就开始严重不适，送到 医院后还是没能抢救过来。

对于王先生的遭遇，即同情又气愤。同情是因为这么点小事 不幸丧命很可惜，气愤是他耳根子软架不住劝，为了让朋友 满意，吃了头孢还冒险喝酒，最后丢了命。

其实和王先生一样架不住劝的人并不在少数。即便身体不 适，也要在酒桌上逞强。只有一瓶的量，非要喝三瓶，照比

别人少喝一口都觉得跌面；吃了忌酒的药，完全不在乎，总 觉得自己是刀枪不入的大英雄。

即便侥幸一次没事、两次没事，但如果长此以往，出事只是 时间问题，并不是运气差。

我们总是过于在乎别人的感受，而忽略了最应该重视的自 己。领导让没日没夜的加班，加：朋友劝没完没了的喝酒， 喝。但命是我们自己的，出事了只能自己买单。曾经的好领 导好朋友只会在葬礼上说一句一路走好，但家人的痛苦可能 一生都无法愈合。

所以，我们无论做什么都要有个度，不能以牺牲自己的健康 为代价换取别人的满意。不能喝就别喝，真正的朋友不会因 为你少喝几杯酒就远离你；不合理的加班就拒绝，好的公司 不会用员工的身体换取效益。

我们的生命只有一次，如果自己都不爱惜那别人更不会爱惜 了。无论什么时候，都要记得爱自己，该拒绝的时候一定要 拒绝，真的没有谁，值得我们以命相陪。
