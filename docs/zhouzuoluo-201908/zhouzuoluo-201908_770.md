# 《即便去不了北上广

《即便去不了北上广深，但也没人阻止你变得比自己更好》

这两天读粥老师的文章“终于，我们逃回北上广深”，感触良 多。目前我在十八线小城，对粥老师说的好多小地方的弊端 我真的感同身受。

凡事靠关系、喝酒才能办事、整个圈子都没有上进心、不撸 串不打麻将会当成异类、人们每天热衷于各种八卦，芝麻大 的事会经添油加醋后变成西瓜大的事…这就是我当下的圈 子，说实话，我讨厌死了这样的环境。

其实我也有一个大城市的梦想，我也想去北上广深闯荡一 番，哪怕撞得头破血流，最后一事无成，那也算我为梦想拼 尽全力。但是由于各种原因，我只能与这样的圈子同流合 污，甚至在可以预见的未来我都离不开现有的圈子。

可能好多人都不理解，认为我还是不够坚定，只要自己坚定 一起都不是问题。但是你们真的理解不了东北、小县城、公 务员、农村人这几个词叠加在一起意味着什么。

我的父母、亲人、甚至朋友都认为我现在的工作对我而言就 是最好的，如果我谈梦想他们会觉得我吃错药了，辞职更是 无从谈起，如果我现在辞职，估计我妈的血压能干到 180。所 以，我暂时真的没有勇气去和一个环境抗争，有些后果是我 承受不了的。

所以，看着好多人都回到了北上广深为梦想奋斗，而我还要

在不喜欢的酒桌陪着笑脸，内心的失落油然而生。

但是，抱怨解决不了任何问题。纵然心中有一万个不喜欢， 我每天还是都要面对这样的环境。与其抱怨，不如让自己在 现有的环境里变得更好，才算不辜负自己。

每个人都知道大城市好，那里有全国最好的资源，那里说梦 想不会被嘲笑。但每个人的实际情况不同，即便都知道那里 好，有些人连尝试的机会都没有。而去不了北上广深，不代 表就没有做梦的权利，只要心中有梦，在小地方也可以做到 让自己闪闪发光。

没有人能阻止我们变得比自己更好。限制我们的不是大城市 小城市、家境、年龄、学历，只要不给自己设限，我们的人 生可以有一百万种可能。如果抓到了一手烂，那也不要放 弃，用心打好每一张牌，少输也算是一种成功。而只要不放 弃，有时绝望也可能变成希望，但前提是我们自己不能放弃 自己。

PS：今天参加了一个局，不胜酒力的我又喝多了。虽然我心 里万般抗拒，但在现有的环境下有些事真的无能为力。其实 现在状态特别糟，回家后睡了一觉，然后吐了，现在胃里还 在翻江倒海。一度想今天就弃更吧，反正也不会有人知道。 但成长是自己的，有些事骗得了别人却骗不了自己，所以我 又爬起来坚持日更。但愿我的抗争有朝一日收到成效，我会 成为自己喜欢的模样。
