# 《认知差距决定赚钱

《认知差距决定赚钱能力》

我们经常会遇到一些很厉害的销售人员，他们会利用一些销 售技巧，我们很容易被他们吸引最后买下他们的产品。我上 次去重庆洪崖洞游玩就被一个变魔术的济公吸引去买了一幅 字，买完走出景区就后悔了。

之后我一直思考为什么当初我会不明不白的就买了那副字， 我仔细回想我从被吸引去看魔术到最后买下这幅字的这个过 程中，那位变魔术的济公其实用了很多销售技巧，我当初就 是不知道这些销售技巧才会被一步步的吸引进去，真的是认 知差距决定赚钱能力啊。

李笑来在 2011 年的时候，看到了比特币这个东西，他就觉得 比特币以后一定会升值，他就买了很多比特币，最后成了中 国的比特币首富。有的人可能会觉得他运气好，很早就遇到 了比特币。他肯定不是中国第一个知道比特币的人，但为什 么就他成了比特币首富。

决定他成为比特币首富的不是运气，而是他的认知能力。他 懂一点英语，懂一点互联网，懂一点编程，懂一点数学，懂 一点金融，懂一点心理学，也有很专业的研究理论。所以他 成功了赚到了很多钱，而很多人对区块链根本不了解，他们 只是随大流，看大家都把钱投进了区块链领域，所以他们也 把钱投进去，最后就变成被收割的韭菜。

在西方哲学史上有一个非常出名的人叫泰勒斯，被称为西方 哲学第一人。他生活在地中海，地中那地方空气非常的透

明，晚上的星星能看到很清楚。泰勒斯非常喜欢研究天空， 曾经预测了一次日食。有一天晚上泰勒斯边看星空边走，一 不小心就掉到了井里，幸好井不是太深，他就爬上来了，但 是因此就有人说“哲学家就是那种只看着天上的星空，却忘了 脚下还有一口井的人”。

后来人们喜欢嘲笑他只会抬头望天空，他为了向人们证明， 只要他想挣钱就能挣很多人。我们知道希腊那个地方产橄 榄，但很多结果子的植物都有一个特点：有大年和小年。今 年（大年）丰收了，明年（小年）产量就会低。

泰勒斯凭着他的知识发现了一个大家都不知道的秘密：这一 年是大年，第二年会接着再出现一个大年——这种事情是比 较罕见的，是黑天鹅，但关键是你只有抓住黑天鹅才能赚 钱，这是索罗斯的思路。

橄榄丰收的时候（大年）会建很多榨油坊，但是到了第二年

（小年），由于橄榄歉收，没有橄榄可榨，一些榨油坊就要

被卖掉，或者以很低的价格出租，这个跟资本市场是一样

的。

泰勒斯预言到了第二年还是一个橄榄产量的大年，所以他以 很低的价格买下或者租了很多的榨油坊。到了第二年橄榄果 然再次丰收，而且比前一年的产量还大，一下子榨油坊就非 常紧俏，泰勒斯就低价买高价出租，赚了很多的钱。

通过这件事，泰勒斯想告诉大家：我一旦认真起来，跟你们 的竞争就是一种不对称竞争——这就是“认知优势”，并且通 过“认知优势”获得了竞争优势。事实上，自有人类文明以 来，总有这样一些人，通过自己的认知优势来获得极大的竞 争优势。

我们身边也有一些人挣钱很厉害，大部分人不是运气好，而 是他们认知比周围的人高。

有一句格言说得好：“在盲人的国度，只有一只眼睛的人也能 成为国王”，这就是典型的通过认知优势而获得的竞争优势。 2019-01-31(22 赞)
