# Day102《你对

《你对工作和生活之间关系的理解可能是错的…》

先说说我自己。

成家之前，我是把生活和工作融为一体的，我随时都准备工 作，工作和学习使我快乐。

成家之后，经过了几个月的调整，我初步建立了生活和工作 的平衡，当然平衡是相对的，更多是建立了平衡的意识和理 念，如今生活让我滋润（甚至滋润得有点发福了）。

我一直相信，凡事都是有代价的，过分投入工作，生活就会 受到一些影响；重视个人生活空间，工作投入就会不够，毕 竟一个人的时间和精力总是有限的。

我甚至认为大多数人都是这么想的。 直到今天看到了一个很有意思的新观点：应该停止追求工作

生活的平衡。

布劳尔是一位社会学家，在她看来，我们使用的语言会影响 我们的思考方式，“平衡”是一个很局限的概念，对我们毫无 帮助。她具体给出了三个理由：

1、工作和生活的平衡论，是人为地把生活和工作对立起来。

本来工作即使无法带给我们快乐，也是我们每个人体现社会 价值的重要方式，如果对立起来，就会失去这种价值感，工 作可能会变得疲惫——只是一个谋生手段而已。

相反，如果把工作看成是生活体验的一部分，就会致力于通 过工作提升生活整体的幸福感。

2、试图平衡工作只会给我们增加更多不安全感。 为什么这么说呢？“平衡”这个词本身就代表了一种随时会被

打破的状态，我们生活注定充满了不确定性。比如本来计划

享受一个周末，突然收到加班通知。强求平衡只会让自己更

心累。

比如我最近一周多，工作上需要额外付出一些时间，生活、 学习、写作等只能牺牲一些时间了，如果强求平衡，更会疲 惫不堪。

3、追求工作和生活平衡，会导致我们把目标定得过低。 布劳尔认为，在生命中的某些阶段，我们确实会有很少的个

人时间，需要把时间更多投入到家庭或者工作中。这是生活

正常起伏的一部分。

如果我们把目标定得过低，就会降低对自己的要求，变得得 过且过。

总之，不要把注意力放在追求平衡上，而是主动在所有工作 和生活中获得积极体验，对未来充满期许并希望自己在工作 和生活中有所建树。

以上，希望对你有所启发。
