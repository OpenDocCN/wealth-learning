# 《人生幸福的四种象

蓝子鱼 : 《人生幸福的四种象限》

《幸福方法课》这本书中作者沙哈尔教授给了我们一个幸福 的四象限，代表了四种不同的人生态度和行为模式。

第一种价值观是“现在幸福，未来不幸”的及时行乐型。沉迷 于网络游戏的人就是这样的类型，他们知道自己这样下去是 没有未来的，但是他们就是管不住自己，非打不可。

我们生活中也会经常遇到这样的人，他们拿着很低的工资， 做着很轻松的工作，每天按时上下班，回到家就打游戏，或 者和朋友出去撸串，从来不学习。他们现在虽然看起来很快 乐，但是等某一天家里需要钱的时候，他们就会觉得自己很 没用，甚至会抱怨自己没有一个好的出身，其实他们只要努 力一点完全能够拿比现在高很多的工资，但是他们就是不想 这么做。

第二种价值观是“我现在很不幸，我未来可能也没法幸福”的 无助型。很多心理不健康的人就属于这种类型，他们根据过 往经历觉得自己现在很不幸，以后也会很不幸，这种很容易 做出一些伤害自己或者报复社会的行为。

郁抑症患者的人基本都是无助型，他们会感觉人生像走在一 个黑暗的隧道中，永远看不到前方有产生乐趣的可能，觉得 所有东西都没有什么意思，干什么都提不起精神。

还有一些心理变态的人也属于无助型，他们会把自己所有的 不幸归结于是这个社会给他带来的，所以这些人经常会做出 一些危害社会的行为。他们不会想到自己现在的不幸是由过

去造成的，而未来的生活是由现在所做的一切决定的。

第三种价值观是“我现在不幸，但是我未来会幸福”的忍辱负 重型。那些喜欢说等我考上大学以后，我就要怎么样怎样； 等我的公司上市了，我就可以过上财务自由的生活，这就是 属于忍辱负重型，他们能够忍受当下的不幸，把一切希望都 寄托在未来。

曾经在第二次世界大战的时候，有一位犹太人被抓进了集中 营，有一天他梦到了两年后的某一天，纳粹主义将会失败， 他们就会被救出去。他开始过忍辱负重的生活，每天还积极 锻炼，但等到了他梦到了那天，纳粹主义还没有失败，他也 没有被救出去，他就变的很消极了，最终死于一次感冒。

忍辱负重型的人是永远不会幸福的，忍辱负重是幸福最大的 陷阱。

第四种价值观叫“我现在很幸福，未来也很幸福”的幸福型。 快乐有不同的样子，要能体会到此刻的快乐。如果有一个人 学习的时候，一直想做其他的事情，那么学习的时候对他来 说就是痛苦的。我们都应该明白每个人都有一段时间是需要 用来学习的，而且学习了以后能给我们带来意想不到的收 获，那他学习的时候就会是幸福的，未来也同样会幸福。

马云曾经说过，他刚开始创业的时候是很幸福快乐的，因为 他知道自己在做什么，他要要创造一个伟大的公司，他要让 天下没有难做的生意。所以马云就属于现在很快乐，未来也 很快乐的人。

我们可以有欲望，可以想买豪车，可以想住大房子，想出去 旅游，这些欲望是值得被尊重的，只要你不被这些欲望折 磨。我们要为未来努力，但也要开心的享受努力的过程，这

才是幸福的方法。
