# 《所有成就感的背

云 :

《所有成就感的背后都是无尽的代价》

大家好，我是名字有点普通的云同学 [月亮]今天是我第 35 次 的分享，我要成为更好的自己[玫瑰]

工作会给你带来成就感吗？ 这是上周加班时，同事给我的一个问题。 当下我并没给出答案，因为疲倦，也因为怀疑，干着额外的

工作，我能获得成就感吗？

直到活动结束的那天，看着热火朝天的气氛和孩子们满意的 笑容，想起这三天的工作，我得承认 :虽然很苦，但确实有成 就感。

但很显然，“成就感”三字的背后是来之不易。

成就感的背后，是无尽的代价 加班那一周，除日常工作外，还要协助开业活动的进行，从

物料设计到流程跟进，用三个字就能概括 : 忙不赢。

两天加班到凌晨，失去休息时间，这是代价。 体力损耗，身体疲惫，这是代价。

写作中断，思考停滞，这是代价。

而且获取成就感这事跟写作一样，最怕陷入自嗨，成为了感 动自己却感动不了别人的傻子。

那坚持下去的动力是什么？ 引用我老板的一句话 : 我只想把事情做好。

成就感有代价，做事需取舍 想把事情做好，就意味着付出。 注意力是稀缺资源，你在这里倾尽全力，就必须在别处适当

放弃。

像学习这件事，从 21 天的写作训练营到最近的听课分享，时 间不够用的感觉越发强烈。

选择了学习，你就得放弃很多的乐趣和时间 。 比如今天我听了混沌大学的大课，占据了一天的时间。课程

内的学术性词语，十分烧脑，让人听得一度想放弃。

虽然坚持听完了全程，只能理解 30%-40%，但却也收获了坚 持学习的成就感。

成就感的升级，应该是使命感 获取成就感已经不容易。

但成就感之上，还有使命感。

混沌大学创始人李善友老师的个人经历就是最好的体现。 面对使命这个词，他是这样描述的 : 将你的产品、你的企业当

做是生命一样去看待，把它看得比你的性命还重要。之后你

会发现，带动你前进的将是你的使命。

李善友老师曾经是一名深度抑郁者，在一次偶然的机会，他 发现了自己的使命 : 打磨好混沌大学的课程。

于是，他变得不再那么抑郁。他花了八年的时间去打磨课 程，这八年里他没去商演，没在外授课，从一名好酒之徒， 成了社交恐惧者。

为了解决课程中的难题，他在斯坦福学习了一年，清晨而 出，戴月而归，不曾外出游玩。过年时，也就在晚上吃了顿 饺子，成了当时最勤奋的学生。

直到今天，混沌大学成了著名的学习平台，影响到了越来越 多的知识付费者，李善友老师的使命得以不断丰富和延伸。

在分享他的个人经历时，已经是课程的尾声了，汗水浸透了 他的衬衫。他坐在讲台前，脸上洋溢着笑意，流露而出的一 种分享的喜悦与满足感。

我想，这种极具感染力的画面就是对使命感一词最好的阐 述。

功。
