# 《小三是第三者，他

《小三是第三者，他们更是》

这是程先生的太太的第十八次分享，希望我给了你正能量。 周六晚，十点，程先生语音过来查岗。正好我在和朋友一起

吃饭。

我听到的是和他一起的朋友说：哈哈哈，你老婆没有你在身 边过得更开心了，还以为你出差你老婆会孤单可怜，原来不 是哦。看来你在她心里可能没那么重要吧？

一字一句，我听得清楚。 程先生，他耳根子极软，谁的话他都愿意听，都觉得有道

理。就是对我的话要思虑再三，还要和我掰扯。

头一年，我听他说的最多的话就是：我朋友说…………我觉 得很有道理。

他还总是喜欢和我说别人家的婚姻故事，我听了小半年，实 在是忍受不了了，才不许他再说了。

我朋友 A 的姑姑，总是喜欢参入她们夫妻之间的事情，特别是 关于钱的时候。姑姑是老公的姑姑，喜欢在她老公耳边吹 风，常说你要管好你赚的钱，别给你老婆，她要是拿着钱跑 了跟别的男人，那你不是杨白劳了吗。

她们俩常常因为这样的事情吵架，没两年，就离婚了。

我知道后，震惊不小。

我朋友 B 喜欢和朋友们分享他和他老婆间发生的不开心的事 情，然后让大家评评理出出主意。他是男生，说：不好和老 婆吵。结果受了委屈就喜欢叫上朋友喝几杯。

让我感到意外的是，每次他这么一吆喝，真的有好几个愿意 凑一起的。

我从来不参与别人婚姻里的事情，最多人家吵架了说句冷静 冷静，再好好聊聊。我真是宁愿闪的远远的。

不是我不愿意帮忙，而是我不愿意成为人家婚姻里的第三 者。婚姻是两个人的事情，两个人之间发生的问题，就得是 这两个人自己解决，其他参与到这其中的人都是第三者。

试想如果我帮人家解决问题了。对朋友来说，我可能是个大 善人，要么以后他们间出问题了就找我解决，总有一天我是 解决不了的时候，要么他们以后不再出任何问题，我这是一 次性帮忙，但这可能吗？

对我来说却不一样，我这就好像你偷窥了人家的私密活动， 扒拉了朋友的外衣，把人家的丑看得一清二楚了，我就好像 别人婚姻里的第三个人。

看得多了我会不会心里受影响？会不会把相同的问题往程先 生上打量一遍又一遍？就算你能解决朋友的问题，那你能解 决自己的问题吗？不一定的。

而那些在程先生耳边说着我俩的事的人，都是我们婚姻里的 第三者。主动以什么过来人，朋友好心提醒的外衣窥探着我

们俩的生活，甚至试图影响我们俩的生活和感情。

前文提到的程先生的朋友就是这样的人。在他的思想里我要 是过得开心就是心里没有程先生，就是不爱程先生。这种人 就是没事爱瞎参和，看热闹不嫌事大。

婚姻里要警惕这样的人，生活中更要警惕这样的人。 许多人都说婚姻很脆弱，很容易就不能圆满了。如果说婚姻

就像是鸡蛋，那么那些凡是在双方婚姻里参和进来的人都是

石头。这些石头不断的和鸡蛋碰撞，怎么能没有裂痕？

婚姻需要经营，需要小心呵护，但这和别人无关，只和夫妻 双方有关。婚姻就是两个人的世界，多一个人都太多余。

愿我们各自有不一样的幸福。
