# 《一条评论引发了我

《一条评论引发了我的思考》

昨天瑞幸咖啡的一组文案火了，很多人把它拍下来传到网 上，在另一个星球里有好几个发了，大多数人觉得这组文案 写的很好，但也有人觉得写的不好，其中一条评论引发了我 的思考。他是这样说的：文案不怎么样······无焦点无特点， 也没有价值观。这个人能说出这些看起来很专业的词，他应 该是从事相关行业或者学过文案课程的人。

我虽然不了解文案，但我觉得它很成功，很多人愿意把它拍 下来并且发到网上，说明很认可这组文案，被发到网上的时 候，瑞幸咖啡已经实现了二次传播，已经算成功了。

得到产品有一条标准就是，用户听完以后能转述，这样能实 现品牌传播，罗振宇跨年演讲上每年都会请用户帮一个忙， 就是拍一张现场照片并配上“我相信”发到朋友圈。这些人发 朋友圈的效果比做广告好一百倍，这些人的特性决定了他们 朋友的特性，触达的都是高质量的目标用户。

对于那个觉得这组文案不好的的人，他很可能在学习上走火 入魔了。他了解一些好文案的标准，所以他见到文案的时 候，第一反应就是用这些标准来衡量文案的水平，一旦不符 合标准就会认为这不是好文案，但他却忘了文案最基本的目 的。

很多商学院毕业的人创业都容易失败，虽然他们从商学院学 了一堆理论知识，但根本不了解创业，遇到事情只会生搬硬 套那些死的知识，当然容易失败。

一个人的知识水平是由两部分组成，一部分是人最底层的逻 辑思维，一部分就是我们学到的各类知识。如果把一个人比 做一台手机，那么底层逻辑就是操作系统，各类知识就是一 个个的 app，有的人能及时更新系统，而有的人只会装上各种 app。如果系统不及时更新，app 装的越多反而越卡。

能自动更新系统的人才算是学习，那些只会装 APP 而不会更 新系统的人只是在缓解焦虑，并没有成长。
