# 《不要高估短时间的

柒月 : 《不要高估短时间的努力，更不要低估十年的付出》

现在很多人都希望自己一夜出名，或者短时间就能获得很大 的成就。而如果没有时间的积累，短时间的爆火，就像烟花 一样，瞬间绚丽，却不能长久。

凡是可以速成的东西都不长久，甚至可能是骗局。

苏大强去银行存钱时，有人给他发了一张收益 9%的理财产品 传单，他立刻请同事老聂一起去解情况。

理财公司的首席顾问给他介绍了一款零风险还能保本的产 品，收益高达 20%！他听后立刻投了一万多。

第二天，就收到了短信提示：收益 200 元已到账，两个月就能 赚 800，他立刻又加 4 万，做起了“发偏财”的美梦。

直到某一天，理财公司人去楼空，首席顾问”的电话成了空 号，他当场晕倒进了医院，醒来后就要跳楼。

短时间内获得高收益的同时，也伴随着巨大的隐患。如果没 有相应的能力，就会直接跌入谷底。

作家李尚龙说过，这是个快餐时代，快到你总希望今天努力 明天就有结果；快到你总觉得今天跑步明天就能减肥成功。

可是，就算是再快的时代，也需要平静不计回报的寂寞时光

去积淀，去一步步安静地积累，随着时间的堆积，才能有明 显的改变。

如果一味的求快，就像拔苗助长的禾苗一样，因为缺少养 分，终将枯萎。

我们追求快的同时，更要看能力能不能达到。不然爆红后， 留下的只是满地疮痍。

心理学中的飞轮效应是指：

“为了使静止的飞轮转动起来，一开始你必须使很大的力气， 一圈一圈反复地推，每转一圈都很费力，但是每一圈的努力 都不会白费，飞轮会转动得越来越快。”

当达到一个很高的速度后，飞轮所具有的动量和动能就会很 大，使其短时间内停下来所需的外力便会很大，便能够克服 较大的阻力维持原有运动。

只有持续坚持，才能让我们的成长飞速，达到一定时间后， 可以像指数一样飞速成长。
