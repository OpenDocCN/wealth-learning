# 《你的习惯性否定，

《你的习惯性否定，会让你变成“反正做不到”的人》

别让现在的不做成为以后的不能——第 200 天的星里浣 今天看到一个小故事。

【俞敏洪说：“我上大学时从来没有追过女孩。我一想到要去 追一个女孩子，我就先想到我自己。敲我这副样子，长得这 么难看，农村家庭出生，成绩又不怎么样，普通话又说不 好，我要是去追，肯定百分百被人拒绝。”

既然百分百会被人拒绝，那干嘛还要去追呢？索性就不追 了。

直到后来，俞敏洪才明白：是否拒绝，是女孩子的事，而要 不要追是自己的事。女孩并不属于你，她拒绝了你，你也没 有失去什么，而且你去追，她还有可能答应你。】

这个故事让我心里出现了一个词：习惯性否定。 所谓习惯性否定，就是遇见事情先告诉自己不行，然后再找 N

个理由来证明自己不行。因为自己从来不去做，所以“做不

到”的事实也就一直摆在那里，极其具有说服力，也害人不

浅。

我也有一个伴随我二十年之久的习惯性否定：在 20 岁之前， 我一直告诉自己，我平衡感和方向感很差，是很难学会骑自

行车的，而且去骑自行车肯定会摔跤。既然会摔跤，那就干 脆不去骑了。

于是，我 20 岁之前都没有学会骑自行车。 后来，随着恭喜单车的普及，我靠着第一代摩拜单车学会了

骑自行车。

当我在朋友的指导下，踩上踏板，坐上车座，把住龙头开始 蹬的时候，我才发现这没啥难的，之前一直以为极度缺乏的 平衡杆和方向感一下就找到了，我也很快学会了骑行。

那是一种又开心又感动的心情。当然，也有着些许遗憾，当 年一个玩得好的女孩子叫我去骑车，然而我不会......

今年春节回家，我摸了一把老爸的车，四个轮的，我也能开 着走啦。虽然仅仅是在平地发动汽车绕圈圈，但这已经让我 非常开心且有成就感了。

当我像去学自行车一样，握住了汽车的方向盘，我发现我也 是能把他开起来走的。而之前我一直习惯性否定自己，认为 我反正也学不好，还不如不学。

今天看到的这个小故事，让我回忆了自己两次学车的经历， 也让我明白了：你的习惯性否定，会让你变成“反正做不 到”的人。

就像蔡康永说的那段话：

“15 岁觉得游泳难，放弃游泳，到 18 岁遇到一个你喜欢的人约 你去游泳，你只好说我不会；

18 岁觉得英文难，放弃英文，28 岁出现一个很棒但要会英文 的工作，你只好说我不会；

人生前期越嫌麻烦，越懒得学，后来就越可能错过让你动心 的人和事，错过新风景。”

就好比，我当年因为不会骑车 没能和那个女孩一起去骑车。 如果我不这两年不学会开车，那么以后有个女孩约我去自驾 游，我难道要说：“我不会开车哎，要不你去找别人吧？”

别让现在的不做成为以后的不能。
