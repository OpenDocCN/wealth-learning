# 《成长的路上，为什

《成长的路上，为什么总是遇到打击？》

你有没有注意这样一个现象，西方的超人总是趴着飞，而中 国的神仙则是站着飞。为什么会这样呢？

被虚构的人物有着不同的形象，一定与人们的认知和环境有 很大的关联。

超人趴着飞，是因为有空气阻力，这个动作能减小空气阻 力；而中国的神仙被杜撰时，根本没有空气阻力这个概念。

所以，再荒唐的现象背后，一定是有道理存在的，就像那句 话说的，存在即是合理。

或许你也一定有过这样的经历，在成长的路上，我们得到的 本应该是鼓励，但却往往得到的是打击。成长不是很好吗？ 绝大多数的人都想成长进步呀，为什么总是会被打击呢？

我们经常说，遇到问题多从自己身上找原因，遇到别人的打 击一定是有原因的，我们一定损害到了别人，否则也不可能 无缘无故被打击，不是吗？

每个人都追求进步，都想成为更好的人，都想变得不一样。 在生活中，总是有人会说某某某又在传递焦虑，或者说他的 行为会让我们感受到焦虑。

身边有个朋友，每天都很自律，天天泡图书馆，参加学校组

织的很多比赛，也拿到了不少奖励，大三就拿到了世界五百 强企业的 offer。

而我却天天沉迷游戏，逃课，熬夜追剧，上课刷抖音。当我 知道这个朋友拿到 offer 的那一刻，变得特别有点不安，我这 种专业本来就难找工作，相当于毕业即失业。

以上情况我能不焦虑吗？ 别人在进步，如果我们原地踏步，与他相比，本来在同一起

跑线，而我是在相对退步的，甚至是被退步。

反过来，在我们成长的路上，我们进步了，就证明别人退步 了，有谁愿意被证明相对退步呢？

而很有可能是，我们还没有进步，在计划某个习惯时，发条 朋友圈让身边的人都知道了，一个习惯的养成，都需要相当 长一段时间，我们很可能最终会放弃。

所以，我们在拿尚未发生的事来刷存在感，为什么会被别人 打击？答案显而易见，是我们让别人不开心了，我们占了别 人的便宜，活该这样。

所以，下次再遇到别人泼冷水，我们一定不要将对方定义为 坏人了，因为本来就是我们自己的问题，是我们让人家不开 心了，是我们做错了。

那我们应该如何做，才不会被打击？ 一个人默默完成进化

我们想尽一切办法成长，渴望进化成为另外一个物种，简单 说就像想提升一个层次或者阶层。

进化是需要长期规划的，想想我们进化成为人类不也花了几 千年时间吗？所以想要进化为另外一个物种，不是今天能拿 出来展示，证明的。

所以千万不要拿尚未发生的结果说事，进化尚未完成之前， 不说，不表现，不好为人师。

在我们默默进化的过程中，如何变得受欢迎？ 做一个鼓励一切人的人

既然我们讨厌被别人打击、泼冷水，我们都知道，那会让我 很不爽，那样的人也不会被别人喜欢，我们也绝对不会成为 那样的人。

相反，我们要反过来做，成为一个鼓励所有人的人，至少那 不会让我们被别人讨厌。

鼓励一定是这世界的稀缺资源，那么做似乎对我们没有坏 处，但是却有两个十分明显的好处：

第一，我们会成为一个受欢迎的人。别人期待得到我们的鼓 励，我们拥有稀缺资源嘛，不仅拥有，还无私地奉献出来 了，无论怎样，你都会喜欢的，对吧？

第二，我们会成为一个不需要别人鼓励的人。我们就是正能 量本身，已然体验到它给我们带来的好处。

鼓励很简单， “我支持你”，“加油，相信你一定会做成的”， 虽然是简短的一句话，却力量无穷，造福他人，更造福自 己。

从今天开始，鼓励他人，也鼓励自己！
