# 《像呼吸空气一样，

《像呼吸空气一样，时刻注重塑造自己的个人品牌》

昨天有个学习群的文艺女孩居然加了我的微信。 长这么大，很少有人主动加我微信，我当时都开心死了。

这个文艺女孩加我时，这么写通过语的： ”你好，冒昧打扰，只是想多认识一个朋友，能有机会和你交 流一下读书或者自我提升方面的心得吗？“

我看到这个，立马就通过了她的微信请求。显然她知道我这 人喜欢读书和投资自己的学习，而我也喜欢和读书和投资自 己的人交流，所以就通过了她的微信请求。

你一定很好奇：她都不认识我，怎么知道我这人喜欢读书和 提升自己呢。

因为，我和她都报了一个自我提升的课程，近期正在上，和

《粥佐罗 21 天写作训练营》课程一样，每次学习完成都要做 作业。

我的作业是比较认真、突出的，且每次学完课程我都会把我 的课堂笔记发到学习群里和大家分享。那个课程是限时免费 的，所以学员真正认真做笔记的比较少，但是那个课程的质 量非常高。

就这样，我成为了群里有些人关注的焦点，那个文艺女孩就 加了我微信。

加完微信后，更有意思的事发生了，她模仿了我塑造个人品 牌的技巧。

我通过她的微信后，立即把我早已准备好的自我介绍发给了 她。

直到昨天晚上，她才回复了我的消息，也给我发了她的自我 介绍。

有意思的是，她的自我介绍格式完全按照我的模板，且自我 介绍中表达了她当时没有及时回复我消息的原因是，她需要 花费 20 多分钟时间来写完自己的自我介绍。

她模仿我的自我介绍模板，显然认为我的模板很不错。 而我的自我介绍，是在之前就撰写好了，并收藏在了我的微

信“个人品牌”标签下。

因为我非常注重建立个人品牌，包括微信头像、昵称、朋友

圈动态等都散发着我自己独有的个人品牌，所以产生了“个人 品牌"这个标签。

你肯定也想像我一样有异性或者同频次的人主动加自己微信 好友。

那你可以像我一样：像呼吸空气一样，时刻注重塑造自己的 个人品牌，包括微信昵称、头像、朋友圈动态，以及平日里 言谈举止等等。

终有一天就会吸收到你想要的东西，包括微信好友。
