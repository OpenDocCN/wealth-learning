# 《关于开始，我想说

《关于开始，我想说的话》

最近因为做写作训练营，被问得最多的就是 我基础很差，不知道怎么开始

我现在学还来得及吗，我学得会吗？

我也想做新媒体，不知道行不行，我该怎么转行？

如果我说，来得及，先学起来，你可以的。 等待我的一定是一系列关于自身困难的描述，然后说“我怕学

不会/我怕来不及/我不知道怎么开始。

最后问我，我要不要学习课程呢？我其实蛮想说，既然那么 纠结，你就别学了吧。留着钱在兜里，把时间省出来去干点 让自己开心的事情，那也很好啊。

可是一个月以后一年之后呢？你还是原来的自己呀，你省下 来的那些钱和时间，似乎也没干成什么事呀？搞不好看到别 人的成长和进步，心中还有点小懊恼自己怎么还是停留在原 地。

不试试看，你怎么知道自己行不行，还没有任何的付出，你 问会不会有收获？我最不会回答这类问题，因为我无法预知 还未发生的事情啊。如果我说，一定会有的，那也是不负责 任的回答。

比如我自己，不会花太多时间纠结只有做了才会有答案的问 题。比如说，来不来得及？有啥来不及的，人只要还活着就 来得及；学不学得会？绝大部分学习没有难到需要挑战你智 力的程度；

关于怎么转行？直接转啊，就像你要走路一样，先把脚迈出 去呗。从实体店主转到我完全不懂的新媒体行业，过程中真 的没有那么多纠结，无非就是买课程学习，先后找相关的工 作边学边做，一边提升能力一边寻找机会。

举个简单的例子，你心里喜欢一个姑娘，还没开始送出第一 支花，你问我以后跟她能不能生完儿子，我怎么知道？

我只知道，学了是不是一定能成，但你不去试一试，就一定 不成，这个道理如此简单，简单到很多人都忽略掉。

没有任何事，是不需要付出就有回报的，也没有任何成长， 是一蹴而就的。比起去纠结结果会怎么样，先开始付出吧。 2019-04-12(28 赞)
