# 【今天，我赚到了见

白茶 : 【今天，我赚到了见识】 今天岩馆来了一摄制组，目的是拍取一些女主角攀岩的片

段，我第一次看到拍电影的拍摄过程，很是有趣～

我数了一下，摄制组来了大概有将近二十人，一开始，我惊 讶于怎么会需要这么多人，当实际开始拍摄的时候，才发现 各自都有自己的萝卜坑，需要按照导演的安排去做，导演负 责整体调度。除了化妆师，没有女性，我想大概是搬运众多 设备器材需要耗费体力吧～

因为是高空拍摄，负责灯光的需要升降机升到大概 10 米高的 位置，负责人怕高，申请把机器安置好，人下来，请求未通 过，我看到他脸上表现出的忐忑，但也是硬着头皮上去了， 当自己对自己狠不下来的最好办法，就是让别人对自己狠， 这就是为什么有的人需要一个教练，大部分时候教练充当的 是督促的角色～

开拍了，我负责演女主替身，第一次体验了一下拍戏的过 程，前前后后拍了有十次了，每次都有不一样的问题需要重 拍，好在这是我比较擅长的项目，多爬几次也不感觉累，这 要是让女主来演，估计就累劈了，想到这里不由得觉得发明 替身的人可真了不起，请专业的人来做专业的事，节省时 间，提高效率～

拍摄结束后，我看了一下回放，仅仅只是一个拍粉（攀岩的 时候，腰上会系一个小口袋放在身后，里面装上镁粉，目的 是手出汗的时候，抹一下镁粉，会增加摩擦性）的动作，画 面被放大放慢了，都特别带范儿，那镁粉就像赋予了生命的

活力，在空中翩翩起舞，这时想到粥老师写作课里说到:一个 作家眼里什么都可以写，只要发现生活中的那些‘刺’那一个导 演的眼里就是什么都可以拍，只要能拍出令人感觉意外的、 新鲜的视角就好～

从开始到结束，拍了 3 个小时，可能最后放到短片里的 3 分钟 都不到，想到 20 个人为了 3 分钟耗费 3 个小时，不禁觉得拍电 影这个造价真是太高了！同时，为了让电影呈现导演眼中最 理想的效果，不得不为 3 分钟的精华片段而准备足够多的素材 以备挑选。这让我想到了刘润老师为了准备 5 分钟的课程内 容，事先要做长达几个小时的准备工作；粥老师告诉我们要 持续、大量、优质的输入，通过自己的思考转化，才会有可 能写出好文章，讲的大概就是这个道理；同样，攀岩要想设 定出好的线路，脑子里要有足够多的动作库，才可以在设计 路线的时候任你调取使用，否则脑子里只有简单的几个动 作，那设计出来的线路大都千篇一律。

这一天，过得赚了。 继续睡觉去～

我是白茶，热爱攀岩，这是我在星球的第 62 次分享～
