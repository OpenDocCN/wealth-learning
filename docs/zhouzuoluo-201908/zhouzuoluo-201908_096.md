# 《最难走的路才是捷

《最难走的路才是捷径》

晚上好 与星友们分享丹儿的第 94 篇原创 7.30

————————————————————————

“最难走的路才是捷径”第一次看到这句话还是在粥老师分享 的一篇文章里。当时只是把它看作道理而已，没有想按照这 个道理去行动，毕竟，谁愿意为难自己呢？轻松点多好！

直到最近，开始创业，做了很多难的事，我发现，最难走的 路真的是捷径啊！就想和大家分享一下。为什么最难走的路 是捷径？

一，先和大家聊聊我们对捷径的理解 星友们是怎么理解捷径的呢？是不是用最轻松的方式，最快

抵达终点？

①惯性思维 我想最快抵达终点没有错，但是最轻松的方式可能并不能帮

助我们最快抵达终点呢！为什么这样说呢？

大家可以先思考一下，平时我们有一大堆事需要处理时，你 会首先处理什么事情呢？

是不是会挑轻松，容易的事先做呢？我以前是这样的，因为 不知道谁给我灌输的思想，先把简单的事做了在做难的，好 像是我妈

用这种方法做事很多年，我发现好像哪里不对，简单的事做 完以后，难得就不想做了，忙这么长时间我需要休息一下。 一休息就把事情耽误了。

所以导致时间花费了不少，但真正让我提升的事情并没有 做！这种先做简单的事的习惯一养成，简单的事越做越多， 但效果却不明显。而困难得事总想放一放，后来遇到困难尝 试一下直接就想放弃了。

大家有过类似的经历吗？不仅是我，我的身边也有很多和我 相同习惯的人，稍微做点困难的事，就不想做了，闲麻烦。 我身边认识的不闲麻烦的人屈指可数。

②少有人走的路速度反而更快 现在，和大家回到最初的问题，为什么说最轻松的方式不能

最快抵达终点呢？

那么多人都想用轻松舒适的方式，咱们竞争多大呀！

就好比两条路，一条看着路很平，离终点近，一条看着路不 好走，离终点还远。你愿意走哪一条？

离终点近的那一条呗！选的好，不止你选择了这一条很多人 也选择了这一条。现在请大家脑补一下国庆假期的告诉公 路，或旅游景点。人那么多，一步都走不动！

而少数选择路不好走的那群人，因为不拥挤反而速度会更快 的到达终点。对吗？

二，分享一个创业过程中的例子，经历了这些我才发现捷径 其实是最难走的路。

①优质的售后服务 因为哥哥姐姐手里有一些资源，所以前期创业就做一些上门

的美甲，美睫，皮肤管理。安全的问题我之前也想过，所以

没在网上挂资料去的都是熟人介绍的家里。

但是哥哥姐姐也只是负责介绍，怎样留住顾客是我需要考虑 的问题，说技术好呢，其实比我好的还很多，顾客也不像以

前因为你的技术好就会成为你的稳定顾客。而且还有像河狸 家那样的大型公司也在做上门。竞争太大了。

那我要怎样凸显出我和其他上门美甲美睫的不同呢？我就想 出了售后服务。一般上门 100 多块钱，她们是不会做售后的， 因为两趟来回路费算起来就四五十。

我做的那些顾客有一个好处就是都是一片的，也就是说我补 睫毛，补指甲可以约几个人一天补节约路费。但即使是这样 赚的还是少。

所以我又想出了套餐的方案，让每位顾客消费的金额超过两 百，这样利润就高了很多。这还是学的樊登读书会的营销方 案。搞活动，并不是便宜就能让顾客买单。

在价值不降下来的情况下，给一些增值服务，当然这些增值 服务在市面上也是有价值的。我就选择了一些时间短成本低 的赠送。

效果很不错，几乎我去过的顾客我都利用赠送增值项目绑定 了她们下一次消费锁定顾客，而且每次去会有不低于两百元 的收入，顾客也觉得我还有售后服务体验感很好，愿意下次 还找我做项目。

有时一天三到四个顾客，刚开始工资就比上班高了。还很轻 松。这还只是开始，随着顾客的增多，建立信任感以后，我

还可以销售产品，或是雇一些兼职，不用做手工就能钱生 钱。

★所以，走那些难走的路，别人不愿意做的事，并不一定会 吃亏，只要你愿意去思考，去行动就可以从这些难走的路中 找到捷径。
