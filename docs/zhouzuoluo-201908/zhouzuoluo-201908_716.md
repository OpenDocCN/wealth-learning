# 《你可以有一百个理

《你可以有一百个理由放弃，但我还是希望你选择坚持》

坚持日更已经超过一百天了，本以为坚持一段时间后，可以 越来越熟练。但实际情况是，我感觉自己枯竭了。每天分享 的时间越来越长，质量却不见提升，甚至还有所退步。

所以，我最近一段时间特别迷茫。我在纠结质量并不高的日 更要不要坚持下？

于是，和几位星友说了我的困惑，想听一听他们的意见。小 峰哥对我说:你想一下是坚持日更困难还是放弃日更困难，然 后你选更困难的那一个。

说实话，当然是坚持日更困难。在我坚持这并不算长的时间 里，真的可以有一百个放弃的理由。

这一百天里，生过病、加过班、和领导吵过架、心情极度低 落过，每一次都有可能断更。

最危险的一次，是有一个推不掉的饭局喝高了，并不是我喝 得多，只是真的不胜酒力。喝完后到家就趴在马桶上吐了一 番，感觉身体已经要崩溃。

利用仅存的理智，我定了一个十点的闹钟，我想缓一下后再 起来星球分享。但那次真的是太难受了，闹钟我没听见，快 十一点的时候我迷迷糊糊有了意识，想到还没日更立马把我 惊醒！

我拿起手机确认还没到第二天，算是舒了一口气。说实话， 那天的分享质量并不高，但是在面对困难时没有选择放弃， 我还是有点为我自己骄傲的。

当然，我并没有想标榜自己多么努力多么优秀。我只是想说 放弃总可以有太多理由，有些理由真的无可挑剔、没有任何 瑕疵。但是，理由是说给别人听的，你真的能过得了自己这 一关吗？

就像我现在，我还是可以找出很多放弃的理由，但是我真的 不甘心。已经走出了第一步，我真的不想放弃。

所以，我希望自己可以再坚持一天、两天、三天…直到有一 天山穷水尽了，一个字也写不出来了，再说放弃。在这之 前，还是少找点借口，多一分坚持。

[我是懂先生，这是我的第一百一十三次分享，很开心遇见这 么优秀的你们。在星球的一年，我希望实现三个愿望，如果 有一天实现了，我会把我的故事说给你听。]
