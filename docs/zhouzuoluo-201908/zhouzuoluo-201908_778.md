# 《你未来的成就，取

老王 : 《你未来的成就，取决于业余三小时》 昨天，和在外地工作的老同学一块吃了个饭。在吃饭、聊天

的过程中，他向我吐槽了一下他遇到的难处。

这位老同学在外地的一家铁路施工单位工作，他的工作地点 不固定，经常会在不同的省份跑。在老家有一位处了很多年 的女朋友，就等着结婚呢！

他的遇到的问题是，如果放弃工作回老家，可能面临失业； 如果还在外地继续工作，将来和女朋友结婚，长年异地不在 一块也不是个事。

像这种情况吧！没必要纠结回不回老家，而是应该考虑如果 回老家工作，你能做点什么？

我给他提的建议是:在网上多接触些不同行业的人，打破自己 固定的社交圈子。多做些尝试，对新鲜事物保持好奇心。在 自己业余的时间去经营一下自己的职业备胎，你可以是学习 一项技能，也可以是做做自媒体。

就拿我自己的经历来说吧！2016 年，月薪可怜的只有 2000 左 右，下班后又有两三个小时的业余时间。那时候，我就想这 么好的时间浪费了挺可惜的。应该兼职好点啥，把那段业余 时间利用起来。

那时候，兴起一股做微商的热潮。不过我觉的卖货挺难的， 我选了一个给微商加粉的项目。由于当时是个小白啥也不 懂，所以需要去学习，无意间加入了一个付费的学习社群。

后来，加粉项目没做几天就放弃了，因为你要花大量的时间 去陪那些做微商的去解释、聊天。

不过在个学习社群里，认识了一些做事情的朋友，他们向我 推介了得到这个知识平台。关注到得到 APP 后，感觉内容特 别有价值，也付费买了几门课程。就这样，三天打鱼两天晒 网的学了一年多。

在得道上，有好多很牛专栏作者，都在强调写作的重要性。 我自己也觉得应该去学习写作，问题在于真是一点都不会 写，又找不到学习写作的课程，结果就不了了之。

直到 2018 年的下半年，我看到有个叫简书的平台是专门写作 的，下载后看了看那些新手写的东西，感觉其实没想象中的 那么难，所以自己也开始胡乱的瞎写起来。

在简书这个平台上，无意间认识了一位做微信公众号的朋 友，在聊天后，我才知道，写作原来是可以挣钱的。在得知 能挣钱后，我才开始站在读者的角度，去认真、用心的去 写。

写了两三个月后，又接触到了知识星球这个平台。一时兴 起，自己也建了一个收费的星球。在那些免费的星球发表一 些文章后，意外的收获了一些星友的付费加入。就这样，倒 逼我去输出更有价值的文章。

你看！就只是写作这么一件很简单的事情，却从 2016 年饶了 那么一个大圈，绕到了 2018 的下半年。所以，你还敢说，在 2016 年做的给微商加粉的项目是没用的吗！要是没去做那件 事，也不会有后面一系列的可能性，更不会认识到写作的重 要性。

你可能有时候会一时兴起，做了一件事情，后来因为各种原 因就放弃了。站在当时的节点看，做了这件事情后，对你的 生活、认知其实是没什么改变，你原先干啥的，现在还是在 干啥。

这时候，你会认为那是一件不靠谱的事情，做与不做都不会 改变自己命运的轨迹。这其实是很多普通人特别错误的思维 方式，他们只会站在当下的节点去看待事情，而不会站在多 年后的角度去回顾现在。

结束语: 还是老生常谈的说一句，人生没有白走的路，每走一步都算

数。你迈出第一步，也许没挣几毛钱，更谈不上改变命运。

但是，迈出的第一步会让你链接到更多的可能性，从而找到

改变你命运的事业。
