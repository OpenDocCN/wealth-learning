# 《把时间放到有价值

《把时间放到有价值的事情上并学会说“不” -- 读连岳文章<你

的屏蔽能力成就的你>有感》

我关注的公众号里，除了粥老师的置顶之外，就是连岳的。 今天我想谈一谈，我读的连岳的文章《你的屏蔽能力成就的 你》。

这篇文章给我带来了深刻启示，我把我的启示分享给你们， 希望也能对你有所启发。

这篇文章中，连岳首先就回答了读者，为什么他不写吴秀波 出轨的看法：明星的绯闻是最不值得花精力去分析的事。

从这篇文章中，我一共总结了六点： 第 1、2 点，说明作者的结论：为什么明星的绯闻是最不值得

花精力去分析的事？

1、有些传播力巨大的事情毫无价值，明星的绯闻就是如此。 传播力大的事情，却毫无价值，这并不值得我们去关注，我

们知道就行了，那些娱乐八卦，也就配我们分配那一点点的

注意力，让我我知道这个事情就行，我不会再去多余的关注

了内在细节，因为这样的事情本身毫无价值，即使你再怎么

深挖它，得到了一个结果，那也是浪费自己的时间，

2、你的时间有没有价值，决定了你这个人有没有价值。

你的时间分配在哪里，决定了你的价值在哪里。我想起了 我，两年之前，也是被王宝强离婚事件刷屏，老是关注着王 宝强马蓉离婚信息，还跑到微博下面去骂。

现在想起来，自己真是个傻逼，明明可以把时间发挥到更加 有价值的事情上，却在那里混混度日，跟随大流，丝毫不珍 惜自己的注意力。所以那时的我，那就没有什么价值体现。

第 3、4 点，是对第 1、2 点的补充。让我们知道娱乐行业给 我们带来的价值究竟有几何。

3、抢夺更多注意力资源，一直是娱乐业的核心诉求。 娱乐业的范围很广，什么娱乐八卦，什么抖音快手，什么种

优酷爱奇艺？这些平台，一大堆的，抢夺你注意力资源的信

息，各种五花八门的信息，扑面而来，冲击着你的脑海，而

很多人却特别喜欢这种信息。这就是娱乐至死的一个典范。

4、某些软件的推荐算法利用人性，不停地给你喂垃圾信息。 现在的抖音，不断的给你喂一些你喜欢的东西。这些沉迷于

抖音的人，就像被算法饲养的小动物，更不客气地讲，那样

的人就像是被饲养员用饲料饲养的猪。读到这里，你有没有

警惕呢？

第 5、6 点，说明面对这些垃圾信息，我们该怎么做？

5、你要有说“不”的能力。你不仅能够拒绝自己不喜欢的人与 事，当然，也包括你能够屏蔽垃圾资讯。

面对那些垃圾信息，我们要有说不的能力，不让他们侵占我 们的大脑，把他们，拒之于门外，不让那些别有用心的算法

推荐饲养我们。

当然，简单来说就是卸载那些娱乐软件。即使有一些娱乐爆 款，我们心中也要知道，注意力是放到自己的成长上面的， 那些娱乐信息想要抢夺我们的注意力，我们对此要说不！让 它们滚蛋，不要影响我们的成长！

6、我们要有价值判断，要有善良的选择，屏蔽垃圾资讯就是 善良的选择。

价值判断很重要，有了价值判断，你才会有善良的选择，屏 蔽垃圾信息就是一种善良的选择，当你判断出那些垃圾信 息，是毫无价值的信息，那么你就不会选择去看它们了。

你会选择的是那些对你成长有益的信息，是有价值的信息。 以上是我对《你的屏蔽能力成就的你》这篇文章的总结，大

家可以看看原文，写得特别有深度，我自己的总结思想深度

可能达不到能够解说那篇文章的程度，但是我真的从这篇文

章中得到了价值，当然，我也打赏了这篇文章，算是表达我

小小的感激之情。
