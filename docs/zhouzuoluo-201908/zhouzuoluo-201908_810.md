# 《对不起，我关注咪

《对不起，我关注咪蒙，拉低了你的朋友圈质量了》

这两天，听说有一个指标可以用来衡量朋友质量高低。那就 是“含咪率”，含咪率越高说明朋友圈质量越差。谁关注了咪 蒙，都要被无情嘲弄一旦，被认为是没品的表现。

很不幸，我也关注了咪蒙，拉高了朋友圈好友的含咪率，真 的是罪大恶极。如果被好友知道，我估计就要被拉黑了。本 想趁没人发现，偷偷取关。但最后的最后，还是没人心下 手。不是被咪蒙的美色吸引，她的图修美了多少大家应该都 知道；也没有被她的价值观洗脑，这点三观我还是有的。

但为什么还是不取关咪蒙？我认为不管她的三观多不正、文 章多虚假，她还是有很多可以学习的地方。若干年后讨论新 媒体，咪蒙是绕不过去的话题。

那怕是有很多槽点，她依然是新媒体领域的标杆，有值得学 习的地方，最差也可以把她的一些文章当成反面教材，以后 自己写文的时候可以有意识避开。所以，最后我还是让咪蒙 躺在了我的公众号列表。

每个人都有多面性，我们不能因为他的缺点就忽略了他的优 点。当然，我没有想为谁洗白。咪蒙的一些文章确实三观不 正，不值得宣扬，我也很讨厌，但她能做到这么大，肯定是 有可取之处，我们应该学习她好的一面，并向其学习。

所以，我觉得不能因为关注咪蒙就代表品味差，质量低，那

有失公允。不是说关注了就对她的价值观全部认可，可能只 是想了解一下她的可取之处，不能一棒子打死一船人。

这个世界越来越多元、包容，希望我们都不要带着偏见看 人，尊重他人的选择。
