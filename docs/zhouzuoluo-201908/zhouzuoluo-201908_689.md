# 《为什么别人进步你

《为什么别人进步你退步，你该如何提升自己》

前两天有个读者和我交流，其中问我一个问题。 自己做前端已经有一段时间，但是总感觉自己没什么进步，

问我有什么好的建议。

我回答你可能需要系统的学习下，多看看书，多和优秀的人 学习学习。

随后他却反问我一句：你是怎么静下心来看书的？看书完全 看不下去，如果看视频还好一点。

我继续回答道，你可以尝试找个女朋友看着你，他却说女朋 友看不住他，然后我继续说：那你想想如何才能给女朋友更 美好的生活呢？

似乎这句话触动了他，他不再说话。 其实不止这位读者这样，大多数人都是如此，人不喜欢学

习，不喜欢看书很正常，毕竟学习是反人性，是痛苦的。人

大多都会败给痛苦，宁愿吃生活上的苦，也不愿吃学习上的

苦。

为什么会出现这种情况呢？其实很好理解，因为生活上的苦 是被动的，而学习上的苦是主动的。人是奇怪的非常不喜欢 主动，更何况是吃苦

说到自己，之前我也是这么的一个人，每天上班好好干活， 但下班基本上都用来玩，开黑打游戏全是一颗上钻石的心， 并没有想去好好地提升自己，也许我认为好好上班就足够 了。

其实当时的认知是错误的，以至于那一段时间我的提升并不 明显，甚至是原地踏步，因为对知识的认知不够，以及对自 己认知不够，很难想到人需要通过学习来提升自己。

我们在毕业以后，大多数人已经停止了学习，在他们的认知 里，好好工作就行，学习对他们来说太难了。所以你如果告 诉他应该好好学习提升自己，是没有用的。

相反也有很多人，认为学习很重要，这样的人你不用说他也 会去主动学习，你认识的大佬中哪一个不是热爱学习的人？ 甚至是终身学习者。

所以你想成为什么样的人？学习 or 不学习？如果你选择学 习。那如何才能更好的进行呢？

淡淡我个人的观点，文章开头我们提到主动与被动。其实学 习也是可以分为主动与被动。

被动很好理解，我们从小到大上学其实都是在被动的学习， 很少有人会主动，当然这里并不是说一个都没有。从普遍现 象看是这种情况。

我们被家长催，被老师盯着，就这样一直被动的到了大学， 这时候没有人管你，基本上都是放纵的自我。这里突然想到 一首歌《爱阔天空》“原谅我这一生放荡不羁，爱自由”。

毕业之后，工作几年之后你会发现和有些同学之间的差距会 越来越大，别人房子车子，老婆孩子热炕头，而你在这边单 间，泡面，小电影。是什么导致的这些差距？

也许你会说他运气好，可能运气好也是一方面，但为什么运 气总会降临在别人身上，而不是自己呢？如果你善于总结， 你会发现那些走的你前面的人，都会去主动的学习。学习别 人身上的优点也好，学习知识也好，学习沟通交流也好，总 之他会主动的去让自己变得越来越好，而不像那些始终对自 己不够坦率的人。

那如何去主动学习呢？

1.我认为你需要从意识上认为学习的重要性，知识的重要性。 因为意识可以决定一个人的行动，比如当你意识到你喜欢一 个女生的时候，你总想着去追她。很多人不学习的原因就是 不知道还有学习这件事情，在他们的世界里，学习被他们遗 忘在某个角落。

有些人觉得下班玩游戏重要，有些人觉得下班继续学习研究 技术重要，你说谁能走的更远？

2.坦率的面对自身的问题与不足，通过学习去提升自己。人总 会自我感觉良好，觉的自己什么都会。如果你觉得自己是周 围的人当中是最聪明的，这是危险。

我以前也会自我感觉良好，总会有这种想法：如果给我这些 条件我也能成功。直到我读到了一篇文章里面讲述了人生中 的四个状态，从此醍醐灌顶。把它分享给你：

不知道自己不知道——自以为是的认知状态。

知道自己不知道——空杯心态，准备丰富自己的认知。

知道自己知道——抓住了事情的规律，提升了自己的认知。 不知道自己知道——永远保持空杯心态，认知的最高境界。 95 % 的人都处在第一个状态 其实超越大多数人很简单，就看你愿不愿做了。

3.道理都明白了，剩下的就是行动了。所以现在就开始行动 吧，这也是最重要的一步，但很多人都败在了行动的道路 上。这不恰恰是你的机会吗？

开始吧，不要焦虑，不要急躁，你今天的努力，是未来的结 果，不是现在。过去的作为，才是现在的结果。明白了这个 道理，你应该就知道如何去做了。
