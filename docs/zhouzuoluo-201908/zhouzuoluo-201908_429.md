# 《不懂得自我推荐，

《不懂得自我推荐，会让你错失好机会》

如果给你一个好的工作机会，你会自我推荐吗？ 可能大多数人会想，这有什么难的，当然会啊！事实上

呢，90%人根本不懂得如何正确的推荐自己。

粥老师在公众号上发了招聘编辑的文案，收到了上百份简 历，90%不合格。首先，并未针对他的需求，做一份适合的 简历；其次除了表达“我想来”之外，并没有表现出自己的技 能，能提供什么价值。

用粥老师的话总结来说就是：我什么都不会，但是很想来， 你养我啊。

凭什么啊？粥老师家里有矿么？ 粥老师又一次在公众号上发布了招聘助教的文案，这次直接

加我微信沟通，本以为会更高效，结果把自己给坑惨了。

为什么呢？ 第一种人：我不太行、我没有经验，但我想挑战一下。 我要的是有经验，完全能胜任的人。挑战？你自己去横渡长

江呗！

第二种人：我先来试试啊，想来咨询一下啊，我需要做什么 啊，需要发什么资料给你啊！ 需要做什么、需要发什么资料，文案里写得清清楚楚，这都 看搞不清楚，回家待着好了，试什么试。

吐槽了那么多，到底应该如何做呢？

1、分析对方的需求 比如说，粥老师需要的是写作能力强的编辑，简历中就要具

体地展现出这一点。像自己的写作经验，然后附带上优秀稿

件，甚至可以根据账号的调性重新准备新的稿件。

总的来说，就是尽可能展现自己适合对方的那个点。 2、别废话，迅速抓住对方的眼球 很多应聘助教的人微信加我，很多人会先寒暄一堆，反复问

文案里有的内容，这样的人我会当做阅读理解能力有问题，

直接 PASS。

最好就是单刀直入，直接按照文案里的提示展现价值。比如 有的人会直接发自我介绍：

我是***，热爱写稿，作品曾被某大号转载，在***训练营当 过 2 期助教，有相关运营经验，下面是我的点评案例和作品。

一下子就抓住了我需要的点，一个助教要具备的两个点：写 作能力+运营经验。

另外，直接截图整理成自己的点评案例，等于把自己的优点 变得可视化，比自己夸自己认真靠谱，来得更实在更可信。

3、别说自己不行

首先，表达出自己不太行、来挑选、来试试，我接收到的信 息就是“这个人真的不行”。

第一，如果你行，你这么说就是一种不自信的表现，pass 第二，诚如所言，你对自己的评价是不太行，那就是真的不 行了。

所以，我们招人是要招不自信、真不行的人么？当然不。 那么你是真的没有经验、目前还不太行，应该怎么表达？ 举个例子：我在写作方面的作品暂时比较少，但是我有参加

过几期写作训练营，目前稳定上稿，我愿意付出比其他人投

入更多的时间和精力，全力以赴地去工作。而且，我在非常

熟悉写作训练营的运营和点评，在这个方面我有几点经

验……

同样是写作经验不够，换了一种表达方式，至少能让人感觉 到，这个人有热情和真诚，有其他方面的优点。假如没有更 好的竞争者，还是可以赢得一线的机会，或者说这个岗位暂 时不用，以后或许有运营方面的需求，还有合作的机会。

总结：要争取好的机会，首先要弄懂对方的需求，其次快速 展现出自己的价值。
