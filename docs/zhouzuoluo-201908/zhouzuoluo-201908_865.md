# 《再“万能”的方法

《再“万能”的方法，你不去用，也是别人的》

我是争青泊。--遇强则强。争做优秀 80 后，与自己死磕到底。 这是星球连续写作的第 46 篇

粥左罗讲写作公众号，上午推送了一篇文章《万能写作法， 你一定要懂的写作底层逻辑》，抽出了时间，逐字逐句的对 内容进行学习。读完后，发现所谓的“万能写作法”与《金字 塔原理》这本书里提到的基本原理“结论先行，自上而下”很 相似，更与“总-分-总”的写作框架一致。既然这么普及的写作 知识，大家都清楚，可为什么自己写的还是不够好？

01 一看到“万能”两个字，你就会觉得“简单” 不论我们在学习何种技能，市面上、网络上都有各式各样的

方法教会你如何去提高，其实，我们要是用心将这些方法“归 还原处”，这已经都是你所学过的内容。有些知识可能小时候 我们就学过，只不过长大了就已经将它遗忘，或是不知存放 在脑中什么位置，一旦你被外部刺激后，就能激活和找到

它。

但找到就用了吗？没有，找到后通常的第一反应是，“啊，这 东西我知道，很简单”，简单到我们不想去用，了解就行了。

02 总想学“特殊的办法”，实际是在找捷径 很多具备大师级别的人物，对于技能的练习并不是通过什么

特殊的方法，而都是“站桩”式的基本功反复练习的，比如像 中国京剧艺术表演大师梅兰芳先生，在早期练习基本功时， 恩师吴菱仙教给他的基本功其中一项就是“跷功”，他搬来一 条板凳，上面放着一块砖头，让梅兰芳脚踏两根半来多长的 高跷站在砖头上，并要求站一柱香的工夫。起初，梅兰芳连 3 分钟都不到，就已经腰酸腿乏。可他刚跳下来，又必须马上 再站上去，因为一柱香烧不完，是不准下来休息的。为了练 出过硬功夫，梅兰芳的腿都站肿了。

如果现在，“跷功”是曲艺界的“万能方法”，你对此方法有了一 定认识，对于一些人来说，也未必会开始练习，而是想找到 那个“不痛苦”的方法。

03 锁定一项“基本功”，勤耕不缀 方法论，是成功人士通过不断的实践，总结出一套属于他自

己的“心法”，现在通过互联网的渠道，快速而广泛的被我们 认识和了解。通过阅读学习，我们当然想尽早成为和他一样

的人。

如果，我们有心去研究，先将方法论拆解，看看到底这个“结 果”是哪个“基本功”结出的果。你拆解的多了，就会发现可 能“师同一家”或是其他派别，然后做出选择，将主要精力、 注意力聚焦在上面，去实践它，练习它。剩下的交给时间， 在一段时间后我们会看到自己的成长和进步。

最后，突然想到好莱坞著名影星阿诺德斯瓦辛格曾说过的一 句话：“看着我健身，你是长不出一身肌肉的”。
