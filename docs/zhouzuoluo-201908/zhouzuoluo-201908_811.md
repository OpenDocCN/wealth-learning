# 《成长与心流》 晚

晚上好[玫瑰]今天是橙橙加入星球的 175 天，和大家分享关 于‘心流’的话题」

今天阅读了《心流》的第四章，我其实没看太懂这一章节的 内容，我字面理解是在分析：心流与游戏、宗教、文化、性 格、家庭环境的关系。

★但我最大的感受是：自我成长的过程也能体验到心流，而 心流或许也是推进成长的原动力。

接下来和大家分享我对成长与心流的理解：

1.  01

    心流的状态：

    心流就是最优体验的状态，书中提到当你在某个活动中注意 力非常集中，完全没有空闲去思索任何不相干的事，觉得自 己有足够的能力应付当前的挑战，并能清楚知道自己的表现 的回馈，自我意识消失，时间似乎在你这里与众不同，这样 的活动会给你带来强烈的满足感，你甚至不会去计较活动的 回报，纯粹为了活动本身而去行事。

2.  02

体验心流的日常活动有哪些？背后的原理是是什么？

书中比较常拿出来做例子的活动一般是倡导心流效果特别好 的，比如做音乐、攀岩、舞蹈、下期、瑜伽等，因为他们的 主要功能在于提供乐趣，所以有助于参加者进入极为愉悦的 心理状态。5

背后的原理是这样的：

①这些活动有竞争性的倾向，参与者会把技巧发挥到极致， 以应付对手的挑战。

②这些活动还有投机性带来的乐趣，比如在下棋过程中你赌 那一方会赢，它能产生一种控制不可知未来的错觉。

不管他们是什么样具体的活动都有一个共同点，它带来一种 别于生活和工作的心的体验，带给你一种新的发现和创造 感，在这个活动中它能促使你有更好的表现，是意识达到过 去连做梦也想不到的境界。在我看来这也是一种成长的过 程，因为这些活动给自己的成长也是心流活动关键。

03

如何在意识成长中体验心流？

任何能改变我们感知现实的活动，都充满乐趣，比如瑜伽， 瑜伽的练习过程，就是一个体验身心，自我探索的过程。

每次静坐的时候扫描自己的情绪、头脑、能量，窥探身体的 适与不适，感知身体能量的储备情况，甚至通过呼吸来提升 整个身体的能量层面，我觉得这就非常喜悦，因为通过瑜伽 的练习，我们发现了日常生活中难以了解的自己，我们是在 单独和自己对话，内观自己，这就是一种超乎常人的能力，

是可以培养出来的感知和觉察能力啊，当你发现自己有这种 能力的时候就是一种自我成长的满足感，也是一种心流的体 验过程。

其实意识是不可能扩张的，我们充其量只能就它的内涵重新 搅和和调换，制造一种扩展的错觉，而对身体的感知和觉察 能力都是通过意识这个媒介去达成的，瑜伽冥想知识其中的 一种方式方式而已。

书中也举了一个男孩正在学打网球的例子，他会经历四个阶 段：

第一个阶段：开始不懂任何技巧的时候只需要过了网就行， 对于他来说挑战不是很大也比较新鲜好奇，所以它可能会感 受到心流但不长久。

第二个阶段：练习技巧的时候，对于过网，他会感到厌烦。 第三个阶段：打比赛遇到一个比自己厉害的高手，他会开始

焦虑，因为悬殊太大。

第四个阶段：要么放弃，要么体验心流，从新确立一个跟技 巧难度相当的新目标，然后加强挑战击败他，就能回到一开 始时的心流了，

但第四个阶段与第一个阶段相比较，他是进步的，也是成长 的，因为他遇到的挑战比知识单纯的打过网更复杂，在这个 成长的过程中，整体阶段他是愉悦的。

▼小结一下： 心流与成长是存在很大关系的，我们不可能长期做同种层次

的事依然觉得乐趣无穷，比如出租车司机，比如工厂的流水 线，他们或许因此感到无比的厌烦，饱受煎熬，在这种情况 下，有些人寻求乐趣的意愿就会出事我们拓展自己的节能， 或者发掘运用技巧的新方向。

从这个意义上说，我们不仅仅在成长的过程中会体验到心 流，心流也成为了成长和发现的源泉。
