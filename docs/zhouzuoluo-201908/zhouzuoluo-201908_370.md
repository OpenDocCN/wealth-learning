# 新人报道，平时爱在

孙文锋深圳 : 新人报道，平时爱在个人公众号写点随意的文 字，都是对日常经历的一些记录和思考。没有太多的章法， 请小伙伴们指正。

第 1 篇：《踏实走好每一步》 去见杨向阳之前，程总发了他的简介链接给我，原来业内大

家都叫杨总为大阳哥。

“北有徐小平，南有杨向阳”，之前在媒体上看到过对大阳哥 的一些报道，但不知道他在投资圈是这么高的元老级地位。

见到大阳哥的时候，是在他古色古香的会所里，和一个老朋 友在小酌畅聊，脸色已经微红，我看了下手表，下午两点 半。

“来来来，快过来坐，都是些年轻人啊，我就喜欢年轻人来我 这。”大阳哥热情招呼着。

桌上摆着一盘坚果，和一支有年份的威士忌。

”我现在不愿出去抛头露面了，时间就多了，就喜欢有朋友 来，吹吹牛，喝上两杯。“

一边和我们说着一边交待工作人员开了两支勃艮第产区的红 酒。

”我这儿酒都不错的，一起喝两杯吧，自己随意哈，我爱喝 点，但从不劝酒。“

在举杯中，3 个小时的聊天开始了。

大阳哥和他的朋友苏总都很健谈，讲了他们一路走来的很多 故事，也给了我们很多建议。

他说，在投资圈他是教博士级别的，不知道我们听不听的 懂。

我想是因为职业关系，天使投资回报周期长，充满高度不确 定性，极其需要智慧、耐心和毅力，造就了“大阳哥”见多识 广、反应敏捷、心直口快、热情幽默的性格。

回到的当晚，群里小伙伴们听说我去见了大阳哥，要我做些 分享，我说等周末吧，三个小时的内容太丰富，我得在静下 来的时候，才能做些整理和提炼。

其实，很多的道理我们都懂，难就难在，有几个人能做到知 行合一？

1，做事一定要聚焦。 特地用了自己的例子，他说早些年赚了不少钱，但亏了更多

的钱出去，总结自己亏钱的原因，都是因为同时在做着几件

事。

做一件事成功率如果是 70%，那同时做三件事，成功率就只 有 34%，那要是还有第四件、第五件呢？

他说，二十年的投资生涯，看了那么多的人和事，莫不如 是。

2，凡事要做窄。他说，在创业这条路上，窄门活，宽门死。

我理解的是要做细分领域的龙头，把自己细分到极致才能保 证生存。

苏总在一旁说，我们作为投资人，会看在你的领域，你是不 是全国冠军，如果不是，那是不是全省冠军，如果还不是， 那市内冠军总是吧，如果还不是，建议就别做了。

说完上述两点，大阳哥说到了曹德旺曹老爷子，当初选择做 玻璃被很多人不看好，还选的是汽车玻璃，结果，做成了全 球第一、福建首富。

3，一个公司先要做健康，再做强，至于做大那是顺其自然的 事，千万不要一开始就想着要做大。

这让我想到了成都那家餐馆，巴蜀味苑的案例，当晚我在群 里面也做了分享。

这间餐馆开了 20 多年，就火了 20 多年，是全国坪效最高的餐 馆之一。老板一直拒绝扩展，精心打磨菜品，很多名人都对 其赞赏有加。而这 20 年来，国内的很多餐饮同行都是一味做 大做强：融资+快速扩张，但最后活下来的又有几家？

4，一个人可以说大话，可以说笑话，但不能说假话。 大阳哥说，说笑话最好的方式是自嘲，马云这点做的特别

好。

5，人只能靠自己。 这个道理谁都懂。

整个聊天里，还有一点印象很深，苏总三次提到了芒格说的 一句话：”在中国这个国度，有太多人相信运气和赌博，这是 缺陷所在。“用芒格自己的话来解释：如果你认为自己的智商 值是 160，但实际却是 150，那你危险了。而以为自己 IQ 是 120，实际是 130，这就好很多了。

我们总会认为自己会是最聪明、最幸运的那一个，而现实却 往往骨感而无情。

踏实走好每一步，共勉。
