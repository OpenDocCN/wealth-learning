# 《父母尚在苟且，我

# 《父母尚在苟且，我

吉橙君 : 《父母尚在苟且，我炫耀诗和远方怎么了？》 佩文是我的同事，刚度过一个不愉悦的春节。

01

故事是这样的。

佩文来自贵州山区，农民的女儿，是家中唯一上过大学，且 在大城市工作的职场白领。

乡下家中有年迈的父母，哥哥，和嫂子。 她跟多数人一样，抢票，挤火车，回家。 但不一样的，今年她不再是学生了，开始上班赚钱了。 赚钱的好处，就是有能力为自己“剁手”了吧。 佩文也是一样，会追求时髦好看的衣裳，五色斑斓的美妆用

品，毕竟爱美是女人的天性嘛。

从土味乡下妹到都市丽人，佩文的改变不仅仅在外表和气质 上。更是变化背后，给她带来的自信，阳光，和向上。

归心似箭的佩文，期待着让家里人看到她的变化。 且跟着她一同回家的，还有网上置办的新颖年货，以及给父

母哥嫂带的礼物。

但万万没想到的是，在父母的眼中，这都变成了不合时宜的 败家玩意，换来的只有脸色和说教，诸如：

“衣服又贵又不保暖，几百块都可以买一件好的羽绒服了” “花钱容易赚钱难，好几千的手机是怎么下了手的”

等等...

网上有人调侃说，回乡前你是城里的 Lucy，回乡后你就是村 口的翠花了。

若用佩文自己的话来说，是因为实力她不允许啊，如果不跟 大伙一个样，注定是要饱受非议的。

02

春节期间，《流浪地球》火了，至今仍刷爆朋友圈。

佩文经不住自媒体的吹捧，架不住内心的好奇，遂有意请上 爸妈、哥嫂到镇上去看场电影。

不知为何，日常三四十元的电影票价，春节涨到 120 元一张 了。佩文略有纠结心疼，但想到这大过年的，就不要太计较 钱了吧，开心就好。

但是，两老可不是这么想的，在他们的价值观里，花上六七 百真金白银，看一场两小时的电影，没有比这更愚蠢的事 了，反驳的理由无非也是：“平时在手机上看电影，还不用花 钱呢”

佩文性子倔，认定的事便一往无前。在跟家里人大吵了一架 之后，一个人闷闷不乐把电影看了。

《流浪地球》格局宏大，视效震撼，若跟银河宇宙比起来， 人类实在是太渺小了。电影中吴孟达饰演的外公，在面对昔 日东方之珠陆家嘴，感慨到“那时候，还没有人关心太阳，大 家只关心一种叫做钱的东西”，佩文联想起刚刚的经过，实在 是有些讽刺。

但电影终归是电影，人始终要回归生活中去。令佩文没料到 是，后面还有更大的委屈在等着她。

当晚回家后，嫂子找她聊天。大意无非是父母种地供她上大 学不容易，知道钱来之不易，且人一辈子穷怕了，有钱就应 该存起来，以后总有用处。甚至还引用了“父母尚在苟且，你 却在炫耀诗和远方”这种标题党句式来暗讽她。

“父母尚在苟且，你却在炫耀诗和远方”这句话像魔咒一样， 萦绕脑海，久久不散。佩文一夜无眠，哭红了眼睛。

佩文是个乖乖女，每月都寄钱回家。所以她想不明白的是， 她在能力范围之内，追求较高的精致生活，难道有错吗？

03

佩文熬过憋屈的春节，一开工便找我倾诉，言及深处，泪眼 婆娑，问我该如何反驳“父母尚在苟且，你却在炫耀诗和远 方”这句话？

我说我不反驳，你父母没有错，你也没有错，错就错在这个 时代，它发展太快了，以至于我们跟老一辈人在认知上，价

值观上出现鸿沟了，我们分隔两岸，你过不来，我过不去。

那么唯一的办法，就是架起桥梁。而其中的关键，需要我们 想明白一个点，就是父母与子女之间，应该是怎样的一种关 系？

言以至此，作为人父的我，想分享一些我自己的观点。 还是讲一个故事吧，就是台湾有位编剧女作家，叫朱天心。 有一次，她在社区做保护流浪猫狗主演宣讲，期间有位老人

嘟嘟囔囔表示不耐烦，都什么破玩意儿，纯粹浪费力气，要

知道在我们那个年代，连人命都是拼来的！

朱天心莞尔一笑，答道“那么，就让我们做一些，你们那个时 代无法做的事吧，这样，才能让你们的牺牲，不至于白费。”

朱天心用了“牺牲”一词，我觉得非常贴切。在我看来同理， 父母养育子女长大，所付出的青春和精力，也是一种“牺 牲”。

那么，这样的“牺牲”到底是为了什么呢？

04

无独有偶，美国第二任总统约翰·亚当斯说过这样一番话：

“我必须研究政治和战争，因此我的儿子们能够学习数学和哲 学。我的儿子们应该学习数学、哲学、地理、博物、造船、 航海、商业和农业使得他们的孩子们可以学习绘画、诗歌、 音乐、建筑、雕塑、织物和瓷器。”

所以最后，我们再结合朱天心的故事，我想答案就很明显 了。

上一辈的“牺牲”，终极的意义就是在追求：一辈更比一辈 好，一辈更比一辈强，一辈更比一辈活的有意义。我们可以 谓之曰：“传承”。

上一辈粗糙，是为了让下一辈可以细腻； 上一辈冷酷，是为了让下一辈可以温柔； 上一辈低头，是为了让下一辈可以抬头； 上一辈现实，是为了让下一辈可以理想； 上一輩种树，是為了讓下一輩可以乘凉。

正是一辈又一辈的苟且，让我们有了本事去炫耀诗和远方， 去追逐星辰大海，去...

所以作为子女，请不用觉得羞愧，请不用难为情，请骄傲地 抬起头颅，请尽情展示你的丰富精彩。而作为父母，请不要 道德绑架，请不要泼冷水，请为子女有本事而骄傲，而鼓 掌。

当然，这一切的一切都是有前提的，是在能力范围之内的， 我们一定要跟“好高骛远”区别开。

总结一句话：子女越活越精彩，便是对父母最大的报恩。 所以新的一年，你该努力了！
