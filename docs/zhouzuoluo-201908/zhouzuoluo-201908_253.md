# 《缓解不安和焦虑，

牧心微笔记 : 《缓解不安和焦虑，你需要知道这个“独门绝 技”》

有一则哲理小故事。 小和尚问：师父，你得道前都在做什么呀？ 师父：洗衣，挑水，煮饭。

小和尚：那你得道后在做什么呀？ 师父：洗衣，挑水，煮饭。

小和尚：那你都沒有成长变化呀！ 师父：不，以前我是洗衣想着挑水，挑水想着煮饭。現在洗 衣就是洗衣，挑水就是挑水，煮饭就是煮饭。

你从这个故事里得到了什么启发？ 我得到的是正念二字。

所谓正念，就是对当下的实相保有觉知。通俗地理解，就是

师父的状态；洗衣时就想着洗衣，煮饭时就想着煮饭。

最近工作特别忙，各种日常性的工作和突如其来的琐事需要 处理，想想自己 8 小时之外制定的那些读书计划、写作计划、 健身计划都还没有完成，就不由地狂躁起来，甚至接电话也 会不耐烦。

而当夜里做到桌前看书码字的时候，脑海里又会想着：今天 某个事情处理得不好、不专心，还被领导指责沟通不力了， 糟糕透了！

这两者是我最近的常态，我非常厌恶，但却不得正法。

直到看到一行禅师的一本书——《正念的奇迹》，我才恍然 大悟：日常生活中要保有正念。

那究竟什么是正念呢？

举个很简单的例子，平时我们要洗碗，那你是为了洗干净而 洗碗，还是为了洗碗而洗碗呢？

“正念”的选择是——为了洗碗而洗碗。 如果洗碗时，我们还想着接下来要赶紧看个电视剧，喝杯清 茶，那么就总想着急急忙忙地把它洗完，好像这个碗很讨 厌，我们就不是为了洗碗而洗碗。

换句话说，洗碗时我们并没有“活在当下”，我们的思绪飘飞 到远处，完全体会不到“正念的奇迹”。 这说的不就是我自己吗！工作时想着额外的精进，学习时又 羞愧工作没有做好，我这一整天都没有活在当下！

长此以往，我们的情绪和精神，一定会被耗得很厉害，得不 偿失。

为了避免这种糟糕的状态再次发生，甚至影响我们的生活， 因此，我们要学会保有“正念”，工作时工作，学习时学习， 娱乐时娱乐，吃饭时吃饭。

在《正念的的奇迹》这本书中，一行禅师还告诉我们 32 个如 何修炼“正念”的小技巧：

比如清晨起来、闲暇时、发怒时、听音乐时轻轻微笑；平躺 并且全身放松；深呼吸；冥想；泡茶时保持正念；每周抽出 一天来作为自己的“正念日”，在这一天专心处理一些简单的

工作，比如打扫房子、做一次饭、洗衣服等等，随顺自 然…………

很多的小方法，其实根本上就是一个道理：干什么是，说什 么事，不要分心，不要胡思乱想。

记得很小的时候，在农村的院落里，看到隔壁家姐姐晾晒的 被子，洗干净的衣服，清新的味道和淡定的笑容，再配上风 和日丽的一个上午，想必，邻居姐姐当时的状态，应该就 是“正念”了吧！

愿我们在忙碌的生活工作中，都能够达到这样一种“正念”的 状态。
