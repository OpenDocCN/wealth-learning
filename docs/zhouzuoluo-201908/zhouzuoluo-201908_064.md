# 《马太效应：一步领

《马太效应：一步领先，步步领先》

[太阳]挑战日更分享第 3 天

你有没有觉得有的孩子一开始就上最好的学校，学习各种特 长，参加各种比赛活动，然后越来越优秀；

你有没有那些跟你同龄却早早成为高管的人，现在已经成立 公司，演讲、出书，跟你的差距越拉越大......

一步领先，步步领先，成功的人更加地成功，失败的人更加 失败；

这种“强者愈强，弱者愈弱”的现象，在心理学上被称作是“马 太效应”。

[玫瑰] 马太效应这个名字，来源于《新约·马太福音》，内容来自其 中的一则寓言： “凡有的，还要加倍给他叫他多余；没有的，连他所有的也要 夺过来”。

美国科学史研究者罗伯特·莫顿就归纳“马太效应”为： 任何个体、群体或地区，在某一方面比如金钱、名誉、地位 等方面获得成功和进步，就会产生一种积累优势，然后就会 有更多的机会取得更大的成功和进步。就是我们前面提到 的“强者愈强，弱者愈弱”。

强者更强很好理解，好的条件，好的资源，然后进入良性循

环，越来越好。

弱者越弱怎么理解呢？仅仅是因为先天条件不足吗？ 其实弱者更多时候是先有了弱者心态：

“我生来就是个弱者，家庭条件不如别人，长得一般，也没有

特长，能有个工作养活自己就不错了......”

这无疑给自己找了个不努力的理由，放纵了自己的懒散。在 别人拼命奔跑的时候，你还在原地踏步，甚至没有意识要去 追赶一下。这样的结果只能是差距越来越大。

[玫瑰]

那么，如何挣脱马太效应的怪圈呢？

1\. 从最基本可控的事情开始，修好每一扇“破窗” 一个房子如果窗户破了，没有人去修补，隔不久，其他的窗

户也会莫名其妙地被人打破；

一面墙，如果出现一些涂鸦，没有被清洗掉，很快地，墙上 就会布满乱七八糟，不堪入目的东西；

人也是如此，失败了任其发展，不去总结经验和教训，那么 它就会成为你的短板，久而久之你将满是“破窗”。

褚时健从“烟王”到“橙王”，一生坎坷。他刚到卷烟厂时，用褚 老的话说：员工软、散、懒；车间跑、冒、滴、漏。可以说 满是“破窗”。

他大刀阔斧地实施了一系列改革，工作激励、技术改进、原 料保证，亲自跑展会，亲自学技术，购买新设备，潜心修补

卷烟厂一个个的“破窗”，最终使得玉溪卷烟厂迅速崛起，他 也终成一代“烟王”。

所以遇到问题，静下心来，发现症结所在，去找到一一解决 它的方法，修好现在出现的每一个“破窗”，自信有了，能力 积累了，成功也就慢慢来了。

2\. 抓住每一次机会，让自己赢一次 不管你现在处于什么境地，你必须跨出艰难的一步，进入马

太效应的良性循环。

也就是说，你必须努力积累第一个大的成功，才能进入成功 孕育成功的良性轨道。

比如一个销售要先通过努力去获得月销冠，才可以复制自己 的成功经验去持续这种情况，然后才能寻求更大的目标，成 为季度、甚至年度冠军。

说到自身，最近已经坚持跑步第 12 天了，发现只要坚定目 标，只要它在你心里是如同吃饭喝水一样必须要做的事，你 就一定能做到。所以给自己定了每天写点什么的小目标，就 是复制了每天可以坚持做点什么的成功经验。

愿你早日找到成功的起点和突破口，获得第一次大的成功， 在优秀道路上越走越远。[玫瑰]
