# 《高手是怎样读书的

《高手是怎样读书的？这一句顺畅的话给了答案》

我曾经向很多读书高手学过读书方法，包括快速阅读和精读 以及市面上很多的读书方法，我从中能够过各取所长。

但是最近我读到了这一句话：「高手读书是先用感性能力帮 助自己选择，再用理性能力帮助自己思考」。顿时我就有所 启发了，然后就写了这篇文章。

这句话是什么意思呢？ 我先说一说彭小六的洋葱阅读法：其中有一个点是记录书中

能够自己“啊哈”的观点，但是这个“啊哈”的启发是什么呢？你 能否结合你的经验来解释一番呢？它为什么能够给你带来启

发呢？像这种灵感转瞬即逝。而如果你当时只是把它记到笔 记里面，没有去思考，你还是很难还原当时的“啊哈”的场景 的。

其实这个“啊哈”的这一点就是你用了你的感性思维。这种感 觉让你灵光乍现，眼前一亮甚至恍然大悟。在这个“啊哈”的 瞬间，你能够结合你的很多经验，很多知识，使得你的大脑 产生获得感和兴奋感。

为什么很多人读完一本书，觉得很有启发，但是过个几天又 忘了，你再回来翻书读原句的时候，又找不到当初那种“啊 哈”的感觉了，因为你没有把啊哈的感觉记录下来，你很难回 忆当初获得的感觉。「美好总是转瞬即逝的」这句话也不无 道理。

所以呢，当我们产生“啊哈”感觉的那一刻，也就是说我们的 感性思维已经感觉到了大脑的兴奋，此时就需要用理性思 维，把这个兴奋点记录下来。结合自己的经验，用自己的语 言把它表达出来，这样你才能够长久的保留这种获得启发的 感觉。

所谓用感性能力帮助自己选择，再用理性能力帮助自己思 考，正是如此。读书不思考就相当于。让文字从脑海中飘过 去，不留半点痕迹。那这样的读书只是徒劳的浪费时间罢 了。

而在成甲老师写的《好好学习》一书中也谈到了「结硬寨打 呆仗」这个方法。使用这种读书方法才是真正的让自己践行 这个准则：「少即是多，慢即是快」。

《好好学习》当真是一本大家都值得精读的好书。当然，我 也希望星友们能够学会这个读书方法。
