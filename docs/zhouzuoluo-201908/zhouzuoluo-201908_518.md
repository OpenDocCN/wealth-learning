# 《如何正确的复盘》

《如何正确的复盘》

大家有没有复盘的习惯呢？我是有这个习惯的，截止今天我 复盘 220 天，今天看完一篇文章，我才发现自己的复盘是无效 的，只是简单的记录，而不是深刻的反思。

一、复盘的方法 留意生活中最触动自己的点，不管是令人喜欢的感悟，还是

痛苦的经历，都把它写下来。

复盘的方式分为三点：

1、描述经过——以便日后回顾时能想起当时的场景；

2、分析原因——多问几个为什么直到有深度的启发；

3、改进措施——尽可能提炼出一个认知点或行动点。 对比我之前的复盘，我只做到了第一点和第二点，最重要的

第三点没有去行动，这样的复盘只是一种形式。

二、复盘的底层原理

1.符合触动学习法 正确的复盘方式是感知生活中最触动自己的点，而不是把今 天发生的事情记录下来。

每天只是单纯的记录，不去反思是没有效果的，根本不会有 所成长。

2.刻意练习

刻意练习就是带着清晰的要领点去学习。 像我之前的复盘方

式就不算刻意练习。

真正的反思是明确自己的行动，知道下一步该怎么去做。 三、开始行动

1.不要被形式所束缚 每日复盘是写下你的触动点，如果你今天没有很触动自己的 点，你不写也没有关系。 复盘的最终目的是为了改变，而不是形式的完美。

2.尽量提炼认知点或行动点。 你通过复盘明确了自己做得不够好的地方，这个时候你就要 去行动。

3.对自己坦诚 复盘是找到自己身上的不足，我们一定要对自己说真话，哪 里不足就去改正，没有必要欺骗自己。

4.多阅读 有时候我们觉察不到自己的触动点，这时候就需要大量的阅 读，去看自己感兴趣的书。

以上才是真正的复盘方法，希望对大家有帮助。
