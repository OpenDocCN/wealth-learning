# 《粥左罗：你的不可

《粥左罗：你的不可替代性，是优势也是劣势，是收益也是 成本》

来花 5 分钟我们一起辩证思考。 大家都说要提升自己的不可替代性，这个我非常认同，但同

时你要学会规避“不可替代性”的劣势。

你可能好奇，这还有劣势？ 当你问这个问题时，就说明你是个不习惯辩证思考的人。

因为万物皆有两极，没有一样东西，只有优势没有劣势。基 本上一个东西的优势出现时，劣势伴随而来。

比如我前面两篇分享的“老板是弱者，员工是强者”就是这个 道理。

那么一个人在职场中的不可替代性提高，会有什么劣势呢？ 第一，老板不需要一个不可替代的员工。 这个世界是矛盾的，你是小白时，老板想培养你，提高你的

能力，当你能力太强时，你的不可替代性越来越高，老板就

开始害怕你、提防你。我讲过我在创业邦的很多精力，但有

一个没讲。当时，我在创业邦写出非常多 10 万＋，而我的写 作充满人格化，同时我打造了个 IP 叫扒哥，时间长了，我在

整个号的读者群体里越来越出名，份量越来越重..... 我本以为 这是好事儿，其实不是，任何一个老板都不需要一个不可替 代的员工，他需要你能力很强，但不是不可替代，因为一个 员工不可替代，对他来说，太危险，你走了怎么办？

当时公司里很多人觉得是我把这个号做起来的，很多读者说 要是没我写，就不爱看，这都是危险。所以有段时间，领导 间接的告诉我，扒哥不属于我，这个 IP 属于公司，而且不允 许我在文章末尾加个人信息，甚至有一次，总编突然开会要 求，所有人在公众号上发文，都不能写个人的名字，只能放 在网站上时标作者名字。这个对我的打击非常大，我极力维 护，吵了几架，才保住了署名权。后面我也开始弱化那个形 象，而是强化作者栏里的粥左罗。

后来的职业生涯里，我明白了一个道理：做人要学会示弱， 要学会掩盖自己的锋芒。你可以很强，但千万不要流露出太 强的不可替代性。否则公司会给你暗地里加很多牵制。

第二，你的不可替代性越强，你的可移动性越差。 我在之前的分享里说了，员工和老板比，很大的一个优势就

是可移动性强，而老板是和公司深度绑定的，他不能想离开

就离开。

一个员工能力越差，可移动性这个优势就越强。比如一个餐 厅服务员，他随时可以撂挑子不干，明天就可以找一个更好 的餐厅上班。

一个员工的能力越强，不可替代性越强，他的可移动性这个 优势就越差。比如我在插坐学院时，我和公司相互高度配 合，我成了一个新媒体名师，公司赚到了很多钱，我的不可 替代性很强，但我的可移动性很差，试问除了插坐学院，哪

个公司还需要这样一个新媒体名师？我当时也观察了业界， 基本没这样的公司。所以这也是我选择破釜沉舟自己做号去 的原因。因为我当时离开后，不可能再找到一个那样的公 司，我只能说再找一个大号做主编之类的。

这个不可替代性强是如何导致的可移动性差呢？ 薛兆丰老师讲过两个词：专用资源和通用资源。 这就好解释了。 一个人在一个公司里越往上走，他的种种能力、资源、人际

关系等，都越是围绕着这家公司展开，他越这样，他对这家

公司的贡献就越大，同时，他越这样，可能他离开这家公司

后，这些东西就越没用，因为他积累的这些不适合其它公司

可能。

所以，这个人就越来越成为这个公司的专用资源。 我在插坐时，作为一个新媒体名师，就属于插座的专用资

源，而插坐的小编，他的不可替代性不强，他的工作换一家

也差不多，所以他就是插坐学院聘请的通用资源。

这就是为什么很多老板不太担心一些高薪员工走，因为老板 知道，你不好走。这也是为什么很多高薪员工离开一家公司 后，找不到那么高薪的工作了。

这个给每个人的启示就是，当你越来越强，在一个公司不可 替代性越来越强的时候，你要谨防过度的变成这家公司的专 用资源，因为那样的话，你很难离开。
