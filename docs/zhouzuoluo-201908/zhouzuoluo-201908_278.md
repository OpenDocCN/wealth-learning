# 为什么我说，世界上

为什么我说，世界上最长的路，不是套路，而是成长之路！

因为成长是一个终生的命题，是一种持续的状态！它贯穿现 在，通及未来，并持续存在。

【现在】:我们每天都在努力进步，思考输出，争取今天比昨 天再多进步一点点。

【未来】:我们每天这么努力，不就是为了未来可以过的毫不 费力吗？不就是为了未来有权利对不想做的事，不想说的话 say NO 吗？

【持续】:越成长，越牛逼，越觉得过去的自己傻逼。为什 么？因为你的认知在不断升级，再造，重塑，再升级...这个道 理就像我们读书一样，你学的越多，越觉得自己知道的太少 了，所以你会更加持续地输入，获得更多的知识。

意识到成长的重要性之后，我们迫切的想要获得成长，那有 哪些方式来获得成长呢？

有且仅有两种方式:

①自己亲自经历后获得的成长

②通过他人的经历获得成长

第一种成长方式，最为直接，也最真实，但成本高，毕竟一 辈子就那么长，时间有限，有些别人踩过的坑总结下来的宝 贵经验你好好听着，学着，以最快速度获得最大的成长值。

这便是第二种成长方式存在的必要了。

通过他人的经历去获得成长。具体怎么操作？比如我们加入 优秀的社群，和优秀的人结交，学习他们的优秀品质，做事 方式，比如我们买某个领域权威人士出版的经典书籍，比如 我们购买网络课程，这些都是相当于花钱去购买别人的宝贵 经验，而这些经验都是别人一步步经历而总结出来的。再次 说明”时间就是金钱”这句真理。

如果能理解这些，你就能想通为什么现在”知识付费”那么火

？因为社会的节奏越来越快，大家都很忙，都希望速成地去

掌握某个技能，所以去购买某领域知名讲师的课程。这便是

通过他人的经历去获得成长。

尤其新兴行业的职业技能，因为这个社会的快速发展而衍生 了很多新兴的职业，甚至行业。而学校课程与社会需求的严 重脱节使得人们没有途径去了解学习这些新兴行业的技能， 所以知识付费便顺势而生。

而这些，都是自己没办法或者没时间去亲自踩坑经历来得到 成长的，所以我们选择付费，为别人的有价值的经验来付 费。

如果你能理解上面这些点，便能理解为什么还有个词很火 了。叫”个人品牌”。因为每个人都有不一样的经历，只要你 的经历可以转化为对别人有价值的经验，那就值得被付费。 如果你可以根据自己的实际情况为他人提供有价值的信息， 并把自己打造成一个 IP，那便可发展为个人品牌，就像微信 公众号的 slogan 所说，”再小的个体，也有自己的品牌”。

以上，拼命成长不止是你年轻时一定要做的事，也是你一辈 子都必须坚持的事，因为成长，无止境！！！
