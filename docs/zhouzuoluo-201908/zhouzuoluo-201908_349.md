# Day185《顺势

《顺势而为，方能大有可为》

曾国藩熟读圣贤书，知道顺势而为的道理，但晚清的局面凭 个人之力已是无力回天，比如大清灭亡跟他间接造就的拥兵 自重的地方军阀，还有民间势力脱不了关系。尽管早已预 见，但趋势无法逆转了。

曾国藩早知大清国运已去，但明知不可为而为之，一次次拖 着病体，劳而无功，这是对国家的责任感使然，最终 61 岁不 到就猝死在了工作岗位上，可谓是鞠躬尽瘁。

说这些对我们个人有什么启示呢？我们比曾国藩幸运得多， 生逢其时，国运昌盛，大势之下给我们创造了无数的个人机 会，所以我们完全可以顺势而为，从而让自己的一生大有可 为。

今天跟各位老铁重点聊一个点：顺势而为，不断放大自己的 个人努力。

为什么有人付 100%的努力只能换回 20%的增长？反之，有人 付出 20%的努力，却能获得 100%的回报？雷军对此进行过深 刻的反思。

答案就是顺势而为，借助时代的趋势，借助外部的力量，来 放大个人的能力。那么具体如何去执行呢？可以分三步走。

第一步，盯紧趋势。除了低头走路，还要抬头看路，关注时

代造就的大趋势，还有所在行业的小趋势，多花时间进行观 察和思考，向有见识的人请教，跟一线城市的朋友多交流。 如果有条件，去最好的城市，去最热的领域积累资源、增长 见识，与优秀者同行。

第二步，做到稀缺。也就是围绕趋势，打造自己的核心能 力。如果迷茫和不确定，就寻找自己的优势、天赋和激情。 如果短期找不到，就先进入能进入的头部公司。如果进不去 头部公司，就在自己的公司成为头部，然后通过小头部不断 向大头部晋级。

第三步，找到杠杆。给自己的核心能力找杠杆，从而放大自 己的能力。什么叫杠杆？就是可以撬动更大成功的资源。比 如团队，资本，影响力等等。

最后，小结一下： 围绕趋势找到高价值区，然后专注打造自己的能力内核，并 通过充沛有效的资源杠杆放大你的能力内核，这就是顺势而 为的真谛，也是我们普通人大有可为的破局之道。
