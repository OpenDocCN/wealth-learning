# 《你的注意力是唯一

智多星 : 《你的注意力是唯一属于你、有价值、可产出的资 源》

李笑来老师说过，人拥有的最宝贵的资源就是自己的注意 力，它是唯一属于你、能产出价值、可无限复利的巨大财 富。

为什么呢？想想我们能拥有和把握的资源(财富)——钱、时 间、注意力。

钱，是可再生的，它并真正属于你，它只有在你决定要不要 花的时候才是你的，而且钱留不住也带不走。

时间，它也不属于你，你没有办法掌控它、驱使它、指挥 它，你只能试着和它做朋友，在有限的时间里做有价值、有 意义的事。

最重要的，注意力，它是唯一属于你、供你支配、谁也带不 走的，如果你把注意力投入在有价值、可复利、能打造时间 壁垒、变成你核心竞争力的事上，那它就能为你所用，给你 带来源源不断的财富和成长。

综上来看，注意力的确是人最应珍重和谨慎使用的资源。 同时，李笑来老师又告诫我们，注意力浪费的“三个大坑”: 1、“莫名其妙地凑热闹”。比如“哎呀谁又出事了”、“发生什么

事了”、“结果咋样啦”、“这谁人模人样的，原来 ta 是这种人

啊”(就不解释了，给大家提供个相同的情景)

2、“心急火燎地随大流”。在大趋势下盲目跟风，什么火就追 什么，比如知识付费、社群运营、比特币等，等别人做起来 了才想着赶快捞一把

3、“操碎了别人的心肝”。对别人的事特别关心，咋咋乎地， 讨论来讨论去，就像营销号上对名人的事说得好像很有理， 但就是在追热点、煽动情绪，然后底下的人也讨论得特别热 烈。

讲到这，我想向大家介绍，采铜老师的“时间之尺”，教人怎 样合理利用时间——“采铜法则”。介绍完这个，再讲讲我 对“注意力应该放在什么地方”、“要怎样小心合理地选择和使 用”的理解。

“采铜法则”主要包括两个维度: 1、收益值。某件事在当下带给我的身心收益，心理层面包括 认知、方法论、系统性思考，生理层面包括物质、身体、感 官反应

2、收益半衰期。某件事是否有长期影响，换言之，衰退时间 有多久

根据这两个维度，采铜分出了四个判断事情重要与否的标尺:

①高收益值，长半衰期。比如一本经典名著；一场信息密度

大的学习会、交流会；一门教人系统掌握某项技能、思维体

系的课。需要人在之后反复思考、练习，具有复利性。

②高收益值，短半衰期。比如给人强烈快感、脑子停不下来 的综艺、游戏、肥皂电视剧。看的时候特别投入、兴奋，然 后屏幕一暗、手一停，又想不起自己这段时间学了什么，做 了什么。

③低收益值，长半衰期。比如练习弹琴一小时、背单词半小 时、写一篇有主题的思考文。这种事只有一点点慢慢地坚持 做，才能带来长期、可持续的收益和回报

④低收益值，短半衰期。比如做一名吃瓜群众，在网上围 观、参与撕逼大战，实时关注、操心各种小道消息、八卦劲 闻。

结合采铜老师的工具和李笑来老师的告诫，显然①③值得 做，应该做，②④可以做，但得分清轻重缓急，在我们注意 力如此宝贵、有限的情况下，先把主要的事做了，在剩下的 休闲时间里选择性地做。

不过，根据李笑来老师说的“三个大坑”，我们应该尽量克制 自己，不做低收益短半期的事，损人不利己，还被有心人士 聚集眼球、变现流量。

拿我自己来说，虽然没有掉过“莫名其妙地凑热闹”、“操碎了 别人的心肝”这些大坑里，但对注意力的选择和使用上，我也 走了好多弯路，浪费了 n 多价值。

以前我想当然地觉得老师讲课很水，这些内容都不实用、都 很落后，这些教科书都是学校为了省钱让老师自己编的，所 以我上课从来不好好听，很自大地想，“不听课又怎样，期末 抓紧复习就行，反正班上的同学一大把都在玩手机睡觉”。

但是我后来看到洛帅说，如果连完整的一堂课都听不下来， 不好好把握大学的学习时光，那就会发现自己做其它的事情 也不会有集中的注意力、高效率、更突出的成就。(我的意思 不是，大学里应该一节课不落、全部认认真真、仔仔细细地 听。我想说的是，如果我连一件事都做不好，就不该只想着

兴趣与否，对这个时候的我来说，兴趣不是最重要的衡量因 素，而是想想现阶段力所能及和努力一点就能做的事，如果 我能忍下性子，做好自认为不喜欢的事，那我就有更多的自 信、踏实和方法去做自认为喜欢和其它更多的事上）

我回想以前，还真是这样，明明是在做自认为最喜欢、一辈 子都认定的事上，却每次都是心思动不动就溜到其它事，想 干别的，觉得烦、枯燥、无聊，不但没什么成绩(当然，能做 出来才奇怪)，还浪费了更多精力和时间，让别人觉得我怎么 这么没用，浪费那么多钱，还什么都学不会、做不好。

所以我就想不要在该集中注意力的事上找各种让自己心安理 得去偷懒的借口，把自己宝贵、有限、唯一能生产个人价值 的注意力资源用到真正重要、高回报、可复利和打造时间壁 垒的事上，其它就先不想了。

每次手痒想玩手机的时候就想注意力分散了，就得比玩手机 更长的时间把它集中起来做事，这也太不划算了，还是先忍 住，该干嘛干嘛吧。
