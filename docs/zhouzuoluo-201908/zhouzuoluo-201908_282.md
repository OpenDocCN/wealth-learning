# 《你有让别人当众吐

《你有让别人当众吐槽的经历吗？这样的益处大了去了》

刚刚和舍友在宿舍喝酒聊天完，大家一起坦白，真心话大冒 险，熟悉的人总是知道你的铠甲和软肋在哪里，也能够更好 的帮你解决问题。

我们围在一起吐槽彼此的缺点。其实自己也大概知道自己有 哪些问题，但我的舍友们和我说的时候，还是有些诧异于她 们的高度凝练。她们说我的最大缺点是：不食人间烟火。

我在现实生活中和大多数人聊不太来，因为大家喜欢的东西 不一样，没什么共同点，我喜欢静，她们喜欢动，我喜欢看 书写作，她们喜欢抖音化妆。不同很正常，但是由此我对她 们很漠然，也就是她们说的不食人间烟火。

我沉浸在自己的世界中，喜欢和线上的人交流，平时听课写 作看书做公众号的时候，注意力比较集中，她们叫我我会自 动屏蔽，有时候反应也慢半拍。这点经常被她们吐槽，但我 总是觉得自己有道理，没有去改过。

我没有主见，对于吃的，喝的，用的，玩的，我都没有什么 意见，她们问我我就说都行，不是我取悦谁，是因为我真的 无所谓。但是我喜欢的事情，我必须去做，做公众号，做书 咖，做公益，做助教等等，她们劝我不要瞎折腾的时候，不 行 我习惯遇事就问人，生怕自己做的决定错误，所以没有突 破，越走越没有底气。

我不懂人情世故，更或者说我不屑，对，这里有古代文人 的“酸”，我不喜欢野心勃勃什么都想要的人，我不喜欢表里 不如一，阿谀奉承的人，我不喜欢天天把钱挂在嘴边的人， 我觉得这些人很俗，很势利，很鄙视他们，所以很多事情我 会把自己“摘”出去。

大家指出来这些问题，其实我挺感谢的，只有真正对我好的 人，才会掏心掏肺地说出来，才会怕我一直走弯路没有进 步。我会努力去改正，活在真实的世界里。

我能够感受到自己的正向成长。我不完美，但同时我每天都 在变好。
