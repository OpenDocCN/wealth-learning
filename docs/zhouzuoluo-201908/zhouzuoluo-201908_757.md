# 《了解一个人，从看

蓝子鱼 : 《了解一个人，从看他如何花钱开始》 昨天粥老师写了一篇文章叫有些钱绝对不能省，否则你会更

穷，让我对花钱这门学问有了更深的了解。想了解一个人可

以从看他怎么花钱开始，这是一个很好的方法。

01 看老板的格局，就看他肯不肯为员工花钱 昨天粥老师讲到他让星里涴买了十几个星球，花了三千多，

他全部给报销了。粥老师还说:后面有值得研究的星球继续 买，他继续报销。文文加入的时候，他也给文文一些钱买

课、星球、书等等。

从粥老师肯为员工花钱这件事，可以看出粥老师是一个有着 大格局的老板。他知道这部分钱不能省，现在花的这些钱是 为了有一天撬出更多的钱。

过年同学聚会的时候，我问了一个做室内设计的同学一个月 工资多少，他说只有一千五还不包吃住，主要靠提成，他本 来这个月做了一单能提成 2200，但是老板说他是新人，只给 了他 700 块。他觉得老板太黑心了，过年的时候顺便辞职了。

我同学的老板就属于那种眼里只有钱，他不会考虑员工的利 益，也不会考虑公司未来的发展，他见到钱只想落袋为安， 这种老板永远挣不了大钱。

02 看一个人的格局，就看他舍不舍的为自己的未来花钱 有人宁愿给自己买一件 300 的衣服，也舍不得花 299 进入星

球；有人宁愿花一百多去吃顿火锅，也不愿买一门可以提升 自己的课程；有人宁愿往游戏里冲几千块，也不舍得为自己 报个培训班。

很多刚出来工作不久的人，由于工资较低，所以特别不舍得 花钱，更舍不得为自己的未来花钱。有人为了省钱，租在距 离公司很远的郊外或者城中村，每天需要花两三个小时在来 回的路上。住着最便宜的房子，挤着人最多的地铁。

他们只关注了金钱而忽略了时间，时间才是决定一个人成就 的最大变量，如果租在公司旁边每天可以省下在路上的那两 三个小时，可以多做多少事？可以选择多干一点活；可以选 择健身；可以选择和一个有趣的人喝一会儿咖啡；可以看自 己喜欢看的书；可以试着在星球写一篇文章。总而言之，这 些时间足以把你的人生各种丰富的可能性打开。

一个肯为自己未来花钱的人，未来才会越来越有钱。

03 看一个人爱不爱你，就看他肯不肯为你花钱 昨天是情人节，你送给伴侣什么礼物？伴侣送给你什么礼

物？早上在微博上看到了有一个女孩子在吐槽自己的男朋

友。说自己送给了男朋友 SK2 男士，而男朋友送自己一盒几块 钱的德芙巧克力和一个暖手宝，暖手宝之前已经送了两个

了。

网上有一句话:“为心爱的女人花钱，是一个男人的基本礼 貌”。钱不是衡量一个男人的标准，但是愿不愿意花钱是衡量 一个男人真心的标准。毕竟，浪漫情话虽然动听，却不需要 什么成本；浪漫的事情看似用心，却没有实际付出。如果他 爱你，整个身心都是你的，那愿意为你花钱也是自然而然的 事情。

谈钱并不一定就是庸俗的，张爱玲也曾说：“爱一个人爱到管 他要零用钱的程度，是一个严格的考验。”

平常我们看到的一个人可能不是这个人的真实面目，但在钱 面前每个人都是最真实的。
