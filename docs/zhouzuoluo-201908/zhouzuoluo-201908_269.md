# 《用行动把梦里的日

Branden : 《用行动把梦里的日子过成现实》

记得是李开复曾经说过，我们的一生可以说是用时间来换取 才华;才华越来越多,时间越来越少。如果一天天过去,时间少 了,才华却没有增加,那就是虚度了光阴。从这句话里，我想到 了今天学到的一个概念——成本。

我们做任何事情，都是需要成本的，我们付出成本去换取我 们所需要的东西，生命就是我们的终极成本。我们花钱买东 西，钱是我们付出的成本；我们花时间去学习知识和技能， 花时间去完成梦想，时间就是我们所付出的成本。如果我们 的成本足够多，理论上来讲我们就能得到我们所想要的一 切。而非常不幸的是，我们所有人所拥有的成本都是有限 的。生命是有限的，我们到了最后都会死，这是一个客观而 残酷的事实。所以，我们必须用有限的时间去换取更多的东 西，去努力完成梦想。

然而，在现实中，我们并不是每时每刻都在用时间去换取我 们所需要的东西。很多的时候，我们的时间在无意义地流 逝，除去必要的休息之外，我们做着一些和我们的目标不相 关的事情，这些事情占据着我们有限的时间，因而我们完成 梦想的效率就要大打折扣。

除了时间，我们每个人的“能量”也是有限的。我们每天所做 的事情，占据我们的时间，消耗我们的能量。每一天，当我 们在为自己的目标和梦想努力的时候，都会不同程度地受到 各种各样的人和事的影响，干扰着我们，销蚀我们有限的能 量。如果我们任由这些影响我们前进的因素干扰我们的话， 长此以往，我们将会离自己的目标和梦想越来越远，难以到

达。

所以，当我们意识到了这点之后，就必须更加敏锐地察觉到 自己是否被一些拖累我们的东西影响了，我们应该更加理性 地掌控我们自己。每个人的人生都是有限的，生命最后会终 结，身躯最后会腐烂，人存在的意义就是你用你这具躯壳在 有限的时间里做了多少于你生命有价值的事情。因而，我们 都要专注于自己的目标，更加高效地运用自己的时间和“能 量”去实现它，让生命变得更加有意义。

现在如果一个人谈目标和信仰、梦想和努力的话多半是要被 耻笑的。在这个鸡汤横行的时代，这些曾经激励人的词汇逐 渐让人麻木和反胃了。现实中的复杂问题不是仅仅靠努力， 坚持或者奋斗就能解决的。单纯的信仰和激励面对复杂的问 题时，一旦碰壁，就容易产生幻灭。因此，那些高大上的梦 想和信仰往往都流于空洞。

不过由于大多数人都没有过上自己想要的生活，每个人都期 待着过得更好，看到有句话说,并不是穷人才觉得自己穷，每 个阶级都觉得自己穷。所以这些刺激人努力的话总是很受欢 迎。但是很可悲的是，单凭这些话语却并不能让人的生活发 生实质上的改变，穷的人依然很穷，这些所谓鸡汤的话最后 成了人们吸食的“精神鸦片”，因为人们乐于听到这些自己想 听到的话，他们认为努力能使梦想成真，他们听着那些抚慰 他们心灵的话，以为这样就能改变自己的生活。

但是，真正能让生活发生改变的只有付出真切的行动。而 且，在现实到梦想或者说目标的路并不是一条平坦的大道， 而是一条充满崎岖和坎坷的路，有山坡，洼地，甚至有河流 和沼泽，一切你能想到和不能想到的艰难险阻都在路上。你 要跨越所有的坎坷，最终才能到达那所谓的梦想的彼岸。

某种程度上，我们很多人都是还没长大的婴孩，所谓的“巨 婴”。虽然身体已经长成大人，但是在心理上，还停留在婴儿 的阶段。在婴儿的时候，我们衣来伸手饭来张口，一切都得 于依赖，以至于我们认为整个世界都唾手可得。而我们长大 到了现实世界，发现现实跟我们所理解的不是一个样子。当 我们眼前发生的事情不符合心里的预期的时候，当他人的行 为没有按照我们内心所设想的那样去做的时候，虽然不会像 婴孩一样嚎啕大哭，但是我们会生气，会愤恨，会让自己陷 入难以自拔的痛苦当中。所以王小波说，人的一切痛苦，本 质上都是对自己无能的愤怒。

是的，在我们有限的生命里，我们会遭遇很多出乎意料的不 如意，有些事情让你恼怒和失望而你却无能为力，这个世 界，实际上并不如你的预期。我们都拥有这样或者那样的缺 陷，都会面临种种难以战胜的挑战，但是，我们是一个活 人，我们都拥有时间，我们可以用时间和行动去充实和完善 自己。用时间去成长去强大自己，让困住我们的外在世界变 得更加可控；用行动，把梦里的日子过成现实。
