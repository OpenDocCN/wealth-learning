# 《这些好习惯，让我

《这些好习惯，让我飞速成长》

昨晚和我学长聊天，我把自己最近的事情说给他听，他欣慰 地说：你终于有点进步了。

去年开始他带我读书，带我写作，开课带我做助教，平时还 总提供机会让我分享。

这些事情我没做过，没信心，也不会拒绝，我自己查百度， 上知乎去问，问好了做给他，他说哪里不行，我再修改。如 果我们意见不一致，我也会直接说自己的想法，不会只听 话，不问为什么。

一年多的成长，我总结了自己的一些好习惯，和大家分享一 下：

1.做事效率高，不会自己去学 我大三的时候课很多，有一次学长让我按照目录，把书里的

文章一篇一篇整理出来，那是他的第一本书，告诉我半个月

做好就行。

我接到任务那天下午就开始做，一直做到晚上十一点，检查 一遍没问题，踩着凌晨的时刻，给他发了过去。

第二天起来他看到了，直夸我效率高。我自己就有这样的习 惯：有任务就要立马完成，不拖延。

小时候经常是周五晚上就把周末的作业写完，10 天之内就把 寒暑假作业完成，每次开学前，好几个同学来我家抄作业[捂 脸]

还有一次他让我做分享，其实我不会，也不知道分享什么， 他告诉我随便来，放轻松。我就自己去百度上搜了一下，也 去知乎看了一下，再看看千聊里的免费课程，听老师怎么讲 的。

几个步骤做好之后，我开始写稿子，改稿子，三遍修改之 后，我给他看，他指出来的问题，我照着修改，然后就直接 去分享了。

事后，他问我怎么还想到回答问题了呢？他忘了和我说，但 我自己去研究，学到了。

自己去主动学习的东西，才是属于自己的，这个道理我一直 记得，也一直践行。

2.积极主动 我以前是一个特别内向性格的人，不善言辞，不主动，总觉

得处处表现自己太锋芒毕露，让人讨厌。

所以好东西只有别人给我的时候，我才会接受。主动去争取 的东西，我从来没上过。

有一次我考了班级第二，但是没有评上国家励志奖学 金，8000 元失之交臂，比我成绩差很多的人竟然能够得上， 我那一刻深深感觉到自己的懦弱。

从此我一改常态，开始自己积极去争取，去参加活动，去参 加竞赛，最后凭自己的实力得了优秀共青团干部。

在生活中的很多事情，我也积极，学长让帮个小忙，我积极 去做；有助教实习，我去报名；有各种课程，我都去积极 听。

这些事情只要我接了，答应了，我肯定是要完成的，要不我 会心有愧疚。

高效 积极，让我越来越幸运。

3.低调谦虚，用实力说话 我一直觉得自己很普通，身上不喜欢标签，可能因为我之前

总写励志文，好多人说我是励志学姐，天天夸我，但是很

累，这次考研失利我就很难去回复他们的各种关心。

进到一个新圈子，我会想怎么认识大神呢？主动加微信，感 觉有点不好意思，不知道怎么介绍自己，感觉耽误了别人的 时间。

昨天的分销群，在我暂时排名第一的情况下，好多人在群里 加我微信，他们想看我的朋友圈，想和我交流，那一刻我深 感实力的重要性。

学长之前有一篇文章是这样说的：优秀的人不是不混圈子， 只是他们的圈子里没有你。你不优秀，别人凭什么要认识 你？

我时刻记得这句话，所以低调再低调，是啊，我们都是普通 人，所以时间要用在如何变优秀上。

我是悦己，只要步履不停，我们总会久别重逢 2019-02-26(22 赞)
