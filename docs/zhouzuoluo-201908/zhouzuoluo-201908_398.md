# 《亲戚为何有时不如

李磊 : 《亲戚为何有时不如朋友？作家路遥讲的很明白》

在亲情面前，爱情都要排第二位。因为，亲情是生发我们其 他一切感情和社会活动的基础，承载着生命的基因密码，是 我们的根。

父母、子女是最直系的亲属，生来无法选择。除此之外，还 有带有血缘关系的亲戚。他们和直系亲属一起构成了一个人 生来最重要、最内向的社交圈层。

但亲戚不总是一个可以发挥正面作用的社会角色。 亲戚其实也分好和坏！好的亲戚，可以是彼此的贵人，互相

关心和帮助，而坏的亲戚，或只知占便宜，功利势力，或产

生破坏性的力量，累及他人。

这样的亲戚，出现在《平凡的世界》中，也在《都挺好》电 视剧中上映。

关于亲戚关系，路遥有一段深刻的描述: 是的，小时候，我们常常把“亲戚”看得多么美好和重要。一

旦长大成人，开始独立生活，我们便很快知道，亲戚关系常

常是庸俗的；互相设法沾光，沾不上光就翻白眼；甚至你生

活中最大的困难也常常是亲戚们造成的；生活同样会告诉

你，亲戚往往不如朋友对你真诚。

亲戚有时候不如朋友真诚。《平凡的世界》中男主孙少平切 实感受过。

孙少平有一个表舅，曾经帮少平介绍了一份在村支书家修房 的伙计，他本人也是短工的一员。在村长家房子快要建成那 天，要放合口石，但是少平的表舅却不慎弄破了手指，粘上 了血，表舅让少平帮忙隐瞒。在当地的习俗，这被认为很不 吉利。这件事也被村支书看见了，追问下来，诚实的少平没 有替表舅撒谎。表舅被村支书赶走，少平也因此得罪了表 舅。

之后，少平到表舅家被赶走，而且挣得钱都被表舅拿走。少 平此刻感受到了来自亲戚的伤害，内心很凉凄。

这是坏亲戚的典型代表，势利眼，有便宜占是亲戚，稍有不 慎就翻脸无情。

这样的亲戚，《都挺好》电视剧中也有。那就是明玉的舅 舅。他不仅喜欢跟娘家姐姐借钱不还，还觉得理所应当，觉 得所有人都应该对他好！当然，这都是明玉妈妈无原则宠弟 造成的。不过幸好记账了，这才让这位舅舅不能再无条件的 问苏家索取一切。

文学、影视作品中存在的这些势力、无赖亲戚，有虚构的成 分，但是实际社会也存在。当然，这样的亲戚产生的破坏力 也不会持续，一次拉黑就再也不来往，这是最好的方式。

如何处理亲戚关系呢？我觉得“君子之交淡如水”也适用。亲 戚之间不易走的过亲密，盛极必衰，时间久了必然会产生间 隙，导致关系疏远甚至恶化。亲戚之间也不能不往来，那样 会疏远，亲戚早晚成为陌生人。

正确的处理亲戚关系应该是:平时淡如水，保持走动，但不要 频繁，不要有经济上的瓜葛，有难时挺身而出。
