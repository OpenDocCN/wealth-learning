# 《高小羊：终其一生

《高小羊：终其一生，我们不会因为优秀而被爱，但我们依 然要优秀》

小学初中同学已经开始晒二胎，而我还在看动漫。 高中同学一胎已经晒了，我数了下，全班唯独剩下我一人依

然还在看动漫。

再数了下大学同学，有晒一胎的，有待孕的，也有像我孤身 一人的，但为数不多。

人这一生，在很多事情上，其实可以做的更好。 这么多年，剩下我一人的，不止在地域上，生活上也是，他

们回家发展的回家发展，结婚生子的也结婚生子。造成目前

这种境况的，其实归咎于我个人的想法。因为，在很多事情

上，我始终觉得自己可以再做的更好。

高中毕业的那个暑假，有一天我站在大门口，穿着脏兮兮的 衣服，等着送通知书的人来，我记得那天我不敢出门见人， 因为自卑，害怕，那时我以为我以后的人生就那样了。

高中时成绩并不好，但我强烈想上大学，学习到夜夜失眠， 最后终于考上大学，而分数比录取线多了三份，因为我觉得 相比起打工挣钱，读书我可能更加有把握。

大学毕业那年，从老家出来读省外大学的高中同学，以及我

在大学的同乡师兄师姐，都选择回家乡发展，父母吵了好几 年的教师考试，我一直无动于衷，还在扛着，因为我觉得相 比做老师，我做其他行业可能会做的更好。

我很后悔没一早开始去学写作，但其实想写作这个想法我是 从三年前开始就有的，现在想开始这件事情，因为我觉得我 应该可以把它做好。

在处理感情问题上，每当遇到对我有好感的，想对我表白的 那刻，我觉得自己还不够优秀，可能对方和我在一起后，对 我失望，因为我看起来并没有那么的好。

当我对某一个男性产生好感时，我当时的内心是自卑的，我 认为我还不够优秀，所以不敢主动。自卑是我此生最大的克 服不过的障碍。

当我没遇到有好感的，或者说对方对我没感觉的时，我知道 肯定是自己不够优秀，所以吸引不了别人。

所以，至此一生，截止目前，我觉得我是因为不优秀而不被 爱，所以就近乎苛刻的要去自己更加努力工作健身读书，培 养一切的所能涉及的爱好。

直到读到武志红老师的文章时，他说至此一生，我们不会因 为优秀而被爱。

不爱就是不爱，跟你优秀是否没关系，在爱情里，有太多的 变量，优秀是一个，除此之外，还有性格，爱好，趣味，身 材，思维，幽默等等。

那我们要放弃苛刻的自我要求吗？或者说还要必要做这些让

我们优秀的事情吗？

我认为要，当然要做，而且还有更努力的去做。 但是前提是你要在自己的能力圈内做事。如果你在自己的短

板里试图超越别人的长板，那你就是在放弃自己的优点，让

自己越来越没信心，围绕自己的缺点做事，怎么可能会有成

就感。

比如说我，口头表达能力和演讲能力一塌糊涂，社交能力基 本为零，如果我近乎偏执的要求自己在这几方面做到优秀， 不断逼着自己优秀，那就是病态的优秀。但是文字表达能力 是我的优势，我就能做到近乎偏执把它做好。所以扬长，请 忘记补短。

就像武老师说的，真正的优秀动力，只有一个：在被爱和被 欣赏的过程中，感受到自己的特别和珍贵，并发自内心地努 力。

我只能在自己擅长的那块，近乎偏执的严苛要求自己，做到 优秀。
