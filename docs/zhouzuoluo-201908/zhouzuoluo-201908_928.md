# Day59《请教别

《请教别人，给的答案不重要，重要的是找到答案的思维》

1、请教第一步，首先是有效提问 如果自己张嘴就问，没有经过最起码的思考，问题也不够具

体，是很难指望别人给你提供有效帮助的。

在我看来，有效提问，要做到关键问题清晰，能问到点子 上，也有足够的细节让对方明白你的处境和目标，你已经有 了哪些思考和结论。一个好的提问会吸引到好的答案，也能 碰撞出更多有启发的思路。

有了这个认知后，你会发现一些问题是属于简单问题，直接 上网就能搜到答案。但还是有人自己懒得搜，简单问题也去 问别人……这样问过几次，可能别人就给你打上了“伸手 党”的标签，甚至你问了一堆网上一搜就有答案的问题，别人 也花时间耐着性子给你把答案送到眼前了，你却连一点积极 反馈都没有，估计别人又给你打上了“不值得再帮”的标签。

2、请教的核心，是学思路，是找答案的思维 除了直接能搜到答案的简单问题，其实，在我们人生各个场

景中遇到的重要问题，往往都是复杂问题。凡是所有复杂问

题，你不能指望别人直接送给你一个答案，然后你照搬套

用，一切问题就都解决了。这种妄念越早戒除越好。

我们请教高手，去听许多人的分享，其实更多的不是“知 道”一个答案，而是梳理出解决问题的思路，从别人分析问题 的过程中获得启发，你拿来借鉴，可以迁移到更多问题的解 答上。

而思路，许多时候就是思维模型，去跟不同领域的高手交 流，你也更容易搭建起自己的多元思维模型，而不是用自己 固有的某种单一思维看待问题。

除了跟比你厉害的人，跟你差不多的人请教，甚至某些方面 不如你的人分享的东西你也认真去听，就是基于对思路的探 讨，而不是答案高明不高明。在这个意义上，虚怀若谷的人 是真正的智者，许多所谓聪明人就容易陷入“自牛逼”的状态 中。

3、请教的闭环，做整理和复盘，分享出去 请教之后，要学会感谢对方，除了口头上的，发个数额合适

的红包。然后结合自己的思考做整理和复盘，分享出来，对

方看到你的有效反馈后，会认为你值得帮，也能看到你成长

的潜力，这样你就不是单方面的索取，而是为更深入的合作

创造了机会。

分享出去的同时，有同样困惑的人看到后亦有所得。通过持 续做这件事情，你也积累了自己的人气（关注度和影响 力）。

其实一个人成长快，很多时候不是因为他学得多，而是他每 一次都真正学有所得，并通过各种方式放大这种所得。

同样，一个人能取得多大成功，也不是他自身有多么厉害

（因为厉害的人太多了），更多的是有多少人希望他成功。

请教这一件事，做好了，你能成长更快，也能更有机会获得 不被人妒的成功。
