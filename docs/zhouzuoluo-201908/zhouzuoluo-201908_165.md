# 大家好，我是小橙姑

大家好，我是小橙姑娘，一热爱写作，渴望未来能活成自己 喜欢样子过上自己想要生活的姑娘。

很开心能有这样的平台给以我们分享我们的成长和感悟。 这是我加入星球的第 6 次分享，还请大家多多指教。

「我后来为什么拒绝拿来主义?」 记得自己以前的时候，只要是朋友需要，或者微信小伙伴问

到我一些关于读书或者其他方面的知识，我都会耐心解答，

甚至自己去帮忙找答案然后直接把他们想要的结果发给对

方，这样也就不知不觉中给别人人很好的印象。

直到前几天，一 96 的小妹妹突然间问我，“你那里都有什么 群？”看到这句话之后，因为我之前都是直接把结果发给对方 的，而她这一次也是有点依赖性地来问我。最后我决定拒绝 拒绝拿来主义了。

一个是因为自己的无私帮助会给人一种理所当然地感觉，还 有一个就是直接把结果发给对方，会让对方形成依赖，对方 的能力没有提升，自己的时间也会被霸占许多。

都说，授人予鱼不如授人以渔。我开始转变思维，那日我就 回那位小妹妹，我说，我不是不愿意邀请你，我可以把所有 的群都推荐给你，但是你要知道，现在所在的互联网时代非

常发达，信息是不断迭代的，你只有自己掌握了寻找资源和 信息的方式，才能不断地更新自己。她好像似懂非懂。

也让我想起，之前我分享朋友赠送大冰刚刚上架不久的《小 孩》的部分文字在朋友圈，后面我看到有朋友在评论区问， 目前哪里有这本书的在售信息？

如果是以前的我，我会私聊对方，告诉她，书不是我买的， 同时会帮她在当当或者京东或者微博或者大冰公众号找到相 关链接发给对方，但我最后没有，我只是说，你去看看大冰 公众号或者当当有没有。

其实有许多方式不需要问人就可以找到的，但是人都是习惯 懒惰的，最好是对方可以发个链接直接购买。

我以前也是这样，特别是在比价商品的事情上，都说货比三 家，而我却不屑于这种事，并不是因为我有钱，而是觉得太 浪费时间，每次都依赖一个同事，她很喜欢在购买东西之前 会在各个购物 app 对比之后再下单，每次她都会把最便宜的分 享给我们。

直到有一次我找不到哪里比较便宜，我去问她，她跟我说了 来龙去脉，还包括时刻关注商品信息，在活动期间比较划 算，还有可能再下一次活动是最划算之类的，我后面才明白 她是怎么去买东西的，而之后我也不需要再问她，也能直接 买到至少不会是最高价格的商品。

都说老好人不好做，做的好了，别人会理所当然，做的不好 对方是责怪，而其实他们忽略了，在对方不断提供你想要的 结果的时候，对方在不断进步，而你却一直原地踏步。
