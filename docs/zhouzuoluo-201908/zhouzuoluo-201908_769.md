# 《没有实力，知道风

公众号:以史悟道 :

《没有实力，知道风口是什么也没有用》

曾经在互联网圈子有一句话风靡一时，这句话就是:站在风口 上，一头猪都能给你吹上天。

这话有些戏谑的味道，不过从出也看出了风口机会的重要 性。

过去的 2018 年经济形势并不好，新年刚刚过完，这几天在老 年过年听到的最多词就是风口二字。

对于当下的经济市场环境，大家的感受非常糟糕，也在努力 去改变这种现状，很多人开始经济困窘起来，大家在极力想 摆脱着这种困境。

有的人发牢骚，直接就说了我开头提到的那句话，站在风口 上，是头猪都能吹起来，大家都再说，我要找风口。

其实乍一听这话觉得很有道理，但是到了耳朵里，仔细的研 究下，发现又不是那么一回事。

站在风口上，猪真的能吹起来吗？ 当公众号红利期来了的时候，我们知道是风口，可是我们的

文采写不了公众号，转载又没有长期坚持的耐心，导致错失

了公众号风口。

现在的风口呢，我知道是视频，可是我又不会拍视频，我不 会视频剪辑，对于视频我一窍不通。

所以我知道视频是风口，又有什么用呢？ 不可否认过去改革开放几十年，在大时代背景之下，有人站

在了风口之上，被吹上了天。

但是呢，大风过后，你又如何安全落地？ 正如马云而言，摔得最狠的就是之前站在风口上被吹上天空

的猪。

其实风口遍地有，知道的也有很多，但是如果没有能力，又 能抓住几个呢？

更多的人知道风口也只能老老实实继续上班，不然就违背了
