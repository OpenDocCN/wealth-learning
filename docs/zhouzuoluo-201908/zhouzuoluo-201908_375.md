# 《你为什么很在意点

老王 : 《你为什么很在意点赞？》

365 天，每天精进一点点。这是老王分享的第 5 篇成长干货笔 记。

我是一个只有初中学历的业余写作爱好者 ，虽然学历低的可 怜，虽然工作也很辛苦，但我对写作的热情，没有随着时间 的流逝而有丝毫的减弱。

时间总是在不经意间从身边溜走，从去年 8 月份开始写，到现 在已经快一年了。在这一年中，与其说每天在写作，不如说 每天在独立思考、反思复盘。

这个动作练习的次数多了，它已经成了我生命中不可或缺的 一部分。说真的，练习写作让我最值得骄傲的一件事就是， 在认知方面，低学历的我可以完美的吊打高学历的同伴。

在刚开始写文章的时候，我就告诉自己，别人对你文章的点 赞数量，并不能说明你文章的质量，人家可能是很随意的一 个动作。

即便是这样，我还忍不住的要看一看别人有没有点赞。为什 么？自己明明知道点赞的真相，为什么还是战胜不了自己的 本能反应。就在昨天，瞬间的顿悟让我找到了到了答案。

对于 大部分普通人而言，写作的意义并不在于是否写写出了 爆文，也不在于你写了多少字。而在于你的劳动成果得到了 别人的认可与鼓励，在于你升级了自己的大脑，在于你对生 活的反思复盘。

你每写 一篇文章，只要得到别人的点赞反馈，就会特别的开 心，有一种莫名的动力继续去写。这其实，就是一个良性 的“回路”。

你动笔了——得到点赞反馈——有更强的动力去写——提升 了你的思考能力、认知水平——同伴对你更加的肯定——输 出正能量。这一圈圈的因果循环下来，你从外界得到了正向 的反馈，也向外界输出了正向的反馈。

成长的路上，一个人走总是孤独的。 为了让我们走的更快 些，为了逼自己一把，老王弄了一个日更小组(一周日更六 天，最后一天休息)。感兴趣的伙伴加我微信。
