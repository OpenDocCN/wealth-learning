# 《朋友间最好的关系

《朋友间最好的关系是可以彼此利用》

我原来有一个误区，觉得交朋友聊得来很重要。 这段时间一直在做训练营，需要组织很多的伙伴一起做这件

事。刚开始的时候，个性活泼，喜欢聊天的人，会给人感觉

更好。但真正在一起合作，慢慢地会更加愿意靠近靠谱的

人。

今天跟朋友聊天说：某个朋友在我心目中一直非常优秀，特 别靠谱，合作过后感到了深深的失望，工作合作不畅，其实 对私人感情也会产生影响。

朋友说：那大不不必，可能她本来很优秀，只是这个事件上 表现失准了。

我的看法却不一样，人与人之间，本来交往的机会就不多， 除了父母、夫妻、兄弟姐妹、近亲，这样的强关系，其他大 多都是合作关系。

有的人通过合作（不狭义地指工作合作，包括一起做一件 事、一起玩）慢慢地加深了解，产生更多的沟通，对方给你 的确定性越强，彼此的关系就越好。

这种确定性，其实就是靠谱。比如说我有个朋友特别爱迟 到，最夸张的一次，她说从家里开车出门了，让我下楼等。 结果我在路边等了半个小时，打电话给她说在半道上了，半

小时后再打，她说马上就到了。

有了在路边吃灰一个多小时的经历，我不太想跟她约饭、约 一起玩。

又过了两年，她使劲约我喝咖啡，我想着咖啡店可以看书 呢，迟到一点没关系。结果约了三点，她快五点到，我刚好 准备要走了。

像这样的朋友，根本不可能做一起做更重要更有意义的事 情。

相反，有的人可能刚开始并不熟悉，通过一次一次提供确定 性，会让人的感觉越来越好，由此产生信赖感和好感。比如 说助教丸子老师，为人低调谦虚，交给她的事情都会完成得 非常棒。假如遇到问题，会主动积极沟通，要么认真听取意 见，要么提出解决方案。

成年人的社交不是毫无目的的。直白点说就是，我们在同一 个水平线，彼此对对方有价值——能够互相利用。靠谱的 人，意味能够为彼此提供稳定的利用价值。这个我觉得就是 朋友间最好的关系。
