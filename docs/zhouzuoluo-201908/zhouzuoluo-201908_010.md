# 大家好，我是小橙姑

大家好，我是小橙姑娘，一热爱写作，渴望未来能活成自己 喜欢样子过上自己想要生活的姑娘。

很开心能有这样的平台给以我们分享我们的成长和感悟。 这是我加入星球的第 23 次分享，期待着小橙的第 100 次分享，

小橙会持续输出的。

《我是如何进入工作的心流状态的?》 什么是心流状态，心流状态就是一种专注忘我的状态，在

《深度工作》里谈到，深度工作的状态，也是一种心流状

态，它可以让人专注于某一件事，在做这件事的时候，精神

高度集中，同时会有许多意想不到的新收获。

而从这个月开始，工作慢慢找到这种状态，一改之前的注意 力不集中，感觉在浪费时间，到后来每天下班之后的收获满 满，我是如何做到的？

小橙是从以下几个方面改善的：

1.记录情绪日志

2.阅读心灵修身书籍

3.制定工作计划

4.制定工作任务之外的任务

5.使用时间管理工具

6.写工作复盘

一

不知道你有没有发现，其实人每天的状态都是不一样的，特 别是当你处于迷茫没有目标，感觉自己每天都处于浑浑噩噩 状态的时候，情绪和状态每天都无法处于同一个比较高的能 量。

我也经历过这样的时候，而且是很长的一段时间。后来我的 做法是，通过记录自己的情绪状态，每天填写情绪日志来监 测自己的情绪状态，同时多阅读一些自我修身的书籍，比 如，张德芬老师的《遇见未知的自己》《深度工作》《当下 的力量》等，让在情绪来临的时候，可以通过觉察的方式与 自己的坏情绪和谐相处。

当能与自己情绪和谐相处的时候，坏情绪就会渐渐远离，因 为情绪也是一种能量，会来也会走，能接受就会慢慢走开。

当我学会觉察自己的情绪，同时每天做情绪日志之后发现， 自己时常会在不好的情绪来临之前，或者来临之时，比如感 受到自己在抱怨的时候，就会有意识提醒自己，慢慢就会终 止这样的思维个情绪。

二 能把控自己情绪的人，才能把控自己的人生。情绪状态好了

之后，再开始每一天的工作之前，我会先做工作前的准备，

每天都准时安排今日的工作任务，即自己的工作计划。

你可能会说，不是已经安排好需要做什么了吗，为什么还要 自己写下来，领导的安排对自己来说，是被动的安排，而自 己需要发挥主人翁意识，对自己有一个主动地要求，只有这

样，才能真正把工作当做自己的事情去做。

在安排工作的时候，需要保持学习新知识的能力和习惯，把 自己需要长期学习的知识贯彻到每一天的工作中去，哪怕安 排自己每天学习 20 分钟，并把这个时间算入到自己必须完成 的任务上去，才能真正起到学习的运用。

有了工作任务的安排，自己也会有一定的紧迫感，同时也会 自然而然缩短玩手机或者浏览娱乐性网页的时间，更加珍惜 自己的时间。

另外，在工作过程中，为了避免自己玩手机，可以使用时间 管理工具来限制自己。我用的是番茄 to do app，可以自己设置 专注工作的时间，画面很唯美，同时文案也很激励人，看到 唯美的画面，都不会想要去玩手机了吧。

而且这样更容易进入专注状态，小橙也是利用这种方式，快 速进入心流状态的，效果非常好，而且一天下来，可以统计 今天专注的时间有多长，统计有助于让自己有成就感，体验 很好。

三 除此之外，在忙完一天的工作，下班前的半个小时，需要对

这一天的工作进行复盘，包括，完成情况，出现的问题，解

决方案以及之后需要注意的地方，最好能把明天的任务也提

前安排好，这样可以避免自己出现遗漏的情况。

以上是小橙这段时间培养工作专注的方式和心得体会，希望 对你有帮助。
