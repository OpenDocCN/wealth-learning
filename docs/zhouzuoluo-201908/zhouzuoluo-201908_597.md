# 《加入星球的第一天

猫 : 《加入星球的第一天，想退群？》

通过插坐学院的课程“认识“的粥左罗星主。听星主的课，自然 而然能感受到他的真诚。今天恰巧在朋友圈看到朋友分享星 主的文章，顺其自然入了星球组织。第一天，我查看了群友 们自我介绍的第三批合集。大部分群友的自我介绍就是按照 格式的冰冷堆砌，没有任何温度。看着几近千篇一律的自我 介绍以及群主对发帖等的各项规定，突然感觉来错了地方。

第一，这里有 6000 多人，大家发的帖子都是值得看的吗？人 数众多怎么能凸显自己？实现人脉链接？

第二，这里有很多专业的新媒体人，高中语文几乎没有及格 过的自己写的文章怎么能够获得足够的赞，不会遭遇删帖的 悲剧呢？

第三，看到群友的自我介绍，大部分都是从事新媒体行业， 感觉没有宝妈没有考证党，没有自己的圈子，怎么办？

因为在选择这个星球前，有一个观望了很久星球一直想入。 结果还是被“帅帅“的粥左罗星主吸引过来。没想到的是，居然 未达到自己的预期。我最终做出了决定——退群。在编辑退 群理由时，因其他事情打扰，这件事情暂时被我放到了一 边。等再回来处理时，我又忍不住把第一期，第二期的自我 介绍合集认真的看完，浏览了精华主题的部分文章。才发现 星球里的别有洞天。

有的群友自我介绍完全显示了自己的优势，让人忍不住想加 好友。群里大部分的文章质量上乘，这完全得益于星主的各

项“苛刻规定”。管理 6000 多的大群，不定规矩怎么成方圆。

事情叙述完了，想跟大家聊聊我的感悟。 第一、做任何事情，做任何决定，千万不要轻易下定论。一 定要全方位的考虑观察调研后再做。 第二、对于付出成本的事情，努力挖掘出最大的价值。 第三、是对星球的一点小提议:为了方便大家快速了解群友， 可以把大家的自我介绍整理成表格，这样方便后加入的朋友 快速寻找到自己的目标友人。

我依旧担心我的发帖得不到群友的认可，会被星主删掉。但 是，想到支教期间，带队老师把我的心得体会当做范文来 读，我问老师为什么选我写的？老师回答说:“因为真实”。

那就让我带着这颗“真实”的心，走在逆袭的路上吧。感谢这 里遇见的每一个人，希望大家能够见证我的成长。

2019 年 3 月 17 日 00:56
