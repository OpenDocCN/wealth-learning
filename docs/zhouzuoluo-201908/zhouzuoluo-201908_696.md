# 《所有的不适，都会

《所有的不适，都会让你成长》

舒适区是一个人习惯一样的生活和工作节奏，在舒适区里， 一个人能获得舒适感和安全感，离开舒适区，就会不安，会 不知所措。

1、每个人都有自己的舒适区 每个人都有自己的舒适区，这个舒适区有性格、行为、思想

上的习惯，只要不偏离这些习惯，就会让我们感到幸福和安

全。

人一旦走出舒适区，会瞬间感到不自在，会有压力。做自己 不擅长的事情会感到恐惧。

我们很多时候渴望待在舒适区，因为一切都是自己熟悉的工 作和学习，丝毫感受不到压力。

我们离开舒适区，会收获很大的进步 。自己的各方面能力都 会有很大的突破。

2、舒适区只是暂时的避风港 现实没有我们想象中的那么美好，面临的是一切的挑战和困

难。一个人不会甘于某一种现状，一旦进入舒适区，就不会

再有进步，生活也会越来越平淡，丧失了理想和激情。

随着年龄的增长，想要逃离舒适区的人就会越来越少。可是 很多人一边想要高收入，一边不想离开舒适区，两者是不能 权衡的。想要高收入，只能逃离舒适区，给自己不断充电， 这样才会有所提升。

舒适区只是暂时的，有时候为了生活和理想，只能逃离舒适 区。

3、不主动打破，你的舒适区会越来越小 舒适区待久了的人，行动力会变得非常差。等待成为了生活

的常态，一件事情要马上去做得时候，都会选择拖延。

在生活上，会特别依赖亲人和朋友，遇到一件事情自己永远 做不了主，想让别人替自己做决定。

这样长期下去，是很危险的。我们不主动逃离舒适区，生活 会把我们逼到绝境，不得不去改变。

我们不如一开始就选择离开舒适区，只有不断挑战，不断突 破自己，才能有巨大的收获。
