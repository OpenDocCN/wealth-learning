# 《不要和女人讲道理

李磊 : 《不要和女人讲道理？错！你只是不懂和女人讲道理的 时机和方法》

不要和女人讲道理？实则不然！和女人讲道理需要正确的方 法和时机，两者缺一不可。

“不要和女人讲道理，因为女人本身就是道理。”这句话是陈 铭在《超级演说家》中讲到的，很多人都很熟悉。它甚至被 很多男人奉为和女人处理感情问题的圣经。

陈铭这句话，倒也不是说女人不讲道理，而是告诉男人在处 理感情问题时，不要太理性，讲一堆的道理，试图劝服女 人，告诉她你是对的。

陈铭的核心意思是——男人不要在和女人吵架时一定要争个 高下、输赢。

陈铭提的这个标准很高，也确实是处理男女感情问题的一个 方法。但是，这句话深入人心这么多年，为什么男女之间的 吵架还是照样上演，该分该离的一样在发生？

不是因为要求太高了，也不是果真不能和女人讲道理。而是 很多不懂和女人讲道理的时机和方法。

当然，陈铭这句话也只是提供了处理男女感情问题的一种方 法，这句话没有义务和能力为男女的幸福之路保驾护航。人 是一个复杂的存在，更别说人最深处的东西——感情。

不要和女人讲道理，很多男人都做不到，尤其是两个人吵架

的时候，更不用说女人冷不丁的生气和发火。

男人和女人最大的区别就是，女人更感性，男人更理性。生 活中的任何风吹草动都会引发女人情绪波动。而男人却更为 理智、冷静。吵架时，很多男人都有把事情和道理掰扯清楚 的冲动。而女人这时候最希望的是，男人可以主动承认错 误，不管是不是男人的错；女人希望男人能包容她的小脾 气，不管吵架事出何因。

因此，吵架时，如果男人试图和女人讲道理，就是将自己的 理性强加于女人。这超出了女人的心理承受范围。

讲道理，会给女人传达一种负面感觉——婆婆妈妈、磨磨唧 唧。这是几乎所有女人的共识。

为什么这么肯定呢？因为我求证过。 我和女友一起三年了，也是几天一小吵。每次，吵架时，我

要是跟她讲道理，一争高下，斗个赢，说很多话。她就会很

反感，直接怼一句“你这个人，怎么比女人都啰嗦，磨磨唧唧 的。”

和女友吵完架以后，我也问过其他女性朋友，为什么吵架 时，女人不喜欢男人说太多？她们的回答很一致——女人不 喜欢男人婆婆妈妈、磨磨唧唧。

其实这个问题可以从物理常识解释:如果想要一团火熄灭，或 不让火势更大，你千万不要往火上浇油。你要做的是要么看 它停止燃烧，要么给它浇一盆水。

吵架时，让女人自己冷静下来，不要多说话，不管是谁的 错，都先主动承认错误，把一切承担下来。这就像是给燃烧

的火，浇了一盆冷水。

不和女人讲道理是指在女人生气发火时或吵架时。如果，你 认为在生活中不需要和女人讲道理，那就大错特错了。

女人其实也是讲道理的，只是和女人讲道理要懂时机并掌握 方法。

如果在女人发完火，两个人吵完架以后不和女人讲道理，那 么因为类似的问题而吵架就会再次发生。因为，问题没有解 决。

那么，什么时候和女人讲道理合适了？这个时间既不能太早 也不能太晚，需要适中。

我的答案是争吵完后 3—5 个小时之间。因为，这个区间内， 女人发完火，或两个人吵完架之后大家的情绪都平静下来 了，双方都会就之前发生的事情总结反思。这个时候，就是 男人和女人讲道理的最好时机。

太早了，大家的情绪都还没有完全平静。而太晚了，会把问 题遗忘或掩盖。

讲道理的时机选准了，那么，该如何和女人讲道理呢？ 第一，要主动，这是态度问题。女人最在乎男人对她的态

度。主动甚至死皮白脸去和女人谈话。这样会给女人传达一

个你在意她，在乎你俩感情的信息。而最大的误区就是让女

人主动，这会让女人觉得你有大男子主义倾向，甚至在女人

看来你这是冷暴力。

因此，主动意味着掌控话语权。

第二，复盘并解决问题。还原今天所发生事情的经过，分析 因为哪个点吵架？女人因为什么事情而生气？吵架但还能继 续一起，说明都不是事，也是可以解决，下次也能避免的。 而女人生气，很多时候都是因为男人无法 get 到自己的情绪， 并做出及时正确的反馈。

第三，做出承诺，让女人有所期待。问题都分析清楚了，也 解决了。男人最后要做的是，承诺会改正自己的问题，自己 会变得更好。因为女人喜欢自己的期待被知晓和满足，更期 待超预料之外的惊喜和期待。

由于男女比例的失衡。如今，已经是一个女人对男人的期待 高于男人对女人期待的社会。好男人、好丈夫的标准越来越 高。处理男女感情问题，尤其是矛盾时，男人要深信并践行 这样一条原则:

你要的是幸福而不是对错！
