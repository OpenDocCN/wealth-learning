# Day188《问了

《问了 3 个问题，我给团队招到了一个好文案》

最近给团队的新项目招人，前天面试的一位文案让我印象深 刻。问了 3 个问题后，我就知道她是我们想要的人。

问题一：你在文案方面有没有野心？ 来面试的姑娘应聘的是新媒体文案，看她的简历非常不错：

已写了上千篇文章，还在各大媒体平台发表过。

我说：在你的简历上我看到了不少成绩，我特别想知道你在 文案方面，有没有什么野心，也就是特别想达成的目标？

她对这个问题并不意外，想了不到 2 秒，告诉我她想做“爆文 制造机”。

她说自己是一个无法躺在过去成绩上的人，她的内心会不断 逼迫她去取得更多成绩，她相信最好的作品应该是下一篇。 每当写出读者喜欢的文章她会非常兴奋，过去 2 年她养成了一 个习惯：每天跟不同人聊天 1 个小时以上，读 10 篇以上爆款文 章，并产出笔记，现在电脑里躺着上千个相关的文档。

问题二：你认为写文案除了能力外，其他重要的还有什么？ 有能力有野心，这样的团队成员肯定是有战斗力的，但能否

持续稳定走得更远，要看一个人的信念和价值观。

她的回答让我有些意外，或者说是惊喜更恰当。她说平常 心。

因为一个成熟的文案不能大起大落，不能写出来的东西一会 儿 90 分，一会儿又不及格；心态上也要稳定，不能一会儿欣 喜若狂，一会儿失魂落魄。写了几百篇稿子后，认识到被客 户毙稿、改稿几十次都很正常，被读者喷也会遇到，重要的 是要有平常心，为了实现“爆文制造机”的目标，她表示自己 内心很强大。

而且她也谈到了敬畏心的概念，尽管没有直接用这个词。什 么可以写，什么不能写，不能为了博眼球踩红线，也不能为 了搞定客户没底线。市场是检验文案表现的第一战场。

问题三：对人心人性的理解。 文案一旦成为工作，就不止于我手写我心，更多是满足工作

需要，需要读者认可。写作不是一个人表达欲的释放，而是

对人性的洞察和理解。

她说爱与恐惧是人心的两大密码，并给出了自己的理解，表 示这两点是她写作中的“道”，不仅仅是写作的技巧层次。

问完 3 个问题后，我知道，我给团队招到了一个好文案。当 然，钱不是问题。
