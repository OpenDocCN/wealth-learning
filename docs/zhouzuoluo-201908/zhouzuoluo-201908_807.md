# 《从咪蒙事件中我到

《从咪蒙事件中我到和你不一样的东西》

咪蒙系公众号「才华有限青年」发表了《一个出身寒门的状 元之死》，状元、寒门、死亡，三个关键词非常抓人眼球， 这篇文章以迅雷不及掩耳之势很快阅读量就破百万了。当大 家还停留在为状元之死而感到惋惜时，一些质疑这篇文章真 实性的文章就出来了，全网又掀起一股打倒咪蒙系的风。

我觉得非常的精彩，就像看小说似的，跌宕起伏还有转折。 我不想去深究这篇文章的真假性，我看过寒门为了家庭为了 梦想，二十几十岁的癌症去世的真实案例。我想说的是我在 这件事中看到了什么。

01 自媒体为了流量选择抛弃了道德

《一个出身寒门的状元之死》用了寒门，状元，死亡三个能 够刺痛人神经的词，先吸引人们的眼球，再通过文章调动人 们的情绪，使这篇文章迅速的火了起来。

现在的公众号喜欢用一些敏感的词来吸引大家，比如“年薪千 万”，“焦虑”，“结婚”，“工作压力”，“生孩子”，“同学对比”。 这些词很容易引起大家的共鸣，是很好的写作题材，很容易 成为爆款文。但有的人为了流量，选择了没有下限。

2018 年，自媒体圈子里发生了一件非常恶性的行文事件，业 内坐拥 300 万粉丝的公众号“二更食堂”在滴滴司机杀害空姐事 件后，在头条推文《托你们的福，那个杀害空姐的司机，正 躺在家数钱》。文章以低俗的手法吸引了大家，完全不顾逝

者尊严和她家人的感受，渲染犯罪，渲染恐怖。

我以前关注一个讲英语和职场的大号，现在已经越来越无 耻。之前一直在撕知乎，天天说知乎药丸，还说知乎里面的 人都是傻逼。但人家知乎最近刚刚融了几十亿，我看知乎很 多人价值观都比较正。之前刘强东事件又在那说刘强东，还 把整个苏北县城的老师骂了一遍，下面就是他的原文:

“我爸早好几年就已经退休了，这意味着现在苏北县城和农村 的最后一批优质教师不复存在。如今当老师的，都是当年成 绩最差，做人最垃圾的混混。也只有这种没出息的腌躜鬼才 会选择考师范，然后回家乡当老师，这帮人不要说教书育 人，用社会渣滓称呼他们也不为过。 再加上高考制度改革，文科题不断以刁难和“考察阅历经验读 书思想”为主，自主招生竞赛降分各种手段大幅度抢占了普通 人考进名牌大学的名额，苏北县城的笨蛋们，彻底沦落到了 陪太子读书的地步。”

粥老师是我非常敬佩自媒体人，他保持了一个自媒体人该有 的良知，同时还把正确的价值观传递给我们。

02 大家对注意力的随意浪费 许多人还没有意识到注意力的宝贵，注意力是我们最宝贵的

财富，注意力的合理分配决定了你的成就。在这次的事件中

我看到的是大家对注意力的随意浪费，最开始大家看《一个

出生寒门的状元之死》的这篇文章时，你的注意力已经被收

割了一遍。后来质疑这篇文章真实性的文章出来，你看了以

后愤愤不平，觉得被咪蒙骗了，还很生气的写文章讨伐咪

蒙。

有的人把后来的这些人当成了英雄，他们带你看清了真相，

其实他们也在收割你的注意力，而且后来这些人收割了你的 注意力比咪蒙的还厉害。像寒门状元这样的题材也不是第一 次出现，为什么大号会质疑咪蒙，因为咪蒙是最大的公众 号，与咪蒙扯上关系的阅读量都不会太差，就像那么多做知 识付费，为什么大家只骂罗振宇一个道理。

质疑的文章我也看了，主人公 2013 年参加高考，高中时看过

《灵魂摆渡》，而《灵魂摆渡》的上线时间是 2014 年，这一

点无疑成为“胡编乱造”的硬核实锤。

针对文章中提及毕业 2 年就成为阿里巴巴 P7 级的员工描述，阿

里巴巴公关部在接受《新京报》采访时回应称：情况不属

实。

即使你看了这些质疑的资料，它也不能帮你成长。我们要学 会管理自己的注意力，防止注意力的浪费。

03 阅读也出现了鄙视链 鄙视链在有些领域就存在了，程序员领域写 c 语言的鄙视

c++的，c++的鄙视 Java 的，Java 鄙视写 PHP 的。昨天我才知道

阅读公众号也出现了鄙视链，大家都鄙视看咪蒙的。

昨天有位星友用含咪率来形容自己的好友质量(含咪率:关注咪 蒙的好友数量)，他的好友关注咪蒙的很少，他觉得好友质量 很高。

这让我觉得很不可思议，好像关注咪蒙的人做了什么让人可 耻的事，下面另一位星友评论说他很喜欢看咪蒙的文章。我 自己有时也看咪蒙的文章，生活这么苦，偶尔喝点鸡汤又怎 么样。

一千个读者有一千个哈姆雷特。即使同一篇文章不同的人看

也会产生不同的思考，在寒门状元这篇文章中我得到的感悟 就是:我要努力奋斗，不管文中的事是不是真的，我都要让我 的家人过上更好的日子，我不忍心让他们受苦受累。
