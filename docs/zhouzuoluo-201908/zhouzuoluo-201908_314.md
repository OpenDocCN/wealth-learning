# 《谁才是最愚昧的人

蓝子鱼 : 《谁才是最愚昧的人？》

谁才是最愚昧的人？肯定是知识越少越愚昧啊。一个人的愚 昧程度，和他掌握知识的量肯定成反比啊。这是我们的共 识。

以前我也是认为知识水平越低的人越愚昧，但最近看了一些 文章，和身边遇到的一些人改变了我的认知。

我们来看一个事实，在美国，很多父母不愿意给孩子打疫 苗。为啥呢？打疫苗不是有助于孩子健康吗？

因为有一篇文章说注射麻疹疫苗会导致自闭症，这篇文章的 作者叫韦克菲尔德。文章发出来后不久，科学家发现，这个 研究有漏洞，结论站不住脚。六年后，也就是 2004 年，《柳 叶刀》就撤下了这篇论文，韦克菲尔德本人也出来跟大家道 歉，承认错误。他本人甚至还被英国吊销了行医资格。

很多文化比较高的父母看过那篇关于疫苗会导致自闭症的文 章，即使后面政府出面解释那篇文章没有依据，这些人还是 不信。依然一意孤行没给孩子注射疫苗，导致每年有很多儿 童感染疾病，每年有好几个小孩死于麻疹病毒。

在一些文化水平较低的非洲国家却几乎每个儿童都注射了。 这些文化水平较低的国家都愿意相信政府，而文化水平高的

欧美国家更愿意相信自己。

今天我和一个老乡一起吃饭，我们两个都是福建人在成都上

学，吃饭的时候谈到有一个人在厦门找了一份程序员的工 作，工资是 8k。老乡他不相信我说的话，他说刚毕业的在成 都都找不到 8k 的，厦门怎么可能找到。我问他：你觉得厦门 比不过成都吗？他很肯定的说厦门没法和成都比，厦门工资 也不可能比成都高。

我让他回去查一下各个城市的平均工资，他居然告诉我说： 那调查都是网友瞎编的。在这个大数据时代他宁愿相信自 己，也不相信数据。

他用他的算法告诉了我为什么成都工资要比厦门高，他说如 果在厦门拿八千，那么在成都就能拿到一万，因为他每年还 要回家，回家的交通费也要算到工资里，所以成都工资必须 比厦门高，不然大家都跑厦门工作了，干嘛在成都。

他这段话完全是从自身出发，觉得离家比较远，所以工资必 须比较高才对。企业才不会管你离家多远呢。很多人求职都 会开一个目标薪资，HR 通常会问为什么要这么多，很多求职 者说：如果少于这个数我就在这个城市生活不下去，企业才 不会管你能不能活下去，他们只会关心你能创造多少价值。

我真的不敢相信我那个老乡读了四年大学居然会说出这样的 话。我想给他解释的时候，他不仅不听我解释，还试图找更 多的论据来证明他才是对的。

他可能觉得自己是个大学生，所以他的论点是不会错的，如 果他文化水平低一点的话，可能就会听听我的想法。那些文 化水平低的人，他们知道自己懂的少，所以他们更喜欢听一 下别人的意见，反而是那些有点文化的不知道自己有多愚 昧，觉得自己什么都懂。

不知道自己不知道是一件很可悲的事。
