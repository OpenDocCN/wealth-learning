# 《成功一次，你就经

《成功一次，你就经常会成功》

无论多难，这一生，你一定要逼自己坚持一次，成功一次。 有这次宝贵经验以后，你会发现自己更容易做成一件事。

01\. 漂浮在水面时带来的未来确定性 前两天晚上我在泳池进行第二节私教课，教练每节课的时间

是一小时。半小时练习之后我们停下来休息，我跟教练说：

看来我还是特别怕水啊，放在浮板上的双手一直放松不下

来，紧绷的肩膀有点酸疼。她微笑点点头说：对。

我继续提出疑问，如果会游泳了以后我是不是能自动漂浮在 水上边？以前跟会游泳的朋友在海边玩，他们就浮在水面不 动，能坚持很久。这时候教练说了一句话，令我十分惊讶和 激动。她说：你要不要试试？

我一下子害羞起来：真的可以吗？ 教练：（甜甜的说）可以啊

她扶着我的腰协助我向上仰着浮在水面上，但是不到两秒我 的腰和臀部还是下意识地下沉，因为我没有完全放松身体。

直立之后，我还想再试一下。第二次，教练一边让我用腰发 力不让臀部下沉，身体平躺舒展浮出水面，一边放松肩部和 头部，我感受到了水的力量，将我托着不沉下去。

连连向教练惊呼神奇，我终于体会到他们说的“放心交给水

吧”那种力量。从此我也坚信我一定很快能学会游泳，那种感 觉特别坚定，特别振奋人心。

这是一个“提前品尝成功滋味”的例子。另外，我还想分享两 个令自己信心倍增的经历。一是一周“模仿练习”而获奖的演 讲比赛，一是坚持六七年后颇有成效的健身。

02\. 偶然的成功令我发现自己的长处——模仿练习 大学时临时起意参加一个演讲比赛，而在比之前没什么演讲

经验的我开始在网上搜索“超级演说家”、“TED 演讲”等视频进

行揣摩和模仿，找到一些适合自己的技巧例如肢体语言、语

速、幽默氛围、真实小故事穿插于演讲当中等，一周后的比

赛我拿了全校三等奖。

在这过程中我没有把握自己会成功，只是专注于快速学习和 运用，拿了奖项之后并不是人生经历的一个小句号，幸运的 是这段经验引出了破折号——它告诉我，我是擅长于模仿练 习的。之后的英语口语比赛、绘画等，我通过模仿借鉴，都 有一些不错的结果。

03\. 只要坚持做下去，其他的交给时间 从决定健身开始，我给自己设定每周四到五次锻炼的任务，

每次不少于一小时，六七年来基本没有间断过。

以前同学叫我肥焦，现在能明显看到身材曲线较好，锁骨明 显，马甲线等应有尽有。高中好友认真说：你再也不是肥焦 了，现在你的身材令我十分羡慕。

看到我现在的蜕变，有一些朋友问我做一些什么运动，其实 什么运动不是特别重要，这几年来我试过长时间慢跑，无氧

哑铃，跳舞跳操，健身房跑步机，室内瑜伽，瑜伽垫上的天 鹅臂天鹅腿系列等千变万化的形式，但不变的是不中断不放 弃。

这是一个渐变的过程，一口气吃不成胖子，一下子减不了瘦 子。经过这么几年之后，今年的改变逐渐凸显出来，身边的 人惊呼：你怎么在不知不觉中瘦了这么多？ 其实，我只是耐着性子，每周坚持做运动罢了。

这一事件，也令我收获颇多。曾经是一个小小的梦想，一天 天过去，一天天不断耕耘，忽然就发现梦想实现了，我做到 了！这种经历极大地鼓舞人心，当你成功做好一件事之后， 你就经常后做好很多其他的事情。

成长过程需要培养一种惯性，一种成功的惯性。至少成功过 一次的人笃信：之前如何做到的，这次我也可以完成。
