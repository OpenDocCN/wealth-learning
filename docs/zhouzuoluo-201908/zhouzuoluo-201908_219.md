# 【何炅：我只是一个

【何炅：我只是一个“自卑”的普通人】

最近看了一本书，何炅 20 年前写的自传，名叫《》，一直以 来何炅都是以高情商别人称道，是娱乐圈难得的零差评明 星，也许是为了窥探他的高情商从何而来，我掀开了这本书 的序幕，却看到了一个荧幕上不曾见过的何炅。

1、“其实我有点自卑”

从小何炅在院里就属于比较瘦小的类型，挤在人群里不蹦一 蹦还真看不见他的身影，也许是因为个子上不占优势，别人 家的孩子都喜欢满院子跑着玩，唯独他，喜欢窝在家里看 书。别人家的孩子成天被父母喊着回家吃饭，而他，成天喊 着他哥回家吃饭。

过于喜静让他养成了内敛的性格，刚上小学时，有一回上课 他想尿尿，可是不敢跟老师说就一直憋着，越憋就越急，结 果还是尿出来了，裤子、地上都湿了一大块，这样的情况持 续了几个星期。

现在的何老师即兴演说张口就来，口若悬河，可是在最初的 时候他也经历过那一个个紧张磕巴的过程。中学时因为普通 话比较好被老师选中参加演讲，第一次演讲的时候，看着底 下乌压压的一群人，他的脑中一片空白，手也不知道该怎么 放，好像放哪都不对，演讲内容接近尾声的时候，突然意识 到中间漏了一大块内容没说。下了台，老师心疼地问他，手 疼吗？原来演讲时他下意识地不停抠着自己的手，都破皮 了。

大学期间，因为一个小品《》与中央电视台结缘，从而获得 了第一次主持节目的机会——少儿节目聪明屋，他的第一场 节目是扮演一个木偶人躲在“书”中，等着他的搭档小角丫过 来打开书本，然后他再趁机跳出来对着小朋友说“大家好，我 是大拇哥”，可是这样一个简单的开场拍摄并没有那么容易， 他反复录制了三次，才终于说出了他主持生涯第一句台词。

何炅在书中说，他觉得自己是自卑的。这个在荧幕上开朗活 泼，情商极高的男人居然说自己自卑？真不可思议，黄磊也 曾在节目中说过，其实炅炅的内心是自卑的。纵观他的成长 历程，我才知道他的自卑从何而来。

因为童年的内敛，他的内心深处始终深藏着一份敬畏和谨 慎，即使后来他不断挖掘着自己的特长在生活和事业上闪闪 发光，但这份自卑早已在他的骨子里养成了习惯。

他的自卑并不是不自信，而是他从不过度抬高自己的价值。 他的每一个阶段都是一步一个脚印沉甸甸地踏出来的，即使

获得了暂时的成功，他依旧如履薄冰，小心翼翼呵护自己的

每一个成功。

也许他的人生是幸运的，可是他更清楚幸运从何而来，每一 个外人看来幸运的机会都是他背后一点一点付出换来的。

2、“如果你在校外见到我，别忘了叫我一声何老师”

自从成名以后，就总有粉丝在电视台门口等着他签名，何老 师一律不签，无论等多久都不签，以至于有些粉丝积怨成 怒，开始抨击他傲气。

不是他傲气，而是他心疼这些粉丝。些许是同时兼顾着教师 这一职业，每每看到那些花季般的少男少女顶着骄阳，冒着 大雨就为了等偶像签个名，他就觉得恨铁不成钢。

有一回录节目录到半夜，走出电视台发现有个女生还在门口 等着，心里头那股师责“窜”的一下就上来了，不仅没给那女 生签名，还把人训了一顿，训完还不忘叫了一辆车让人赶紧 回家。这就是何老师呀，那么较真也那么可爱！

有一次在台里被一个十六七岁年纪的男生叫住，想请他帮 忙，原以为可能又是要签名之类的，准备要一口回绝，谁知 那男生却说，能不能请你假扮我的哥哥。这个要求可新鲜 了，原来那个男生和同学打赌，赌的是何炅是他的哥哥，赌 注是一千块钱。这下可让我们的何老师为难了，帮吧，不能 助纣为虐呀，不帮吧，一千块钱也不是小数目呢。

结果何老师彻夜写下一封“教导信”，第二天如约来到了这个 学生的学校配合演了一出“哥哥探校”的戏码，然后留下了这 封满满当当的书信。

信里的大致内容是：

“弟，其实人生就是一个大赌局，到时候生活自然会逼你下 注。你们现在该是好好修炼、积累人生筹码的时候，又哪有 时间玩这种无聊的打赌游戏？另外我不知你们家的经济状况 如何，我目前正为把系里读不起书的贫困学生留在校园而忙 碌，当你们一掷千金的时候，请想想他们。”

在大多数人眼里，何炅是一个华丽闪耀的明星，可是在他自 己眼里，他一直都是那个五尺讲台上的老师。别人都说他情 商高，他只是习惯了用谦卑的心态去与人相处，无论在娱乐

圈获得了怎样高的地位，他都依然把自己的能量看得很淡， 很淡。

他常常说，要做最坏的打算，尽最大的努力。 何炅最被人称道的是他的救场能力，正是因为他的每一场主

持都做了超过一百分的准备，他才能在恰当的时候施以援

手。

生活在聚光灯下的明星闪闪发光，我们看明星就好比雾里看 花，但其实明星也是普通人，他们和我们一样一步一个脚印 在人生的旅途中不断前行。
