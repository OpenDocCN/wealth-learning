# 《如何避免争吵？试

《如何避免争吵？试试这个沟通方法！》

晚上好 与星友们分享丹儿的第 100 篇原创 8.5

————————————————————————----

“没法沟通”这个词，在生活中大家是不是经常感受到呢？这 个词一般会出现在什么场景呢？是不是与他人大吵过一架之 后，就会觉得与对方无法沟通？大家有思考过让我们产生“没 法沟通”这个想法的原因是什么呢？

是因为三观不合，观念不同吗？是不是觉得思想都不一样， 话不投机半句多？跟你解释你也听不明白，你说的都不对， 就听我的就行了。但是，每个人都有自己的想法，完全听一 个人的太难了，所以就产生了不断的争吵。

可是，争吵能解决问题吗？是不是吵完不仅没解决，双方还 闹的很不愉快！今天就和大家分享一个有效解决问题的沟通 方法。

一，首先我们先看看导致我们争吵的原因 想想我们与别人争吵时是不是大多数是因为对错的原因呢？

你说你是对的，我说我是对的，都想说服对方听自己的，好 嘛，那就开始吵了。

但是在对方表达观点时，我们都有用心听吗？ 是不是在对方张口提出反对意见时，我们就在思考如何反驳

他，接下来的话都被我们屏蔽了呢？心里是不是在想你说的

不对，应该是怎么样怎么样的？

还有的时候别人刚一张口，就会有人打断说，我懂你意思， 你说的不对，应该是怎样怎样的。为什么不能听别人把话说 完呢！

★我们不要通过推理的方法去跟别人沟通，就是我猜你是这 样认为我的，进而先得出一个结论！没有人愿意在别人得出 结论以后在来谈话！这就是交流中争吵的主要原因。

举一个自身的例子 最近在培养自己谦卑的品质，要求自己一定要听别人把话说

完。

今天和朋友聊天中提到电影银河补习班中邓超的教育方式。 我是很认可的，因为和我接触的正面管教的教育方式很像， 帮助孩子培养独立完整的自尊体系。

而我的朋友却有不同意见，她一说邓超的教育方式有问题， 孩子不应该放养。我在心里就给她定论了，她一定是认同传 统的教育理念，孩子只知道读书，不知道读书的意义在哪有 什么用呢？

如果是以前我肯定没等她说完，就会说你说的不对，然后噼 里叭啦说一大堆，让她认同我的观点，但是现在不是要练习 听别人说话嘛，我就打算把话说完，在噼里啪啦说一大堆

于是继续听她往下说完。

等她说完后，我发现她的观点居然是正确的，她并不是说让 孩子明白学习的意义是错误的，而是电影中邓超没注重培养 孩子的合作精神，她认为是不可取的，如果一个孩子太独立 凡事独来独往不注重合作也不行。

电影快结束时马飞独自决定出去修理飞船，他想赌一把，但 如果没有后来同伴发现他出去后与他的语音配合，单独靠他 自己很难脱险。

把自己变得独立很重要，但是与他人之间的连接合作更重 要，这就是从独立型人格到互赖型人格的转变。

★所以你看，认真的听别人把话说完，不要过早的得出结 论，可能会减少很多争吵的机会。那么，什么是有效的沟通 方法呢？

二，什么是有效的沟通方法呢？ 沟通的目的是想解决问题，很明显争吵是解决不了问题的。

所以接下来，我们可以试试心平气和的坐下来聊聊，不带任

何的情绪。

①认真的倾听对方想表达的话

②不要给对方过早的得出结论

③共情，占在对方的角度理解她

④当对方的情绪被理解了，她也可以做到理解别人，这时候 你就可以陈述事实，说出你的感受。

⑤共同思考更好的解决方案

1+1＞2，两个人共同思考的解决方案，双方更容易接受，还

可以通过合作共赢。
