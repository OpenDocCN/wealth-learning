# Day214【用好

【用好精力管理，过有质量的一生】

这是一个重视个人体验的时代，而拥有美好人生体验的前提 是有一个良好的精力状态。读了张展晖的《掌控：开启不疲 惫、不焦虑的人生》后，我改变了自己 N 个错误认知，并学到 了 N 个改善生活质量的秘密。

整理 10 条精华清单，与你分享。

001 精力充沛是掌控人生的关键，改变对生活有心无力的失控 感从精力管理开始。精力管理的目标是达成每天有足够的精 力完成工作和学习任务而不疲劳，并有余力享受休闲生活， 还能应付突发状况。

002 精力管理的本质是让人体持续表现出最佳性能，我们可以 从运动、饮食、休息和心态四个维度来全面提升自己精力管 理的能力。

003 大家对运动往往存在着一定误区，很多人把锻炼肌肉当成 是最重要的事。如果从精力管理的角度看，心肺功能才是保 证精力充沛的第一要素，跟身体状态以及寿命之间呈正相 关，所以心肺功能的训练是运动的首要任务。

004 在减肥这件事儿上入坑的人更是不计其数。根据研究发 现，要减少 1kg 脂肪需要燃烧 8000 卡里路，相当于跑 130km 消 耗的能量。可见，运动并不是控制体重的最有效方法，控制 饮食才是最重要的。

005 采用节食减肥一定会产生反弹，因为节食减肥减掉的脂肪 很少，大部分是水分，一恢复正常饮食分分钟就弹回来了。

006 不会休息的人就不会工作，一张一弛，才是生活之道。充 分休息+高效工作才能活出想要的人生。

007 睡眠是最好的医疗手段。身体和大脑的修复都需要在充足 的睡眠中进行。所以，人生用三分之一时间用来睡眠，是为 了保证另外三分之二活得更有质量。

008 心态管理的秘诀在于三个词：安心、真诚、认真。用安心 避免情绪化的消耗，用真诚面对人和事最高效，对每一件小 事都保持认真的态度能耗最少。

009 做事前，先为这件事赋予重大意义，可以大大提升做事的 动力，而不是依赖所谓的坚持或者意志力。

010 社交学习是一种非常有效的学习方式，可以减少个人随意 性，实现团体互相监督和赋能。你的圈子决定了你的成长速 度。我已经将社交学习升级为联机学习，并打磨成我在高效 学习系统中的关键一环。
