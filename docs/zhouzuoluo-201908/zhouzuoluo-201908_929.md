# 《我越来越坚定，要

《我越来越坚定，要努力提供好的内容》

之前一个@粥左罗笔记的粉丝，他负责管理健身会所，邀请 我免费去游泳，昨天再去游泳，觉得一直免费不好意思，带 了一些水果放在前台。

这位粉丝得知后，发微信说了我一通，并且直接送了一张健 身卡，搞得我更不好意思。

他说，之前很少看文章，就是偶然一次看到我发朋友圈的文 章，觉得非常干货，不像有的内容喜欢煽动情绪，其实好看 只是觉得爽，爽过了并没有什么收获。

后来他就一直关注我发的朋友圈，然后关注了@粥左罗笔 记，慢慢习惯每天看一篇文章，了解了很多从前他不懂的东 西，提升了自己的认知。他经常把文章分享在朋友圈，偶尔 也会跟朋友分享和讨论文章内容。

以前我很少认真去思考，这些内容一天一天的，到底会带给 读者什么。对于有的人来说，不过就是一篇文章，一划就过 去了，甚至都懒得点开。

但是，对于有的人来，他喜欢一个号、相信一个号，每天从 这个号的每一篇文章中吸取营养和知识。

我想起曾看过的一篇文章，有句话：你不要觉得几万人很 少，几千阅读量很少，你要想想，这几万几千人里面，真的

有人会被影响。

粉丝的相信，让人越来越坚定，要努力提供真正有价值的内 容，除了追求阅读量之外，心中更要有一份责任。
