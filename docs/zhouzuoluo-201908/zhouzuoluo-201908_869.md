# 《选择异地

《选择异地恋，背后是不配得的心理》

我有个来访者，她来咨询的原因是长期失眠。 一个人带着孩子，老外地工作，经常半夜会想念老公，尤其

是白天遇到不开心的事情，就更渴望老公在自己身边，可以

抱抱自己，可是越想就越睡不着。

有一次在店里面认识了一个小姐姐，俩人很很聊得来，当来 访者心情不好的时候，他们就会相约酒吧，互相倾诉，相谈 甚欢，有时两人会喝很多酒。

有一次小姐姐谈起自己和丈夫是两地分居时，她告诉对方:“姐 姐，我也是”，于是两个女人眼中瞬间噙满泪水。

用她的话说，老公正是努力拼事业的时候，自己不应该不明 事理，可以醉但不能失去理智。

当他她想念老公的时候，她总是告诉自己要知足，因为每年 生日的时候，老公都会送她好多礼物。

今年元旦，一大早她发微信祝老公新年快乐，老公发给她一 个 2019 元的大红包。

她总是告诫自己要知足，孩子 3 岁了也很懂事，可是她说自己 对婚姻还有许多不适应。

她说你可能听说过“留守儿童”这个词，可我现在是个“留守妻 子”。

有一次在电视上看到徐静蕾在《朗读者》朗诵史铁生的《奶 奶的星星》，她想起来奶奶临走前对自己说的话，在自己最 脆弱无助的时候，她总会想着奶奶就在自己身边。

原来她小时候是在奶奶家长大的，晚上基本上都是和奶奶 睡，虽然每周都能见到爸爸妈妈，但她还是总感觉自己是“留 守儿童”，自己有家不能回，只能住在奶奶家。

于是我对她说的“我是留守老婆”多了几分理解，虽然她长大 了，结了婚生了子，可是两地分居的生活，让她体验到了被 抛弃的感觉，这种感觉，就像小时候被爸妈“抛弃”，寄住在 奶奶家一样。

有些女性还没谈对象，就会提出择偶标准，那就是坚决不要 异地恋，这样的女生以后的爱情婚姻就不太可能是异地。

但有些女性却从来没有考虑过这件事，他们觉得只要有一个 人陪着自己，对自己好就行了，好像异不异地都可以接受。

能接受异地恋的女生，潜意识或多或少都有不配得的心理， 害怕生活过得太平静。

1、不配得，不配享受亲密关系。

当我想老公的时候，我含着眼泪告诉自己要知足，和别人比 我已经拥有了稳定的工作和生活，这背后其实是一种不配过 上更好生活的自虐心态。

其实他们内心也很渴望有一个男人天天陪在自己身边，心情 不好的时候，可以抱抱自己，听自己抱怨、唠叨，晚上踏实 的睡在老公身边。

“我有吃有穿，老公工作稳定，家里还有存款，婆婆对我也挺 好，我应该知足，怎么还能有情感甚至性的渴求呢!”

选择异地恋，尤其是长期分居的女性，他们很能吃苦，很不 爱惜自己，有的带几个孩子，又当爹又当妈，最后活活把自 己累死。

2、在平静的生活中，觉得不真实。

有些女性小时候成长环境太恶劣，缺少妈妈的疼爱爸爸的保 护，他们从小就习惯了:天有不测风云，有人旦夕祸福。

他们习惯了生活中经常发生不好的事情，在这样的生活中， 他们才有存在感，他们才觉得自己活着。

当日子慢慢变好，遇到对自己好的男人，他们开始的感觉会 很好:终于等来了拯救自己的人，终于过上了自己想要的生 活。

但这种平静美好的生活对他们来讲又太陌生，这样的生活没 有他们熟悉的味道，于是他们恐惧担心，不知道自己将去何 处。

过去的生活虽然不喜欢，但是熟悉啊，人对熟悉的东西本能 的会产生安全感。

一而再再而三的异地恋或长期两地分居，其实也是心理学讲

的“强迫性重复”，西方一句谚语：我们跟一个人交往的方 式，其实就是我们跟所有人打交道的方式。

怎么打破呢？ 讲道理找方法吗？no，道理谁都懂，但问题是做不到。 在道理或者方法和做到之间缺少了一个“体验”，“体验”更多的

时候是一种感受或感觉，而非道理或方法。

方法或道理是头脑层面的东西，感受感觉更多的是内心甚至 潜意识的东西，当你勇敢的开始觉察和面对自己的内心，你 慢慢就会有新的体验。

当你可以用语言表达出你内心的想法、感受，你的心就会慢 慢打开。

突然有一天你也许就会顿悟，选择异地恋或两地分居原来是 我自己创造的：因为我自卑，因为我不配过更好的生活。
