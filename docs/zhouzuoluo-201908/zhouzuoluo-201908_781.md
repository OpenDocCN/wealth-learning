# 《我嘴上很丧，但我

一个悦己 : 《我嘴上很丧，但我比你想象的努力》 很多假装看透人生打算得过且过的人以为遇到了知己，以为

李诞和他们一样晃荡着度日一切都不在乎，那你们就大错特

错了，李诞一点都不消极悲观，也没厌倦处世。

他以一种 " 积极废人 " 的状态存在，嘴上说着很丧很懒什么 都不想要，行动却很积极，非常清楚自己想要的，并努力付 诸现实。

他自己也说过 " 基本上什么事儿我都不想做，但做的话会认 真 " 这样的话。

大学毕业后，李诞先是从内蒙老家到南方做媒体工作，后来 又到一家广告公司写文案。

一年后，有一家找上门的工作，因为钱多，他就去了上海。 他给相声演员王自健的节目《今晚 80 后脱口秀》写了五年的 文案。

那个名不见经传的蛋蛋积累了足够多的经历与能力，养精蓄 锐，从没停止努力，策划了脱口秀节目《吐槽大会》，带红 了仅有高中文凭的小兄弟池子和高颜值程序员庞博，得到了 王思聪的投资，成为了笑果文化的 CEO，身价一跃十几亿。

怎么看都不觉得这是一个佛系不作为青年的人生脚本，更像 是一个准备充分的王牌大戏。

蔡康永曾经打趣李诞说： " 李诞你初出茅庐的第一个节目，

就打败了马东在电视圈修炼了几十年下来做的节目 "。他还是 那一幅戏谑调侃的口吻说 " 你说巧不巧 " 。

这个看似轻松蹿红的不正经青年巧妙地藏起了自己勤奋刻苦 的准备和练习，以丧且轻松的态度面对着每一件事情,回应着 我们的猜测，一边漫不经心，一边小心翼翼。

他经常自嘲自己是 " 滞销书作家 " ，他写的书不如他做的节 目火，但是他还是愿意坚持，一边笑话自己卖不出去一边继 续写。写作是李诞的梦想。

那个看上去什么都不在乎、随性又潇洒的人，心里有着他自 己坚定的坚持。

他好像真的看透了人生，深谙世间的道理，但人们忘了他的 那句 " 人间不值得 " 还有前半句 " 开心点朋友们 " ，他说的不 是从此就凑乎过吧，他想说的是人间不值得你不开心，你要 开心地好好过。

李诞招人喜欢的地方就在于他过着我们向往的懒散自在的生 活，但却从没放弃自我打磨。

我们都知道人生难，都知道这个社会向来不公平，也都经历 过努力得不到成绩，付出没有收获的时候。

我们惊叹着有人出生在罗马，感慨着有人是锦鲤体质，羡慕 别人轻而易举的成功，叹息着自己的不如意。

面对理想的生活和现实的差距，有人从此淡泊名利归园田 居，也有人逆流而上做那不轻易放弃的大马哈鱼。

聪明的人会抱怨生活不易，也会偶尔吐槽自己生不逢时遇人

不淑，但从不会因为一点生活的不如意就放弃追求，得过且 过随波逐流，因为他们不会轻易被外界打败，除非是自己自 愿放弃。

人生是个不进则退的过程，你得往前走，哪怕走得慢一点， 哪怕很不情愿，哪怕一边叫苦一边喊累，你也得走，因为只 有走下去，才有希望。

之前看过许知远采访李诞，李诞在里面曾说道：我很长一段 时间都觉得啥都没劲，后来包括逼迫我自己，因为我确实喝 三块钱一瓶的啤酒也很开心。

但是我觉得我这样下去是不是就完了，我就回内蒙了，我就 回内蒙自己找个屋，我就喝一辈子啤酒，什么都没有也行 啊。

最可怕的就是这个也行，因为在我心里这种选择真的行，所 以我就一定要变成现在这样，打一个小粉领带，我才能健康 地活着。

活得通透也需努力，这就是李诞的智慧，我欣赏他有趣有能 力又自知要努力。也希望自己如他般豁达。
