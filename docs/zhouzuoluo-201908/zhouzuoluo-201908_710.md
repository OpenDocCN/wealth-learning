# Day105《万万

《万万没想到，职场技能千千万，最重要的竟然是…》

关于职场能力图谱，前人有现成的总结，我下面直接贴出 来，如果从中选择一项你最想精进的，会是哪个呢？

放在以前，我一定会说是学习力。 因为我有一个简单的判断逻辑：职场就是价值交换，而一个

人的价值体现在能为公司解决问题的效率，除了工作资历和

经验，学习力是背后的关键能力。

直到最近一年，我后知后觉地意识到：风风火火的互联网时 代早早就来了，新媒体时代也来了，学会放大自己的个人能 力才能赢得未来。

罗振宇说过这么一句话：“职场，或者说当代社会，最重要的 能力是表达能力。因为在未来社会，最重要的资产，是影响 力。影响力怎么构成？有两个能力：第一是写作，第二是演 讲。”

你可能并不完全认同他的话，但你估计不会反驳职场人脉的 重要性。

人脉不是来自一厢情愿地搞关系，而是你的影响力。你的职 场影响力足够强，何愁没人脉？

香港第一自媒体人 Spenser 说：“在没有资金、资源、人脉的情

况下，写作是提升职场影响力成本最低、效果最好的方式， 没有之一。”

这我同意。因为我从事新媒体行业这几年来，我眼睁睁看着 一些能力不是最突出的人，靠能说会写在职场一路高升；还 有的人干脆通过写作搞起了第二职业，甚至收入远超主业。

1

在过去，只有媒体机构才能发出自己的声音，而如今人人都 是自媒体的时代，你可以向全世界发出自己的声音，让别人 看到你。

如果你不会写作，那真是可惜… 先说一个正面例子：

刚毕业的 S 叔从事了金融理财工作，他持续通过自己的公众号 写一些关于资产配置的理财科普文章，慢慢收到了全国许多 读者的咨询，其中不乏有海外资产配置需求的人，结果很自 然，他们成了 S 叔的客户。就这样，在没有人脉、渠道、资金 投入的情况下，他一个初出茅庐的小伙子竟然短时间内迅速 把事业做起来了。

你想想，一名业务员一个个去拜访客户，一年能拜访多少 个？最终又能成交多少？如果是通过写作建立起自己的专业 性，一篇文章可能就触达成千上万的准客户。效率完全不在 一个量级，套用一句时髦的话：不懂写作的人被降维打击 了。别人一篇文章产生的成交量可能抵上他一年的成绩。

写作，是个人能力的放大器。是连接更多人和机会的强大武

器。

再说一个反面例子： 朱小峰，也就是本人，毕业后辗转多个行业，最终进入了广

告行业，后主要负责新媒体类项目。本来在新媒体的认知上

有一些优势，却完全一副睁眼瞎的样子，在 14 年就注册的公 众号上写一些很轻松的随笔，从来没有持续写作超过一个

月，最长一次 20 多天，还是最初找工作的时候以“有原创能 力”作为敲门砖，入行成为一个新媒体文案的时候。

从来没有认真考虑通过写作建立自己的影响力，并通过持续 输出对所从事领域的研究成果来为自己开拓事业，从而赢得 更多主动选择的机会…

2

如今反应过来，已经是 2019 年了。 于是我从头开始学写作，基于新媒体环境下的传播特点，摸

索了一个多月，成绩平平。

再后来才认识到，跟有结果的、实战派高手直接学习岂不是 比自己摸索更快？我怎么就那么蠢！

于是我加入到了自媒体大 V 粥左罗（一个被称为爆文制造机的 人）的社群，并通过他的写作课来建立和打磨自己的写作系 统。

到今天，我第一次做到持续写作超过 100 天，也渐渐找到了新 媒体写作的感觉，也有文章获得了大号的认可，当上周第一

篇文章被大号转载的时候，我知道 2019 年会是我的一个新开 始。

3

今年，我成立了一个写作精进小组，与十多位小伙伴一起学 习粥左罗的写作课，通过每天日更几百字找到写作的手感， 通过每天一起做选题，以投稿的标准要求自己，去接受市场 的检验……

如果你也认为写作能力是职场最重要的能力之一，欢迎你加 入我的写作战队。

如果你能立下军令状，有足够的意愿，我会考虑自掏腰包送 你一套粥左罗的写作课。

我相信这会是一笔划算的投资。 对我，对你，都是如此。
