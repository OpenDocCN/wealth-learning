# 《高小羊：掌控关系

《高小羊：掌控关系：说完“谢谢之后”》

上周日，闺蜜邀请我去她家吃饭，亲自下厨做了家乡特产擂 茶，同时还配了精致小菜和焖豆。

因为我所在城市在连续下了一个月的暴雨后，周日那天终于 迎来了太阳，我将要进行暴晒的东西全部摆好出门后，时间 已经不容许我折角去反方向买水果，所以在去的正方向的路 上，顺道买了其他礼物带了去。

每次去闺蜜家吃饭，我都会带水果或者其他礼物，这样我们 在吃完饭后就会有水果食用，同时我想让朋友知道我很注重 这次聚会。更重要的是，在我表达了对朋友邀请的感谢后， 表达自己实际行动上的感谢。

我一直都保留着这种习惯，不管是在工作上，生活上，还是 在经营关系上。去朋友家玩，或和朋友约在外面见面，或是 回家探望父母，我都会准备礼物以作为我的心意，而不是两 手空空。

比如朋友请我吃饭，或是公司前辈在工作上提供了帮助，在 当面表达“谢谢”之后，我更注重后续的“回礼”。比如找合适的 机会，再请朋友吃饭，工作之余帮前辈解决一些能力之内的 问题。

比如每次回家探望父母时，我会选择将礼物装在行李箱里一 起带回去，而不是单独寄回。如果礼物太多，那我会选择寄

一部分，自己带一部分。这样的意义在于回到家中，当你一 件一件从行李箱中拿出礼物送给家人时，那种迫切想要知道 对方收到礼物后的感受，以及期待，对方是否真心喜欢，会 作何处理等，这种无以言表的开心，是无法替代的。

记得青年作家李尚龙曾经写过他和作家熊太行吃饭的经历。 他们第一次吃饭时，李尚龙告诉熊太行说，你人来就行了，

我请客。

熊太行回复说：好的，那我带瓶酒。接着又问了句：你这边 有多少人？

李尚龙说，熊太行的回复堪称绝妙。 同时李尚龙给出了最差的回复模板是： A 说：你人来就行了，我请客。

B 说: 不用你请，我请。

A 接着说：不，还是我请吧。 B 再次说：不不，还是我来。

…… 两个人抢单的样子不光很丑，还很浪费时间。 李尚龙还给出了比这更差的回复是： A 说：你人来就行了，我请客

B 说：好的。

然后就空手来了。 为什么说熊太行的回复很绝妙？

一：不抢单，另外带酒，就是礼节，这是两个人关系的升 华。

二：问你这边多少人，就是保证酒不能带少，扫了大家的 兴。但也不能带多，容易喝大。

在工作生活上，擅长表达感谢的人，反过来他肯定也会被对 方所珍惜。

我们日常工作生活中所要的一切，就是把身体照顾好，把喜 欢的事情做好，把重要的人待好。

而把重要的人待好，如果只是一句谢谢就没有下文的话，是 远远不够的，重要的是说完谢谢之后，第二天的礼数，这样 才能足以表达出我们在一份重要的人际关系上，赋予了和对 方同等重量的感情。
