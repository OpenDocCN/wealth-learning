# 《你偷的那些懒，总

《你偷的那些懒，总有一天会拖垮你》

跟北京一个朋友聊天，她说：刚刚又开掉了一个员工，明明 努力工作可以做赚钱，可就是怎么劝都不愿意动。

在我的定势思维里，北上广的生活压力大，工作强度高，懒 得员工应该比较少，毕竟环境天天追着你跑。但朋友还是吐 槽，年前已经裁掉了几个懒得动 的员工了，有一部分人留下 观察，结果还是无动于衷。

我说：“对于那些愿意在职场中偷懒的人，也许怎么劝都是没 用的。”

就像我的一位前同事，我入职前她已经待在公司做了三年老 总助理，月薪 3k，三年如一日，没涨过一毛钱。她当然有怨 气，跟我们新来的几个同事抱怨：“老板是不会涨工资的。”

一两个月后，她偶然发现新来的小助理，月薪涨到 4.5k，她忿 忿不平地对我说：“比我多几百块就算了，居然比我多一千多 块，太不平公了，凭什么呀？”

私底下我们的关系挺不错，我没敢跟她说，其实我的工资也 涨了一千多，也比她多一千多。

凭什么？ 上班的时候，她偷偷看视频，拖到快下班了才勉强干一点

活，干不完照样到点就走了。第二天以来她又给忘了，然后 就是一拖再拖，最后被老板训斥。

我看她经常挨骂，帮她做了个工作备忘表，别错过重要工 作，免得总挨骂。结果没什么用！

她的理论就是，干活不要那么快，反正每天上班的时候是固 定的，工资是固定的，多干也不会给你发钱。并且她对此深 信不疑。

有一次公司全体员工一起出差做活动，晚上大家准备一起吃 宵夜，老板微信我让写一篇第二天的演讲稿，我跟大家说要 写稿子就不去了，给我打包点回酒店就 OK。

她劝我：“大晚上才给你发微信，你就假装没看见啊，我都是 这样的，谁叫老板自己不早说呢。”

另一个女孩也说：“早就跟你说过了，以前我做文案都要拖到 最后时刻才交稿，这样就可以少改几次稿。你呢？每次一天 都搞定，老板知道你干活快，所以什么急事都找你。笨了 吧？”

从职场经验来看，她们都是我的前辈，或许也是为我好，能 少干点活少受累。但是真的是这样吗？

在这个世界里，没有谁是完全不可替代的，只是替换的成本 不同。

比如说，在我们这个城市，3k 找一个文案或许不容易，但用 四千多块找一个文案就太容易，进一步说，要找一个任劳任 怨，干货速度又靠谱的文案，就又没那么容易了，而且，还 需要付出一定的时间和精力去寻找，还要给新人一段适应

期、磨合期，这成本太高了。

但要替换掉一个懒拖成性的助理，成本就太低了。 像她们说的，我当然可以装作看不见老板的信息，只是我想

把被替换的成本拉得高一些，越高越好。甚至反过来，由我

来替换付不起高薪水的老板。

我的前同事，三年合同期满没被续签，还有另一位被辞退后 回了县城。临走时吐槽老板，太特么没人情味了。

其实，坑她们的不是别人，正是从前自己偷过的那些懒啊。
