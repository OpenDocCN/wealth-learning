# 《我在星球活跃起来

《我在星球活跃起来了。为何？我有这几点与你分享》

今天粥老师在星球上发了三张图片，说我们星球很活跃，月 活跃量快一半了。其中活跃度最高的居然是我 -- 熊小帅，我 感到特别惊讶，随之而来就是感到一阵阵惊喜，这也是对我 的一次曝光吧，我现在回想起 12 月份让我活跃度确实挺高， 每天都会给星友们点赞与评论。

我为什么在星球上活跃呢？我觉得我有六点要分享给你。 第一点，因为我正在坚持日更，我不是把文章写好发到星球

上就完了，我还会看一看星友们的文章，星友们的思考。你

们的文章，字里行间中，总会有些闪光点，让我眼前一亮。

我想的是，我能够从中获得哪些启发，或许还能给我的写作

增加灵感。

第二点，这是与星友们互动的好机会，与你们交流，我发现 每个人的思考方式不同，所处的行业又不太一样，这样更能 有思想碰撞，给我带来启发。

大家也可以看看别的星友的文章，或许能从里面学到对自己 有用的知识，比如孙瑞希姐姐的职场分享，我觉得每一篇几 乎都是干货，都能给我们的职场生涯添砖加瓦，带来启示。

第三点，用心的评论，其实就是自己用心的思考，从星友的 回复来看，不一样的思考角度，会给我带来不一样的视角， 可以使得自己的思考更加全面。

第四点，我给星友们的点赞与用心的评论，也会带来星友们 的点赞和用心的思考评论，这样的话，双方都可以在思考中 进步，这也是一种练习思考的方式，相当于远程对话，交 流。互相点赞，互相鼓励，其实就是一种双赢的局面，多好 啊。

第五点，活跃起来，可以在星球上交到你现实生活中交不到 的高质量的朋友，星球上的朋友是与你共同进步的，一起向 上生长的，与你同频。

我相信，你肯定愿意交这样的朋友吧，那还不赶紧活跃起 来，与大家互相，点赞，甚至加好友，互相探讨学习，与成 长。何乐而不为？

第六点，也是最后总结性的一点，如果我和大家都一起活跃 起来了，那么就从一个小小的火花，成为燎原之势，这样也 可以增进整个星球的活跃度，从“我”做起，增加我们星球的 凝聚力。每一位小伙伴，在和谐友善的气氛中，共同成长共 同进步，你能想象这有多美好吗？
