# 《有效学习》我们之

我们之所以学习没有效果是因为不懂得如何学习，不懂得如 何思考，不知道如何问问题，不知道如何设定目标，更不知 道习得知识意味着什么，所以有时候学习没有效果，就像没 有学习能力一样，茫然无措。

以下几个学习方法，可以让我们学习更有效。 学习要有价值感：通俗说就是这件事是我们自己本身想要学

的，如果我们本身不想学，那么就不太可能学习，更说不上

精通，其次，我们要知道，我们准备学习的知识和技能是有

价值的，而且必须发现其中的意义学习的过程，其实就是赋

予某种事物具体意义的过程罢了。

学习要有目标：在精通某项技能的初始阶段，专注才是最关 键的，我们要清楚自己要学习的究竟是什么，并且设定阶段 性目标。

学习要刻意练习：某些行事的练习，可以让我们此其他人取 得更好的成绩，因此，在学习阶段，我们要通过一些有针对 性的练习打磨技巧，从而提高技能水平。

学习要实践：在这个阶段，我们掌握了基础内容之后，需要 把所学知识与技能运用到实践中去，因为只有在实践中，才 能让我们对知识与技能的实际意义理解的透彻。

学习要融合：这个阶段，我们要把所有掌握的知识与技能融 会贯通，明白各个组成部分是如何有机地结合在一起的，在 学习目的方面，了解某一个具体细节或者一个步骤是不够 的，我们需要知道这个细节或步骤如何与其他部分互动。

学习要反思复盘：在学习过程中，我们很容易犯错误或者过 度自信，所以我们需要反思与回顾，重新审视我们的理解， 从而反思学习过程中的收获。

学习是一种智力活动：越是积极主动地参与收获就会越多， 如果你在阅读一段文字，那么问几个问题：这段文字讲的是 什么事情？作者希望表明的观点是什么？这里没有没看起来 说得不清楚的地方？

在学习过程中还要有一些反馈机制，比如准备做演讲，可以 录下来看看，写文章，总之学习过程，我们需要为学习设立 目标，搞清楚我们的学习目的。

最后，别忘了审视一下自己的思维过程，你真的看懂了吗？ 有没有想过为什么总是会忘记某些内容？在这方面，一段时 间内持续学习是非常关键的，实际上，我们总会忘记某些事 实和细节，所以我们需要在几天，几周甚至几个月以后复习 一下所学内容。

最最后，今天的世界，仅仅变得聪明一些或者掌握大量的事 实是远远不够的，学习的目标在于成长为一个高效学习者， 成为一个高效利用 21 世纪所有工具的人，让我们充分发挥自 身的无穷潜力去掌握新的技能。
