# 大家好，我是争青泊

大家好，我是争青泊

--遇强则强。争做优秀 80 后，与自己死磕到底。

这是我在星球连续写作的第 28 篇

《学习这种事，带着身边人一起最好》

我曾经想过这样一个问题：如果在公司里，学习的人不多， 而只有你自己在提高，那么你有可能学有所得后，离开公 司，转投能让你的所学有用武之地的企业。那如果在家庭 里，你学习提高，可你的另一半要是还没有这方面意识，你 会怎么办？

我的答案是：你一定要带着她一起。 公司里的情况我觉得还在于受体制、环境、文化的影响。确

切的说，应该是由领导者的风格决定的，这位一把手什么风

格、什么习惯就有一些人会愿意投其所好，慢慢的形成一种

风气，一种群体风格。大多数个体决定少数人的命运。

如果大多数公司员工放弃学习提高，整天刷抖音、玩网游， 那小部分员工纵然提高了也无济于事，如果上级也是这样， 那就彻底完蛋了，因为在这样的公司里可能只能换来工资， 得不到成长。你不是做决策的那个人，而做决策的人你却很 难改变他。

经营家庭与公司也是一样的，我意识到了这点，所以，我并 不是光顾着自己学习、写作，我想用些行动并以身作则慢慢

影响她，使她能够意识到在家庭里，没有完美的某个人，只 有完整的家庭，两个人不仅要对自己的成长投资，更要给两 人经营的家庭注入一种氛围。

最近日更写作，我刚开始写时，每次都会将我写的内容读给 她听，星友们的点赞鼓励和粥老师的打赏我都及时跟她分 享。她看着我高兴的样子，也很受影响，就在我连续写作的 第 9 天，她也开始有计划的进行了学习，购买了英语流利说的 课程，专攻英语。

从那次开始后，每次我在书房写作，听到她在外面朗读英 语，毕竟她扔了很长时间，刚开始读起来很生硬很拗口，她 在外面读，我在里面笑的不行。但笑过后心里面还是暖暖 的，真的十分开心，因为我知道，一个人的成长提高可能是 在负重前行，唯有两人携手并进才能使双方受益，家庭最终 也能达到“经营”升级。

就是在这样的环境下，我们互相勉励，相互鼓励，通过这 种“外力”彼此在时间的积累下，一定能完成一个又一个目 标，我为此，相信！
