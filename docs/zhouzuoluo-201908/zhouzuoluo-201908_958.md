# 《从 0 读书到 3 月 7

《从 0 读书到 3 月 7 本书，攻克重度懒癌，我有这十点实操经验 要告诉你！》

作为从不读书的资深懒癌，《微习惯》这本书可以说是我的 启蒙之书，带我完成了“从 0 读书到 3 月读完 7 本书”的蜕变。以 下就是我的微习惯践行经验，欢迎指正！

001 明确微习惯的大力量 微习惯的精髓在于目标足够小，对于大多数人来说不需要耗 费意志力就可以完成，比如每天 5 个俯卧撑。 微习惯的大力量在于积累，通过时间的累加，足够小的目标 也能带来大的成果。

002 将目标定的足够小 贪心不足蛇吞象。只有目标足够小，每天去践行的时候才不 会有困难。例如每天运动 1 小时，是不是想想就觉得难，而每 天 5 个俯卧撑就不一样了，是不是想想就觉得简单？

003 将目标定的足够具体 目标应该是具体可量化的，不能模糊不清。 比如，我定的目标是每天坚持读 5 页书，而不是读书 10 分钟。

004 将目标写到纸上 确定践行目标后，一定要写下来，人的记忆力是最靠不住 的。 我的做法是制成表格打印出来，贴在墙上，每天提醒自己坚 持。

005 享受微习惯达成的愉悦，大胆给自己奖励 不要因为目标小就漠视完成后应得的奖励，要郑重对待，给 自己仪式感。 我的做法是完成一周的读书计划后，请自己吃一顿大餐。

006 不要盲目扩大目标 不要因为目标简单就随便扩大目标，有力气不如去超额完成 任务。 我的做法是 5 页书读完了还有力气就多读。超额完成 20 页的心 情贼爽！

007 预留“犯错空间”，确保“自我心理获得感” 最怕你习惯养成过程中被打断，就懒得坚持了。 我的做法是，提前暗示自己，你这周可以有一次“免死金 牌”，以防被临时事件打断。

008 享受微习惯带来的大力量，完成正向循环 以周、月为周期，及时回顾自己的成果，让成果激励自己继 续下去。 完成 3 月 7 本书之后，我的自我效能感和坚持下去的信心大大 提高。

009 公开：辅助坚持的好习惯 人都愿意在他人面前展现自己完美的一面，如果方便请找人 监督，更容易坚持下去。

010 加料：同类书单放送 私以为，《坚持，一种可以养成的习惯》与《微习惯》，很 互补，推荐阅读，双剑合璧，效果更佳！
