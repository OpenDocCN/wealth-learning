# 《你要相信，你是来

智多星 : 《你要相信，你是来创造，不只是服务的》

今天看小马宋的文章《我跟这位做按摩的姑娘都聊了些什 么》

一开始小马宋以为姑娘就只是一个按摩员工，他跟她聊天， 发现姑娘对按摩行业有很多自己的想法，她不单单只是做这 么个服务，她是边做边学，内化成自己的一套经验和理解体 系，然后出来合伙开店，既是老板，也是员工。

虽然按摩行业的利润不高，上升空间也不大，但是姑娘没有 用打工的思维出来做事，而是不断提升自己，学完技术以 后，主动了解店里的组织管理、经营理念、盈利模式，以及 行业的动态、趋势。

自己开店以后，她也没有懈怠，选好开店地址，算好店面和 人力成本，在大众点评上搞促销，了解对手的服务体验。虽 然她开的只是一个小店，处在一个没有多大前景的行业里， 但是她这种“不只想现在，为以后考虑”的职业意识和进取精 神，总能让她看到更多机会，拓展自己上升的渠道。就算以 后她去其它行业，也更容易被贵人赏识，升职加薪，或再次 出来当老板。

不管时势如何，我们又身处哪个行业里，都不能只是低头做 事，一定要学会抬头看路。就算大家现在的能力相当，但只 要比别人多一层眼界、多一份见识、多一点学习的自主性， 危机来临前你就能更好地躲过风险，保护自己；机会到来 时，你也能抢先一步展现自己的价值和想法，让别人看到你

的决心和可塑性。

就像文文姐和星里涴，最初进入职场打工时，不嫌自己做的 事小，混日子过，把自己该做的都做好，下班以后主动学 习，不断迭代自身能力和认识，最终争取到和粥老师一起工 作的机会。

可能别人觉得我们很傻，没事找事，浪费时间，但他们才是 见识短浅，鼠目寸光。主动学习不只是为了升职加薪，而是 蓄势待发，在被动离开前拥有主动选择的权利。

在文中，小马宋举了一个例子:吴军（原腾讯副总裁）和一位 做互联网视频的工程师聊天，他简单问了工程师，关于视频 收入的两个问题，但没想到，工程师一概不知，认为，这和 他的工作没多大关系，老板给的活做完就了，至于反响如何 他没有考虑和关心过。

吴军最后评价，“一个从业者连自己所做产品的基本收入情况 都不去了解，就永远没有机会做老板或者高层。”

每个人都不能局限在自己的一亩三分地里，只关注收入，只 在乎事少。你来工作是为了赚更多的钱，同时让自己不可取 代。如果不积极开掘和保持对行业的前瞻性，看问题更全面 更敏锐一点，那你职业的天花板就会一直卡在老板手里，没 有选择的权利、say no 的勇气和跳槽的决心。

也许有一天你会撞大运，碰到牛人，有机会和他说话，但是 能不能产生更深的联结，有进一步的交流，完全取决于你平 时的点滴积累。只有现在充好电，蓄好能，未来才有可能搭 上下一班浪潮的列车。

你要相信，不管在哪个行业哪个岗位上，你都是来创造，不

只是服务的。为自己铺路，让以后好走。
