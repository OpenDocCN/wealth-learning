# 《我为何决定继续参

《我为何决定继续参加粥左罗写作训练营》

第一期写作训练营，回想起来仍感到脸颊发烫，热血沸腾。 在那 21 天里，每天醒来第一件事打开小程序，看新课题，预 想新作业，开始酝酿，只要作业还没提交，那天就没法安 宁。

认真一比对起来，跟现在的情况也有点像，现在要求自己每 天日更，虽然没有要求字数，但设定自己必须要有新的学习 或者思考输出，实在没有也要理顺一些心中模糊概念、想 法、计划。

前几天看到继续参加二期的同学发圈，又有 300 人毕业收获满 满，令我感到羡慕和蠢蠢欲动想继续参加三期训练营复盘的 原因是——他们明显变强了！

变强的方面有很多，我最看重以下三点。

01 写作能力进一步提升 我特别喜欢训练营里的氛围，积极主动，热情好学，每位同 学都是自发自愿来到集体里面，有压力就更有动力。因为我 们知道方向是对的，只管用尽全力奔跑即可。

温故而知新。这也是我填写参加第三期的申请理由，如果参 加一期训练营就能写出好文章了，我想很多其它课程早就被 下架了，而市场的垄断趋势也早该浮现出来了。事实上，这 是不可能的，首次训练营结束之后，我们对这些新概念有了 全面了解，真正开始运用还需要进行足够多的刻意练习。

练习是枯燥的，独自练习是枯燥无味的。所以我想加入一个 朝同一个目标前进的不同团体，学习以前的知识，获得未来 的认知。

02 利他就是更好的利己 继续加入学习有两种方式，一是当学员，学费减半；二是当 志愿者班长，学费全免。我选择当班长，这是在考虑了相当 长一段时间后做的决定。

其实我根本不差那几百块钱，选择继续当学员反而容易，我 只需像第一期一样，认真做好自己的作业，成为大家口中 的“学霸焦”即可。

但是我却顶着压力逼自己选择当班长，因为我要挑战自己去 承担责任，做得更多，做得更好。我缺乏一种很重要的品质

——无私奉献。有可能是因为从小到大的生活条件不佳，也 有可能是价值观的不同，我感觉到自己缺乏“博爱”精神。

偶尔乐于助人并不代表总有无私精神，助人的事件是偶然发 生的，可能还要分场合和自身条件去决定可否提供协助；而 博爱精神却意味着：无论何时何地都想着能为他人提供一些 便利，仅仅是看一篇文章或者新闻都能从心底里散发出同情 和爱护。

我承认更多时候把注意力放在自己身上，这让我想起之前第 一期的同学，她的作业完成特别好，但是更重要的是她的个 性特别突出——乐于助人、分享有用信息。结营的时候很多 同学对她表示感谢和喜爱，因为他们都承受过她的帮助。这 个事情令我特别震惊，也令我反思自己，才发现自己一直以 来只默默地做自己的作业，很少互动沟通，很少表示热情， 很少主动为他人提供帮助。

所以这次我选择当班长，选择一条比较难的路，给自己一个 目标，去用心感受每位同学的特别之处和不同需求，学习并 尝试跨出“利他利己”的第一步，而且必须要踏稳踏正。

03 认识更多的同道中人 很幸运在第一期认识了二十几位好友，其中有成为书友的， 有成为约稿小伙伴的，也有交流写作的，我们是在当时的营 中看到对方优点和出于欣赏建立联系的。

那么，这样的延伸到第三期，何乐而不为呢？ 人脉不是一蹴而就的，它讲究时机，即你跟我都还没那么厉

害的时候相识相知，一起一走向你跟我都变得很厉害的时

候，你就是我的人脉，我就是你的人脉。

以上，我在三期训练营等你[爱心] 2019-06-18(23 赞)
