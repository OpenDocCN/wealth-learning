# 《你的急于求成，只

《你的急于求成，只会让自己万念俱灰》

文稿：橙橙第 165 篇分享 时间：2019 05 22

——————————————

「如何让你的文章做到论证精彩」

知识点整理 1）观点对口（共鸣+新知）

2）论据有力（真实+犀利）

3）案例精彩（与论点相关+大众理解）

刻意练习

论点：

＃你的急于求成，只会让自己万念俱灰＃

论据： 当期望过高，而现实的努力又太少，你的急于求成与焦虑，

其实是在搬起石头砸自己的脚。

案例： 家附近有家做了 10 多年的瑜伽馆了，可这两年生意都不景气

做不下去了，现在着急转让出去。

按理说，这么多年的老馆了，会员是相当稳定，馆主只要踏 踏实实的做，虽挣不了大钱，但也能维持正常运营挣些小钱 的。

去年开始馆主怂恿瑜伽老师们都去做某减肥瘦身产品的微 商，这倒好，瑜伽老师们一心想着怎么让会员吃产品而不是 努力上课了，馆里的课程教学质量都没有心思打理了，会员 自然不乐意了。

很多会员年卡一到期都不愿续卡，慢慢的大部分老会员都流 失到其他新开的瑜伽馆了。

这个社会就是这样，太多人急匆匆又功利心特别强，到处都 是“如何快速挣到更多的钱，买大房子，买豪车，创业成 功”。

馆主踏踏实实做了十多年的瑜伽馆，转身去做微商无非也是 太急功近利，想要快速变成有钱人，可这种急于求成不但逼 走了大部分的会员，甚至瑜伽馆都没法经营下去了。

当初那个瑜伽梦，毁在自己急于求成的手中，万念俱灰！
