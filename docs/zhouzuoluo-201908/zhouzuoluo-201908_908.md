# 《你所认为的好，未

《你所认为的好，未必适合每个人》

「 你好呀，我是温雅，这是我在粥左罗星球里的第 135 篇分 享。坚持很难，一起才不孤单。」

最近我们开展了一场丝网花展销会，今日上午一位阿姨购买 了 900 元的丝网花，在这之前，她已经买了 200 多元的花了。 起初我还比较惊讶，因为从我的角度来说，花这么多钱买一 堆仿真花不太划算，而且家里有一两盆点缀一下就可以了， 不必摆放太多。

我站在我的角度，对阿姨进行了一番建议和劝说，结果阿姨 依旧没有改变最初的主意，而且一直念叨房间的家具颜色比 较暗沉，马上要过年了，孩子们要来看她，想让房间变得更 漂亮一些，还说这些花盆的颜色太暗了，回家要把花取出 来，插在她自己的花瓶里。

最后我好像突然明白了点什么，不再劝说阿姨少买一些了， 而是尊重她的决定，并找同事一起将她买的花送到她的房 间。

在买花时，阿姨提出一个请求，希望我能帮她找一些泡沫。 因为她房间的厕所开始是蹲便，后来改造成了马桶，但马桶 周边依旧是蹲便踩脚的瓷砖，不太美观，所以阿姨想用泡沫 块将它遮住。我答应了阿姨的请求，送她到房间后，又专门 去看了一下她房间的厕所，正好该区域的负责人也来了，于 是他让工程师傅想办法整得美观一点。

通过这件事，我试着分析了一下这位阿姨的心理：

01-感到孤独，通过改变环境获取快乐 在我看来，花盆的颜色和整体的搭配还比较和谐，但阿姨一

直念叨这些颜色太“黑”了，说自己本来就长得黑，房间的家 具也黑，回家后要将花取出来，插在自己的花瓶里。

阿姨是长期一个人居住，年龄较大了，平时的兴趣爱好应该 不多，和她耍的好的有一位阿姨，但她俩又不住一起，再加 上房间的光线不是太好，家具的颜色又都是深色，所以推测 阿姨平时的生活应该比较单调，甚至有点孤独。

所以当她看到五颜六色的仿真花时，她会产生愉悦的心情， 于是想要在自己房间里摆上这些花，让自己每天居住的环境 更温馨，心情自然更开心。

02-不愿麻烦别人，自己想办法解决问题 在购买花时，阿姨向我提出了自身诉求，而且她强调了一

句，“我唯一有一个要求，就是……”也许阿姨是一个不愿意 麻烦别人的人，所以她有了想法后，也是自己想办法去达

成，当自己也不能解决时，突然看到丝网花的花盆里用的是 泡沫，于是想寻求我们的帮助，帮她找一些泡沫，她自己再 修剪成合适的形状去填充。

03-活得通透，用金钱换快乐值得 之所以阿姨和我们的消费观不同，大概是因为年龄不同，所

处的阶段不同吧。到了她这个年纪，能快乐一天就是一天，

省再多钱也带不走，不如健在时好好享受当下。如果金钱能

换来快乐，何乐而不为呢？

04-渴望得到关注 在交谈中，我询问阿姨过年是否要回子女家，她说子女让她

回去，但她还是想子女都过来陪她在这边过年。她买花布置

房间的目的之一，也是希望有客人到她房间时，感到很温馨

漂亮。

其实，这也是渴望得到关注的一种表现，期望子女来看她 时，能感受到她过得很好，能多陪陪她。

孝顺孝顺，既要孝，也要顺。有时候父母的要求、想法、做 法，或许我们不能理解，或许我们不太赞同，这时千万不要 否定他们、反对他们，最好的做法就是顺从他们的心意。

甲之蜜糖，乙之砒霜。你所认为的好，未必适合于每个人。 己所不欲，勿施于人；己所欲，亦勿施于人。
