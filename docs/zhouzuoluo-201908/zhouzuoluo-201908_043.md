# 《深度思考：当你的

《深度思考：当你的才华撑不起野心时，不妨调动一下旁边 的资源》

今天在朋友圈分享了一个很励志的销售成单记录。2 位前同事 同事介绍一名未曾谋面的第三方伙伴给我认识，伙伴介绍终 端用户，她需要 1000 盒月饼，当下酒店圈正疯狂卖月饼，这 个需求就像海里鲨鱼群中的一袋鲜血撕开来，几个酒店开始 厮杀争夺，两天内客人就定下来，这个从天而降的十几万月 饼大单，最终落在我手里。

朋友圈中我重点感谢了几位牵线人，也是我的贵人，同时确 信是因为自己有幸依附于所在酒店的良好品牌以及协助满分 的同事。在销售这个非常现实的行业里，大家只看结果。所 以我只呈现结果给老板，同事和客户，过程在自己内心，无 论内心有多少波澜，表面永远不惊。

这个过程，虽然仅有两天，我却焦虑得吃不下晚饭。现在尘 埃落地，我在细细品味这份成就感时，忽然有个声音在脑子 里边响起：那你是不是要回顾一下，这个过程有没有可以做 得更好的地方？有没有不必要的挣扎？是否陷入大可不必的 泥潭？

有！！真的有！！ 客户在香港，竞标需要寄出试吃月饼，快递如何破？

餐饮部同事一口回绝称最大快递公司某川页丰不收，理由是

含蛋肉的食品不能寄出境。

我不相信，让经常跟川页丰联系的礼宾部同帮忙联系确认， 这中间来来回回折腾了三四个回合，一会儿说可以寄，一会 说不能寄，过了一会儿又说可以寄但是需要寄件人身份证复 印件和某检验检疫证，十分烦人。

一个人做事是否靠谱在这一点上非常完美的显示出来，一件 事情一旦涉及多个部门协作时非常考验团队的各个环节衔接 性。晚上八点多时还在问询确切信息，仍然没有一个周全的 答复。

同事兼好朋友来找我一起去食堂吃饭，我没有胃口。她了解 情况后，提醒了我一句话。

她说：这个事情已经超出了你的掌控范围，这时候你应该汇 报上去，让餐饮负责人去想办法。如果试吃月饼寄不出去我 们就失去机会，而你又搞不定的时候就不能一个人硬扛着 了，适当时候求助上级，能做到最好，可以一起达到目标； 不能完成我们也已经竭尽所能。

我恍然大悟，如果我没有上报，这单生意拿不下来，虽不至 于被批评，但是没有尝试过所有方法就放弃，其实是我放任 着机会流失，总归十分可惜。

后来，餐饮负责人联系到一位川页丰的客户经理，对接上之 后全部问清楚确认第二天十点收件。客户在第三天收到样 品，查看确认后马上转账过来，订单成交。

今天回过头想起这件事，对我的启发很大。向来要求自己要 能独当一面的我竟然陷入了误区，当某事超出自己的能力范 围时，我竟忘记了要向别人求助。

三年前初入职场我非常好学，不懂就问所以快速成长，别人 用时 5-6 年而我只用 2 年时间成为销售经理，从那以后我慢慢 转变自己的思维，跟自己说从此要扛起大旗，不要再动不动 麻烦别人，不能让别人觉得自己胜任不了。没想到扛着扛着 变成了惯性思维，漠视身边那些随时可借的资源。

当你的能力撑不起野心时，不一定要等自己变强，因为等你 学成归来时，机会早已不在，或许你可以调动一下资源试 试。
