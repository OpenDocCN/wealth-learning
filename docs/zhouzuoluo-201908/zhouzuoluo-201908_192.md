# 「要写多少篇稿子才

「要写多少篇稿子才能月入过万」

通过写作训练营的搭桥引线，有幸混在几个沟通群里边，大 家每天沟通不一定多，但是主题明确，那就是写作练习，投 稿中稿。

今天有一个同学又中稿了，又有很多同学发来贺电，又有人 问绝招。她总是谦虚的说只做一件事：一直练习一直写。

这位同学目前投的公众号每篇文章稿费是 300 元，是她这两个 月中的第六篇。不禁有别的同学为她盘算起来，一天写一 篇，则一个月收入 9000 元。

这激起了另外还没出声的群里其他同学兴趣，参与话题说： 那只要勤快点，上午一篇，下午一篇，一个月靠投稿都能有 18,000 元。大家乐了，一边祝贺一边羡慕。

这时候，群主出现了。 她说：不是这样算的。你要去思考如何写出稿费 1000、2000

的稿子，通过写作能力，全面提升赚钱能力。不止要会赚

钱，你要懂一点运营、懂一点营销、懂一点个人品牌。

举个例子，比如做社群，你得会写话术，擅长沟通；比如做 演讲，你得会写作，把稿子写得出彩；就连吵架，会写作的 人也会逻辑清晰得多。

深以为然。学会写作，不仅仅为了投稿，还是做各种技能提

升的底层能力。就算只是为了投稿，也不能局限在 300 元的水 平上，还应提升自己去中 1000 元、10000 元的稿。

要写多少篇稿子才能月入过万？刚开始 33 篇，渐渐会是 10 篇，以后可能是 0.1 篇。

难道不是吗？共勉
