# 《我为什么说自己是

《我为什么说自己是“傻逼”？》

如果有人说自己是傻逼，你怎么看待？真的认为他的脑子秀 逗了，居然自己说自己傻逼？然而，对于我来讲，一年以前 的我，就是个傻逼；而我也有自信说：未来的我，看现在的 我，还是跟看傻逼一样。我为什么这么说？有两个原因。

★原因①：入了混沌大学的坑，被李善友教授带偏了。 虽然，他老说自己是“傻逼”，但是，我从他的话里解读的

是：自己说自己“傻逼”，指的是未来的自己看现在的自己

是“傻逼”。

先追溯到两个月之前，我加了几位星友为好友，然后，大家 一个讨论组里面，分享各种知识付费产品的学习心得。然 后，有一天，有位朋友，发了混沌大学李善友教授的演讲。 然后，我就好奇的点进去看了看，这一看，就被李善友教授 带入坑了：在演讲结束之时，他说道：我是傻逼，我说的都 是错的。。。

其实，我对李善友教授这句话的理解是：现在的自己，在不 断的成长，心智在不断的加强。当未来的自己成长到一定程 度，心智也越来越成熟，再返回来看当前的自己，会感觉自 己有点傻呼呼的，倒不如当下就给自己下一个简单粗暴的结 论：我是傻逼。然后，继续成长。

回答我的标题：★我为什么说（未来的）自己（看现在的自

己）是“傻逼”？★

因为：在未来的生活中，我是一名愚者，我永远能保持空杯 心态，我要不断的学习与成长，在学习与成长中，并不断地 改正自己，优化自己，摆脱现在的自己的不成熟的心智，提 升自己！

★原因②：我看了《原则》一书，对其中一句话见解特别深 刻。

《原则》一书中，我有深刻见解的那句话是：如果你回顾一 年前的自己，而没有为自己所做的傻事感到震惊，就说明你 还没有吸取足够多的教训。

我对这句话的见解是：我回顾一年前自己做的傻事，居然觉 得不震惊，甚至都觉得平常，那我怎么改正自己？那么，我 就自己说自己“傻逼”，即现在的自己看过去的自己是“傻逼”。 然后，我会督促自己，复盘自己过去的所作所为，发现自己 的诸多不足之处，甚至做的傻逼事情。

简单粗暴点，就是说：过去的自己就是傻逼。所以，要改正 过去的自己，优化自己。

现在再来回答：★我为什么说（现在的）自己（看过去的自 己）是“傻逼”？★

因为：我通过回顾过往的自己，复盘过去的所作所为，不断 的反思自己，过去有哪些地方做的不好，甚至做的跟傻逼一 样，可以发现，现在的自己对现在的自己的认识更加深刻， 发现自己有哪些不足之处，才能逐步改进自己。

★总结：

未来的自己看待现在的自己是傻逼，即放眼未来，是为未来 的成长谋大计；现在的自己看待过去的自己是傻逼，即回望 过去，是为过去的傻事做复盘，同样能促进自身的成长；

承认自己是“傻逼”，这是一种难能可贵的品质。从发现自己 傻逼的那一刻，并改进“傻逼”的自己，你就会逐步的获得成 长了。

再总结性的回答标题的问题：我为什么说自己是“傻逼”？ 因为，我要成长，不断地向上生长啊！！！

你我共勉。
