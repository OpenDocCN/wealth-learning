# 《抛弃鸡汤，选择有

《抛弃鸡汤，选择有筋骨的内容阅读》

今天是我第 97 次分享，主题是:学习作者有筋骨的气魄。 我又细读了《关于赚钱我听过最扯淡的鸡汤:赚快钱，会毁掉

你》读的很认真，我在一句句的读着体会，并不是想从中找

到赚钱的方法和技巧。而是被文章语气体现的气魄、底气震

撼、吸引了。

请你读读“你不要的赚快钱的机会，能不能给我，我不怕被毁 掉。”“有钱不会毁掉你，它会让你更从容的掌控自己的人 生，这也是财富的意义。”“赚快钱是需要实力+对机会的敏感 度和捕捉的能力+运气”

这些话分明是在讲:你根本看不到，抓不到，不知道赚快钱的 机会；你根本碰不到，见不到靠赚快钱高潮期赚的钱；（这 需要知识、见识、能力和魄力）你根本没什么资本去赚快 钱，就连“快钱”的内涵你都缕不清呢，（只片面的关注那些 游走在法律红灯的阴暗面）却在散布不靠谱的“鸡汤论点”， 是不是幼稚可笑了点!

铮铮话语的气魄、底气就如有强健筋骨的人亭亭而立。我感 悟到:我喜欢这样的文章，要选择这样的文章读，要学习这样 的文章去写。

1.载道之文必有筋骨

“仰天大笑出门去，我辈岂是蓬蒿人”我喜欢李白流芳百世的 风雅气派，我爱屈原“吾将上下求索”的探求精神，我们的血 液里流淌着中华祖先的强大基因，他们的深刻和强悍在一代 代传承中永生!

如今，互联网时代信息铺天盖地，尤其是新媒体壮大到媒体 行列，人人可发表看法，表达见解的今天，鱼目混珠现象存 在。再者，我们的阅读方式，学习方法都发生了翻天覆地的 变化。

我们不用去图书馆，找百度就可以查到想要的研究的专题内 容；我们一划屏幕就可满足精神阅读的需求。但是内容再 多，“选择”却是我们要把握的第一道关口。

春节期间学习，我接受了一个观念:优质的文章，必然把有筋 骨摆在谋篇布局的首要条件位置，评判文章不仅要看文采， 更重要的是看文章是否“载道”。可以说载道之文必有筋骨。

读文章就是会作者，作为一个有思想，有见地，有能力的作 者，他所表达的必是掷地有声的启迪，必是见物见人见思想 的多棱镜。他传递给我们的是一种鼓舞、引导、辩证的力 量，温暖的是人的心灵。

选择这样的文章给自己铸锭观念的基石，肯定不会错，所以 读要读有气魄有底气的文章；写要写能承载道理的文章、有 筋骨的文章，这样的文章是活的，是可以教人有追求的。

2.写有用的，写身入其中熟悉的东西

加入星球那时，我是被粥老师给我们讲《用公众号增加收 入》这套课时的魅力和句句入心的话语打动了，成了老师的 铁粉和追随者，当看到老师开了星球就毫不犹豫的加入了。

惊喜的是进了星球，读老师的一篇篇干货和有筋骨的文章收 获很大，长了见识，学了写作，更好的是不断的打造和宽展 自己的认知系统。这是我入星球前没想到的。直到现在，老 师的新媒体知识、写作方法和做人做事的观念以及语言表达 的金句蘋出的精彩不断地征服着我。

我原来只知道“干货”文章就是好文章，具体好在哪儿，为什 么叫干货文章一概不知。现在老读老体会慢慢琢磨明白 了，“干货”这个词是引申的网络语言，指这样的文章明细 理，很实用。

用干货概念写出的文章实战性很强，都是一些课程和经验分 享因为写这些东西的作者的观念、思路都来源于自身真实的 体验和真切的体会。

想想粥老师为什么能说到我们的心坎里，正是因为他写的都 是真实的、经历过的那些自己熟悉的却是共性的心理，对我 们来说，才是珍贵的，有实用价值的东西。

我向老师学写作还处在“完成比完美重要”的阶段，不知什么 时候能锻炼的得心应手了。最近我一直在问自己，“你最熟悉 的东西是什么？你的专业有什么干货可以写一写？”我特别希 望自己有些突破向上走一走。

回到主题上来，把好“选择”关口，无论是读还是写，都要学 习作者做人要做有筋骨的人，写文章亦是。
