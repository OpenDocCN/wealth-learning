# 《高小羊：什么是真

《高小羊：什么是真爱好？人为什么需要真爱好？》

爱好？你说，我有呀，看电影，听音乐，看书，我都喜欢。 不止你，很多人都有这样的爱好。求职时，简历上爱好那 栏，几乎每个人都写着看电影，读书，听音乐等。

我必须残忍的告诉你一个事实，这些都不是爱好，这是消 遣。爱好不等于消遣，我们太容易把消遣和爱好混为一谈。

看电影不是爱好，但是你在看完电影后，逼自己写一篇影 评，这就是爱好。

看书不是爱好，但是你能在读完后，逼自己写一篇读后心 得，这就是爱好。

听音乐不是爱好，但是一句歌词与你大脑中某一想法产生共 振时，你能将这个想法写下来，这就是爱好。

然后你的简历爱好那栏，或许可以改为：写影评，写书评， 写音评等。

那消遣和爱好有什么区别？ 它们的区别在于，消遣是你身体之外的东西，你只是看看就

过了，你是一种被喂养的姿态，在享受别人的拍出来的电

影，写出来的书，作出来的曲，那是别人的成果。当你看

完，读完，听完，日后提起时，你可能依稀记得大概，但是

经典台词，书中表达的思想，歌曲中流露的情感，你可能忘 的一干二净。

而爱好是你身体之内的东西，是你投入了大量的时间甚至金 钱，将它变成你身体的一部分，比如看了这部电影后，你通 过复盘后，电影中的思想影响了你日后的言行，判断。

再进一步解释消遣和爱好的区别： 消遣，能够带给你愉悦，是顺人性。 爱好，能够带给你痛苦，是反人性。

甚至说，爱好是先带给你痛苦，痛苦之后滋生愉悦，而绝不 是一上来就是愉悦，是从痛苦到愉悦的过程。

比如你的爱好是运动，写作，或者弹钢琴等。 对于写作之人，一开始他肯定是绞尽脑汁半天，也想不出要

写什么主题，苦恼写的主题对别人是否有用，遣词造句怎么

安排等。一段时间后，他会觉得写作有益身心健康，因为他

将自己吸收的东西经过消化糅合呈现给读者的这个过程，大

脑是在思考的，同时随着一天天翻过去的日历，他的生活是

被记录了的，而并非一张白纸。

它会变成你生活中不可或缺的一部分，甚至变成你身体的一 部分，它会让你约束自己并享受这个过程，它和你的身体会 融为一体。

爱好是你在投入了时间后，通过你自身的自律，能让你的身 体培养自制力。

划重点：它可以培养你的自制力。

好了，现在再想想你的真爱好是什么？
