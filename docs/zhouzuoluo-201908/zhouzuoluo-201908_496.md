# 《有些东西就像巴黎

蓝子鱼 : 《有些东西就像巴黎圣母院一样，如果之前没去，现 在就去不了。》

昨天一天被巴黎圣母院刷屏了，800 年历史的巴黎圣母院，一 场突如其来的大火，哥特式的塔尖在火中轰然坍塌，就像生 命的陨落。

对于巴黎圣母院的陨落，大家有着褒贬不一的评价，有的人 觉得这是全人类的损失，也有的人这是报应，而作为一个普 通人的我，只想说我都还能去过，它怎么就没了。

01 有些事情现在不做，以后可能就没机会了 寒假的时候我经过深圳，特意去了一趟世界之窗，它是深圳

著名的一个景点，世界之窗就是把世界上著名的景点按不同

比例仿建。在逛完时间之窗时，我突然有一个梦想，就是以

后如果有能力了，一定要到各个国家去参观这些景点。现在

巴黎圣母院已经没了，也许还会有其他的消失，一想到这就

感到很遗憾。

在这个千变万化的世界里，有些事情如果此时没去做，可能 一辈子也做不了。

去年我跟团到四川阿坝州旅游，在第一天晚上到阿坝州的时 候，导游推荐我们去吃一下藏餐，这个藏餐需要 150，如果不 想去吃的旅游团也有安排晚饭。同行的好几个人就不去了， 觉得既然有免费的晚餐干嘛还要花钱去吃藏餐。

我是来自农村的孩子，150 块对我这种农村来的人来说，还是

觉得挺多的。后来我仔细想了想，虽然现在的 150 对我来说很 多，但等到我有钱的时候，我还会想吃吗？有还有时间来这 吃吗？我知道如果我现在不吃，可能这辈子都吃不到了，吃 完以后我真的不后悔，因为这种经历是独一无二的。

02 小时候想要却得不到的东西，长大了也许就不想要了 奇葩说有一期的辩题是：“该不该刷爆卡买包包”，大部分人

都觉得怎么能刷爆卡买包包呢，这以后得生活可怎么过啊，

这不是理性消费，要坚决抵制。

其中最让我记忆犹新的就是正方选手花希的发言，他觉得我 们应该刷爆卡买包包。他讲了一个他的故事，花希小时候特 别喜欢看美少女战士，他当时特别想买齐那一套美少女战士 的 DVD，但是他根本没钱买。后来他有一天逛商场的时候， 他正好看到一套美少女的 DVD，虽然他现在已经买得起这些 DVD 了，但是当他站在这些 DVD 的面前，他却再也找不到小 时候的那种冲动了。

我是一个很理性的人，做的每一件事都会经过深思熟虑，但 随着年龄的增长，我会对一些事情感到后悔，后悔当初没去 做。

03 年轻的时候没去大城市，等你想去已经去不了 现在很多年轻人大学毕业了，都有一颗想去大城市闯荡的

心。有些人勇敢的到大城市打拼了，但有些人前怕狼，后怕

虎，选择了一个安逸的小城市，等到他想去大城市的时候已

经去不了了。

之前在知乎上有一个问答，就是没去大城市打拼你后悔吗？ 其中有一条回答引起了我的注意。

有一个人他是这么说的，他毕业于国内某 985 院校，学的是金 融专业，毕业时有机会去大城市工作的，但经不住父母的劝 说，他选择了回到家乡的小城市。后来进入了一个国企，现 在年薪 20w，每天过着朝九晚五的生活，但他很不开心，因为 他觉得国企这种一眼望到头的工作，不是他喜欢的，很后悔 没去大城市。

他说他现在已经去不了大城市了，因为这几年在小城市生 活，他已经完全适应了这种安逸的生活，在国企工作了几 年，感觉自己能力也没得到增长，还有自己的人脉圈都在 这，他现在已经没有勇气离开这个小城市了。

年纪越大，就越不愿意换城市，对熟悉舒适环境的不愿割 舍，对陌生环境的恐惧越大。

最后这辈子的命运，就被这座城市给绑架了。和这座城市的 命运一样，平淡，甚至走向衰败。有时候，一个人和一座城 市的命运，是一样。

所以，如果你有想去的地方，就趁年轻赶紧去，说不定它那 天就毁了；如果有想吃的美食，就趁年轻赶紧吃，不然到老 了什么都吃不了；如果有喜欢的人，就趁现在赶紧表白，不 然说不定他或她就要结婚了。
