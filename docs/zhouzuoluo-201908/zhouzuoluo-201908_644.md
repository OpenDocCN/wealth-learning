# 《做适合自己的事，

《做适合自己的事，才能做得长久》

1、 记得我刚毕业出来找工作时，我压根就不知道自己想做什么 适合做什么。学习成绩一般般，学校和学历也没有什么太大 优势，于是，我顺着学校学习的专业方向找工作。

我读的是外贸英语，找的都是外贸公司。当时我就稀里糊涂 地干着外贸的工作。可是不知道为什么，尽管我又年轻又努 力地付出一切地去工作。在外贸这个行业，我一直做得不 顺，不开心，没成就感，没存在感，更看不到发展前途。

我最终下定决心转行。我多次告诉自己我不要继续在这个行 业耗下去。尽管去到另外一个全新的行业要我重头做起，我 也要转行。

我一开始以为我喜欢英语，我可以把自己的专业工作做好。 现实生活中尝试过，我确实没有在这行做好。即使，现在回 想起来那段外贸公司的职业生涯，我都不知道那段时光带给 了我什么。

说句不好听的话，就是试错的一段经历和沉淀成本。没有收 获一帮好同事，没有一段美好快乐的时光，更没有积攒什么 能力或者人脉关系。

2、 当我转行去做定制家具公司销售后，我整个生活状态都像是

会发光一样。每天都过得特别开心，充实，快乐和幸福。

我无论什么时候回忆起那段时光，做销售三年的职业生涯， 我都有说不完的故事和满满的回忆。正因为这三年的工作积 累，也为我现在的职业生涯奠定了基础，也因为以前的同事 关系积攒了人脉关系，才有今天的创业机会。

在我做销售那三年，我以为我可以不用再做销售了，我的销 售生涯已经到了尽头。我要转岗，从销售转到管理层。当我 拼命地去争取管理层职位时，再次落败。

后来，得知在公司做管理层并不是因为能力很强就能胜任， 而是人际关系要很强这个才是关键。当我得知这个潜规则 后，我终于明白自己为什么当不了管理层了。至少对于当时 的我来说还不行。

3、 当不成管理层，又觉得做销售没有意思。刚好趁公司改革， 应发展需要成立很多新部门。我顺势转岗去其他部门做小组 长，后来兼职做培训。

做了一段时间小组长带过新人，组建过团队，又去全国各地 做培训后。我以为我可以往培训方向走，在去年我想过我未 来的职业有可能是培训师。为以后能做培训师，我去接触和 参与了一些培训类的课程和组织去学习。

4、 在互联网迅速发展，微信公众号崛起，写作风刮起时，我曾 经也想过想通过写作变现实现个人创业。因为我发现我越来 越不喜欢本职工作，越来越不喜欢公司体制化朝九晚五的上 班模式，一眼望到头，平平庸庸地度过下半生。

我应该要趁还有机会的年纪去尝试做自己喜欢做的事。不管 成不成功，赚不赚钱，最后我是按照自己的意愿过一生的， 这样的人生就足够了。

于是，我又下定决心辞职，离开公司离开稳定的体制，选择 去做自己想做的事。

5、 等我出来之后，我发现做自由职业并不现实。我想做的事是 什么？原本以为我理想的生活是时间自由，选择见哪些人不 见哪些人是自由。每天可以看书，写作，健身，有时来一场 说走就走的旅行。

这样理想的生活多好啊，也正是我想追求的。我尝试了一个 月之后，这样的生活不是长久之计。首先，家里人不会支持 的，来自家人的压力，社会关系上的压力等等。最重要的还 有经济上的压力。

虽然现在每天都很忙碌，每天都想着如何赚钱。目前的这种 生活状态反而让我感觉更加踏实有安全感。

为什么？因为我知道了所有享受型的生活不是建立在幻想和 异想天开，或者不劳而获，或者是家里条件好给予的经济支 持。

所有享受型的生活，都是经过辛苦劳作之后，获得经济或获 得奖赏，才去享受才是心安理得的，连快乐的感觉都不一 样。

于是，我更加笃定想要获得享受的生活，必须要先去努力奋 斗争取回来。并不是靠异想天开。

6、 我觉得并不是你想做什么，就能做什么。而是你适合做什 么，才能做成什么。

一开始并不知道自己适合做什么，所以，不断地尝试想要做 这个那个。最后，实践和现实会告诉你适合做能做什么。

我以前做外贸的时候，一直找不到成就感，存在感。什么意 思？就是平时工作你做得累死累活，忙忙碌碌，付出很多。 但是做成的这件事是与你无关的。我觉得这份工作不开心没 奔头没劲。

一开始迷茫是可以理解的，不知道自己想做什么能做什么， 那就去尝试。尝试过你就有想法有体会。我觉得我们的内心 欺骗不了自己。我们的内心知道自己真实的感受是怎样的。

我做销售那段时间，很开心很愉快。这件事有值得深挖的地 方。越做越发现这个职业其实有很深的学问，值得我去学 习，做得越久越能显示出自己的价值。

7、想做的事和适合做的事 我想读书，写作，旅游。好处是什么？每天都呆在家里看书

写作，减少不必要社交。但想想这真的又是好的吗？我真的

是想把写作当作一份事业吗？至少暂时没有想清楚。

我想做培训，培训做着做着会变成理论派。销售是实战派， 有成就感和存在感，我用我的业绩告诉别人我的实力。

我以前特看不起销售这个职业，因为起点低谁都可以做，感 觉不受人尊重，很低端。能不能做得好又是一回事。

可是，偏偏我却又离不开销售啊，别人看到我的价值好像也 只有销售。

我想做销售吗？没有。我适合做销售吗？好像适合，要不然 怎么会做了那么久。而且，我做销售做得挺顺的，外人也一 直很看重我销售这点价值能力。做销售的这个过程中，我能 感受到自己的价值感和存在感，是一种被需要的感觉。

8、 想做什么和适合做什么到底是不是一回事？不是。

或者说，想做的事和适合做的事，选择那种？我选择适合做 的。适合代表了一种舒服的状态，一种长久的稳定性。在做 适合的事情里，我希望找到自己的价值，通过一点点积累价 值和财富，才有选择权去做自己想做的事吧。
