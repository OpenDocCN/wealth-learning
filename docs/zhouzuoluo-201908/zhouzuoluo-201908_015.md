# 这世界就怕两种人：

章子 : 这世界就怕两种人：一种是超有天赋，一种是死拼，你 是哪一种？

我们身边经常有人被夸，有天赋。天赋是什么？天赋是说， 一个人生来具有的生理特点，天生就具有了后天才智发展的 物质基础。但是，人的聪明才智只能通过后天的社会实践， 才能真正形成。所以，两种品质都有的人，才能成就一番能 力。

文学家莫言就是个具有文学天赋基础，后天又勤奋训练，而 成的文学人才。

今天我听了，莫言上学时的老师讲，他小学二年级的时候， 写的作文就与众不同，别的学生写运动会场面，总是面面俱 到，平铺直叙，讲了赛跑，讲跳绳，然后还不忘讲一讲篮 球。

而莫言却懂得突出重点，就抓住篮球详细描写，篮球场上人 物奔跑的动作，跳投时的神态和展现性格的语言，来表现同 学们快乐，勇敢和执着。其他的，一带而过。

莫言自己也回忆说，他小时写的作文，经常被老师当作范文 读给全班同学听，有时还被隔壁的中学，拿去当作范文展 览。

听了这些真实的述说，我感到莫言的确属于有文学天赋的 人，但他能成功的成为文学诺贝尔奖获得者，后天有多拼， 只有他自己知道。

我曾看过一段写他沉迷读书的情景。那是一本《青春之歌》 他的朋友只借给他看一天。他就躲到草垛上，精神全部沉迷 到书的情景中，忘了吃饭，忘了自己放羊的任务，也忘了有 可能挨打的风险。

莫言虽然只有小学文化，但是他不停止努力的步伐，始终追 求不断的成长，终于考入解放军艺术学院，他一步步踏实的 走，扎实的练，忘我的写，才写出了世界公认的，有人文情 怀的，有深蕴内涵的小说。如《红高粱》《蛙》等。

这就应了那句古话：世上无难事，只怕有心人。 我们虽然没有交往过，但无比熟悉的人中，还有一个人。她

也是一个天赋和死拼，两种合体的人，就是孙俪。

孙俪是一个公众都喜欢的人，从她演的两个著名角色，甄 嬛，羋月。融入人物到位，心灵体现真切。就能了解她的表 演艺术才能多么高超。他对人物的把握准确，刻画到位，表 演娴熟，无论是帝王高贵，还是百姓平凡，神情仪态，落落 有度，没有任何违和感。

生活中，孙俪低调平凡，与普通人没有什么两样。就是快乐 平静的过日子，计划到位的教孩子，平淡而幸福。

从她身上我们看到：越是有艺术水准的人，越是无浮躁负气 的人。

今天我看了孙俪的书法。更是惊异她的勤奋和努力。又更深 一层的了解了她的内心世界。平时孙俪喜欢书法，画画和做 手工，看她的书法，小楷精美，唐楷纯粹，真是字如其人， 那功底的扎实，那笔画的秀美，那笔法的华贵，让人感到和 她本人一样优雅淡然，气质绝佳。

心中有大气，外在必淡然

一个人成功，必定是有原因的。孙俪是个日积月累，常年坚 持清晨练字的人。她的心得有多定，神得有多闲？如果是一 个艺术才华，与上善若水结合起来的人。那是不会辜负天赋 赐予的。

其实，每个人都具有一定的天赋。找出天赋再加上后天不抛 弃、不放弃的努力。会怎么样？你看，粥老师成功了，文文 成功了，林克成功了。你离成功还远吗？

挖掘天赋，坚持努力。就算不让世界害怕，也让自己吓一 跳，“原来我也这么强大！”
