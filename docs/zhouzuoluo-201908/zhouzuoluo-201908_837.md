# 干货：高效学习成长

猫小呆 : 干货：高效学习成长 我相信世界上所有的事情都是有一定的运行法则，也就是

——规律和套路。只要你能找到，就能比绝大多数人做得 好。

一 人是与生俱来建模欲望的 为什么说人是生来具有建模欲望的？举个例子，现在和你说

7345069218，你很难记住这里的数字。但是换个排列顺序， 数字还是那些数字，0123456789，秒记对吗，因为脑子里已 经有了基础的模型。因为好奇心，我们有了寻求知识的动 力，我们会去搜集例子，建立模型；而因为建模欲望，我们

会把所看到的例子、知识进行分类，慢慢构建出自己的底层 系统。有人说好奇心会随着年龄的增加而降低，其实不是， 是因为年纪越大越容易被一些权威思维和自己的固执心理框 住。

我用印象笔记建立的知识管理分类放在照片里 二 学习原则

我以前是个屯课狂人，各种类型的课都爱屯，从营销，写 作，文案，运营，裂变，ps、PPT 等教程到易经八卦，历史， 健身，语言学习，区块链，代码，手绘等，因为觉得自己什 么都不会，算是个标准的松鼠。但是在屯了 10 个 T 的课程资料 后，我明白了一个道理——把最需要解决的问题最先学习。 学习的内容是要用来解决问题的，漫无目的的学习最有可能 的结果就是搭进去时间和精力，但是最后慢慢遗忘的干干净

净。所以现在的学习我只会在明确任务后，再进行输入和输 出

三 学习方法 第一步，观察

观察那些在这个领域里做的成功的人是怎么做的，这个是收 集原始素材的过程

第二步，总结 把信息加工处理成有逻辑的数据，提炼信息之间的联系，在

有限的例子里找出规律，把信息压缩成知识。我一般会用卡

片或者思维导图做记录，这个有一个好处，输入的是庞杂的

信息，输出的却可以是知识网络

第三步，模仿 一个烂开始总比不开始来的强，更何况不去使用的知识等于

白学。前两天我在疯狂的看别的英语大牛是怎么学好英语口

语的时候，突然有了一个感触。小时候我不会游泳，去游泳

池爸妈就把我放在一个充气沙发上，我玩着玩着就掉进了水

里。水是一口一口的喝，然后求生欲让我不停巴拉，每次巴

拉几下就能往上浮一会，就这么自动学会了狗刨，再慢慢演

变成了蛙泳。之前爸妈也和我说过游泳的方式和原理，有用

吗？没有。有很多技能的学习是靠你的肌肉记忆来的，英语

其实也是，接受了十多年的学习，一次性过的大六，为什么

口语还是烂？因为不说，不用，大脑和肌肉都没有形成记忆

和快速的条件反射。

第四步，反思复盘

如果想要做一张好的思维导图或卡片，第一步是看导图和卡 片达人是怎么做的；第二，看看能总结出什么共同点，无论 是内容版式字体色彩还是工具；第三，去结交这个领域的达 人，把自己的作品发给他们，让他们告诉你哪里做的不好， 该如何改进，如果暂时结交不到，就把作品放在网上，听观 众给你的反馈。不复盘只能叫经历，复盘了才能叫经验。当 遇到自己有兴趣学习的领域有很棒的作品时，不要只觉 得“嗯，做的真好”，而是要去挖掘“为什么觉得这个好”。然后 在下一次制作的过程中自己也去尝试，这样水平就会快速提 升。

第五步，再总结 经过复盘之后的规律和套路会更加精准完善 四 小结

个人学习成长之路：观察——总结——模仿——反思复盘

——再总结
