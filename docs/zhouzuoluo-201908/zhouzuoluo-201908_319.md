# 《想到就去做，成功

《想到就去做，成功可能会迟到，但你不可能会一直输》

何可乐在星球的第 1 篇分享 时间：2019.06.11 天气：阴转[太阳]

—————————————————————————————————————————

这是我首次在“粥左罗和他的朋友们”这个星球里发言，但我是在 2018.08.12 就加入了，再过两 个月就又得交保护费了。

在这个星球里我觉得我算不上粥佐罗的朋友，顶多算个追随者？毕竟有哪个人有朋友会默默 在你身边一言不发快一年？

从开始踌躇满志的加入这个星球，暗中观察大家发言。感受到了一个又一个写作小白因为坚 持和努力写作而蜕变成长，又见到了一个又一个各种领域的牛人大佬吸了一波又一波粉。

以前我一直相信一句话“无论哪个时代，都不会亏待肯思考的人”，所以我选择在毕业的时候 从事新媒体运营工作。

后来我又听到“莽撞定律”这个词，还有浅读了《曾国藩》，粥左罗老师也说过“不要想清楚了 再去做”。我才知道那句话后面应该是“无论哪个时代，都不会亏待肯思考又肯行动去做的 人。”

以前为什么不在群里发言？可能有些人跟我心态一样，在喜欢的人粥左罗老师和星友们面 前，都希望自己不鸣则已一鸣惊人。我第一次在这个星球写的东西，虽说不指望粥老师能给 予肯定和打赏，但是也希望能有一些素不相识的星友给我点赞评论啊。

于是我就天天暗中观察，看看别人怎么写爆款，看看别人怎么吸引注意获得打赏。结果呢？ 跟我同期或者后期进来的人，只要坚持每天动脑思考写的都获得了成长。

像我这样暗中观察，蓄力非得憋大招的结果越来越没信心了。 原因如下：

1.光说不做假本事。

我光思考，不去做，哪里来的真实反馈？没有真实反馈你怎么完善自己不断进步？ 一直自信自己肯定能写出爆款文章而一直拖着不写简直是灵魂自嗨永远飘着。

2.心灵鸡汤喝的太多。 沉浸与粥左罗和星友们的励志故事太久，误以为这么真实的例子会是以后的自己。第一次因

为粥左罗老师的爆款文章而关注了创业邦，后来有一天又在朋友推荐的插座学院见到了粥老

师的文章。

那时候突然开始想了解这么一个人，当时正好是我毕业去北漂。北京这座城市我一次没去 过，我是山里的孩子。为什么去北漂？因为没去过所以成了我毕业时候想去的唯一理由。那 段日子真的很难，我又看到粥老师写的东西。

他的案例太真实了，真实又接近生活，好像就是身边一个朋友通过几年努力变得非常牛逼。 所以当知道粥老师创建星球的时候，我毫不犹豫的加入了。

然后我见到除了粥老师以外其他星友的励志案例，我沉迷于这些分享而误以为我就是以前的 他们，他们就是以后的我。

大学之前心灵鸡汤盛行，老师推荐下我就马上去图书馆借阅了一本。那时候高中学习真的太 苦了，我需要心灵鸡汤奶我一口。

大学之后，我更喜欢毒鸡汤。因为鸡汤能让我充满能量，觉得只要去做我就成功了。但毒鸡 汤能让我认清自己的定位，处境。逼迫我动脑思考，如何找到出路。

而现在，我好像又沉迷在星球的鸡汤里。 突然吓出一身冷汗，我竟然不会思考了。我只看到了粥老师和星友月入 N 万，用了 X 年从打

工仔逆袭到高管或者摇身一变当老板。而没有思考过，他们为了达到这样的高度，都做了些

什么？经受了多少苦难？而自己又付出了多少，收获了多少？我凭什么能达到甚至超越他们

的高度？我凭什么能说我会成功？我哪里来的底气？谁给的我？

3.想的多，做得少，提笔不知写什么。 我这个年纪，虽然才毕业一年。但是网上贩卖的焦虑，我买了不少。 我不再像儿童一样那么容易获得快乐，我因为需要考虑的过多而变得现实。 我因为经历过从山里出走大城市遇到的人或者事而变得心有感慨。 我因为赶上知识爆炸的年代而自己作为一个知识青年而变得心中一直有所想去表达。 正因为我还没过上想要的生活，活成当初自己喜欢的模样，而变得心浮气躁。我内心有很多

想表达的，但是我却不知道如何描述自己心中所想，提笔一写就语塞，甚至写着写着就不知

所云。

我写东西或者拍照片的理念就是：“如果自己都不能感动自己，又怎么指望去感动别人？” 但就是这样，你越不去写，你越不知道怎么写。

很典型的例子就是：

你从来不发朋友圈，突然想发照片配文却不知道怎么写文字。于是改来改去，最终就是不满 意删了不发。

你从来不拍照，你突然去拍照肯定会觉得屏幕里的自己表情不自然，甚至嫌弃自己长得不够

好看。

当我不做的时候，别人做了，别人就进步我就原地踏步甚至退步。 当我想做的时候发现跟别人已经有了差距，觉得还是再观望观望结果还是没有进步。 你看，所有的一切，你越不去做你越不会做。你越逃避，你越恐惧。

今天早上突然有个电话打来，是快递小哥。出门拿了快递，发现原来是前两天打开星球看到 粥老师推荐的《学习写作》。于是中午吃完饭，我就打开书，前面几页依然是熟悉的粥老师 心灵鸡汤味道，鼓励着我们继续学习，向上成长。

转过头看了眼窗外的天气，不错，放晴了。我感受到了好像一股强大的力量在召唤着我，我 不能再低调了，我决定要打开星球写作啦！我今天就要把我的星球第一次分享发出！即使没 人为我鼓掌，我也能自我欣赏！（好非主流，突然 BMG 响起来超级女声主题曲）

发现自己问题不可怕，可怕的是不去思考解决办法，而是自我感觉良好，还无动于衷。 鉴于以上问题，我做出以下决定：

1.既要说，也要做，而且是想到就去做。

因为现在写的少，所以获得的反馈少。根本不能通过一次两次就能获得太多有用信息。想到 就做，即使莽撞出现错误，但那也是宝贵经验。

前期先多做积累经验，通过量变引起质变。后期精进，提高层次水平。不能老是重复做着已 经不能让自己成长太多而且浪费时间的事儿，应该把时间用在学习提上自己或者去做自己喜 欢的事享受生活。

2.不管鸡汤是否有毒，只要有用都喝。 虽然我们一直都在倡导快乐成长，但那只是理想状态。像我这样小时候开始学着去放牛觉得

特别痛苦，后面我就觉得特别有意思啊。

没有那种非常优越的条件从小无忧无虑的成长，但是父母已经尽力为我们遮风挡雨了。当我 们觉得翅膀硬了，出了社会得自己成长，那种落差还是非常大的。大家不是你爹妈，没人会 一直惯着你。

上了大学后我一直明白一个道理： 成长必定是有一段痛苦而漫长的过程，很多苦难都是你的人生投资。 所以我去北漂，我痛苦并快乐着。 痛是因为生活真的很苦，工作真的很累。 快乐是因为我知道，我要的是什么，我应该怎么去做。

未来的你，应该感谢现在多灾多难却不放弃的自己。

3.想的要多，做的更要多。 很多时候我们确实想得多，特别是晚上容易想多。

悔创阿里杰克马曾经说过：现在的年轻是，晚上想到千条路，白天依然走老路。

现在大多数人都接受过教育，不是文盲。而且知识爆炸年代，我们想了解什么基本网上都能 解决。

所以我们想的越多，做的越少，就越浮躁。 以前在公司里端茶送水，复印文件，大部分人觉得大学生毕业是来上班的不是来打杂的。但

是通过前期我很低调的主动完成这些后，我发现老板和高层对我都很好，去谈事情愿意带着

我，团建时候总是让我去安排。

有些人会说：“你们老板还是当你打杂啊？” 但是如果我不去做我怎么知道每一件小事的流程？如果粥老师不是从小编做起怎么知道管理

团队时候怎么团结人心？如果董明珠不是从销售做起，当了老板还是会经常到生产车间查看

怎么敢自信给自家产品代言？

按照目前的情况，调整心态。从小事做起，把小事做好再做大事。通过小事打地基，为了自 己以后筑高楼有个牢固的基础保证。

可能是因为山里出来的原因吧，性格有点不服输。 以前考起大学，我也是村里的骄傲。 虽然现在还没有活成自己喜欢的模样，但我坚信。 成功可能会迟到，但你不可能会一直输！

—————————————————————————————————————————

第一次写分享，开始写了几百字的时候脑子就开始混乱。想到以前还经常写高分作文就觉得 汗颜，提笔忘字，逻辑表达偏题不知所云。

好几次差点就想关了浏览器不写了，看看电影放松下，但还是决定试一试吧。 开始觉得 1000 字都难得写，没想到我写完的时候已经 3000+字了。哈哈哈！ 可能文笔没星友们写的好，但还是希望能努力追上大家的成长脚步，所以请你们注意点，老

子要来追你们啦，你们快跑！

现在，我也发言了，而且会坚持每天日更，我发现写作确实能让我的逻辑思维更加清晰合

理。为了不让我在二十多岁的年纪提前痴呆，我要多动脑子。

终于鼓起勇气跟写出来，能自信的说我是这个星球里粥左罗的朋友啦！我和大家是朋友啦！ 向上生长，是年轻时候需要做的事。

与众人成长，是年轻时候最快乐的事。 爱你们，爱这个星球！
