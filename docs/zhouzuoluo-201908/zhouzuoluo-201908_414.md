# 《写给成年人的童话

《写给成年人的童话书，教你什么是爱》

我们真的懂什么是爱吗？——第 180 的彦初 这几天二刷《小王子》，在看第一次的时候，我就对自己

说，这本书要隔几年就重温一下，因为每一次读，都会有新

的感悟。

前段时间香港娱乐圈的「安心出轨」事件，从许志安的道歉 发布会开始，到郑秀文的原谅声明，很多人不解，为什么那 么优秀的天后郑秀文，要原谅许志安。

媒体列举过往的妻子原谅丈夫案例：马伊琍的「婚姻不易， 且行且珍惜」，谢杏芳的「给认错的男人多一次机会」，为 什么会原谅？

《小王子》里其实有说出原因。 狐狸告诉小王子，“正因为你为你的玫瑰花费了时间，这才使

你的玫瑰变得如此重要。”小王子也终于知道他那朵玫瑰和花

园里玫瑰之间不同，之所以成为他独一无二的玫瑰的原

因：“她单独一朵就比你们全体玫瑰花更重要，因为她是我浇 灌的。

没有人能拒绝爱情的魅力，认为爱情刻骨铭心，不正是因为 我们对对方花费了许多时间和精力吗。爱情，爱人，丈夫或 妻子，对某些人来说是唯一重要的，不正是因为我们为之付

出过一切吗？

因为如此，你才会在对方犯错后，选择原谅，这也是你对自 己，对对方，对你们的婚姻或感情所承担的责任。

对啊，爱是一种责任啊，爱与被爱都有责任，我们都需要对 我们付出过的感情，对对方，对自己负责，不是吗？又怎么 可以背叛对方呢。
