# 《品酒体驗: 懂酒

《品酒体驗: 懂酒比死勤奮更靠譜，你相信嗎》

今次是我第 105 次分享。 大家互相学习。 昨晚与友人出席由青商举办的品酒晚宴，前后品尝五种不同

餐酒。 由前菜沙拉，日本哈蜜瓜，餐汤，主菜到甜品，以致

认识不同朋友，渡过丰盛一晚。

同时，我有另一番深度思考，为何人懂饮酒可以在社交上深 受欢迎?

这令我想起粥老师曾在文章提过一比喻，我不会在能力 A 上跟 你比拼，我只要在能力 B 上战胜你就可以。

等于你用剑与我比试，我只要一粒子弹便足以 KO 你。 今天很多上班族包括我自己，可能忽略这项能力，社交能

力。

我们从小到大一直被灌输一种观念，就是努力读书，考试， 入大学，找份好工作，早日建立家庭，早日生小孩，再为小 孩提供最好教育，然后助他成材，再鼓励他/她 找到好归属， 抱孙，三年抱俩，等等。

我们人生，是否就被像工厂流水线般复制成差不多的"公仔“ 呢？

多少人能逆向思考，不要求自己成为最勤奋的学生，加班最

厉害最早上班那个，反而成为最有社交能力的中心人物？

在学校，老师不会教你怎样结识朋友， 也不会教你怎样成为最受欢迎的人， 也不会教你如何将心比己，避免成为自我中心的人。 我认识有位朋友 A，他学业成绩不错，有数学，语文及音乐天

份，可惜因为独生子，性格比较孤僻及自我中心，大大影响

他建立人缘及社交能力。

相反，我另一位自小认识的好友 B，他父亲是福建人做生意， 小时候参与过童军经歴过群体生活及训练，到初中认识他时 已发现他为人较成熟，懂得深度思考， 及为人设想。 在往后 的日子他每逢参与过的群体都深受众人欢迎与爱戴。

事实上，在社会实况中，能够成为领袖或老板都不是最勤奋 或知识量最丰富，相反主要因为他超强的交际及组织能力。 例如我有个供应商，老板最会交际及与人沟通，工程及跟进 等工作他都下放到更适合的同事下属。

这次品酒晚宴上，由浅尝第一枝法国香槟(champagne)，到第 二只枝新西兰白酒，再到另一款味道更特别的法国白酒，到 最后一边品尝甜酒，一边享受最后一道菜， 焦糖布丁。

在过程中大家渡过愉快的晚上之余，更愿意开放自己畅谈一 切。

品酒在不同层面的社交场合中越来越受欢迎，若果我们能培 养出对品酒有一定认识，将会为社交能力加分。

例如当你能告诉友人红酒配牛扒，白酒可配海鲜及大部分食 材，更可用作煮饭的调味料，及红酒应存放在温度 16 度的酒 柜，这些都能增进新知，及令 友人对你另眼相看。

千万不要忽视社交能力的培养，有时它比盲目努力工作更有 效。

数年前我上司的女朋友学习过插花，当她进入新公司上班 时，原来她公司的高管也喜爱插花，最后她们二人兴趣相 投，从此有很多重要项目都由她负责。

无数人上班只懂得把工作做好，追求准时下班，却缺乏思考 能力，不懂思考如何追求工作学习并行，不懂思考培养更多 社交能力。

我们最终能够逆袭成功，可能就是靠某项社交能力突围。
