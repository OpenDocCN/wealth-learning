# 《真实的人生并不存

《真实的人生并不存在一条既定的跑道，但存在一个决定的 自己》

关键是走在那条路上的你，有没有决定在那条路上走下去

——第 152 天的星里浣

“真实的人生并不存在一条既定的跑道”。这句话是我在 @粥 左罗笔记 的推文 中看到的。在这篇文章中，这句话是我们应 该在 25 岁之前明白的第一件事情，也是我目前感受最深的一 件事。

我对这句话的理解是：每个人都有每个人自己要走的路；不 同的路，源于不同的选择。

在学生时代，我们并不见得知道自己以后会走上哪一条路； 现在好好学习，以后出人头地，应该是我们最为明确的一 条“大道”。在高考之前，我们都会在这条跑道上奔跑。

但即使是这样一条长达十多年的既定跑道，我们也是在“各跑 各的”。在沿着这条跑道前进的过程中，有人继续好好学习， 有人开始荒废学业；有人去学了美术，有人开始搞运动，也 有人早早进入社会谋生。

小学毕业、初中毕业、高中毕业、大学毕业、这些节点就像 一个个路口，每到一个路口，就会有一部分人走上不同的 路。就像文章中说的那样，”我们和同龄人浩浩荡荡地一同出 发，然后在分岔路口奔跑着散开”。有些人跑得快，有些人跑

得慢；有些人还在原来的路上，有些人已经更换了跑道。

我也在这条路上跑了 21 年，再有几个月就 22 年了，回顾这一 路走来的自己以及旁人，我看到了让我们走上不同道路的原 因，是我们当初所做的每一个选择决定的。当然，也有父母 或他人帮你做的选择。

我以前选择了逃课上网，所以我走上了荒废学业的道路；我 选择随便选一个专业；于是去了一个毫无兴趣的学校，度过 了一段没有看点的大学生活，开始越跑越累，越跑越偏，好 在后来及时转向了。

我的高中“知己”选择了辍学逐梦，后来我看到她在朋友圈 PO 出的照片，光鲜靓丽，潮范十足，但是她说过得很辛苦。也 不知道那条路对她来说，是好还是坏。

我的表姐选了一段不成熟的感情，婚后的生活充满了鸡毛蒜 皮的争吵和的分外红眼的责任推卸，现在正在闹离婚；两个 孩子，一个 7 岁，一个 2 岁不到。

人生的道路，你选择逃避，不一定能逃避到底；你选择逐 梦，也许会掉入进退维谷的两难；你选择早早成家立业，未 必就能幸福美满。

其实重要的，不是一条怎样的路，而是选择哪一条路的你。 你选择的路是错的，不一定就不能回头；你选择的路不好 走，也许后面就慢慢好走了；你选择的路上满是坑洼，未必 就不能填平修补。
