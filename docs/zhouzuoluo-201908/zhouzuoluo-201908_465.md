# 今次是我第 74 次的

<先不要自我设限, 去向一流的大师取经吧!>

今次是我第 74 次的分享, 请大家多多指教。 如果你渴望能够在一个行业中快速成长, 你会选择在这行业中

找第一名的高手, 还是第二三四名的高手学习呢?

最近的电视剧中重播着新一辑的金庸名著<倚天屠龙记>。先 不谈论这一辑谁所扮演的角色或美女, 大家有没有想过为何张 无忌的武功会进步神速?

其实最核心的答案很简单。张无忌所学的功夫全都属于顶级 的包括九阳神功, 乾坤大挪移, 圣火令神功, 太极拳, 太极剑, 九 阴真经, 降龙十八掌等。所以, 最后造就了他以 20 岁小小的年 纪, 能够力压群雄, 并当上了明教教主带领会众推翻元朝。

张无忌

香港每年都有个十大杰出青年的选举。主要是每一年, 都会在 香港选出十位在各行各业中最有代表性及成就杰出的人士。 例如在单车奥运中拿到金牌的李慧诗, 滑浪风帆奥运金牌得主 李丽珊, 单车选手黄金宝, 著名舞台剧演员彭秀慧, 歌手刘德 华, 黎明, 陈慧琳等。

至于在新媒体写作上称得上一流的大师的, 应该是咪蒙吧。当 然, 粥老师也称得上是其中一位常写爆文的大师, 所以我们齐

向他学习便是在写作上能快速成长的其中一个途径。

昨天我做义工的青年商会机构有幸能够访问到其中一位十大 杰青龙子明先生(1995 年杰青)。他现在是香港宏利保险集团的 区域总监, 在从事保险之前, 他在政府机构工作了 12 年。但在 访问中他透露当他第一年在政府机构中工作时, 便已萌生退 意。主要的原因是他即使付出的努力明显比人多, 但每月的工 资也是和其他同事一样。他了解到在公营机构中工作并不是 按努力行赏, 而是按年资排队, 所以最后促使他进入做保险的 自雇行业。

为何他最后可以比同业做得出色? 主要是他从来不怕辛苦, 也 从不自我设限。在访问中, 他认为他的一生中没有特别大的挫 败, 但困难从来就每个人都有, 主要是看你如何去面对。

在我角度看, [自我设限] 这四个字看似简单,但实践起来绝对不 是件易事。

大家可以想想, 你有没有曾经自我设限?

在小时候读书时, 老师告诉你没有数学天份, 从此你便提不起 劲读数学。你英文读不好, 默书不合格, 从此你便不爱读英语, 对吗?

你第一次, 第二次买股票亏了钱, 从此你便觉得自己没有运气, 没有投资的天份, 于是提不起劲去学习投资, 对吗?

你在星球发了帖文, 但没有星友赞好, 于是就认为自己没有写 作天份, 不再日更写下去, 对吗?

好多时候, 自我设限来自于你屈服了别人对你的评价及目前的 结果。

在访问中, 龙子明先生道出了小时候的他非常自卑。由于他父 亲有其他的老婆, 甚少花时间照顾他, 导致他学业成绩不好, 最 后被赶出学校。直至他转校时遇到了另一位对他不离不弃的 老师, 从此为他带来了希望, 人生于是开展了新的一页。

只要你愿意踏出第一步, 打破[自我设限], 你的努力便会慢慢开 始累积。虽然不是马上带来改变, 但你的目标将会离结果越来 越近。

为何本文要强调找一流的大师学习呢? 因为通常一流的大师都代表着该行业的最高标准。 例如香港的著名喜剧演员及导演周星驰, 他所拍的电影如少林

足球, 功夫, 长江 7 号等电影都带来超过六千万及五千多万的票

房收入, 他饰演的搅笑形象深入民心,至今仍无人能及。但周星

驰的演技也不是先天的, 由他初出道参与儿童节目时, 便已经

非常努力。其后他在电影中的每一次演出即使担任配角时, 都

比任何人认真及用心。最后, 这位一流的电影大师便这样诞生

了。

做事极度认真及用心似乎都是一流大师不可或决的条件。 假设你明天上班, 后天便开始放大假。你会选择准时六点下班,

然后马上编排如何去玩乐, 还是依然认真用心工作, 并且稍为

加班以预备放假回来后的一些工作交接吗?

甚至乎现在你要写一篇文章, 你会很简单的随便写下四百字, 加一个例子便结尾了, 也不做任何的检查核对错别字便上载发 文, 还是先再反覆添加多数个不同的洽当例子做对比, 认真核 对后才上载吗?

有时候, 微少的差别, 长期累积后便可以产生不同的结果。一 切都是源于勇敢打破自我设限开始。
