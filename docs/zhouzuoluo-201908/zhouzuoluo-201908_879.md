# 《选择前要谨慎一点

《选择前要谨慎一点，选择后要坚定一点》

有人说“选择大于努力”，我认为对一半。得分是选择去还是 选择后。

1、选择了下降趋势的行业，努力事倍功半。 我在 2010 年选择了开实体服装店，特别努力拼搏，刚开始几

年还能说努力就有收获，事实上 2015 年之后，实体行业受到

电商冲击越来越明显，加上房租和人工成本增长，举步维

艰。

而我的朋友呢在广州也开实体店，2012 年她开店的那条街， 大多数因为生意萧条而关门，她就此转行做电商。虽然说淘 宝最好做的是 2009 年，她错过了最好的时机，那最好的开始 时间就是马上。

坚持到 2014 年底，她摸索出来经验，一下子爆发了。 做事之前，选择其实是大于努力的。 举个例子，有的姑娘选择了一个滥情的对象，那么再贤惠对

方都有可能“被绿”，选择了一个不思进取的人，那么再劝诫

都可能是“皇帝不急急死太监”。

就像梁宁说的，人的底层操作系统天生注定了大半，后来的 能力只是装在系统上的一个个 APP。你要把安卓系统改成

IOS，那不是太为难自己了吗？

选行业看环境，选伴侣看本性。选择，比你本身的努力重 要。

2、选择之后，则更多要靠自己努力 跟朋友一起做电商的很多，有些老乡比她更聪明，开始得更

早，结果还是一败涂地。为什么？

一个行业再好，也不可能保证所有人都赚钱。总会有些人因 为懒惰、经营不善而落败，甚至有些人早在 2009 年就说，一 边做一边怀疑“哎呀，电商不行了，做不下去了”。

反过来说，实体行业真是不行了，我开店的几年时间里也都 是盈利的，最惨淡的季节也从未亏损。因为一个行业再不 好，总是会有人赚钱。

所以选择了之后，自己的行动最重要，要么你拿出足够的勇 气和实力，再做一次选择——换行业/换伴侣，否则别逼逼。 抱怨和颓丧不仅没半点用，只会让人更快跌入谷底，难以翻 身。

还拿婚姻举例子吧，一个朋友婚后生活不太如意，特别是看 到前男友已经买了宝马，丈夫还在开马自达，她就一直假想 如果跟前男友没分手，现在坐进宝马的就是她了。

对丈夫横挑鼻子竖挑眼，嫌弃他赚钱少，连婚前认可的好脾 气，也被吐槽成“窝窝囊囊”。三天一小吵，一周一大吵。

你要是劝她“实在过不下去，那就离了吧”。

人家说：“好歹他工作还稳定，总是让着我”

你要是说：“既然老公人好，对你也好，那就好好过呗”。 然后又是各种吐槽和嫌弃，这就是一个死循环。

柏拉图说，在花园里选花，你永远无法选到最美的一支，在 麦田里选麦穗，你也永远无法选到最大的一支。

在做选择，要谨慎要全面考虑，在选择之后，还是得靠自己 努力啊。
