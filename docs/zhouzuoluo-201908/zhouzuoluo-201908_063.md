# 《所谓效率高，就是

《所谓效率高，就是会列清单》

查理·芒格说：“最优秀的飞行员也会在起飞前按检查清单检查 一遍飞机，没人拿生命开玩笑。我们生活和投资中的决策也 应该有一个检查清单，这能使我们少遇风险。”

飞行员的训练，是依照一个严格的六要素系统进行，其中之 一就是：“必须养成核对‘检查清单’的习惯。”例如，波音公司 就曾有一份协助飞行员驾驭 B-17 轰炸机复杂操作系统，确保 飞行员不会遗漏重要细节的检查清单。

检查清单用得好，就能实现高效工作。 无论何时何地，我特别喜欢建立检查清单，这个过程不但能

减轻大脑的负担，还能防止人为因素（大脑忘记/疏忽）造成

的麻烦。减少大脑使用功率，避免犯错；不仅如此，在写清

单的时候我们能够重新审视任务类别，分清楚“轻重缓急”进

行归类。

检查清单的技巧有哪些？

1\. 设立清晰的检查点

2\. 简明扼要，不宜太长

3\. 用语精炼，准确清晰

4\. 版式整洁，强调重点

5\. 实践检验，时时改进

养成建立自己的清单检查习惯，通过制定日常任务目标来提

高时间和精力的管理能力，而不是仅仅依赖于自己的人脑。
