# 大家好，这是我加入

大家好，这是我加入星球的第 146 次分享，请大家多多指教

《真正让你感觉人生完了的，压根就不是自律》 今天下午看了一下粥老师推送的新文章，里面的这几段讲的

发人深省

对自律的极大宣扬，很容易让我们下意识的认为：自律是通 往成功之路的必要条件。为了成功，甚至有人不惜自虐。

但成功之道哪有那么简单呢？ 有时候我在想，很多人大力宣扬自律，也许并非是 100%认同

它，只是想缓解一下内心的焦虑感。

也许大家也都知道自律，并非灵汤妙药，只是想多看几个自 律的故事，刺激一下停滞不前的灵魂吧。

我看完文章仔细想了一下，真正让我们感觉人生完了的，压 根就不是自律，而是因为我们追求的，其实是过度自律

为什么这样说呢？ 现在问你一个问题，世界上最想控制，又最难控制的人是

谁？

其中一定有一个是你自己。

我们都希望对自己有更强的掌控力，每个人年初列的

wishlist，里面的目标几乎都和自律相关

比如说要坚持锻炼、不吃垃圾食品、要早睡早起、定下计划 就去执行、不要拖延等等

自律，对我们来说，像一粒神奇药丸，似乎可以帮助我们实 现人生绝大多数的目标。

从心理学上来说，自律还有很多名字，比如说自控、自制， 还有咱们平时说的延迟满足感、抑制不必要的情绪、抵抗外 界干扰与诱惑，在困难面前不退缩，这些都是自律的表现。

这些表现，是不是听起来特别棒？ 人们通常会觉得，自律能力越强越好，再多也不为过。 但最新一期《新科学家》的封面文章，却颠覆了我们对自律

的固有认知

适度的自律，能帮助你实现目标，对生活有更多的掌控，但 如果太过自律，反而会损害你的身心健康，比如降低你的幸 福感，让你更容易患上疾病。

《新科学家》这篇文章叫做《放轻松》，从标题就能一眼看 出观点了。

文章的作者大卫·罗伯森是一名科普作家，他在文章中引用了 以色列心理学家利亚德，发表在《心理科学最新指南》上的 研究论文

《心理科学最新指南》，是美国心理科学协会 APS 旗下的期 刊，而 APS 是美国的权威心理学机构

简单来说，APS 是科学化程度最深的心理学家组织，能够发 表在它旗下期刊上的文章，都是经过心理学行业权威认可、 被认为能够推动心理学发展的重要研究。

明白 APS 的权威性，也就明白了今天要讲的这篇文章的分量 了。

一.过度自律会降低幸福感 利亚德认为，过度自律会从三个方面影响你的幸福感。 首先，当你取得成绩时，过度自律会让你错过人生的喜悦时

刻。

纽约州立大学奥尔巴尼分校的学者马克·穆拉文（Mark Muraven）就曾经做过相关研究

他发现，自律水平比较高的人，他们会在无意识中压抑强烈 的情绪，这就形成了，一种“情绪惯性”。

当这类人取得了好成绩时，本来应该欢呼雀跃、特别高兴， 他们的“情绪惯性”却让他们用冷淡取代了喜悦，这会让他们 错过那些本该为自己骄傲自豪的时刻。

其次，过度自律会让你对身边的人不满。 这是怎么回事儿呢？ 文章里说，一个人的自律程度越高，服从度也就越高，这样

的人会更遵守组织的纪律，会更认真严格地完成安排给自己 的工作。

这也就是我们平时说的“靠谱”。 你可能说这不是挺好的么？

问题就在于，人们会习惯这种靠谱，并把这种靠谱当做理所 应当，而忽视了靠谱的人在背后是做出了多少牺牲，克服了 多少困难才完成的任务。

文章里说，这个现象不是特例，而是人类的心理共性。 美国杜克大学就曾经做过一个相关实验，研究者召集来志愿

者来做一个测试

在这个测试中，志愿者会阅读几段描述不同人的文字，志愿 者要根据这些描述，来给这些人安排工作，并且评估他们为 了完成任务，需要付出的努力。

研究者就发现，如果在文字描述中透露出这个人比较自律， 那么志愿者在给这个人安排工作时，就倾向于给他安排更多 的工作，同时，还低估这个人为了完成工作需要付出的努 力。

问题就出在这——高度自律的人，他们不是没有困难，他们 只是在遇到困难时更能坚持，他们只是更善于隐藏抑制那些 负面情绪。

说白了，他们不是不累，他们只是更能“忍”。 而旁人却把他们的隐忍和付出当成理所应当，这当然会让高

度自律的人心生不满。

另外，从长期来看，过度自律会带来悔恨。 你可能就有这样的经历： 上学时，每次放假都立志要在假期好好学习，但假期过去

了，一页书都没翻，光玩了，然后就很后悔假期没有好好学

习。

那你想不想知道，那些高度自律，在假期一直都好好学习的 人，他们最后悔的事儿是什么？

研究者曾经做过一项调查，他们采访了美国哥伦比亚大学毕 业多年的校友，让他们回忆他们当学生时期的寒假经历，再 谈谈对当下生活的感受。

你猜怎么着？ 按说哥大出来的学生，都是精英中的精英，绝大多数都会在

职业上有出色的表现。

但结果却是，他们中很少有人对自己在毕业后取得的成绩感 到骄傲，却有很多人后悔自己的假期都在学习，没有出去多 玩玩，多走走。

他们最大的愿望是自己在当学生时，少一点自律，多感受生 活，而不是再自律些，取得更大的进步。

听到这你可能也明白了，适度的自律确实可以帮助实现目 标，但过度的自律，反而会让我们错过一些人生的难得体 验，影响我们和同事家人的关系，甚至是带来悔恨。

过度自律除了会降低一个人的幸福感外，对一个人的健康也 会产生负面影响。

二. 过度自律引发的健康问题 文章引用了《美国国家科学院院刊》曾经发表过的一项研

究，这项研究调查了 700 个美国贫困家庭儿童的人生轨迹。

他们发现，那些小时候就非常自律的孩子，更有可能在成年 取得更好的成就，比如说进入名牌大学。

但这种自律也让他们付出了健康的代价，调查发现，这些孩 子普遍都有高血压和肥胖问题。

这是为什么呢？ 因为他们压力大。这些孩子，本来家庭条件不好，现实生活

已经让他们压力很大了。

而在学习上，他们要取得更好的成绩，为了出人头地，就要 有更强的自控力去克服一个个难题，面对更多的挑战，压力 当然也就更大。

长期处于高强度的压力下，健康当然就会出现问题。 而这只是过度自律对健康的第一个影响，成年后，过度自律

对一个人健康的破坏力会更大。

问你一个问题，你认为，是自律能力高的人容易酗酒，还是 自律能力低的人容易酗酒？

你的第一反应可能是——当然是自律能力低的人啊，因为他 们控制不住自己。

但研究显示，自律能力高的人，却更有可能酗酒。为什么这 样说呢？

自律能力低的人喝多了，身体难受，他们就不喝了。 但自律能力高的人，他们反而能够压抑这种难受，为了和人

拼酒，或者是满足同桌人的期待而继续喝，代价就是伤害自

身健康。

不仅如此，对于工作狂来说，为了实现工作目标，他们的自 律会让他们不计代价地坚持完成工作。

英文里有一个单词专门用来描述过度自律的人，叫 over- controllers.

过度自律的结果，就是工作狂们，为了完成工作，会缺乏锻 炼、饮食不规律、睡眠不足，长期下去，这几项带来了健康 危害，我就不多说了，咱们都知道。

说到这，你也明白了过度自律在健康上，会对人们造成什么 影响了。

那是不是说，我们就不要自律了呢？ 当然不是这样，《新科学家》认为，要发挥自律的优越性，

最重要的是掌握好分寸和尺度。

三.如何调节自律的分寸？

关于如何掌握自律的分寸，《新科学家》在文章里提出了三 个建议

第一个建议就是：重新评估你的目标。 作者说，适度自律当然是好的，但更重要的是，我们要想一

想，在什么时候、什么情况下更需要自律。

想想刚才那个调查，哥伦比亚校友后悔年轻时错过的欢乐时 光，你就会明白，你应该做的，是重新评估你的长期目标， 看看它值不值得你用过度的自律来实现。

文章还提醒说，那些高度自律的人，他们一旦设定了一个目 标，即使这个目标对他们来说已经没什么意义了，他们还是 会坚持不懈的去实现这个目标，这就是典型“把手段当目 标”。

这不仅不会让他们更幸福，还会影响他们的身心健康。 第二个建议是，在工作和生活上，你不仅要管理自己，也要

管理别人对你的期待。

如果你是个高度自律的人，你需要和别人谈论你遇到的困难 和挑战，而不是把所有的压力和不满都压在心底。

最后一个建议是，你要适当地降低对自己的标准，不要对自 己太过苛责。

你应该让别人知道，你也是个普通人，你也会累、会辛苦， 不要让他们把你的付出当做理所应当。

最后我们来总结一下

大家都知道自律，是有好处的，在生活和工作上，很多人也 给自己，设定了条条框框来管理自己。

但看完今天的内容，你就知道，自律再重要，也不能过度， 否则也会伤害到我们自身。

在努力让自己更自律的路上，也许我们也该停下来想一想， 更自律有没有让我们，真的过的更好。

我们应该记住，自我控制只是一种工具，学会什么时候不使 用它，才是更明智的选择。
