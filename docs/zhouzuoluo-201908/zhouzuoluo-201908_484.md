# 《混圈子的方法应该

《混圈子的方法应该是多说多做》

昨天文文姐和我聊了几句，她说我在圈子里太沉默了，我说 少说话多做事不好吗？她说你不说谁知道你做了事情？多说 多做才是对的。

我就在想啊，自己平时在社群里几乎很少说话，因为我觉得 群里的大神好多啊，各个都是运营管理，文案策划，要不就 是创业人士，海归人士，我一个普通大学生，感觉自己弱爆 了。

所以我很少说话，怕说话露怯。 文文姐就一直鼓励我，在很多群里她都会说悦己是粥老师分

销的第二名呢，悦己做事情很认真的，悦己是我喜欢的宝宝

之一。

当我弱弱加了助教老师的时候，她们都说文文姐经常提到 我。听到的时候特别感动，虽然我自己很少说，但是文文姐 一直在带我，她真的帮助我很多。

我觉得她的性格特别好，和我有些互补。特别喜欢她的干 脆，爽朗，所以我也想像她一样，积极主动一点。

可能很多时候别人并没有那么厉害，是我们把自己想的太弱 了。不如就多做事多说话吧。成绩和努力同样值得肯定。
