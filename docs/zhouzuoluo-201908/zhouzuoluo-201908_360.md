# 《这样练习，你的文

老王 : 《这样练习，你的文章也会被加入精华》

365 天，每天精进一点点。这是老王分享的第 11 篇成长干货笔 记。

过去的我，写文章很佛系、想起什么写什么。直到有一天认 识了丹儿，才被一棍子敲醒。

原来，放纵自我 的写作方式，就像一盆不经打理的花儿，没 有目标的四处野蛮生长，越长越糟糕。

写文章的正确方式，应该是有目的的刻意练习。一篇好的文 章，也就是在那么几个点上打了高分。

一、选题

上周写了一篇文章《你为什么会很在意点赞？》 发表当天得 到 18 个赞，后来还被加入精华。

而昨天写了一篇《高效能人士必备的一项能力——联机学 习》 收获却甚微。是第一篇文章的质量高吗？不是的，造成 这种反差的是选题。

一个好的选题，是与读者相关联的。可能是他正在经历的 事，或者经历过的事；也可能是她比较关心的话题。

记得某位高手曾告诉我，一篇自带传播性的文章 是 50%的选 题+30%的标题+20%内容。这样看来，选题对了，你就成功了 一大半。

选题是道，标题和内容是术 。大方向对了，剩余的就是细致 末梢的东西了。

二、标题

昨天的那篇《高效能人士必备的一项能力》 ，这个标题起的 其实挺失败的。因为读者看到这个标题，内心不会起到涟 漪。

再看一下其他星友的标题《竞争，根本不是为了要去赢 谁？》 《你做事情还是三思而后吗？错了，应该是三行而后 思》。

这两个标题，第一是我们所熟悉的事物，第二是它颠覆了我 们的常识，有很明确的站队。

所以呢！新手在起标题的时候 ，要么能引起读者共鸣，要么 颠覆常识。这两种方式比较初级，只要用心的去琢磨一下， 重视市场反馈，效果也是立竿见影的。

三、内容

星球里有一位叫 link 的伙伴， 他的文章我比较关注。因为信息 量大，能讲出很多我不知道的东西。

这很可能是长期输入积累的结果，我们普通人 可能做不到。 怎么办？

真诚。

写你身边的小事情，普通人会遇到的事情。只要 真诚的去

写，也是可以得到读者认可的。

写作无捷径，唯有勤更文。有目的的刻意练习，重视市场反 馈，你的文章也会被加入精华。
