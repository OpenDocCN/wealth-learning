# 《电影［少年派的奇

《电影［少年派的奇幻漂流］教给我的 3 件事》

好好利用恐惧；永远不要绝望；勇于逃离舒适——第 167 天的 星里浣

今天，趁着假期睡了个懒觉，顺便躺在床上看了之前想看却 一直没看的电影《少年派的奇幻漂流》，电影主要讲述了少 年“派”和一只孟加拉虎的奇幻漂流故事。故事非常励志，传 达了很多道理；其中有 3 个道理我的感触最深：

第一，好好利用恐惧 少年“派”遭遇海难，失去了家人，阴差阳错下还被迫要与一

只成年孟加拉虎“同船”漂流求生。但这只老虎恰恰成了他活

下来的重要原因。他说，因为有老虎的存在，他不得不保持

警惕，想办法和老虎周旋，老虎带来的恐惧被他变成了生存

下去的动力。

对于我们来说，学会利用恐惧，将给我们的成长起到催化剂 的作用。恐惧除了是痛点，是边界，还是一种动力。这种动 力甚至比愉悦、成就、梦想这些动力更加强劲。因为，你越 恐惧说明就越想拒绝什么。比如说对穷的恐惧，会促使你拼 命赚钱；对无知的恐惧，会推动你不停学习；对落后的恐 惧，会激励你拼命追赶等等

我们应该把恐惧理解成“想拒绝” ，而不是“想逃离”，要学会 利用恐惧，为拥有“能拒绝的能力”而努力。

第二，永远不要绝望

在和老虎漂流的过程中，少年“派”在救生筏的补给箱里发现 了一本《海上求生指南》。里面有一条准则：永远不要绝 望。在尝试驯服老虎的过程中，在看到船只发射信号弹却没 有获得回应之时，少年“派”都告诉自己：远不要绝望。

这份一直保有的希望之心，可能是为了求生，可能是为了追 寻，可能是为了救赎，不管为了什么，它都成了少年“活 命”的鼓励，不至于让他在风雨无常的大海中丧失活下去的信 念。

你可能曾经对自己，对未来，对这个世界失望过，在希望和 失望的交替下尝到了一丝丝绝望的味道。但我也希望我们可 以永远不要绝望，人一旦绝望，就等同于戳瞎自己的双眼， 看不到世界新的机会和可能。

有个广为人知的故事，说的是一头驴掉入了一个坑，它爬不 上，主人也无法将它拉出来。于是，绝望的主人准备用土活 埋这头驴，以减轻它的痛苦，但驴没有绝望，它将主人用来 埋葬它的泥土抖落在脚下一一踩紧，最终靠着不断堆积的土 堆跃出了这个坑。

你要知道，所有的绝望，都是自觉希望。

第三，勇于逃离舒适 在电影中，少年“派”和老虎历经艰难险阻后意外发现了一

处“世外桃源”，那里没有风浪，资源充足，是一个非常棒的

休息地；对于前路未知的危险来说，目前已知的安全无疑是 令人舒适的。

上岛后的少年，有着劫后余生的满足，还准备在岛上好好“修 整一阵子”。但晚上睡觉的时候，他意外发现了在岛心湖里被 逐渐溶解的鱼，人类的牙齿被包裹在植物中，这些异常的情 况猛然令他惊醒：原来这是一座“食人岛”。如果他没有在树 上睡觉，如果他贪图这里的安逸，可能他就会被这座岛悄无 声息溶解掉。于是第二天一早，少年“派”就果断选择了离开 这个岛，继续向着未知的前方进发。

尽管前路有着不可预测的危险，但也有着无处不在的希望。 最终他终于靠岸获救，老虎虽然离开了他，但他们也各自重 获了新生活。

我们的成长路上，也会有像少年“派”登上的“安全岛”那样的舒 适区。这些舒适区，往往会在已知的安全中，藏着未知的危 险，你要学会及时离开这些令人麻痹的舒适区，千万不要贪 图暂时的安逸让自己退化，最终被舒适区慢慢“溶解”。

电影《燃点》中有一句这样的台词：你对这个时代适应得越 好，就越有可能被这个时代抛弃。
