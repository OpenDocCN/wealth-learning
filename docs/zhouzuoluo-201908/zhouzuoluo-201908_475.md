# 《高小羊：生命的意

《高小羊：生命的意义在于记录，写作亦如此》

一朋友留言问，我每天怎么做到突然产生灵感，并快速写出 一篇文章的？

答案是，生命的意义在于记录，而写作，从来不是源于灵 感，写作也源于记录。

我之前的文章经常会提到记录这个习惯，不论是工作，生 活，还是学习，我都会用不同的笔记本做记录。记录今日计 划，今日会议纪要，老板指令，同事分享的心得等，我都会 一字不落的记下来，有些知识点，流程你当时可能听懂了， 或者记住了且有收获了，但一段时间后，你可能什么都不记 得。

记录的好处在于，很多工作技巧是相通的，有时候你无意中 通过交流获得了来自老板或同事的一些想法后，将这些想法 和自己的想法进行糅合，会得出一个新的想法，你会发现这 还是一个很不错的想法，这对于你的工作会有很大的提升。

生活中，大到诸如办理证件，小到诸如买菜，我都会提前计 划并做好后续跟进。特别是周五晚上，我必须计划好周末要 做的事，这样周末才能有序度过，并且周日晚上必须要进行 复盘，这样你才能对以后的安排进行相应的调整和改善。

写作上，不管是我看过的书，听过的课，读到的文章，以及 和朋友之间的聊天，我不会单纯的把它们当作是看书，听

课，看文章或聊天，我脑子里时刻在转动，这句话表达的意 思是什么？它带给了我什么启发？对我哪方面有帮助？当带 着思考，有方向做这些事时，脑袋里时刻就会蹦出新的想 法。然后将他们记录在自己常用的 APP 中，记录时可以不用 做到有逻辑，可以是一些碎片化的思考，自己能看懂就行， 日后写文章时再进行整理。

写作并没有所谓的技巧可言，就是这种笨功夫，时时刻刻记 录生活，记录一切，这也是生活的意义。

说到记录，我想分享我有一个写日记的习惯，这个习惯持续 了十年，空闲时我最大的乐趣就是翻阅日记，那种无法言语 的喜悦是其他事都给不了的。我能从中看到自己以前的无 知，幼稚，可爱，调皮。穿越时空，这些想法和现在的我链 接起来，也有无知，幼稚，可爱，调皮等，但它们却是不同 心境的东西。

前段时候翻看 16 年日记时，有篇日记标题是“本想以男神为奋 斗目标，结果两年过去了，男神不再是男神，而我却变成了 女神”(女神是来自朋友的戏称)。那篇日记写的是我 14 年开始 健身，把一个曾经为男神级（现实中认识的人）的人当做奋 斗目标，结果 16 年有次无意中碰到男神，但男神当年的容颜 和肌肉都已不再，剩下的是岁月给他的一身脂肪和油腻。

这些都是一些极有意思，对于生活有意义的事情，从中你可 以反思出很多东西。所以我常说，生活在于记录，生命在于 记录。记录美好的事情，不好的事情，让你感动的那一瞬 间，这些东西都是组成了我们的人生，我们的每一天生活。

若干年后，当你回忆起过去的岁月中都经历了什么时，这些 就是你留下的最有价值的事。若无所查询，那你就是一张空 白的人生。
