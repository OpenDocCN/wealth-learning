# 《粥左罗：商业的本

《粥左罗：商业的本质，是让人多花钱，而不是省钱》

小小的思辨一下。 瑞幸，虽然更便宜了，但是它没有让本来喝星巴克的人省钱

喝自己，而是让本来不怎么喝咖啡开始花钱喝咖啡，让本来

喝更便宜的速溶咖啡的人开始喝它的咖啡。

淘宝，虽然更便宜了，但是我们好像没有因为淘宝出现而在 穿衣上省下钱，好像是花得更好了，因为买的更多了，隔三 差五的买，甚至买一堆自己根本不穿的衣服。

双十一，本来是打折省钱的，结果你发现，双十一现在是很 多人花钱最多的，买一堆很多仍在家里不用的东西。

滴滴，比出租车便宜了，但是我们打车更多了。 不止是花钱，还有花时间。 微信，张小龙是最倡导用完即走少玩手机的，可查看一下手

机电量，就微信用的最多，一个要帮我省时间的人，做出来

的东西，占用我们的时间最多。

不从别的角度讲的话，你会发现一个很奇葩的现实： 比如一直号称要帮大家省时间的得到，他把那些爱惜时间的

人的时间都抢走了。而天天刷抖音的人，省时间并不是他的

需求，他竟然根本用不着得到。

所以，所有号称要帮你省时间的人，本质上也是要让你多花 时间，只是他们认定，时间花在他们的产品身上，你的人生 会更成功，哈哈。

线下服务线上化，是为了帮大家省钱、省时间，可以省下来 的时间和钱要干嘛？

还是得花。继续花在别的地方，或者再回到线下花。 商业让人花了更多钱、更多时间，那商业的意义是什么？ 上面的逻辑已然包含了，商业创造更高的交易频率，交易频

率提高的意义是什么？我之前讲过，交易创造财富，更高频

的交易创造更多的财富，所以虽然我们都没存住钱，但我们

享受了更多物质，享受了更多便利，就想所有手机 app 最终没 有帮我们省下时间一样，它们确实让我们的一天时间效率更

高了，我们可以做更多、更丰富的事情。
