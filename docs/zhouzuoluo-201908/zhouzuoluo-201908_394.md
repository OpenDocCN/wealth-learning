# 《真尴尬，我们已经

《真尴尬，我们已经到了耻于和父母要钱的年纪》

01

已经记不得是第几次在心底里说出这样的话：真尴尬，已经 到了耻于和父母要钱的年纪。

这样的感受还要源于我的“苏大强”版爹最近给我作妖。 这不是快夏天了嘛。我爹让我给他买衣服。起先是想要运动

短裤，他平时总打网球，眼光一直很高，除了阿迪耐克都不

看，看了没有满意的，就要看菲乐，安德玛。

我把 ipad 给他，让他自己看，相中的加入购物车，就去忙自己 的了。结果晚上的时候，我爹问我：车里的都买了吗？我一 脸懵逼，啥？？？

他这人性格就这样，以前他说五分钟让我下楼，我要是没在 规定时间内下去，他直接就走了，这样的事情没少发生。没 想到这样的性格还体现在买衣服上，明明你连颜色，尺码都 没告诉我，也没和我说要哪件，我哪能买了？？？

听我这样说了，晚上我爹就召开了家庭会议，三口人一起给 他看短裤，黑的吸热不要，花的随意不要，太紧的不方便运 动，太松的肥大不好看，我的爹，到底想要啥？？？

在我妈的拍板下，终于把他的短裤定下。

结果没两天，他又说需要防晒服，嗯？明明去年给他买了， 结果我爹说不行，不是阿迪耐克的，质量不行，今年要新 的。

他明明有好几个耐克的防晒服，非说是两层的太厚。我的爹 啊，你什么时候这么爱攀比了？

后来我才知道，他的球友 A 儿子又给买了新球鞋，球友 B 又给 买了新球拍，球友 C 又买了新手环，球友 D 又买了新运动裤。

他之所以这么作，想要穿得好一点，有面子。 听到的一瞬间，我突然有点难受。记得很清楚，在我六年级

的时候，看上了一双 600 多的运动鞋，我妈说太贵，但就是没

动弹地方，我爸看着了，拉着我就买了。

一双 600 多的运动鞋，在我那么小的年纪还不懂这是个什么概 念，现在想想真是太不懂事了，但他竟然二话没说就给我买 了。

再想想他的作妖，我也就理解了。一句句我姑娘不懂事，我 就忍忍吧。

02

去年我签约了一个平台，每个月有固定的稿费，虽然不是很 多，但是基本上每个月都不需要家里给生活费了。

考研时暑假的房租是自己付的，资料费报名费是自己出的， 复试时车费住宿费都是自己的钱，我不知道这算不算是独 立，但是那阵子我真的挺开心的。

今年由于不可控因素我解约了，这阵子一直没什么收入。我 自己倒也佛系，在家实习不怎么花钱。但是我爹这几天说买 衣服，我看了看自己的银行卡，买运动裤可以，买防晒服也 可以，但是一起买，得需要财政支持一下，说出这句话的时 候，我还挺心虚的。

在他们的印象中，我大概是一个独立的孩子了吧，但是抱 歉，还没有。

上班的时候，和我舍友说起这个事情，她也说：本来想着带 小饭桌挣点钱，回去给爸妈发红包，没想到钱到手，买点东 西就差不多没了。昨天还被解雇了，给妈妈打电话说失业 了，妈妈还说没事会养我。可是哪里还好意思呢？

好尴尬 ，我们已经到了耻于和父母要钱的年纪，但是还没有 持续不要钱的本事。

而真正独立是什么时候，我们也不知道。

03

在知乎的热榜上，我时常会看到“大学生怎样实现经济独 立”“大学生有哪些赚钱的方式”“大学里做什么能快速赚钱”“大 学生怎么靠公众号赚钱”的话题，而且关注问题和回答问题的 人都不在少数。

点就去一看，充斥着的都是替课，做写手，运营公众号。我 下意识的想问怎么现在的大学生都这样急于求成？

紧接着我也会想我自己呢？有什么傍身的技能吗？可以做什 么让保证自己的基本生活吗？

不是说现在的大学生急于求成，而是大家都有很强的危机意 识，看到身边有经济独立的人，也会自尊心受挫，想要做些 什么赚快钱，好证明自己已经是大人了。

以前的我并不理解这样的行为，耻于把钱钱钱这样的字眼挂 在嘴边，我觉得自己不能去做他们那样的事情。但是马上毕 业的我，好像能够理解了一点了。

我在父母的庇护下，安稳无忧地长大，但并不是所有人都有 这样的环境，人们想要证明自己，想要做一些事情来丰盈自 己，这不是问题，问题在于我们意识到做些什么，和这个社 会接轨。

我开始放下固有的观念，去学习那些人的拼劲，开始试着做 没做过的事情。考上了研究生，只是为自己争取了三年的时 间试错，还是不可以放弃成长。

不是小孩子了，那就好好为自己开一条路，走下去，自然明 白未来在哪里。
