# 《关于“越描越黑”

《关于“越描越黑”的一点小思考》

我们可能会喜欢完美，却更愿意相信自然——第 147 天的星里 浣

今天在一本书上看到一段有意思也很有道理的话：

“如果你怀疑杰克偷了你的钱，这时候你可能会得到两种说 法。

1\. 杰克是个诚实的人。

2\. 杰克是个特别诚实的人。

如果你听到的是第二种说法，你会更加怀疑杰克。对不加修 饰的形容词或名词，人们倾向于二元对立的理解——要么诚 实，要么不诚实。而添加修饰词，会凸显程度的差异，“特别 诚实”有多诚实呢？”

这算是一个“越描越黑”的例子，从这个例子里我们看到，过 分修饰，会让人觉得模糊，缺乏力度，自然也就没那么有说 服力；同时，我也有了一个新思考：

我们可能会喜欢完美，却更愿意相信自然。 真的是这样吗？真的。 我刚入行新媒体的时候，做的是“内容搬运工”，粉丝群体是

农村县城中老年用户，每人负责十来个号，每个号发八条， 只发视频，就是纯复制内容，再用标题党博流量的方式来运 作。

那时候，我的主管和我说，选封图的时候，尽量用偏生活化 的，比如“美女图”要用那种“农村实拍美女”，不要用那种被精 修过的“写真美女”，因为真实的封面更容易被打开，而那种 很好看但是不真实的封面打开率不会很高，就算被打开了， 可能会因为图文差距太大，会让读者有“被骗”的感觉。

情况确实如此，那些封面比较接地气的更容易被打开和传 播，而那些高大上的封图，数据都不太好，当然这里面也有 公司的原因（内文质量不高）但这个例子也说明了一个现 象，就是人们更愿意相信那些真实自然的东西。

这也有点像明星们的海报图和街拍照，个个都是又美又帅， 我们也许会喜欢，但是通常不会那么相信，所以“素颜”的吸 引力总是居高不下。

那为什么越加修饰，我们就越难以相信呢？ 我想很大一个原因就是因为我们“够不着”吧。其实回到写作

上，也是如此。 @十点读书 的创始人林少说过，好的内容，

应该是那种大多数人踮起脚就能“够到”的内容。那些被华丽

辞藻修饰，被复杂语法加持，被专业术语包裹的文章，通常

让人难以理解，不会产生多少说服力。

不管是人、物、还是文章，过于“完美化”的背后，通常都充 斥着怀疑；我们可能会喜欢完美，却更愿意相信自然。 2019-01-05(23 赞)
