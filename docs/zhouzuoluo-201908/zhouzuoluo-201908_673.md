# 《成长是孤独的，总

《成长是孤独的，总有一段路需要一个人走过》

终于完成了一个推广小活动，mark 一下。 粥老师的写作课写完后，我们就放假过年了，我一直在想怎

么去推广，但我啥也没做。因为感觉没有资源、没有能力、

没经验，甚至从哪里入手都毫无头绪。很希望我老板告诉

我，具体应该怎么做，第一做什么第二做什么……

然而并没有，我们唯一的共识就是，需要做一些推广工作。

在我稍微有一点点头绪的时候，我跟他说搞个活动吧，下午 给你方案。其实，我根本没有真的想清楚，这么说只是为了 逼自己，这话说发出去，我的内心是巨想撤回的。但我还是 忍了两分钟，事情就成了定局。

终于要动手去推进了，我期待老板给资源出主意，没两下我 就死心了，他说“我以前从来没干过这些事”，我更没有 啊……%*&#%&*

只有乱码能表达我的内心感受…… 我对效果毫无把握，感觉到可能存在风险，但我把计划发给

运营朋友看，他说非常完美的方案，good。这个回答并没有 令我彻底打消疑虑，只不过事已至此，我只能继续推进，尽

可能完成目标，避免最坏的结果发生。

这个过程非常的煎熬，奖励方案已经逐步告知主要的 kol，即 使明知有问题也不可能再改，骑虎难下。

其实从最开始计划到 执行，我特别希望有人能告诉我到底应 该怎么做方案，怎么规避风险，怎么保证最后的效果，每一 个具体的节点又应该怎么做…….我询问了很多人，得到的只 言片语里，或者赞同或者质疑，但没有一个有逻辑有思考的 意见。

整个活动过程中，遇到很多琐碎的问题，比如说做的资料二 维码扫描不了，有人投诉分享了课程没有收到佣金，几次三 番要求我向粥老师要红包；有人表面说帮助，其实只是想借 此加上粥老师微信，有人进活动群抢红包、用软件加人，只 想薅羊毛……

每天都有人问你这里该怎么弄，哪件事要怎么处理，但是就 是没有人会告诉我，要怎么办。所有的东西，都得默默的承 受，然后在摸索中前进，根据具体的情况不断地迭代方法。

至此很长的时间，我都有一种依赖的心理，总觉得有一个厉 害的老板会直接给出解决问题的办法。事实上，当你真正一 个人完成一件事时，更多的感觉是——孤独。需要独自去思 考，一个人去想办法，没有轻而易举的答案（即使有，因为 不用承担任何责任也很可能是错的）。

昨晚跟一个朋友聊起，他说也有同感，目前正在做一个大项 目，自己以前的很多经验用起来才发现“然并卵”，一切都只 能靠自己摸索，这条路很孤独，但是若是将来做成了，就真 的很厉害。
