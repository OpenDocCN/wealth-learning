# 大家好，我是名字有

云 :

大家好，我是名字有点普通的云同学 [月亮] 今天是我第 33 次的分享，我要成为更好的自己[玫瑰]

《摆脱无效社交前，你得知道这三点》 最近以高情商著称的黄磊被骂上了热搜，只因他在节目里的

两种表现：

面对流量新星 他爱答不理，借口退席，场面尷尬。

面对知己老友 他兴奋不已，热情待客，把酒言欢。

黄磊的做法，让“摆脱无效社交”成了自媒体人的热议话题。 然而，在摆脱无效社交前，你得知道这三点。

摆脱无效社交前，你得明确什么是有效社交 与其追究如何摆脱无效社交，不妨先反过来问一下，什么才

是有效社交？

很多写“无效社交”的文章都对只强调对案例的渲染，却鲜有 人对“如何判断社交是否有效”做出恰当的诠释。

直到看到《练达：如何成为社交高手》的一观点，我才眼前 一亮 : “一个社交管理的标准 ，或者高兴，或者推进。”

怎么理解？

无论是滋润心灵，唤起愉悦感还是满足感，只要能让你高 兴，这是有效社交。

无论是工作还是生活，是快乐还是烦恼，只要对你有推进作 用，就是有效社交。

那无效社交就很简单了：所有不让你高兴，不能推动你的社 交。

为什么我要强调区分有效与无效社交？ 因为对的认识才能创造正确的思路。

摆脱无效社交前，你得明确背后的代价 这次黄磊热议的背后，总少不了一个男人的身影：汪涵 作为仅有一百好友的社交自由人，汪涵成了摆脱无效社交最

好的案例：

1）微信朋友一旦过百，就开始删除好友

2）工作后也开始删好友，连钱枫、范冰冰、陈坤也不例外

......

然而，无论是黄磊还是汪涵，他们早已功成名就，一个是桃 李天下的表演系老师，一个是国内顶级主持人。

他们拥有足够的地位和能力，去拒绝一切的无效社交。

但平凡的我们呢？

我们有勇气对无效社交说不？ 哪怕成了别人眼中的怪物？

我们有能力拒绝无效社交吗？ 能承担随之而来的后果？

这是一个艰难的回答。更何况，在我看来，作为人脉积累的 重要方式，社交从来不是说放就放，因为无效社交也非一成 不变。

微商就是最好的例子，它是将无效社交向有效社交转变的过 程，与之相仿的是个人品牌。

当我们渴望摆脱无效社交为人生减负，随之减去的，还可能 是未知的机会。

摆脱无效社交前，你要掌握明确的界限感 你可以摆脱自己的无效社交，但你也可能成为别人想摆脱的

无效社交。

前面提到，无效社交不是一成不变，它是可以被转化的。 举个栗子 加了位大咖，如果鸡皮蒜毛的小事都找他，你便成了他的无

效社交。

但如果你聊他感兴趣的事，提供一些他欠缺的东西，你就能

成了有效社交。

从社交的无效到有效，看的是做事的分寸，待人的情商，这 都强调界限感。

什么是界限感？ 唱 K 时，能把握气氛，知道点什么歌 工作时，能分清主次，知道干什么事 聊天时，能分清场合，知道说什么话

有界限感的人，待人有亲疏，做事有分寸，不越界，不张 扬，这才是当代社交的正确算法。
