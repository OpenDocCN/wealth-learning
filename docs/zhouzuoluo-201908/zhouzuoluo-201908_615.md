# 《别人告诉你的不算

《别人告诉你的不算答案》

晚上和学长聊了一会，说到未来的打算，他说：你不要总在 内蒙呆着，多出去见见世面，真的，你会成长很快。

我说：我也挺想去的，但是我是独生子女，还是女孩子，和 你不一样，我爸妈未来需要我。再说我也没有那么大的野 心，不想只身去大城市打拼，太累了。

我觉得自己一直是个小富即安的人，我看得很短，但是很现 实。

首先我是独生子女，那我爸妈未来肯定是需要我来照顾的， 我走太远，他们肯定会比较担心。

其次我是女孩子，可以拼，但是不需要太拼，工作有优势， 也有劣势，这个我要好好权衡一下，尽量做到利大于弊。

最后最重要的一点是我不具备去大城市的能力，这一点我很 清楚。我既不是去北上广深找梦想的，非那里不可。也没有 过硬的能力，肯吃苦的能力。这是我最大的弊端。

心里默默比量着这些，学长又发来几句：你不想离开家也挺 好的，读个研，回家找个工作，不忙的时候读书写作，网上 赚点钱，真挺好的。大城市不是唯一的选择。

确定的一点是外部环境的改变，绝不会改变我向上生长的脚

步，所以别人给的只是路标，答案还是自己说了算。
