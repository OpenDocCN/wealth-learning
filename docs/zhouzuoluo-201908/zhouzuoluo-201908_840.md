# 《给未婚青年的相亲

《给未婚青年的相亲建议：看对方的最低处，展现平常的自 己》

正在热播的《知否》不知道大家看了没？其中有一段女主明 兰说“看一个人就要看他的最低处”，这个观点跟梁宁在课里 说的有异曲同工之处。一个是说看人要看对方的低点，一个 说不要表现自己的“闪光点”。

看一个人的最低处很容易理解。在一段关系开始之处，一般 都会尽力表现出自己的优点，甚至是自己平时根本就不具备 的优点，当关系逐渐深入，激情退却，对方渐渐放松了对自 我的粉饰，你可能会发现居然无法接受对方真实的样子。

当探知了一个人的底线在哪，并且你能够接受他最差的样 子，这就可以考虑交往下去。

但为什么不能在第一次相亲就努力表现自己最好的一面呢？ 一般很多很多人在希望引起对方注意的时候，会表现得非常

用力。会努力拿出自己最好的一面，给对方留个好印象。

比如，一个计算机高手。他可能会在第一次相亲的时候，把 话题引到自己最擅长的领域，然后在自己最擅长的领域里充 分展示自己，希望给对方留下一个深刻的印象。

这样到底对不对呢？第一次见面，到底是应该照着平淡如水 来，还是去玩命地表现自己？

如果是追求短线关系和片面接触，当然应该玩命闪耀。比如 表演，比如做销售。

但如果是长期关系，两个人认识相处的目的是缔结婚姻，成 为长期合作、伴侣关系，那么就应该平淡处理。

为什么？ 因为每个人现在呈现的样子，其实都是自己过往所 有经历的累加。

有句话说：“你的样子里，有你爱过的人，走过的路，看过的 风景，读过的书。”

相亲时，对方看你一眼，喜欢就是喜欢，不喜欢就是不喜 欢。适当的时候，主动结束约会。如果对方不再约你了，就 放弃这个人。

你也许会想：会不会是因为他没有看到你最闪耀的点，所以 没有喜欢上你，这样是不是会错过一些好的缘分？

如果你追求的是长期关系，就不能基于自己最闪耀的点去缔 结关系。因为你所有闪耀的点，都有它没用的时候。

做生意有亏钱的时候；你擅长计算机，但是技术也在不断更 新迭代；你是名记者，但是媒体行业也在不断变革，也不是 所有人脉永远买你的账。

如果对方是因为你闪耀的这些点才喜欢你，那你是不是为了 保持对方对你的兴趣，要不停地闪耀呢？

长期关系里，没有人做得到。 你平平常常拿出来的样子，就 是你真实的样子。这才是你可以稳定提供的样子，才是可预

期的样子。

从这个稳定的点开始，作为一段长期关系的发端，有可能才 是靠谱的。

所以，初次见面不要努力表现自己最好的一面，平平常常的 样子即可；也 不要谈自己最擅长的东西，不要努力营造氛 围，平淡如水即可。相反，在日常相处中，不要仅仅去看对 方“闪光”的那些点，要多去看对方的最低处。
