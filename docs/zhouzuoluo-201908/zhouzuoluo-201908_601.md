# Day124《所谓

《所谓融会贯通，就是形成一套方法论》

武林高手学功夫，练到炉火纯青，心法招式浑然一体；我们 学知识，学到融会贯通，是形成了自己的一套方法论，就跟 掌握了一套绝世武功一般。

有效学习学的是什么？在我看来，正是一套解决问题的方法 论。

专家之所以成为专家，是因为他们不但掌握了基础知识，更 是在实践中形成了一套解决专业问题的方法论。我们要成为 行家，除了学习必要的知识外，还要更进一步，打磨出自己 的一套解决问题的方法论。

那么，如何通过学习形成自己的一套方法论呢？ 首先，以问题为驱动，找到学习动力。 我们学习的目的就是要解决问题，以问题来驱动学习，通过

解决具体问题来测试学习效果，这也是学习的底层动力建

设。

其次，运用系统化思维，关注问题的深层特征。 当我们面对问题的时候，要区分表面特征和深层特征两个层

次，把注意力放在深层次的概念和方法上，而不是表面的、

具体的要素。

被问题的表面细节分散注意力是我们普遍存在的认知障碍， 这也是我们阅读《有效学习》这本书要留心的一件事。这是 我们有效学习的认知建设。

最后，打磨方法论，成为解决问题的高手。 区分一个人是不是高手，就看他有没有形成自己的一套方法

论。面对现实中的各类复杂问题，其实你拆解它、解决它的

思路往往是相通的，学会这一套解决问题的底层方法，你才

更有可能成为解决问题的高手。这是我们有效学习的终极建

设。

世界著名数学家波利亚最早提出了解决问题的系统化方法， 他将解决问题的过程分为四个阶段：理解阶段、规划阶段、 计划实施阶段和回顾阶段。

在理解阶段，你要学会明确和理解问题，用问题引导自己去 思考：“我遇到的问题本质到底是什么？”

在规划阶段，要学会拆分和定位问题，将复杂问题拆解为最 基本的元问题，然后逐一击破；

在计划实施阶段，提出解决方案，并执行和验证方案； 在回顾阶段，反思解决方案，通过总结持续精进。 小结一下：专家之所以厉害，是因为熟练掌握了一套解决问

题的方法论，如果你能做到这一点，那你也就是行家了。
