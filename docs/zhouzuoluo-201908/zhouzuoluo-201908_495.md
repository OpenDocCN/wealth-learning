# 《做自己的发电厂，

《做自己的发电厂，同时也要找到自己的蓄能方式》——第

11 篇

感谢点开我的分享，希望在这儿停留的你有所收获。 记得小时候听过一首儿歌这样唱，我心中有个太阳，我心中

有个月亮，我眼前有一片红花绿草，我听到小鸽子的歌唱…

…

虽然过去这么多年，但直到现在，我有时候还会不自觉的哼 唱起这首歌，因为这首歌给人希望感，好像真的拥有了太阳 和月亮，眼前就真的出现了红花绿草小鸽子的美好景象一 样。这也是我鼓励自己的一种方式。

关键时刻，自己能够自我鼓励一下，也许会看到不一样的蓝 天，获得更好的机会。

高中时期我们学过思想政治课，其中讲到内因和外因，内因 是根本，外因是起到推动作用的，所以说，内因是关键。鸡 蛋，从内打破是生命，从外打破是食物，讲得也是内外因的 作用。

我加入星球很久，都没有写作，但却一直蠢蠢欲动。因为我 每天都关注粥左罗公众号的文章，买他的书，购买写作课， 每天都看到那么多星友分享，然后很多星友点赞，星主定期 还打赏星友好的分享，并整理出精华阅读，每天都在关注， 这些外在的力量多么强大啊，我被感染，最终我开始写了。

在这个过程中，我发现一直以来我都很爱写东西的，只是以 前觉得自己随便写的东西不算文章，作家才能写出好文章， 通过在星球的学习，能够重新唤起我内在的写作动力，这太 让我兴奋了。所以，决定写，持续写，报名参加写作训练 营，学习更专业的写作。这是一个决定，也是我内心自然而 然的事。

不管外在的影响力多么强大，归根结底，起决定性作用的还 在于自身。有一部电影《风雨哈佛路》，讲得就是一个小女 孩成长环境极其恶劣，她完全可能成为坏孩子，但就因为一 个决定，并付出非常人的毅力，最终她走进了哈佛校园，获 得了不一样的人生。这并不是否定外在的力量，内因和外因 是辩证关系，都不可忽略。

每天关注星友的动态，给了我力量，促使我开始写作，但决 定写，还在于我自身，我的内在动力是促使我写作的根本， 但是如果没有加入星球，对粥左罗一系列的默默跟进，也就 无法唤起我的写作动力，这是多么重要的力量啊。

我常常说，要做自己的发电厂，而不是蓄电池，但也要找到 自己的蓄能方式，比如说写作，决定着喜欢写，就是原始动 力，但是也要蓄能，找到激励自己的方式，比如持续关注公 众号，买写作书，听写作课，进知识星球，查看星友的分

享，进入写作训练营，同时自己每天写，每天有星友点赞， 星主打赏，这又给了自己动力。

你，就是自己的发电厂。找好蓄能方式，获得更多动力。 一起向上生长，期待明天更好的你！❤
