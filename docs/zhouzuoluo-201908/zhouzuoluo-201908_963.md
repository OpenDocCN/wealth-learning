# 《如果没有办法，那

《如果没有办法，那就用笨办法先干起来》

如果没有办法，那就用办法先干起来，是我最近几天最大的 感悟。

自认为智商和情商都有限，也不算很笨，但离开舒服区接手 一些完全没处理过的事，会是会焦虑。前几天有位星友说， 看到我在其他星球的提问，觉得我是太着急。

怎么能不着急呢？不知道你有没有这种经历，在工作中遇到 需要你处理的事，发现自己竟然无能为力。强调一下，特别 是遇到那种心大的老板，还笃定你能搞定一切，真心越着急 脑子越混乱。

怎么办呢？我半夜三更问了一位老师，得到了一个答案“如果 你不知道怎么办，那就用最笨的办法”

1、最笨的办法就是先干起来，边做边学。 我做过大半年电商，在行业里算是比较“水”的水平，比起我

那么年入百万的朋友，实在是差远了。但是真要跟学了三四

年电子商务的大学生比起来，那还是他们“弱鸡”。

这不是自夸，负责招聘过几位相关专业的大学生，他们几年 前的教材知识，来应对当下的环境。怎么可能成呢？

况且，我那时做电商投入的资金，全都是自己辛苦赚回来

的，每一个错误都是资金的损失，当然也带来一次深刻的反 思。在这种压力更大环境下，成长和学习的速度自然更快。

只有你真的做起来，你才会知道具体会发生什么问题，然后 努力寻找解决的方法。每一次遇到问题，甚至是犯错误，只 要你不放弃，都会变成自我成长的催化剂。

2、怎么学？不耻下问和阅读相关书籍 找人聊天一直是我比较喜欢的一种学习方法，可以帮你快速

找到入口。比如说，他们会告诉你先从哪里开始，帮你打开

思路，推荐优质的书籍。

阅读相关的书籍，我觉得也是了解一个新领域最快的办法。 虽然不一定完全正确，大概率上，行业内比较优秀的人才能 写书，留下的大多数也是经验和智慧。

如何避免“偏听偏信”？ 我曾听到一个很好的方法，比如你找这个领域观点完全不一

样的两本书，找出他们认知相同的部分（说明这部分大概率

正确，你可以先接受这部分）；找出他们互相矛盾的部分

（说明这部分有争议，你可以仔细思考这部分）。

就好像你从南极走到北极，一下子了解了一件事物的两极， 也就确定自己认知的边界。对这个领域也有了大体的了解， 心里就不慌了。

经过几天时间的阅读，我已经找到了大致的思路，打算继续 这种“笨办法”，做到心里的目标。共勉！
