# 【读曾国藩传的三点

【读曾国藩传的三点感受】

001 要看懂一个人，要看他最初的来路 市面上很多成功学、励志学，大多会渲染他们成功后的辉 煌，成功前的努力，但是不会告诉你他们最初的来路，比如 说马化腾的成功的故事里，很少有人提到，他有一个个早在 1998 年就能开着奔驰给他做账的爸爸，王健林的地产王国 里，很少有人强调，他有一个当过老红军有着丰富军队资源 背景的爸爸。

而曾国藩的故事里，从他的祖父一代开始扒起，在财富积累 上平淡无奇，读书做官的起点上，他的家族真的算是真正的 毫无根基。正是因为这样，所以他的经历更加值得我们普通 人去学习，找到值得复制的点。

002 比起努力，更重要的是坚持 每一个优秀的人，或许聪明或许努力，但是真正让一个人走

向卓越的，却是坚持。书中提到曾国藩从 14 岁开始可靠，21

岁才中秀才，这在同龄人中并不算早，有的人年纪轻轻甚至

中了举人，后面却默默无闻。

论聪明，大量的案例故事佐证曾国藩资质平庸；论努力，我 们无从考究，就像读书时代，无论学习成绩好坏，我们都一 样在教室里从早到晚地听课、读书、做题，似乎都一样的努 力。

唯一不同的是，他的坚持和和韧性。从他能坚持几十年地写 日记，这一点心智没有坚韧性的心性就做不到。我认为，这 一点正是他后来的成就能普通人，非常宝贵的品质。

003 一个人的挫商，决定他最后的高度 曾国藩人生故事里很有意思的一点，就是他四十几岁时自杀 过两次。

那个时候他已经担任朝廷要职，功名成就已经达到了祖辈无 法企及的高度，足以光耀门楣。

为什么要自杀，而且是两次？可想而知，当时的兵败对他的 打击是毁灭性的，然而他没有继续一蹶不振，而是迅速从泥 坑里爬起来，继续往前蹦跑。

拉回到现实，我们可能会因为一次受伤的恋爱，不敢再相信 爱了；因为一次应聘的打击，不敢再尝试新工作了；因为一 次投资的失败，不敢再挑战创业了……

没有人会喜欢逆境，但世人不如意十乃八九，有时候你可能 已经足够小心翼翼，还是一头扎进坑里去。怎么办？呼天喊 地抱怨时运不济，赖在泥坑里不起来？

不！读曾国藩的传记，给我最大的心灵力量是，败了就败 了、痛了就痛了，擦干眼泪，站起来，我们再干一次，再一 次。
