# 《年终反思 | 此

《年终反思 | 此生不逞强，余生不将就》

新年快乐[玫瑰]今天是橙橙加入星球的 145 天，和大家分享关 于‘反思’的话题」

回首我的 2018 年，有太多让自己都震撼的事件：

①第一次创业，把瑜伽做成了事业，和几个姐妹一起做喜欢 的事情，有共同经历风雨分享喜悦也有闹得不可开交的时 候。

②跟随师父 2 次在南通 120 小时的深度学习阴阳瑜伽，更加清 楚瑜伽到底是什么。

③上天赐予了我二胎，说不定是我曾经许下的女儿，充满期 待。

④加入星球和一群人向上生长坚持日更。

这几件事让我的整个 2018 年充实而忙碌，但这一年，我最大 的收获是我开始思考了！原先我是属于傻白甜的，别人说什 么就什么，师父老说我没心没肺，现在我学会了思考，学会 了反思别人说的话和做的事，也会定期的对自己反思。

2018 年已经成为过去，但这些过去将会为我的 2019 年做基 石，而 2019 年我只需要做一件事：沉淀自己！

1.  想要做成一件事，只要把简单的事情重复做好就离成功越来 越近。

    我记得我的师父曾经跟我说过，瑜伽的晋升不是去挑战越来 越高难的体式，那些炫酷的体式虽然可以在公众场合让你赢 得掌声和欢呼，但那可能需要你付出更大的代价，比如身体 某些部位超出负荷，那些美美的高难体式不是你瑜伽路上最 终的追求。

    练习瑜伽无非是为了身心健康，而往往越简单、越柔和的体 式学具有疗愈性（比如盘坐冥想）只要我们重复的练习，深 入感知身体，倾听身体，体验练习给身体带来的变化，你会 发现，瑜伽让你心平静和，这或许是我们所追求的瑜伽。

    ★2019 年，我会一次次的提醒自己，安驻在当下的体式练习 中，用心感受瑜伽带给我的安详，慢慢的把心沉淀下来。

    写作也是一样的，需要每天坚持练习，重复的练习，要相信 时间积累的力量是很强大的。

2.  不要被工作的忙碌遮蔽了双眼，你身边还有家人孩子和你想 要的生活。

从 2018 年的 3 月到 11 月，我都是忙得团团转，我觉得吧，既然

要做事业就是有做事业的样子，就应该是如此忙碌的状态， 于是我开始忽略对孩子的照顾对父母的看望对老师的关心， 甚至对自己的学习也停滞不前，直到 11 月份怀孕了，我看清 了自己的工作和生活状态，医生说我必须停止做体式保胎， 老公很担心我。

我在事业和身体的抉择间最纠结的时候，师父告诉我：幸福 是需要取舍的，你不能太贪心，遵从自己的内心，做回最原 本的样子。面对忙碌的工作了吃不消的身体，我放弃了全 职，把手头的工作交接了其他人，只是在家养胎、学习、兼 顾一点点工作，我觉得这样一个状态是我可以把控的，也是 最适合我的状态。

★人忙着忙着有的时候就会迷失方向，忘记了最开始自己想 要的是什么，会被忙碌推向其他的地方，到最后精疲力尽， 得不偿失。2019 年，再也别忙丢了自己！

03

做一个有价值的人不是挣多少钱而是让自己的时间有价值。

我们总想在自己的工作或者事业上挣多少钱拥有什么权力和 待在什么位置上来证明自己的价值，对于普通人来说，这个 价值衡量的方向就错了。

之前听罗胖的‘知识就是力量’脱口秀，提到过如何做一个有价 值的人：我们死去的时候不带走存款，不带走房子车子不带 走曾经的辉煌，墓碑上也不会刻下我们存款数目，只会刻下 名字和一段在世的时间，所以我们其实最珍贵的就是时间， 把时间利用好，安排好就是最大的价值。

昨晚的跨年演讲，罗胖的‘五问’当中，其中有两个问题都是和

时间有关“我的时间够用吗？”“时间愿意和我做朋友吗？”

★在 2019 年，我觉得自己最该把握的是时间，普通人的价值 体现往往是在一段长期的时间复利下，积累出来的成就。

▼对 2018 年的反思也是我 2019 年需要不断成长和努力的的方 向，那么在新年的第一天我有几个小小的愿望与大家分享：

1\. 顺利的生下宝宝（此生再也不生娃了）

2\. 趁着孕期将我这些年学的瑜伽知识沉淀下来，做成知识系

统输出。

3\. 坚持在星球日更，加强自己的写作能力！

最后愿家人身体安康，丈夫工作顺利，自己身心愉悦，也祝 福星主粥老师和星球的宝宝越来越牛逼！
