# 大家好我是艾薇，这

艾微 : 大家好我是艾薇，这是我在星球第 12 次分享

《梦想的最初不就是妄想吗？》 在遇见粥老师写作训练营之前我便接触过写作学习的课程，

那时的我偷偷摸摸，生怕别人知道我在学写作。

是的，我的梦想是当作家，而我从未敢跟别人提起过，我怕 别人说“字都不认识几个，还想当作家。”！我是没什么文 化，也没有写作的天赋，难道仅因为这些我就要放弃写作 吗？我就不能把写作归纳为梦想吗？

我不要，不要放弃写作，网上不是有一句话这样说吗？只要 开始，一切都不会晚。

终于有一天我鼓足了勇气踏出了那一步，没文化是吧？那我 给自己每个星期一本书的计划，文笔不好那我就练，从这一 段开始，从碎碎念开始。

难，做什么事不难呢？没学车之前不也觉得好难好危险吗？ 三个月后不是拿到驾照了吗？梦想怕不是难，而是只当它 是“梦想”

电影《月球救援》，男主的一段话让我影响深刻，在那种情 况下，你要不接受命运，要么开始想办法，就是这样，你会 开始计划，你想办法解决一个问题，然后解决下一个问题， 然后再解决下下个问题，等你解决所有问题时你就可以回家 了。

梦想不也是这个步骤吗？从解决第一个问题开始，再解决下 一点个，再解决下下个，等你解决足够多问题时就到梦想之 家了。

进训练营第十天，我从不敢让别人知道我在学写作到主动拿 作业让别人提意见，解决了第一个问题，剩下的我会把所有 的问题都一一解决掉。
