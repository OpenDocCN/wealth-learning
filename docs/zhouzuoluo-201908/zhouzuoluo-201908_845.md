# Day76《学到的

《学到的就要教人，赚到的就要给人》

今天本来想进一步写写第一性原理，但思考了半天，发现还 是难以深入，就暂放置一边了，即使我能勉强写出来，估计 大家看着也会比较累。

今天是休息日，写点轻松点的吧。聊聊我 2019 年践行的几个 原则。

原则一：永远只选择鼓励他人 被重视，被鼓励，被夸奖，被理解，被支持，是你的刚需，

也是别人的刚需。但我们大多数人都倾向于通过把别人比下

去，让自己感觉更舒服，而不是相反。于是我们常常会通过

打压别人、否定别人以证明自己更牛逼。

这个倾向明显是非理性的，也不高级。你想想看，这种做法 除了让你心理上稍微得到一点安慰，对自身成长没有一点帮 助，对人际关系也有害。所以，对一个理性人来说，肯定要 反向操作。

真诚、毫无保留地、甚至丧心病狂地鼓励和夸赞别人吧。前 一阵子，我分享过一个“每天对一个人好”的方法，鼓励就是 对他人好的一个方法，成本低，见效快。

原则二：相信教是最好的学习方法

在这个时代，知识基本上等于免费，我们很容易获得非常多 的知识，比如 199 元你就可以在得到 APP 上买到某个领域顶尖 高手的一整套课。但是，你草草学完了，那些知识就成为你 的了么？

包括我在内，许多人听完学完看完，哪怕记了学习笔记，回 头还是忘，也发现基本用不上。

最近我在研究高效学习的方法，基于任务驱动学习，今天再 补充一条，把学到的东西教给别人，也作为任务来驱动学习 吧。

如果你仅仅是看过一次或者听过一次，你很难给别人讲明 白，你得充分理解和吸收，甚至要仔细备课，写逐字稿，才 有可能顺利教给别人。

带着教的目的去学，你的学习效率至少能上升一个数量级。 不信，你试试？

原则三：做了再说，而不是说了也不做 正是因为移动互联网的存在，我们获取各种信息变得无比便

利，似乎谁都能讲一番道理出来，各种新词、大词、流行词

层出不穷。

谈资易得，实力难求。这个时代，夸夸其谈者尤多。 以前我喜欢给别人提免费建议，很多却没有经过自己的验

证，只是理论上的可能性，给人的感觉是不够接地气。

如今我保持思考，每天写作，尽可能用最接地气的方式去 写，同时在践行中优化自己的内容。这样把见解和心得分享 给需要的人时，也得到了更多的反馈和认可。

我不能保证我百分之百正确，但因我亲自做过，于是能做到 百分之百真诚。如果有幸帮到了别人，那种感觉真的超爽。

“学到的就要教人，赚到的就要给人。”

刘德华在帮助濒临破产的张卫健时对他说了这句话。今天把 这句话也送给大家，2019 年我们共勉。
