# 《从他人身上获取力

《从他人身上获取力量》

今天上午有位男同事过来跟我问了一个事就走了。这不问还 好，一问就问出了大事！小伙子是旁边分部门的同事，属于 同个大部门。

有一团队中间经手的人太多，他不问的话我以为主负责那位 女同事已经处理好，因为前天她告知我“客人提前一天入住， 我这边安排好了，你知道一下”，我回应一声“好”之后没放心 上。

如果不是男同事过来“确认一下”，明天提前入住的客人将不 会有之前承诺给他的待遇，只因为他提前一天入住，所有的 安排都要修改系统。但主负责同事没有修改这个系统，也有 可能是忘记了，放由这件事发展，最后客人不开心或者投诉 是理所当然。

且不说届时会由谁来负责，但我一定是背锅的人，再者令客 人不开心，在服务要求极高的酒店里是天塌下来的大事。我 很感激这位同事，一会儿之后跟他说好在有他提醒，不然明 天麻烦就大了，然后应允要欠他一杯奶茶。

之后我在思考和总结这个事件，归因是自己疏忽大意，太过 于轻信别人会做好自己的职责，即使这位主负责是旁边部门 的一名总监，但还是差点被她误了事。当我把系统改过来之 后并发了邮件再次通知相关部门此活动的变动并抄送这位总 监。两个小时后她看见了，还把我叫到办公室问责：修改为 什么不跟她说。由于了解她的风格和思维，我没有较劲，平

静地跟她说如果客人提前住，就要把之前答应的东西都准备 好在房间里头的，没有理由住进去第二天才去人家房间布置 的。她自知理亏，停了两秒之后说“下次有什么变动麻烦你提 前告诉我，好吗？”我点头，离开，不再这件事上对她有多一 秒钟的纠结。

回到自己的工作台之后，我细想后仍很庆幸有这位男同事的 善意，虽说他只是不经意的“问一句”，却免去了我和女总监 之后要面临的麻烦。在系统看到异常时，也许他可以选择忽 视因为不关他的事，也可能会认为我和女总监应该沟通好了 的，选择不“多管闲事”。但他还是站起身来，走到我身边， 说了这么一句话。

很开心地感受到来自他人的力量，同样的例子，在前两天也 有出现。因为一个沙龙需要公关部帮忙写推文，前提是收集 好所有的推文资料给到他们，本来口头跟公关部说周一会给 到，但是合作伙伴出了一些问题只能周二才给，周二那天按 照原计划补休安排了几件私事紧凑安排，几乎把这件事情忘 了，下午公关部的其中一个同事发来微信“你今天休息 吗？”我给了肯定的回复,随即她说“弱弱提醒你，要给某某

（她的老板）的资料尽快给喔” 。 我心中一惊，才想起要让 部门同事帮我查邮箱里合作伙伴发来的资料转发公关部。调 皮地跟她说“好的，马上！不瞒你说我还真就忘了”，她回复 一个敲打我的表情。

当然，最后还是十分诚挚地跟她道谢，感谢她的提醒和关 心，对我十分重要。

总结上述两件事，虽说不是陌生人，但跟这两位同事平时的 接触不深，能得到他们的帮助时给我带来感动之外，还有结 合今晚一本书《他人的力量》带来的启发。我思考自己平时 与同事间的相处方式，基本处于“融洽，平等，关怀，真

诚”的基础上，当我能够提醒他们一句，关怀一声的时候从不 吝啬，良性的关系是经营出来的，我也会一直传播这种力 量，给他人带来温暖。
