# 从我开始在公众号写

从我开始在公众号写文起，从我开通了赞赏账户开始，快二 十篇了，每一篇都会收到我爸爸的赞赏。每次收到的时候， 我的心情很复杂，感受到，“亲人一定是你最支持的后盾”， 这句话的力量。

但有时候我感觉好像这个赞赏似乎是给我爸专门设的通道一 样，我不想他老人家花这个钱。

但后来我没关，我希望等到有心灵触碰愿意真实想赞赏的读 者，当然，我也等到了好几个哈哈。

我希望我可以越努力越幸运，然后在父母年老的时候有足够 的能力给到他们有份量的支持和照顾。

愿我们踏遍人间，仍然认真生活！ 嘿嘿，今天写了一篇送给高考后的小伙伴的！反响还不错！ [高考后，有些心里话我想对你说](https://mp.weixin.qq.com/s/04Eag8ntnSZEiRLUfw3I2Q)
