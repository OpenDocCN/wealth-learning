# 《有关读书的两种观

《有关读书的两种观点》

今日读书分享两句名言，关于读书的两种观点。 其一是黄庭坚所说：“人不读书，则尘俗生其间，照镜则面目

可憎，对人则语言无味。”

黄庭坚这话有意思，同时也有点狠，意思是人不读书容易俗 气，照镜子都会讨厌自己，与人交谈言语必然乏味。

其实这话不完全对，我有一位小学同学，讲故事绘声绘色， 我们都爱听。他只读完初中，然后早早出去走南闯北，接触 了很多地方不同的文化风俗，每每听他讲这些地方的故事， 我们都特别喜欢。

实践出故事，有些人天生就有讲故事的天赋，即使他不读 书，也不妨碍他的这项本领。但有天赋的人毕竟是极少数， 像我同学那样的人在我生活中认识的人里也是特别稀缺的， 对于大多数人，尤其不读书的人，确实如黄庭坚所批评—— 言语无味。

这些例子生活中比比皆是。 比如同学聚会，与多年不见的老同学往往搭不上几句话。 比如上学时的一些课堂，老师在讲台上滔滔不绝，台下学生

睡倒一大片。

比如许多公司的培训，培训水准之乏味可见一斑。

读书或许不能让每个人，都能成为讲话有趣的人。但是，假 如不读书，而又想讲话有趣，这类人生活中还真是少的可 怜。

其二是朱晦庵所说：“书册埋头何日了，不如抛却去寻春。” 朱熹圣人这句话主要针对沾染书癖的人，也就是说如果读书

沉迷成瘾，势必呆头呆脑，变成书呆子，这样的人在人生的

舞台里可能就没那么好运。可见朱老夫子是比较反对死读书

的。

幸运的是，现代人普遍不爱读书，连书都不读了，所以死读 书的人理所当然也就成了稀缺物种，简直比国家一级保护动 物还少见。

最后总结一下，两位历史名人的话给后人的告诫是，我们要 钻书窟，也还要从书窟钻出来。春节期间大家难免走亲访 友，难免与人交谈，想必我们都希望自己的讲话有趣和有意 思，让更多的人喜欢听，这时就有必要适当读一读书了。
