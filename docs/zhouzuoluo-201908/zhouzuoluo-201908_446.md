# 《为什么别人不喜欢

辉 : 《为什么别人不喜欢听你讲道理》

我们经常会听到这么一句话:“道理我都懂，就是不愿意去做。

”

为什么会这样子？

☆喜欢站在自己角度 深入一点思考，有没有发现，这些道理跟自己或者对方当下 遇到的困难或寻求的答案从根本上就没多大关系。

喜欢讲道理的人，只是把自己的认为所谓行之有效的方法、 理论，简单描述了一遍。

没有真正意义上考虑到当事人真正的问题所在，而是把看似 有效的方法，强加到这个当事人身上，仅此而已。

道理说完了，当事人的核心问题还是没有解决，该从何入 手，还是一脸茫然。

所谓道理千千万，说白了就是一个笼统的概念，对于要解决 的实际问题的关键细节，还真挨不着边。

也许是真有心帮助一个人解决问题，但也只是一些不能解决 实际问题的道理，听的人真是急得火烧眉毛呀!

大人教小孩就多半是喜欢讲道理，“小宝要玩具让给弟弟玩 哦！这才是好哥哥，”结果相信当妈妈的懂，弟弟玩的很开 心，小宝很郁闷!

长期如此小宝也开始叛逆了。

说到这里应该大家明白，一个爱说到道理的人，更多只是站 在自己的角度去处理问题。

可能更多的只是体现在这一领域，我比你懂，有经验，听我 的准没错，潜在意识中还带着一种炫耀的感觉。

☆放下道理着手细节 也并非道理，经验百无一用，前提是这个道理应该可以帮助 当下解决实质性的问题。

需要我们实际的了解当下问题出在哪里，根据当下所存在的 问题进行全面性的了决之后，再给出自己的建议，才能实际 意义上的利用好经验和规律，将问题予以解决。

再简单的说今天遇到一个电脑开不了机，可能是 CPU 不工 作，也可以能，是内存不工作，还有可能是主板出问题了， 还可能是其他问题引起的。

这只是一个机器的故障问题，对于一个人，一件事!可能还会 有更多的因素造成提的出现!

还请你放下道理，拿出正真意义同理之心一起研究存在于当 下的实际问题，从细节着手，这比任何方法都更有效用，相 信也更深得人心。
