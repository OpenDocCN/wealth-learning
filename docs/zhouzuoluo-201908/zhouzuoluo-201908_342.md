# 《普通人要想成功逆

老王 : 《普通人要想成功逆袭，这件事你必须要去做》

对底层出生的普通人来说，有一件事情十分重要。你要不断 拓宽自己的边界，这个边界包括认知边界、视眼边界、人脉 边界、能力边界。

一、认知的边界 什么是认知？

简单讲，就是对事物的认识。同样一件事，认知低的人可能 只会看到一个点，也就是事物的表面现象。

而认知高的人，会从一件事的不同维度看到很多个点，并且 分析出点和点之间的联系。从而挖掘背后隐藏的机会。

要想拓宽认知边界，就要多思考，多和别人交流，多发表自 己的观点，要做一个独立思考者。当然，写作是你提升认知 最低成本、高效率的方式。

二、视眼的边界 视眼，就是你的心有大多，你能看多远。在我刚开始写作的

时候，我就和朋友说，我会最少持续的去写五年。

朋友就很纳闷，明天的事，谁都不知道，你怎么能看的那么 远？

是的，明天会发生什么样的事，谁也不知道。但把时间倒过

来，在你回顾过去的几十年时，就会发现，每一年的变化都 是循序渐进的。

我们每个人都生活在大大小小的系统里，每天发生的小事 情，都是系统中的一个点，每个点和过去、未来都是有因果 关系的。

就像粥老师说的一样:现在的平庸是过去造就的，现在的努力 也会在未来见效。

拓展视眼的方法，就是多读书，多感受身边细微的变化。那 些细微的变化，最终都会积累成一些大事件。

在你经历了从不起眼的事，变成大事件的过程后，你的视眼 边界就拓宽了，你看事情的眼光也会更长远。

三、人脉的边界 得到 CEO 脱不花曾说过:成长就是一个抛弃旧朋友，结交新朋

友的过程。在不同的人生阶段，会认识不一样的人。

在你还是一个菜鸟的时候，你会认识一帮菜鸟朋友；你有一 定能力了，会结实一些能能力的朋友；有一天，你成为高手 了，就会和一帮高手打交道。

随着能力的提升、资源的积累，每个人的社交圈都是在不断 更新的 ，人脉圈的边界也是在不断的被拓宽。

四、能力的边界 能力，是每个人生存的根本。要想提高生活质量，你就必要

要不断拓宽你的能力边界。

就像我自己，在专业技能达到天花板后， 我开始学习写作， 是写作提升了我的认知水平、思考能力。

能力，不仅是指你的专业能力。还有你思考问题的能力，你 的学习能力，接受新鲜事物的能力等等。

前段时间，读完一位日本老奶奶的真实故事后，对我感触非 常大。

若宫雅子，60 岁开始学习计算机，81 岁开始自学 swift，83 岁 成为全球年龄最大的独立开发者，仅半年就成功上线了一款 iOS 应用。

和这位老奶奶相比，现在的我们是多么的惭愧。 你的学习能 力，很大程度决定了你的一生，到底能有多远。
