# 《婚恋市场的经济学

《婚恋市场的经济学，很有意思》

马上春节了，意味着相亲高潮又要来了，提前读点婚恋市场 经济学，兴许对你有用。

朋友相亲 17 次之后跟我吐槽：“哼！现在相亲，都是先看条件 的，现实！”

我没接茬，等她倒完苦水之后问她：“你说现在的人都现实， 那么古代的人，不现实？”

你觉得古代人是不是就比较“淳朴”，或者说不在乎条件，特 别是经济条件？

1、古代婚恋讲“门第”

古代的婚恋市场，颇为讲究“门当户对”，相亲首先看“门第”， 包含一个家族的声望地位、经济实力、文化背景，接下来才 看女子的个人信息，比如生辰八字、性情、容貌。

《红楼梦》中贾府挑媳妇，首先看的就是门第，说白了女孩 子本身反而被忽略，重要的是她娘家的地位和经济实力。

最近火热的电视剧《知否》，也是如此啊，就是年轻人宁愿 爱到死都不放弃，最后却输给了“门第”。

2、“门第”是一个女孩的原始竞争力。

古代人为什么这么看重“门第”？ 很简单，第一，门第好的女孩家，必定陪嫁很多，这笔嫁妆

就是女孩的婚后生活，很重要的经济支撑，有个有钱的老

婆，万一遇到什么难事，就能靠嫁妆扛过去。

第二，门第好的女孩，娘家助力很多，或者在官场或者在经 济上，都能对夫家有一定的帮衬。中国社会自古以来讲人 情，不可否认，裙带关系就是其中很重要的一种。

第三，最关键的一点，古代的女孩自身创造经济价值的机会 少，能力弱，婚后的生活基本上依靠夫家。

所以，两个的性格和感情合不合，没那么重要，男女双方首 先看的是经济实力。婚姻更像是一场交易，尽可能选择强强 联合。

3、现代婚恋市场 现代婚恋市场，讲究的是自由，我觉得应该说越来越倾向于

公平了。

一个女孩家庭不够好，靠自己的努力还是可以改变，所以现 在相亲，在意女孩的娘家的实力，而是在意女孩本身的综合 实力，比如说学历、工作单位、个性、外貌等等。

过去一个女子财力的顶峰了，就是在出嫁时（主要看娘 家），而现在需要看，女孩现在，以及未来潜在的经济能 力。当然需要了解更多所谓的“条件”。

4、建议现在和未来结合着看

一些守旧的长辈，格外在意对方现在的家境，殊不知现代社 会充满了变化和机会，个人财富并不是守恒的，而是是会流 动的，改变的。

当然有好的家庭背景，未来的生活会相对容易。但未来的生 活到底如何，是基于原有经济基础和能力。

家世好赢在当下，有能力的人能够创造未来。要考虑眼前， 也要考量未来。
