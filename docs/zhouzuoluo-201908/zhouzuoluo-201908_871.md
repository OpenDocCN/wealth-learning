# 《你的需求不是罪，

《你的需求不是罪，（）要大声说出来》

会闹的孩子，真的有糖吃——第 162 天的星里浣 我高一那年，特别想要一件以纯的格子衬衣。那时候家里只

有换季或过年才会给你买衣服，而且也只是去街边小铺买那

些“非名牌”衣服，装修精美的专卖店是从来不会考虑的。毕 竟，一个可以“讲价”，另一个不能，而且“名牌衣服”动辄两三 百，可能父母觉得我还远没有这个需求。

直到有一天，我小心翼翼向我妈说出了这个需求，我妈犹豫 了一下，然后答应了。

居然答应了！！！ 后来我妈带我去了以纯的专卖店，买下了我从小到大的第一

件“名牌衣服”，我也为此高兴了好一阵子。我才发现，原来 我的需求也可以被满足，只是之前一直没有提出来。换句话

说，因为我没有说，我的父母就认为我没有这个需求。

今天这个故事，是我看了一个故事想起来的。这个故事是关 于张卫健的，大意是说张卫健跑了十年龙套，一直演路人甲 和匪兵乙，直到有一次他鼓起勇气向公司提出了想演主角的 想法，然后公司就开始注意他，后来给了他一个演主角的机 会，最终，靠着这个机会，张卫健一步步获得了成功。

这也说明了一条古老而有有效的俚语：会闹的孩子有糖吃。

确实是这样的，很多时候你不说，别人就以为你没有需求， 所以你也会失去很多机会。如果长期不说，需求没有被满 足，不仅对一个人的发展不利，甚至还会影响心理健康。

我在向我妈提出想要一件以纯的格子衬衣这个需求之前，心 里想的是：衣服这么贵，我从来没有穿过超过 200 的衣服，爸 妈肯定觉得我没必要穿这么好的衣服。后来，在拎着购物袋 和妈妈一起回家的路上，我妈说：“你要是想买衣服，就要和 我说，但是你要好好读书，知道吗？”

“恩恩。”

惭愧的是，我学会了买衣服就说，却忘记了要好好读书，今 天回想起这件事，心里确实挺不是滋味的。但我把这个故事 分享给你，是想告诉你：你的需求不是罪，要大声说出来。 但是，你一定要好好管理自己的需求，也不要让满足你需求 的人失望。
