# 《学会了这个方法，

《学会了这个方法，你的生活便不会有痛苦》

晚上好 与星友们分享丹儿的第 59 篇原创 6.22

————————————————————————

因参加好友婚礼，回哈尔滨待三天，不知不觉已经过了两 天，爸妈和我抱怨，回来几天都看不到人影，怪我没好好陪 她们。其实我已经推掉了很多好友饭局了，这两天都在帮好 友筹备婚礼，才没时间待在家里，心里也很愧疚。

这两天，我也发现了一些让爸妈苦恼的问题，今天就根据这 些问题，做一个解决方案，明天开始实施，走之前帮助爸妈 解决一些问题。

【爸妈苦恼的问题】 一，老妈的睡眠问题

老爸睡觉打呼噜，老妈睡眠轻，因为家里是开饭店的，所 以，老爸忙完就已经十一点多了，回来后洗漱＋玩手机睡觉 时就已经凌晨一点多了。老妈听到声音睡不着，老爸睡着了 打呼她就更睡不着。

长期下来，老妈的睡眠质量就有了很大的影响。

二，老妈的心情易怒

她和我说的原因是，老爸干活慢性子，干活效率低，她每天 早上都要收拾家里的卫生，下楼又要管饭店的事，觉得太累 了，而老爸只是干好少的活，却没有闲着的时候。帮她分担 不了。

她性子又急，做这么多事，她是不情愿的，觉得不公平，就 很容易发脾气。

三，员工工作效率低的问题 饭店一到饭口会来很多人，顾客着急吃饭，而员工人一多就

开始慌了，尤其是老妈嗓门比较大，一喊就更慌了，不知道

做什么，工作效率不高，上菜慢。

【分析问题根源】 可以看出这几个问题都是围绕着我老妈的，睡眠不好，就会

影响心情和情绪，但这不是最重要的原因。我妈每次发脾气

是因为她骄傲了。

她觉得自己比所有的人都强，所以一看到别人做不好什么事 情，她就会喊，但是喊是解决不了问题的，她一喊，我爸也 慌，员工也慌。（小时候我也慌，现在不慌了理解她为什么 喊了 ，这就是学习心理学的好处。）

别人可能干的还不如以前，别人做不好，她就要自己做，替 别人承担很多。她做的事情一多，身体也累，心也累，又觉 得不公平，所以喊只是一种发泄方式，老妈的内心已经很焦 虑，很烦躁了。

★所以问题的根源就是，老妈的情绪主导她做一些事情，导 致出现这么多的问题。那么怎么解决呢？

【解决方案】 一，和老妈沟通

①沟通时不要说教 因为老妈属于情绪化的人，我和她说怎么做是对的，不能怎

么做，她会觉得我不理解她，在责怪她，她也听不进去。

②沟通时做到反应情感 什么是反应情感，就是理解她，对于她说的话表示理解，不

要反驳，其实对错，她心里都知道，只要让她把心里的话说

出来，每句话都做到反应情感，她就可以自己梳理好。

③引导老妈自己想出解决方案 为什么自己这么累呢？因为别人做不好的事都要我来做。

为什么别人做不好呢？是因为他们笨吗？

有的事她们也能做好呀，尤其是我不在的时候。 为什么我不在的时候，她们就能做好呢？

★就这样引导，她自己就意识到，她的痛苦其实是因为她的 骄傲，她觉得她和别人不一样，这些事只有她能做好，其实 她不在时，我爸的事他自己也能做好，员工也能做好自己的 事。

当她把事情分配出去，就不会这么累了，心情自然也就好 了。

二，与老爸沟通 与老妈沟通已经可以解决 2,3 两个问题，第一个睡眠问题，就

需要与老爸沟通。

我爸和我性格很像，比较容易接受新的事物，很多道理一点 就通，就不用像和我妈沟通一样要引导。

①从饭店回来后声音轻一点，别打扰我妈睡觉。

②关于打呼噜的问题，侧躺就可以解决，我爸喜欢平躺

（《睡眠革命》里提到过，睡觉姿势平躺就会容易打呼）

③自己的事情自己做，不要总依赖我妈，做事要专注别一会 做这个一会干那个。

★和老爸沟通完，就可以解决第一个老妈睡眠的问题啦～

＊＊＊＊＊＊＊ end ＊＊＊＊＊＊＊＊

▼文中老妈痛苦，难受这种情绪我想每个人都遇到过。

生活中的痛苦就是来自于你觉得你不应该承受这种痛苦，你 认为你和别人不一样。

因为你劝别人的时候劝的特别好，这些事情都没什么的，走 过去前面是片天，你觉得没什么大不了，但为什么到你身上 过不去了呢？

可能是比别人很小的痛苦，你都觉得很难受，为什么呢？因 为你骄傲了，你认为你不应该承受这些。你认为你和别人不 一样，所以别人可以承受这些，你不可以。

自我一膨胀痛苦就来了，当一个人能做到发现自己的时候就 不会给自己那么多膨胀的标签。

我们都只是芸芸众生里的一个，不要认为自己是不同的，去 发现自己，虽然这很难，希望有一天，我们所做的事是我们 自己去做，而不是我们的情绪去做。
