# 《关于写作的短期目

《关于写作的短期目标》

粥老师在星球说他跟编辑谈出书的事儿，编辑分享经验：沟

通类书籍好卖，演讲的也行，写作的就差点了。

看起来大部分人对于写作这件事的认识还停留在专业人员或 相关从业者，比如新媒体行业才能做的层面上。

写作这件事是关于专业写手或从业人员来做的吗？ 其实并不是，现在这个时代，人人都是自媒体，人人都可执

笔写作。

大部分人还是对于写作保持观望态度，认为这是一门技术 活。没错，它是一门技术活，但是你可以通过持续的写作， 也能够变成一个有影响力的作家。

那么多人直到写作很好，为什么还不去写呢？其实他们并他 们并没有不写，而是写了几天或者几周或者几个月就放弃 了。

短期内看不到效益，没有收入，没有积极的正反馈，很容易 令人产生畏缩心理。

即使在粥老师的星球里面有奖金加精打赏，还是有很多人有 没有写呀。6000 多人中，有 10%的人写吗。有 1%的人每天都 坚持写作吗？

如果一开始就把写作的目标定为赚钱的话，这个目标就太长

远了。短期内看不到收益，那么也就很有可能放弃。

但是如果我把短期内的目标改为锻炼自己的逻辑能力，提高 自己的语言表达能力。那我就可以去刻意练习了。

其实，我们大多数星友处于写作初期。我认为，写作初期的 目标旨在锻炼自己的逻辑能力。

能够把一篇文章写得逻辑通顺，能让人读得下去，可以算是 及格线了。

如果刚写作就追求赚钱的话，我觉得有点好高骛远。最终目 标可以是它。但是短期目标不可以是它。星友们，你们觉得 呢？
