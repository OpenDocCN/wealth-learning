# 《粥左罗：人微言轻

《粥左罗：人微言轻，你弱没办法，自己变强啥都好说》

最近有个读者问我：左罗，我最近在职场上发生了匪夷所思 的不公平待遇，你说我如果整理一下发表一篇文章，会有作 用么。我觉得你很客观，而且善解人意，希望你能替我支支 招。

我给她的回复是：这个事情你做不好，只有一个标准，你综 合评判一下，如果你这样做了，你是否有 80%的自信能解决 掉，事情会按你预想的路径走，如果可以，你去做，如果不 可以，就别搞了。

因为如果不能，你还去那样做，可能的结果是不仅没解决问 题，还变得更糟糕。

你把这件事通过文章发出来，无非是想借住外部的力量和舆 论解决，但这个世界很残酷，对的不一定赢，错的不一定 输，这个世界有时不在乎事实，而是看谁更有权势谁更有影 响力，即谁更有话语权，谁说了算。

前几天我看到一个小自媒体，洋洋洒洒写了个长篇，证明一 个千万级大号洗他的稿子，我的微信好友基本都是这个圈子 的，我在朋友圈搜了一下，只有 3 个人转发了这篇文章。

这是什么？这是人微言轻。 你影响力小，你行业地位不够高，你后台不够硬，你说出来

的话再有理，可能也不被重视，而圈子里的人，谁不是混口 饭吃，不会随便为一个行业小兵得罪一个行业大佬。

这是挺无耻的，但现实就这样，不是么？ 你弱，再好解决的问题，也会变成大难题。 你强，再难解决的问题，都有人替你出头。 只要在职场里待过几年的，谁没被欺负过，谁没受过委屈，

怎么办？你伸手就能戳到对方有内伤，你就伸手，否则你就

只能咽下去，谁让你弱呢，你弱没本事办了他，你出来闹，

一是看起来像跳梁小丑，二是可能会被更多的报复。

到底怎么办？答案只有一个，忍辱负重，自己变强。自己变 强是解决一切的办法，你真的变强了，有一天那个问题会解 决。
