# 《你还能做得更好一

《你还能做得更好一点吗？哪怕一点点》

我决定从今天开始，每件事问自己 这个问题：你还能做得更 好一点吗？哪怕一点点？

这缘于一件小事的刺激，昨天我的一条分享，被知识星球官 方精选发出来，粥老师看到了把链接发出来，还特地发了一 个红包鼓励我。

一般都是精选星主的帖子，这回竟然轮到我啦。本来我心里 特别开心，点开文章一看，一篇文章里，至少有三个以上错 字和不通顺的地方，结构也相当的随意。

这并不是我在星球里点赞很高的一篇，但发出去多多少少代 表咱们星球，所以我心里感到有点遗憾和难受，默默地想： 唉！早知道我就再写好一点啦，用词再准确些，多斟酌一 下，论述再严谨些…..至少拿出我最高的水准吧！

想到这里，我吓了一跳：难道不应该是我所写的每一个字， 都代表我此刻的水准吗？难道不是每一次写作，都拿出自己 的最高水准吗？

如果不是，那代表我自己还没有尽力。没尽力，是什么破理 由啊？那些跑步比赛差了 0.001 秒，错失冠军的运动员，难道 去跟全世界喊“我只是没尽力，差一点点冠军就是我的了。”

每天差一点，结果就是“差之毫厘，失之千里”。

不知道星友们会不会看官方的精选贴，粥老师的帖子经常被 精选，我诧异他怎么平时这种“小随笔”似的分享，竟然结构 清晰，逻辑完整，语句准确有力。

平时我总是有这种“小侥幸”，觉得自己“还不错啦、差不多 啦”或者说“今天太累了，明天再更好一点啦”。

其实，每一天每件事都应该用“最高水准”来要求自己，只有 这样，才能不断地进步和提高。就像读书时的每一次作业， 都应该认认真真对待，而不是平时马马虎虎，却寄希望于重 要考试时再发挥出最高的水平。
