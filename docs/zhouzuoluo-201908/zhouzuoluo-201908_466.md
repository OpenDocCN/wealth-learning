# Day154《积极

《积极回应，才能赢得人心》

《绝佳体验》的前三大原则都是铺垫，决定体验层级的是你 的回应。今天跟大家聊聊本书的第四大原则：回应。

回应分为积极主动和消极被动，显然只有积极主动，才可能 创造良好以上级别的体验。被动回应，作者直接称之为“反 应”，反应源自冲动，更多是应激性反应。而我们需要通过积 极回应做到两件事：满足需要和解决问题。

那么，具体如何做到以上两点呢？ 受迪士尼公司启发，我们从书中获得了关于回应的十大次级

原则：细节，参与，紧迫感，认知，同理心，过程，适应，

认可，预知和补救。

为了便于理解，我直接用清单体的形式，为大家提炼出要 点。当然这个要点是我比较有感触的点，你也可以去发掘自 己心中的“要点”：

001 保持对细节的专注。这是创造绝佳体验的核心因素。

002 具体如何做呢？我们可以针对绝佳体验所涉及的重要细 节，列出清单，制定具体的行动，然后加以实施就可以了。

003 急对方只所需，快速回应，尽可能别让你的用户等待。

004 有效倾听和真正弄懂用户的需求，而不是靠你自己的主观 想法去理解用户。

005 培养你的同理心，也就是学会站在对方角度思考，让对方 觉得舒服。

006 建立一套清晰的流程和规则来回应用户的需求。其实，运 营就是通过优化每一个流程，从而达成运营目标。或者反过 来说，就是将运营目标拆解成一个个具体的流程，然后进行 优化。

007 除了掌控流程，还要学会适应变化。变化无处不在，所以 要掌握应变能力。不管是适应环境，还是用户。

008 人性的根本原则是渴望被赏识。每个人都希望拥有的感觉 就是：自己是重要的而且是受重视的。被重视，被鼓励，被 夸奖，被理解，被支持，是你的刚需，也是别人的刚需。

009 除了满足已知的需求，还能够预见需求。除了能够解决已 经出现的问题，还要能够预见可能出现的问题。

010 不要怕犯错，要学会适时补救。千万不要隐藏错误、制造 借口或者推卸责任。

以上就是关于回应原则的具体行动要点了，希望对你有所启 发。
