# 《找到人生六匹马，

《找到人生六匹马，找准自己的职业定位》

最近在看《定位》，这是一本理论性很强的书，其中最主要 的一个点，就是定位的最新定义是如何让你在潜在客户的心 智中与众不同。

作者有提到：定位的基本方法，不是去创造某种新的不同的 事物，而是控制心智已经存在的认知，去重组存在的关联证 据。

也就是简化信息，抢夺心智。比如我们现在一提猴子就会想 到孙悟空，一提牛奶就会想到特仑苏，这些信息就是定位准 确，抢夺了我们的心智，让我们潜移默化将两者划为等号。

同时，为什么我们首先想到的是孙悟空，特仑苏，而不是其 他呢？因为他们已经做到了第一，或者是已经包装成了第 一。

成为第一，就是进入心智的捷径。越容易被看到，就越容易 被记住。这也就是定位的优势。

那个人职业怎么才能够定位准确呢？作者提出了人生六匹马 理论。

第一匹马是你所在的公司，你的公司发展方向是什么？或者 它有没有发展的方向？

第二匹马是你的上司，关于你的老板，他有没有前程，如果 没有，谁会有？要永远争取为自己找到最精明，最出色，最 有能耐的老板。

第三匹马是朋友，除了自己私交的朋友，也要在工作上找到 一些能够为你提供帮助的朋友。人生中很多重大转变之所以 发生，就是因为能够有朋友的推荐。

第四匹马是好的想法，好的想法能够比任何东西都更有助于 帮助你提升。好的想法虽然冒险，忍受非议，但是耐心等待 时机成熟，总会有机会。

第五匹马是信心，对自己及别人的想法要有信心，超越自 己，找到外面的财富很重要。

第六匹马是自己，要学会竞争与合作，找到自己的闪光点， 并且去突破。

对职业规划，不如从六匹马开始找起，会豁然开朗许多。
