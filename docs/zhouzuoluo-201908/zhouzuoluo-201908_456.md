# 《收获最多的，永远

《收获最多的，永远是那些懂得主动付出的人》

1、主动付出 在训练营里，发现一枚可爱的小仙女，其实起先我并没有留

意到她，300 多名学员里面藏龙卧虎，优秀的人才很多，小仙 女的履历和能力都不出挑。

但是几天之后，发现大多数学员的评论区留言里都有她，在 微信群也很乐于回答同学的学习疑问，也时常主动提供学习 资料和链接。

在自己的能力还不够突出时，付出了自己的热情，慢慢地大 家很快跟她熟络起来，愿意跟她交朋友，前天我也把一个写 文案赚学费的机会交给她。

2、价值吸引 其实链接资源和人脉除了姿态上的主动，还有一种，那么就

是价值提供。

有个学员每天纠结质疑，学员的水平良莠不齐，这里没有优 秀的人才。

遭到了我的反驳，社区里有学了三十万字要出书的学员、也 有自己把公众号做到十几万粉丝的学员......

对方说自己不知道还有这样优秀的人，但是一个一个去了解 和链接，又太花时间了。

我回她说，所以才要学习写作啊！比如我就很懒得主动链接 别人，但我可以写一些干货分享出来，如果在一个社群中长 期做这样的事，自然会有志同道合的朋友主动找你。

3、有付出才有收获 很多学员进入训练营之前，都说要塑造个人品牌，打造个人

品牌，这些听起来都比较虚。

有句话说，所谓个人品牌，其实就是你离开这个房间别人对 你的印象。

这个印象很大程度产生在，你对别人付出过什么。那些愿意 主动帮忙，愿意提供给别人价值的人，其实往往能给人更好 的印象，链接到更多的资源，最后自己得到更多。
