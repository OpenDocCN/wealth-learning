# 《2019 年的好坏

《2019 年的好坏你无法决定，但你能做好自己》

自强则万强——第 148 天的星里浣 前段时间，看到一篇文章《2019 年，将是决定国家和企业，

以及个人命运的一年》。其实哪一年都是如此，2018 年决定

命运的一年，2020 年依然是决定命运的一年，就像马云在今

年的开年演讲中提及的：2019 年管它好和坏，只做好自己。

时间在流逝，时代在变化，你的命运从来都不是靠哪一年就 能决定的，那些改变命运的人，其实都是改变了自己。

“犹太王大卫在戒指上刻有一句铭文:一切都会过去。”

所以我们会告诉自己，明天又是新的一天，几月对我好一 点，明年一定会更好。

“契诃夫小说中的一个人物在戒指上也刻有一句铭文:一切都不 会过去。”

我们也会发现，第二天的生活还是会重复，下一个月依然生 活不易，也难以看到明年的命运转折。

我以前就特别喜欢说“过阵子就好了”，觉得只要时间长了， 问题就会自然而然地解决，或者期盼时间的背后藏着时来运 转的改变。

上大学的时候，我觉得找工作这事毕业后自然而然就会搞 定，顶多也就是工作差一点，只要做久了，也会慢慢好起 来。但后来毕业后我被现实狠狠地打脸了，根本就不存在什 么“车到山前必有路，船到桥头自然直”，你没有做好准备， 没有自己付出行动，就只能一成不变。

后来我学聪明了，开始对任何事情都更加上心，也学会了主 动和坚持，并且放弃了“一切都会好了”的“时间疗法”，也不再 幻想“机缘巧合”。后来我发现：当自己付出行动后，你就改 变了；当你变得更强后，你就有能力作出更多决定了。

成长的改变，生活的改善，人生的改观，并不会随时间的推 移而变好，时间也无法替你决定命运的走向，还是要靠自己 啊，就算机会来了，也是要伸手去抓的。

在新的一年，你肯定会有新的想法，可能也列出了新的计 划，那么要与之配套的话，也请拿出新的行动吧。

“你不能左右天气，但可以改变心情；你不能改变容貌，但可 以掌握自己；你不能预见明天，但可以珍惜今天。”

改变靠自己，自强则万强。
