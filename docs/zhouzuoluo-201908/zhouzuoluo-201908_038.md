# 大家好，我是李晶晶

动感光 biu : 大家好，我是李晶晶，这是我的第四篇分享《外 行看销售:销售的魔力在哪》。我今天想聊聊销售——让人又 爱又恨的职业。 今天在广场上等朋友，有一个房产销售在屡次被忽视后，走 到我的身边说“这么漂亮的小姐姐应该买一套房子”，因为我 本来没有意愿，我淡淡的回了一句“谢谢，不用”，这是我惯 常的回应。那个销售年龄是一个微胖的中年男人，我感觉他 心态应该有点崩，说话都中气不足了。虽然我内心泛起一点 同情的涟漪吧，不是自身有优越感，因为我觉得做销售真的 要心态好。 和朋友逛街的过程我接到表哥的来电，说想和我合作，当时 那个欣喜若狂啊，满心期待的回去了。然后我表哥跟我说， 让我替他做线下课程推广，我的开心就消散了一半，我哥让 我多拉几个朋友，看我犹疑不决的样子，就和我聊了聊提 成，就好像一摞钱摆在我面前。想起那个销售那愁眉不展的 样子，我说我会考虑的。回家打给朋友，她第一反应是拒 绝，连朋友都无法说服，赚钱的心思又被压回去了一点。但 是我是一个有点轴的人，不想放弃任何一个证明自己的机 会，特别是亲人面前。（因为我表哥是一个自主创业的人， 对我也是一种人脉资源。） 如今自媒体如此盛行的年代，线上营销的用户真的爆炸性增 长。我得到的回答是还没线下推销的速度快，又掐灭我心中 的小火苗。回家在知乎上搜索销售技巧，评论里一片怨声载 道，还有专门给销售“浇冷水”的文章。 销售是一个仁者见仁，智者见智的工作。现实中，销售这个 岗位在各个领域几乎是无处不在，他门槛最低，但是流动性 也非常大。而且销售也有很多种类，比如项目销售、门店零 售等等，我们生活最常见的就是电话销售和门店零售，还有 最直接的扫街。在我们最常见的销售中，有赚得盆满钵满的

销售精英，也有揭不开锅的销售小白。

有人说，销售既是销售产品，也是提供服务。销售是一个与 人打交道的职业，开发顾客和维护关系就很重要，过程中还 会应对一些奇葩的顾客。今年的暑假晚间档《我的真朋 友》，看过的朋友们都加入吐槽大军，感觉自己被强行灌了 鸡汤。不过剧中还是有接地气的存在——郝美丽，她的身上 没有主角光环，外表光鲜，八面玲珑，有很多客户资源，在 维护关系中也有比较尴尬的经历。但在卖房的过程中被客户 性邀约，被美丽给巧妙的化解了。 虽然我不是销售，对销售的认识并不是很全面。不过隔岸观 火，也能看到火光，所以我就先替你们先食用了这份《顶级 销售实战指南》，主讲是插坐学院，分享一些共勉。

对客户来说，销售的专业知识可以帮助客户节省时间和费 用。所以销售要了解产品，清楚客户需求和痛点，提高成交 率。一个好的销售人员还需要掌握商务谈判的技巧（从插座 学员上学习到的） ，产品不错，售后服务不错，回头客和潜 在客户就有啦，总得来说，就是要有产品知识和销售技巧知 识。

这是一些非常实用的销售理论： 1.MNA 法则，客户的定义： 有钱、权、面以及需要的组织或个人。 2.7 秒钟定律：顾客会在看到产品的七秒钟，决定是否购买。 销售人员要精心设计自己的外形、语言和心态，激发顾客的 交流欲望。 3.沉默法则：谈判中先开口谈话的一方往往失去的利益最多， 而后开口说话的那一方可以掌握谈判主动权，并且获取最大 利益。 4.FABE 推销技巧：F 是产品特点；A 是特点形成的优点；B 是 优点给客户的利益；E 是例证，真实的存在。 作为一个销售员，专业知识占 15%,人际交往 25%,人际交往又 可以展开为四点，分别是投其所好、呈现自己的高价值原

则、友善原则和场景知识。这几点从字面意思上非常好理 解，真的要做好还需要我们自发的练习。 最后呢，很欢迎大家能够指正我文章中的不足，积极探讨 哟。
