# 《关于阅读一篇优秀

简三十六 : 《关于阅读一篇优秀博文时，是否应该点赞和赞赏的思考》

【事件描述】

2019 年，小六经历了一些不好的事情，个人的状态比较消沉。为了走 出阴影、找回真实的自我，小六做了许多尝试，比如回顾过去的经 历、阅读思考、反思自我，终于在年中的时候，小六走出了阴影，开 始学着接受自己不好的和好的种种人格。从接受自己内心的那一刻 起，小六变得更有力量、变得更有自信。

然而，过去的经历教训、阅读的知识概念，由于缺少结合自身体悟的 归纳总结，对于小六而言，大多只是了解知识的概念，却没有真正地 将其转化为自身的认知能力。为了加深对知识的理解和掌握，小六决 定在生活之余尝试着写写网络文章，发表小六的生活感悟。

于是，在 0713 日小六写了第一篇文章《从“听到一个道理”，到“本能地 运用这个道理”的方法探索》，分别发布于知乎和知识星球。文章的内 容大部分是对一篇网文的观点总结，只是其间加入了少许个人的人生 体悟。文章发布后，收获了一些志同道合的网友的点赞，小六的内心 也获得了小小的满足。本以为这篇文章的影响也算是告一段落了，但 是另小六倍感意外惊喜的是：在“粥左罗和他的朋友们”知识星球里，粥 老师对小六的文章表示了肯定，并给予了额外的赞赏，令小六产生了 很多的思考。

【关于阅读一篇优秀博文时，是否应该点赞和赞赏的思考】 在收到粥老师的赞赏后，小六本能地想到“难道我的写作能力竟出乎意

料地那么优秀？以至于竟然得到了粥老师的肯定？”，于是小六重新回

顾了自己的文章，然后得出了答案：“粥老师之所以给予小六肯定，大

多原因是对小六认真总结态度的一种肯定、对小六发布文章分享经历

的行为的一种鼓励”。但是，即使如此也令小六产生了许多思考：

思考一：和志同道合的人一起做事的效率，要远远高于一个人单打独

斗的的效率。

过去几年来，小六一直有写日记的习惯，所有的喜怒哀乐都默默地对 自己说，但是由于没有人交流，所有的心事体会往往不够深刻，不能 从中汲取教训，导致一次次地重蹈覆辙，也因此小六经常怀疑写作的 作用，经常性地中断写日记的习惯。

然而，在网络上发布文章却截然不同。首先，你需要发时间对文章的 内容进行修饰和整理，在此过程中不仅能提高你的写作能力，而且能 够令你更深刻地思考心得体会；其次，在文章发布后，你能够收获一 些志同道合的网友的点赞和中肯的评价，通过和网友的交流讨论，更 能够反过来作用于你，令你也获得成长。正因如此，小六更加坚定了 要写作下去的信念。

思考二：如果你从别人那里获得了帮助，那么你应该真诚地表达自己 的感谢。

发布的文章获得点赞和赞赏，让小六意识到，假如从网友那里学到了 知识，拓展了自身的认知，那么，出于网络的礼节，应该向对方表示 感谢，比如点赞和留言感谢；如果你获得了很大的帮助或认知改变， 那么在经济能力允许的范围内，你可以给作者赞赏，来回馈作者的辛 勤付出，表达自身的感谢。你真诚的感谢和中肯的评价，不仅表现出 你个人的人格素养，同时也是对作者的一种积极反馈，以此形成一种 良好的交流闭环。

思考三：要相信积累的魔力，时间会看到你的每一分努力，必定会给 你超乎想象的回报。

在小六的潜意识里，“被赞赏”一直是网络大牛的专利，然而，没想到平 凡如小六这样的路人甲，也有机会有资格获得赞赏。因此，令小六端 正了自己心态，不再只是心怀敬畏地仰望巨人，而是学着用平等的目 光去审视自己、去学习巨人身上的优点，以平等的心态去处理问题。
