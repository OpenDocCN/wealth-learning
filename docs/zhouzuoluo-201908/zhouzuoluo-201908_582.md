# 《刘媛媛|独家演讲

《刘媛媛|独家演讲术：根枝叶果理论，同样受用于写作！》

昨天听了刘媛媛的关于演讲的根枝叶果理论，迫不及待的想 给大家分享，废话不多少，开始吧！

一.演讲表达中最容易犯的四种错误，我觉得写作也同样受 用。

第一个错误，就是在交代背景的时候，说一堆没有用的。 比如要说这样一个主题，关于孝顺，说自己是怎么孝顺母亲

的，大部分都在说我自己公司做的很棒。这些交代跟主线没

有关系，所以会让大家觉得罗嗦。然而真正当你去展开你要

讲的事情的时候，大家又觉得莫名其妙，因为你背景没有交

代清楚。

第二个错误，就是本来应该说的详细的地方，说得特别干。 比如说自己的奋斗过程三句话讲完了，一点细节都没有。 这

个点我记得星里浣，写了一篇文章，写自己的成长过程就写

得非常好，该交代的地方交代的特别好。

第三个错误，就是说不清楚。 比如我的故事跟我的结论之间没有关系，就论点、论据没有

关系，所以别人就会听得很难受，或者云里雾里听不懂，逻

辑上有问题是很可怕的一件事情。

第四种错误，就是讲了一堆没有什么记忆点，听众听完了之 后什么都没记住。

很多时候，我们看一篇文章，标题特别好，把我们吸引进来 了，结果就是看完了不知道作者想表达什么，很乱，没有重 点。所以我们写文的时候就要反复问自己，想传达一个什么 观点？整篇文章有没有说清楚？

这个点我非常有体会，因为我们公司有时候，会组织员工进 行 ppt 分享，这种分享最怕的就是讲睡一大片，很多人都是不 擅长分享，所以，就会出现讲了很多，但是大家啥都没记 住。

二.下面就是重点的根枝叶理论

1.根就是你演讲时候的目标，这根是一定要有的。 第一，你只能有一个目标 演讲只能有一个目的，要传达想法就只传达一个想法，一定

不要贪心。有些人的表达欲旺盛，他讲了一个故事，给出一

个结论，说得非常完整的时候，突然又来了一句话，说：我

还有一个故事想分享给大家。这就是非常错误的策略，演讲

永远只有一个目的。

写文章也一样，为什么一篇文章只能有一个中心思想，凌棱 两可的案例故事不要讲，之前粥老师也说过，有一些案例可 能适应于两个立意，我们尽量不要用这样的例子，会让读者 产生误区，对你这篇文章的中心思想产生质疑。

第二，你这个目的必须是非常清楚的。

清楚到什么程度，清楚到可以写出来。你在跟听众讲话之 前，你先问问自己，你到底想表达什么目的，看看能不能写 下来。 比如我今天演讲的目的，是希望公司的员工意识到公司现在 面临着危机。那你把这个目的写下来，然后最终看看有没有 达到这个目的。 所以这是我们对演讲目的的要求，两个：一个是只能有一个 目的；另一个是清楚到必须可以写下来。

2.枝干：就是框架，逻辑，来佐证你的观点的，观点不好，我 们就会跳出去。造成观众的迷惑，就会出现下面三个问题：

第一，听众会很茫然，他不知道你在讲什么； 第二，他有冲动去打断你去提问，或者他问旁边的人，为什 么讲到这儿了？ 第三，他可能会思考，想刚刚自己到底错过了什么信息，为 什么现在说到这儿了？

当我们看一篇文章的时候很有多文章也是看到一半就退出来 了，这就是枝干不清晰，框架不清晰，乱的很。

3.叶：体验的细节，细节决定演讲成败，一篇好的文章也是有 细节的，比如粥老师写人物的文章，他会把人物发生的一些 事情的细节，描述的很清楚，让人有非常好的体验。

4.最后一个果实是什么？很简单，果实的定义就是听众听完 了，他能记住什么，能带走什么。能记住和能带走的就是果 实。

可以是一个观点，可以是一个金句，一个画面等。。就是让

读者看完了有点收获。
