# 《写在开学前一天》

《写在开学前一天》

不知不觉大四最后一个学期要开学了，刚刚收拾好东西，望 着越来越小的行李，突然很多感慨，去年这个时候在公众号 后台说自己要去考研了，希望大家在我回来的时候还在。

快三年了，经营这个号的三年，我认识了自己团队的小伙 伴，认识了后台的朋友们，认识了一起写作的同龄人。时间 很长，长得我记不清来了多少人，走了多少人，长得我写文 做活动越来越慢，逐渐知道自己最欠缺什么，还可以怎样改 进做到更好。

我几乎每年都会在公众号写文总结，大二的时候我说希望自 己一直活在大二，大三的时候我说希望回头望，把故事从头 讲。大四的总结我迟迟不敢写，仿佛写完了文章就要匆匆结 束大学时光。说说自己的这个假期吧。

01 投身公益，影响他人 这个假期做了很多事，最开心的还是通过自己和小伙伴们的

努力，在四川省凉山彝族自治州普格县夹铁乡，给瓦达洛小

学的小朋友们捐赠了 560 本图书，落地了我们的【阅书悦己团 队公益图书室】，小半个假期我都在做这件事情，刚开始的

时候，根本没有人理会，我自己发朋友圈，发微博，发推 文，发海报，天天介绍，一个个宣传，组建了自己的团队。

一起活动的半个月生气过，烦恼过，但是看到大家和我一样

积极宣传，积极参与，感觉很值，很开心。做一件事情，影 响一些人，帮助一个小学。我只想说，心之向往，行必能 至。

记不得多少人问我，你在干什么啊？天天发朋友圈好烦。但 我记得有谁问过我，怎么参加，怎么捐书，什么书籍符合， 还有人为我们捐款，在网上买书捐过去。对此我满怀感激， 通过他们看到了温暖与希望。

当别人质疑你的时候，不必说什么，做出来就好了，用行动 告诉别人你的坚持是对的。

02 助教学习，输入+思考 在学长的推荐下，我做了一个线上新媒体助教工作，认识了

一群很厉害的新媒体人，大家的效率都超高，而且特别认

真，经常是学员十点交作业，助教们凌晨还在改作业，一起

交流，一起侃大山，不亦乐乎。

这次的实习对我来说是一次很好的锻炼机会，助教先学习课 程内容，自己消化好了再批改学员的作业，自己因为点评太 宽泛，还被当了一次反面教材，但是被指出来问题让我更好 的进步。

同时，我还在怀左学长的写作社群做助教，每天坚持给大家 讲一个干货分享，输出倒逼输入，我报名了粥老师的写作课 程系统学习，跟着学长共读《高效能人士的七个习惯》《深 度工作》《深度思考》，自己学习广告学，学手帐，看高分 电影在微博上分享，这样的生活让我每一天都很充实快乐， 如果还有时间，真想一直这样。

学习的每一个瞬间，我都能感受在心中有一团熊熊烈火，变

得更好就要重建自己。我不怕学习新的东西，只是怕自己原 地踏步，没有成长。

假期和学长交流了很多次，每一次都是醍醐灌顶，特别感 恩，写作的路上，成长的路上，遇到这样一个贵人足矣。

03 反思自己，学会沟通 整个假期经历了很多事情，反思自己太多的不足：不能有效

沟通，不会拒绝别人，不能独立作出判断。

而这些问题都是学长说了我才知道，之前自己完全没有意识 到。不能耐心和长辈们沟通，总喜欢自己一个人呆着；不会 拒绝人，一次次当烂好人，还在背后被说；不知道自己想要 什么，纠结于每一次的选择，犹犹豫豫，没有进步。

我敲下这些字的时候，真的觉得自己弱爆了，太糊涂。 二月有一天突然好多广告商加我，想要投广告，在他们和我

说的时候，我没有直接拒绝，还和他们沟通，也是自尊心在

作祟吧，看到别的朋友们都在接广告，自己无人问津还是很

心酸的。

但是真的有机会的时候，我才发现自己在乎的不是广告有多 少钱，而是害怕喜欢的读者失望，读者在我心中是很温暖的 力量。

我一遍遍和他们解释，好抱歉自己没有能力接下广告，更抱 歉自己耽误了对方的时间。还好对方理解了我，还直说我靠 谱，下次有机会一定要合作，那种被陌生人信赖的感觉让我 倍感幸福。

在成绩出来后，我不知道以怎样的心情来说这件事，停更了 很久。一天深夜我刷知乎，看到很多很多考研人的故事，我 深深自卑，在朋友圈发了一句：害怕别人都在期待我，害怕 让别人失望。好多人点赞评论，我甚至都没有勇气点开看。

那天晚上达令姐和我聊了很多，帮我打开了心结。我不是你 们眼中完美的样子，我自卑，我害怕，我逼着自己走上去， 但是如果没有做到你们期待的样子，是我真的做不到了。

是这个号，是朋友圈让我有了一种错觉：我是一个励志学 姐，我成绩好，我不迷茫，我可以帮助别人。但真的，这些 标签让我很累很累，快找不到自己了，学长说我有“偶像包 袱”，是吧，越来越不像自己了，我在努力做别人心中的悦 己。

今天开始，我要蜕变了，这阵子应该会少更新。因为我要调 剂了，为自己争取一个更好的可能。收集资料，准备复试， 不知道未来在哪里，但是我学着自己做决定，让自己满意， 才是最好的悦己。

只要步履不停，我们总会久别重逢。
