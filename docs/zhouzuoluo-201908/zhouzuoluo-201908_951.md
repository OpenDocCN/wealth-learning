# 大家好，我是阿楠，

林 :

大家好，我是阿楠，这是我加入星球的第 111 天。从 2019 年

起，我决定要每天写一篇文章，无论长短，希望大家监督并

指正，这是我的第 5 篇文章。

《宇宙中，是否还有另一个你？》

今晚在同事的推荐下，去电影院看了《蜘蛛侠：平行宇 宙》。电影的故事很科幻，说的是同样的宇宙有很多个，蜘 蛛侠也有很多个，因为时空错乱，他们会聚在一起合作打 怪。

所以，今天我们来聊一个有点烧脑的话题：宇宙中，是否还 有另一个你？

不要以为这个话题离我们很远，生活中，另一个你可能随时 会出现在你身边。因为有一个呼声很强的科学说法就是，我 们经常会做的梦，其实就是平行宇宙的通道。

梦境中，意识可能来自现实生活中的反应，也有可能是平行 时空下的一面镜子。所以梦境中的场景在现实生活中根本不 存在，人们却会觉得和自身有很大联系，因为那很可能是你 看到了另外一个宇宙的自己。

还有一种情况可能很多人也遇到过，我自己就多次遇到。 即在某个时刻，你突然觉得自己所处的环境、自己的言行和

他人的言行似曾相识或觉得和经历过的完全一样，甚至可

以“预感”到下一时刻某人会说什么话或做出什么动作。然而

这种感觉不会维持很久，在产生这种即视感后过了一会儿或 者就在下一瞬间，眼前的景象就会和记忆中大相径庭，之后 一切归于正常。

有心理学家把这种情况的原因归结为大脑内部的潜意识冲 突，但也有人认为这是两个平行时空在某一瞬间碰撞到一起 了，引起的错乱所致。

读到这儿，是不是觉得背脊凉飕飕的？ 好了，还是正经说一说“平行宇宙”这个概念吧。请注意，“平

行宇宙”这个词是物理学中的重要概念，是一个专门的天文学

术语，并不是只存在于电影里的一个想象的词。

“平行宇宙”概念的提出，得益于现代量子力学的科学发现。 在 20 世纪 50 年代，物理学家在观察量子的时候，发现每次观

察的量子状态都不相同。而由于宇宙空间的所有物质都是由

量子组成，所以这些科学家推测既然每个量子都有不同的状

态，那么宇宙也有可能并不只是一个，而是由多个类似的宇

宙组成。

通俗点说，就是在离我们非常遥远之处，存在一个和我们的 银河系一模一样的星系，而那其中的地球上正有个一模一样 的你。虽然这距离大得超乎人们的想象，却毫不影响你的“分 身”存在的真实性。

还想对“平行宇宙”了解更多的，可以看看相关的片子。还记 得我在传媒大学读硕士时，上的第一堂课，老师便给我们放 了一个关于平行宇宙的外国纪录片，说的就是宇宙中可能有 很多个“你”。

具体片名叫什么忘记了，网上搜了搜，相关题材的纪录片有

《宇宙的构造》《穿越虫洞》等，涉及到无限空间、多重宇

宙、时空穿越等内容，十分烧脑，据说看完会感觉思维都被

颠覆，感兴趣的朋友可以自己看看。

其实，相比于看不见摸不着的平行宇宙，我更愿意聊点实际 的。

宇宙中的另一个你，在现实中如果要找到真实的例子的话， 最显著的莫过于双胞胎了。他们几乎同时出生，拥有几乎相 同的外貌，拥有同样的父母、同样的成长环境……如果说现 实世界中有谁最能体会到另一个我存在的，无疑是那些双胞 胎的当事人。

我一直很想知道，有一个双胞胎兄弟姐妹到底是什么感受， 是不是会像看到另一个自己整天在身边走来走去？特别是看 到那些什么双胞胎兄弟还娶了双胞胎姐妹的新闻，感觉更是 怪怪的，他们就不怕婚后认错么？

为了寻求答案，我特意到知乎上查了查，其实结果倒也还 好。大部分人对自己的双胞胎兄弟姐妹都会觉得和有正常的 兄弟姐妹差不多，只是更加亲密和默契，成长过程中也更容 易引起周边人的关注。

不过，当双胞胎中有一个成绩好、一个成绩差时，容易引起 差的那一个的心理负担问题，也是事实。而那些喜欢上双胞 胎兄弟或姐妹的，会对自己的恋人有特别的感应，也不存在 太大问题。

除了在双胞胎身上能看到另一个自己的影子，在夫妻身上也 能看到。这就涉及到一个很有趣的概念——“夫妻相”。

我们经常看到一些夫妻会在眉目神色间相似，有的夫妻长得 简直就是异性版的另一个自己。这是为什么呢？

《法制日报》的官方微博上，曾发布过一段爆红网络的有 关“夫妻相”的短视频。视频中，生物学家对“夫妻相”进行了解 释：两人接吻十秒，就会交换八千万个菌群，两个人的菌群 趋于一致，就会产生“夫妻相”。看到这个解释，很多网友不 禁欣喜，如果嫁给帅的人，岂不是自己也会逐渐变美了！

当然，对于“夫妻相”的原因，更贴合实际的普遍说法是：夫 妻因为生活在一起，所以在饮食结构、作息规律、生活习惯 等方面基本一致，而且夫妻一般是兴趣相投，表情动作也会 相互模仿，时间久了，夫妻俩的外部生理特征也逐渐趋于一 致，让原本外貌有差异的两个人看起来也有了相似之处，这 便是“夫妻相”的由来。

所以，不用想着下辈子投胎变成男人或女人会怎样了，看看 身边的伴侣，他（她）就是这个世界上异性版的你自己。

最后，回到我们每一个人自身。 在网上，有一个很流行的游戏，就是给未来的自己写一封

信。比如给二十年后的自己写一封信，诸如此类的。不管有

没有平行宇宙中的另一个你，其实每个人自身也是会变的，

今天的你，和二十年后的你，已经不是同一个你了。

这也是为什么每一年新年伊始，都会看到很多人在朋友圈发 今年新立下的要完成的目标，上面写着“今年要多健身”“看完 50 本书”“早睡早起”“多陪伴家人”等等。这是因为每个人都对 现在的自己多多少少有不满意的地方，都想成为一个更好的 自己。

《奇葩说》第五季有个选手叫杨奇函，他说了一句很著名的 话：“我每天都问自己一句，你今天变得更博学了吗？”尽管 这句话被很多人嘲笑，但如果真能做到的话，是很值得钦佩 的。

因为这意味着，每天的你，都比前一天的你更好了一点。日 积月累，用不了多久，一个全新的你就会出现。这时，我们 就不需要再问宇宙中是否还有另一个你了，因为你自己就可 以创造出来。

愿 2019 年，每一个人都能成为心中最理想的另一个自己。
