# 《成长，让你学会报

《成长，让你学会报喜不报忧》

说到过年回家团圆，更像是一场年度总结和汇报，和工作总 结不同的是这场“汇报”以报喜不报忧的形式上演。

大年初一，我去看了《新喜剧之王》理论上翻拍的剧情已经 知道结果会很平静的看完。但是里面每一个镜头都像是年轻 人生活的缩影，其中有两个部分戳中泪点。可能，只有在外 漂泊打拼的人会更有共鸣。

1.受了委屈也要笑着说很顺利 一个是女儿在演戏的时候被大牌演员拳打脚踢后，带着一身

伤坐在化妆间里，边卸妆边和父母通电话。母亲问她过得怎

么样。她刚刚被人欺负完，那一句你过得怎么样一下子刺痛

了她的内心。

瞬间，她的眼眶红了，声音哽咽，但是她尽量让语气显得轻 松，说：“挺好的，大家都特别喜欢我。”

那一刻，我想起了自己每次和父母打电话。 我妈问我过得怎么样，我总是会说挺好的，即使过得再不

好，也会和他们说过得还不错。

到后来，我不在经常打电话，而开始通过微信打字聊天，因 为有时候很怕她们听到我忍不住的哽咽。

不知从什么时候起，渐渐和父母学会了报喜不报忧，因为我 不想他们担心，不想他们因为自己的一点小事情而担惊受 怕。

2.只有自己知道成名之前的付出和磨难 还有一个看哭我的细节是在如梦的父母参加她颁奖仪式的时

候。

台上放着她跑龙套时期的搞笑经历，观众们全都被逗得哈哈 大笑，包括她自己也在笑，而只有她的父亲眼眶通红在咬牙 忍泪。

所有人看到的是你成名之后的璀璨，只有爱你的人看到的是 你获得这份荣誉背后曾遭受的那些磨难。

也许，只有获得成就之后才能笑着回头看曾经的磨难。否 则，回头看仍是咬牙忍泪。
