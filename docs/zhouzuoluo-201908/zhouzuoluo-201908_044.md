# #新认知思考《看了

杨同学 : #新认知思考

《看了抖音粉丝 3000W+的抖主广告视频：好广告是不是都是 这样的？》

刚刷抖音看『多余和毛毛姐』的视频，是一条广告。但点击 率也不低，还有网友留言说，“这是我见过为数不多把广告都 打得这么走心的视频！”这一条留言点赞 20000+。

翻看了毛毛姐之前接过的广告视频，内容精彩度依然不亚于 平时的内容段子视频。

或许很多人跟我一样，一提起广告脑壳疼。 小时候，那时候都是黑白电视机，还不多，经常大伙集中追

剧，好不容易赶上正剧播放，正精彩的时候，“不孕不育哪里

治，快到＊＊医院来…”于是就见看得正入神的大伙，恍神过

来，拍手开骂，然后各种洗发水，肥料，农药的广告汹涌而

来…

有次学习谈到广告时，一老师问，“平时看电视大家都看什么 啊？”

大家回答，当下热播剧，综艺… 然后老师说，其实他现在最爱看的是广告，一到广告他就聚

精会神，比看正剧还认真，因为一条十几秒的广告汇聚了各

种营销传播的智慧，它要在十几秒里传递有用价值还要让你

记住它…等等。

这一听，好像还真是这么个味。可大家还是没心思看广告， 毕竟很多人不做生意学营销啊，就是想看剧看综艺…

而现在新媒体广告也是一波流量带来的趋势。 如何做到打广告达到目的还不掉粉呢，很多新主也是花费了

不少心思。毕竟好不容易积累的一点粉丝，就因为接了一个

广告，全部跑差不多了，那真是得不偿失。

记得粥老师在 谈写作中逆向思考这个话题时说，很多人接广 告，标题取得欲盖弥彰，引得很多粉丝点开看。

而他想来想去，觉得这样做不妥，有种伤害粉丝信用的感 觉。于是他跟其他媒体主打广告不让读者看出这是一条广告 不同，反其道而行之，要让读者看出来这是一条广告。

一看广告标题一目了然，这样让不需要的读者没必要点开节 省时间，让需要的读者尽快链接，避免了透支读者信用。

这样做的结果还有，就是阅读量虽然不高，但转化效果特别 好，商家也满意。基本一次合作，多次合作。

回过头来看“毛毛姐”的广告，她把产品融入了故事段子里， 还是原来精彩的表演风格和故事，广告产品只起到了一个点 缀陪衬的作用。

这不由得让我想起了《歌手》节目里拼多多打的那个广告， 它用了一个开播时的倒计时形式，在节目演播正式来时时， 正中间出现一个声音“拼多多和您一起倒计时”，然后倒计时 数字和拼多多 LOGO 出现在正中间…

就 5 秒左右的时间，而且在观众期待节目开场时，拼多多广告 站在观众的角度，一起同呼吸共期待…毫无违和感，而且很 有情绪推波助澜作用。这种感觉，对于一个追了很多年《歌 手》的我来说，很 nice！

广告总是会层出出穷，而有意思的广告设计师，总会在节 目，观众和自己的目的中找到一个绝佳平衡点，不但不会令 观众反感，而且还会将广告溶于观众关注的画风中呈现，起 到画龙点睛之效，这个分寸点便是智慧。

我想，在生活和工作中，很多看似复杂不能两全的事情，有 时候用心思考，也可以找到两全其美的解决方式。
