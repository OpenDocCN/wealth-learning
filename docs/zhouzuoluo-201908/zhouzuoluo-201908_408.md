# 5.13《机会来临

5.13《机会来临时，要不要抓住呢？》

晚上好 与星友们分享丹儿的第 23 篇原创

今天领导找小 A 谈话，主要内容是想让小 A 去新店当店长，谈 了一下他们对未来的规划，是想提拔我和他们一起带团队， 不要把眼光放在一个店，一个项目上。工资待遇方面也会提 高，看起来是画了很大的饼，关于去与不去新店的问题小 A 从 当下情况和长远发展方面进行了一些思考。

一，分析问题，去新店会比现在好吗？

①新店需要拓客，管理员工，营销方案，汇报工作，制定计 划等处理令人头大的问题。

而老店就不同了，有稳定的顾客，只是当员工也不用操心别 的事情做好自己本分的工作就可以了。

②工资方面 老店底薪虽然只有 2800，但是因为顾客多，提成也很多。

新店底薪会涨很多，但是前期拿提成是很难得，算下来还是 没有老店的工资多，而且当了店长后会多干很多活，压力还 会很大。

③住宿方面 老店离地铁比较近，交通方便，新店的地址有点远。

④人员方面 和老店的同事已经很熟了，都有感情，每天也很开心。

去新店要重新认识同事，而且在和同事沟通方面也需要技 巧，员工和店长很难成为朋友。

二.从长远方向考虑，去新店能带给我什么？ 去新店是一个很好的锻炼机会，以前在店里总是依靠店长，

有依赖性，去了新店什么都要自己做，可以提升自己的能

力。

①提升管理能力 其实开一个店，最难的就是管理，管理好一个团队，能让员

工成长，主动愿意给老板创造高业绩很难。能让员工留下工

作的理由一般都是两个，一个是能让自身成长，一个是能挣

到钱，希望以后我也可以帮助店里的员工得到成长，让她们 靠自己的努力能赚到高工资，就和我们领导一样。

②提升管理能力给我带来的好处 以后就可以不用在把心思放在专研技术上，工作量那么大

哦，当有一天自己不能工作时就会失去赚钱的能力。

而当管理者就不同，可以通过管理手下的人分配工作就可以 把店运营的很好。前期虽然很累，但是后期人员都熟悉内容 后自己会有很多空闲的时间用来提升自己。无论是以后自己 开店还是去别的单位都比只是一个手艺人有优势。

三.长远方向考虑一直待在老店，我会怎么样？

①工资不会有大幅度的提升

因为一个人的工作量在那，只能做那么多活，可能也会通过 销售多赚一些，但是并不稳定。

②长期待在舒适圈，会不想走出来 现在工作的氛围特别好，做喜欢的工作，和同事们相处愉

快，每天很开心，不过长期这样下去，太舒适了，会有些遇

到困难时不敢挑战，可能也会进步，不过不会有太大的提

升。对自己比较放松。

③等年纪大了，就不适合做这份工作了

会有一些职业病，可能会去当美容老师不过那时候又要重新 学习新的东西，也还是靠自己在赚钱。身体上会很累。

＊＊＊＊＊＊＊ end ＊＊＊＊＊＊＊

▼经过小 A 自己的理性分析，从长远方面考虑还是决定去新 店，能够脱离舒适圈得到锻炼，对以后的发展也比较好。虽 然以后可能会遇到无数的困难想要退缩，但是只要我认为它 足够重要，一定可以找到解决困难的方法。
