# 《寻找一个锚点比较

《寻找一个锚点比较靠谱》

晚上好[玫瑰]今天是橙橙加入星球的 152 天，和大家分享关 于‘锚点’的话题」

这段时间段训练写作学会了怎么抓住一个主题去展开阐述， 我记得我以前写作是没有框架和系统的，往往是在文章写完 后才将就着随便弄个主题草草了事。

我觉得这样对自己写的文字很不负责任，这就意味着写到最 后都不知道自己在写些什么，那么读者又怎能看懂，所以我 觉得标题要预先设好，标题往往也是文章的主题，更是我们 写作的一个锚点，它穿透着整篇文章。

1.  01

    锚点就像一个中心轴，始终围绕这个焦点转，不易脱轨。

    锚点这两个字我第一次听是去年参加师父的阴阳瑜伽培训， 每一次的练习我们都需要找一个锚点去练习，比如关注呼 吸、关注肩背的区域或者关注到根基的稳定，当确定一个锚 点的时候从头到尾为的练习都会用这个锚点来引导练习，这 样的练习效果才会更有针对性。

    经常码字的我们是否有肩颈疼痛的症状？根源是因为我们在 码字的时候没有留意到肩旁的区域，在工作的整个过程肩膀

    不自觉的会紧张起来，一个坐姿待久了，长时间的紧张和压 力无法释放机会造成肩颈的疼痛，假如一个小时的瑜伽练习 都以肩膀区域为锚点，同时所有的体式练习也会围绕这个锚 点展开，那么这节课的练习你的肩颈一定会得到有效的修 复，因为无时无刻在提醒自己这块区域要放松下来。

2.  02

锚点也是一个灯塔，它让你非常明确自己的航向。

正如我所经历的，写文章的时候没有先设立标题，你知道后 果会怎样吗？基本都是云里雾里，东南西北分不清，囫囵吞 枣一把抓，整个文章就特别散，你不知道自己在哪里将去何 方。

没有一个明确的标题就不会有明确的小标题，也没有文章的 灵魂了，没有锚点的写作，就像是失去灯塔的航船，找不到 行驶的航向了。

03

那么在写作训练中如何去寻找一个好的锚点呢？

① 学会积累锚点 在写作中锚点往往用来作标题或者代表文章的观点，它往往 出现在你走过的路看过的风景，经历的风雨甚至是一本书或 者一场电影里，那么我们就要细心的观察，将随处可见的锚 点记录下来，累积下来，当哪天不知道写什么的时候去翻翻 你累积的锚点，发现有很多选择。

② 质疑和更新已有锚点

查尔斯·福里尔曾说：“作为一个无需捍卫任何一个派别的人， 我采用的是绝对怀疑的方法，并且将其首先用来研究文明及 其最根深蒂固的偏见。”

我们要像查尔斯·福里尔一样，学会质疑自己累积下来的锚 点，并不是所有的都值得展开研究，我们需要在今后的阅读 和知识增长的基础上不断的质疑和更新我们的锚点，质疑本 是就是一种对思维的训练，更新后的锚点会更加精华具有代 表性。
