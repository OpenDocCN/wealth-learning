# 《如果想长线发展，

《如果想长线发展，就不要想即时满足》

心越小，事越难——第 164 天的星里浣 今天晚饭时，文文姐讲了一个故事: 一个女孩，她很想在新媒体领域有所发展，却一直不太如

意。女孩也不是不努力，她非常努力，各种线上线下课程学

了很多，月薪三四千还舍得花钱参加学费上万元的训练营，

也很有学习的欲望，但入行新媒体三年多了还是发展平平。

问题出在了哪？ 原来，女孩特别热衷于“赚钱”，在工作之余会做很多兼职。

比如那些一两百一条的广告文，甚至几十块稿费的文章，她

乐此不疲。但这不是问题的所在，问题的所在是她耗费了大

量的时间和精力，做的事情却对自己的成长几乎没有任何帮

助。

你想想，一两百甚至几十一篇的文章，它的要求肯定不会很 高，你老是做着低水平的工作重复，不知不觉也就会对自己 放低要求，这对你的成长是有非常大的限制的。

我们常说，一个人能有多大的发展，很大程度是由工作 8 小时 之外的时间决定的。但这句话应该再加一个限制条件:做好本 职工作的前提下。

故事中的女孩学了很多课程，知道很多知识技巧，但她很少 静下心去研究理解，去想着好好磨炼一门专长，去优化自己 的本职工作，而是迫不及待的用“学艺不精”的技能去应付“收 入尚可”的各种杂活。

而她之所以热衷于做那些“要求不那么高”的兼职，是因为可 以获得即时回报。毕竟升职加薪看起来是那么不容易，但写 条一两百的稿子却是手到擒来。所以，她一边想着要长线发 展，一边选择了即时满足。

如果她好好吃透自己所学的课程，打磨好知道的知识技巧， 在自己的工作岗位上做出成绩，升职加薪是必然的，工资涨 个三五千是不难的，远比她一个月花大量精力写上二三十条 低水平稿件的付出划得来。

粥老师说:“你以后千万别做这种傻事，这种一两百一篇的稿 子，写上一百条也只有一两万，你不如好好积攒能力，以后 写两条就能赚到那一两百条的钱。”

如果你的心太小，事就越难做。如果想长线发展，就不要想 即时满足。
