# 《一件趣事给我的启

《一件趣事给我的启示：每个人都有长处，换一种思路找出 路》

前几天我去部队完，听说有了一个有趣的故事。 有位战士在部队当了几年兵，也做了几年的班长，后来复原

回乡，部队补贴了一笔钱。他拿着这笔钱做点小生意，可惜

没啥经验，做这也亏那也不行，最后一无所成。他又不甘心

去打工，最后听人家说养猪赚钱，他想反正在部队也养过野

猪，就跑到一个山脚下租了块场地养野猪。

别以为这就赚了。纯野猪不好养，他花了大价钱买了几只“养 殖野猪”（是一种被驯养过的野猪种），好不容易养大几只大 公猪，差不多可以出栏了，偏巧到了发情期刨烂了围栏跑 了。

这兵哥崩溃了，这可是最后的老本了，欲哭无泪啊。 他没办法，经常上山想抓回大公猪，结果无功而返。到最后

他都放弃了，偶然一天上山发现了很多小野猪崽，这是怎么

回事呢？原来是他养的那些大公猪跑出去跟山里的野猪生了

的，满山跑的小野猪崽。但要抓回来，又不容易。

怎么办呢？ 这位兵哥想出了一个办法，他找亲戚朋友借了几万块钱，又

买了几头野猪种，全是公的，每天吹哨训练。这几头公猪长

大后，一听见哨声就集合到他身边排成队了。

大公猪到了发情期，他把它们全部放上山。过了一段时间， 山上有多了好多的小野猪崽。这时候他跑到山上一吹哨子， 大公猪全跑出来了，背后跟着一群群的小野猪崽，他一路吹 回猪圈，那些大公猪一路跟他着走，小野猪崽一路也跟着。

最后，他把这些猪全部圈起来，小野猪崽作为猪种卖掉，狠 狠赚了一大笔。

这位兵哥在部队待得太久，不熟悉社会上的一些规则和情 况，做生意屡屡亏损。连最后的几头猪都跑掉了，眼看着血 本无归，但是最后他利用了自己的长处——擅长吹哨搞训 练，居然把猪给训练好了，最后成功把满山的小野猪崽一举 俘获。

我想说，每个人都有长处，换一个思路去用，也许就是你新 的出路。
