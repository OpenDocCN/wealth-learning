# 《从樊登身上学到的

《从樊登身上学到的创业知识》

晚上好 与星友们分享丹儿的第 86 篇原创 7.19

————————————————————————

今天和大家聊一下，樊登的低分享创业。打开樊登读书会的 app，发现有一个活动庆祝樊登读书会的会员超过 2000 万。粗 略的算了一下，每个会员 365 元，2000 万人就是七十多亿。

不仅如此，樊登讲书还提到过他现在平均一个月会有两次演 讲，还有软件里在卖的知识课程和书，还有一些日用品都在 盈利。

由此看来樊登的创业是非常成功的。他真的把书中自有黄金 屋变成了现实。不仅自己成功了，还帮助了很多没时间看 书，不爱看书的人培养看书的习惯。

我对樊登的创业为什么这么成功进行了一些思考，和大家探 讨一下。

【樊登的优势】

樊登他之前是中央电视台主持人，他总采访优秀的人，他从 他们身上就学到很多东西。

还有他本身就很优秀，从小就是学霸，他看书比我们看书看 的要深层一些，所谓听着听深，浅者听浅。同样的东西他就 能看出很多深层次的方面。

他讲书时表达也很多逻辑思维，等于把知识嚼碎了吐给我们 的，很容易记忆，而且还会讲很多真实的例子，因为他的真 实精力在那里嘛。

这是别的讲书人比不了的，因为像他这种经历的人很少能有 来讲书的。

【樊登读书会的优势】 对比其他听书软件，樊登读书会不仅有音频，还有视频。 后面还有测试题，这个测试题不是听完做的，而是训练读者

用脑记忆，一周以后用来检测是否记住了的。

以前在别的地方听书，是强迫自己听，觉得对自己有用，很 多时候对方讲的并不好，慢慢就没有兴趣坚持下去了。而樊 登读书会讲的书都很有意思，樊登很注意和读者的互动。听 他的书就是喜欢听，不是为了任务去听。

【樊登读书会的营销策略】 樊登老师选择的是低风险创业，讲书这个事赔是有数的，他

们用来宣传的钱很少，基本用户都是靠口碑推荐。但是它挣

的钱确实无限的。

推荐好友后可以换取积分，这个积分可以再知识超市里买 书，也可以买课程。用这样的方法还让顾客有了良好的体验 感，以后也会付费买课程。

樊登读书会里的书是正版书中最便宜的，樊登以前和读者说 过，他们卖的的书要比任何地方都要低！其实也就低一毛 钱，但是可以通过积分购买，积分你邀请一个人体验听书就 能获得十积分，只要有人体验就有很大机率变会员。

积分兑换是他的一个拓客的方法，并且很有效。有效的原因 是什么呢？是他真的很用心的把每一本书讲好了。

我们出去创业也是一样，别人之所以能成功，本质还是产品

确实很好，这个好还要与市场上同类产品不同，有自己的优 势。最重要的是能给用户带来价值。

我还发现樊登每次搞的活动，表面上是优惠其实都是想让我 们花钱，618 搞的 365 元一年会员，赠送十门课程。看起来真 的很优惠，因为这些课程平时都要 99。

当大多数人知道有知识课程后，樊登开始宣传张展晖的精力 课程了，这时候听过课程的人就有很多花钱买精力课程的， 包括我。

后来又搞抽奖，其实是为了让我们关注微信公众号，因为在 樊登读书会里面发活动很多人会看不到，一关注公众号以后 果然收到了一些直播课程分享的信息。虽然花钱了，但是确 实讲的好，很受益。

当你的产品做的特别好时，顾客是心甘情愿花钱的。

＊＊＊＊＊＊＊＊ end ＊＊＊＊＊＊＊

▼如果你想要创业，首先应该先去研究一下创业的风险，优 秀的企业家从来都不是冒险的人。而是一个更善于控制风险 的人。

反脆弱的创业模式更容易成功。再好的营销方案，也要产品 真的自身有价值才可以。
