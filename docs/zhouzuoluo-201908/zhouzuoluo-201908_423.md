# 《有能力的好人做运

《有能力的好人做运营，用户不付费都难》

运营类更多的是和用户打交道，真正时刻为用户着想的人更 容易长久地做好用户维护工作。

我大学期间做过寒假工代理，就是招募学生到工厂去做流水 线作业员。

招募学生的过程中，我记得比较清晰的一件事情是： 有个学生通过 QQ 来找我咨询寒假工的事情： 我问她：之前有没有做过假期工呢？ 她回复说：有做过一次。 我就问她：为什么还要去做假期工呢？ 她的回答应该是：想赚钱。 我说：我不建议你第二次还是去做假期工，在工厂当流水线

作业员，有一次经历就行，体会到那种环境和那份艰辛就

行。趁着假期可以去做点别的，更能锻炼自己，对自己成长

有帮助，第二次去工厂一点也不划算。

不过那个学生后来应该还是找我报名去做了假期工。 我就是属于那种，从用户价值角度考虑问题的人，首先考虑

的不是对销售额负责。

但是长期来看，这种做事方式是一种复利效应。 我做过 2 次假期工代理，第二次做时候，我自己比较忙。宣传

推广比较少，基本上是之前我带过的学生推荐过来的。

这是一种信用积累，用户自动会帮忙推荐。 做运营就需要这样的思维，优先考虑能给用户提供什么价

值，把自己要得到的回报滞后。

比如发一篇文章，发送之前你根本不知道文章会有多少人看 的，也就没办法知道回报如何。只有保证发出去的内容质量 足够好、能为用户带来价值，才能吸引更多的用户去阅读分 享。

一直为用户考虑，久而久之，一定能号召和感染更多的用户 心甘情愿掏钱买你的服务。

我做运营一方面是兴趣驱动，另一方面以上的用户价值优先 思维也是我的优势（这种价值观挺难修炼的，我是成长环境 塑造出来的）

运营之路漫长，唯用户和爱不可辜负。
