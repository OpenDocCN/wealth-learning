# 《我就是明明知道，

《我就是明明知道，却偏偏做不到》

大家有没有这种情况， 有时候因为朋友的一句，你太胖了， 你就办了健身卡，下定决心减肥，然后过了半个月早已忘记 当初的决心。

大咖都说要养成阅读的好习惯，于是你就买个一大推书，决 定开始看书，看了一星期你觉得看书太痛苦了，你又放弃 了。

我们明明知道这样三分钟热度不好，但就是做不到。下面分 享一下如何才能既知道又做到呢？

1.正面思考，及时跟踪 我们决定做一件事情的时候，千万不能否定自己，一定要养

成正面思考的习惯，要想着自己肯定能完成，给自己信心。

我们还可以借助外部力量监督自己，比如我想早起，就可以 加入早起社群，通过打卡和同伴的监督让自己做到早起。

2.简化目标，聚焦行动 简化目标是抓住重点，保持专注。比如看完一本书，一定要

让自己写下收获，最好把这些收获转化成行动。

聚焦行动就是当下的这个目标是你的重点，是每天需要完成

的任务，然后去执行。

3.建立危机意识，先行动起来 危机意识的强化，一部分来自于自身的经历。比如， 身体受

过伤之后，就知道了健康的重要性，从而会去运动。

先行动起来，我们不需要用计划来指导行动。而是用行动来 带动计划。

4.真正接纳并认可你所做的事 想清楚你做这件事情的初衷，比如我想减肥，是想有好的身

材，能穿漂亮衣服，还能让自己变得自信。可以用这件事情

的好处来激励自己去行动。
