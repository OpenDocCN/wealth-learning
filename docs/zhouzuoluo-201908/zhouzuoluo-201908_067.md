# 大家好，我是小橙姑

大家好，我是小橙姑娘，一热爱写作，渴望未来能活成自己 喜欢样子过上自己想要生活的姑娘。

很开心能有这样的平台给以我们分享我们的成长和感悟。 这是我加入星球的第 15 次分享，期待着小橙的第 100 次分享，

小橙会持续输出的。

《关于加入知识星球之后的一些思考》 最开始加入粥老师的知识星球，也是犹豫了许久，主要是怕

没时间看，因为之前也入了关于朋友圈营销的许多星球，后

面我都放弃不看了，因为后面发现自己并不喜欢那种营销气

息太浓的东西，不管别人怎么评论，而我只想按照自己的意

愿来。

那个时候我是先看粥老师的公众号的，基本上每天都看，偶 尔会用小号评论，因为工作关系大号很少做点赞或者评论操 作。

看了一段时间发现其实自己还蛮喜欢这个作者的，后面就入 手了星球。我是一个在选择适合自己的付费产品比较谨慎的 人，一般都会观察一段时间，再查看边缘的东西，合适我再 买，星球也是一样的。

加入知识星球之后，第一天就发现，粥老师的星球并没有像

其他星球那种有微信群，而且每一个模块都分的非常清晰， 一目了然，因为刚刚开始，信息量有点多，自己也没有花特 别多时间研究，后来是一小伙伴加入了之后，再跟我交流的 时候，我才又慢慢研究了一番。

或许，跟我是一个比较慢热的人有关系吧。那一次居然都看 懂了。而且刚刚好那天看了粥老师发的精华帖里面的星球内 容之后，才发现，原来自己也可以在里面发布自己的感悟和 观点。

最开始，因为对于那种观点性的文字，自己并没有写过，而 且也不知道如何下手，也就没有分享自己的文字。而在看了 精华帖之后，自己才开始慢慢进入状态。也是告诉自己，希 望一周最少要输出 3 篇文章。

最开始的时候，输出的不频繁，都是看别人的文章，点赞别 人的文章，打赏文章居多，记得第一个星期分享文章之后， 发现居然有人点赞，还有人会评论，而且之后还收到粥大大 的打赏，也就更加坚定自己继续分享下去的决心。

而也就是在那个时候开始，在自己文章的开头增加了“期待自 己第 100 篇的分享，我会持续输出的。”

而每个星期开始的时候，在周末定的计划是，这个星期最起 码 3 篇，故而希望自己能在越前面完成越好，而往往在周一周 二的时候，是处于输出不了的情况，因为这两天都没去思考 这件事的，往往是周末的时候才会去思考。

而我现在觉得，写作这件事，其实不是有空了才去写的，既 然热爱写文，那就是要最好每天都能在输入的同时坚持输 出，也应该把写作这件事融入到自己的生活中去，像工作那 样去对待它，做好它，完成它，也只有这样，才能不辜负自

己对写作的喜爱。

所以从这个星期开始，我尝试在调整。尽量不再像上个星期 一般，周日才想着输出两篇文章，这对在星球输出这件事来 说，如果是这种节奏，最后只会导致放弃。

在星球的这段时间，也有关注粥老师的朋友圈以及在星球的 动态，我还是蛮喜欢粥老师的一些思想和观点的，不张扬， 踏实地做好自己的事情，也很喜欢星球里面小伙伴的文字。

也有想过，去参加粥老师的写作训练营，因为写作这件事， 虽然自己断断续续也写了不少文字，但始终是缺乏坚持的， 而这种坚持往往也是需要环境的，但我知道自己现在每周坚 持 3 篇都不太容易，天天写而且还需要要求质量，再加上经常 要加班，那会是很吃力的一件事。

我不是一个喜欢一直活在一个群体的人，在随性地做着自己 喜欢事的同时，也会发现或许这也是没能有大成就的根源所 在吧，而我还是想再尝试一段时间每周的文章能稳定在 3 篇或 者更多的输出之后，再去参加，再去抱团，或许对我来说， 收获会更大一些。

一个人唯有喜欢一个环境，待得舒适才会一直待着并且愿意 一直分享下去，而这里无论是环境和氛围都是我喜欢的。

我不太喜欢微信太浮躁的信息，在这里，人与人之间，说陌 生也熟悉，说熟悉也陌生，我挺喜欢这种关系的，不会太近 也不会太远，但是又彼此信任，在需要帮助的时候，星友第 一时间给以我帮助，其实还蛮感动的。

而我，也有出现很想输出，但不知道写什么观点的时候，我 会看看星友的文字，看看精华帖，再看看微博的一些资讯，

看着看着，就有素材可以写了，或许，这也是一个成长过 程，一个培养自己敏感的写作触觉的过程。

我会继续坚持下去的。期待在输出感悟的时候，遇见那个更 加优秀的自己，写出更多有价值的深度思考文，也希望看到 更多有助于我的文字，也期待遇到更多优秀同频且可爱的小 伙伴。
