# Day80《懂得这

《懂得这个道理，没准你今年赚钱的本事能翻一番》

今天看罗辑思维推出了吴军老师的一篇文章，关于对待金钱 的态度。其中，对金钱本质的定义，我认为非常值得大家思 考，没准搞明白后，你赚钱的本事能很快翻一番。

1、钱的本质你想过吗？ 钱的本质是什么？不知道你有没有思考过这样的问题。表面

来看，它是一种度量工具，但度量的到底是什么呢？

吴军说：“金钱实际上是对各种资源的所有权和使用权的量化 度量。”

什么意思呢？举个例子，你买了一辆车，除了换回制造车的 材料之外，你其实花钱买回了这辆车背后生产线工人及一切 付出劳动的人的时间，是你用钱调动了一部分自然资源，还 有一些人的时间资源。

因此，每个人钱的多少，反映了他能调动社会资源的总量。 在这个意义上，钱只有花出去才是自己的，也就是完成了对 资源的有效调动。

2、如何花钱才是明智的？

每个人掌握的钱的数量不同，调动的资源量级自然不同，不 同量级之间很难用想象力去弥补。但基于本质思考，我们不 管钱多钱少，花钱的思路应该是一致的。

想让钱发挥最大的效能，最好的办法是利用它把今天过好。 当然，由于未来的不确定性，我们需要用一部分钱来抵御风 险，其他的就应该让它流动起来，而不是为了攒钱牺牲当下 的生活。

说把钱花在今天，并不是说“今朝有酒今朝醉”全部用来挥霍 一空，许多时候，钱花在自己身上不一定是最有价值的，同 样花在自己身上，不同的用法产生的结果也是非常不同的。 这涉及到了具体的金钱观，智者见智，我这里就不展开说 了。

核心是我们明白钱的背后是资源，我们应该用这个资源让生 活变得更好，而不是变得麻烦。

在这个意义上，贫穷和富有的区别不在于拥有多少钱，而是 看你从这个社会获得资源的能力。

3、赚钱的正确姿势是什么？ 赚钱是每个人都关心的问题，似乎是每个人的刚需。其实仔

细一想，赚钱并不是真正的刚需，对大部分人来说花钱才是

刚需，赚钱只是目的。如果你有一辈子花不完的钱，估计就

不再出去赚钱了。

言归正传，当我们明白了钱是对资源的量化度量后，你自然 会了解到越稀缺的资源就会越值钱。那么，赚钱的正确姿势 是创造自己的稀缺资源，比如磨练稀缺的技能。

当你越感觉到赚钱困难，就说明你越需要稀缺的技能了。而 人都是被逼出来的，当你足够困难，那你磨练稀缺技能的意 愿会越强烈。

很多时候，你明知道道理却依旧不行动，只能说明你还不够 困难，你还是挺舒服的。
