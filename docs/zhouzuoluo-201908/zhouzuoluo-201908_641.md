# 《粥左罗：员工是强

《粥左罗：员工是强者，但大部分员工都没有利用好这个强 者优势》

上一篇分享我讲了一个新的认知，老板是弱者，员工才是强 者。我看了星友的评论，很有意思，比如有评论说“现实是老 板赚大头，员工工资低”，有的星友说“星主可能没有遇到过 黑心的老板，自己当老板又比较大方，才会得出这样结论 的”。

你看，这都讨论的不是一主题。这也是我基本不与人争辩的 原因，因为大部分争论是因为，你在跟甲同学讨论 A，甲同学 却用 B 反驳你，冲出来个乙同学说：你俩都不会。然后他讲了 C。ABC 本就不是一个事，当然不需要争论，也没有意义。

好，我继续来讲。 我们在讲员工是强者的时候，说员工可以更容易的伤害老板

（合同有力量平衡作用）。

资本家投入的厂房、设备、办公室、预先的时间成本等等， 他的脑袋无法安在这些资本上面，他不可能 24 小时看着自己 的资本。

劳动者的脑袋长在自己身上，随时可以掌控他为资本家提供 的劳动力的品质和数量。

按薛兆丰老师的话是：企业管理的本质就是为了防止劳动力

滥用资本，即员工欺负老板。

当然我们从上述论证种能看出员工有个很大的优势：不同于 老板和企业的深度绑定无法脱身，一个员工其实几乎有随时 离开一个企业的能力。

老板没有这个能力，他不可能想抽身就抽身。我仔细的度过 俞敏洪的经历，俞敏洪多次表示，他已经被新东方锁死了， 他从前想抽身发现不可能，同时他羡慕徐小平和王强，说他 们俩赚到足够多的钱就可以抽身离开，而他必将与新东方绑 定一生，这个绑定的解除，要么是新东方倒闭了，要么是俞 敏洪去世了。

一个员工便是自由的，几乎拥有随时离开的能力。 但我今天要说的是：大部分员工都没有珍惜自己的这个能

力。

我看到很多星友提问，说自己的公司不适合自己，岗位不适 合自己，和领导处事不搭，工资收入不公平，得到的机会不 公平等等各种问题时，特别纠结要不要离开，有的人甚至因 为反复纠结，在一个不适合自己的公司一干就是两三年。

所以，你们根本没有意识到自己拥有可以随时离开的能力， 你们根本没有珍视这个能力。

大部分人的描述里，就好似自己离开的成本有多大似的，其 实上面我们讲了，你之所以用随时离开的能力，正是因为你 离开的成本很低。

所以，本来老板是弱者，员工是强者，因为员工们都不珍视 自己的那项能力，最终变成了“不是劳动力在滥用资本，而是

资本在滥用劳动力”，谁造成的这种结果？你自己。

而且提出这种疑问的人，很多都是在这家公司混的并不好的 人。

这就更不对了，一个员工在一家公司混的越不好，位置越不 重要，他“随时离开的能力就越强”。反而是那些混得好的， 位高权重的人随时离开的能力越差，因为越往上走，他的员 工属性就越弱，老板属性就越强。

你本来就一无所有，又为何担心失去？我们说，暂时混得差 的人，更应该珍视自己的“可移动”能力，并借由这种能力让 自己越来越好，你本来就混得差，还放弃了自己的“可移 动”能力，你会一直差。
