# #新认知思考《与人

王威 : #新认知思考

《与人相处的技巧——换位思考》 今天是我加入星球的第 80 天；我会每天坚持成长，长成更好

的自己。

如果你希望别人喜欢你，那么了解对方的兴趣，在他感兴趣 的领域交流是一种不错的方法。

我特别喜欢钓鱼，每次回家都要去自己家的池塘里钓半天， 我自己很喜欢吃荔枝和口香糖，但是我发现鱼却喜欢吃蚯 蚓，所以每次我去钓鱼的时候，不会想我所喜欢吃的东西， 而是琢磨鱼儿喜欢的美味佳肴，我不会鱼钩上挂口香糖做鱼 饵，而是挂上一小截蚯蚓，然后垂到鱼儿面前，说：“你想吃 这个吗？”

当你“钓”人的时候，不妨使用这种方法试试？ 要想钓到鱼，鱼饵必须符合鱼的口味！但是我们为什么总是

对自己的需求大加讨论呢？这像小孩似的荒缪做法。

我们关心的是自己的需求，所以我们对自己所需要的东西永 远都会感兴趣，但是别人对你所需要的东西并不感兴趣，别 人都像你一样，只会对自己的需求感兴趣，所以，能够影响 他人唯一的方法，就是谈论他的需求，并告诉他如何去获得 他。

不管你怕对待小孩，或者小猫，小狗，这条原则都要牢记。

举个例子，有一天我想把牛赶进牛圈，我只想达成自己的目 的，我在前面拉，弟弟在后面推，正如我们一样，这头牛也 只想它自己所要的，所以它蹬紧四腿，在门口一动不动，这 时姥爷看到了，他什么也没说，他知道牛想要什么，拿了一 把干草在牛面前晃悠了两下，牛就跟着他走进了牛圈。

当你知道别人的需求时，并且在相处时谈论、甚至满足他人 的需求，那么你一定会获得人们的喜爱。

看帖如有收获，点赞是种态度——这里是成长实践星球@粥 左罗和他的朋友们：每天睡前，都要比早上醒来的时候更强 一点，365 天后你就是自己最大的惊喜——高手的战略，是持 续性和稳定性。
