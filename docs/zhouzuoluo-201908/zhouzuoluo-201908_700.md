# 《高小羊：不要试图

《高小羊：不要试图鹤立鸡群，你应该离开那群鸡》

我是一个重度健身爱好者。 每天不管忙多晚，我都会去健身房锻炼一下。 当你把时间花在哪里，你的成就就在那里。 所以，我的健康和身材得到了回报，我在我们健身房，被他

们开玩笑说身材排名第一，所以被他们称呼为一姐。

他们经常开玩笑说，我不去比赛，不去当教练可惜了，我比 健身房那些教练的身材都好太多。

不过说实话，我暂时到目前为止，没有想过当教练，也没想 过要去参加健美比赛。我只有两个目的，保持健康，然后让 自己的线条看起来优美一些。

其实我每天大概就训练 1-2 个钟，然后日积月累的坚持，和饮 食上的稍微控制，自然而然就能够实现这两个目标。

反观我的很多朋友，三天打鱼两天晒网，办的卡基本上都废 弃了。

所以这个东西难吗？不难。难得是你是否能日复一日的坚 持，以及你是否相信时间会给你想要的一切。

我坚持了四年了。

很多人其实就已经死在了开头。 我以前在健身房，会遇到几个志同道合的朋友，一起交流训

练心得，有时候也相约结伴一起训练，相互加油打气。而且

我们也私下打赌比赛，谁能一个月坚持来的天数最多谁就

赢。

慢慢的，这些人消失了，要么是半途而废的，要么就是换了 工作，顺带写把健身房也换了。

慢慢的，我从一群人结伴训练，到一个人默默的去到健身 房，默默的训练完回家。所以在健身房唯一交流放松的机会 和兴趣也没有了。

当时打赌比赛的几个项目，到目前为止我还保留着第一。我 们健身房，说句不好听的，素质都不大高。曾经有个朋友建 议我在健身房把私人问题解决了，我想了想，算了，因为很 多来健身房的人，大都是来拍照发朋友圈的，也有很多是来 撩人的，还有霸占器械的和乱扔器材的。

而且我今晚算了下，我在这家健身房已经待了有四年了，所 以保持着这四年最高的打卡记录。

我觉得是时候要换一个健身房了，换一个新的地方，随之而 来的是不同的环境，心情，和新的朋友。朋友之间相互帮 忙，相互打赌比赛的层次也会不一样。

有个朋友和我一样，也是待了有四年了，最近我们碰到时， 都会说，啊，这里待的好腻啊～
