# 《到底是什么毁了我

《到底是什么毁了我？》

这是程先生的太太 第十五次 分享，希望大家看了能有收获。 这是一个创业摇滚的时代，这是一个毁掉无数年轻人的时

代。

1、毁掉我的是我想的太多 这两年是知识付费盛行，以前接触大量知识只能是通过图

书，并且必须深耕细作才能有自己见解。

随着知识付费的发展和普及，我随时随地都能获取到很多知 识。我被告知了很多的道理，每一个大神都以过来人的经验 告诉我，应该如何做，不应该怎么做，我甚至不用思考任何 已和决定，就有许多人帮我做好决定。

我看的越多，想的越多，顾虑也就越多。 我焦虑我的选择不能大于努力，而使我的努力白费了。然后

我就想如果努力注定白费，那么我为什么还要努力呢？

我不知道自己是在做正确的事，还是事情本身是正确的。这 个时候没有人告诉我正确的标准，学习了更多的知识之后， 我连做基本的事都不会了。

我每天学习，为知识付费，也为自己焦虑。

如果我每天都必须花心思取思考如何才能更好的选择，如何 才能做正确的事，那我用什么去认真做事呢?

除了知识付费，还有自媒体，很多还是不良自媒体。 面对自媒体，我们永远可以只选择自己喜欢的部分，至于我

们不想接受的，我们可以完全屏蔽，但是我们关注的真的就

是正确的嘛？咪蒙每天的推送除了让自己心里好受点，还有

什么其他用处。就算每天看一篇丁香医生，还是照顾不好自

己的身体。

不过是让别人安慰自己，心安理得的成天幻想。 2、毁掉我的是我做的太少 韩寒一句：道理我们都懂，却依然过不好这一生。戳中了多

少人的内心？

对我，它是一剑封喉。 我知道运动是极好的，可是我起不来床，迈不开腿，管不住

嘴。我也知道英语是极其重要的，可是我不喜欢英语啊，我

就是记不住单词，不敢开口说话。

我都知道，我真的都知道，只要我掌握了这些那些技能，我 的人生就会不一样。可是我懒啊，就是烂泥扶不上墙啊。

执行力，一个被生活推着走的人能有什么执行力呢？自律， 更跟我说这些，那么多人都做不到的事，干什么我就非得做 到？

你知道每天早起有多痛苦吗？我不是没有试过。

你知道每天说着一口蹩脚的英语需要多大的勇气吗？我也不 是没有试过。

你知道每天死磕高深技术需要多少耐心和时间吗？我真的很 想放弃。

我总是能找到一万个理由放弃，真的找不到一个理由坚持。 3、毁掉我们的是自己 许多人常常将自己的失败归咎于对方要求太多，运气太差，

又将别人的成功归于运气太好，对方要求太低。

其实不过就是不想承认自己无能，自己失败。 我们总听别人说毁掉年轻人是让他赚快钱，是让他闲着，是

让他高配，是让他在体系里。我们除了自己骗自己，还让别

人骗自己，甚至附和人家说的对。

到底是什么毁了我们自己，我们自己心里没点 B 数吗？ 是我们做事不专心认真毁了自己，没有愿意全身心的投入。

做一点小事就想着别人来夸你几句。这是做事还是表演呢！

是我们的不自律毁了自己，坚持一天是坚持吗？这是你试 水！什么是自律，是外界无论有什么诱惑有什么阻碍，该做 的事还是的做。

是我们毫无执行力毁了自己，学富五车博古通今又如何？梦 想着出本书，却连一个字也不愿写。日不积跬步何以致千

里，道理朗朗上口又如何？每次都嚷嚷着世界那么大，我要 去看看，可哪个周末不是躺在家里追剧？

看了那么多书，读了那么多文章，懂得了那么多道理，可都 用来麻痹自己，欺骗自己毁掉我们的是这个社会，是另有他 人。

清醒一些吧，毁掉我们的只有我们自己。
