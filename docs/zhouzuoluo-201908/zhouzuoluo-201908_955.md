# 《时间管理的问题和

《时间管理的问题和解决方法——上》

今天听了关于时间管理的一些问题和解决方法，我先总结了 前面 4 个部分，分享给大家，下面 4 个部分，明天分享。

1、凡事要提前完成，不要让截止时间追着跑 每当我们做一件事情的时候，都要想办法提前去完成， 这样

会让你工作和学习更高效，心情也会特别好，如果老是拖

延，到最后时刻才去做得话，我们到最后不一定能完成，因

为每天总会有突发事件出现，有时候因为突发事件，你没有

办法去完成了，你会感到自责，然后会焦虑，效率又会低

下，所以提前完成是个很好的习惯。

2、每天早晨做好每日管理，做 to do list

我每天在用时间效率手册，确实对我的工作和学习带来了改 变，我每天会在效率手册上写下今天要做得事情，这样你一 天下来是明确知道自己是要做什么的，提升效率，而且每天 把自己列的事情做完，你是很有成就感的。

3、正确的利用碎片化时间 我们碎片化时间只能来做碎片化的事情，比如你可以利用碎

片化的时间来看看星球的分享，或者回复不需要思考太多的

邮件等等，如果你碎片化时间用来看一本书，那肯定是不合

理的，你刚看了前面一部分，就要忙别的事情了，这样思绪

就会被打乱了。看书、学习需要花大量时间的事情，就要找 个安静的地方专心的去阅读，碎片化的时间就不适用了。

4、事情排列焦虑症 我们有的时候想让自己一下子完成好几件事情，今天晚上既

要看一本书，又要写文章，还要去听课，那么我们下班的时

间是有限的，不可能把这些事情全部完成，我们就需要做一

下排序，比如今天下班看一本书，明天下班去听课，后天写

文章，就是不需要把每一件事情都排在同一天去完成，可以

根据自己的需求做不同的安排。

以上就是今天的一部分分享，希望对大家有帮助，下部分明 天见。
