# 「人人都该掌握的 7

「人人都该掌握的 72 小时定律」

在《小狗钱钱》一书中的有这样一个观点“当你决定做一件事 情的时候，必须在 72 小时内完成，否则你可能永远不会再做 了。”

人总是陷于琐碎事情的忙碌，所以很多事情明知道很重要， 也总是没时间做。

01 决定做的事，在 72 小时内行动 科学上对“72 小时定律”的解释是：因为我们的大脑在做出某

种决定时肾上腺皮质会分泌出防御性荷尔蒙，在 72 小时内，

无论是多么不情愿做的事情也能坚持下去，且信心十足，因

此事情的成功率高。

15 岁觉得游泳难，放弃游泳，到 18 岁遇到一个你喜欢的人约 你去游泳，你只能说“我不会耶”。

18 岁觉得英文难，放弃英文，28 岁出现一个很棒但要会英文 的工作，你只能说“我不会耶”。

想做一件事，就要在 72 小时内立即行动，否则，我们就失去 你做这件事的最佳时机。很多时候我们都觉得自己做不到， 其实不是做不到，而是根本不想做到。

02 一旦超过 72 小时，拖延症便占上风 为什么会存在拖延症？因为我们有个普遍存在的“鸵鸟心

态”，像鸵鸟遇到危险时，以为只要把头埋到土里，就可以远 离危险。

事实上我们也有这个体会，一旦拖延，没有把握住重要事情 与紧急事情的合理安排，我们就会忙得像热锅上的蚂蚁，永 远也找不到工作的重点。

一直想去做的事，一直在拖延，最后白白浪费有兴趣的好机 会。

很多人做事情也是这样，觉得只要不去解决问题，把它放在 一旁，时间久了自然就会有解决的办法，这是永远不可能发 生的。我们每天都很忙，但常常没有做那些真正重要的事 情，要么是因为太难，要么是太麻烦，要么是没有即时收 益，要么是本能的靠惯性做事。

03 即时行动：完成永远比完美更重要 通过坚持每天写作以来，我也深刻意识到：以前的那种“要么

就不做，要么就做到完美”的心态调整。之前一直在想，我要

先学习大量的写作知识，阅读足够多的好文章，拆解无数爆

款文章之后，我才能开始写作，不然我都不好意思动笔。

事实证明这是不对的，粥老师的音频课告知我们，一定要写 起来，马上练习，边写作边进步。回过头来看，确实是正确 的做法。

太多时候我们搞错了方向，花费巨大的时间精力做准备工 作，而迟迟不展开行动，给自己的拖延找理由。

72 小时定律证明，当我们执拗地将某件事情列入 72 小时计 划，放在比较重要的位置，我们就有时间去学去做了。

最怕的是，一直在想，却一直没有开始行动。 心中有什么未尽的梦想，就立即勇敢地去追逐吧。路要一步 一步向前走，登山要一个台阶一个台阶往上迈，才能最终登 上山顶。没有一蹴而就的事情。

想要得到，就要拿出行动。种一棵树最好的时间是十年前， 其次就是现在，72 小时之内。
