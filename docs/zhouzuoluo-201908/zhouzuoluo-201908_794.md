# 《你以为自己对工作

照明邦 : 《你以为自己对工作感兴趣，其实是你没得选！》 前两天晚上，徒弟的父母来我家做客，说了一些客套话后，

就问我这一年徒弟的进步情况。

一年了，进步肯定有，只是做父母的不懂，我只能讲一些他 们懂的，比如相处的这一年中，我发现他的一些优点和缺 点。

优点：

1、安排的任务，他会努力去完成，加班加点时没有任何怨 言，算有责任心。 2、不愚笨，因为做事的过程中，他会主动寻找更便捷的方 法，不像很多人不愿动脑，有事直接干，不管方法路径是否 高效。 3、三观正，徒弟才 19 岁，他跟我说过，希望趁现在年纪小， 压力不大时，学会一门技能，以后想去深圳广州那些大城市 闯一闯。现在也不想分心谈女朋友，等到以后工作稳定再 说……

缺点：要求加班就工作学习，不要求就把自己关在房间内， 不会主动给自己安排学习任务。

他母亲听完后，表示知道他这些缺点，和他爸爸一样。不过 只要他喜欢这行，对这行感兴趣就好。临走时，塞给我小孩 一人一个红包。

他父母走后，我一直在想他妈妈的那句话：“只要他喜欢这

行，对这行感兴趣就好”，而徒弟以前也说过喜欢干这行。那 他真的对这行感兴趣吗？我觉得未必，他之所以说喜欢这 行，也许是因为他没得选！

徒弟和我一样性格内向，不善言辞。由于两家相隔不远，每 次我开车回家都会带着他，而在路上的这一个多小时，我都 会主动和他聊天。慢慢地对他有了更全面的了解。

一年前，邻居跟我说，他有个朋友想让他儿子到我这里来学 设计，经过一番交谈了解后，我答应了。后来我才知道，其 实那时徒弟有三条出路：

1、做电话销售员 徒弟职高毕业后，自己去找了份不需要什么门槛的中国移动 电话销售员，就是充值送网线，升级套餐之类的业务，偶尔 还要去街头卖卡。他说打电话经常被劈头盖脸地骂，由此产 生了厌恶心理，觉得自己不适合这行。

2、去舅舅那里学室内设计 他舅舅是个室内设计师，而且懂施工，自己能接工程。按理 说，去他舅舅那里学习是挺好的一条出路，为什么不呢？他 说舅舅说了，自己太忙，没有时间教……

3、跟他爸爸去学泥水工 他爸爸是工地的泥水工，曾经开导他，让他跟自己学泥工， 虽然有点累，但收入差不了，一天挣三五百，现在不稀罕 了。

以上三条路，明眼人都知道，哪一条都不好。而跟我学设 计，优点就非常明显。我俩性格相似，而且贴身学习，加上 这些年挣了点小钱，相信我邻居一定添油加醋说得更夸张， 总之在他们看来前景更好。在这种情况下，应该谁都会说这

行不错，要跟我学设计吧！

我记得当年自己进电脑培训学校学习之后，也是迷茫得不 行，因为找不到相关的工作，很可能得继续去工厂打工。刚 好在这时，有人介绍我进设计公司做学徒，我想傻瓜都知 道，要抓住这次去设计公司的机会。而这并不是兴趣驱使我 的，而是我没得选，没有更好的选择。

写到这里，想起作家龙应台写给儿子的那句话：“孩子，我要 求你读书用功，不是因为我要你跟别人比成绩，而是，你希 望你将来有选择的权利，选择有意义，有时间的工作，而不 是被迫谋生。”

是的，很多人用功读书，并不是因为兴趣。就像徒弟选择跟 我学设计，我选择去设计公司当学徒一样，都不是兴趣驱 使，而是为了将来不再被迫谋生，自己有选择的权利，能够 选择有意义、有尊严的工作和生活！
