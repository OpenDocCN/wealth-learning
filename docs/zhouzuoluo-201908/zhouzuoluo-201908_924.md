# [大家好，我是懂先

[大家好，我是懂先生。这是我的第六十九次分享，很开心遇 见这么优秀的你们。在星球的一年，我希望实现三个愿望， 如果有一天实现了，我会把我的故事说给你听。]

《谁都不欠你的，别忘了谢谢每一个帮过你的人》 我们总是容易把别人对自己的好当成理所当然，而没有真诚

的对帮助过我们的人说一声谢谢。好多帮助在我们看来都是

理所当然的，但其实谁都不欠我们的，没有人有义务必须要

给予我们帮助。

对于那些愿意帮助我们的人，请真诚的说一声谢谢。而对于 不愿意帮助我们的人，也请不要诸多苛责，每个人都有权利 选择帮或不帮的权利。

单位新来了一位同事小超，由于大学刚毕业没有工作经验， 领导让我带一带他。对此我也十分乐意，能够给他人提供帮 助自己也会十分满足。所以工作上不管有什么问题问我，我 都知无不言。

但由于最近上级部门年终考核，我手头的工作忙到爆炸，而 且身体出了点问题，工作效率很低，检查材料一直没弄完。 小超这时让我帮他改一个稿子，我和他说了我的实际情况， 当前真的没有时间，但我讲我的思路给他说了一遍，不能说 我的思路就对，但起码可以给他一个启发。

但小超却对我说：“当初领导说有问题就让我问你的，而且你

也答应了，现在怎么不管我了？”说实话，这种典型的学生思 维我真的不知道怎么回答。

入职将近半年，已经记不清帮了他多少次，他重来没有正式 的说一声谢谢，也没有说因为我的想请我吃个饭什么的，虽 然请我我也不会吃。但我都没有计较，全当他小不懂事。但 他今天的话确实让我很失望，我帮他没有多挣一分钱，完全 是处于个人感情，可他却把这当成了理所当然。

通过这件事我也在想，时我是不是也忘了感谢那些曾经帮助 过我的人。我离开第一个公司已经两年多，和之前的朋友联 系已经越来越少，包括给我很多帮助的老师也只是过年的时 候一个短信问候。

所以今天我联系了第一份工作时带我的师傅，叙旧过后要了 他的地址，打算给他邮一些东北大米。虽然他并不缺这点大 米，但我相信他会很开心，因为这说明他曾经带过的徒弟还 记得他，没忘了他当年的帮助。

工作压力越来越大，生活节奏越来越快。但不管多忙多累， 我们都不应该忘了那些帮助过我们的每一个人。工作上帮助 过我们的同事、跌倒时拉过我们一把的朋友、星球里给我点 赞的星友…

他们本可以不帮我们，但是他们帮了。所以，不要吝啬自己 的感谢，是他们让我们的生活更加明亮，请用心去感谢帮助 过我们的每一个人。

谢谢你们，每一个善良可爱的人。
