# 《个人品牌，你应该

《个人品牌，你应该知道的三件事》

「写在开篇」接触这个词语有一段时间，并没有很系统的了 解和认知，今天在粥老师的直播课得到一个详细完整的框 架。

一、个人品牌是什么 无论你知不知道、承不承认、接不接受，事实上我们每个人 都是行走的一个品牌。何为个人品牌呢？——别人对你的认 知。

你的朋友、同学、同事、领导、同行甚至更高层次的合作伙 伴对你的印象，为你贴上的标签，就是你的个人品牌。 如果你不想被分为某类人，就别做某类人做的事，体现在日 常行为举止、微信朋友圈、微博等言论变现上。同理，如果 你想保持某个形象代言，就坚持做好相关主题的事情。

注重个人品牌思维，注意加分项加强，减分项不做。因为品 牌积累过程日积月累，但是毁灭却在一念之间。刘强东的企 业家形象好不容易建立起来之际，因为偷吃被抓住，无论违 不违法，在大众心中的形象已经严重受损，公司股票也一落 千丈。

二、经营好个人品牌的四个要素 个人品牌伴随一身，它就是你的“身份证”，经营好它，从长 远来看是一个明智之举。

1.认知度

外界对你的评价和品质类型归类就是你的品牌认知度。这是 一块大基石，不熟悉你的人如果碰巧有认识你的人可以打 听，事实上，这种情况常常出现，认识你的人传播出来对你 的印象此时尤为重要！

我在面试第二份工作的时候，因为公司比原先的高级很多我 很重视，面试都完成得很顺利，最后我的未来直属上司说只 差 HR 去做一下背景调查，没问题就可以来上班了。如果这时 候你是我，会忐忑吗？

我不会。那时候我心中已经十拿九稳了，因为我深知无论询 问到我任何一位同事，都有把握得到肯定的评论或者是中肯 的评定。 最后我顺利在当天收到录用通知，入职之后听他们说，前同 事对我的评价都特别高，无论是工作能力还是待人接物。

2.知名度 知名度建立在认知度的基础之上，名人的这点尤为重要，特 别是需要粉丝流量等增值的人。如果知名度不高，粉丝很 少，即使认知度很高也鲜为人知。

3.美誉度 美誉度显然是最重要的。无论认知度多全，知名度多高，如 果美誉度极低的话，就变成了“臭名远扬”。

4.普及度 普及度要求较高，需要你不知在圈内广为人知，还需要在不 同行业、圈子上有交集，交叉面积越大，普及度越高。这一 点，可以将“马云、马化腾、刘强东”等不同行业能被大众所 熟识的理解方向。

三、升级个人品牌的注意事项

1.作品和人品：两手抓，两手都要硬 这里作品可以理解为不同领域的不同产物。作为职员来说， 就是工作风格和能力；对于写作者来说，就是文章输出；对 于企业家来说，就是公司产品质量。作品体现人品，人品决 定作品。

2.标签和行动：高度统一，互相加持 前面所说的，陌生人打探你的认知后，还会进行实际验证， 这时候统一性显示出重要性，如果其他人都评价你守时高 效，这个陌生人约你会面时你却迟到了一个小时，这就不符 合标签和行动的统一性，无法形成统一认知。

3.建设和消耗：建设具有长期性，消耗具有速废性 建设个人品牌是一辈子的事情，在你行动的每一天慢慢积 累，设想马爸爸，从大学便是学生会主席、连长等出类拔萃 的学生代表，到初出学校开始创业再到今天做出的世人皆佩 服的成就，是一步一个脚印踩出来的，试问这样一个好不容 易建设起来的完美个人品牌，如果被暴露出涉嫌偷税漏税等 违法事情，或者被查出其他不为大众所崇尚的丑闻，个人品 牌将在须臾之间崩塌。
