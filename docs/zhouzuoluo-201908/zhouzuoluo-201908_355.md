# Day183《拥抱

《拥抱限制，方能自由》

曾国藩一生功成名就，是公认的一等一高手，但在 52 岁回顾 一生遇到的各种挫折时，痛切认识到自己的力量太有限，发 出了一番感慨，大意如下：

之于漫长时光人生太短，之于广袤天地自身渺小，之于浩瀚 书海所知太浅，之于繁多世事能做太少。

我们往往觉得高手可以享受更多自由，真相是他不得不面对 更多的限制。放眼天地，纵观古今，任何一个人的一生都充 满了限制。而修炼自身、成为高手的过程，就是不断看清边 界，不断受到约束的过程。

于是，由时长我短的限制学会豁达，由地广我窄的限制学会 取舍；由书多我少的限制学会谦卑；由事繁我简的限制学会 专注。

1、关于豁达 人生不如意事十有八九，但当你懂得了在宇宙漫长时光之中

人生只有一瞬间，你所遇到的伤痛沉浸其中则无法自拔，跳

脱出来则充满希望。

高手明白人生所限，于是心境超脱，获得自由；普通人希望 人生自由，却感受到各种限制，于是常常陷入忧愁。

2、关于取舍

白居易有首诗是这么写的：蜗牛角上争何事？石火光中寄此 身。我们多数人总感觉得到的太少，孜孜于未得到的东西。

高手却可以平衡进取和退让之间的界限，先看到限制，然后 追求属于自己的东西。

3、关于谦卑 成熟的麦穗，谦卑的低头，其实它们籽粒饱满；空瘪的麦

穗，个个骄傲的昂着头，其实轻飘。相比于人类知识的总

量，我们每个人都是无知的，而人类的知识在无穷真理面前

又是非常有限的。

高手看到的是未知，谦卑而上进，不因自己的一点所得和成 绩而沾沾自喜。

4、关于专注 世事繁多，我们多数人想法也总是太多。但想法越多，就越

难抵制干扰，去专注于少数重要的目标。

高手深知一个人的时间和精力总是有限的，只有专注在少数 重要的事情上，才有更大概率做成事情，也有更多时间享受 生活。只有坚持以少赚多，才能获得真正的自由。

结语： 普通人看不到边界，行动自由，结果受限；高手看得清边 界，行动受限，结果自由。愿你能够懂得，然后拥抱限制，

最终获得自由。
