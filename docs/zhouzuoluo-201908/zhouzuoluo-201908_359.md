# 《公交上听音频，阅

增雄 : 《公交上听音频，阅读，是我 3 年来犯的错误》

学习是讲究环境和氛围的，之前粥老师说过，我们学习不能 太过于矫情。比如说，利用上下班坐车回家的路上，可以看 一下文章，可以听一下音频，或者坐地铁的时候，带上耳 机，听你所听，学你所学。表面上我们利用时间，实际上在 损害健康。

坐公交车，车的噪音比较大，你不得不提高手机的音量，这 样听多了，很容易使听力受损。车是运动的，晃动的手机屏 幕，视觉也不是那么好，长期以来，会使你的视力受损，不 值得。

这么说来，我利用如此借口，说明公交车地铁不适合学习。 难道要给你找一个非常安静，环境非常优雅，才能让人去学

习吗？

这样再好不过，一方面，人在工作或地铁上学习的时候，很 容易分散注意力，至少你要关注，是否到站了？是否礼让老 人小孩？是否钱包安全？另一方面，摇摆或奏停的地铁或公 交环境，会使听力和视力受到影响，不信你对着镜子看一下 眼睛红色的小血丝（过度用眼），以牺牲健康为代价的学 习，不值得。

那怎么办？通勤 2 小时的车程，难道就看窗外看人头？可以理 清一天要做的事项，有条理有目标，或者反思一天所做过的 事，趋利避害。

下面这张图，我是早上 7:30 坐公交看到的情景，很多人都低头 看手机，我觉得信息包围我们太多了，不管出于什么目的看 手机，都离不开用眼睛看。就在我拍完这张图片后，前面的 一个女孩就放下手机，从包拿出了护眼贴，贴好，靠在座 位，享受护眼的时光。
