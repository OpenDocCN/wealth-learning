# 《成长的真相：真正

王徐人 : 《成长的真相：真正自律的人，坚持其实不需要毅 力》

1.

寒假过后我就重新恢复了低碳生酮饮食的状态，从开学至今 我低碳生酮也差不多一个月了。

这一个月内我经历了从十分渴望高碳水食物到慢慢适应低碳 饮食，对米面也不怎么渴望的过程。

稍微提一下，低碳饮食不是素食，低碳饮食可以简单理解为 只吃肉蛋菜和部分低糖水果，不吃米面的饮食。

低碳之前我是一个饮食不规律的人，会不定期暴饮暴食，而 且会暴食高糖食物。我曾经不止一次地在一个小时内吃光 120g 的高糖巧克力或者奶油蛋糕。

不过因为体质问题，我体重五年来一直保持在 80-83 斤左右。 我很瘦，但是不健康的饮食方式终究不是好事——我看起来 不胖，但脂肪可能堆积在肝里，不良饮食习惯的瘦子其实更 容易得脂肪肝。

一次偶然让我走上了低碳生酮的道路。一开始是真的很辛苦 的，因为米面是我从小吃到大的食物，不吃米面我总觉得饥 饿——不是生理上的饥饿，是心理上的饥饿，是对高碳水食 物十分渴望的一种感觉。

等过了半个月，身体开始有些习惯了低碳水的饮食之后，我 的状态也渐渐变得更好了，脑子清醒，精神更好了，身体也

更加轻盈了。

2.

让我真正觉得自己进入低碳状态的是我发现自己对高碳水食 物已经没有那么渴望了。戒糖对我来说不是问题，虽然我以 前会暴食高糖食物，但并没有上瘾症状；真正让我上瘾的是 高碳水食物。

所以，降低对高碳水食物的欲望是我在低碳生酮道路上的一 个里程碑。

在低碳生酮的道路上，我也不断关注相关公众号，查阅相关 资料。我所得到的信息是，真正完全进入低碳生酮状态的 人，对高碳水食物几乎是 0 欲望的。低碳生酮饮食的维持终将 成为小酮人的习惯，不需要靠意志力坚持。

对于低碳生酮达人来说，低碳生酮饮食是能让他们获得快乐 的事情，而不是旁人想象中的那么辛苦。

这也是我的低碳终极目标——将健康的饮食习惯完全内化为 自己的本能，而不是靠自制力去控制。

3.对低碳生酮饮食的进一步了解，也让我联想到了我们的日常 生活。

我们总是说要坚持，要努力，要成为更好的人。 但其实在我们说这些话的时候，我们的潜意识里往往将努力

变得更好想象成一件很不容易、时刻需要靠毅力控制的事情

的。

我曾经问过一个很自律的朋友，为什么她能坚持每天晨跑、 每晚阅读的习惯？她当时很理所当然地回答我：不用坚持的 啊，不做这些我会觉得很不习惯。

这位朋友是我的发小，我们一起度过了人生前二十多年时 光。从她七八岁起她妈妈就一直培养她晨跑和睡前阅读的习 惯，我见证了她十几年的坚持——现在其实无需坚持，因为 这已经成为她的生活习惯，让她不晨跑、不睡前阅读，她反 而难受得紧。

去年的时候看到一篇文章，写的是村上春树坚持跑步 36 年的 故事。当时我就想到了我的这位发小。我想，或许在他们的 心里，坚持真的不需要坚持，自律也只是一种习惯而已。

4.

前行会让人们感受到真正的美好——这种美好的前提是坚 持。

前行的道路应分为三个阶段： 第一阶段就是止步不前，我们肉体很轻松，精神却很煎熬；

第二阶段就是克服惰性，努力前行。这个阶段肉体和精神都

会有不同程度的煎熬，但内心会觉得踏实；

第三阶段就是自律成性，坚持其实不是坚持，而是习惯。这

种状态下，人的精神和肉体都不会难受，相反，他们会很享

受这种状态。

只有不断前行，不总是把注意力放在坚持的辛苦上，坚持才 会越来越简单。

成长的真相就是：真正自律的人，坚持其实不需要毅力。所

以，去追求更美好的事物吧！不管是健康，爱情，事业，或 者是其他。终有一天，坚持会让自律成为我们的习惯，我们 也能享受努力而不是煎熬地努力着。
