# 《高情商的孩子，不

《高情商的孩子，不跟父母讲道理》

看过粥老师的文章《我见过情商最低的行为，就是不停地讲 道理》，记得里面有一段是说，高情商的孩子，不跟父母讲 道理。

今天终于派上用场了。 我爸的诺基亚手机已经用到第十个年头了，只有打电话的功

能，他却一直没舍得换。我们跟他说，要不换个智能机吧，

屏幕大又能拍照，下载个微信跟我们语音视频都方便，好说

歹说就是不肯。

他吐槽我的苹果手机每天冲电两回，不如他的手机声音大， 还经得摔，用了这么多年一点毛病都没有。但是从去年，他 的手机开始经常无缘无故死机，铃声也有时响有时不响，以 至于有时候打他十几个电话都接不到，让人挺担心。

所以，我今天又说：“我帮你换个手机吧，我给你买好寄到家 里，如果你用几天不喜欢，还可以退款的。”

他还是犟嘴说：“你要节约用钱啊，不要工资高一点就乱 花……”

我说：“我买便宜一点的给你，花不了多少钱……” 他：“不要不要，智能手机不方便.....”

我突然意识到，自己又陷入跟父母讲道理的困境了，无论我 怎么说，也说服不了他啊，情绪上来还容易起争执，我为什 么还要跟他讲道理啊？

然后我说，那好吧，就挂断了电话。然后打开京东，直接下 单了一款排在最前面的手机，填写了家里的地址和我妈的电 话（怕填写我爸的他直接拒收了）。

发了一条微信给我妈，告诉她已经买了一部手机给我爸，帮 他收一下货。

我妈告诉我，其实我爸的手机早就有毛病了，他死犟着不肯 换，也不是舍不得钱，而是他不会用智能机，怕我们笑话 他。所以嘴硬说他的手机好用。

你看，我一直还以为他不舍得钱，以为他不知道智能手机功 能好用，死命跟他讲道理。对于父母，道理没那么重要，有 的时候，只要用自己的方式，去爱他就够了。
