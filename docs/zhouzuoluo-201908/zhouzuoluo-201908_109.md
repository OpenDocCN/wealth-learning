# （大家好，我是阿萌

（大家好，我是阿萌，这是我练习写作第 3 篇）

《一场座谈会，让我学会避开职场三个坑》

今天总公司召开座谈会，参加人员是 2011 年特招的同一批入 职到各分公司的 13 位职员，我是其中之一。

会议上老总让参会人员发言汇报自己几年来工作情况、感想 和一些工作建议，做为考核提拔干部的根据之一。

我观察到，从一个简单的汇报，就大概能看出一个人平时工 作的样子，也发现一些职场人员不应该掉的坑。

1.别老拿内向说事，学会选择汇报。

13 个人中，几乎都是精心准备了发言稿，也说出了多年来工 作心历路程，得到了老总的肯定。

但我发现，A 同事工作繁重又非常勤奋能干，但汇报时很拘谨 内容乱，根本没能把他的工作成绩展现出来。B 同事岗位业务 较清闲，工作业绩一般，但非常会汇报，有条理带感情，把 工作亮点用事例摆数据等，说得老总频频点头赞同。

两者一比较，A 就是一直都埋头苦干，会做不会说，他说自己 太内向，在我看来就是不会作汇报，比起会写会说的人，不 会汇报的吃亏很多啊......

为什么说会向领导汇报工作很重要？

李嘉诚创业之初做过推销员，他说，再好的东西，只有推销 出去才会转化为利润。人在职场也一样，工作动手又动脑， 会做更要会说，要学会推销自己。

因为你想升职加薪，你需要领导的认可和赏识。这需要领导 对你有了解。

那么职场中把和领导在一起的时间算算，再把有其他同事在 场的时间剔除。你会发现，你和领导汇报工作，是你最主要 的能够向领导展示自己的机会。

所以，领导对你的看法在你汇报工作的时候基本就定了。比 如这次，老总对 B 同事的印象肯定好过对 A 同事的印象。

做出业绩与善于汇报，相辅相成。不会写不会汇报的，赶紧 练起来学起来吧。对了，加入粥左罗写作训练营第四期就是 个好办法。

2.别老拿个性说事，学会选择情绪。 轮到一位女同事汇报工作，说着说着突然哭了起来，她说工

作太辛苦，自己性子易情绪化…听得老总眉头深皱。

情绪化是影响职场晋升的重要因素。据调查显示，约有 87% 的企业经营者会因为担心情绪化的员工无法控制自己的情绪 而不敢委以他们重任，而几乎所有的企业经营者都对习惯性 抱怨表示出了强烈反感。

太情绪化＝难担大任，过于情绪化可以说是职场的禁忌。 那些负面情绪较突出的员工，是老板不喜欢的。因为性格很

难改变，但情绪是可以调节的，而且对行为有着直接的影

响。有负面情绪的人往往容易失去工作的积极性和主动性， 还会给团队带来较多负能量。

要求员工心理强大，是很多老板的共识。曾任蚂蚁金服首席 执行官彭蕾在一次会议中发表自己对于“什么是合格人才”的 看法，其中很重要的一点就是要“皮实”。

滴滴总裁柳青在谈到对团队的期待时，第一点也是希望团队 要有“心力”——放下玻璃心，换一个钢的、铁的回来。

在职场上，谁又容易呢？但不要总拿自己的个性说事儿，不 要有太多的情绪化，领导和同事不是你父母会处处包容你。 学会为自己的情绪负责，学会选择好情绪，化解坏情绪，才 是一个成熟的职场人应有的职业素养。

3.别老拿机遇说事，学会选择努力。 我们 13 位于同年入职，刚入职的岗位几乎一样，但是八年过

后，有人仍然是普通员工，有人已经升职加薪为中层管理人

员，有人甚至已经是分公司的一把手。

有同事说，是机遇不同。我相信机遇，但我不相信机遇是偶 然得到的，努力了才会有机遇。

知乎上有句话很皮但很在理：如果不努力，你的选择无非就 是“我去蓝翔是学汽修好呢？还是学厨师好呢？”

我那位已经当了分公司一把手的同事说，工作这几年，他也 有低谷期，也遇到太多糟糕的事，他也曾不知道明天究竟在 哪里。但他用实际行动告诉我们：努力比任何东西都来得真 实，用坚持换机遇，用努力换天分。

也许，升职加薪他有运气的成分，但成功恰巧就是运气撞到 了你的努力。努力永远不会错，即使现在用不到，未来的某 天也会派上用场。

人生好像一场赛车，一旦你停止不动，给你再好的装备再好 的补给都没有用，你不努力去跑，会有无数人从你身边一越 而过。只有持续不懈跑好每一段路程的人，才能保持前行的 速度，最终顺利达到理想的终点。

越努力才会越幸运，我们要始终相信努力奋斗的意义，让未 来的你，感谢现在拼命努力的你。
