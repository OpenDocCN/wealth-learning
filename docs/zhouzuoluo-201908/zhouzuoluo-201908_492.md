# 这是我在星球的第三

这是我在星球的第三十七篇分享《先把自己活成一个“超级个

体”》

公众号大 V 粥左罗老师慨叹：“四年混成超级个体，他很难 过。”

粥老师，他凭借自己不懈的努力，日复一日的坚持写作，使 得公众号拥有了几十万的粉丝，知识星球社群有大几千的成 员跟随他学习，他出了书，创作了写作课。

对于一个自媒体人来说，他真是已经很强大了。所以他说他 混成了一个超级个体，个人能力极强，创业初期，赚钱效率 非常高。但是，到一定天花板后，不好突破。这应该是他觉 得不爽之处。

但，目标都是一个一个完成的，每个人都是根据自己的优势 资源，先把自己混成一个超级个体，然后才会向着更全面， 更专业的方向突破。

著名的木桶理论：一个木桶能装多少水，取决于最短的那块 板。在全球互联网时代的今天，这个理论已经演变成：当你 把桶倾斜，你会发现能装的水量决定于你的长板（核心竞争 力）。

当你有了一块长板，围绕这块长板展开布局，为你赚取利 润。如果你把这块长板做到极致，你就能达到别人无法企及 的高度。粥左罗老师的长板就是他的写作能力，围绕这个核 心竞争力，他的公众号，写作课，书，知识星球社群都取得

了不错的成绩。

相对于大多数人，可能还没有达到粥左罗老师那样的高度， 所以我们现在首先要做的就是先把自己活成一个像他那样的 超级个体。

我们要怎么做呢？ 1）尽你一切努力，挣你能挣到的合法的钱 不是怂恿大家拜金，确实是年龄越大，越体会到：你的赚钱

能力，对你自己，你的小家庭，你的大家庭至关重要。

你想做什么花钱的事情，买什么大额的东西，从自己口袋里 面掏钱，总比从别人口袋里掏钱要容易的多，最重要的是理 直气壮，我自己挣的。

孩子的哪个同学又出国去玩了，还给同学带回漂亮的礼物。 看着孩子艳羡的表情，想想自己的口袋，是否也有勇气带孩 子来一趟说走就走的旅行？

老人生病，现在动辄几十万的医药费，你是否有能力轻松承 担？

如果不是，那就努力挣钱吧。尽你一切努力，挣你能挣到的 合法的钱，这才是你当前最重要的事。其它的小情绪，小矫 情都让它一边去吧。

聚焦自己的木桶中的长板，使它成为自己的核心竞争力，同 时不断提升它，使长板更长，正向循环，不断增强自己的赚 钱能力。

同时，要有意识的开发自己的斜杠能力。

白天上班，利用晚上时间学英语，练书法，学写字，学编程 等等，可以提升自己能力，又可以以此为兼职开拓自己的副 业。

2）不要等，很多事情都耗在这个字上面。 “等哪天有空，咱们约上一起吃饭。”基本上没有空的时候。 “等你放假，我带你去玩。”放假了，可是又有别的安排，出

去玩就泡汤了。

“等我有钱了，带父母去旅游。”结果父母都走不动，也不想 出去了，你还没有行动。

“等过了这阵不忙了，就开始学英语。”几年过去了，也没学 过。

“看别人写公众号挣钱，等孩子大点了，我也写。”结果孩子 都从牙牙学语到小学三年级了，还没有动笔。

一个“等“字，说明你不是立刻马上去干。你在灵光乍现的时 候，做的决定，有的想法，如果不立刻去执行，时间走了， 当时的劲头也松懈了，你的想法也就不会去实现了。

如果把“等”字，换成“今天”，“今天咱们约上一起吃饭”，“这 次放假，带你出去玩”，“今天带父母出去旅游”，“今天开始学 英语”，“今天开始写”，是不是结果会不一样呢？

有的人之所以等，是因为他在犹豫与纠结：这件事我没有做 过，我能不能干？干不干得了？失败了怎么办？

逻辑思维 CEO 脱不花有一条莽撞定律：先搞起来，你就成功 了一半。

如果你不开始干，你满脑子里面转的都是“要不要干”，一旦 你确定开始干后，这时你的大脑都会在想“怎么干”。

同时，你多干一步，就离成功近一点。因为问题都是在你干 每一步出现的，也是在你干的每一步解决的。当你解决了过 程中的所有问题，你就成功了。

还没有把自己活成“超级个体”的人们，努力吧，为了这个小 目标。
