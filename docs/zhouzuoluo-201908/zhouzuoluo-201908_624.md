# 《他是一个在星球布

《他是一个在星球布满脚印的星友：不仅自己发光，还给别 人点火》

几个月后，看到这名星友，星里浣突然想起，收到第一个点 赞时那个遥远的下午——第 212 天的星里浣

今天，我一口气刷了几小时星球，发现了一个特别感动的现 象：有一个星友的名字，始终出现在我的视线里，他几乎给 一条帖子都点了赞，至少几百条。

他的名字是 link。我翻看了之前的分享，有整整 50 个【多维思 维模型】相关的分享，简直是一个科普认知狂魔的定位；现 在他的分享一般是思考认知类，也都颇有见地，大家可以去 翻翻他的分享，相信一定会让你有所收获。

不过，今天我想分享在星友 link 身上看到的一个特点：不仅自 己发光，还给别人点火。

link 不仅输出了很多高质量分享，还给很多星友点赞，鼓励他 们继续创作。

给人点赞，是一种鼓励，是会带给人动力的。就像粥老师曾 在文章里说的那句话一样：“在人生的很多阶段，我们都在等 一句话，那句话到位了，发动机就打着火了”

其实，在加入星球的很多时候，我们也在等一个反馈，也许 是一个赞，一个评论，就能让我们成长的发动机打着火。

我还记得第一次收到星友点赞和评论时那种“动力感”，它驱 使我有了写第二篇分享的念头，也有了“写好一点”的想法， 然后一路写下了一百多篇分享，最初的第一个赞也变成了今 天的 5030 个。

我曾经看过一个公众号在文章底部要“好看”的文案，叫“你的 每一个好看，我都当成了喜欢”。对于星球里写分享的星友们 来说，你的每一个点赞，他们都会当成荣誉，会像小时候被 老师奖励一颗五角星那样开心。

有一个概念，叫情感账户，是指人际关系中相互信任的一种 比喻。我们将人际关系中的相互作用，比喻为银行中的存款 与取款。存款可以建立关系，修复关系。取款使得人们的关 系变得疏远。

我们每个人都在星球开设了自己的情感账户，一起组成了一 支成长基金；你看完星友的分享后觉得不错给个赞，就是往 他的情感账户里存钱，每个人情感账户里的钱存得越多，我 们的成长基金就越稳健，大家的成长收益也会更高。

你传递出去的力量，终究会以另一种形式回流给你；希望我 们都能成为既能自己发光，也能为别人点火的人。

感谢星球有你，一起互相成长。如果觉得不错，给我们的成 长基金点个赞吧【手动比心❤】
