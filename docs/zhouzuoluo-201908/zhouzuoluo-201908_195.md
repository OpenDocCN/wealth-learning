# 『怎样通过团购锁客

『怎样通过团购锁客？』

晚上好 与星友们分享丹儿的第 71 篇原创 7.4

————————————————————————

表姐今天发来一个其他店在做的美甲活动，

49.9 元可以做一次美甲＋足甲＋面部清洁＋美睫。我一看这么 便宜，手工费都赚不出来，就想着跟着去看看，看看人家是 怎么做项目升级，促进办卡的，顺便看看这么便宜的项目她 们是怎么做的。

【操作流程】

①活动中 49.9 的四个项目只能做两项（应该是想让顾客二次进 店促进消费）

②选色卡，只有四十多个不常用的颜色。不太能选的出来。 同时拿出来的还有几个色板，直接说选其他色板的颜色要加 钱。

③选睫毛，直接说团购的是最差的睫毛，持久度不高。建议 选貂毛。

④睫毛和足甲同时做，泡脚时才告诉顾客磨砂去死皮需要另 外收费，睫毛用的眼贴也是撕开后和顾客说需要加钱。

【看到的不足】

①服务态度：给顾客的感觉就是你就是团购的，我不用给你 好的态度，给顾客传递的信息也是团购用的所有产品都是最 不好的，想做好的就要加钱，尤其是我姐说不加钱之后服务 态度更差了。

②服务细节：接触顾客前手没有消毒，工具也没有消毒，也 没有提供水，过程中没有询问顾客是否有不适感。

足甲也没有用分趾器，做完后也没有检查，更没有涂指缘 油。

我询问美甲师店里 280 元的胶也是这个操作流程吗？她说是 的，只是胶不同，我就想怪不得没人做贵的胶了。

③销售话术：想做项目升级时，她是这么说的“团购的胶最不 好了，你可以看看店里其他的胶，加个几十块钱。”

睫毛项目升级她是这么说的

“团购的睫毛很硬，持久度不高，你用貂毛吧，毛很软，持久 度还高，就是贵一点。”

还有很多不足，环境，卫生之类的我就不说了，重要的就是 这三方面。

【引发的思考】

①服务态度：拓客低价吸引来的顾客，目的是什么呢？就是 让她们体验，即使第一次进店不消费高项目，她觉得好了第 二次还回来这时候价格自然上来了，还会推荐好友来。

她们的服务态度，就是把顾客往外推呀，给我得感觉就是， 你就来这一次，不做项目升级，以后别来了。给我的感觉就 是团购用的东西都是最差的，你不用好的就说明你消费能力 不行，我之后的态度就会变。

我们店对拓客来的顾客，即使是免费的项目，服务也是很热 情的，就是要让顾客感觉到，即使你不花钱做免费的项目， 效果也很好，那么花钱的，贵一点的效果就更好了。而且第 一次我们不会强制推销，只要让顾客感觉到效果好，服务好 就可以了。

过程中会给顾客讲一些专业，培养二次进店的习惯，因为只 要人来就有机会。

②服务细节：做的真的是比较粗糙了，如果高价位的胶也是

这么做的话，价格肯定是做不上去的。

首先消毒就没有做到位，工具很多人都在用，很容易交叉感 染，我看美甲师剪死皮的手法感觉分分钟要把我姐剪出血， 事实证明真的出血了。

③销售话术：她们也比较注重对比，关键是总说团购的胶不 好。贵的胶哪里好，没有说出来，只说是环保胶。

我们店里的团购胶是 68 的，一般来的顾客都会升级 98，或者

198。我们是怎么做到项目升级的呢？

我们不会说团购的胶不好，也有 80 个颜色，很漂亮，只是适 合秋冬季节。夏天来说颜色偏深，所以一般顾客来选胶，我 会拿三个色板给她看，并且介绍夏天更适合哪个色板。

贵的胶说明为什么这么贵，比如是日本进口的，成份是树脂 成分的，树脂成分是烤瓷牙里也有。因为烤瓷牙是放嘴里 的，能用这个成份说明它很安全，以后会说一下胶的属性， 色泽饱和度的对比，告诉顾客贵的胶为什么贵。

一般这一套说法下来，很容易项目升级。也不会让顾客感觉 团购的胶不好，只是会让她更想做好一点的胶。

＊＊＊＊＊＊＊＊ end ＊＊＊＊＊＊＊

▼很多店现在都有团购项目，很优惠，那么我们做团购的目 的是什么呢？就是想吸引顾客进店，体验！让顾客下次还想 来。次数多了就可以变成会员，甚至推荐好朋友一起过来。

现代社会年轻人消费居多，他们最注重的是什么？就是体验 感。

有很多团购的顾客，虽然抱着占便宜的心态来，来一次下次 就不来了。但是能否转变她的想法还是取决于我们，如果顾 客一进店就伤客，让她下次再也不想来，正好让她觉得他的 想法是正确的。那办团购也没有什么意义，耗费了时间成本 和人工还不挣钱。

让团购的顾客感受到我们和其他店的团购不一样，才是出 路，现在市场那么乱，什么价格都有。也有很低的价格，但 是只要做的好，价格还是可以收上来的。

通过团购吸引顾客进店，提升顾客体验感，培养出挑剔的顾 客，让她体会专业的感觉，这样她去别的店是做不了的，很 多能将就的顾客是因为她没有感受过好的，她也不会看，她 觉得做的都差不多。我们要做的就是教会顾客看不同。这样 价格才能收上来，也能锁住顾客。

通过这件事以后，我想我应该多去做做市场调研，看看同行 中哪些做的很好的学过来，哪些做的不足的我们要做的更 好，这样才能更有竞争力。
