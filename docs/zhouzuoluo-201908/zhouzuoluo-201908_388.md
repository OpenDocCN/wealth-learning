# 《好心被当做驴肝肺

丁丁 : 《好心被当做驴肝肺？只怪你情商不高》

上午，我正带着孩子在图书馆看书，手机收到一条微信：“亲 爱的，今天在家吗？找时间来我家把娜娜（她家孩子）的玩 具钢琴和一些乐高积木搬走吧。”

一看，是孩子同学的妈妈，和我们住同一小区。我知道她家 最近打算重新装修，看样子是要把家里的东西处理一下。

我迅速地在脑海中检索了一遍，是不是曾经表达过想要娜娜 玩具的想法，或者她询问过我是否有类似的需要。但是，没 有找到任何相关的信息。

我想了好一会儿，给她回了微信，感谢她能想着我家孩子， 然后说家里玩具已经太多（这个是事实）都快放不开了，委 婉地谢绝了她的好意。

这个事情，多少有点令人不愉快。 就像不饿的时候强行被人塞了一嘴的红烧肉，明明吃不下，

不吃又觉得负了对方的好意；执意不吃吧，没准对方还会怀

着“好心当做驴肝肺”的想法来揣测你。

总之，不舒服。 本来是件挺好心的事儿，怎么到最后就变了味儿呢？说到

底，还是跟情商有关。

1、情商高的人，不会把自己的需要当别人的需求

送别人东西，是件好事，但不意味着可以简单粗暴。 送之前，先询问下对方是否需要，是一种最起码的尊重。不

是你认为别人可能有需要，别人就一定有需求。

孩子同学家条件不错，她妈妈想要给的玩具之类应该也不会 太差。或许考虑到都是小女孩，对玩具的兴趣方面比较相 近，所以同学的妈妈认为我会想要这些玩具，直接让我过去 拿。

我需要吗？不，我真的不需要，家里玩具已经多的快堆不下 来，最近我一直在考虑如何稳妥地处理一批玩具，好让家里 腾出更多空地儿来。

退一万步说，即使我有心想要那些玩具，这种没有任何铺 垫、类似通知一样的微信，也会让人心里产生一种抗拒。

毕竟，吃人嘴软，拿人手短，即使拿的是别人要处理的东 西。

在这种问题上，情商高的人，往往会先询问对方的意见，然 后找恰当的时机将东西不动声色地送出。

2、情商高的人，恰当的时机做恰当的事 把自己不需要的东西送给别人，表面上看是帮了别人，实质

上别人在帮你。

为什么？因为要送出去的东西，是你不再需要的，放在家里

只有占地儿的功能。

你当然可以选择扔掉。但是，它看着这么好这么贵这么新， 扔掉它，你实在于心不忍，你决不能容忍自己如此浪费。

有人接收它，既解决了占地儿的问题，又消除了你心中的不 安，你说，占便宜的到底是谁？

所以，送人东西这事，得找对时机，你送的开心，别人拿的 舒心。

我有个邻居，有次接孩子回家的路上碰到她，聊着聊着，我 说起孩子特别喜欢那种超级仙的裙子，我打算给她买一条， 但是平时幼儿园不让穿（因为不方便运动），可能穿的次数 不会太多。

邻居说，买太浪费了，真的穿不了几次。她家孩子去年买了 一条类似的裙子，就穿了 2-3 次，今年就穿不了了。然后问 我，对了，好像你家孩子今年差不多能穿，如果你不嫌弃的 话，就送她吧。

我当时觉得很好啊，这样大家都不用浪费。没想到回家后， 邻居很快就把裙子给送了过来，还特意找出白裙子上一块很 小的黄色部分指给我看，解释说，这个实在洗不掉了，希望 我不要介意。

其实那块黄色污渍很小，不仔细看根本不会注意到。最重要 的是她的态度，让人很舒服。

3、遇到情商实在低的人，远离为妙 S 老师曾经讲过一个事，说他有个高中同学，有次搬家时有一

大堆时书需要处理，找到他，说我这有什么什么书，你应该 比较感兴趣，要不快递给你？

于是 S 老师想，我得助人为乐啊，于是便要了这些书。很快， 书就寄到了，但是，快递费到付。

钱虽不多，心却被伤了。 后续是，该同学还多次关心地询问，书好不好看啊，对你是

否有用啊？

这种低情商的人，离远点比较好，否则一不小心，下回还有 可能被伤。
