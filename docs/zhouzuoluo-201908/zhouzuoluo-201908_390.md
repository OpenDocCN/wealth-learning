# 《高小羊：写作 3 个

《高小羊：写作 3 个月，我接到一份 5000 元月薪的代约稿兼 职》

距离我真正开始写作，时间已经过去了 3 个月。其实我是从去 年十一月份开始就接触这个行业，当时还参加了一个训练 营，不过这个训练营是教我们学习公众号平台，并不是教我 们写作。

后来买了一些写作课程，有粥老师的，spenser 的等，开始断 断续续的写作。有时一个月更一次文，有时一周更一次，还 有每天更一次文，但都是不持续的，拿起来又丢下，之后又 捡起来。

后来无法忍受不上班的生活，就找了份工作。从三月底开始 工作后，同时就暗暗发誓，写作也要重新开始。我凡事都喜 欢追求仪式感，所以我把四月份当做人生新的起点，工作的 起点，写作的起点，以及生活的起点。

从四月份以来，我的写作正式步入了正规，虽然没有每天在 公众号上发文，但是不发文的当天，我会在写作软件上练习 写至少 500 字。

这前前后后的写作时间加起来，大约有 3 个月。 上周三，之前和我一起参加训练营的一朋友突然找我，说我

写的文章不错，想把我推荐给她认识的一家公司，这家公司

想找一个代写公众号文章的人。

她将我写过的文章推给了这家公司，获得了公司的认可，就 这样，我和这家公司取得了联系，并开始了后面的洽谈。今 天收到了协议，目前协议还在校对中，不出意外，下周一就 是正式开始的日子。

这家公司要求每个月提供 10 篇原创文章，每篇 500 稿费。事以 秘成，按照这个逻辑，其实我不应该这么早和各位星友分享 这个消息，因为还不确定最终能做多久。但后来一想也没什 么关系，最坏的结果就是，我中间写的文章没达到这家公司 的标准，然后解除协议。

今晚下班后，从健身房回来，就开始在检查协议，然后写了 这篇分享。

从写作到目前，这 3 个月，我从一开始的不会写，到会写但写 的很僵硬，到现在的流利，全部认证了那句话：先相信，后 看见，时间是最好的证明。

最另我开心的事是，当时我朋友将我的文章推给那家公司 后，那家公司给我的评价是：你写的文章我很认可。

这句话让我开心了整整一天。 写作这条路确实不好走，但走下来的人到最后都会觉得这条

路太好走。

写作这 3 个月，我最大的感受就是你的文章必须要有自己的特 色。

一篇文章，我不署名，要让读者一读，就知道是高小羊写 的。别人想洗稿，都会洗的四不像。因为文章里面糅合了太

多我个人思考的东西和亲身经历。

就像粥老师之前分享的李叫兽的回答：任何商业成功的关 键，都是要有持续地创造并提供差异化价值的能力。但我见 过的绝大多数年轻人，他一直在想如何变得更优秀，同样的 考试我如何考得比别人高，同样的岗位和工作我如何比别人 超一分。但是这就像那些超市比谁价格低一样，这样做的本 质上你是没有办法创造超额利润的。

所以你必须要给自己的作品创造附加值，差异化，这样你才 能顺着这条线顺利的走下去。
