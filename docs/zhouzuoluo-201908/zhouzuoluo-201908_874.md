# 《人生只有一个课题

《人生只有一个课题——战胜心中的恐惧》

大大小小的恐惧，就是大大小小的山坡，为了到达更远的地 方，突破跟大的边界，翻山越岭，是我们一生都要做的事情

——第 161 天的星里浣

梁宁说，恐惧是一个人的边界。看到火，你就不会伸手；看 到风高狼大，你就不会出海，因为这些让你感受到恐惧。所 以，恐惧会困住一个人的手脚。

再回过头来看今天看到的这个观点：人生只有一个课题—— 战胜心中的恐惧。为什么说一个人一生要解决的问题，就是 战胜心中的恐惧。

因为你不战胜恐惧，你就只能停留在原地，甚至会后退。更 要命的是，恐惧会让你无法拥有很多可以得到的东西，失去 很多能够拥有的机会。

记得小时候，我很羡慕那些能上台演讲的同学，觉得这是一 种自带主角光环的能力：你往台上一站，一开口，全场所有 的目光都会望向你，大家也都会认识你。我也很渴望成为全 场的焦点，被大家认识，可是我恐惧上台演讲，所以我就无 法得到这种能力，也失去了展现自己的机会和舞台。

恐惧，会随着时间的增长拼命累积，到最后成为一座横亘在 你面前的大山，成为更大的恐惧，给你造成更多的麻烦。

就好比我从小就一直害怕公开表达，所以我一直活得像个小 透明，也失去了很多机会和可能，不能体会那些别人习以为 常的经历。比如，去 ktv......（仅有的几次都是被拉着去的）

直到今天，我还是不太敢公开表达，虽然已经在努力克服 了。但当年的那份恐惧已经长成了一座大山，我要费更多的 力气才能翻越它。

如果你也有恐惧的东西，最好趁早去翻过它，不要想着来日 方长的自愈，不要等它变成一座大山。

大大小小的恐惧，就是大大小小的山坡，为了到达更远的地 方，突破跟大的边界，翻山越岭，是我们一生都要做的事 情。
