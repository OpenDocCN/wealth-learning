# 新认知：靠知识写作

新认知：靠知识写作赚钱，本质是利用信息差。

01

信息差的概念

也许你以为，信息差仅仅是，你知道的比别人多。 其实，在一般情况下，信息差仅仅是 A 领先于 B 的一个外在表

现形式。

本质上的原因，是 A 比 B 先知道了一个知识，然后，A 比 B 先去 实践，总结，最后分享出来，变成 A 自己的知识。从而达 到，A 在某方面，比 B 领先。

于是，在这方面有需求的人，比如 B，如果他想进步的快一 点，于是就付费找这方面比他专业的 A。

举个例子，粥左罗老师写作很厉害，这是因为他比我们先知 道了写作的套路，于是先去实践，于是先达到专业 的结果。

当我们想学习写作，快速上手写作，于是就花钱买了粥左罗 老师的写作课程。

02

如何与别人产生信息差：阅读

如何阅读？

①针对某主题，大量，迅速地输入

想扎根某一领域，在一开始最重要的是建立对该领域的底层 认知框架。

于是，我们大量地收集该领域的书籍。 然后，快速的阅读，迅速的建立认知。 如何阅读呢？ 预览目录，序言，了解书籍的架构。

快速扫读，对于感兴趣的关键字部分，图片部分，以及结论 部分，不停下，先做好标记。

跳读，最后只读标记了的部分，读完第一个，就直接跳到下 一个，不用管中间的。

②限时和计时 规定每天放在输入的时间，我一般是两小时。

在你进行输入时，最好用番茄钟计时。第一个目的是为了保 持专注，第二是可以知道自己看完一本书要多长时间。

③贯彻信息洁癖 一本书，并不是说要一字不落的看完，才算是看完。 看书的本质是为了获取知识。

只看对自己当下最有用的部分。

如果一本书讲的很专业，但我们理解不了，那么也不必看。 一些貌似很有用的知识，但是对我们现在没什么卵用的，也

不必看。

④ 精读与速读切换

精读的目的：为了掌握你未掌握的新事物，所以这部分要仔 细读，做好笔记。

速读的目的：对于简单易懂的或已经掌握的，我们侧重于前 后之间的逻辑框架，搞清楚这些部分与前后，与整体的联 系。便与建立起完整的知识结构。

03

最后再掏心窝子的说一点

人的进步，就是不断地学习新知识，掌握新知识的过程。 如何判断已经掌握呢？

能付诸于实践，能教会别人。

如何赚钱呢？ 流水生生不息，新人年年有来。

总有不会的人，他出钱请你，你出力教他，这便是钱。
