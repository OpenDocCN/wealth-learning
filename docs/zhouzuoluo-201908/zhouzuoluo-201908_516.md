# Day141《会铸

《会铸造社交货币的人，大部分都是有钱人了》

学会疯传，让我们变得更有钱。《疯传》第一章是讲社交货 币的，首先你是怎么来定义社交货币这个概念的？

货币是交换权的契约，反映的是个体与社会的经济协作关 系。社交货币同样是交换契约的一种，只是反映的是个体在 社交网络中的影响力。

人活于世，不只肉体上有生存的需要，精神和情感上也有生 存的需要。货币满足肉体生存，社交货币满足情感需要。当 然两者也是不可割裂、彼此关联的。

社交货币可以用来推广产品或服务，也可以推广自己的人设 或观点，可以帮助人们打造良好的印象和在社交圈的影响力

。

乔纳·伯杰在书中提到了三种铸造社交货币的方法：第一，发 现非凡之处；第二，利用游戏机制；第三，让人们觉得自己 是内部人士。

1、谈论非凡之事 非凡之事是指不同寻常且值得关注的事情。 因为人天生就有好奇心，对不寻常的事有出乎本能的关注，

进化心理学也证明了这点。所以发现和创造一些与众不同、

鲜为人知的事，就更有可能获得别人的关注。

不论是广告创意，还是一个文章标题的打磨，甚至只是一个 新鲜的知识，利用人的猎奇心理来传播，就能形成社交货 币，往往会有出奇制胜的效果。

平时多搜集一些涨知识的信息，比如说，皱眉比微笑能够燃 烧更多的卡路里；蚂蚁能够举起超过自身体重 50 倍的东西等 等，还有金句、段子、奇闻异事等，多储备一些这样的社交 货币吧。

2、利用游戏机制 游戏机制背后对应的是人的心理，甚至是所谓的一些人性的

弱点。利用游戏机制，本质上是利用人性，这么说似乎不太

光彩，但事实确实如此。

偷偷做，别说破。这里面套路很深，我就不多说了。 3、成为内部人士 这一点利用了稀缺性，让人觉得自己与众不同，获得了专属

待遇，比如说机场的贵宾专属休息室。就跟一首歌唱的那

样：特别的爱给特别的你。

人天生喜欢比较，当比别人更有优越感时，内心就会觉得舒 服，相当于在精神上生存得更优渥，反之亦然。

那就把优越感给别人吧，你也能源源不断铸造出社交货币。 别人精神上舒服，你在赚钱上舒服。
