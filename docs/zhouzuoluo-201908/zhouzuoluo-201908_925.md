# 大家好，晚上好，我

贝姬狸 : 大家好，晚上好，我是贝姬狸

—日更 38 天分享

《再累，我也要做个温暖的人》 蔡康永：我劝你冷淡一点，做个温暖的人太累了。 可是，再累，我也要做个温暖的人。

以前的我，个性淡漠，比如朋友圈很少给人点赞，比如在一 个群体里，只是淡淡的围观，不曾参与一二。

有一天突发奇想，我想做一个能带给别人温暖的人。 于是，我在朋友圈积极点赞。 于是，我融入群体，积极去发现别人的优点，去称赞、去认

可。

演讲训练营最新的一期，我申请当了助教，是义务的。 我每天陪伴我的学员，经常跟他沟通，监督他作业完成。 如果收到他的打卡作业，我会在吃饭的途中，跑到一个安静

的角落给他进行点评。

在平时，我会抽出时间，尽快为他点评，生怕耽误他作业评 精选。

听/看他的作业，我会把要点记在纸上，生怕漏掉一点什么。

我尽我所能地去帮助他。

没想到，得到我的学员认可，他向其他老师表扬我认真负 责。执行助教把此信息发到我们助教大群上，那一刻我温暖 无比。

你看，我温暖别人的同时，别人同样温暖着我。 今天收到我们演说俱乐部一个姐姐的礼物及一张亲手写的贺

卡。

平时我在群里积极与他们互动，互相分享，互相鼓励。 贺卡上的一字一句，温暖地打在我心窝。 我真的觉得无比感动。 我在另一个心理学训练营里，当上了一名赋能天使。 赋能就是给予他人力量。 我每天都会点评一位学员的作业，她因此受到很大的鼓舞。 而我，也倍感开心，因为能帮助到别人的感觉，真棒。 所有人都渴望被看见，我们只是稍微停一停，一句鼓励、一

句认可就能让人感觉到温暖，何乐而不为？

不要惧怕做个温暖的人，你献出温暖，就得到温暖，甚至这

温暖是加倍返还。

一个温暖的人，就像一道微光，虽不能照亮大地，但能照耀 人们的前行。

这也足够了。

轻轻点个赞，温暖我心窝。
