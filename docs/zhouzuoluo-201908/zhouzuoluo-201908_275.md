# 《意外来临之前，希

《意外来临之前，希望我们有所准备》

当下看到很多“活在当下”的年轻人，要么是月光族：每个月 等着工资到账还花呗，剩下的一点钱半刻都不能闲着，马上 找代购买面膜买精华买漂亮裙子买买买。要么是年光族：省 吃俭用积攒一年的工资来一场说走就走的旅行，飞到马尔代 夫、法国巴黎拍个照打卡，游玩几天回来，钱包空空不过心 满意足，感觉生活过得特别精彩。

最令我震惊的还是屡屡出现的网红爆款哄抢：星巴克猫爪 杯，价格被哄抬到五百多元一个；兔子 T 恤一件一百多元在服 装店内进行“叠价”交易，即 a 顾客抢到了衣服，直接在店内加 钱兜售给 bcd 等顾客，带着他们前台交钱后再给 a 佣金。

但是，一旦什么意外来临：如公司倒闭忽然失业；家人忽然 生病需要医疗资金等，他们毫无准备，惊慌失措，口中唉声 叹气：上帝为什么这么不公平？偏偏对我下毒手？

殊不知，意外无法预测，我们应早有防备。

01 比赛出现重大意外，仍奇迹般打破记录 美国游泳名将菲尔普斯是个传奇人物，他拿过 23 枚奥运金

牌，打破过 39 项世界纪录。

有一次重大赛事中，菲尔普斯的泳镜入水，眼睛无法睁开， 但他却从容不迫完成比赛，还拿到冠军并且打破了世界纪 录。

观众目瞪口呆，记者采访他为何能在出现重大意外时还能破 纪录，菲尔普斯轻松地回答：“这种情况我经历过无数次。”

这是为什么？ 原来，菲尔普斯的教练在日常训练中给他做过特别多的“设想

意外”状况：忽然断电、让他在漆黑的泳池中训练；故意往他

身上丢东西，甚至会提前弄坏他的泳镜，让他在水下无法睁

眼。

在这些意外情况下不断学习做好准备，真正遇到问题时，他 自然能够从容面对、轻松跨越。

02 买房子选好地段，为未来生活便利准备 今年五月底，广州大雨不断，一夜之间降水量惊人，第二天

醒来，有的人面对门前高处膝盖的水位傻了眼出不了门，有

的人门前道路干净宽阔，只是湿的路面提醒他昨晚下过雨，

至于雨多大，与他无关。

朋友圈疯狂转载一张图，白云区和越秀区相差一条街，路面 水位却截然不同：白云区水位高涨通行困难，旁边的越秀区 路面排水工程显然做得很好，没有任何积水。

原因是白云区房价一平方米均价 3 万，越秀区均价 6 万。

03 母亲突然得做手术时，我们不需要众筹 我一直记得，五年前我还在读大三的时候，我妈在姐姐要求

下来广州做第二次的每年全身体检。至今每每提起，我们都

庆幸做了例行体检。

因为那次，我妈查出肝的良性肿瘤。

当时报告一出来我们都吓坏了，医生让我妈继续做完几项深 入检查后告知我们不用担心，是良性肿瘤好在发现得快，只 要马上做手术就好。

那时候我没有参与金钱的筹备，因为我姐全部承担得失。所 有的医药费，住院，手术费全部都是我姐姐出的钱，后来听 她说只需要跟朋友借一小部分钱用来购买药物，大部分的钱 她的存款能够负担得起。

注意到重点吗？——存款。 不同于“月光族”和“年光族”，姐姐一向都很节俭，即便在她工

作八年的以后，从月薪几百到几千，她从不舍得多花一分

钱。当意外忽然降临时，我姐姐作为家里长女，从容不迫协

助我爸爸，陪我妈妈轻松渡过难关。

试想，如果她没有任何存款，当时家里还有五个在读书的弟 弟妹妹，基本没有多余的钱可以周转，爸爸是否得在全村挨 家挨户的借钱？更有甚者，我们是否要在网上发出众筹水滴 筹等爱心呼吁？

生活不易，希望我们一辈子都没有意外发生。 万一不幸发生了，希望我们有所准备。
