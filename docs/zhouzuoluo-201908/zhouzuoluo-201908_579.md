# 《粥左罗：婚姻是一

《粥左罗：婚姻是一项极其重要的投资，很多人做得不够》

我有个同学去年结婚，我回山东老家去参加他的婚礼，后面 才知道这个婚结的很有意思：婚礼办了，结婚证还没领。

为什么？岳父那边有个硬性要求，必须买了房子，才能领 证。

我说这也太奇葩了，你对象怎么说。 他说：还能怎么说，她听她爸妈的。 这是我实在不能理解的事情，怎么都无法理解，你们是两口

子，你为什么难为自己男人呢？父母有那样的要求，可以理

解，爱自己孩子嘛，但是你应该是站在男朋友这边的，将来

一起过日子毕竟也是你们两个。

后来这个同学买房时还打电话借钱，他跟很多同学借了钱， 他的爸妈跟很多亲戚借了钱。

何苦呢？买房子当然很重要，但超出自己承受范围去买，后 面十年二十年未必过得更好。

金融巨子摩根说过一句话：一旦婚姻投资得当，你的事业也 将随之达到高峰。

婚姻这个投资实在太重要了，仅次于自我投资，其它投资的

重要性都在其下。

因为这个人要跟你每天生活在一起，要生活一辈子，人活着 可不只是吃饭喝水，一个人的想法、三观、思想、智识、性 格、情绪等等，才是真正影响生活质量、人生质量的因素， 如果这个不匹配，甚至很多时候拉倒车，你怎么可能人生幸 福、事业成功呢？

婚姻是一项需要慎重，最不能凑合的事，一个人越急着脱单 的时候越要注意，这时候很容易忽略你心里的一些标准，标 准不是指高低，而是是否合拍，在一起后是不是互相促进，1

＋1 大于 2.

选择很重要，后期经营同样重要，包括处理一些关系，比如 跟双方父母的关系，我比较赞同夫妻关系第一，大于父母关 系，也大于跟孩子的关系。

我举个我自己的例子。我女朋友很优秀，但再优秀的人也不 是完美的，我记得女朋友第一次去我家时，我妈就偷偷跟我 说了一个不足，这种指出不足的心态不是说要改正，而是 说“你可以再选选，她好，你可以找更好的”，我当然非常理 解爸妈对我好的心态，但我还是非常严肃的跟她说：我不允 许你说她不好，一点都不可以，为什么呢，因为我们已经确 定关系了，也就是我们现在已经是一家人了，你跟她是一家 人了，你怎么能背后说她不好呢，而且你说她不好，就是说 我不好，她是我选的，将来也是跟我生活的，不是跟你们。

从那以后，我妈就明白了这个道理。 同样，我也跟女朋友说，回家要多顾及老人，她们没什么文

化，一辈子生活在农村，生活习惯、说话方式啊都那样，我

们只要记住一点就行：她们爱我们。

我去她们家也一样。所以直到现在，我们从来不用花心思用 来处理婆媳关系之类的。

其实很多人的行为我是不理解的，比如女朋友向着自己爸 妈，男方父母也有自己的小算盘，婆媳之间也不融洽，他们 大概忘了最重要的一个宗旨：根本目的是让两口子幸福。一 切违背这个宗旨的不做就完了呗。

当然经营婚姻最重要的是经营两个人的关系，一辈子的太 长，人都在变化，经营的目的就是在长期变化中还能匹配， 除了生活上的互相照顾，更重要的是脑袋层面的，是不是能 互相支持，互相理解，互相促进，是不是每一方除了自己变 好，都还挂念着怎么让对方也每天变好一点。
