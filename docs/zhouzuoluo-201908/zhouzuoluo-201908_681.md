# 《从小就知道要好好

《从小就知道要好好学习，但你真的会好好学习吗？》

今天向星友们推荐书籍啦。推荐的是哪一本书呢？是成甲写 的《好好学习》。我们从小就被教要好好学习，但是我们真 的会好好学习吗？当然，这本书就教给我们真正的好好学 习。这是一本实用类的书籍。看完之后，其中有很多地方都 可以用来实践。我强力推荐这本书是因为这本书不仅仅能够 带给你认知升级，而且还能给你行动，然后改变你的生活。 这正是学以致用的典范啊。

在本书中，作者并没有讲解多少知识管理。反而讲到了知识 和知识管理的误区。本书中最重要的概念就叫临界知识。

现在我就简要说一说书中的内容对我有启发的地方。 第一点是关于知识和知识管理的误区。 能够提升认知和改变行动的信息才叫知识。知识管理并不是

把你从碎片信息收藏的干货以及书上的知识摘抄放到笔记里

面，放到仓库里面就完了，这种收藏以及摘抄叫做管理仓

库，而不是管理知识。

所以做了一上来就给我破除了这个误区。然后呢？作者又劝 告我们：要具有绿灯思维。就是不要看到与自己认知不符合 的东西、自己不懂的东西以及突破自己认知的东西就否定和 质疑。抱着这种想法是很难提升自己的思考能力的。

第二点是我们要想洞悉事物背后的规律。想要探索事情的底

层原因，就需要用到临界知识。临界知识是最底层的最通用 的知识，是能够跨行业的底层知识。比如说数学里面的幂函 数：其实用在经济学里面它就是一个复利效应，用在生活 中，它又是一个滚雪球。

像这种底层的知识，非常重要。几乎可以一通百通。我们洞 悉事物的规律，就靠它了。混沌大写的李善友所说的第一性 原理就和他类似。查理芒格所说的思维模型背后的底层东西 也和临界知识有类似之处。你们对于您见之时感到心动吗？ 他可以探索规律并且能预测未来。至于怎样预测未来，且看 书中讲解喽。

第三点是应用临界知识。比如黄金思维圈（what、how、 why）可以用在很多地方。你可以把它用来解决问题。也可以 把它用来写作，你还可以通过多问为什么来提升你自己的思 考能力。作者举例：得到 app 为什么这么火？

在表面层回答的人说用户量大呀。这是几乎不用动脑思考的 问题，所以这个回答基本上也没什么意义。 在技术层面的人回答得到用了很多技术，而且赶上了风口， 赶上了互联网的风口。而且罗振宇自己也有点有有两把刷 子，会带团队，找到了人才。

在底层的解释是几个临界知识：用户价值第一、复利效应、 从众效应、规模效应、品牌效应通过这几个底层知识。把得 到火起来的原因洞悉的很透彻。这就是我们所谓的认知很 深，思考很深。原因就在于临界知识的灵活运用。

总之，这是一本集认知与实践、信息密度极大的书，我现在 推荐给星友们。这本书我已经看了三遍了。并且做了近万字 的笔记，还写下了近 5000 字的思考。好书就是这样，值得你 反复咀嚼与消化，耐读又耐看。
