# 《选择工作时，很多

《选择工作时，很多人都忽略了这一点》

在社群里，时不时会小伙伴问我，关于工作选择的问题。大 家关心的有很多共同点，比如说：这份工作的收入怎么样， 是不是适合自己，有没有成长空间…..

但很多人在强调个人成长诉求的时候，容易忽略老板对你的 需求。

有的人可能会说：“我为什么要关心老板的需求？” 1、个人的成长诉求和老板的需求，有重合的部分、也有矛盾

的部分。

即每份工作的内容，不可能都是你想要的，比如说大量的重 复性的工作，或者对提升自己核心技能毫无帮助的部分。

我一个朋友一心想提高自己的写作能力，他去应聘一家公司 的编辑，就只想写稿子。事实上他可能还得负责公众号的运 营、推广、商务合作。

“写稿子”是符合他成长诉求的部分，他很愿意做，其他事情 则不符合他的意愿，但老板要求做，他就不太愿意做。

2、即要考虑个人诉求，也要分析老板需求。 只考虑自己的诉求行不行？可以。只不过完全符合自己预期

的工作，不太可能存在。

为什么这样说呢？老板请你是给工资的，他肯定首先站在自 己的立场安排工作，而不是以是否符合你的成长预期来考 虑。所以，一份工作里无法避免的会有你不太喜欢的部分。

应该怎么办呢？入职前你就得主动分析未来 1-2 年双方的主要 诉求。除了你自己，还得了解老板到底更希望你写稿子，还 是做运营，他的需要跟你自己的规划重合的部分有多少。

重合的部分越多，这个工作就越适合你，符合自己的预期。 总之，工作是一种双向选择，选择的时候，除了考虑自己的

诉求，也要考虑对方的需求。双赢，才算彼此最好的选择。
