# 坚持每日输出一个月

坚持每日输出一个月后我发生了什么样的转变？

① 最大的感受就是感觉生命被延长了，每一天的进步轨迹都 清晰可见。以肉眼可见的速度在进步。好像每一天都认真活 过。这一个月跨度之大，比以前浑浑噩噩过一年的感觉都令 人回味和印象深刻。

② 感觉越来越清晰了。不管是对生活，对自己，还是对事 业，对感情，各方面的思考都有了全新的体会和感受。就像 是划了一条直达目标的线，我只需要顺着这根线去不断努力 靠近目标就好了。

③ 因我的坚持而影响了一部分人，他们会因为这些而加我为 好友，私信说每天看我写的东西感觉很好，很给人力量；还 有的小伙伴说每天坐等我更新。还有的小伙伴给我加油，鼓 劲，说看好我，我一定可以的！这些都让我很感动，也很幸 福。

希望持续带给大家正能量，一起互律成长，成长路上不孤 单，至少还有小白呢！

不知道你们坚持了多久了呢？又有什么样的改变和感受？欢 迎分享交流哈！

一个认真生活的平凡人 白羽曳
