# 《凡是我赶不上的，

《凡是我赶不上的，我就去未来等他》

越对未来有信心，越对当下有耐心——第 149 天的星里浣 每年的《时间的朋友》跨年演讲都是一个金句频出的现场。 2016 年：爱上一个人的感觉，就好像突然有了盔甲，也有了

软肋。

2017 年：岁月不饶人，然而我也没能饶过岁月。

2018 呢？目前还不知道，但我想每个人心里都有自己的年度 金句。对于我来说，今年这句“凡是我赶不上的，我就去未来 等他”，就是我的年度金句。同时，这句话也让我想起了粥老 师的一篇文章

因为“凡是我赶不上的，我就去未来等他”，所以我们要“越对 未来有信心，越对当下有耐心”。

毕竟我们绝大部分人的一辈子，都至少有 70 年，几年的失 败，几年的成功都不能说明什么，谁又能保证未来没有可能 和变数呢？谁都不能保证。

就在半年前，我还是每天玩好几个小时游戏的咸鱼，赶不上 我的同学，赶不上我的消费水平，也赶不上时代前进的脚 步；成长靠顺其自然，工资靠通货膨胀，人生靠时来运转。 我当时没有看到未来，也不知道其实时间就是一个人最大的

逆袭资本。直到今天，我回过头去看那时候决定改变的自 己，每天进步的一点点，我才发现，我已经在今天等到了曾 经赶不上的很多东西。

王兴曾经在接受《财经》采访时说：太多人关注边界，而不 关注核心。

我想，这个边界也可以理解为那些赶不上的东西。我们大多 数人只关注自己面前那些赶不上的东西，看到的边界也只是 它们，而很少去关心自己的核心——能力，更没有想过未来 的可能，所以我们被“赶不上的东西”形成的边界困得死死 的，既没有对未来的信心，也没有对当下的耐心，满眼看到 的只有“有限的资源”和“无望的前路”。

最后，我想分享一个书评，是王兴读完《有限与无限的游 戏》后写下的：

有限游戏在边界内玩，无限游戏却在和边界，也就是和规则 玩，探索改变边界本身。实际上只有一个无限游戏，那就是 你的人生，死亡是不可逾越的边界。与之相比，其他的边界 并不是那么重要了。

如果以那些“赶不上的东西”为边界，当然只能看到有限的资 源和无望的人生，而你以“未来的自己”为边界，你看到的一 定是有无限可能的，充满希望的人生。
