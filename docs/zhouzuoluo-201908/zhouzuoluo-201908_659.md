# Day115《思考

《思考越深，践行越狠，成长越快》

最近听大咖的分享，发现几乎场场分享都干货满满，而且思 考越深，拓展越广，收获就越大。

1、关于如何分享 做行家，必学做分享；做分享，要学大咖。 首先，结构要清晰。 以总分总为主，每次都分几个重点来讲，深入浅出传达要

旨。要打磨到小白级别都能听懂，高手级别也有启发。

其次，聊天式分享风格。 这让听众感觉亲切，就跟听朋友聊天一样，不端不装。语调

语速都拿捏得恰好。而我在最近的几场分享中发现自己语速

太快，如果分享的东西再干一点，大家的接受度就会很糟。

第三，个人专属道具的使用。 孙悟空专属道具是金箍棒，在三国演义电视剧里面，诸葛亮

的专属道具是那个扇子。那咱们在成为行家的路上，也需要

有自己的专属道具。比如说，你的口头禅。

设计自己的专属道具并在每次分享中使用，可以建立强感 知，虽是小小的细节，但对个人品牌的价值不容忽视。

那我以后做分享的专属道具是什么呢？可能先讲个跟分享相 关的段子吧，哈哈。

最后，重点分享自己的经历。 原创思想是很难的，但经历却是独家。分享自己独一无二的

故事，可以更好建立感知和信任。

可以按照“开始+结果→经历→干货”框架来分享自己的经历， 基于受众思维，一方面能引起兴趣，抓住受众注意力；另一 方面也更容易让别人获得启发，做到听得懂用得上。

2、关于如何成事 做行家，要成事，离不开智囊团。

首先，找到合适的路径链接行家，建立自己的智囊团。

除了在各大付费社群和优质人脉圈主动链接各领域优秀的 人，还可以通过更多行家平台去拓展。比如在行 APP。

我计划 2019 年每个月都至少链接一位学习领域的高手，与他 深入沟通和碰撞。

其次，学会通过提问高效请教。 除了听干货，还要主动询问他的故事，他是如何做到的；要

试着通过他去链接更多的学习资料和行家资源； 主动询问能

帮对方做点什么，甚至再预约下次的沟通，能把他慢慢变成 自己人就更好了。

最后，一定要及时反馈。 行家都是需要用户证言的，获得帮助后，一定要及时反馈，

写评价，写文章等等，不要吝啬你的感激和赞美。这个时候

你可以想想，有一天你作为行家为用户提供服务后，你希望

对方能有怎样的反馈？

思考越深，感悟越大；践行越狠，成长越快。祝大家早日成 为行家，我们来组智囊团呀！
