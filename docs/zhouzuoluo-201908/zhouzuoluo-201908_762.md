# Day95《和菜头

《和菜头：写作的真正压力，来自于完成一件事情的压力》

最近在翻看旧文。在日更 100 天之际，再聊聊写作这件事。 和菜头说，写作的真正压力来自于完成一件事情的压力。 尤其是当这件事情你几乎没做过，或者做的时候体验并不太

爽的时候，你每次面对这件事情往往都会充满压力，会本能

地逃避这件事情。

曾经我是逃避写作的压力的，后来写过一些零零碎碎的东 西，也算从无到有创造了一些所谓的作品，但如今动笔还是 会有压力。不是怕写不好，而是在未完成的过程中会持续地 煎熬，这个感觉非常痛苦。

所以，如今我考虑的并不是写多么好，而是首先要完整地完 成它。

现在我参加比稿写提案，也是在不断经历这种完成一件事情 的压力。

这件事情现在做起来依然很吃力，一方面没有足够的来自他 人可借鉴的经验，这不像写作，我读过非常多的好作品，起 码会有各种参考；另一方面我本身也没有多少经验积累，所 以会感受到恐慌，做起来也显得很笨拙。

同样我第一考虑的不是写多好，而是首先要完成它，然后再

优化。否则就会因逃避心理而被无限拖延下去，未来每当面 对这件事情的时候，也容易在巨大的压力下逃之夭夭。

“当完整地经历一次创造，而且是在一定的心理压力和干扰下 完成的。从而，你就拥有了创造者之心。”和菜头如是说。

对于写作这件事，不要想太多，专注于完成它，不断去体验 完成一件事情、创造一个事物的过程。让创造者之心先诞生 出来，然后慢慢成长，直至壮大，终有一天，写作这件事将 变得越来越得心应手。

重要的是，你要跟内心的压力达成和解。否则更大的可能还 是会放弃。

除了写作，再去做别的事情的时候，比如我现在写方案，可 以带着这种已经培养出的创造者之心和相对和谐的内心状态 去做，而不是想得太多，做得太少。

这样看来，写作真是我们普通人最容易上手的必备武器。它 帮我们做好了完成一件事情的心理建设，同时还训练了必备 的思维，让我们在做更多事情的时候，敢于去做、去尝试， 从而打开新局面，获得更多的经验和体验。

所以，马上开始写吧！ 你的经历，你的见闻，你的思考，甚至你的纠结和挣扎，都

是写作的素材和土壤，缺的就是一颗行动的种子了…… 2019-02-14(27 赞)
