# 《张扣扣案件为母报

《张扣扣案件为母报仇应该被处以死刑吗？》

晚上好 与星友们分享丹儿的第 85 篇原创 7.19

————————————————————————

今天与朋友聊天，聊到最近的热点新闻。张扣扣为母报仇 案，对于法院判决张扣扣死刑这一判决，引起了很大争议。 很多人认为他是为母报仇有原因，应该从轻处理。对他为母 复仇的想法表示理解，但是对他复仇的做法不是很赞同，其 实有更好的解决方案，和大家分享一下。

【交代一下案件背景】 张扣扣因为母亲多年前被邻居打死，但因对方是未成年人只

判了七年牢狱。有一句古话叫“杀人偿命”，张扣扣认为判决

不公平，心里一直想为母亲报仇！

再加上年少在外打工，总被骗，感受不到人与人之间的温 暖，又因为没钱娶不到妻子。对人生不抱有希望了，在二十 二年后终于在邻居回乡祭祀时杀了邻居，和他的两个儿子。 二儿子因为去女朋友家逃过一劫。

为什么要杀邻居的儿子呢？因为当时打他妈妈的人很多，最 后是未成年的三儿子出来认罪说他打死了张扣扣的妈妈。

在杀人后的第三天选择了自首。

【现在有两个争议的声音】

①是觉得应该法外开恩，因为当时的判决真的不公平，很多 人能理解张扣扣。觉得他做的对，坏人应该收到惩罚。既然 法律没有惩罚恶人，他用自己的方式解决也可以理解。

②是觉得不应该法外开恩，如果这一次对他法外开恩了，就 等于告诉大家，以后有私人仇恨，都可以通过自己报仇，不 用受到法律的制裁。

【我的观点】 其实我是比较赞同第二种说法的，对张扣扣的复仇的想法表

示理解，换做我们任何人都会想去报仇。但是他的做法我不

认同。

因为他虽然报了仇，但是自己的一生也毁了。而且仇人还死 的很痛快，一天之内三人全死了，我并不觉得他们得到了应 有的惩罚，可能他们只是死前害怕了一下，也并没有很痛

苦。

无论是复仇前还是复仇后，真正痛苦的都是张扣扣。复仇前 的二十二年里，每天生活在痛苦里，走不出来，生活也不如 意，感受不到人与人之间的温暖。复仇后又经历这么长时间 的判决，即使从轻处理，不死了，也会坐很多年的劳。

所以你看，这样的方式复仇，仇人并没有很痛苦，而你却一 直走不出来。

那么更高级的复仇方式是什么呢？别误会，我分享这篇文章 不是教大家怎么杀人哈，往下看才是重点哈哈。

【复仇并不能帮你走出阴影】 不知道大家有没有看过一本大仲马写的小说《基督山伯

爵》，大仲马可能大家不太了解，金庸大家应该是知道的，

金庸的偶像就是大仲马，他就是因为看了大仲马的《三剑

客》才想写武侠小说的，而且文章风格和大仲马也很像。

《基督山伯爵》也是写的复仇类的小说，但是却写出了治愈 人心的味道。主人公也是要复仇，但是他并没有采用雇凶杀 人或自己去直接把仇人杀了的复仇方式，因为他觉得这样让 他们死的太痛快了，他必须让他们活着感受到痛苦，摧毁他

们认为最重要的东西，让他们生不如死才能达到复仇最好的 效果。

他花了十年的时间去布局，让他的三个仇人都受到了应有的 惩罚，并且最重要的是他没有犯法哦，他所做的一切都是在 法律允许的范围内复仇的。结尾还收获了自己的爱情，放下 了仇恨。走出了阴影。

为什么他能走出阴影呢？不是因为他复仇成功了，让他的仇 人都生不如死了，他感到快乐。而是他在复仇时导致了一个 仇人的儿子意外死亡，他做了反省。最后放过了本来要饿死 的第三个仇人，他才放下了仇恨走了出来。

因为他的心还是善良的，没有被仇恨蒙蔽双眼。所以最后他 控制住了情绪，没有让复仇的情绪掌控他。从而获得了真正 的幸福。

大家看出这两个复仇的不同了吗？最大的不同不是基督山伯 爵的复仇更高级哦。而是他们一个是法内复仇，一个是法外 复仇。

【法内复仇与法在复仇】 法内复仇在没有违法的情况下，做到了复仇并且自身不用我

会坐牢，仇人受到了比死亡还通痛苦的惩罚，看起来更解

气。

但是报了仇我们就会变得开心了吗？以前的那些发生过的事 就能当做不存在吗？我们还能信任别人吗？

其实无论是法内还是法外，都希望那个复仇的人，不要被仇 恨蒙蔽了双眼。冤冤相报何时了呢？

表面上看，被伤害后，除了复仇和怎样复仇我们没有别的选 择能让我们开心。能让我们解气。但是如果你学过第三选择 就不一样了。

【第三选择】 除了复仇，我们还有什么第三选择能让我们变得快乐呢？ 之前听过一个案例，有一位夫人老来得子，她很爱她的孩

子，可是有一天她的孩子被车撞死了，司机是酒后驾驶。判

决也很不公平，她非常的愤怒伤心。

她觉得她的孩子不应该死，是那个司机的错，他为什么要喝 了酒开车？很长很长时间她都很痛苦，她想她必须杀了司机 才能解恨。

但是后来她冷静了下来，杀了他，我一辈子将会在牢狱里度

过，我能接受吗？还有司机的孩子也会失去爸爸，一下子会 毁了两个家庭。

我不能这么做，那我怎么办呢？我好痛苦，大家猜后来这位 夫人是怎么走出来的？

她去当了志愿者，专门给大家演讲不要酒后驾驶，和酒后驾 驶的危害，帮助了很多人从酒后驾驶的阴影中走了出来。她 自己也感受到自己的价值，她觉得因为这个事情，她女儿的 死也变得有了价值，她的行为可能避免了很多别的家庭因酒 后驾驶带来痛苦。

当我们治愈不了自己时，最有效的方式是什么呢？就是去帮 助别人，去做对社会有价值的事，这会让你感觉很有成就 感，你就是对社会有用的人。《自卑与超越》这本书里也详 细的提到过这个方法。

当我们不在纠结于自己的情绪，试着去帮助别人，提高自己 的格局，就可以走出自己的怪圈。那些烦恼的，想不通的事 都会变得不那么重要了。

＊＊＊＊＊＊＊＊ end ＊＊＊＊＊＊＊

▼对张扣扣的案件表示惋惜与同情，希望大家通过他的案子 能给自己一些警醒，不要让情绪控制了我们，冲动下去做那 些可能会影响我们一生的事。

治愈自己最好的方法，就是放下那些自己纠结的事，去帮助 别人，做一个对朋友，社会有价值的人。这个世界上温暖和 爱还是很多的，一直生活在阴暗面当然会感觉不到，如果你 愿意走出来就会发现，外面阳光很好，很温暖。

今天的分享就到这里啦，希望我的分享能让你受益。
