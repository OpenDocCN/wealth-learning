# 然果打卡第十天近日

然果. : 然果打卡第十天 近日收获

1\. 一天之内会有七八个朋友找我询问各种问题。

2\. 有朋友愿意付费定期找我咨询。

一天七八个朋友找我咨询各种问题。

（聊天记录看图哦❤能给别人提供价值的感觉很好！）

学习高质量打卡、输出自己之后，每天都有不同的朋友加

我，我也能帮助到不同的朋友，包括咨询心理问题、咨询如

何打卡、咨询定位、交流演讲等等……

而且交流完之后很多朋友都表示很收益，看着他们因为自己

有些改变，自己也特别开心！感觉自己也有了价值感！

有朋友愿意付费定期找我咨询。 想想我进入社群差不多一个星期，就有朋友咨询之后愿意发 红包给我，甚至还有人愿意付费定期找我咨询。（小小激动 一下>o<）

重点来了

（也有朋友问我是怎么做到的，回头我整理一下自己一些经

验再与大家分享~希望跟大家一起进步！）

我能提供：

①对输出价值（星球打卡）和社交过程中有疑惑的朋友欢迎

咨询！

②微商、个人品牌等需要朋友圈打造和变现知识的朋友欢迎

咨询~

③写作、文案策划有疑惑欢迎咨询。

后期将分别推出课程分享。敬请期待！
