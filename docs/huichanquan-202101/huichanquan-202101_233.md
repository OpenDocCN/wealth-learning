# 骗子专在直播间钓鱼

最近一种新型的诈骗手段浮出水面，诈骗嫌疑人专门在网络直播间里**“钓鱼”。** 

**“一次性”购物链接**

近日，家住萧山新塘的修先生在手机上看直播时，浏览到一个专门卖二手货的直播间，**被弹幕留言里的一条卖手机广告吸引**，对方报价不仅比主播提供的价位低，而且**明显低于市场价**。

修先生根据弹幕留言添加对方 QQ，很快就收到一个所谓**“购物链接”**，让修先生点进去选购。

![](img/371cdc0b5a3c21e22f4b5ba93ac7118a.png)

▍嫌疑人发送商品链接截图

网站有各种款式的手机可供选择，图文并茂，看上去比较正规。修先生觉得有这样一个平台，比一般推销二手手机的“个体户”靠谱多了，便不疑有他。

选中心仪的苹果新款手机后，对方要求**必须一次性付款且只能在购物网站上操作**，修先生照办，付款 4900 余元。

![](img/b67eeb8c91e8bf598d2dfc872ef4695a.png)

▍嫌疑人忽悠受害人直接付款转账

没想到这个看似正规的购物平台竟是“**一次性**”的。**等修先生付完款再点链接想查看发货情况时，却发现链接已经无法打开，对方 QQ 也已把他拉黑**。如梦初醒的修先生赶紧报了警。

**跨省收网抓捕**

接到事主报案后，新塘派出所立即开展调查，**综合指挥室发挥小脑作用**，通过对涉案账户以及资金流向进行分析研判，**成功锁定 2 名嫌疑人，**分别为四川籍男子孙某和朱某，并查明两人的落脚点在江苏苏州市。

明确嫌疑人身份后，新塘派出所当机立断，组织抓捕小组赶赴江苏，通过连夜排查蹲点守候，**在一酒店客房将嫌疑人孙某和朱某成功抓获**。

![](img/5708ee612c15e3f67dd7854a56751c6a.png)

▍嫌疑人孙某审讯画面

经查，**孙某与朱某是高中同学，**结伴离开家乡到外闯荡。

![](img/8e3a5c00fd83f5107b319a70b0f851e8.png)

▍嫌疑人朱某某审讯画面

打工嫌辛苦，好吃懒做，很快便手头拮据。这时听狐朋狗友介绍，发现一条赚钱门路——**去直播平台“钓鱼”骗钱**。

两人通过不法渠道，生成诈骗链接，**伪造正规购物网站假象**，然后在直播平台发布低价售卖二手手机或者游戏账号的广告。**当有人上钩后，便通过 QQ 发送虚假购物链接。一旦受害人付款，链接便会失效，无法再次点入**。

令民警意外的是，**嫌疑人孙某自己就曾被人以卖游戏账号的名义诈骗过**，本该对诈骗分子深恶痛绝的他却以此为“灵感”，走上犯罪道路。

孙某等人通过这种方式多次实施诈骗，部分受害人因受骗金额不大、嫌麻烦等原因未向警方报案。

目前，**嫌疑人孙某、朱某已被依法刑事拘留**，萧山警方已查证案件**6 起**，已联系相关受害人制作笔录，案件正在进一步深挖侦办中。

警方提醒：

市民朋友观看网络直播应选择正规平台，对于直播间内陌生人发布的购物、招聘、游戏代练、装备交易等各种弹幕广告应保持足够警惕，切莫轻信。交易时尽量不要直接打款给对方，而是选择那些有第三方平台保障的。

**直播间弹幕广告暗藏陷阱，**
