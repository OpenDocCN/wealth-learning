# “接到校园贷逼债电话的晚上，我真的想去跳黄河。”

 大家好，我是老黑。

之前我写过很多关于校园贷、裸贷、P2P 的文章，之所以今天再次提起，是想谈一谈当年的校园贷对曾经借钱的那些学生的深远影响。

在我看来，曾经的校园贷虽然早已远去，但它就像一颗曾经爆炸过的生化武器一样，影响着那些以贷养贷的学生群体。

这颗“生化武器”诞生于 2014 年，从 iPhone 等电子产品入手，结合电商、地推和分期付款的优势，让校园分期平台迅速在全国大学蔓延开来。

接着，越来越多的网络借贷产品、P2P 贷款平台争先恐后的侵占大学校园。

那会大家会发现一个现象，当代大学生里面有这么一群人，他们的家庭状况并不好，甚至还有些贫困。

可是他们的生活就是该吃吃该喝喝，用高档手机、穿潮牌衣服，搞各种奢侈聚会，花天酒地自然是常有的事。

如果你不了解他们的家庭情况的话，你会以为这哥们家里应该挺有钱。但实际上，这些钱都是从当年的各种 P2P 平台借的。

大家都知道当年的网贷有多猛，利息有多高，当年那些无知的学生们哪会想这么多，超前消费，以贷养贷，到最后这颗雪球越滚越大，滚到最后都还不起了，只能通知家长。

有些可能只借了几千块，到最后滚到几万，十几万的都有，只能把父母一辈子的血汗钱搭进去，最终害人害己。

也有不敢向家人开口的，男同学可能会走上犯罪道路，女同学无奈可能会选择外围等不正当，来钱快的渠道，甚至有人因此丢掉性命。

但其实，只要陷入网贷，想要凭自身能力上岸的，少之又少，有人甚至毕业工作几年都无法上岸，大好青春就这样奉献给了各种网贷平台。

【差评】曾经拍过一个纪录片，记录了几个曾经深陷网贷的大学生，讲述了他们毕业后的生活。

我至始至终都认为，基本上大部分去网贷的学生，都是为了满足自己的虚荣心而已，但仅仅只是满足了虚荣心而已。

这点虚荣心的代价，有可能会让父母搭上一辈子的血汗钱，有可能会消耗自己毕业后的大好青春，甚至是自己的性命。

视频里的受访者有句话很是无奈，他说自己可能到三十岁才会有自己的人生，三十岁之前的人生都是属于那些 P2P 经营者的。
