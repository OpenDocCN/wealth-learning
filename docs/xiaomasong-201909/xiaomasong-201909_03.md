# (16 赞)《》我们上一篇说过

我们上一篇说过，商品是文化母体的道具，商家是道具的提 供者，而消费者就是道具的购买体验执行人。

然后这就讲到了生意问题，你做生意，首先就要考虑，这个 文化母体够不够强大，这决定了你生意未来的规模和消费者 购买的动力。

中国是没有咖啡传统的，所以咖啡馆在中国就一直不怎么赚 钱，因为它的文化母体不够强大；但茶是传了几千年的东 西，所以改造过的奶茶就是中国人喜欢的，那中国的奶茶品 牌就特别多，奶茶店几乎是咖啡店的 100 倍，还都能赚钱。

你选择咖啡，就是选择了一个弱的母体，那就要准备好接受 它市场不够大的事实，因为中国消费者的生活里咖啡并不那 么重要。

《》

那些寄生到强大母体中的商品和策划，就会获得更好的发 展，甚至将来会成为母体的一部分。

西贝的亲嘴打折节，常常会跟春节冲突，因为情人节常常在 春节放假，那为什么不在 520 或者七夕搞呢？因为 520 和七夕 这个母体不够强大。520 是在网络时代兴起的，七夕在古代并 没有情人节的意思，只有情人节有强大的文化母体，虽然来

自西方，但它在中国已经很流行，所以亲嘴打折节就寄生到 情人节，而不是 520.

我有个朋友，每年大学新生报道，他都会做一波生意，比如 新生三天内就要测试英语水平，所以他就卖收音机。但收音 机在那里卖更好呢？他在新生入学的前三天去学校超市门口 租一个地方，那里是新生入学必然要去买生活用品的地方， 而且新生也必须用收音机和耳机，所以购买必然发生。

要么成为目的地，要么让自己在目的地的路上。

你也可以考虑一下，自己的生意，是如何寄生在母体之上 的。
