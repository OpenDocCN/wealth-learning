# (27 赞)美团外卖的广告。P

王东魁 : 美团外卖的广告。Pacing 型的创意是能够唤起同感 的，但是定位区隔并不清晰。这些想法很棒，体现了“懂 你”，但是并没有体现美团与其他品牌的优势。

2017-05-04(27 赞)
