# “你屁股那么翘，我顶顶怎么了？”

这个社会向来不缺少灰色地带！很多事情不说，也许有些人一辈子都不可能知道，一批又一批的人正在被剥削着，不为人所知。也正是因为这些事见不得光，一些含义隐晦却又生动形象的词也应运而生——顶爷，扫楼，打胶人！ 

![](img/d9fe63a586d4fc142280931b88398066.png)

顶爷相信大家不陌生，也就是我们常见的“公交色丨狼”，“地铁色丨狼”，这些人利用公共交通拥挤的便利，经常用自己的身体摩丨擦女性性感且敏感的地方，而且大多以自己的 Xing 器官去“摩擦”！

一项中国北京的调查数据显示，26.6%的女性遭遇过公共交通上的 Xing 骚丨扰，除了采取“瞪他”的反应之外，最多的反应是“默默离开”。

这也更加使得这些“顶爷”气焰嚣张，每个季度，警方的“扫狼”行动，抓获的猥亵嫌疑人也都不少，但“顶族”依旧强盛不衰。

网上随便一搜就是他们的“讨论群”，其聊天内容也是不堪入目，直接就把女性当成了他们的 Xing 玩具。

![](img/97a82cc723e6dc6189857f2b1feb83d9.png)

曾有一个新闻采访，采访的对象是一个被地铁猥亵的小姑娘——小苏。

小苏是一名每天都需要地铁通勤的都市白领，当时车厢里人不算多，她叉开腿呈一个大字状，扶着地铁的把手刷手机。

大概是坐了一站后她感觉自己的屁股痒痒的，好像有什么东西顶着她，但绝对不是坚硬或是锋利的东西，然后……好像还有点热？？？

![](img/8fa554ba4c8c53d02f1f0f376a6d85c4.png)

她的第一反应是挪了挪位置，但没想到那个变态干脆贴过来开始顶她大腿。那位猥亵者是个皮肤黝黑，脸上布满褶皱，手上还有纹身的中年男人。 

![](img/2af0363ffc4c03f1e32fc2166cb07dcb.png)

![](img/689321f742f98e66b89d7ceec1585cf3.png)

小苏表示当时很想张口骂他，但只要一想到他反问“你怎么啦？你搞错了吧？你有证据吗？你是不是有病？”就硬生生把到嘴边的脏话咽了回去。

其实大多数“顶爷”都是一些胆小如鼠的虫蚁之辈，就是由于女性的怯懦，而导致他们的气焰越来越嚣张。

本来也许只是“顶一顶”，现在大多数顶爷会选择直接“She”在女生身上。

![](img/ee1773136735d6cf8b1ebf39b9d5cfc6.png)

微博上有一个人叫“涂鸦达叔”，此人在自己的微博上各种晒“扫街图”，偷拍女性的隐私部位，拥有不少信徒，最爱在公交、地铁上寻找目标。 

![](img/dd0531fa14d687e8b1700434dce18023.png)

![](img/93e1136be6b7ef5dd3ab4e92de3d6d62.png)

其有些行为，甚至已经直接触碰到了法律的边缘。

![](img/73d79edbdc83d0ccb7303a740e007916.png)

你难以想象，你什么都没做，仅仅只是站在那等朋友，但你的身体，早就被人在群里变成下流的谈资，被当成了下流的“Dang 妇”。 

曾在网上看到这么一句话：“女孩子打扮的漂漂亮亮的，觉得外面的世界五彩斑斓可爱有趣的时候，有那么一群人，站在黑暗里，想着怎么毁了她。”

![](img/6b25bb3275d3cc1bedc0e373c5c71746.png)

![](img/b0f8867bb0379b3730f2948fab1ce516.png)

由于被害人的各种纵容，社会打击力度低等各种因素加起来，这些顶爷猥亵者仿佛得到了纵容一般，越来越胆大，其行为也越来越变态。 

慢慢衍生出另一类扫楼者，称之为“打胶人”！

什么叫打胶人？

这些人已经不满足于只在大街上蹭蹭了，他们会尾随女孩回家，等女孩走进家门，他们就在女孩的家门口拿起女孩刚脱下的鞋子，趁着上面还有余热，脑海中回想着刚刚一路上尾随女孩的身姿，对着鞋子“打手 Q”。

最后打在女孩的鞋子上，并且在群里和他们的 BT“同好”一起分享。

![](img/46eb83f8d17679a8d80435e4bc251421.png)

![](img/9de636430ab1d2e43a6c98a04fb6719a.png)

其中有自己的表姐...

![](img/74a30b3c84988f590044a9d63c831ee7.png)

甚至有自己的亲妹妹...

![](img/9911d251cce778abfbe1764f006eed9a.png)

刚知道存在着打胶人的时候，心中很难平静，细思极恐。

因为犯罪是讲究升级的，虽然现在打胶人仅仅只是在女孩家门外，只对着鞋子猥亵，但之后呢？

随着社会对罪犯的越来越纵容，谁能保证这些变态下一步不会进行入室强 J？

![](img/48b0d49e15e800ad982903d2c8cd979e.png)

欲望无止境，当女孩的贴身衣物也不能满足罪犯变态的欲望时，那些独居女孩年轻的肉体无疑是下一个目标，罪犯的欲望是永远不能满足的，他们会榨干被侵犯者最后一滴价值，直至被害者被消磨殆尽。

所以不要对罪犯有同理心，也不要奢求罪犯会止步于哪个阶段，“顶爷”、“打胶人”正是罪犯早期的萌芽，我们要坚决抵制！
