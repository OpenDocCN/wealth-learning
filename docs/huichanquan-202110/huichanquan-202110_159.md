# 数名村民花呗的钱忽然消失，都是因为注册了政府医保？民警揪出“真凶”

激活电子医保后

支付宝的钱没了

咋回事？

![](img/6ac2b8dc3c4d8e8929e05151dd8bc724.png)

近日，昆明市禄劝县公安局马鹿塘派出所接一个来自到辖区某村民的**“特殊”报警**。

“我支付宝花呗的钱不知道怎么回事少了 460 块......”

接到报警后，马鹿塘派出所高度重视，赶紧向报警人了解详细情况。

据报警人称，自己既没有用支付宝买东西，也没有转账等行为，连警察经常宣传的电信网络诈骗情况也没有遇到，可是短信却提醒自己支付宝花呗支出了 460 块钱。

听报警人这样说，民警觉得很奇怪。

经过进一步沟通，民警发现了一个异常：在报警人支付宝的资金变动之前，**曾注册过电子医保小程序！**

“这不可能啊，医保程序是政府官方的，不可能注册后出现这样的情况啊？”

进一步梳理事件始末，民警感到不妙！

“或许还有更多的受害者，尤其是农村的群众，有很多是老人，不会用智能手机，如果银行卡或支付宝的钱被盗走，他们也许都不知道。”民警心想。

而让民警更担心的可远不止这些，报警人少的是支付宝花呗里的钱，如果还有其他受害者，如果其他受害者被盗刷的也是花呗里的钱，那将是一件更加棘手的事：“花呗是借贷性质的，这里的钱被盗用了，而受害人不知道不按时还，直接影响征信的！”

对此，当地政府和禄劝县公安局高度重视。

马鹿塘派出所一边结合正在开展的大走访活动迅速组织精干警力下乡走访调查，一边积极联系辖区内各村委会，让村干部组织人员协助走访，筛查出遭遇报警人相同情况的群众。

“这个案件的性质非常恶劣，在公安机关没有揪出真凶之前，老百姓只会单纯地认为，是注册了政府官方的医保电子小程序才出现这样的问题，这对一直以来十分信任、依赖、支持政府的老百姓打击非常大！对受害人来说，这样的精神打击远远超过几百块的金钱损失。”办案民警表示。

 **受害人越来越多

警方全力追盗捉住嫌疑人** ![](img/d83367c10a53c9d317bb6eeff0c3ffce.png)

在走访中，民警又陆续接到几名群众反映：今年 8 月，有自称是政府工作人员到家中帮助安装电子医保小程序，在注册支付宝并激活电子医保小程序后，发现自己支付宝内的资金被盗。

而今年 8 月份以来，禄劝县医保中心确实委托第三方工作人员，到各乡镇帮助不会操作的群众激活电子医保。此时，民警通过技术手段侦查发现，资金被盗的几户群众的支付宝内，有一个相同的转账收款人！

“这个收款人会不会就是嫌疑人？嫌疑人就藏在这些帮助激活电子医保的工作人员里？”办案民警想到这，便迅速联系上禄劝县医保中心委托的第三方负责人，调取了当时到各乡镇的工作人员资料。

民警在大量的资料中加班加点查找，终于，那个与收款人同样信息的工作人员出现了。

在深入调查、取证、核实后，民警确定其中的两名工作人员就是嫌疑人胡某某（男，22 岁，重庆人）、刘某（23 岁，男，重庆人）！

随后，马鹿塘派出所迅速出动抓获两名嫌疑人。

而在进一步调查时民警发现，其中一名嫌疑人并非初犯，此前曾在重庆有贩毒的前科。

 **临时起意图财

胆子越盗越大** ![](img/d83367c10a53c9d317bb6eeff0c3ffce.png)

据嫌疑人交代，他们在帮群众激活电子医保时，发现很多人不知道如何安装支付宝，就直接将手机交给他们来操作。

在拿到群众手机那一刻，嫌疑人发现，自己轻松就能得到群众的短信验证码、登录密码、银行卡等信息，便在自己的手机上登录群众的支付宝账号。

而这一切操作下来，十分顺利！

在第一次偷偷用群众的支付宝账号转走一笔数额不大的钱，且没有被发现时，两名嫌疑人开始“飘了”！

于是两人不断寻找下一个猎物。

据嫌疑人交代，他们并非凭运气随机挑选盗窃对象，而是有针对性地挑选：“第一种是不懂支付宝不用支付宝的新用户，这样不容易被发现；第二种是愿意把手机交给我们操作的；第三种是已经安装支付宝，平时很少用，也不怎么懂，但是账户上有钱的。”

就这样，嫌疑人陆续盗窃了 5 户群众，共计 3460 元人民币，直到被抓！

经警方再次核实，二人对所做违法事实供认不讳。

至此，该案成功告破！

目前，二人已被依法刑事拘留，案件正在进一步侦办中。

 **追回不仅是金钱

更是群众的安全感** ![](img/d83367c10a53c9d317bb6eeff0c3ffce.png)

案件侦破后，警方将受害群众被盗刷的金额如数追回并发还。

而禄劝县医保中心委托的第三方负责人则在案件侦破后，主动按照被盗群众的损失金额，分别对受害群众进行了全额补偿。

领到失而复得的现金和补偿金，几位受害人悬着的心终于放下来了，表现得十分激动：“从这个件事情上真切地感受到人民警察的高效办案能力，以及全心全意为人民服务的热忱，我们就感觉很安全，人民警察和政府值得信任和依靠。”

办案民警表示：涉及群众的案件没有大小，对警察来说每个案子都是“大案”，我们会更加用心用情用力为人民群众破民生案、追民生财，不断提升人民群众的获得感、幸福感、安全感。

“我为群众办实事”之“问计于民大走访”活动开展以来，禄劝公安局马鹿塘派出所牢记“服务人民”的神圣责任使命，切实为群众办实事、办好事，以“人民群众得实惠、矛盾纠纷快化解、风险隐患能发现、警民关系更和谐、公安工作有提升”为目标，多渠道问政问需问计，为群众排忧解难，以真情实感换来群众的真心实意。
