# 在境外遥控四件套买卖，4 个月从国内运出近 2 万套

> 原文：[`mp.weixin.qq.com/s?__biz=MzIyMDYwMTk0Mw==&mid=2247522080&idx=7&sn=2aece83c21abe3f02eb4166a3d264b64&chksm=97cb5c18a0bcd50e2660baae40800f8865027a90c9be4fef5af488d7eebd6796c875bb1fab5d&scene=27#wechat_redirect`](http://mp.weixin.qq.com/s?__biz=MzIyMDYwMTk0Mw==&mid=2247522080&idx=7&sn=2aece83c21abe3f02eb4166a3d264b64&chksm=97cb5c18a0bcd50e2660baae40800f8865027a90c9be4fef5af488d7eebd6796c875bb1fab5d&scene=27#wechat_redirect)

据安徽含山警方消息，偷运银行卡“四件套”为境外电信诈骗、网络赌博等团伙提供犯罪工具，短短 4 个月从国内运输银行卡“四件套”近 2 万套。

日前，安徽含山警方成功破获一起特大妨害信用卡管理案件，缴获银行卡“四件套”7600 余套，价值超过 8000 万元，抓获犯罪嫌疑人 60 余名。
