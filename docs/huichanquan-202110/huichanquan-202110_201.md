# 任由警察喊破喉，刷单诈骗依旧有？

**刷单就是诈骗！**

刷单就是诈骗！短短六个字就能说清楚的骗术，按道理应该很快被消灭。可是这种骗术丝毫没有消退的迹象，反而长期占据高发骗术的前三名，这到底是为何呢？![](img/e078047970a38a32d3b14e4994a53fb4.png)**案例回顾**

近日，阿紫（化名）在网上找兼职，没过多久就找到了一份可以在家完成的组装笔工作。但是对方称组装笔的材料要一个星期才能寄到，问阿紫有没有兴趣做一下兼职。

刚开始的任务是在“某宝”上关注店铺，关注一个就有 10 元的佣金，阿紫想着这钱来的十分轻松，于是就欣然接受。接着对方又推荐阿紫购买该平台数字货币的任务，称佣金可以高达 20%，阿紫尝试充值了 28 元和 108 元，结果都顺利收回了本金和佣金。

有了这两次成功经历，阿紫继续充值 808 元的任务，结果这次对方称阿紫做错任务需要补单，于是阿紫按照要求充值 2424 元去完成补单任务，完成后对方还是以各种理由让其继续转账。就这样，阿紫分多次向对方转账共计 103823 元。最终在爱人的提醒下，阿紫才终于意识到自己被骗，选择报警。

![](img/c9f456e5e784d9d6f063632ca06a7a88.png)

**虽说刷单诈骗手法不断变化，迷惑性大大增加，但是这种骗术归根到底离不开这三招：**

**一、建立联系**

与以往简单直接的招聘信息不同，现在刷单诈骗物色受害人的手法更加隐晦，最开始都是以一些其他名目与受害人建立联系，比如“**进群领水果、招聘短视频点赞、招聘手工加工**”等。

**二、抛砖引玉**

骗子最开始会让受害人做简单并且不用垫付本金的任务，然后再试探性向受害人布置需要小额垫付的任务，两次都把本金和佣金派发给受害人以获取信任。

**三、巧立名目**

当获得了受害人充分信任后，骗子就会下发需要垫付大额资金的任务，并且承诺更高的佣金。部分诈骗团伙为了显得更正规，还会搭建一些虚假的 APP 来下达任务。当受害人完成任务后，骗子就会以“完成任务不正确、连单”等借口让受害人不断转账，直到受害人意识到上当受骗。

![](img/0a37bf3ffbc67d638687bc58910e9d82.png)

**除了要记住刷单就是诈骗，**

**网上要你先垫付资金的工作同样是诈骗！**
