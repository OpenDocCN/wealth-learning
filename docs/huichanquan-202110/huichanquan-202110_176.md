# "阿里女员工案张某妻子"再发声，用 4600 余字长文控诉周某诬告陷害

> 原文：[`mp.weixin.qq.com/s?__biz=MzIyMDYwMTk0Mw==&mid=2247523513&idx=3&sn=22bb3c8bb49acb91afb0130d8a169061&chksm=97cb5781a0bcde97a84b8122d329189e1f212667bfcc06b071e12a4a12a637f84fa9c7ad4002&scene=27#wechat_redirect`](http://mp.weixin.qq.com/s?__biz=MzIyMDYwMTk0Mw==&mid=2247523513&idx=3&sn=22bb3c8bb49acb91afb0130d8a169061&chksm=97cb5781a0bcde97a84b8122d329189e1f212667bfcc06b071e12a4a12a637f84fa9c7ad4002&scene=27#wechat_redirect)

今天，一个认证为“阿里女员工案张某妻子”的用户发布一则 4600 余字的长文，指责周某利用颠倒黑白，毫无底线的编造虚假事实，意图使张某和王某文受到刑事追究。

据“阿里女员工案张某妻子”介绍，11 月 2 日，其又去了槐荫区人民检察院就阿里女员工案的诸多疑点递交了材料。她还称，这一个月去了多个部门反映周某谎话连篇，诬告陷害。知道这个维权是个异常艰难的过程，但不会放弃，为了保护真正的女性权益……

在这篇长文中，“阿里女员工案张某妻子”详列了 9 条关于周某的谎言和诬陷情况，并一一进行了反驳。她表示，相关事实都有大量的人证物证，恳请司法机关能公平公正的依法办理，做到法律面前人人平等，不冤枉一个好人，也绝不放过一个坏人！

此前，阿里女员工案王某文妻子也在社交媒体发文称，周某发文和警方通报表明，周某颠倒黑白，隐瞒对自己不利的信息，捏造、散布虚构的事实，造成多名当事人和阿里员工名誉和职业生涯受损，涉嫌诽谤。

彼时，王某文妻子表示，其已委托了控告律师，开展控告周某的准备工作。
