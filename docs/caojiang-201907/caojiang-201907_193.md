# 应大家的热情，主屏

世程 Cheng : 应大家的热情，主屏推荐来啦 一个一个介绍 吧， 1、雨时的界面太美了，不同的天气对应不同的诗。 2、 倒数日，主要是放在首页，提醒自己这今年还剩多少天了， 也是偶尔看看。 3、calendars 5 配合 omnifocus 真得是太好用 了。一个安排日程项目，一个列清单动作。 4、forest，喜欢 种树。一般看书时，工作时候用。 5、纽约时报，翻到墙外看 看。文章质量还可以，也不多。 6、即刻，关注你感兴趣的， 即刻提醒你。 7、分答，碎片时间。可以听听免费的问答语 音。质量蛮高的 8、锤子便签，用来写长文并归类，还不错。 9、anki,记忆神器呢。自己可以做记忆卡片，没事拿出来看看 复习下。 10、有道云笔记，主要是记录与工作有关的东西。 11、方片记事，由于记录太方便。我一般把他当做碎片文字 收集箱。之后收集有价值的知识点，用 anki 加上自己的见解转 换成记忆卡片。 12、reeder 关于 rss 订阅离线阅读的神奇。可 以看看知乎日报，好奇心日报，还有一些好博客 13、最后一 个微信，大家应该有经常用的
