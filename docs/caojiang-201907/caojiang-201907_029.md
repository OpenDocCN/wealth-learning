# 今天给大家推荐几个

大野宰 : 今天给大家推荐几个喜马拉雅上我一直在听的英语类 FM，很适合利用碎片化时间进行学习 ①潘吉 Jenny 告诉你 收 听链接： 推荐指数：★★★★★ 推荐理由：氛围轻松，声控 党福音。 以外教+中国老师搭档的形式，主要讲一些比较地 道的日常口语，比如：外卖有哪几种说法？还有就是中国人 英语口语里常常会犯的一些错误和闹的一些笑话，时间很 短，一集一般 10 分钟，很适合学生党在路上听或是在上班族 的通勤时间听。个人尤其喜欢女主播的声音，很温柔~ ②夏说 英文晨读 收听链接： 推荐指数：★★★★ 推荐理由：新闻类 题材，短小精悍，收听人数最多的节目。 主播是 05 年世界英 语演讲比赛冠军，原新东方知名讲师夏鹏，内容主要讲时事 新闻，并且分成了讲解版和朗读版，讲解版是逐句进行讲 解，而朗读版目前除了夏鹏老师外还加入了其他外教。不过 唯一的缺点就是新闻的内容一般不是最新的，时效性不是特 别好。个人建议收听时身边最好有纸和笔方便记下讲解内容 并及时回顾。 夏鹏老师还组建了很多个晨读群，在群里可以 和喜欢英语的小伙伴一起早起晨读打卡，能坚持的小伙伴可 以尝试。 ③英语曰曰曰 收听链接： 推荐指数：★★★★ 推 荐理由：内容丰富，题材有趣。 同样是新东方前优秀讲师， 流程上是先朗读，朗读的文章长度会比夏说英文稍长，然后 逐句讲解。除了讲解文章内容，重点单词还会挑出来，进行 原声扩展练习，很实用，之后还会鼓励听友进行造句练习。 最吸引我的还有片头的英文歌曲~ ④Albert 说英文 收听链接： 推荐指数：★★★ 推荐理由：牛人主播，专业性强。 主播 25 岁就是总统翻译，可见功力不一般。讲解模式主要是一句英 文一句中文，讲的都很细致，难度比前三个稍难一点，其中 包括英语使用习惯，用词和语法都会拆解。也建议准备纸 笔，有时候讲语法还蛮多的。 每天听一点，收获大不同~ 2018-09-11
