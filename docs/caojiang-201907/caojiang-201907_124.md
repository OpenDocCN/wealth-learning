# #晚安# 会议的

曹将 : #晚安# 会议的前和后。 今天和合作方开会，分享一下 之前和之后的过程。 在会议开始前： ①和前台沟通好，有几 个人过来，需要她倒点茶水。 ②再次看一遍前一天准备的素 材，方便沟通时有话可说。 ③接客人时，迅速打开话题。这 时候，可以： A.把知道的内容再以疑问形式抛出来。比如： 红璞今年在广州规模做的很大啊？（即是对对方的尊重，也 是对方舒适的话题） B.找到一些背景信息，进行延伸。比如 知道他们来自深圳，于是抛出话题：广州就比深圳堵好多， 听说深圳外地车过来也要限制？ ④随时带着名片，见面时双 手递给对方，并做自我介绍：你好，保利蒋志超。 在会议结 束后，赶紧做三件事： ①把对方的名片存到「名片全能王」 里。这个软件可以快速识别名片信息，并同时保存到手机通 讯录里，减少了自我录入时间。 ②给姓名贴标签。比如：曹 将｜保利研究院。 ③加微信好友，万一之后还有来往，沟通 会方便很多。一般开完会后加，对方都会同意。 很多都是小 细节，背后的逻辑有两个： 第一，打破尴尬，让对方感觉舒 适。 第二，创造未来沟通的前提，对自己而言也算是积累了 一个连接。 以上。 好梦。
