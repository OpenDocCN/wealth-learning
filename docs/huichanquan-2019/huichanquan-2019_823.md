“灰产”中的“黑产”

![](img/68407704a41e9a984c8afe5512de448b.jpg)

为什么叫灰产中的黑产，因为接下来我要说的几条产业链都是真正的黑产，涉嫌金额，以及对社会的危害性都是无比巨大的。

一四件套

这个四件套可不是床上用品，而是指以一个人身份办理的银行卡、U 盾、身份证、手机卡。

![](img/49ebfdfdba9b6383ace4eb427473e986.jpg)

这东西能干嘛？洗钱、外汇、金融犯罪、大笔金额犯罪、电信诈骗、收受贿赂，能做的事情非常多不过都是违法的。

那么四件套对线上博擦行业又有和作用呢？

**主要是三点：**

> **收付款**
> 
> **洗钱**
> 
> **注册账号隐瞒身份**

线上博彩，线上的收付款都是在网上进行。博彩公司老板不可能把公司财务流水和自己的账户牵扯上关系。所以，就需要四件套以及洗钱。

玩家如想娱乐就需要充值，如果赢了就要退款。这一切都需要四件套来运作。这样是防止有关单位通过账户来往把他们一网打尽。

最重要的一点，可以隐秘身份，防止泄露个人信息。

菲律宾目前来自国内的四件套已经泛滥，国内卡头通过走私或者海运到东南亚各国。大部分都是二手三手四手卡商。黑吃黑的现象也经常发生。

四件套是哪里来的呢？

一些不法分子跑到深山老林里，以一百到二百人民币收身份证，然后在通过关系自己办理到银行卡手机卡等。

还有就是发布一些兼职工作吸引缺钱的学生，民工来询问，然后以几百元的价格收齐一套。

这就是卡源，从卡源---卡商---一级卡商---二级卡商---境外卡商到了境外国家稳定的一套价格可能会到 3000-6000 不等

![](img/86af8ffa5db5d7a041d4f7c79057ceac.jpg)

二第三方支付系统

网络时代，网上交易是很稀松平常的事情，为了公平公正便衍生出了第三方平台，如：**支付宝、付费通、拉卡拉**

第三方支付平台实际上不属于灰产，但是有些第三方平台就没那么白了。

一些日流水很高的博彩公司用第三方是没有办法满足的，所以就衍生了第三方平台。

![](img/853fea060f9a0c40ccbfa16ebbe959a5.jpg)

第三方的好处在于，出款速度快、资金安全。而出款速度对于一个网络博彩公司来说是命根子。第三方帮你出入收款对接客户省了博彩公司很大的麻烦，除此之外还提供微信、支付宝、银行卡等收款渠道，大大改变了博彩公司尴尬的收款渠道。

而第三方则抽博彩公司百分之 0.3-2 的占成费用，对于博彩公司来说，这是很可怕的费用，因为赌博来说可是流水收入。

三人力资源相关产业链

线上博彩行业其实属于 IT 公司和金融公司的组合体。不管什么公司，他都需要员工来运行。而东南亚的线上博彩公司基本面对的市场都是大陆市场。所以从各方面来考虑，东南亚的线上博彩公司是离不开大陆员工的，一个线上博彩公司只要是主做大陆市场，那么公司百分之 90 都是大陆员工。

有意思的是，这行在国内属于灰黑产业，博彩公司没有办法通过正规渠道从大陆招聘员工。但是大陆员工又是必不可少的。

## **签证保关（受贿的移民局）**

签证是一国政府机关依照本国法律规定为申请入出或通过本国的外国人颁发的一种许可证明。

大部分国家签证都会有两种类型，一种是旅游签证，一种是工作签证（工作签证一般比较难签，不仅价格贵，耗费的时间也比较久）

![](img/1d23127ae80d8a458bff3c1ea60932d0.jpg)

应聘到博彩公司的国人都是以旅游签入境，不少公司甚至连工作后也是沿用

> **旅游签证**

只要你需要旅游签证，很简单，都无需亲自去领事馆。只要把相应的材料寄给某些旅行社就可以帮你代办，价格 250-500 之间一周内百分百下签。

从旅游签、临时工作签证、9G 劳务工作签证等都可以帮你轻松的办到。

> ## **机票**

想要证明你是不是来打工的，最简单的就是看你的机票。

如果你是旅游的，那么你的机票一定是来回机票。如果你是单程票，那么你百分之百会被卡。

从此，又衍生出一个小型的黑链服务“借票”

航空公司内部帮你买一张回程票，然后你出关后航空公司就会帮你退掉。然后航空公司内部收取少量的服务费。

（说个题外话，不管你在海外从事什么行业，都要遵守本地法律法规。签证这个东西能办一定不要省。

最近当地移民局也在不定期的抽查，说不定前脚你还在吃面，后脚你就进移民局了）

![](img/08699c47f030d6830ccd0f58630795c7.jpg)

> ## **保关**

保关一词，在全世界也只有菲律宾能听到了。专为中国人量身打造的。保关保关，保证入关。

16 年至今有超过 300 万中国人来菲。无一例外所有人都是以旅游签证入关，其中打工人数占大多数。很多人以旅游签长期停留引起了当地的重视。使得现在菲律宾的海关以及移民局对中国人格外“热情”

![](img/01209e2b92c3ab8487fe2ca4c100d7a3.jpg)

当局为了遏制中国人来打菲黑工，菲海关对中国人的检查力度比其他国家的高好几个档次，只要看着你像来打黑工的，轻则拉到小黑屋进行盘查，重则遣返拉黑护照。

可是博彩公司的员工如果被遣返，那么损失又会很大。

结果，（保关）这一项服务就顺应天意的出现在大家的面前。

保关服务 500-1000 人民币之间，购买过保关服务的基本上都会安全的出关。那些没有保关并且被带进小黑屋受调查的中国人就没那么幸运了，如果调查出有问题，不仅遣返，甚至你的护照都会被本地移民局拉黑。如果你的护照有了这种污点，那么对你以后它国工作以及旅行有很大的障碍。

所以大部分了解其中利害的中国人都不会省这笔费用。

保关，真的算是博彩影响下的畸形产业。

保关这个产业就是移民局官员受贿后的产业，除此之外各种移民局服务给有钱人开着大门。

如果在菲律宾，你有不爽的人或者钱被员工卷走，那么只要你有他的个人信息并且花一笔费用就可以让他凭空产生一个罪名，并且被移民局拉黑。

> **洗白：**

菲律宾官员真的是赚钱的一把好手。被仇人拉黑了？不要怕，洗白业务等着你。只需花一笔比拉黑稍高的费用就可以洗去一切不良信息，并且从移民局黑名单系统中剔除。

## **包网**

包网是什么？如果没有包网公司没有包网产业，那么面对大陆市场的线上博彩公司的数量会大大减少。可以说包网是这个行业的第一大助力。

![](img/bcda4a4c340c3fa0deb33128da470864.jpg)

这是一个包网公司的自我介绍，大家可能对它的自我介绍不是很了解。我给大家详细解释一下。

线上博彩公司其实等同于互联网公司。创立一个互联网公司或许不难，可是要创建一个网络博彩公司就没那么简单了。绝大多数的线上博彩公司不仅没有法律保护、还要时刻防止黑客攻击以及 ddos 攻击敲诈、还要担着出入款账户被冻结等等。可以说要想平稳的运营一个网络博彩公司无异于痴人说梦。

从产品开发到运营需要大量的资金作为后盾，不少公司初期都是摸着石头过河，能成功的真的都是用钱砸出来的。

![](img/8b169ba0ee628dcbafe8402f33465d0b.jpg)

包网公司的出现打乱了这个市场，你不需要有大量的资金，不需要有网站开发维护的技术人员，甚至连第三方平台都不需要有。

你只需要花非常少的钱就可以买到一体化的建站服务，从建站—维护—活动策划—支付接口—域名—服务器—漏洞防御—反套利—后台系统，等等一系列。

而你所需要付出的代价就是每月的维护费、开板费、以及最重要的平台盈利分成。

如此小的投资就可以运营一个博彩平台，致使这个市场变的拥挤不堪。一个又一个的人投身进来，一个又一个死去，唯一不变的就是包网公司越来越肥。

![](img/aa48c171b9e4866efd50f5738fb9d3fc.jpg)

东南亚面对大陆市场的网络黑彩，百分之 75 是出自包网公司之手。

## **API 游戏厂商接口**

API 这一词可能大多数人不了解，简单的说 A 开发出一个产品，B 调用 A 产品的功能到自己的平台上。B 完全不用开发就可以让平台拥有 A 产品的功能。（最简单的一个例子，百度某地区的天气，结果就是百度调用了气象局的数据，让你可以在百度看到某地的天气）

![](img/a46dbd44f143bf1660f3581a2c40f5e0.jpg)

而线上博彩的 API 服务就有意思多了，我为什么要叫 API 游戏厂商呢？是的，因为这些 API 服务相当于游戏厂商，更像一种专利技术持有者。

开发—测试—维护一款博彩游戏是很困难的一件事情，而且某些大型的 API 游戏厂商已经做出品牌效应，不少玩家只认品牌。所以，目前市场基本上所有博彩公司都会接众多的 API 接口。

> **彩票类**
> 
> **体育类**
> 
> **真人视讯类**
> 
> **电子游戏类**

每种都有很多接口可以选择，而博彩公司只要对接好接口然后把网站搭建好就是一个网络博彩平台。

**1.彩票类**

目前市场上主营的彩票基本都是官彩。比如：六合、时时彩等官方彩票，线上彩票玩法以及开奖结果与官彩都是一样的。所以线上的彩票也被称为“私彩”“黑彩”相比官彩的优势就是，赔率高、有活动、有返水。但是同样有着各种不确定因素，比如账户被黑、资料泄露、不退款等风险。也是国内各方机构重点打击对象，因为给官彩造成了很大的影响。

![](img/6068314ea3643e464bf4caaa0c58a85d.jpg)

线上彩票 API 接口比较少，所以也没什么关于彩票 API 的话题。反的是彩票盘口数量在整个东南亚里绝对是首屈一指。原因就是彩票盘口结构比较简单投资少。

**2.体育类**

体育竞技线上博彩与国内体彩大同小异，不同的是赔率更高、比赛更多、体育种类更多（甚至电竞比赛也有盘口）除此之外体育类的博彩更容易被人接受。与别的彩种不同的是，从购入到开奖中间有很久的时间、以及观看比赛结果不确定性带来的刺激。

![](img/6d1c7ae7dc4af4ffb748fb06d0d33ca9.jpg)

体育盘口的难点主要是赔率的定制，所以每家体育博彩公司会请很多的精算师、操盘师等职位来运作公司。

真正专业的体育博彩公司不多。而体育博彩是很吃品牌的，所以有的专业体育博彩公司是不轻易的放 API 接口的，所以大部分体育博彩公司的水位以及赔率会跟着比较专业的公司一起走。

体育博彩带来的利润性远远不如其他种类，而且体育博彩的玩家是非常非常认品牌的。如果想要自己的品牌被玩家认可最简单的方法就是赞助大型体育比赛。赞助一场大型体育比赛的价格是很恐怖的。

从表面上看体育盘带来的利润远远比不上其它彩种，那为什么那么多公司使劲在这上面砸钱。因为体育竞猜型的博彩是最容易被大众接受的。而博彩公司的平台上远远不止一种彩类，而玩体育博彩的玩家是有很大的几率被引到其它即时开奖类的游戏。

**3.真人视讯**

真人视讯，优势在于可信度高,模仿赌场场景、设备、玩法等让玩家最大体验在赌场的感觉。

这些游戏厂商会招聘各国的美女来做主播荷官，为了吸引顾客穿着会很暴露、甚至有的公司还会招一些知名的日本女优来做荷官主播。她们的工作内容就是在镜头前从事荷官工作。主流游戏为：百家乐、筛宝、转盘等

![](img/2083276e8b3715621d7b793e5ab3cb4b.jpg)

目前市场上真人视讯这一块儿被几家大公司把持十几年，而真人视讯的玩家也只认这些品牌。所以市面上的线上博彩除彩票盘外都会接这些游戏厂商的身影。

由于菲律宾政策原因、大部分的厂商都会把摄影棚以及总部设立在菲律宾。但还是因为政策原因能拿到手续运营真人视讯的公司少之又少。

![](img/048c6d1f439e630ace82bbb61cdbaa5a.jpg)

**4.电子游戏类**

电子游戏最早起源于水果机拉霸，也叫角子机、老虎机等电子机器。玩法非常简单易懂，赔率是设定好的，由于随机性比较大，有以少博多的概率、过程也比较刺激。所以电子类的博彩游戏在全球都很有市场。

![](img/5ebc4a3c19a67f27a55c211a5a15bd46.jpg)

而线上博彩电子类没有什么老牌公司能霸占市场的，电子类游戏难点主要在于活动策划以及优惠活动。所以电子类的 API 接口众多，博彩公司选择也更多。
