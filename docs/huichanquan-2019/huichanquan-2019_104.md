结尾

![](img/2bc1fda818d1d044f3bd459476bf6d93.jpg)

作为币圈韭菜，每年不见识几十个跑路项目，这个年过的都不香。实际上对于跑路崩盘这样的事情，大家都已经见的足够多，也都麻木了。

所以每当有维权群建立起来的时候，开始还有人同仇敌忾的讨伐项目方，然后慢慢的就被广告党和段子手们占领，由此还产生了一批维权网红。

此次全球看点跑路之后也毫不例外，维权群刚建起来没多久，各种带着哭腔讨伐项目方的维权段子手们就纷纷开始登台表演，向韭菜们展示着自己精湛的演技。

当然也可能是真的亏了钱，只是在用阿 Q 精神自嘲罢了。

这种带有表演性质的讨伐，一方面自带喜剧效果，另一方面则非常利于传播，估计现在币圈的各个社群里都在转发这些段子手的作品。

但是真的有人因为全球看点的跑路，被逼的没法活。

![](img/b233b061aef1caf723d9b3b22ae8e48f.jpg)

全球看点的跑路，不是个例，在接下来的这一段时间依旧会有不少项目方割韭菜跑路，还是那句话：你想赚钱过年？很好，狗庄也是这么想的。

据说维权群已经在调查操盘手的详细信息，准备让他们吃不了兜着走。

与其这样出了事情再费时费力的去维权，不如在没崩盘之前迅速撤离，你都知道狗庄要割韭菜过年了，就别去贪恋最后一个铜板了吧。

很多时候你会发现自己不是被狗庄割了，而是被自己的贪欲割了。
