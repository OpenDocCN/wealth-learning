开通微信支付分竟然要“收费”？而且只能在网约车上用？不！先捂好钱包听我说。

微信支付分是对个人的身份特质、支付行为、守约历史等情况的综合计算分值，覆盖的场景也越来越广泛，包括[充电宝免押租借](https://mp.weixin.qq.com/s?__biz=MjM5ODEyNzQ4NQ==&mid=2651761884&idx=1&sn=8b6e9063422dce9ed8c57e330493fee2&scene=21#wechat_redirect)、[先乘后付](https://mp.weixin.qq.com/s?__biz=MjM5ODEyNzQ4NQ==&mid=2651762480&idx=1&sn=3a904dd1aca24ccb5333917d8b28e88a&scene=21#wechat_redirect)、[微信机上付](https://mp.weixin.qq.com/s?__biz=MjM5ODEyNzQ4NQ==&mid=2651762148&idx=1&sn=2b79ece96535715fc3797ee5ea60ac1e&scene=21#wechat_redirect)、[酒店免押预订](https://mp.weixin.qq.com/s?__biz=MjM5ODEyNzQ4NQ==&mid=2651761645&idx=1&sn=000e70ba7d23882e1e038c00b6f9d76f&scene=21#wechat_redirect)、物流快递先寄后付、娱乐设备先玩后付等数十个场景。

开通微信支付分不需任何费用，且步骤简单便捷。

你可以在已接入微信支付分的商户所提供的服务中开通。![](img/2cc249e57235dc773e9a1a148f52a6e3.jpg)

**如何查看分值**

开通微信支付分后，依次点击“我-钱包-支付分”，就可以查看自己的分值了。
