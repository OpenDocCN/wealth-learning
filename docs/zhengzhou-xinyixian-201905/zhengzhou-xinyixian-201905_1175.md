# 1175.星主，北龙湖有没有

星主，北龙湖有没有已经洗白的盘？你觉得哪个最适合做为 北龙湖的上车盘？

评论区：

1 : 粘贴一句名言 学我者生 似我者死

“洗白”是个很有意思的事情。 一线城市，一手房屋价格会比同 地段二手高很多，如果是一手+豪宅（定位），那就高更多， 所以存在一个洗白过程。 二线城市不是如此，你观察一下过往 数据，会发现，打比方豪宅/高品质住宅一手卖的时候比同地段 二手贵了 3000/平，过了三年，不但“洗不白”，价格差距反而越 发大了！ 所以当我们做推断的时候，不能生搬硬套，类似在重 庆建仓的话，贵 2000 的好品质江景的价值是高于普通小区，未 来差距只会拉大。郑州同理。 北龙湖目前上车的话，550W 的 精装四房算是不错，正商龙湖上镜还有正商即将开的另一块差 不多也是这个价格。兰溪上苑有单价 3.1 出头的毛坯也不错，但 需要关系去找。
