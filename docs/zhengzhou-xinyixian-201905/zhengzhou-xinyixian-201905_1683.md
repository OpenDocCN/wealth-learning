# 1685.北龙湖是不是不适合

北龙湖是不是不适合投资？只能自住？怎么选标的物？现在 适合入手吗？

评论区：

1 : 唯一需要注意的就是标地别太大，大标的周期太久

2 : 我说的这俩楼盘都有中等户型 尤其是龙湖上镜

3 : 洋房 140 平是鸡肋，食之无味。

4 : 以变化眼光看问题 客户画像和需求随时在变化

并非不适合投资。纵观其他城市豪宅片区发展历史，选的好也 是有可观收益的。 我们评判贵与便宜，不是看价格绝对数字。 价值高于价格=便宜，价格高于价值=贵。类似 4000 块的苹果 8= 便宜，400 块的一瓶矿泉水=贵 目前北龙湖区域 4 万以内/略出头 的从投资角度来讲是可以的。比如正商龙湖上镜，兰溪上苑， 单价不高，总价可控。
