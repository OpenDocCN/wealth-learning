# 832.我现在家里有一个表

我现在家里有一个表姐，又在建议我们拿 100 万首付款去买一 个二手房，这样马上就能交易，并且马上就能租出去，对我 们来说是资金最大化……因为期房要 2023 年以后才能交易。 我的问题是，是拿着 100 万首付去买合适的期房，还是买二手 房？期房其实我已经有明确的标的了，二手房还没有，你这 里有哪些标的推荐吗？谢谢!

二手房是能够马上交易，但盈利好的二手房不一定能补充多少 租金现金流。 如思考之后，觉得想要二手房的话，可以代选筹 笋盘。 我方合作渠道已打通，整体费用与普通中介门店费用一 样。 除了获得笋盘溢价，一些环节还可以省不少钱。
