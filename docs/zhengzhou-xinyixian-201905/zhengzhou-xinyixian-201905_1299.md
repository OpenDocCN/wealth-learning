# 1300.老师好哈，问个谈判

老师好哈，问个谈判小细节。 房主，一上来就说，另外还有一个客户要买我的房，比你多 出价五万。你不买，我就找别人买了。确实也有人出价比我 高些。

这种情况如何处理。前提是这套房子我想买。

了解一下他的需求，和在意的地方，看是不是和自己恰好吻 合。 比如，假如他想要全款，而你正打算全款买下来抵押； 假如他在意买家是否靠谱，而你却可以表现的是个体面、诚 恳、靠谱工作的人；假如他房子里还有人，不能及时搬走，而 你可以接受，那么也可以作为价格商量的条件。 总而言之，知 其需求。 如果他其他需求皆无，只要多的钱，那就认真理性想 一下，此标的，自己是否愿意多支付，既不要被逼定，也不要 因小失大。
