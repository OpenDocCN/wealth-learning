# 1656.女神，我是外地人，

女神，我是外地人，不想迁户口，准备缴纳公积金拿购房资 格，问：1.感觉现在牛市尾巴，但看好郑州，缴纳两年公积 金，看市场以伺机买入，是否可取？2.公积金贷款在本地已经 用过一次，郑州缴纳后是否还能再贷款？3.郑州缴纳公积金 后，只要购入住房，账户内余额就能支取出来对吧。谢谢， 问题较多，小白很困惑

评论区：

1 : 4，指的是所有楼盘还是个别的？

2 : 二手，东区为主。 其他区域费用太高，因素不可控。 一手房需要开发商环节，因素不可控。

3 : 晓得了

1、无户口，仅有两年郑州公积金，不能构成购房资格。 2、有 户口/社保连续缴纳 2 年的情况下，郑州缴纳公积金满六个月后 可以贷款。 3、余额可支取。 4、外地三无可破限购 有资源
