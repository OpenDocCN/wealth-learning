# 1254.S 老师能简单讲下郑

S 老师能简单讲下郑州东区的房价和低容积率的关系（个人感 觉 new money 都在东区这是主要因素:)，还有老市区和部分三 环边大部分高容积率和房价的关系（个人感觉有人气能跟上 大势:)，但欧神不看好所有城市的新区，能简单说下郑州有什 么不一样吗？会不会若干年后郑东新区也和浦东一样失去郑 府的扶持，整体容积率又这么低，人气不足失去耀眼的光芒

评论区：

1 : 呵呵，重庆两江四岸也是新区头一次听说，提前祝贺 S 老师新年快乐！

2 : 老城区是朝天门和解放碑 江北嘴就是“江北新区”

这是三个大问题。[抱拳] 会提问。 1，东区有以下鲜明特征： 土地价值最高*容积率低*高端岗位密集*住宅供应相对少=高房 价； 土地价值最高的*同时密度又最低的小区域=最高房价。 具备真实的最优的居住价值，new money 第一批来到这里。成 熟度提升之后，居住价值与投资价值双 Up，一批又一批的 new money 再次来到这里。 2、老城区的高容积率是历史规划原因， 存量产业和人口也比较多，居住价值显著低于东区，价格也相 应低一些（十年前相反）。当我们看一个片区的繁荣，思考他 未来房屋价格关系时，要注意，当前的繁华与落寞在未来是可 能发生逆转的，后发优势不仅仅出现在产业领域，在楼市中也 时有发生，尤其是在二线城市。 3、欧大总体不看好新区，但 并非绝对。重庆两江四岸严格说来也是新区。郑东新区同理。 所谓“新区”，是有差别的。郑东新区发展已有十年，是二线城 市新区中耀眼的明星。 一线城市的新区，发展起来后，一般也 仅是区域/某个产业中心。 敲黑板，二线城市新区，则是有可 能是成熟度和中心化的双重提升。全城行政力量、商业总部、 优质居住条件、医疗教育连根拔起重造新城，旧的老城不断被 失血。 郑州东区即是这样的“新区”，发展十年，已是熙熙攘攘 流光溢彩。只有进一步上升的确定性，成熟度和中心化的进一 步集权，而无凋零可能性。
