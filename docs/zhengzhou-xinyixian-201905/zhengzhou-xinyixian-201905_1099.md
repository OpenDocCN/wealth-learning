# 1099.请问老师设计怎么收

请问老师设计怎么收费呢？另，在郑州简单自住，按照水库 标准，克勤克俭，每平多少装修费合适？500-1000 可以么？

评论区：

1 : 这个简单自住标准应该大概对应我们的投资客户中等标准，硬装费用可以降低。

2 : 基础硬装造价需要看客户对储物空间的需求，一般自住型客户对这方面要求高于投资客户，所以造价也会适当高出一

些。

3 : 自住房屋如不做个性化的一对一咨询采购，可按投资类走，做全案打包，降低部分费用。

在郑州的自住型客户家装设计费起步价 200 元/平米，以克勤克 俭为原则，基础硬装在 800~1000 元/平米是可以完成的。此外， 会产生采购活动家具、灯具、窗帘等软装和家电的费用。
