# 167.s 老师请教下，尔湾

s 老师请教下，尔湾 172 和华府 178 以长持的角度来说那个有更 高的天花板，自我感觉差不多，都在同一区域，配套和户型 也各有短处和优点，想知道您的看法？

尔湾： 板块和户型毋庸置疑，三成熟的坑洼路面，阻碍了当前 真实需求，但一旦发力就是再次飞起。要说缺点就是开发商的 靠谱性，多关注于酷炫的外立面等前卫元素，内在实际的东西 或缺。 比如，同时期发售的建业天筑 VS 瀚海晴宇，作为当时 高铁区豪宅代表项目，如今天筑居住品质和二手交易都算得上 不错，而晴宇至今无自来水，无正式电，马上冬天来了无暖 气，前几天在给业主送小太阳弥补点电费，产证还是未知数。 书香华府： 板块由三成熟逐步变为五成熟，第一轮赚钱效应这 一年来已经看到了。单价有所上升，首付比之前有所提高。但 价值支撑非常扎实，利好在三年内不断释放，所以目前依然是 很好的入手时机。就单价而言，比尔湾略高。成长速率就去年 相比，会放缓一点。但买房不是炒股，希望速成的是穷人心 态。
