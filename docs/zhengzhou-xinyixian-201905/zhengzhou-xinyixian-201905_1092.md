# 1091.老师，晚上好。我是

老师，晚上好。我是今天新加入的星友。有关些问题请教 您。现有北三环片区一套 2 房。在广州工作，有可动用现金 35 万左右（努力周转可到 40 万）。估计过 2-3 年回去郑州发展。 想买一套小三房自住。但现在没有名额，打算把家人的户口 迁来郑州一个。

请问您是否有可靠迁户口中介呢？ 另外您有什么地方或楼盘推荐吗？ 十分感谢。

评论区：

1 : 第二个问题漏了。 小三房可以看一下白沙和滨河。价格较低，空间较大。

2 : 滨河国际新城老师有哪个盘可以推荐吗？

3 : 前 2 周在星球里有说过 1.1-1.2 的保利春风十里，还有 1.4 的绿地澜庭。回报率会比较好。 这几天几乎没有了，保利还剩下

几套小四房，1.4/平总价可控，也不错。其余都是 1.6-1.9 的盘了。

户口方面，你有房子，那么就自己可以直接落户，然后你的直 系亲属比如你的父母就可以迁入。这种方式费用是零。 如果你 不想落户口在郑州，仅想家人迁入，那么学历在中专/技校及以 上即可，如果落地不流畅，私信我。
