# 1657.咨询：夫妻二人去年

咨询：夫妻二人去年已办离婚，名下各一房产，媳妇名下房 还未交房，户口依然还在我名下的房产，这个时间老丈人的 户口还可以签到我的我这吗！派出所会查吗？

评论区：

1 : 如果媳妇这边的房还未交房，可以凭购房合同先迁过去户口吗？

2 : 我记得是这样的，如果原户口和新购住房在同一个区，拿购房合同可以迁去，成为一个单独户主。但是跨区，就不行。

不过政策一直在变化，你询问一下具体片区的户籍警，他们有最新的答案。

3 : 郑州市区内户口，可以凭购房合同先迁，你媳妇的户口可以先迁入，前提是房子在派出所备案了门牌号。但外地户口必

须等房产证下来后才能迁。

4 : 对 还得先有备案 需要具体问本片区户籍警

迁户口的时候，需要你出示你和太太的结婚证来证明亲属关 系。 如果之前的结婚证盖上了作废章，那就不能用了。你俩再 结一次吧。
