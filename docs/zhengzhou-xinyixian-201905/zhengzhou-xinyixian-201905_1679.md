# 1681.女神，你好，在郑州

女神，你好，在郑州，三环内的老破大有投资价值吗？比如 二环边地铁口，无电梯的顶楼六七楼复式是三楼价格的 7 折， 十年房龄，可以投资吗？（郑州已有五套 140 平以下的刚需 房）

评论区：

1 : 谢谢，我神 [愉快][愉快]

2 : 大部分银行现抵 15 年以上只能 5 成，超 2 百平降一成。

当我们说“老破大”的时候在讲些什么？ 1、一线城市老破大单 价和中等品质楼盘差价很大，很低的价格基数具备投资吸引 力。 2、一线过去永远涨, 大水在三五年内灌满每个洼地。老破 大只要被灌到，单价增幅*面积数=可观收益。 3、前期成本 少，核心在于评估贷款环节，不能以低成本买入的老破大不是 好的老破大。 以上三个维度对比分析二线城市郑州 1、三五年 内属于中等次新快速增长的时期。老破大单价跟中等次新目前 还未拉开足够距离，吸引力不充足。 2、二线城市水灌没有这 么快，买入老破大最佳时期是在牛市前期，以小博大。否则需 要等待比较久的时间。 3、如果抵押环节你可以做到前期成本 极低（具体咨询你的中介 了解评估价格），那么买一套小赌怡 情，可以。
