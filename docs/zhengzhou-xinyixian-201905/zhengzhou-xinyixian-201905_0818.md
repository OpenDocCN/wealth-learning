# 818.S 老师 我想请问以

S 老师 我想请问以卖为前提的时候 什么样的房子适合凤变冰？ 有无刚需盘刚改盘的区别呢？ 因为我有个常西湖片区房子今年想卖掉 今年该区域二手市场不太好 地铁、学校都在修建中 在考虑是否有必要凤变冰一下

评论区：

1 : 冰变，结合房屋的具体户型和区域特点，小投入达到好的美观效果，没问题的。

对于冰变，刚需盘和改善盘的区别在于溢价数字。 大户型改善 盘的提升总值更为明显，有时候可以达到惊人。但对于产品设 计的整体把握，要求比较高。 刚需盘投入小，提升总值也要小 一些，但按比例来看，也不错。 楼市淡季期+房子区位不佳， 面对的不是能多卖多少钱的问题，而是不打扮漂亮根本就无人 问津的问题。
