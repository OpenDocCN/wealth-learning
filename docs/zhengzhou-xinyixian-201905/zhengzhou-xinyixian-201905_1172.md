# 1172.S 老师好，请问全款

S 老师好，请问全款郑州商铺/写字楼可以抵押贷款吗？抵押 最长年限是多久呢？哪个行利率较低？谢谢指点！

评论区：

1 : 商铺评估价五成（评不高），超一百万经营贷，一百万内消费贷。 利率上浮六十。 放款后存单质押。

2 : 约年化 9%左右

3 : 是哪个银行的？

目前信贷政策严格，大部分银行目前不接件。 写字楼商铺抵押 成数不超五成，部分要求必须装修已租赁状态。 民生银行写字 楼可做经营性贷款，客户要求有实体经营且经营状况良好，不 要求必须有营业执照，利息 6.2 里到 7.2 里（大概） 临街社区底 商或优质市场商铺可做浦发银行，负一层，虚拟商铺不做，利 息 5.8 里，三年循环贷。 但难度很大，客户经理不接件。建议 找贷款中介，私信我给你发个。
