# 1262.关于房价 一

关于房价 一、房价支撑：居民收入+房贷+赚钱效应

二、郑州现状：100 平，150 万，首付 50 贷款 100，月供 5300 三、几个假设： 1、若 M2 不再脱缰野马，保持在 8%左右，居民收入提高幅度 也当在此附近，按目前普通三口之家年收入 15 万计； 2、省会城市向北看齐，首付 7 成，贷款 3 成；

3、房住不炒，每人只允许拥有 60 平方的私有产权住宅；

4、新建房屋不得买卖，只能出租（郭嘉不耽搁土地财政）

四、结婚：房价还会想以前的涨吗？

假设不太成立。 M2 实际增速比官方公布的要大，年轻艺人动 不动 2000w/部电视剧，价格膨胀很直观。 房住不吵，参见新加 坡，有房和无房形成阶级差别，无房的有房住，但一辈子买不 起，买不起的意思就是房子继续很贵。 未来房子还不会像以前 这样涨？ 如果说是像 16 年这样爆发性普涨，那肯定是要等，但 具体到标的，我们可以做到每年都有优质涨幅。 举一个最实际 的例子，排除普涨年份，仅以每年 10%的保守涨幅来做测算。 第一年，以首付 100W 作为资金成本，购入一套 330W 的大三房 住宅（郑州）。 第四年卖出，大概剩余 225W。以此资金作为 首付，购入一套 740W 的房子，第七年卖掉余 500W，以 500W 作 为首付，购入 1650W 房子一个，到第十年房子价值 2196W。 以 上数字是以无脑买无硬伤房，无科技树，也无附加值高的操作 为例。收益可以参考。
