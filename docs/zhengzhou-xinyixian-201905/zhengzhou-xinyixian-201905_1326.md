# 1327.泰禾，某某府，开盘

泰禾，某某府，开盘卖了两套，一个月卖了四套，一个多 月，卖了 6 套。从招聘到项目开发都是北京公司占主导地位。 销售成绩看起来贻笑大方，但想想其背后，确有其原因。

北京一手盘是可以无视地段的，面和线？不存在的。 周围二手 7W，新项目 100000+，OK 的。 土地供应严格控制，土豪买家源源不断，支撑其价格的价值 是多维的，单看其平面价格，那是支离破碎。

在此路径依赖下，在郑州项目做出如此定价，也就不难理解 了。

降维打击四个字，不是简单的路径复制。作为投资人，真金 白银上战场，则更要看清规律本质。 “学我者生，像我者死。”并非儿戏之言。
