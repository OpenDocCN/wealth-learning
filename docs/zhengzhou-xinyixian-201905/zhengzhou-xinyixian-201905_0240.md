# 240.S 老师 为什么大玉

S 老师 为什么大玉米周围的房子涨不过 西抵港附近的房子呢 能否讲解一下

评论区：

1 : 大玉米周边双气的房子，有推荐吗？

这个问题太有趣了，我观察了 4 年数据变化。 首先，你提问的 方式很好，问的不是“为什么不涨？”而是“涨不过 cityon 周边” 我 已经听了数位问我为什么不涨？？（本地人）和“为什么这么 低？这么棒的夜景房！这么高级的 view！” （北上深人） 真实 的现象是：1、他涨了，而且以短期回报比来说，很惊人。16 年是 8000-9000/平遍地走，负首付，高租金，正现金流。半年 翻倍。 2、18 年回落较明显，价格停滞，二手出售难度较大。 租金依然还不错。 预判：短期两年价格无起色，主要靠租金。 为什么短期猛涨： 属于目光可触及的繁华东区，牛市一来就飞 天。 为什么平稳期总是涨不过 ciyton 周边： 根本原因在于东区 人群的购买力远大于板块房屋均价，对于居住氛围、小区光 环、板块光环、洋房舒适度的溢价完全乐意买单，大玉米周围 就算是离上班再近，可是就是不喜欢没绿化没居家氛围，不喜 欢隔壁有公司办公，便宜也不要，就是这么任性。
