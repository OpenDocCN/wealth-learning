# 1622.郑州，这十年来地段

郑州，这十年来地段财富版图一直在重新架构，城市中心的 转移毫不留情。超额利润在未被深耕过的区域。 S 老师，请问您一直推荐的东望、象湖一号符合未被深耕过的 区域、后续价格上有爆击的可能忙？是现在出手好还是等等 明年上半年出手好呢

评论区：

1 : 象湖一号需要全款

2 : 是的 可能性比较大；东望有转让的可以贷款。

3 : 为啥在链家上看不到象湖

4 : 因为还没进入二手市场 还是一手的嘛

5 : 目前郑州市场还是限购的吧？

6 : 新盘为啥需要全款？

7 : 因为这个价格太便宜 买的人太多

8 : 是 东区二手可破

这俩盘目前都属于价格实实在在比较低，在楼市平稳期，近郊 盘会挤掉不少泡沫，短期半年可能无利润暴击（区别于书香华 府二手房），但两年后交房之时会有明显利润。首付成本低， 所以是不错的选择。
