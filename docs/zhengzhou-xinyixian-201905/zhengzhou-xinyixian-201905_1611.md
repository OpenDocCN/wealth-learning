# 1613.请问 S 老师对郑州航

请问 S 老师对郑州航空港区房价怎么看，目前北港大概均价 11000 左右，南港目前好像只有两个楼盘，对港区产业经济现 状不够了解，对港区规划的框架及未来可能的影响也不清 楚。与港区京港澳高速一路之隔的薛店镇，工厂较多，外来 人口集聚，在港区工作住在薛店镇的人也不少，目前楼盘大 概均价 6000 左右，我老家新郑薛店村里的，考虑在老家置业 一套，最好能自住投资兼顾。由于长期在外工作，在家时间 很少，最近开始考虑这方面事情，想听听 S 老师看法。另请问 郑州接力贷这种产品有没有，刚过 65 岁的老人有没有渠道可 以办理，本人户口因工作已迁出不满足限购条件。

评论区：

1 : 老人 60 岁以上做接力贷就很难了。65 岁不能做了。今年比较严。

当我们做价值判断的时候，首先要有对的现象材料。 “北港均 价 1W1”不正确。 价格要看二手成交价，而非一手开发商价

（倒挂情况除外）。想买港区的话，去买二手，均价打七折。 老家置业的目的是？纯消费不必考虑投资回报。有实际的居住 需要就买。 四线城市没有房地产。
