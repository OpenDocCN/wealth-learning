# 1567.上海户口来郑州出差

上海户口来郑州出差 3 个月，手里 80w 现金，想咨询投资方 向：入一手还是二手？看东区还是市中心？

首先，东区就是市中心。郑州市以中州大道为分界线，分为老 城区和郑东新区。但，“新区”已有十余年，不新。新娘不是 娘，老婆饼没老婆。 财富制高点、高收入岗位、改善型住宅集 中于东区。东区核心区价格高，可以选择近郊城乡结合部，目 前出于政策缘故一手价格很低，万元的价格很划算。东区人民 对于房屋总价的承载力比较强。买两套扔那里，待三年再来 看。 也可考虑成熟区二手房，不过就是需要淘，以提高杠杆 率。 另：无户口购房，郑州政策这几天已松动。可以观察一下 后续，洞在不停的挖。买一手就问你的置业顾问，买二手就问 中介，他们会有办法的。
