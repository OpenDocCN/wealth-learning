# 1112.请问星主：

1，郑州相对其他二线城市，他的核心优点是什么。

2，18 年的市场现状是什么？

3，有哪些异于其他二线城市的机会点。

4，主流的投资者（比小白强一些的）心态如何。您可能了解

的多一些，还望赐教。

1、郑州的优势在于无可争议的人口基数巨大，在校小学生数 量遥遥领先，单核城市，全省首位度第一，无分流，房价基数 相对低，产业发展蓬勃，易于获得持续的、确定性强的收益。 2、18 年信贷政策较紧张，贷款和抵押环节不太友好，价格或 有波动。 但对于熟手而言，平稳期反而是黄金时代，笋多易 抓。信贷方面，我们有专业团队帮助大家实现利益最大化的目 标。 3、经过 16 年市场教育，投资者心态被激活，对于构建以 房贷为核心的资产包有了初步认识。 最近一周开盘的数据可看 出投资需求依然活跃，名门翠园和华南城中原几秒钟清盘，只 要价格不太高，房子要抢。
