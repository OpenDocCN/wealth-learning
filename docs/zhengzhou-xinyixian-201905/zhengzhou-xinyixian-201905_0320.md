“平日时间要有大量用于自己阅读和沉淀”

好奇 s 美大量的阅读都是哪些方面？可以推荐一二 我家客厅是没有电视的。 最近几天在读《城市里的陌生人》与

《空间的力量》。 前者是一本人类学著作，讲社会生产与流动

人口的空间重构。从上个世纪的数据开始收集，直到现在。有

很多细腻的量化分析与有趣的现象综述。 包括女性权利空间的

变化。 后者是一本经济地理学论文合集。比作者的《大国大

城》难读一些，研究方法和理论较多。但很值得读下去。 对于

市场的分析，价格的分析，未来楼市的溢价推断无一不是小小

的田野调查和考证。如同，近期几个月有数位朋友提出老城区

二手房有“下跌明显“。认真看星球的就会发现，此现象早有预

警，星球内强调购买的其他区域和明盘楼盘大半年来上涨明

显。区域个体差异极大。 而就其部分区域二手房价格有所“滑

落”缘故，我看到的，除了之前星球所述预判，有新增因素。

中介因素与前两年产生较大变化。今年，开发商普遍大量签约

中介店，让中介来卖新房快速成为普遍现象，中介向来以快速

成交+高提成为目标导向，自然纷纷重心转移，二手房带看动

机和积极性显著降低，量持续下降几个月必然传播到价格变

化。投资环境中 ，有 N 个因素会影响价格波动。房东急着卖不

出，以为是价格问题，其实是交易环节中的因素发生变化。尤

其是总价不高+成交量本就不大的区域，会较为严重。提成少

+带看费时成交难，房子会被中介忽略。
