# 1606.请问斯嘉丽，打完自

请问斯嘉丽，打完自有资金后，下一步只能开发 xyk 和 xyd 了 吗？这样的话使用多少比例是安全的？您是如何规划现金流 的？

评论区：

1 : 多少比例因人而异的，与自身和家庭其他成员的收入，负担等情况有关。只能自己把握好分寸

2 : 怎么运营到租金超月供的[呲牙]

你不留月供吗。。。 XYD 和 XYK 可以申一申，尤其是低息的 XYD，作为资金的补充。 我呢 1、会留够资金作为月供的 2、 部分房屋拿出来运营，租金超过月供，正现金流 3、持有的标 的由于涨幅不错，负债比较低，房屋可抵押出来的钱全部超过 原有贷款。后期可选择部分截断抵押，以更好利用资金。
