# 1334.近期一两个月，好几

近期一两个月，好几个朋友提到省内三四线城市房价 8000+的 现象，洛阳、许昌、南阳过万。

虽然一手价格和二手价格是不同的评估体系，但深挖一下， 三四线城市之前陈旧的产品和现在新式住宅是有比较大的差 异，人民愿意为品质付出真金白银，符合消费升级的需求， 不能机械看二手老旧房子价格。

价格对比之下，郑州好多洼啊[嘿哈]

货币化安置，不可能出现爆发式的上涨。等货币化安置完成，这股洪荒之力也不可持续了。”
