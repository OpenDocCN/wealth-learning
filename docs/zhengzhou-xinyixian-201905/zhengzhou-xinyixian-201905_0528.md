# 528.郑州好多一手楼盘有

郑州好多一手楼盘有一些中介可以拿到比售楼部低不少的价 格还可以正常贷款，只收取一些好处费，有的项目还可以前 期定门定户但是对比售楼处的价格还是低很多，包括一些性 价比不错的楼盘都有这个现象！这个怎么解释？（自己心中 也有一些答案）

不是中介“拿到”，因果关系错了。 第一种是，是有员工/领导/ 内购时期买的老客户/利益相关的人有一些相对低价格的房子， 想以转楼花的方式转出去，赚个块钱，所以挂到中介那里找客 户。这种转让价格会明显低于售楼部释放价格，或者售楼部压 根就不开盘，都在私下转让中去化完毕，以北龙湖的诸多限价 楼盘为例比较多见。 第二种是，售楼部价格与中介那里价格没 差异，中介店从自己的提成里给客户一点返利。这种是因为现 在一手楼盘普遍会让中介店作为一种分销手段，因中介店遍布 城市，人数众多，可以以比较低的成本将楼盘信息快速传播到 市场。对于中介来说，都是一样的卖房子，卖一手房更简单快 捷，也乐于去做。
