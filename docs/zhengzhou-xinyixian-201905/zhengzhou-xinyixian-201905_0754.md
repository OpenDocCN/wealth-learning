# 754.S，有其他投资人认

S，有其他投资人认为杨金康桥位置一般，周边环境不好，怎 么看这个说法

这个问题很有意思。 从小片区区域来看，确实不如东三环以东 的鸿园片区低密、整饬、环境优质。鸿园片区未来是仅次于北 龙湖的低密度+豪宅聚集板块，天花板足够的举高高。 目前 1 年 以内，鸿园片区除了尔湾，没有性价比高的一手房了。不出 2 年，就会出现新开高层近 3，洋房近 4 的情况，对于原住民来说 是喜笑颜开，但对于彼时建仓者来说恐有肉痛。 尔湾本身从区 域、户型到价格，单从盈利来说，有明确的空间。但其也有不 可忽视的问题，证件推进太过缓慢。有错失机会成本的风险 性。 康桥此盘，虽然从完整性及自然环境因素弱一筹，其地理 位置确是距离北龙湖更近，补回一分。证件推进更为可控，加 分。区域客户与杨金西板块大有不同，从客户画像支撑点来 说，与东三环以东区域别无二致。 基于其具体户型来说，如果 选择到最具盈利价值的标的，此环节溢价也可提升 30%+。综合 来说，是可以考虑的。
