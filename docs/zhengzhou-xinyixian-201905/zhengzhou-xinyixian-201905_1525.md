# 1526.教瘦老湿，白沙的楼

教瘦老湿，白沙的楼盘（比如东望或者其他的）和名门翠园 给分析下吧 个人情况：自住为主，但是两三年内不住。也 带有投资的成分，哪个好些？

白沙的东望等楼盘，对于东区工作的人来说，是非常实惠又便 利的选择，但是三四年之内没有像样的租金收入，对于租金很 在意的人则需要计算一下。 翠园对于老城区工作的人来说，是 还不错的，价格比去年预期的便宜一些，地铁、学校皆有，不 住的话租金没太大问题。 如果投资，我较偏好买大面积低单价 的白沙。但如果你已经有一些类似产品，想配置流动性好的作 为补充的话，翠园选择紧凑型户型，租金/总价溢价未来还是可 以的。
