# 1178.请问首套用公积金贷

请问首套用公积金贷款，再买第二套时用商贷还能按首套的 首付和利率算吗？记得星主曾经这样说过，但也有很多人说 用过公积金贷款再办商贷就不能再按首套了……到底什么情 况？求答疑！

首套和首贷是两个事情。 之前不限购数量的情况下，公积金贷 款买了不上征信的话，再商业贷款就是首房首贷。 但后来限购 限贷之下，名下已经有一套房子了，不管你有无贷款，有的银 行都会按照二套去执行，是这样的。但是呢，少量银行可以按 照首贷（公积金贷款没有上征信的情况下，这个有时候不一 定，大部分不上，少数情况也不排除上征信，那么你自己先去 查询一下，或者说抓住时间窗口两套一起）。 房管局那边则是 按照二套缴纳税费。
