# 1500.一句话概括各城对待

一句话概括各城对待人才的态度

1、北京：我送你离开，千里之外……

2、上海：我欢迎你默默的离开。

3、广州：我希望你是有耐心的人。

4、深圳：来呀，快活呀，反正我有大把时光。

5、南京：我真正的菜是研究生哦。

6、杭州：我最爱的是博士和硕士。

7、天津：北京不要你们，我要你们。

8、石家庄：北京天津不要你们，我要你们。

9、成都：我这的火锅和美女都在向你招手哦。

10、重庆：我特么只引进了一群炒房客。

11、长沙：我钱不多，意思一下，你快来嘛。

12、武汉：我这有 20%折扣的低价人才房，速来。

13、合肥：我这有 20%折扣的低价人才房租补贴，速来。

14、郑州：我爱“双一流”大学生，落户买房就送补贴。

15、济南：我只给博士和硕士发补贴。

16、东莞：我已盛开，蝴蝶自来。

17、福州：我只给创业创新人才租房补贴。

18、南昌：我只给你户口，不给你补贴。

19、西安：我这有兵马俑，你来不来？

20、鄂尔多斯：我这有的是房子，求求你来吧。

21、拉萨：我这有诗和远方，等你哦。

22、长春：老铁，你别走啊……

23、沈阳：老铁，你别走啊……

24、哈尔滨：老铁，你别走啊……
