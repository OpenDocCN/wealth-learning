# 1145.星主，龙子湖 2 万多

星主，龙子湖 2 万多的高层和龙子湖北部 2 万多的叠拼或一楼 洋房该怎么选？

这个问题选择涉及到一个综合效用的衡量。 星球中的回答一般 是针对单个楼盘的价格或价值作出判断，但要做出利益最大化 的配置是要看整个家庭的全链条投资情况。 有的时候我们筛选 出来 ABCD 楼盘都是好的，但对于甲而言，A 楼盘最适用于他， 对于乙而言，则可能是 C 能给他带来最大利益。 以此问题为 例： 单个看这两种标的，都是不错的。 龙子湖 2W 高层对于投 入资金中等，且对于资金现金流有要求的人来说，是合适的选 择。可追求到稳妥的房产升值收益+租金补贴。未来贾鲁河修 建好，还会有一波价格拉升。 龙子湖北叠拼（我猜你说的是清 水苑）或一楼洋房（带院子），对于自己有置换自住需求或心 态稳、愿意以时间来换取高总值收益的人来说，非常好。他的 增长总值有惊喜，但你要等待 2 年去换。延迟满足体现在这 里。 这也是大户型/偏豪户型的一个妙处，片区一旦激活，他 的增长可能就是半年两百万。在 16 年这一波已有体现。更大的 T 户型在这三年内则默默从 600-700W 提 1200W。当前此价格成 交或有难度，但价格是三维的，有的时候时机一到，又是一个 势如破竹的跳跃。
