# 94.S 大 善水上境降价

S 大 善水上境降价了还分期？听说郑州楼市的大白之前三万 多买的 是真的吗 他们公共号自己人买的时候没有内幕消息吗

楼市跟炒股炒币不一样 股市和币圈，“内部消息”真假参半，大 概率是以内幕消息为名来割韭菜。 楼市投资要做的好，一个是 看资金实力，一个是看选筹水平和对金融的理解。跟所谓内幕 消息关系不大，况且开发商有什么内幕消息？投资和开发是两 条线，完全不同的领域。贴近一个领域，看似切的很近很近， 实则隔了一条银河系。打个比方，中介、置业顾问接触的都是 一线真实的数据，so what？写文章的记者同理。 善水这个盘， 高层一两年前卖三万出头，即使不降价，也是妥妥的站岗，很 正常。之前早就说过。
