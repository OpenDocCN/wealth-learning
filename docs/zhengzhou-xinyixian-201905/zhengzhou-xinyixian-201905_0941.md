# 941.京沪广深大城市，永

京沪广深大城市，永远都会涨。 无论经济如何变化，有个基本面是肯定存在的。五亿多农 民，没什么大文化，又要年年增收，住小楼，开汽车，吃海 底捞。呵呵，农民工现在可牛气了。那么，有什么产业能够 维持这么多人就业，且是高质量的岗位。年年加工资，年入 十五万！我们一一分析： 一，农业，不可能。现在土地要大面积流转，前提就是农业 不值钱。如果农作物值钱了，怎么可能流转。粮油涨价，cpi 一动，就要扯出利率问题，人民币估值就要立即动，外汇也 要动，那可是大的无边无际的事情了。 二，重工业，不可能。中国的重工业产品，主要是金属和化 工，例如唐钢，石化。这些东西有竞争力的前提是，环保呵 呵了。如果我们按照欧美标准搞化工厂环保，这些东西是没 什么实际竞争力的。如果五亿农民工，大量的人搞重工业， 环境是难以承受的。现在，唐钢基本消停了。小化工基本废 掉了。重工业不可能承载这么多就业。 三，轻工业。裤子皮鞋玻璃杯。以前我们积极搞这个，虽然 满足了快速大面积就业，但是轻工业利润不高，无法满足农 民增收。劳动法一来，各地小老板一片片的躺死狗了。 四，m2 主导的基建，必须挑重担。这个厉害了。m2 是自己印 的，基建是大家喜欢的，农民增收，充分就业是可以的。那 就搞了。 五，这么多票子往哪去？现在，各行各业都是产能过剩，就 连游戏行业，也是九死一生。那么多普通人拿到了钱，能够 创业吗？不可能的。除了买房，一般家庭还有什么可以留给 子孙的？ 结论：暂时的报数字需要，不会改变经济大趋势，不会改变 老百姓的真实需求。
