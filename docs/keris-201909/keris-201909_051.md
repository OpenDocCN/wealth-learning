论如何一天确定婚纱照

```
我是一个"温柔"的急性子，从来不生气，但做事情总是雷厉风行恨不得一下子完成。原本婚期定在 12.30，后来两家算算日子又改到 10.1 啦，男朋友的妈妈也是那种很有主见的人(对我还超级好)，这一提前了三个月意味着现在就要开始筹备各项结婚事宜，婚纱照首当其冲。

其实我也没有刻意去搜，就是看朋友圈广告有弹出婚纱照公司，就加进去简单了解下，结果天天都是扫码优惠进店送礼拼团低价等等信息，总感觉哪里不太对，但因为一直比较忙也没仔细研究。直到前天晚上十一点左右终于空下来了，就在网上搜搜关于婚纱照的评价，才发现原来有这么多坑(⊙o⊙)

最常见的坑就是打着低价团购的名义，看起来一两千就拍下来了，但是处处包含隐形消费，什么婚纱礼服选择有限，多了或者想换好的就要多加钱，好的彩妆要加钱，场景要加钱，洗片子要加钱，要不然就是摄影师一天接 N 个活儿，发给你看的不是客片而是找模特拍的，千篇一律的风格，拍照还要排队…总之算下来一样要花八九千 重要的是还闹心！

吓得我赶紧把之前加的微信删掉了，开始搜口碑比较好的婚纱摄影工作室，(我的好感比较奇怪，有时候看两眼就会有种某种感觉，一旦觉得什么很好就会坚定不移的认准这个) 也就搜了十分钟左右吧，就看准了一家比较有名的私人定制婚纱摄影(不说名字啦以免有打广告的嫌疑哈哈)，添加客服微信之后就睡了。昨天一早看客服加上我了，就先看她的朋友圈，里面的客片每天都有更新而且原创氛围浓厚，又再网上搜搜评价，这家基本不怎么宣传但口碑很好，一对一定制，婚纱照拍出来像大片，从不降价却没有任何隐形消费。

于是询问档期、价格、和包含内容，感觉对方很专业态度也不卑不亢，发来的套系报价单每一项都列的清清楚楚(服装不分区，换装定型套数，摄影团队质量，拍照修图数量，照片装裱，路线场景选择等等)，仿佛你能想到的内容全部涵盖进去了，剩下的就是选择地点、档期和套系。因为我喜欢旅行，婚纱照也想采用旅拍的方式更有纪念意义，对比了本地、丽江、大理、海南、厦门的套系内容，基本上价格在四千到两万不等，对应服装有两套到九套，拍摄时间可以一到三天自由选择。本来上午我在外出工作，只是在车上才有时间简单询问，可能就是某种感觉对了吧，到中午回来就决定先下定金预定这家，后面决定档期和行程。(把我主管都看呆了哈哈，怎么早上刚加微信这就定了？！或许某种信任感就是来往几句话就建立起来的吧，后来才知道我加微信的这个是本地店的店长，我觉得人挺不错的ʘᴗʘ我还跟主管说回来看看效果好不好，好的话蛊惑他也拍吼吼)。

选择地点就比较纠结啦，因为男朋友在香港读研，只有四月中下旬几天放假，火车飞机有些城市也不直达，所以要多考虑。提供旅拍的地方我基本都去过，比较倾向于大理和三亚。 我是那种喜欢操心安排行程的人，海陆空每种班次都研究清楚，得出结论这两个地方都不好走蒽，让男友选，说没有去过大理，所以最后就定在 4.21 号到大理拍啦^o^(我俩认识十年，恋爱近六年，其中五年都是异国异地恋哈哈，感情还是超级好。刚确定恋爱关系没几天他就回澳洲了，三个月后放假回来我俩就去的丽江泸沽湖，对云南也是蛮有感情的) 

微信给主管请公休假再次把他愣掉了(⊙o⊙)他一边说婚姻大事重要，一边开玩笑的问你这么草率你爸妈知道吗？答：兜里有钱心里不慌哈哈，我还问男友妈妈说能不能把他儿子拐到外地拍婚纱照呐^o^(虽然刚工作工资不多，但凭着我不逛街不化妆，花钱只买书只旅游的份上还是攒了不少哒~还是花自己的钱舒坦，爸爸和男友妈妈说要赞助我，都被我"无情"的拒绝了哈哈)

其实整个算下来也不到一万块钱，套系选择的 6900(含五套，因为我觉得一天五套衣服就比较满负荷了，服装太多换衣服换造型也挺累的，如果选择两天拍摄我时间不够)，含两晚客栈住宿，所以我只用安排行程就好，决定周五晚上下班后先飞昆明，第二天动车到大理选衣服，第三天拍摄，第四天选片子回昆明，第五天回家，男朋友从深圳往返昆明。婚纱照行程机票车票都定好，就开始跟男朋友嘚瑟啦吼吼，这也是最开心的时刻^o^

就这样一天把婚纱照确定下来啦，当然最终好不好还有待证明，不过雀跃的心情是真的好。分享一下心得，希望对有缘人有所帮助~
```
