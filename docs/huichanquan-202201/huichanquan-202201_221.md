# 一份教你识别诈骗电话的方式

请收好！ 

👇

**未知境外电话多为诈骗**

**以“00”开头电话多为**

**境外 IP 电话或非法网络电话**

**一般都是虚拟号码**

**所显示地址也不一定真实**

**有些甚至是不法分子**

**通过部署在境外的服务器或交换机**

**接入个别国际运营商网络拨打的电话**

**![](img/6fc7fab79aec9209f1b3684e21745198.png)**

**这些境外电话存在不断变号的特点**

**往往一个国际号码只呼叫一次**

**单个号码的骚扰特征不明显**

**很难通过传统标记号码**

**或黑名单等形式进行屏蔽**

**![](img/e274658c83acd1166f434727d7899dff.png)**

****警方提醒****

****1.陌生来电不轻信。收到以“+”号、“00”开头的来电或短信，务必要提高警惕！****

****2.未知链接不点击。诈骗分子往往会先通过社交媒体添加好友，随后再发送未知链接让事主点击设计好的钓鱼网站，或者下载有涉诈风险的 APP。****

****3.个人信息不透露。不要在未知的网页或者 APP 上填个人的信息、银行卡号、验证码等。****

****4.转账汇款多核实。凡是陌生来电、短信、未知网站、非官方 App 涉及到钱财的，一律多留心、提高警****

****5.下载“国家反诈中心”APP，收到可疑诈骗分子来电、短信或可疑 APP 应用时，国家反诈中心 APP 可智能识别骗子身份，提前预警，大大降低受骗的可能。****
