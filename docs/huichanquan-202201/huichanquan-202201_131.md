# “空姐都是玩物！想睡哪个，我安排！”飞行部领导侮辱空姐录音曝光

> 原文：[`mp.weixin.qq.com/s?__biz=MzIyMDYwMTk0Mw==&mid=2247528189&idx=1&sn=640ce84ab5dc5b2011c0fa51b98ac65a&chksm=97cba5c5a0bc2cd35a462e5a2955f12aa2551e0802db410aeb24571df25f8f3e474503a77f1e&scene=27#wechat_redirect`](http://mp.weixin.qq.com/s?__biz=MzIyMDYwMTk0Mw==&mid=2247528189&idx=1&sn=640ce84ab5dc5b2011c0fa51b98ac65a&chksm=97cba5c5a0bc2cd35a462e5a2955f12aa2551e0802db410aeb24571df25f8f3e474503a77f1e&scene=27#wechat_redirect)

一则关于某航的乘务人员曝光了某飞行领导给飞行员讲话的的惊人语录，引发网友关注。

领导开会当着所有飞行员尤其是年轻的副驾驶说：“对呀每次喝完酒我就会给他说，玩嗨了有本事没事，但是没本事玩嗨了有事！”

“你知道我跟他们副驾说什么？我说你们只要是想去泡妞，管你北京的，广州的，只要是能泡，我就可以给你安排！”

更加恶心的还在后面！“你看上了哪一个空姐，没关系你想跟她一起飞航班，没有关系，你来找我，我给你调班！我都同意！”

“这是娱乐，动物从根本上讲就是性！不要泯灭动物的本质，从古到今，无论怎么发展，科技可以变好，飞机可以变好，但是有一点是什么泡妞不变！”

随后还不忘了告诫一下：“我可以帮你安排，但是如果事情泄露了，你不要过来找我，扯不平不要来找我扯犊子，我喜欢有本事的人，有本事干完你啥事情都没有，这人厉害！”
