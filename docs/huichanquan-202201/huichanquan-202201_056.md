# 传销里的女孩，都是“迫不得已”的吗？

**一：老实人出狱后**

再次见到周超是他三年出狱以后，真没想到从小到大这个家人、朋友、同学眼中的腼腆老实人，会有判刑坐牢的一天！之所以被判刑的原因，是周超拿板砖拍了一个美女的脑袋，故意伤害他人身体被判刑三年。

出狱后的周超感觉比以前更白了，也更健谈了，从之前的腼腆大男孩，蜕变成一个成熟小伙子，不得不感叹监狱里也是一个历练人的地方。

**我问他：“后不后悔，拿板砖拍人？”**

**他扶了一下鼻梁上的眼睛，还是有点愤恨地说道：“一点都不后悔，唯一后悔的是心疼那一笔赔偿款！”**

周超的回答在我的意料之中，如果打人不犯法的话，我也想替他给那个女的一板砖！

**二：一个叫玲子的网络美女**

这件事的起始还要从 4 年前说起，那是周超大学毕业参加工作的第二个年头；26 岁的年龄，从出生到现在足足单身了 26 年。内向、腼腆、不愿多言、老实沉默的性格，以至于他在男女比例 1：9 的大学校园里，都没能脱单。步入社会，他所在的公司，从事的工作让他一天能见到一个陌生人的女性都难，更不用说找对象了！又没多余的兴趣爱好，不愿意参加社交活动，每天都是公司和宿舍之间 2 点一线的生活，重复且单调。

一天周超正在办公室办公，QQ 上响起了好友申请的提醒；点开信息一看是一个年龄显示 25 岁，网名叫玲子的女生，看 QQ 头像是一个大眼睛披肩发的美女头像。

周超怀着激动的心情，通过了美女的好友申请；出于男人的本性，周超先点开了对方的 QQ 空间，想去空间里验证一下是不是头像上那个美女本人。进到空间里，翻看一下最近几天对方发的动态，周超很确定 QQ 头像上的那个美女，就是 QQ 号主人本人；单身多年的周超心情更加激动了，能有美女主动加自己好友，那感觉就一个字超爽！

老实男人的聊天都是从一句“你好”开始的，周超也不例外；周超这个人虽然平时不怎么爱说话，但是在网络上聊天是话语连篇，一句接着一句！差不多聊了一周，周超感觉找到了人生中的另一半，被玲子每天准时的早安，晚安，感动的是不要不要的。没多久周超就主动表白，想要对方做自己的女朋友；很快两个人确立了异地恋的关系！

**三：决然辞职去见女网友**

周超每天除了工作的时间，脑海里都想着玲子，幻想着能跟她见面，在一个城市生活打拼，幻想着以后能跟她步入婚姻的殿堂。总之对她的喜爱已经到了走火入魔的程度，周超期间提过几次要去她的城市看她，玲子都以工作忙先在网上处一段时间再奔现，为理由，委婉的拒绝了周超几次。直到两个人相处了 8 个多月的时间，周超期间又是给她发红包，又是快递礼物；玲子才同意周超让他来武汉看自己。

周超为此找老板打算想请一周的假，老板没同意；为了追求爱情，周超果断做出决定辞去了干一年多的工作。收拾了些行李就直奔武汉，想着以后就跟玲子在武汉一起打拼。可谁曾想到，周超这样抛家舍业去到一个陌生城市，却是自己牢狱生涯的开始！

**四：短暂的甜蜜时光**

见面后的几天，周超一直住在酒店里，玲子下班后就会来找周超逛商场、吃饭、看电影；那几天的时光是周超最开心的几天，第一次感觉到了有女朋友的快乐！

讲到这里我坏笑地问了一下周超：“你就没跟她发生一下那事？”

周超闷哼了一声说道：“没有，那时我特别单纯，出门逛街连她的手也没牵过，好像她不喜欢我牵她的手；还以为她跟我一样也是单纯害羞，当时真以为自己找到了跟自己性格相似的女生。那天正赶上七夕节，为了表达我对她的爱，我顶着烈日，走了好几条街才找到一个花店，花了大几百元为她买了一盒白色的玫瑰花，这是我第一次给女生送花！我在酒店的房间里把花送给了她，借着那浪漫的一刻，征得她同意以后，我第一次亲吻了一下她的嘴唇，真的很润！”

![](img/35248fd5c2da1d729c150ae21bff4541.png)

**五：初入传销窝**

相处了几天后，她提出要带周超了解一下她现在正在做的生意；本身就已经辞职，打算跟玲子以后在这个城市打拼的周超，想都没想就同意了，于是从酒店带着自己的行李，第一次进到了传销窝里。

周超所进的传销窝，是典型的南派传销，这个传销窝是以一片新开发的区域，附近的几个小区为据点，人员分散地租住在小区里；都是一套房子整租下来的，租住环境还不错。

就这样周超跟玲子住在了一起，当然两个人是分开房间睡的，周超跟另一个男生一个房间。不得不说，里面的人非常非常热情，对待周超就跟一家人一样，这让周超很快适应了里面的环境！

住进来第二天，玲子就带着周超开始了解起她所做的生意，经过一天的串户讲课，周超当天就感觉有点不对，貌似自己遇到传说中的传销组织。

于是周超晚上躺在床上等自己的室友睡着以后，就上网查了起来。浏览了几个网页，看了很多网友的经历陈述，周超很确定自己这是进了传销窝里。其中有个网友发布的一篇文章里写到：有的传销窝里的女性，为了发展线下；会以感情为诱饵，以谈对象的名义从网上找外地网友，哄骗他们加入传销组织！

周超联想到了玲子，但是想到玲子纯真无邪的面孔，还是打消了自己的怀疑，觉得玲子肯定不是这样的女人；虽然她现在在传销组织里，但是她对待自己的感情肯定是真的，她现在只不过是被出传销组织的利益诱惑给蒙骗住了。

周超此时没了任何睡意，手不自觉地点开了玲子 QQ，翻看起她的空间；当他查看玲子的访客记录的时候，发现访问她空间的基本都是男性，并且每天都有不同的访客；以他对 QQ 软件的了解，这些访客基本上都是对方已经加了的好友。周超在这些访客里看到了几个，竟然和自己同一个地址城市的人，周超一一加了他们的好友，想通过他们来打听了解一下玲子！

**六：老实人被骗了**

转天第二天，玲子继续带着周超去串门听课，周超假装配合没敢直接揭穿他们；中午休息的时候，昨晚上周超加了的几个同城好友，有一个叫“把子肉”得通过了他好友申请。

周超先跟他客套地聊了几句，然后就问道：“你跟玲子成为好友多长时间了？你感觉她这人怎么样？”

把子肉回复道：“差不多 3 个月之前，我跟她就成为好友了，聊了几天，她告诉我她是单身，在武汉跟着小姨一家做什么建材生意；聊天基本上都是她主动跟我聊的，看了她空间里的照片，挺漂亮的有些姿色，这女的屁股挺翘的！我说哥们你不会对这女的有想法吧？”

周超此时心情有点复杂，要知道早在玲子加把子肉这个好友之前，他跟玲子已经确认情侣关系了，没想到玲子还跟别人说自己单身！周超机械地给把子肉回复了一个“有”字。

把子肉这时发过来一个捂脸痛哭的表情，然后说道：“哥们别天真了，我虽然没见过这个女的，但是我很确定地告诉你，这个女的肯定有问题，不是浪就是骚！你翻看一下他的好友访客跟留言，基本上都是男的，通过访客就能推断出这个女的几乎每天都在加好友。这种网络上的好友，你把控不住的，可千万别跟她深交！”

跟网友把子肉聊了一会，周超脑子里已经一片空白，懊恼和愤恨已经席卷全身，周超第一次尝到了被爱情背叛的感觉，他接受不了喜欢的人对自己感情的戏耍！

![](img/a84165c8f3009885fdcb67ccf8d80828.png)

**七：爱恨情仇一板砖**

下午天空下起了小雨，玲子没发现周超的不对劲，只是跟他说了一声下午要去帮朋友一个忙，让周超自己在家休息一下，就拿着伞出了门。

望着窗外阴雨的天气，独自身处一座陌生的城市，没了感情的支柱，周超感觉到了深陷泥潭中的绝望！

从来不吸烟的他，也不知所措地拿起茶几上烟盒里的烟，大口地吸了起来，一支，两支，三支.......窗外的雨下了一下午，周超保持一个姿势不动的，在窗户边站了一下午。

楼下有个熟悉的身影映入了周超的眼帘，他站在 3 楼的窗户边上看得很清楚，正是玲子的身影；跟她走在一起的是一个瘦高个男生，两个人看起来非常亲密，手牵着手正向单元楼这里走。

周超看到这一幕，即崩溃又愤怒，一句“泥马的”脏话随口而出。甩开门就下了楼；楼道门口玲子正背对着楼道跟那个瘦高个男生拥抱在一起，这一幕像极了狗血爱情剧里男主把女主送到家门口，拥抱分别的情景。

急冲下楼的周超，顺道从堆放在楼道的一堆板砖里，拿了一块砖；助跑几步，垫脚一踹，踹在了玲子的屁股上。玲子当时就跟那个瘦高个一起倒退地倒了下去。周超目标很明确，手拿搬砖，就狠狠地拍在了玲子的头上；两砖下去以后，搬砖碎成了两半，鲜血从玲子的头部大股地流淌了出来，很快路面的积水就被染红了！

好在板砖的质量不过硬，玲子并没有受多大伤，周超的一时冲动换来了自己三年的监狱生涯。这件事唯一庆幸的，就是周超这一板砖下去，成功地打掉了那片区域的传销团伙，这也从片面上做了一件好事。
