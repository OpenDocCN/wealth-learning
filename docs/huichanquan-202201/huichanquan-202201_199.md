# 速抢！快来领红包！

“年终福利放送”“瓜分十万红包”

春节期间类似广告词数见不鲜

诸多活动让人眼花缭乱

虽然福利诱人但可别放松警惕

因为不法分子正混迹其中

意图**诈骗钱财**

大家一定要注意甄别

不要看到红包就先抢为敬

有些“好事”真的不能信

![](img/2df3c3c16db6127d774dec3107f0406e.png)

拆个红包，竟成了被薅的羊毛

勤勤恳恳一整年 

终于能放个长假了

老周躺在沙发上玩手机

一打开朋友圈就看到好友分享的

**“百万红包 疯狂派送”海报**

刚开始他以为是广告

没有太过在意

可连续看到好几个人都在发

老周有些心动

莫不是真能赚到钱

不然大家怎么都在分享这个海报

![](img/a27db6798b976329f440a5d77e056d68.png)

他心动不已决定参加红包活动

扫描二维码后出现返利界面

界面包含**“8 元红包”“66 元红包”**

**“148 元红包”“788 元红包”**

**“1388 元红包”**五个选项

老周尝试着点击 8 元红包

而后跳转到充值界面

系统显示**充 8 元可返还 16 元**

虽然对交钱有些疑虑

但想到金额低

就算有损失也只是毛毛雨

可活动若是真实

能领不少钱

抱着侥幸心理他充值 8 元

结果真的领到了 16 元

老周再接再厉

充值**66 元后又获得 132 元**

两次尝试均成功返现

他放下戒心不断解锁大红包

查看余额时已到账 1864 元

![](img/96fbfca9d0ecaaa3a5b225ee3a85056c.png)

于是他决定将金额**提现**

拿到钱后再拆最后一个红包

不料系统显示提现**繁忙**

要求**开启所有红包后再申请提现**

老周没有怀疑

充值 1388 元拆开了最后一个红包

却仍旧无法提现

原因是**缴纳中奖金额 25%的个人所得税**

才可以返还金额

望着屏幕上要求充值的弹窗

他心慌意乱退出链接

找到朋友圈发推广海报的好友询问情况

对方却一头雾水

称自己只是做了个发朋友圈**兼职**

不知道为什么会出现这种情况

看到回复老周陷入沉默

这是被骗了吧

报警后被告知自己遭遇了

常见的**红包返利骗局**

![](img/03dde9aca7f6fe3f37170ee2923720cb.png)

![](img/bf9d5ef89f4abba6838f2732b56a56ad.png)

引流手段多样：一个 H5，几个人力

类似骗局并非第一次出现 

早几年便有不法分子

以“新春大回馈”为借口

向用户推送**虚假的双倍红包返利活动链接**

受骗群众不知凡几

这么多人上当的骗局

实现手段却简单粗暴

一个 H5、几个人力

首先发布招聘信息

寻找**兼职人员**

要求其在朋友圈或其他个人平台发布

送新春红包海报/视频与福利截图

诱骗受害人扫码参与

受害人扫码后出现 H5 界面

小额红包正常返现

等加大投入便以**点亮所有红包才可返现**

**缴纳个人所得税才可返现**等理由

诱导继续充值诈骗钱财

![](img/4ff4c15d990cc8e0a9c9125c62969a03.png)

新春回馈只是“外衣”

类似噱头还有

**粉丝福利、开业大礼、周年庆典**等

骗局的传播渠道也不仅限于朋友圈动态

可能是**一个公众号、一个视频、一篇文章**

虽然手段多样

但万变不离其宗

注意甄别福利真假

**不点不明链接**

**不扫描陌生二维码**

**不贪小便宜更不心存侥幸**

谨记**充值翻倍是假象**

**金钱返利是骗局**

如自己或身边亲友遇到类似骗局

请保留证据第一时间报警处理
