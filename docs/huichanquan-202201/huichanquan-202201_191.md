# 又一“海王”被曝：仨妹子都恋上，傻傻献身惨被榨干！

都说现在的人精的很、女人不相信爱情看来只是没有遇见对的渣男、每次看到这种新闻，咱都肃然起敬。 

![](img/2479efdadc546b00febb40bb2869648c.png)

贵阳周女士日前遇到了一名男子，两人情投意合走在了一起，她感觉遇到了真爱，谁知道后来是无尽的伤害。

因为这名姓杨的男子，是个名副其实的“海王”，甚在贵州每个城市都有女友，而且还骗走每个女友许多钱。日前，这些受害女生集合一起，决定报案挽回损失。

![](img/ece8a93e6c521091057135827e809ca2.png)

据媒体报导，周女士控诉，“情人节那天我很忙，他就去我店里一直陪着我忙到半夜然后我们就回家了，第二天他说他有事就走了。

实际上他就去酒店去找他山东的女友，然后等山东女友回去了，他转身就去了江口，去找他江口的女友，这个事情是后面我们几个女生联系在一起之后才知道的，他情人节这天，转来转去陪各种女生。”

![](img/b9d57656352f8833e6bb5d116bd66b8f.png)

彭女士也是杨某其中一名女友，她的遭遇和周女士差不多，还有从山东赶来贵州的韦女士。彭女士表示，“他平时表现得很体贴，不是一般的体贴，给我提供了安全感，他的哥们都会叫我嫂子什么的。”

韦女士也称，“我们是从交友软件上认识的，当时觉得他这个人看着挺老实，说话也比较温柔，才相信他的，他说要和我结婚，在山东定居，让我在山东等他。”

![](img/ad66ea4698cc83793ae6b934ea0b3a9b.png)

这几名女士原本素不相识，可一通电话，让她们的生活有了交集。周女士回忆，“12 月初，他停掉了一个手机号，我问他为什么，他说债主太多了，不想有那么多人打扰他做事。后来，有债主打到我这边，然后才知道他借了很多女生的钱。”

倘若不是无意间接到债主电话，几名女子还一直被蒙在鼓里。杨某和她们每个人都有过甜蜜的点点滴滴，可如今看来，这更像是一种讽刺。

![](img/7fdf108011e735d4fee7753d90f842a9.png)

周女士表示，—我们这些女生联系在了一起之后彼此说了一下情况，发现他的说辞和方法都是一模一样，甚至有时候借钱的理由都是复制，给我发一条，给那些女孩子也发一条。” 

![](img/4d326c70b6f7b8044244dc7da7cb5d90.png)

这时，所有女生才知道，自己认为可以托付一生的人，是个名副其实的“时间管理大师”。 

韦女士控诉，债主说他在江口、福泉、铜仁、遵义、贵阳都有女友，在很多地方都有女友。

此外，杨某不仅劈腿，还骗每个女友的钱。周女士表示，自己借给男方大概 8 万元。韦女士则被借了 20 万元左右，而陈女士则被骗了 8、9 万元。可现在，她们都已经联系不上杨某。目前，她们已经报警求助，希望可以挽回损失。
