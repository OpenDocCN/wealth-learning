# 28 岁送 28 件礼？女孩省吃俭用摆摊给男友庆生：作践式“上头”

这年头在亲密关系里，大家也都更懂得爱自己了。

也就是说，菩萨应该已经不多了。

但也不是没有，我绝对没想到 2022 年还能刷到这样的事情：

**女生摆摊赚钱、吃白馒头配辣酱、白米饭配榨菜，冬天在公厕里冷水洗脸，甚至睡楼梯。**

**就为给足男友生日仪式感——28 岁送 28 件礼物。**

![](img/e796b696c584c25f9f50c64d17d2eff6.png)

为了给喜欢的男孩买礼物摆地摊赚钱，整个生活看起来就是：**“一块钱恨不得掰成两半花”。**

![](img/c2d0994772ac5463f5a80d7d74382335.png)

![](img/8d21b5055ba2436d2c5de3691cb0c361.png)

为了努力摆摊赚钱买礼物，女生吃饭也都是凑合对付、勉强吃饱。

**白馒头配辣酱，白米饭就榨菜。**

![](img/5be5cf0aa5c7dc5bf61f004337f25284.png)

![](img/eca0ee834c24376b313fa8a4640cc6e9.png)

摆摊前在路边和伙伴一起吃个汉堡会觉得特别满足，看起来吃饭全都是瞎对付。

**好好坐下来吃一餐饭的时间都没有，感觉都是路途中仓促解决。**

![](img/bafbf796e20e6874127192bc1edc4a99.png)

![](img/512d87bd5f5d15b3470aba50edf21853.png)

而她每天给她男友准备便当，不仅荤素搭配，而且碳水、蛋白质、蔬菜、水果一样不缺，营养均衡极了。

**因为男友每天吃得饱饱是她最大的心愿。**

![](img/583272d0ca565e1bfe7cabf55c72dfaa.png)

![](img/6a2674a228c8770026fb6c9e40d0b916.png)

![](img/a8029fdeb5dbf23cc1aa7e22841567e1.png)

女生为了赚钱买那该死的 28 件礼物，不仅省吃，还俭用。

而且是只对自己的那种省吃俭用，比如冬天在公厕里用冷水洗脸。

![](img/67716d394d41c0d60c085c5f9a0c8f9e.png)

![](img/fcd5a9244940e8572faca23afb2a384a.png)

**女生做了这么多，一心想和对方走到最后，再辛苦也要分担男友的压力，并坚定地认为这是作为女友的义务。**

![](img/d6fd586081b764fe30f335909a5e4ab9.png)

![](img/5a19e6c2c24cf4cccecd9dc8e9f7abf9.png)

不过 28 件礼物倒也是被她攒齐了，还有模有样：手表、球鞋、小汽车......

**仪式感爆棚！这真是活菩萨啊！**

![](img/fc6c1509f84832c5704abdb274393654.png)

![](img/0d95a21609483ab770378fc6ef5cb2e3.png)

![](img/72088e1436430857b8690f3ade189d84.png)

**这礼物的确到位，都有种他俩活不到 29 岁的感觉了。**

但都让人分不清这是谈朋友，还是养儿子啊？

对亲生父母怕是都没这么上心。

![](img/ae02075526d663650439a8d79abeae19.png)

![](img/a42f3ab6e3b1ff9b95a0a75fe52f2cb9.png)

![](img/88ef00e6f32d8bd12069d3a6909c2e79.png)

这一系列操作看完之后，大家都不看好这两个人，觉得不会有好结局，只会越来越离谱。

![](img/7af9246438534fd1b71f7e003fd38488.png)

![](img/22a79c3db9a00e4d70b0e7fb073ae879.png)

但现在这种感情里的“上头”是拉不回来的，只有自己看清了才会回头。

所以，尊重！祝福！

![](img/b83d7b769b0a593298653b714722cde0.png)

![](img/1d2316975520df2140dd773cb6630161.png)

虽然但是，这种作践自己式的“上头”，大可不必！

要先学会爱自己，再去爱别人吧。

![](img/97414449d028fc3dd60fd8f6c029e94e.png)

![](img/218c0ccbd653e5e23be8585811e03c00.png)

**真正爱你的人，不会舍得你这样作践自己的，怕不是被 PUA 了？**

![](img/6ff312cead6a3e31ba97b749cefac981.png)

之前还在网上看到过类似的事情。

巧了么不是，也是 28 岁 28 件礼物，花里胡哨一大堆。

女生给男友准备了大大的 surprise，最终却以男生出轨为结局。

![](img/4902e06de47622ca27a838d2bf0b73be.png)

![](img/06102f050effcd4670ec1b14f18e27fb.png)

不管对方接受还是不接受，也不管最终两个人是什么结局，都还是要多爱自己吧！

**这样沉重的付出，对谁都没有必要的，到最后只能感动自己。**
