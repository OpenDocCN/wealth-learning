# 杭州郑女士开小号，擦边整上，看来想明白了...

*![](img/bc54a1fe4c4ada3633c539697bcd4135.png)*

去年一则新闻让郑女士成为了很多网友心中的“女神”虽然这位女神一开始从未露脸，但很多网友坚称：郑女士说的都对！ 

后来的她，成了一个几十万粉丝的网红，每天发些什么，不知所云，莫名其妙。略显土味的同时，又有点想不开、放不开。

![](img/44d40fc5e349f574f8c62ce598fb73e0.png)

什么相处的小技巧啊，什么长不大的孩子啊，纯纯不禁想问，谁会没事关心这么些东西呢？

其次，给她安排的造型师，就更加的令人费解了。

这种家常的穿搭，看的让人着实有亲切感，但又偏偏毫无意思。

![](img/29625bc194935d1856d322f67fd2ea21.png)

衣着保守、穿搭朴素，无怪乎，网友惊呼，郑女士在凉凉的道路上越走越远了。

![](img/958501f3cde16419ac1d0aba5a2a00da.png)

正所谓，越穿越多，越穿越保守，越来越知性，流量越来越低。更核心的问题是，她的运营团队找不到正确的财富密码。

![](img/c773270d2727781c08b21991b1f92912.png)

这其中的问题，其实很简单，就是人设和粉丝画像完全不符。

郑女士的大部分粉丝，都是喜欢看火辣身材的“正经”男性，对于荷尔蒙的渴望，明显而又浅显易懂。

![](img/4f7cb940881a78a0048fbcccbf7ee17b.png)

纯纯其实特别能够理解，她迫切想要赚钱的心情，更是理解身为一个 38 岁成年女性的心态。

![](img/e8b97e2471421643fb24155d98389a22.png)

她难道不清楚，人家关注她，觉得她说的对，是因为什么吗?其实郑女士很清楚，但又偏偏不能明说，更不敢、放不下面子去走那样的路线。

![](img/d850eac74d6b6f3008702a3a4c1697b8.png)

1 月 23 日，有网友晒出一组网红杭州郑女士直播带货时候的视频截图，画面里是杭州郑女士和身边的工作人员为某款酒水带货。

![](img/658700092a85fb1c3679df50c21e2a36.png)

# 但令人尴尬的是，以前郑女士啥都不懂的时候，直播带货能有数十万的人气，大量网友追随，而如今郑女士啥都懂了，直播间人气却只有数十人，连破百都成了奢望。

# 尽管郑女士和身边的工作人员依然卖力推销酒水，但奈何大势已去，郑女士好像真的凉了。

但令人想不到的是，最近，郑女士似乎想明白了。

有网友提供爆料，说郑女士已经悄悄开设了小号，与大号的“情感导师”人设完全不同，小号里的郑女士，则完全掌握了“流量密码”。

![](img/c98a5085450c7cf300050f8e22ab9a51.png)

郑女士的小号一反大号中端庄的举止，视频内容也似乎为了迎合粉丝，穿衣风格也十分大胆，并且通过一些跳舞等行为让很多网友都津津乐道。

![](img/bda91ca58b87baa54b87bb960df135f2.png)

有人说，她找到了流量密码，该怎么对，就怎么对，漂亮是漂亮，就是年纪有点大了？

![](img/c61854e75f44348edd3fc82662126dce.png)

和年纪无关，也很身材无关，咱们纯粹就是喜欢郑女士一本正经的胡说八道，你说对吧？

![](img/b4f2dc4b0c7309418ef5f7310ea34c0d.png)

最令人想不到的是，郑女士不知道和哪个网红学的，开始在视频中“吐舌头”了。

![](img/2900492a5ee2566e27fc2f7fe52c5f00.png)

但郑女士的粉丝好像非常吃这一套，在郑女士吐舌头打擦边球的视频下面拼命夸赞，说郑女士说的都是对的，郑女士终于想通了，越来越知道大家想看什么。

![](img/9127cc1719b1e6487d702e3436bf6eb2.png)

看小号更新的频率，几乎天天都有新的视频，而郑女士的大号依旧一个礼拜没动静了，似乎郑女士有意努力在经营小号，大号只用来带货。

![](img/4b550dde31c60172e92411a1c5548306.png)

对于郑女士最近的改变，是你乐见的吗？![](img/988e912179ae2c654b623ebf1ba8f86c.png)
