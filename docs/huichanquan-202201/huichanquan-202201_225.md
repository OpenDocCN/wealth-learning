# 辛苦一年不容易，远离赌博要牢记

> 原文：[`mp.weixin.qq.com/s?__biz=MzIyMDYwMTk0Mw==&mid=2247529039&idx=3&sn=8ef32c8fc024f19d8d2cead6f02b292d&chksm=97cbb977a0bc3061794b2b5e7df04e3b36c28fd698b875a1c4a1b6c0205a601a91db610ccef2&scene=27#wechat_redirect`](http://mp.weixin.qq.com/s?__biz=MzIyMDYwMTk0Mw==&mid=2247529039&idx=3&sn=8ef32c8fc024f19d8d2cead6f02b292d&chksm=97cbb977a0bc3061794b2b5e7df04e3b36c28fd698b875a1c4a1b6c0205a601a91db610ccef2&scene=27#wechat_redirect)

辛辛苦苦又一年，为的就是回家好好过年。每年春节，和亲朋好友一起抢红包，已经成了一个新传统。 

虽然发红包抢红包是一件开心热闹的事，但微信安全中心还是友情提醒大家→[《群里这种红包别抢，对钱不好》](http://mp.weixin.qq.com/s?__biz=MzIyMDYwMTk0Mw==&mid=2247525910&idx=7&sn=a40eddceaad88832a3cdb5dc391881dd&chksm=97cbad2ea0bc243811bd3f42445ac637ade4bb4d36351f64b2e4ca6f1b7049da6d1b1b3fcb2e&scene=21#wechat_redirect)。

原本是亲朋好友间发发红包调节群里气氛，热闹一下没什么，可总有人利用红包功能设立游戏规则，并以此牟利，那就成了赌博，性质也就不同了。

在微信平台利用红包功能参与或组织网络赌博，**不仅属于明确违法违规行为，而且极易衍生欺诈骗局，严重损害微信用户权益**。

所以，对这些违法违规行为，一经投诉核实，微信安全团队将依据国家相关法律法规以及《腾讯微信软件许可及服务协议》《微信个人帐号使用规范》《微信支付服务协议》等，视严重程度对违规帐号、违规群采取阶梯式处理：

◆ 对参与或组织赌博的帐号限制支付能力； 

◆ 对参与或组织赌博的帐号进行封号处理；

◆ 对涉赌微信群限制群功能；

◆ 情节严重的将证据线索移交公安机关处理。

看到这里，小编想对那些曾经或正想参与网络赌博的人说：请放弃吧，打击网络赌博我们是认真的。

**对于网络赌博活动，**微信团队一直保持零容忍态度：**我们会持续治赌，并将在春节期间保持高压的打击态势，为微信健康、良好的网络生态环境保驾护航。**
