# 理发店一块屏幕的一

破晓 : 理发店一块屏幕的一些思考 今天去理发店注意到在镜子旁边有一块电子屏幕，想起来在 以前去的一些理发店也见着过，今天格外注意了一下。由于 老板不在，只了解到屏幕不是他们自己装的，本来还想打探 更多的消息也只能作罢。 屏幕的右下角是一个二维码，左上角是某某传媒，内容主要 放一些搞笑视频，每个视频结束都会插入一段广告，广告内 容就是扫描二维码的某商城的介绍。 我猜测的盈利模式是这样：传媒公司出设备和人去线下布设 备，需要打广告的公司付广告费，理发店收租金，打广告的 公司获得客户。

但是我有几个疑惑： 1、这些打广告的公司想要获得的客户是否主要是女性，一般 女性因为染发或者造型会在理发店逗留较长时间，有机会拿 出手机，出于好奇去扫码。 2、扫码进入的某商城首页并没有很吸引人的介绍，也没有大 公司背书，现在又是淘宝京东垄断的情况，这个商城的首页 很难留住人，为什么还花大力气去打广告，或者说为什么不 优化下主页。 3、什么样的产品适合在这样的场景下打广告，广告和产品应 该注意什么？ 4、这里边看来稳赚不赔的是理发店，但是很多高档的理发店 都没有，这是高档理发店拒绝了还是那些公司想要吸引的客 户就是中低端的消费用户呢？

花爷梦呓换酒钱 : 先给你讲讲分众传媒的模式。 分众传媒靠电梯广告起家，它们就像一个中间商，上游对接各大楼宇物

业，按年承包电梯广告的投放权，下游对接广告商，收钱做电梯广告业务。 这个模式看上去不难理解，也很简单，但有两

个细节： 第一，电梯广告，很多人闲着无聊都会看，但你会扫码去买吗？99%的人不会。 这就是效果广告和品牌展示广告

的区别，效果广告追求投一分钱赚两分钱，但品牌展示广告，追求的是曝光量，是知名度，一般背后的品牌会配合其他渠

道，会消费者进行多次触达，最终应该购买决策。 第二，分众独占电梯广告的鳌头，地推、渠道能力，是非常强的，模式

虽然简单，但人家能打通全国楼宇的商业合作关系，这个就是护城河。 所以，理发店的电子广告屏也一样，品牌方想向特 定人群打广告，理发店只是出点电费，就能拿到广告分成，最后电子屏的平台从中牵线，拿走差价做利润。 当然，你说理 发店的广告屏，效果好吗？我觉得肯定比电梯广告差一点（覆盖人次太少）。 但你也要理解，很多品牌方手里拿着广告预 算，他们其实并不清楚怎么花效率最高。看到一个跟电梯广告类似、但是价格低于电梯广告的理发店渠道，难免可能动 心，加上广告屏的销售人员吹吹耳边风，投点钱尝试一下，是很正常的。
