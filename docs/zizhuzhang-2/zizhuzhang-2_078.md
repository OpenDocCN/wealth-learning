# 都市和农村的婚姻市场走入了二个极端

都市和农村的婚姻市场，虽然都是男婚女嫁，但是走入了截然不同的二个极端，**都市结婚，房子是绝对大头**，基本必须要有；汽车其次，有最好没有也行；彩礼这东西，基本就是象征性的。其婚恋核心的核心就是房子，动辄几百万上千万的房子就是这么顶起来的，婚嫁走入了一个极端，万千青年为了婚房而痛苦。

而农村的婚姻市场却走入了另外一个极端，彩礼是绝对大头，不是基本必须要有，是绝对要有，区别只是五万八万，还是十万二十万而已；汽车其次，有最好没有也行；房子这东西，有肯定是有的，自家在农村建的房子，在我看来就是象征性的了，因为都是出去打工，根本不住。婚嫁走入了极端，那就是**彩礼是绝对重心**。

农村和都市的婚嫁重点，几乎是完全反过来的，到底谁更先进文明呢，在我看来谈不上孰优孰劣，纯粹是限于自身环境做出的最佳选择而已。

对于都市人来说，为什么那么侧重房子，因为房子承载了都市的一切，只要有房子，在都市的生活成本并不比农村高太多，吃穿花不了几个钱，都市生活成本高昂，很多人被迫离开都市，很大的原因就是负担不起房租，如果房租是免费的，随便找一份什么工作，肯定是饿不死的。更别提很多城市把房子和户口、学区进行挂钩，没有房子，小孩上学都是个大问题，户口也很难上。

所以房子是必须要有的，**只要有了房子，小两口就有了在都市立足的资本**，具备了成家的基本条件，没房子并不是说不能生活，只是生活的很困难而已，所以在都市里，房子是绝对重心。

**至于为什么汽车是可有可无的，而彩礼只是象征性的，那是因为房子实在是太贵了**。大部分普通家庭男方凑够房子的首付，都要倾尽三代之力，家庭储蓄基本耗尽。汽车属于生活改善品，有的话最好，没有的话也并非不能过，而彩礼要来有什么用呢，与其要彩礼，不如让男方买一个更大的房子。。。

所以都市的婚姻标准配置是，男方出首付买房子，女方出装修和汽车，彩礼象征性，然后房产证上写二个人的名字。这种配置是男女双方博弈的结果，并非女方不想要有房有车长的又帅，上来就把几亿资产送自己名下的男神，而是你找这种男神只有在梦中才可能；男方也并非不想要美丽温柔多金，家里几亿资金巴巴的送给男方当事业起步金，长的祸国殃民还言听计从，每天在家炖汤等自己回家的女神，而是你找这样的女神也就是在做梦。大家都现实一点，取一个平衡点，就是我上面说的标准配置了，男女配对成功率最高。

农村的婚姻市场又是另外一种极端了，农村青年分二种，第一种属于精英派，衡量标志就是能否在县城买房，不管是打工打成了包工头，还是在都市企业里上班，都算，当然还有超级精英，就是能在都市里买房的，那属于农村里的人中之龙，概率极低，已经算是都市人了，略过不谈。**我们考虑的是第二种，就是普通农村人，不管是本地生活还是外出打工，大部分农村青年属于这种**。

这类人的婚恋现状就是常年跟随父辈在外打工，每年春节回家都是一次结婚的机会，结婚速度有多快呢。年前刚认识女孩子，年后就结婚了，全部闪婚，然后女孩子随着刚认识的新丈夫外出打工。如果男女在春节的一个月里没有婚嫁成功，那只能等第二年再重新认识其他异性了。闪婚速度快，没有感情基础，彩礼费用高昂，离婚率高就是目前农村婚恋的现状。

造成这一切的首要原因，就是贫穷，农村收入太低，为了获得更好的收入，农村人被迫外出打工，他们不能像都市人一样，生于斯长于斯工作于斯，他们必须离开故乡，到很远的地方讨生活，都市的房价对他们来说就是天文数字，不管他们在都市打工二十年还是三十年，最终都是要回到老家的。不管对于男孩还是女孩来说，都是找一个老家的伴侣最放心，他们没有扎根都市的可能性，就不存在都市男女那种生于天南海北照样结婚的底气。

由于他们都是外地打工，那么对于他们来说，异性见面的机会只有春节这一次，过了春节各奔东西，想相亲难如登天。所以在这一个月里，男女都是快马加鞭的见各种相亲对象，争取在最短的时间里，找到一个合适的对象。

找到了对象之后怎么办，是不是像都市里一样，谈恋爱谈个三五年，慢悠悠拖到二十七八岁才结婚。怎么可能呢，要是谈个二年后面闹掰了，年龄都大了不好找对象了，这个损失谁负责，尤其是对于女孩子而言，因为农村女孩子一无所有，年龄就是她们的最大优势。

所以，一旦双方看对眼觉得可以谈了，马上就男方就要开始给女方买衣服、手机什么的，随后就要立刻定亲，交十万二十万的彩礼，这样女方就放心了，可以慢慢谈恋爱了，因为男方反悔成本就太高了。女方放心了男方不放心啊，彩礼送给女方了，慢慢谈恋爱，年后各奔东西去打工，女方在外面找其他人了怎么办，怎么能让男方安心呢，那就只能直接结婚，把自己嫁过去，年后跟着男方一起出去打工，双宿双飞，这样男方才愿意掏大笔的彩礼出来。

我们可以看到，彩礼在农村男女的取信过程中是非常重要的，农村的婚姻都是闪婚，一个月内从素不相识的陌生人到同床共枕的伴侣，中间的核心纽带就是彩礼，彩礼交到了女方手上，女方才愿意相信男方是打算认真谈的，女方把自己嫁过去了，男方才愿意相信女方是愿意认真谈的。所以博弈的结果就是略过恋爱步骤，直接结婚，大笔的彩礼一手交钱一手交人，活脱脱的像卖女儿，但是实际上这就是农村无奈的现实。至于为什么是男方出彩礼而不是女方出，那自然是因为农村男多女少，城市男少女多的现实，以及农村妇女地位低下的原因。这也是博弈的一部分，和中国传统文化没什么关系，中国传统文化的确有男方送彩礼的规矩，但是也有女方送嫁妆的规矩啊，是彩礼多还是嫁妆多那就完全是婚姻市场博弈出来的了，南方有很多富裕都市的婚姻，女方的嫁妆都是男方彩礼的双倍，而很多贫困农村，彩礼要耗掉男方几十年的积蓄，女方的嫁妆就一点家电被子而已。

所以，在打工时代，彩礼在农村婚姻中提供的纽带作用是怎么强调都不为过的，当然，如果男方能昭显自己强大的经济实力，比如在县城甚至都市买一套房，或者有极高的学历及高收入工作，彩礼大减甚至不要都可以，贫困农村的女方找到了乘龙快婿的确是可以象征性给彩礼的，甚至不要彩礼，直接领证都有，这种现象和高额彩礼是同时存在的，不过属于极度稀有事件。

因为大部分的男方，只是普通的打工者，他们昭显实力的方式就是拿出十几万的彩礼，修一个农村小楼房，买一辆国产小汽车，这就属于农村婚恋市场的顶配优质男，别看农村男多女少，达到这个标准的男方也是不愁娶的。

但是即便是这样的顶配优质男，也不可能把彩礼标准降低，因为现在钱多只能说明你家现在的储备多，几年之内能日子不会太差而已。大家都是打工的，前途一目了然，只要肯吃苦，收入不会差太远，没有高额的彩礼，是无法让女方安心的。相反，一个重点大学毕业生+都市政府/国企的正式编制，只要这个男方愿意娶，

随便给一点彩礼都能干脆利落的击败十几万彩礼的对手，如果还能有套房，倒贴都可以。。。理由也很简单，这种人的前途是不好说的，而且就算混的再平庸，也不会比打工的差，稍微提个几级，就甩的不止一条街了。

都市和农村的婚姻市场走入了二个极端，都市重视房子轻视彩礼，农村重视彩礼轻视房子，**都市先恋爱后结婚，农村先结婚后恋爱**。这种截然相反的状态是由中国现状决定的，是人民所自发决定的，都市丈母娘要房子谈不上多高尚，农村丈母娘要巨额彩礼也谈不上多低俗。

人类的适应性是很强的，什么样的社会状态就能孕育出什么样的婚姻规则，大家觉得现在都市和农村的婚姻规则奇怪吗？其实还算好，我可以告诉大家，**如果到了战乱年代，那个时候被扭曲的婚姻市场，才真的是光怪陆离。**

所以，我听说农村的男青年要准备巨额彩礼，而且是一个月内就闪婚，性质接近于买老婆，我并没有歧视他们愚昧落后，而是对农村男女抱有深深的同情，他们都是很无奈的，如果他们能在老家里安居乐业，这些奇闻都不会出现。![](img/fdb589e67d0f60e6242165cfff2477bd.jpg)
