# 关于「产品党每周话题」

产品党是小密圈里的一个免费公开圈子，希望能和产品经理 们有更多思维碰撞。「产品党每周话题」是其中的一个项 目，希望以每周一期的频度，发出问题，大家一起讨论。也 欢迎大家提出自己的问题（如果能提供奖品最好，奖品不在 贵，一份心意即可）。

我们将会对每次话题进行总结并发布。

评论区：

4、流量大于价值观的企业都应该关门。
