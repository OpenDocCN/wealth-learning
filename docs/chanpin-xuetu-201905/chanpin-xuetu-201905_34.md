# 产品党每周话题 01：你怎么看「贵圈」？ 产品党每周话题，每周四一期，欢迎关注。

在阿混那里看到「贵圈」这个产品，觉得蛮有意思的，借着 这个，启动「产品党」这个圈子的每周话题——每周我（或 者大家有希望拿出来讨论的内容更好）找一个产品/产品的功 能点/交互等来聊聊天。

「贵圈」这个产品，在朋友圈里一露面，很快有人关注和转 发，说明有点刺激到用户的需求，但马上又被举报进而被封 禁，有几个问题：

*   技术上怎么实现「贵圈」（要尽量实时，数据量大了怎么 办？微信封禁怎么办？）；

*   商业模式应该怎么建立？

*   如果你是他的产品经理，你现在该做什么？

还有什么更多的延伸问题吗？也欢迎一起探讨。 欢迎大家：

*   动脑筋思考；

*   充分提出自己的观点；

*   和圈内的朋友们展开讨论。

不欢迎：

*   骂人；

*   觉得「盗用」了你的智慧（真的没有这个必要）。

评论区：

2 : 我之前想做这个产品的，它其实满足了人们的好奇，甚至是偷窥欲望。

3 : 还没关注到，先玩了在来聊

4 : “在别人的墙基上砌房子，再大再漂亮也可能经不起风雨，甚至会不堪一击。”

5 : 不懂

6 : 你开了一个好大的话题...... 我当时看到及目前还剩的想法有： 1、无限好奇“贵圈”的前端交互，做的太自然了，特别想 知道这个是怎么实现的或者类似的思路 2、这么存留的资金池，一旦被封禁冻结，会退回么？（或者官方强制逐笔退 回？） 3、因为当时等有时间去准备支付试试，已经被封了。不知道订阅朋友圈的体验是不是真的和“朋友圈一致”（就是界 面上的链接都可以正常点击，就像真的访问朋友圈一样） 4、去年类似体验的某个产品还在么，昨晚回来累挂了，早上接 着找了下，还在，之后我会贴出来。是一个做朋友圈微信书的产品，挺赞的。 至于你问的“技术上怎么实现（我是技术 白）”，商业模式怎么建立（...这种铁定招封禁的我一般不深想，深想的花时间有点多了）， 唯一剩一个可答一点的：我是 他的产品（善后，复盘反省） 这个产品本来是可以成为一个现象级产品的（即使被封禁），成为现象级产品的好处就是曝 光之后其他的可能性，只是我搜了搜，没有任何水声，莫非是全栈工程师独立做的？o.o
