# 那些做熟人生意的人，最后都怎么样了？

如果做生意都只是卖给熟人，那么任何生意可能最后都会失败。

一般而言，特别是对于刚刚创业的人来说，一开始卖产品，总会想着先从熟人“下手”，也就是我们通常所说的开辟“熟人市场”。这种观念无可厚非，但如果是想着把生意做大做强，那么“熟人市场”并不是我们的支撑点。

在社交电商里面，我们都比较看重中关系，强关系反而不是我们成交的重点，也就是说依靠熟人市场其实是很难做起来的。这个逻辑也很简单，因为无论从哪个方面来说，更多的都是我们不认识的人。

的确，很多刚开始创业的人都希望通过熟人打开市场，先做熟人的圈子，然后再向外扩张，但很多时候，往往事与愿违。并不是说关系没到位，他们不愿意帮忙，相反就是因为关系太好，所以导致根本没办法开口。

而且因为关系好，做熟人市场，就太利益化。虽然我们相信是因为产品好，才去给他们推荐，但对方并不会这么想，在他们看来，我们就是想赚他们的钱。这也是导致在熟人圈子做不大的另外一大原因。

以利相交，利尽则散;以势相交，势败则倾;以权相交，权失则弃;以情相交，情断则伤;唯以心相交，方能成其久远。

谁都不希望与他人相交，是因为自己有利可图，被关系变现。

当然更重要的是，不管是亲朋好友，合作是可以，但要在对方身上赚钱，就显得过于现实和不被接受。所以在中国式的社交关系里面，一直都很排斥这种做法。

基于此，要想把生意做大做强，第一个应该摒弃的观念就应该是放弃熟人市场，开发陌生市场。

而相对于我们说的陌生市场，如果是放到移动互联网上来说，其实就是一个等待引流和转化的市场。显然，这个比熟人市场要庞大得多。

传统 pc 电商注重流量转化，而社交电商注重情感营销，信任成交。两者有着本质的区别。

但有一点是相通的，引流。也就是说，不管哪种形式的商业形态，尤其是和互联网相关的，都必须要做的一点，引流。简言之，你得有足够的吸引力和关注度。(关于引流相关文章，可以查阅我之前的文章。这里不赘述。)

所以对于很多创业新人，其实大可不必把所有希望都压在身边的所谓熟人上，去做熟人圈子，相反我建议当他们不存在，全心全意去开发陌生市场。

值得一提的是，不要因为熟人不帮忙而变使得关系变生疏。帮是情分，不帮是本分。不要道德绑架熟人关系，没有谁是必须要帮谁的，也没有谁帮谁就是天经地义的。所以，即便是熟人不帮忙，也不要道德绑架于他人，致使原本要好的关系遭到破坏。

不过有一点我需要单独强调一下，我文章所说的不要指望做熟人市场能把生意做多大，可从来没否定过熟人在我们创业过程当中的重要性。这是两码事情，希望大家不要理解错。

尽管我们想依靠陌生人市场把生意做大做强，但也不能完全不顾熟人的存在，导致最后自己可能真的赚了钱，但却失去了身边最要好的亲人和朋友。显然，这是最不可取的。

因此，这也是一个相对而言的问题，切不可顾此失彼，最后赚了钱，失去了一切。

但记住，要想做大做强，就别指望做熟人市场。
