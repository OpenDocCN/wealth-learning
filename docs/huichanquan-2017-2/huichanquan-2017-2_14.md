# 3 点做网赚的精髓，明白了你也可以赚 20 万以上

草根互联网创业赚钱这个行业，小项目特别多，但是由于这些项目的高复制性、高盈利性、短周期性、私密性的特点。导致一些项目发出来之后，可能已经有不少人进入这个小领域了，然后，后面的小白再加入的时候，就已经来不及，因此他们会喊烂大街之类的。

那这些发出来的文章项目还 TM 的有意义吗？

有，文章项目的意义不在于傻瓜式复制操作，意义在于积累点滴,、优化、管理自身的已有项目。

很多人一味追捧最新、最热项目，可以，我简单问你几个问题：

（1）测试出最新、最热项目，成本高吗？

其实厉害的团队一般有流量部门、项目部门等等，他们有一群头脑很灵光的家伙测试项目，一个是人力成本，一个是流量成本，流量成本比较高，要测试一个项目的稳定性，可能要花几万，甚至更多。那人家会马上给你吗？不可能！他会自己玩一段时间，差不多了，可能会分享给他信任的朋友，比如社群里边混得熟、混得好的朋友。所以，处在这个行业食物链中低端的玩家，基本是比较被动的，企图看看【财神大咖】一篇文章就想咸鱼翻身，不太现实。

（2）给你最新、最热项目，你玩得动吗？

我觉得这个行业的朋友要认清楚一个形势：互联网人口红利期已经退去了。现在不像以前那样，随便搞搞就能赚钱。很多人就是被之前的行情给惯坏了，习惯了简单、直接、暴利。然后到了今年，就算有什么最新最热项目，都有门槛，而且同行越来越多。 

（3）你所认为过时的项目，真的赚不了钱了吗？ 

不懂什么时候开始，人们喜欢把过时项目跟赚不了钱划上等号。以前我也是觉得要不断的换项目、换玩法才能在这个行业生存。当我走到线下交流会，我发现：其实优化、管理项目也是很重要的。

优化项目，一直都是很多人讨论的话题，但是也没有很多人去总结分享一个比较系统的方法。有个做微信生态业务的朋友，也是团队在操作，有一回他和我聊起他的团队。他有一个宣传推广部门，就是类似招代理的部门。人员只有 3 个人，一个主管，两个不同渠道的维护助手。她们的业绩加起来一个月有 A 万（这里就不公开），而且做得很吃力。

他问我能不能让他们赚更多的钱。我回答：没问题，而且可以翻倍。他很吃惊。我又问他，如果想让效益短时间翻倍，你有什么办法？

（1）他说：挑出最赚钱的业务，然后集中人力去优化。

感觉这道理挺对的，符合集中精力办事情原则。我问了一个问题：3 个人都去做一个容易来钱的渠道，按照现在的业务水平，同时也要考虑放弃其他来钱的渠道。能做到收入翻倍吗？他觉得不能，也不现实，就算再怎么优化也做不到。

（2）他说：在原来的业务上增加人手

感觉也不错，广撒网，说不定人均产出全部加起来还凑合。但是要考虑招人的难度？培养的时间跨度，能正式上手的时间？他的城市到 9 月份这时候基本上找不到什么人，已经过了高校生就职的高峰季节。而且业务又涉及到买粉、卖粉的细节。想要在短时间翻倍收益，也不现实。

（3）他说：对接资源更大的客户，成交大单

这感觉也不错，一般大客户成交量更大，业绩就会增加很多。我问他一些情况，就是目前这 3 个工作人员维护的是什么样的客户。他说是老客户。我又问，开发一个大客户难度有多大。他说如果开发新的大客户，至少需要一个月以上的时间，因为需要测试各种渠道流量的效果，需要大家熟悉，彼此信任。而且影响原来的老客户维护，影响原来的业绩。可见，短时间翻倍收益也不太可能。

（4）他说：外包业务，合作赚钱

这个也觉得很不错。大家合作赚钱嘛。然后，我问：老板会同意吗？他说不会。是哈，任何老板都不愿意把核心客户交给别人，何况在我们这个行业有比较高的复制性，比较高的盈利性。设想是美好的，现实是骨感的。这个办法也不能短期让收益翻倍。

最后他说让我提一点建议，我告诉他：要用优化项目、优化业务的办法来解决问题。

A、首先分析他们的业务类型。他们做的是对外推广与合作业务，属于外向型的商务合作。所以要把业务搞得红火，就一定要对外，对外，对外。线上对外，线下对外，线上的效果不如线下的面对面沟通、对接。

B、其次分析当前的人力情况。很明显，因为业务专业性比较强，就算你招新人进来，也不能直接放到现有业务的岗位上，就算放上去也不会有产出那么快。但是难题就是依靠现有的人力，你又不能把业绩翻得那么快，还是要招人，而且招到的人能短时间能用上。

C、接着看一下当前的业务潜力。当前的业务潜力可能很大，但是你们 HOLD 不住，那一定不能死死抓住。换个简单的方式做业务，一定要最简单，简单到什么程度就好。简单到，扫地阿姨进来都能马上上手，而且你可以快速的招到这类型的扫地阿姨。

D、综合前面三点来看，他们需要：往线下去拓展业务。微商不是经常搞什么会展、会销，甚至做各种品牌赞助之类的。而他们的业务也是对外的，需要找更多外面的人来参加。那他们只需要增加更多的会议协助人手，把规模做大不就行了吗。

这就是微商经常干的事，搞个微商大会，收几个大的品牌赞助商，同时提高参会门槛。这个难度不大，招一些维护会议的人难度比招业务专员难度要小。而且上手快，你把会议流程全部给他们培训一遍，基本都能懂。关键是能带动自己原有的业务。

这个业务的优化流程还真不是没有依据，前段时间微商界，不是有人组织一些大会赚了几百万，这个懂的人都懂，我就不具体说。那短时间收益翻倍你说难不难？当然，前提是你有号召力，这个东西一般对外宣传推广业务部门都有自己已经稳定熟悉的客户，如果一个部门几百个客户都没有，那还留有何用？

当然，不同的业务，优化的角度又不一样，以上只是一个例子。我和不少的网赚高手交流过。发现他们玩项目就是靠三个点：优化项目、管理项目、资源维护。
