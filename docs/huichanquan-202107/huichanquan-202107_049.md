# 杀猪盘竟盯上高瓴！“杀手”做局：财经博主损失惨重 怒而曝光

近日，一张印有“高瓴资本邀请函”的图片在网上流传，招募高瓴中国代理人，仅有一个名额，获得代理人资格后可入股高瓴资本，获得 1%股份。

![](img/ecbc23dc9409c24ff2d5abe7f57e5c92.png)

图片来源：网络

真有这等好事？中证君向高瓴方面求证，高瓴人士回应称：假消息！

中证君调查发现，这背后或是一场杀猪盘骗局。

高瓴招募中国区代理人？假的！![](img/22de6dfdeea9715b4173319ba3c1be91.png)

这张印有所谓“高瓴资本邀请函”的文件中写道，2021 年“高瓴资本”招募代理人在中国赛区仅有一个名额，比赛第一名即可成为中国区代理人。

而获得所谓的代理人资格后，将可以拥有三大非常具有“诱惑力”的权限：一，可决定高瓴中国区域投资方向；二，获得高瓴全球私募和投资机构内幕消息；三，可获得高瓴资本 1%股份。

奇葩的是，这唯一名额的考核指标只有两项——“人气”和“团结”：一是参赛者吸引到的粉丝数量，二是和粉丝共同参与测试“高瓴资本花费 5000 万的资金调研 78 种方法”，以此考察参赛者的人气及协同作战能力。

获得这份邀请函的，是一位名叫“张景”的先生。

对此，中证君第一时间联系了高瓴方面求证，高瓴人士回应称：假消息！

背后或是一场杀猪盘骗局![](img/22de6dfdeea9715b4173319ba3c1be91.png)

中证君调查发现，这张照片源自某财经博主的曝光。他表示，这是一场杀猪盘骗局的最新骗术，邀请函是骗子用来标榜自己身价的工具，邀请函中高瓴资本的章是假章。

![](img/f9275ca1ebe4a52530b377a88a3bbfec.png)

图片来源：微博

中证君联系到该博主，其介绍称，这张“高瓴资本邀请函”来自一个 QQ 炒股群，群管理员就是邀请函中的被邀请人“张景”。张景称，高瓴从 2018 年开始持续调研和关注体育行业，体育产业将成为国民经济支柱性产业。

![](img/2e535546a41d9ae25df1866404c2d45b.png)

图片来源：被采访人提供

中证君通过群聊的 QQ 账号尝试加入群聊，却发现该群目前“不允许任何人加入”。

该博主向中证君表示，这个群和杀猪盘群的套路是类似的。他曝光了另一个群里的部分聊天记录，群管理员张婷之前称：“明天的布局，大家都期待已久了，一会大家认真听，我把这支票的详细操作，布局时间和大家讲一讲，让每位支持张婷的朋友，都能拿到 50%的利润。”

第二天一早开盘，张婷便在群中发布股票信息，召集群成员“立即挂涨停板价格 7.14 元买入，八成仓买入，速度执行”，而且买入时间“只有 2 分钟”，并称：“一定可以买进成交，已经预留仓位给大家。”

而该博主就在这场杀猪盘骗局中上当，损失惨重，因此特意曝光出来，希望大家不要上当。

谨防杀猪盘骗局![](img/22de6dfdeea9715b4173319ba3c1be91.png)

值得注意的是，中证君曾多次报道此类事件——冒充基金经理或私募大佬建群荐股，其实是杀猪盘骗局。此前，私募大佬东方港湾但斌遭不法分子冒充，注册微信号、开公众号、建群荐股等；星石投资、保银投资等知名私募也均有遭遇“李鬼”的经历。

从诈骗的手段来看，除了通过微信号、QQ 号荐股收取指导费之外，有的对群成员收取“群费”，甚至还有开“投资团”的，号称万人坐庄抄底，短短几天就能有 20%-30%甚至 50%的收益，以高收益利诱投资者。不少群里大部分群员都是“托儿”。

此外，骗子的工具也在升级，除了传统的微信、QQ 之外，骗子更是有自己的电脑客户端、手机 APP，连硬件智能手环都有配套。

骗局惊心，投资者千万要小心！
