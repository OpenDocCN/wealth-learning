# 这些网红直播也太恶心了吧？？？

现在，可以算是网红时代了，

直播的热潮，到底有多火，我就不多说了。

不知道大家是否跟我一样，

直播火了这么久，却没有真的看过几个，

这段时间密集刷了一下才发现，

**原来，市面上有这么多奇奇怪怪的直播啊？？？**

1

话不多说，我们直接来看一个动图吧：

![](img/b85e0bd09e984b1943168c63c0a55bbf.png)

这就是某著名女主播的直播画面，

明眼如你，应该也看出来了，

清凉的吊带，短到快要到大腿的裙子，

紧身到几乎要坦胸露乳的衣服质感......

这...这不是妥妥地打擦边球么？

**结果，摇着摇着动作幅度太大，发生了什么呢？**

![](img/c3468de698b65b24481ae516002ae185.png)

**一对奇奇怪怪的东西，从她的身体中跳了出来......**

而我们的女主播呢？

竟然淡定地跳完了一整段舞蹈，坐下来后，解释说，

不好意思刚才是为了防止走光的胸贴，

**我真的呵呵呵呵了。**

可就是这样一个严重擦边的直播，竟然再过了十几分钟后，

才最终被举报封禁。

所谓软色情，我相信大部分人都能看得出来，

可是这样的直播，却真的是屡禁不止。 

CF 有一个火线演播室的节目，在直播中，镜头都是些什么特写呢？

![](img/b342aa117e2405356d3e4b18b924312d.png)

从脚部特写：

![](img/bd4e8731589e3e00cdf7c43270988695.png)

到黑丝特写：

![](img/b444cdd5a6606a6117a9172a03e877d4.png)

这镜头运作，这重点特写，

这到底是是游戏直播还是美女直播，

我还真分不清楚。

2

郭老师，大家还有印象吧？在爆火之后的很长一段时间，

我几乎很少再听到过什么关于郭老师的新闻，

这样的例子有很多，在凭借着某些爆梗火起来之后，

这很正常，很多类似的、缺乏持续吸引力的网红，会后劲乏力。

只是，让我没想到的是，让我再一次关注到郭老师的，不是别的。

**而是因为，在直播中疯狂打擦边球......**

![](img/efbd5be114c689e36820f1ef06004df0.png)

你能想象么？郭老师，打擦边球？这...不是要人身攻击，只是，真的有人看么？

可是，这又怎么样，人家郭老师不仅打了，

还打得别出心裁，让人难以直视......

在某一次的直播中，她不知道是抽了什么风，

直接拿出自己穿过，还没有清洗的内衣，对着镜头：

**（下图还是我截过的，辣眼睛的部分已经被我截掉了）**

![](img/4d578b010403c07afbbf8f032f05ee28.png)

不仅如此，郭老师还有意无意地，在视频中展示自己“私密”的时刻，

比如，穿裤子，吊在屁股上，里面的内裤已经清晰可见，

如果裤子还往下掉，下半身就随时会展示在直播画面里，

![](img/ada44e0e81342525f0809f1317742fca.png)

甚至还有这样穿着内裤直接出现在镜头的画面： 

![](img/57414840afdf02f7f91540217ab116b6.png)

**啊，我真的是想自戳双目。**

3

**穿的少、各种摇，**

我不知道这样的画面，会出现在未成年人的眼前， 

只是这样辣眼睛的画面，真的比我们想象的猛烈的多， 

有这样梳着双马尾背心短裤蹦蹦跶跶的女生，

![](img/6923e02f38f748965f5303ed8a09643b.png)

有这样录着录着，穿着吊带就开始各种摆动的女生：

![](img/47d8d972d1c90459c189d76efbb5c6f4.png)

还有这种，无数扭动着身姿对着镜头谄媚讨好的女生......

![](img/ef258c741d20cad3f65c37444633a501.png)

说真的，上面那些女孩，他们的身体自然是美的，

只是女孩的身体，以这样的方式，

展现在镜头面前，

企图去靠这个博得话题，

**赢取流量，那一切，都变得丑陋了起来。**

当然除了擦边球、软色情，在直播界让人无语的事还有很多很多，

有人直播自己被打， 

![](img/c388d1649cadef0c2627f500bf8908e6.png)

有人直播卖假货，结果被观众当场戳穿：

![](img/aa4e9392c136d2fed1b457d0e4da42e5.png)

还有人直播撕逼，对着对面的人破口大骂，

甚至大打出手......

**我不知道，这样粗俗、低劣的直播，**

**这样博下线的磨脸皮哗众取宠，**

**究竟还能火多久。**

**写在最后**

但是，大家知道最讽刺的是什么么？

最讽刺的是，

那个在跳舞的时候把胸部甩出去的女孩，

那个对着镜头自扇巴掌自我伤害的女生， 

那个卖假货被当中戳穿的女生，

还有那个把内裤露出来给观众看得女生， 

**他们竟然，都能有一批死忠粉......**

有人为他们应援， 

有人为他们花钱，

有人为他们点赞排队求合照......

一句话，他们虽然低俗，虽然丑陋，虽然让人感到害臊，

但是，靠着低级， 

他们却赚了一把又一把的钱，

拿着普通人根本拿不到的收入， 

呵呵，呵呵呵呵。

看来，娱乐至死，是真的。
