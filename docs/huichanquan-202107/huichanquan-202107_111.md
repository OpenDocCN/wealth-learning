# 为给女主播打赏，私家厨师盗窃雇主家的名酒，获利 174 万

2019 年，小万被北京的一位富商俞某请回家里做家庭厨师。他每天的工作就是外出采购食材及做饭，俞某对小万很信赖，让其在家中居住。可是，让俞某没想到的是，自己竟然会引狼入室。

![](img/e8d73297d649553fb1ea81e66cbf3dad.png)

原来，小万米上了迷上了一个网络女主播，通过给对方打赏。他觉得能够博取对方的好感。可是，他每个月几千块的工资，根本无法支付打赏费用。他发现俞某家里的酒窖藏有大量茅台酒，于是偷卖出去换钱继续来给女主播打赏 

![](img/61bec1d9dac9dd1b852cb9c71ad252a8.png)

![](img/4fc346e7ea5f08de8e3523208893e8d2.png)

![](img/86b49eb810e62670a37bdfa8e5ac7e53.png)

![](img/a212c1fb1926da2ae6e692971372c04d.png)

仅半年时间，小万盗窃名酒获利 174 万元。不久前，顺义区人民检察院以小万涉嫌盗窃罪向顺义区人民法院提起公诉。

![](img/6aafabe6cffa7bdb6cbc4705bfc5d454.png)

最终，**小万因犯盗窃罪被判处有期徒刑十年六个月，剥夺政治权利两年，并处罚金人民币十万元。**
