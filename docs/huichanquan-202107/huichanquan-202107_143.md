# 这些年最大的骗子！出轨还家暴！林生斌被前岳父母公开撕了

长达四年时间里，林生斌利用惨死的妻子和三个可爱的孩子，高呼着只要公道不要钱，利用舆论的巨大支持，却悄悄的与多家单位私下和解获得巨款，绝口不再提半句公道。
朱小贞生前清楚的控诉了林生斌家暴出轨，当朱小贞死去再不能开口时，林生斌却利用围绕于身边的利益写手和贴身跟随的摄影师，将自己打造成中国第一深情丈夫，史上头号体面人，亿万国人的林爸爸。 ![](img/b875a7665e9e414e76a76315569a76f8.png)他赚得盆满钵满，他娶了纵火案前就认识的小十五岁的娇妻，直到孩子出生数月瞒不住了，才选择于 6.30 这个百年大庆之日报喜。
算计了妻子，算计了孩子，算计了岳父岳母小舅子，算计了全国人，最后连党和国家都给算计了。凤凰周刊记者去采访朱小贞年事已高的父母了。信息量好大。给大家提取一下重点：1.朱小贞在生前，曾告诉爸爸：老公在外面养小三了。
![](img/45a0447c57166a4643f9ea4bbc09dc19.png)2.虽然从农村出生，但朱小贞和哥哥朱庆丰一起创业，生活过的不错。
3.朱小贞去理发店做头发，认识了洗头工林生斌，两人恋爱了。朱小贞的爸爸从始至终都反对林生斌和自己女儿在一起。
4.为了说服父母接受林生斌，朱小贞自己拿钱当做彩礼给父母。
![](img/4f9c984f175c6a4f9ecf9ee7c6ce7b1e.png)5.朱小贞生前就告诉爸妈，丈夫林生斌很迷信。
![](img/a22ccb5088c53be748e3ea81196c1985.png)6.结婚后，小贞带着林生斌做生意，随后林生斌又把自己妹妹带进来做生意，朱小贞退出来，在家里带孩子。
7.朱小贞被林生斌家暴过，母亲曾看到女儿头上鼓个大包，她说是被丈夫打的。
8.朱小贞的爸爸提醒过朱小贞，说让她注意这个保姆，感觉不像好人。
9.纵火案后，朱小贞爸妈希望拿到女儿生前的手机，林生斌不愿意给，说烧毁了。父母起疑，因为女儿浑身上下都没有被烧过的痕迹。
![](img/bf12eb72cdf812a00c812059464998fa.png)10.朱小贞去世后，两位老人想去查一下女儿银行卡余额，她一共只有八万。
11.纵火案后，朱小贞父母去案发地小区，林生斌家人质问他们是不是来分家产的，林生斌妹妹告诉大家：我哥的东西，干嘛给他！
12.林生斌结婚后，曾把家里上百万的钱分给了老家农村的兄弟姐妹，也不要回来，让朱小贞很生气。
13.林生斌靠着深情人设和思念妻儿的故事，赚了很多服装钱，朱小贞的父母算一下女儿的财产，林生斌总说没钱。
14.林生斌发微博时，展示自己给小贞妈妈转 5 万。其实那时候林生斌偷偷卖掉了两个女儿的服装店，每家店 120 万，这 5 万是林生斌卖店给他们的补偿。
15.上个月，小贞的爸妈希望林生斌把生前分给老家亲戚的上百万元还回来，林生斌翻脸，要求走法律途径。由于林生斌的不配合，老两口至今也拿不到小贞卡里的 8 万元。![](img/5cd7280f23706e6e118bc9f532ec3c9b.png)
