# 来，继续说人性，游

Caoz : 来，继续说人性，游戏或虚拟世界中的一个很重要的 用户吸引的思路是，人在现实中想做而做不到的事情。

如果游戏完全写实，你现实做不到的事情游戏也做不到，谁 还要玩，活着还不够辛苦么。

成人游戏，儿童游戏，都可以这么思考。 有哪些事情我们现实中想做 却做不到。
