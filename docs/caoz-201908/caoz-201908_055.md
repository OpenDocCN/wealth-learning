新币和美元可以自由兑换，无额度管制。但来回兑换损失还 是有不小的。

提示 3，虽然你可以拥有美元户头，但在新加坡的银行里，是 无法取现美元的，只能取现新币。

以上，有需要的，可以给我私信。
