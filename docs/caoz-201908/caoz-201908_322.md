# 朱姐怎么做到在不同

(提问)条形马 : 朱姐怎么做到在不同圈子有不同画风的？非常 酷炫。

2017-03-18

回答：因为在我的圈子里我是众多读者的作者；在大辉哥、 池哥的圈子里我是他们妹子，撒娇耍赖也没事；和阿禅是特 别聊得来的朋友，所以喜欢玩笑；和曹政哥……基本大部分 时候他都以善意的“故意不给我面子”的姿态跟我说话，不过 我知道他是善意，他爱什么态度什么态度，我只说我的。没 有刻意画风吧，还是看我和圈主的关系是怎样的。(30 赞)
