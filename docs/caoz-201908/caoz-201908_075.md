# 曹大大曾教导我们，

加油丫头 : 曹大大曾教导我们， 春节回老家，要留心大朋友、小朋友、老朋友，手机里在关 注些什么？置顶些什么？

昨日元宵节，我便留意关心了一下一已上大学的侄女在手机 里玩些什么，然后发现她一直在刷 QQ 动态，我说你不玩微信 吗？她说不玩啊，我们同学都是转账缴费才用微信，平时都 玩 QQ，

这让我瞬间有点脱钩了，QQ 有啥可玩的？我都好几年没有看 过 QQ 的动态了…然后侄女就跟我介绍他们大学里 QQ 很好玩 啊，群里人多 2000 人，可以闲置交易啊等等，学校里还有社 团组织的一个“**大学表白墙”QQ 账号，全校同学几乎都在关 注，大概就是比如你喜欢谁谁谁，然后你就可以聊天告诉“表 白墙”，然后表白墙会把你这一侧的头像昵称，打上马赛克遮 盖后发出，全校同学都可以看见，还可以发相亲信息等。

然后我大概看了一下侄女的 QQ 动态，确实是一个很生动的小 生态…我忙拿出自己的手机，跟她说我都不知道在哪找这个 动态入口，然后她帮我找了出来，所有人都笑了，我的动态 里十几条消息全是一个卖手机靓号的人发的…

我大概拍了两三张图，都是孩子们。所以马赛克遮盖了一 些…（虽然我也不大，哈哈哈）

感觉自己的朋友圈，都好久没有看到这么多真诚的晒生活 了，嗯嗯…我不知道是我落伍了还是这只是个别现象，

有需要大学生流量的朋友是不是可以关注一下？？
