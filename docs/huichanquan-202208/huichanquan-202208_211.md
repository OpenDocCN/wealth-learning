# 柬埔寨说 99%电信诈骗不是本地人，在金三角看到的电信诈骗

不是本地人，和本地人没关系，这话表面是这个意思。 

看见这条新闻突然想起十几年前某省人的一句口头禅“做传销的不可能是 GX 人”。 

肚子里嘀咕嘀咕，想说点什么，想起一段往事。

**旧事** 

前些年有朋友让我去 MD，这故事现在讲起来，开头看起来像极了是电信诈骗，因为我在电话里反复追问到底什么事儿，对方就是不说，一定要人到了以后再说。 

被我拖了两个月后，对方终于扛不住，买好机票告诉我时间，硬逼着我去，加上中间人起哄，于是就硬着头皮去了。 

飞昆明转普洱，一个越野接到我，司机除了接人时确认电话号码，全程几小时一言不发。

此时国内电信诈骗风潮刚刚起步，菲律宾电信诈骗被老杜排挤，开始大面积转移到东南亚，我来之前一直犯嘀咕，这要是找我过去搞电诈，那不是天赐良机给我一个卧底的机会，心里是乐开了花。 

到边境，换摩托车，走山路，再换摩托车，来回转车好几次，黑灯瞎火的，打开手机连地图都没办法定位，无法标记路线，这要是被卖了就彻底歇菜了。

![](img/6874666319a97e55d0ed8e6b08cd12f2.png)

摩托车又换轿车，走大路，遇 MD 检查站登记，弯弯绕绕到了一个小镇。 

四 T 某酒店，名字已经忘记了，只记得介绍时，遥看河对面一个中式房顶的平层别墅，说彭 J 生就住在哪儿......

当家的来见面寒暄半天，还是没说什么事儿，就是招呼吃吃喝喝，遮遮掩掩的让我觉得这事儿不见光，心里是更确定是电信诈骗了。 

休息后次日早，到酒店一楼等车，说是进山里去。

一排排的丰田面包车从路上跑过，大清早就这么浩浩荡荡的去了。

![](img/356b58ce66fb150bdb10a39aa5721249.png)

一下车，傻了。 

![](img/49e97ca565a3344880c0c79f042aae47.png)

麻蛋，赌场啊，这特么和电信诈骗有个毛的关系。

肚子里骂了千百遍，硬着头皮继续吧。 

参观完赌场，给一堆筹码但啥都不会，斗牛看了半小时还没看明白，大厅左侧内里有一个大号墙柜，全是现金，乐颠颠去拍照，被保安现场要求删除，财务过来给我解释，我顺手把筹码给他做个人情。

![](img/ca1d36cfd84e1ed209cb9e2543b7003a.png)

第一天就这么打法时间了，晚上回去吃饭时，当家的把话挑明了。 

项目名称：网络赌博，而且还是最时兴的 VX 群，介绍隔壁团队认识，七个湖南人男男女女挤在一个二十多平的房间里工作，一问果然是湖南南部某县。

一个月流水几百万了，抽水多少，发展前景等等一通介绍后，了解电信诈骗的美梦就这彻底破灭了。 

**暗自盘算** 

我心里琢磨着，按鹅的风控速度，两个月做火了，鹅一个月就能扑灭了，后来事态发展证明了我对鹅的信心，这茬风潮来的猛去的更快。

但在当时现场，这话是消极思想啊，不能说，说了就等于否定“机遇”，没饭吃没酒喝了，我调用最大意志力管住了自己的嘴贱，在脑子里把这个常识灭了。

虽然不是电信诈骗，边境赌场也是比较有吸引力的，而且现在身不由己，总不能白来吧........

说服自己很重要，先别管谁是鱼儿.......

**鬼混** 

后面几天都是例行混赌场，和财务混熟了，去二楼财务室溜达，索然无趣时，发现赌场机房就在旁边，下午终于找机会混进去。

一进机房，六七米的一面墙全是监控屏幕，好强大的气场，十几个服务器机柜，齐刷刷的刀片式服务器，交换机路由器的机柜仔细看了看，有电信级路由，至于光纤哪里来的，看看满满的手机型号，我自然是信服的。 

机房的人自然都是 IT 背景，拉开话题闲扯淡半小时，鱼儿告诉我，机房设备两百万，机房全套来自深圳 IT 公司，当时手机搜索了一下，还是个比较有名气的系统集成，硬软全包，定期还有人过来维护升级。

全然是流水线式产业链化了。

第五天，谈合作细节，入戏状态极好，月打底三万，提成比例多少。

当家的吹牛逼，说自己多少条线来自澳门，多少来自菲律宾，还有济州岛的，现在势头如何，月流水如何如何，当晚扎帐时还给我看看流水，一天三百万左右，但完全不懂行啊，就假装算账，浑浑噩噩的过去了。

最后讲到人手问题，当家的说“自己在国内招聘啊”，

我顺嘴就溜了一句话出来“这和电信诈骗路数一样啊，招的到吗”。

“赌场隔壁写字楼三楼，就是菲律宾搬家来的电信诈骗啊，他们都这样啊，你应该更懂啊”.......

我假装害怕，“尼玛他们不打人的吗”。 

“不会的，你去看看就知道了，人家很正规的”.......

卧了个槽，很熟悉的味道，

机会来了，

晚上躺床上，把最早追查电诈的“终结诈骗”公众号的相关文章，从上到下捋了一遍，可惜的是故事太多提纯度太低，不是我这种人需要的。

第二天，

瞪着红彤彤的兔子眼，来到电信诈骗公司，格子间里的人就三十多人，貌似搬过来不久，看起来很像中介公司。

头头不在，找来个管培训的副总，喷着一嘴的时髦词汇，招聘的套路看起来还不如北派传销。可惜的是，这鸟公司很多琐事还没捋顺，交流时总有人打扰，绕话术的时间又比较多，唯一的窗口就这么错过了，吐血三升。

**煎熬**

再往后几天就是煎熬瞎浪，和赌场老板去 M 波新赌场浪一下，过去要七八个小时，透透风也好。

路上经常有各种检查站，这时候拍照就无限制了。

![](img/836db9687af38e5aecdd6e1a4225c98c.png)

偶尔还遇到错车的，

![](img/0ccebe4c0f245087f71353e24fcbcc54.png)

挺紧张的，

570 在山间土路上狂飙，这车即便是顶配，对他们来说太便宜，完全放开了，对，就是那种感觉，我想开坦克也就这感觉了，

就像是绝地求生的特效全开，这是整个旅程里最刺激的画面，想起来头顶还疼，喝水时也总被洗脸。

![](img/61abc49c720c019d3fd38b25ae767ae0.png)

回途中还去靶场娱乐，东西拿手上看半天，没看明白是五六还是阿卡，悄悄拍一张还被瞪一眼。

![](img/debbaac06c03f33e1e8a626d739490ff.png)

回来修整一天就找机会撤了，回程在检查站就我一个人被卡了半小时，一同回的人在机场候机时笑问我为什么被卡，我也是莫名其妙啊。 

回来后半个月后，找到中间人，打开地图让他说说在地图上那个位置，完全说的不着边际，这就预示着中间人也比较懵，这事儿都混沌，别的技术细节就更难了解了。

只能弃钩了。 

**几年后**

某次路过，和中间人喝酒闲聊，“还记得 XXX 吧，人没了，我也搭进去两千万”。

我愣了一秒，注定的事情不多说，起杯喝酒带过。

后来偶然有人说有个 fcx 当年改行做这个，一个月两三万轻轻松松，也是湖南人。 

这个最失败的卧底经历里，法外之地，有牌照，有伞（鲍姓 LV 长），有货，有技术支持，全流程全要素，发财的机会唾手可得，他们只缺个熟悉网络的人引流而已，我摆出的是全盘掌控姿态。

看到赌徒里的大咖，多数都是一身罪债，也有传闻一些大案逃犯，也有些经济犯落魄于此，赌场门口还有各种赌术秘笈贩卖，各种奇门遁甲的洗钱套路（彼时地下钱庄还能运转，比特币洗钱刚刚开始尝试），变身套路花招更多，确实是开了眼。 

山里的产业园和小镇中心，是全然两种状态，小镇里忽略掉文字和车牌，很难感觉到不习惯，语言不同但通的人也很多。菜市场里各种奇奇怪怪的动物，什么都敢吃，但吃起来都是油炸，很崩溃。传闻这里不能打架，后果貌似严重，所以治安很好，这听起来也很洗脑的囧感。

![](img/3903514f898c8eecbcfcc38db15807ae.png)

赌徒圈子口头上是鄙视电诈圈的，但也只是局限于微词而已，至于如何发展到卖人的这种情况普遍化，很难在国内看到电诈变异流程的文字，只是再好奇也是找不到答案了。金三角的“传统项目”在这里被所有人害怕，基本生态已经被赌博电诈冲垮，也许有时候打败魔法的只能是魔法吧。

以抽离视角来审视赌场生态，看到赌徒的各种心态，以及周边生态个环节支配者的心态，相比传销这些经济犯罪，也只是少了些赤裸多了些伪装，终归是人，以自我利益为驱动。

多年后回头再看时，各种经济犯罪的社会心理背景，又何尝有区别，人性经不起诱惑，人性的不堪，也经不起审视。

对人性绝不幻想，因为，对利益从未放弃。
