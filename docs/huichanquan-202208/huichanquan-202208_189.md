# 柬埔寨已经成了人贩子的天堂

位于柬埔寨西南部的西哈努克港（简称：西港），曾是无数人的逐梦之地。前几年，在当地政府对娱乐和观光业的大力扶持下，大量海外人员涌入西港，试图在这座新兴的海港城里大赚一笔。

2018 年，西港只有 20 万本地人，而前来淘金的，光是中国人就达到了 30 万之多。
