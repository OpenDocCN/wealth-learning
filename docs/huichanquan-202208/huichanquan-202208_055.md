# 国内“陪聊”和境外“裸聊”诈骗团伙是什么关系？

近年来陪聊服务

在网络空间悄然兴起

用户花几十元或几百元

就可以获得

“聊天”“哄睡”“叫醒“情感代追”等服务

很多人向陪聊者倾诉交流、吐露心声

“网络陪聊”服务被称之为

“空巢青年”缓解城市孤独症的疗方

![](img/8d25d4aee6756be80421c9f7c8832b65.png)

在发展初期陪聊行业缺乏监管

接单群中充满“污麦”等“特殊单”

后因打击和监管力度的增强

行为虽有所收敛但涉黄并未杜绝

一些店铺打着

**“绿色陪聊，拒绝低俗”**的口号

涉黄陪聊转为更隐蔽的方式

这恰恰又被“走肾类”诈骗团伙相中 

一特殊的境内外“接力”的诈骗模式

火热地展开了

近日福建厦门警方

捣毁一境内外勾结的特大裸聊诈骗集团

抓获包括张某在内的犯罪嫌疑人 32 人

![](img/b0998c797390de5820bfb588e4400bb6.png)

经查张某在境内以“网络陪聊”为由

招募了一批声音甜美的女性

这些“聊手”通过小众交友软件

随机寻找男性聊天

获取被害人的信任

引诱他们下载安装带有木马病毒的 APP

播放事先录制好的涉黄视频

趁机录下男性裸聊的视频

![](img/3e06446e6daebd9c39dd7afcc10843c3.png)

一旦录下裸聊视频 

聊手就会联络在境外的团伙成员

境外成员会以不雅视频敲诈勒索被害人

并要求受害人将钱汇入指定境外账户 

再流回境内

![](img/6fa84e5be25a590845972ce59f128398.png)

初步查明该团伙在全国

敲诈勒索的被害人多达数千人

涉案金额达 2000 多万元

实际上招募“陪聊、陪玩”

是诈骗集团招人的常用技俩

**40 元一个小时**

**一天兼职 7-8 个小时**

**只要按照固定的文本**

**与社交软件上与男性用户聊天**

**轻松月入过万**

陪聊圈、陪玩圈群聊中不时出现的广告

吸引了不少年轻女性

2021 年年底莆田警方也捣毁一窝点

专为境外“色播”团伙引入受害男性

年轻女性应聘成功后

境外团伙成员对兼职人员专门培训

包括安装远程操作和虚拟机等软件

并提供相应的 QQ 账号和“话术”

这些女性听信了诈骗团伙的洗脑 

“只是聊天，没有参与敲诈勒索” 

一头栽进去

成了诈骗分子的帮凶

**反诈君想说**

在网络求职过程中一定要擦亮眼睛

明辨工作性质

避免落入犯罪分子的圈套

成为电信网络诈骗团伙的帮凶

最终令自己身陷囹圄

同时提醒广大男性网友：

**要洁身自爱**

在日常网络社交活动中

不轻信并接受网友的“激情”邀请

不轻易安装陌生人发来的链接
