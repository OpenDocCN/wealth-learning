# 我被小三短信轰炸，辱骂，威胁恐吓，甚至收到寄来的花圈...

> 原文：[`mp.weixin.qq.com/s?__biz=MzIyMDYwMTk0Mw==&mid=2247542624&idx=2&sn=5c3653f823e6209c702d8c80fcb4b67c&chksm=97cbec58a0bc654ed17f337c242d7300c210e0caa531c6f1a8fd6e63540af068ce3edbba0ba2&scene=27#wechat_redirect`](http://mp.weixin.qq.com/s?__biz=MzIyMDYwMTk0Mw==&mid=2247542624&idx=2&sn=5c3653f823e6209c702d8c80fcb4b67c&chksm=97cbec58a0bc654ed17f337c242d7300c210e0caa531c6f1a8fd6e63540af068ce3edbba0ba2&scene=27#wechat_redirect)

自从今年过年前曾经的丈夫突然离家出走，抛弃我一个人留在家里。男方一直逃避问题，爆出小三事件却一律不承认（婚内出轨，结婚半年后就一直瞒着我跟婚外异性发生关系），不肯回来跟我面谈离婚事情；最后我选择了法律诉讼程序去执行，到刚刚一审结束，调解书下发，耗了大半年时间。自从男方离家出走后，男方自身以及男方家人也一直没有找过我，甚至男方也屏蔽我，拒绝与我沟通。

![](img/361ef486a01220d58c3382d49e2dffc2.png)

我沟通是为了离婚的事情，如果男方依然用逃避敷衍的方式对待，我也只能继续走法律程序。但我依然一个人去面对很多事情，本来以为法院审理判决后事情终于可以得到解决，终于可以解脱...但为何作为一个男人却做事如此渣，如此不作为，还纵容小三如此猖狂。事情一日没有解决我依然一个人，小三知道我的住所和家里老人住所，威胁恐吓。是不是都没有天理了？我也很想大家去帮帮我，我不想将自己的私人事这样露骨摆出来，但真的会害怕之后自己会不会再被恐吓。与李先生这八年多九年的感情，这段婚姻教会我怎样好好过日子！这段日子我已经慢慢放下，所以我真的不想再让这件狗血事件，这些里外不是人的人一直精神折磨我.......

![](img/8a8753437c3bc00eecd06839852f799c.png)

由于离婚后需要处理一些交接事情，对方也没有跟我主动联系，我便打电话予男方沟通，但电话不接，短信敷衍说一直在工作，不接听，到最后也没有回复。之后熟悉的恐吓事件又上演了：

![](img/7d94928e74dff1f948e85492fb1cc4af.png)

这位婚内异性——小三半夜打电话，然后短信。就是怎样？我也不想联系，是要处理问题，需要过户，问时间，大家准备的材料怎样处理等等？搞得是我是小三似的，我要抢他“对象”？就是男方跟我联系是无敌大罪？

在没有开庭前的这大半年里，不断收到很多骚扰短信轰炸，电话；在我微博，小红书，工作室大众点评等社交平台收到很多来历不明的恶意评论，私信。甚至是恐吓我，甚至我家人，年迈 80 的奶奶，给我们寄花圈......
