# 跨国电诈组织国内头目：我们基本按这方法骗！

**“一道进、二道出、三道进、四道出、五道进、六道杀……”**这应该是你从来没有听过的一个套路，它通过短短三言两语，将电信网络诈骗犯罪分子实施刷单诈骗的套路和逻辑揭露了出来。  

近日，浙江嘉兴警方抓获一名嫌疑人何某，他就是这个套路的讲述者，而他的身份是柬埔寨、缅甸等地电信网络诈骗犯罪组织在国内的接洽人，涉案金额近百万元。

![](img/a223cce0debb72e56700f158760e8261.png)

**“养养再杀”**

电信网络诈骗团伙，都是怎么骗钱的？

这名参与骗取无数群众血汗钱，妄图逃避抓捕的犯罪分子被南湖公安擒获后，经过多轮讯问，终于交代了通过多名下属帮助境外犯罪集团洗钱的犯罪过程，及一个重要的刷单诈骗套路。

**何某说：“在我们行业，基本****<typo data-origin="上都" ignoretag="true">**上都**</typo>****按照这个方法去骗。”******

****民警通过讯问、研判，得知了一个套路。****

****<typo data-origin="上都" ignoretag="true">**“一道进”，**</typo>就是受害者相信并进行第 1 次刷单；****

****<typo data-origin="上都" ignoretag="true">**“二道出”，**</typo>对应的则是第 1 次付佣金给受害者，进一步骗取受害者的信任；****

****<typo data-origin="上都" ignoretag="true">**“三道进”，**</typo>再次提高佣金，诱使受害者进行第 2 次刷单；****

****<typo data-origin="上都" ignoretag="true">**“四道出”，**</typo>就是第 2 次付佣金给受害者，此时受害者趋于更加信任屏幕另一端的犯罪分子；****

****<typo data-origin="上都" ignoretag="true">**“五道进”，**</typo>是利用各种理由，任务、套餐、组合……通过更大收益，诱使受害者进行第 3 次刷单。通常来说，这将是诈骗分子最后一次诱使受害者进行刷单；****

****<typo data-origin="上都" ignoretag="true">**“六道杀”，**</typo>当受害者信心满满将大量钱财投入进去，正准备获取大额佣金时，犯罪分子已经拿着钱卷款潜逃……****

****![](img/1195370ff197ba11f5c5c051ac362244.png)****

******“你想要我的佣金？************而我们其实只想要你的本金”******

****犯罪嫌疑人，一定会按照这个套路诈骗吗？那我是不是可以先试着刷两单，给诈骗分子来一波“反杀”？不要太天真哦。****

****何某交代，这种诈骗，套路就是先骗取受害者的信任，所以前期看起来都很正常，甚至有的刷了三单还没有实施诈骗的也有。<typo data-origin="上都" ignoretag="true">**总结起来一句话：养肥再杀。**</typo>“你想要我的佣金？而我们其实只是想要你的本金。”什么时候被“杀”，团伙都是有综合判断的。****

****也就是说，狡猾的犯罪分子并不是每次都在第 3 次刷单时卷款潜逃，也有捞一票就跑、第二次刷单就“杀”的情况。****

****所以要记住：<typo data-origin="上都" ignoretag="true">**千万不能存有侥幸心理，自己挣的血汗钱不能花在骗子身上。**</typo>****
