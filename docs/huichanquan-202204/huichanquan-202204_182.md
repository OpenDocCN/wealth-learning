# 虽然我们没有买冰柜的土豪，但我们有最良心的团长！

疫情的两个月，我们从一开始抢菜、

在小区挖野菜，到现在的衣食无忧

其中脱不开团购群团长的功劳

这次疫情中脊梁也看到了各式各样的团长

有的团长尽心尽力，为团员服务

有的黑心团长漫天要价赚的盆满钵满

![](img/2192944877b2dab95aadaa3b102378be.png)

很幸运的是，脊梁所在小区的团长非常良心

**每次团购都直接按成本价发出去**

**从来不另外加价，有时候甚至还倒贴钱**

昨天晚上分完物资后我按照成本价算完账
告诉她考虑到蔬菜的折损
自己适当加一些利润再发给大家
结果**她都没有加价直接按成本价发出去了** 

![](img/36b2bca6757ee776e446572f5246dbfa.png)

团里的人都拿她开玩笑

**“你这是卖蔬菜还是送钱啊”**

而这个团长只是谦虚地说

**“大家都是邻居，多点少点没事的”**

![](img/dd64fb6627508d179136d69b0553367f.png)

这是她在静默期前

在小区发起团购的**26.8 元蔬菜大礼包**

26.8 这样的品质

真的是相当良心了！ 

![](img/d23009f6efc14ca940f67f3207b6b172.png)![](img/34373237df81c870e786836bd7e6c364.png)

除此以外，疫情时间的鸡蛋

她也是按 30 个按成本 25-30 元卖给大家

没有赚过黑心钱

![](img/5d118e56381a0aba7873621d70191bda.png)

这个团长平常就是做团购的

所以相对而言能找到的渠道就特别多

但能找到靠谱且性价比高的供应商

并不是件简单的事

**她现在找到的这些渠道**

**都是她自己掘地三尺到处比价**

**自己先下单到货后确认货源和物流靠谱**

**费劲心思才筛选出来的**

所以有些小区的菜贵也不一定是团长赚差价

供应商本身给的价格高也是理由之一

但也因为这位团长的良心

所以她每次发起团

购买的人都特别特别多

除此以外，愿意帮助她的人也特别多

脊梁小区静默昨夜团购物资到货

为了让她可以不用再贴钱请人搬东西

我们楼的居民都自发愿意帮她去卸货拉货卖东西

相当积极

![](img/65643628d4f7db2359fd1183870cc697.png)

所以大家才会看到我那天深夜

突然变身菜农在楼下卖菜

![](img/ccff5938b0a147303763c7228480e696.png)![](img/97c06542a9035e1cfab299b0b7cb7238.png)![](img/8e47bb30c253aa35da95d7b8540ca5fc.png)![](img/8951f577c1e87388559bc16d6447cd6f.png)

我们小区能有这样的好团长

真是再幸运不过的事

并且这样的良心团长在上海肯定还有很多

![](img/f52c34dd9e50e218b30d3b1a1a4d64e0.png)![](img/dcdf59498f6210c3f0b2406405f33ce3.png)

虽然我们小区没有买冰柜、送蔬菜的土豪

但是我们有最最良心的团长！
