# 防疫证明怎可伪造？再怕麻烦也不能因小失大

在当前阶段

我们经常需要提供、出示

核酸检测报告等防疫证明

但是有人因为种种动机

竟在网上购买虚假的报告、记录

这怎么能行呐

近日警方查获了一个

造假防疫证明、记录的团伙

也顺藤摸瓜发现了某些人

的造假行为

**不想重复做核酸，检测时间改一改**

在某通讯公司上班的小王，单位要求提供 48 小时内核酸报告才能进入办公场所。

他觉得自己没有去过外地不会有问题，懒得再去做核酸检测，于是通过网络渠道让商家对其过期的核酸检测时间进行了修改，并支付了相应费用。

![](img/efd9ce83b570c698e1b9acb30c0811dc.png)

据汪某交代，其购买的虚假核酸检测报告仅上报给了所在单位，未有其他用途。他前期没有认识到事情的严重性，现在十分后悔，保证再也不犯此类错误。

**为了虚荣炫个富，检测地点变一变**

居民孙某曾在某微信群内吹嘘自己住在高档小区，然而，事实并非如此。

当需要上传核酸检测证明到该微信群时，为了不露馅儿，孙某通过网络渠道找人，将核酸检测记录截图中的检测地点修改为某高档楼盘。

![](img/da62c50998c690a28343025eae2a31f4.png)

孙某表示，自己就是一时虚荣心作祟，觉得改变一下检测地点没有什么影响，以后再也不会犯这样的错误了。

**没有打疫苗，接种证明造一造**

李某某是一家公司的驾驶员，工作单位要求上传疫苗接种证明，但其因自身原因并未接种疫苗。

为能提供疫苗接种证明，李某某通过网络渠道找到自称能够修改健康码接种记录的商家，向商家提供了一张接种疫苗记录截图。

支付一定费用后，商家将其提供的疫苗接种记录截图中的接种人姓名改为了李某某，后李某某把修改后的截图提供给了公司。

![](img/b949c0d08cc9efe4046fdf05108bd811.png)

经过民警教育，李某某认识到自己的这种行为对自己和他人都是极不负责任的，表示将尽快接种疫苗。

事发后，经对上述涉及人员进行核酸检测，结果均为阴性。

有朋友看到这里肯定想问

那是谁给这帮人造的假呢

放心啦

这个专门修改防疫报告的团伙

已经落网

负责人谢某某、王某某

因涉嫌妨害传染病防治罪

已被警方依法采取刑事强制措施 

提醒一句，防疫大局当前

咱们可别因小失大

再怕麻烦，该做的也得做呀
