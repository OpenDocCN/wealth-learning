# 老师，余，男孩，生

(提问)苏酒滴小窝［失焦世界* : 老师，余，男孩，生于 7 月 3 日上午 11 点，求赐名。拜谢

2019-07-10

回答：余宅兹——西周初期《何尊》青铜器铭文：“唯武王既 克大邑商，则廷告于天，曰：余其宅兹中国，自之乂民。” 这 是出土文献中最早出现"中国"一词，是周武王定都洛阳的战略 宣言。男孩起这个名字，霸气侧漏，又内涵丰富，意蕴深 厚。(6 赞)
