# 唯大，可以说说“草

(提问)匿名用户 : 唯大，可以说说“草船借箭”吗？真实情况是 怎样的呢？

2019-06-04

回答：《三国志·魏书》确有“以船受箭”的史实，不过，主角 不是诸葛丞相，而是孙权。 建安十八年，赤壁之战 5 年后， 吴、魏两国爆发濡须口之战，“权乘大船来观军，公使弓弩乱 发，箭着其船，船偏重将覆，权因回船，复以一面受箭。箭 均船平，乃还。” 嗯，这就是后来的小说《三国演义》“草船 借箭”的最初源头。诸葛亮粘了孙权的光。(4 赞)
