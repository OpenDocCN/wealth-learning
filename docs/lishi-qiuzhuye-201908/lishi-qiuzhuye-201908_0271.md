(提问)Akko Akko : 老师，李白“高冠佩雄剑”见韩朝宗，这究 竟是时代风气，还是他自己不守规矩？

2019-06-22

回答：风气。 岑参《奉和中书舍人贾至早朝大明宫》，当时 上流社会，都是剑佩铿锵啊： 金阙晓钟开万户，玉阶仙仗拥 千官。 花迎剑珮星初落，柳拂旌旗露未干。(1 赞)
