# 春秋时期的兵车科技

(提问)匿名用户 : 春秋时期的兵车科技后来是怎么淘汰了？

2019-03-10

回答：公元前 541 年，晋国荀吴在太原与狄人作 战，因地形 险扼，战车无法展开，于是采用魏舒建议，“毁车以为行”。 放弃车战，改行步战，一举取得胜利。 这场战，标志着车战 已开始走向衰落，中国军事开始进入步战时代。 根本原因： 一是春秋中晚期后，战争地域越来越广，不 仅要在中原大地 作战，还要在华北、华中的山地与南江水网地区作战。 二是 战车极其昂贵。(4 赞)
