# 司马懿来到丞相营垒

回答：反讽意味浓郁。其实，司马懿给魏明帝的正式报告， 对丞相是这样定评的: “亮志大而不见机，多谋而少决，好兵 而无权。” 请注意，这里的“权”字，不是指权力，而是指权 变，说丞相喜欢打仗，但就是没有灵活机动的战略战术。[呲 牙]呲牙
