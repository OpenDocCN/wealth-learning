# 读到一首咏史诗，吴

唯为 : 读到一首咏史诗，吴融《陈琳墓》： 冀州飞檄傲英雄，

却把文辞事邺宫。

纵道笔端由我得， 九泉何面见袁公。

陈琳，汉末建安七子之一，擅长写文章。

他一开始是大将军何进的主簿，曾向何进献过诛灭宦官的计 谋，但不被采纳；

接着，董卓入京后他避难于冀州，投奔袁绍，为袁绍起草过 讨伐曹操的檄文；

再后来，官渡之战后，陈琳归附曹操，曹操军国书檄，多出 自陈琳之手。

群雄纷争乱世，陈琳一介文人，笔端如剑但终究不是剑，注 定是颠沛流离的命运。

他一生三易其主，于吕布何？
