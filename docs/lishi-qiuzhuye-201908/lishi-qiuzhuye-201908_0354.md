# 羊姐您好，问个比较

(提问)匿名用户 : 羊姐您好，问个比较沉重的问题。孩子 3 岁 多点，平时好玩但是性格敏感内向。他爷爷癌症晚期，基本 进入最后阶段了，状态也不好。是要让孩子经历这些事情好 还是让他回避好？会对孩子心理有什么影响？多谢！不方便 回答也没有关系，谢谢！

2019-06-02

回答：家里发生的事情，孩子隐约都知道的。瞒也没有用， 反而会让孩子自己瞎想。临床好多儿童行为问题都是这么来 的。 大人先好好消化一下，然后告诉他，看看他什么反应， 帮助他去面对。(2 赞)
