# 做“征信修复”那些人，都是如何骗钱的？

> 原文：[`mp.weixin.qq.com/s?__biz=MzIyMDYwMTk0Mw==&mid=2247539157&idx=6&sn=677fa44bde199d2d37f73a9aacd68bee&chksm=97cb92eda0bc1bfbca0955a345f605e2dd34d7c1286c31ca99d825f01b5c17fcb6ab26a4722e&scene=27#wechat_redirect`](http://mp.weixin.qq.com/s?__biz=MzIyMDYwMTk0Mw==&mid=2247539157&idx=6&sn=677fa44bde199d2d37f73a9aacd68bee&chksm=97cb92eda0bc1bfbca0955a345f605e2dd34d7c1286c31ca99d825f01b5c17fcb6ab26a4722e&scene=27#wechat_redirect)

如今，征信越来越受到大家重视，与征信有关的虚假宣传和诈骗套路也层出不穷，如有人打着“**征信洗白、征信修复**”的幌子，骗取个人信息或手续费等。

部分金融消费者办理贷款业务时，发现信用报告上有逾期记录，便想着找人修改，骗子利用这一需求，**编造理由、伪造材料、帮助作假**，声称可以修复征信，最后却是卷钱消失。

这不仅损害金融消费者经济利益，导致个人信息泄露，还可能造成**法律风险**。下面来看看常见的“修复征信”骗局。

1

**本想修复征信**

**却遭遇骗局**

**搜索“征信修复”关键词，网上常出现“消除征信逾期，成功后再收费”、“逾期不要慌，专业修复找我们”等类似**广告**，看到这些宣传时，你会相信吗？**

****案例 1：消除不良征信被骗****

**前段时间马女士接到一个陌生电话，对方自称是某金融公司工作人员，电话中称马女士存在不良征信记录，**需要及时消除**，否则会影响以后生活。马女士信以为真，按照对方指示下载了某贷款软件并借款，再向对方提供的账户转账 3 万元，转账后才发现被骗。**

****案例 2：交定金修复征信被骗****

**陈先生因忘记还贷发生逾期，在网上找到一家“专业修复征信”的机构，对方表示可以根据他的情况定制修复方案，但**需要先交 2 万元定金**，事成后再付剩下的 3 万元。陈先生交钱后，还给了对方自己的详细信息，十多天后，对方表示修复成功，还发来信用报告截图，要陈先生付余款。陈先生到银行核实，发现相关信息并未修复，修复成功的截图是作假，再联系对方时已被删除。**

****案例 3：加盟征信修复被骗****

**王某在网上看到汤某经常发“征信修复成功”的案例，便联系对方想代理这门技术，汤某表示大多数人的逾期问题都能修复，未来有很大赚钱空间，只**要交 5 万元加盟费**，就可以代理修复技术。王某信以为真，交完加盟代理费后，便将一些客户的逾期记录交给汤某修复，汤某却已经卷钱消失。**

**出现逾期后，要及时联系银行处理还款，**终止逾期行为**，如果认为信用报告有误，可以提出并要求更正，这种处理不会额外收费。**

**接到陌生来电，**不要相信**所谓消除不良征信的作假手段，要第一时间向银行核实，也不要去加盟或代理征信修复，里面欺诈风险极高。**

**2**

****所谓修复征信****

****步步是陷阱****

****骗子利用很多人不了解征信、**怕麻烦**的心理，声称可以代操作修复，在受害人恐慌和焦急时，不断施压并收取修复费用，主要套路如下：****

******0****1************收服务费******

****骗子所谓花钱就能修复征信中心的报告，其实是**收取高额费用后失联**，或者让人编造虚假理由去无理申诉。****

******0****2************骗取贷款******

****有的还会仿冒金融机构人员，声称受害者办理过贷款要注销、不良征信要消除，以此骗受害人**去****贷款平台贷款**，并转到指定账户。****

******0****3************要代理费******

****协助或指使受害人伪造证据，用虚假材料进行投诉，借机收取高额的**代理费**，一旦申诉失败，骗子便卷钱消失。****

******04************骗取信息******

****在修复征信过程中，骗子堂而皇之地让受害人提供征信信息、身份证、银行卡、手机号等**隐私信息**，再通过买卖信息获利。****

****修复征信，步步都是陷阱，骗子既骗取了受害人的个人征信信息，也侵害了**财产安全**，还扰乱了社会征信的正常秩序。****

****3****

******保持良好征信******

******警惕修复骗局******

******征信是指由中国人民银行征信中心统一建立的“**信用档案**”，它记录了个人或者企业过去的信用行为，是申请贷款必备的重要参考资料。******

****个人征信报告由中国人民银行征信中心统一管理，其他任何机构、任何人都无权删除或者修改。《征信业管理条例》第七条规定，未经国务院征信业监督管理部门批准，任何单位和个人不得经营个人征信业务。****

****所以，看到“个人征信可以伪造材料修复”的信息一定要警惕，平时的金融生活中，我们要注意以下事项，**警惕征信修复骗局**。****

******0****1******

****养成良好的还款记录，记住还款时间，按时足额还款。****

******0****2******

****妥善保管个人信息，不随意向陌生人提供自己的个人隐私。****

******0****3******

****遇到逾期情况，及时向正规金融单位咨询，勿轻信陌生网友。****

****请大家**擦亮双眼**，提高识骗防骗能力，守护好自己的财产和信息安全。对征信记录确有异议的，可以自己申诉或先咨询有关部门，不要相信所谓“征信修复”机构的片面说辞。****
