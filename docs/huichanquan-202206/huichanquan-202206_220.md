# 骗子想从缅甸回国，万万没想到……

也许很多人都知道

有不少犯罪团伙和诈骗分子

藏在缅甸等国

隔着千山万水

进行这样或者那样的诈骗

然而你知道吗

他们中的一些人因为太想回国

也遭遇了骗局

**表哥的故事**

**一个多月前的某天，浙江某地派出所，来了一位报警群众，称他和表弟被人骗了。**

**接警民警拉过来一把椅子，坐坐坐，先说说咋回事，然后把手续走了先。**

**![](img/3a671061699343344b192ba59ce9a6e4.png)**

**你说你和表弟都被骗了，那你表弟人呢？报警人回答，还在国外。**

**嗯嗯继续，然后呢？**

**报警人继续说，他的表弟之前去了缅甸，但因为那边疫情严重，表弟想快点回来，可手续非常麻烦，现在还在那边隔离排队。**

**后来，表弟看到某视频平台上有个人声称可以帮别人回国，就加了这个人的微信。这个人说先付 5000 元订金，直接找关系帮表弟回国，事成之后再付尾款 1 万元。**

**表弟身在国外身边没有这么多钱，于是便向表哥求助。报警人也就是这位表哥也没多想，当即把 5000 元订金转给了这个自称可以找关系插队回国的人。**

**结果就如寻常的诈骗案一般，对方收完钱直接跑路了，钱没了，人还一动不动，没法回来......**

**民警随口又问了一句，那你表弟是什么时候去缅甸的呀？表哥老实地回答：“有段时间了，他是偷渡过去的。”**

****偷渡可还行？！****

**![](img/3eff06d6c05a05579cfc43734f2e18ec.png)**

**过了半晌，民警对这位表哥说：**

**案子我们马上开始查，等你表弟人回来，务必要到我这里报到哈。**

****表弟的自述****

****我真不知道一开始怎么就脑袋糊涂，来了缅甸，所谓的“好工作”又脏又累又危险，分不到几个钱，不知道图个啥！****

****不知不觉，我在缅甸演“高富帅”骗人，也演了几个月，我累了。****

****听说我的同学、朋友都拥有了稳定的生活，有的结婚，有的做生意，而我却在缅甸过着猪狗不如的生活。****

****![](img/4f0c86d18eb25292acdc45ed33a1913e.png)****

****啊，我想要回到国内过上平凡但是安稳的日子啊，我要回国……****

****不少人离开了，就像来时一样，走，也是偷渡走。可我不想再忍受偷渡车里闷热的空气、偷渡头子蛮横的态度和沿途的重重危险。****

****我要光明正大地回国，即使回去就被抓，我也愿意。可缅甸这地方疫情肆虐，想回去的人又太多，所以只能排队隔离，要等上很长时间。****

****![](img/f528f8d4fc4eded64333408a584a46c9.png)****

****我真的太着急回国了，于是在网上乱搜，发现了一个自称有办法的人。****

****这个“贵人”说有关系，已经帮好几个人弄回去了，要我先给他 5000 元订金，事成之后再给一万。****

****可我在缅甸的日子里，根本没挣到几个钱，没办法，借吧。我找到了最亲近的表哥……****

****表哥一听我要回来，高兴坏了，这让我心头温暖，还是家里人好。****

****没过几分钟，表哥就说已经转账了。“贵人”说收到了，现在开始安排，估摸着三四天之后来接我。我一想快回国了，饭都比以前多吃了一碗。****

****可是 3 天之后，并没有任何人来接我，我立马质问那个“贵人”怎么回事，可是聊天框却出现了一个刺眼的感叹号。****

****![](img/114f830b7b66c95a0aa8b426d84d3208.png)****

****终于，我，也尝到了，被骗的滋味。****

****现在，表哥已经去报案了，我则老老实实地排队等着回国。****

****无论什么时候回去，我都很确定一件事——以后我再也不会来缅甸搞诈骗了！****
