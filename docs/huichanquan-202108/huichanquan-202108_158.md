# 隔壁男歌手被前嫂子揭发私生活混乱，与发小老婆独处整夜，还在综艺上秀恩爱！

> 原文：[`mp.weixin.qq.com/s?__biz=MzIyMDYwMTk0Mw==&mid=2247521025&idx=1&sn=436243abd096639b4cf1720cdcba8465&chksm=97cb5839a0bcd12f573edcf2f3b6db1c0c813ef6ec7d30f2cffc25002cedd77f365e2aa1cdb4&scene=27#wechat_redirect`](http://mp.weixin.qq.com/s?__biz=MzIyMDYwMTk0Mw==&mid=2247521025&idx=1&sn=436243abd096639b4cf1720cdcba8465&chksm=97cb5839a0bcd12f573edcf2f3b6db1c0c813ef6ec7d30f2cffc25002cedd77f365e2aa1cdb4&scene=27#wechat_redirect)

正在《女儿们的恋爱》中秀恩爱的歌手隔壁老樊（本名樊凯杰），被网友@小田田有点甜揭发生活作风问题。她这次发长文举报“失德”艺人樊凯杰，控诉隔壁老樊的种种不检点行为，发文要对方给自己道歉。 

![](img/5255286bd580bb12be5e8c980220ea0c.png)

该网友称，她是隔壁老樊亲哥哥樊凯华的前妻，也就是隔壁老樊的前嫂子，因为他哥家暴离婚。

![](img/f0e3b70411499dd2b0fb229cfc3d3cb4.png)

她的长文只针对歌手隔壁老樊，有六个诉求，每一点都信息量很大。 

![](img/452ae411c64a9a3a103ba0484eb64de8.png)

第一，隔壁老樊曾经试图插手哥哥的感情，让哥哥玩够了就把女方甩了再找个好女孩结婚。为此，隔壁老樊也和女方当面道歉，但地点选在 KTV，在道歉的同时还不忘给发小安排“小公主”。

![](img/12b40f9d4a116390f281776c8d9575fd.png)

![](img/2e73fa44961af10227ebde15a3b284bd.png)

第二，在道完歉当晚隔壁老樊没有回家住，反而在有对象的情况下跑去了发小妻子家里过夜，这样的行为多少有点不合适

![](img/9463920966e233617a728f4c3d78eaa6.png)

有个粉丝后援会的会长跟他在一起过，还晒出截图。

![](img/7796b1feb82a2c344ce1a622b96c1ee5.png)![](img/8072a3e407766be9e83e42694bc7530d.png)

前嫂子晒出当时拍下的视频作为补充，她也用文字标注出了每个人的身份，不难看出，现场确实有很多人在，而且所谓的“小公主”也不止一人。

![](img/da008296e79d8ac31c553d1f294348a3.png)![](img/93247f39099351d020a032ba6dc7c6c6.png)

甚至还有粉丝爆料称自己也被樊凯杰骗过，侧面证明了樊凯杰确实玩的很花，而且还说他很会假装。 

![](img/3589e856e2103f52dcfd2a5d994e7dfa.png)

![](img/80b125ca5ffcdf9d2e52288d6b579835.png)

前嫂子还补充表示，他在有女友的情况下也会骗别的小女生，甚至还以此作为谈资拿出来炫耀，正因为樊凯杰会告诉哥哥，所以这位前嫂子也看清了隔壁老樊的真面目。

![](img/95d4f09947fb213737827d856f619105.png)

另外，她说老樊在李莎旻子面前很乖，但是在他们面前又是一种样子。他带李莎旻子去的都是正常的 KTV，但是跟他们在一起就会点公主，质疑他表里不一。![](img/f5f2f79971cf6a4a891ab2f0cbaff931.png)

最后，老樊的爸爸不觉得儿子的行为有错，还说他们那里的人在 KTV 里都会这样，毁人三观。![](img/f8602237b4fb42d113aa578f6c9ce313.png)

由于内容涉及的隐私太多，有网友质疑@小田田有点甜是不是真“嫂子”，她立马晒出与樊凯华的结婚照，证明是嫂子本嫂。结婚证显示，两人是在 2020 年结婚的。

![](img/12c0e21ab3dc8e28e66b08bf94158e6d.png)

![](img/b6debd7cee402f14a99be6558070b251.png)

上文中提到的李莎旻子，也是个歌手，算个公众人物，2021 年 3 月份两人公开恋情。网友疑问她秀恩爱的对象私生活如此混乱，她知道实情吗？

对此，前嫂子回应称，早就把 KTV 的视频、聊天记录都发给她了，但是她不介意。

![](img/6e4656ad28e327e9f660cb3642a11418.png)

![](img/2c92d9d7bf1e782999107368df34cf87.png)

李莎旻子怎么选择，跟@小田田有点甜的控诉无关，她的诉求很简单，只是想要一个道歉。虽然她和樊凯华是因为家暴才离婚的，但老樊之前发的消息，还是让她耿耿于怀，她曝光出这件事，一方面也是为自己出口气，另一方面也是想揭穿老樊的真面目
