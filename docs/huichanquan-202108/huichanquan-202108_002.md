# 千万别上当！网红主播设计剧本卖惨带货，网友纷纷上当受骗！

“我们家的石榴又大又甜...无人收购，很多石榴裂开烂在地里，只能做肥料。”8 月以来，随着凉山州会理石榴的成熟，网络平台上出现了少部分“网红”以“作假”“卖惨”直播的方式带货。

卖惨带货的套路有多深 记者辗转找到了上述视频中的两位老人，面对会理县市场监督管理局工作人员的询问，两位老人都否认自己说过这样的内容，坦承是有陌生人叫着帮忙出镜拍视频，但文字内容是捏造的。

除了这类卖惨视频，有些主播在直播时会选用品相更好的石榴造假，或故意压低石榴价格，称会理石榴几毛钱一斤都没人要，以引起果农恐慌，低价抛售石榴，从中渔利。
