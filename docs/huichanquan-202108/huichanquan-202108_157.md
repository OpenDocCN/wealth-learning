# 紧急预警！警惕“债权转让回款”骗局

> 原文：[`mp.weixin.qq.com/s?__biz=MzIyMDYwMTk0Mw==&mid=2247520967&idx=7&sn=67bad93f747c1977fea5b589f3535f32&chksm=97cb59ffa0bcd0e90cfc96ef3a8fcd66b58e4972d8b4afad70ec9ab14e916d8369539d1ce79a&scene=27#wechat_redirect`](http://mp.weixin.qq.com/s?__biz=MzIyMDYwMTk0Mw==&mid=2247520967&idx=7&sn=67bad93f747c1977fea5b589f3535f32&chksm=97cb59ffa0bcd0e90cfc96ef3a8fcd66b58e4972d8b4afad70ec9ab14e916d8369539d1ce79a&scene=27#wechat_redirect)

近日，网上疯传着这样一条预警通告：“近期，有不法分子假借某些 p2p 公司名义，打着落实国家清退政策的旗号，以私兑回款为由，骗取投资人的身份和投资信息后，引导下载快三，北京 28 之类的赌博平台，骗取投资人的钱财。

并且作出了提示，关于 XX 公司的资金清退和回款兑付工作，目前我市只对接了深圳达晨创投数字区块链交易所进行本息回款兑付工作，目前 XX 投资官方回款工作对接企鹅群向出借用户公示:16247833X……”，落款为北京市公安局反诈中心。

****![](img/1654abd91c8c03efa4dd0b365f148b22.png)****

******而这种通告已经在被各地公安局进行辟谣，但倒下的成千上万个 p2p，背后都是一个个庞大的易受骗人群，这一次骗子们显然经过了精心的谋划，环环相扣，让人陷入可以拿回损失的幻想中，不料却是再次掉进深渊。******

********骗子们这次是真大胆，直接冒用对手的名义，全国有公安局的地方就有反诈中心，警察每天必做的一件事就是反诈宣传，所以用这个名号足够响亮。而且 p2p 受害者更是各省市都有，所以这次的受骗基数庞大而且极易中招。我们一起来扒一扒这次骗局的新套路。********

******来看看骗子的套路吧************![](img/22de6dfdeea9715b4173319ba3c1be91.png)******************01************************伪造文件、公告************

************不法分子伪造**红头文件**或者公告，谎称自己为深圳市小牛在线互联网信息咨询有限公司、政府机构，然后召集网贷平台投资受损人加入官方清退群。************

**************02**************************虚假回款方案************

**********很多 P2P 出借人在平台兑付出现问题后，维权心切，病急乱投医加群。此时不法分子以公司清退的名义群发消息，与受损人取得联系，提供回款方案。**********

************03************************套取钱财跑路************

************诱导受损人按照指示，一步步进行回款操作，从中套取受损人的资金账号、银行账号等信息，实施诈骗。************

**********所谓的回款只是让你加群，给你两种方案让你选择。可实际上就是让你注入资金，也就是只有转账才能给你分批返还之前被骗的资金。醒醒吧！已经立案侦查或者进入审判阶段的案件，返款是需要经过法律程序，而且 p2p 相关案件也不是反诈中心管辖范围。虽然骗子用心做了带公章的行文，可是请大家牢牢记住。**********

************全中国的公安局不会通过 QQ 群、微信群办案的，更不会让你转账的私人账户的。如果你也收到此类诈骗信息，请通过全民反诈 APP 举报，或拨打 110、96110 进行咨询。************
