# 【观点】用技术的角

阿辉（私人号） : 【观点】用技术的角度看商业趋势 最早做 推荐式的其实是淘宝，也就是我们之前说的千人千面，标签 是升级版的一人千面。但目前推荐式算法，做得最好的是头 条系。因为信息流的用户时长比商品流要长的多，所以采集 用户行为更精准，推荐也更容易得到用户喜欢。 微博当时也 遇到股价暴跌，后来阿里成为大股东，然后在微博也用推荐 式算法。也就是我们常说的千人千面。由于微博的主流信息 生产还是去中心化生产，也就是大 V 生产的，而今日头条的主 流信息早期是采集的，还去掉了外链，绝对的中心化运作模 式。 再到后面头条引进内容创作者，就如现在的抖音引进视 频创作者。表面是去中心化产生内容，但实际是绝对中心化 运作的。这种中心化的运作方式，对于平台而言确实大大的 提升了用户的停留时间，减少了用户流失。 接下来两年，如 果推荐式算法的技术门槛降低的话，很多中小型平台都会 用。一旦是这样的话，相当于命脉掌握在别人手里。 比如录 了抖音，你想持续创作内容获取粉丝需要消耗大量的精力。 然后粉丝变现由于平台管制，变现链条就变得很长。再加上 意向粉到精准粉又存在转化率的因素，用户价值链条也非常 长。 所以，我建议，一定用最快的方式去圈粉，不要管有没 有用。两年后，就会知道今天这个决策多对了。
