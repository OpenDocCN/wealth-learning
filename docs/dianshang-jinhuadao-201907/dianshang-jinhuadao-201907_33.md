# 项目初创时期，活着

阿辉（私人号） : 项目初创时期，活着找方向比什么都重要。 例如我们的新平台，第一步就是招一部分商家，这样可以活 着。然后，不断围绕商家和用户去验证，验证流量成本，验 证效果，验证规模可能性等不确定的因素。 初创时期切不能 操之过急的去赚钱，因为没有验证清楚是否能给用户提供独 一二无的价值，流量成本等因素的话，很难搞清楚自己到底 是老鼠还是大象。 创业中期，加速比什么都重要。 买家秀网 的模式，当时验证了对商家效果非常好，验证了一个流量成 本 36 元。那么接下来要做的，就是快速去扩大 B.C 两端的用 户，形成规模和标签效应。让对手很难超越。
