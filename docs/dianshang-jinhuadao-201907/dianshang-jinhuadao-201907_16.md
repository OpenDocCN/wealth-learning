# 我现在在公司内部从

阿辉（私人号） : 我现在在公司内部从来不教具体方法。因为 我发现方法太容易过时了，，我只给方法论和建议。 例如下 面 4 张图，是跃龙的提问，涉及到用户使用场景，用户需求， 闭环的设计，以及用户的获取成本。 我如果直接去回答，我 肯定是错的。因为不是用户，而最好的方式就是看用户怎么 说。有些产品，不太好了解用户的场景的，最好就学会自己 去变成用户，或者用户打电话，或者去看百度指数，微信指 数，看产品相关的关键词的指数。 补充一点，再做调查的时 候，很多时候用户不愿意说真话。我也吃过很多亏。现在总 结是，不要太相信用户说了什么，要思考用户为什么这么 说。
