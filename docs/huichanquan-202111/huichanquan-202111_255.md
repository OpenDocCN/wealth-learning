# 某公司老板娘的“自述”：我被假老公骗了 16 万

我真的以为这种事

离我很远

被我亲手转了出去

![](img/795e1c98eae2b1028909acb69b121997.png)

***01***

我，在顺德一家公司从事财务工作，同时也是公司的老板娘。

因为工作的性质，我比他人的警觉性更高。理所当然的，我自觉受骗这种事，也不会发生。

那天，我回到公司后，登上了自己的 QQ，一个从未见过且以公司名称命名的群聊弹了出来。

我的“老公”也在其中，聊得火热。也许是察觉到我上线的信号，下一秒，我就收到他的消息。

> ***“老婆，最近是不是有人联系公司进行账号年检？”***

奇怪，这个群我是如何进来的？是他专门为年检工作建立的 QQ 群吗？怎么不直接来跟我沟通呢？线上联系就不怕遇到骗子吗？

无数个疑问萌生在我的心头。转念一想，算了，同床共枕多年，这口吻跟头像都是我老公的模样，应该是他无疑了。

“是的，就是前不久的事情。”我如实回答道。

***02***

没有过多地怀疑，随后我便依照他的要求，截图公司账户余额，发到群里。

> ***“我给你发了个文档，里面有需要转账的款项和账号。”***

> ***“老婆你对照着账号转款就可以了。”***

这钱款还真不少，怎么这么大的事儿也不提前跟我商量一下。不过，他应该有自己的考虑吧。

于是乎，我按着文档内的项目，先后往“老公”提供的账户一笔笔转账，**共计 161000 元**。

搞定，收工！

下班回家见到老公，跟他闲聊时提到 QQ 群的事。

> ***“我今天没有让你转账啊。”***

话音刚落，我从沙发上惊坐起来。

等一下！难道？！

原来，群聊是假的，里面的“老公”也是假的。

“报警吧……”

***03***

故事讲到这里，相信大家都知道，**这是典型的冒充老板 QQ、微信诈骗**，不同的是，我的老板也是我的老公。

回看整个骗局，你或许会觉得很多地方并不合理，甚至可以称得上是漏洞百出。

然而现实中，相似的案件却还在发生。不法分子通过非法手段盗取员工 QQ、微信，掌握公司内部人员架构，再建立虚假 QQ、微信群，把财务人员拉进群后，冒充公司老板 QQ、微信向财务人员发送转账指令。

与其“亡羊补牢”，不如“防微杜渐”，防止受骗。

1.完善相关制度

公司老板和财务人员要制定严谨的公司财务制度。

2.不轻易转账

转账汇款均需严格按照制度执行。通过 QQ、微信等指令转账、借钱的不要轻信。 

3.再三确认身份

财务人员要与老板当面或亲自拨打电话确认后再进行转账汇款。 

无论骗术如何变化，话术如何精湛，归根到底，不法分子的目标也只有一个：**让你转账！**

所以转账前不妨再三深思，“钱袋子”再系紧一点。
