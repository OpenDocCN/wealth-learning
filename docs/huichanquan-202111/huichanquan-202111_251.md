# 爱你的人不谈钱，谈钱的人不爱你

**0****1****婚恋平台偶遇“真命天子”**

叶女士，通过自己多年的经商打拼，攒下了一定的积蓄，只可惜感情世界依旧一片空白。

**闲暇之际，她在某社交软件上注册了个人账号，寄望于网络可以解决情感问题。**

一陌生男子突然添加了叶女士的微信好友，**称**自******己是通过珍爱网获得叶女士的联系方式的。**

![](img/d3db75ba52e16cc928fdc7aa89ccf727.png "8998267305229333758.jpg")![](img/f8004ed294d1ff936f209adf95f5b1a6.png "8685420210673294132.jpg")

骗子发来的自拍照

接下来的日子里，该男士开始对叶女士展开了猛烈的情感攻势，每天嘘寒问暖，关怀备至，**这让久未感受“爱情滋味”的叶女士倍感甜蜜。**

![](img/06ff381bdf9ecc52b1d61d886e0886e1.png "7841481571708691216_ 副本.jpg")

![](img/df8d74412dad420f01301bb3f62e3329.png "5039300608608276140_ 副本.jpg")

（甜言蜜语攻破心理防线）

**很快叶女士便彻底沦陷，仅仅 10 天就和对方建立了恋爱关系。**

**0****2****为了“爱情”掏空积蓄**

不久，该男子称自己最近在**某货币交易平台投资赚了不少钱**，为了两人将来的生活，让叶女士也跟着一起投。

![](img/1e11a6caead66f5f0461e7bb4adb231f.png "2512764938212960490_ 副本.jpg")

此刻，被爱情冲昏头脑的叶女士没有思考太多，在“男友”的指导下，两天内投入资金 90 万元。**令叶女士没有想到的是，她在该平台投入的资金在三天就暴涨到了 300 余万元，顿时让其欣喜若狂。**

![](img/c0dff0f7fb27974455fdc43f408e445e.png "8766560513210384197_ 副本.jpg")

（暴涨的资金其实都是后台随意操纵）

**0****3****“男友”失联，人财两空**

**但快乐的时光总是短暂的，变故和这“爱情”一样来的猝不及防。**

叶女士准备提现，却发现资金被冻结，联系客服后，对方让其缴纳 50 万元的解冻费。

![](img/14e5f980a342c3f095e4aa726c12431d.png "373371254167983622_ 副本.jpg")

叶女士发现自己可能被骗，联系“男友”却发现对方失联，其随即打电话报警，民警确信叶女士遭遇了杀猪盘诈骗。

（案例来源：93 反诈）

![](img/8d5300c615133ce86eaeee91bc2aaab8.png)

网络一线牵，当心是诈骗。希望大家在网上投资、交友时，能够保持警惕，不要轻易相信所谓的“稳赚”项目，更不要对任何非法投资获利抱有幻想，避免成为骗子砧板上的“待杀猪仔”。

**警方提醒**![](img/7fa457caf18b6850f327610afea940b6.png)

1、网络交友提高警惕，不要被陌生人的甜言蜜语迷惑，不要和没有见过面的人谈钱；

2、不相信“有内幕、稳赚不赔”的博彩、理财，守紧自己的钱袋子；

3、不向陌生人提供个人身份证号、银行卡号、密码、验证码等信息。不扫描陌生人发来的“二维码”，不点击来历不明的网页链接；

4、如不慎被骗或遇可疑情形，请注意保护证据，立即拨打 110 报警咨询。
