**好家伙，今天刷到一条狗血新闻，你敢信，从头到尾除了记者竟然没有一个正常人……**

这段狗血纠葛要从 2017 年说起。

2017 年，卢小姐与郑先生在网上相识，两人聊了一个月就陷入热恋，郑先生说自己四年前离婚，而卢小姐说虽然郑先生有过婚姻，但她也不介意。

![](img/3e6ddd6048dfc8b603425a6b940f3283.png)

![](img/5e3842129a7476a7a617c8fbe994ba51.png)

![](img/8e7f57f06778704fdabcbb5a691c11a6.png)

之后两人一起去到深圳办厂并同居，卢小姐陆续为郑先生生下两个儿子，大儿子今年两岁多，小儿子只有 8 个月大。

![](img/8073bb04010a1daa6466d6de1463a12a.png)

虽然生了孩子，但两人一直没领证，因为郑先生说自己欠银行 50 多万，**等还清了再和卢小姐办结婚证。** 

![](img/080294537bd17bdbf886f8cd13a961c6.png)

![](img/2224fd958de1fe7f9824ed46d12d5928.png)

![](img/6df9fc3d2b517d28116ffffd43f09e5d.png)

孩子的户口也要等郑先生还完欠债再说。

![](img/2a47f2cf8b22b338cfb6098d4af2b735.png)

去年 4 月份，工厂搬到了佛山，郑先生开始不断出差，此后，两人分居在佛山和深圳两地，两人的感情渐渐出现裂痕。

有一天，卢小姐突然发现，郑先生和一个赵女士有染。

对此，郑先生的解释是：**跟赵女士在一起是为了赚钱，给小孩买奶粉。**

![](img/cf2245c136be3d040dc99bd1ee10456f.png)

![](img/f833402154d317be82cbe1c73ad80ec5.png)

顺便一说，这名赵女士就是郑先生工厂的投资人，赵女士还将他俩的聊天记录和照片发给了卢小姐。

![](img/b07e2ac09d140dc767162f61f77ab841.png)

![](img/e2a6457c39a08f01f1f38eeb6467cca8.png)

**郑先生不仅否认出轨，还称一切都是卢小姐自愿的：“她非要生！我一直强调说不要生。”**

![](img/b85ceb59f703e4ce5cc91037449c7644.png)

![](img/8a24439ad76abcca7b06b7e60948f8a1.png)

郑先生不以为然：“小三现在很多的。”

![](img/7a92775a7e20aeeaeb651caea969b3ef.png)

**此外，郑先生还提到：“我已经结婚了，（卢小姐）从一开始就知道，我没有承诺跟她结婚。”**

![](img/5c72d6cffd9baef0689f124b984a88dd.png)

![](img/7c1c734b01c61cef884bee196f44fcb7.png)

![](img/fdafe4916cbf9b52b403cdac5246d072.png)

![](img/a1dc08049a1720d8c66568faafbbdb99.png)

**郑先生：“男人谁没在外面玩过。”**

![](img/bb87056e0701a655d9b35e6148bc75ae.png)

没想到，那头赵女士竟也为郑先生喊冤指责卢小姐难缠：**“男人嘛在外面玩一玩很正常。”**

啊这，是赵女士亲口说的吗？？男人嘛在外面玩一玩很正常的。

目前，卢小姐已报案，哭诉自己的青春和名誉全都毁了。
