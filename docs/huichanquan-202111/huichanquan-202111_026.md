# “120 升油箱被加到 161 升”，当地凌晨通报

据微信公号“盐亭市场监管”11 月 17 日凌晨消息，针对某加油站“120 升油箱加了 161 升油”事件，[此前报道👉《“120 升油箱被加到 161 升”，警方介入》](http://mp.weixin.qq.com/s?__biz=MzIyMDYwMTk0Mw==&mid=2247524177&idx=4&sn=ebc24c88ae1cd96a84847f56e0635838&chksm=97cb5469a0bcdd7f11d47ed748270912c424111ac0eca17c68e442786218e72279b3bd3e2f5f&scene=21#wechat_redirect)

四川盐亭县官方发布情况通报称：

经专业检测机构对该加油站加油机检定，**加油机为“不合格”**，专案组对不合格加油机采取“查封”的行政强制措施。

此外，经调查，**排除加油站工作人员限制当事人人身自由情况**。

目前，相关调查工作有序进行，调查终结后，将依法从严处罚。
