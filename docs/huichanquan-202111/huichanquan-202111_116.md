# 被骗到缅北做诈骗，怎样逃离缅北诈骗基地?

> 原文：[`mp.weixin.qq.com/s?__biz=MzIyMDYwMTk0Mw==&mid=2247525212&idx=3&sn=f404e544ebea8b60f0a90a0cc0702a0b&chksm=97cba864a0bc2172ff4b2824ad472f1068026b5fb347ec775ea2863e6bfc81922cd6bcc82ebe&scene=27#wechat_redirect`](http://mp.weixin.qq.com/s?__biz=MzIyMDYwMTk0Mw==&mid=2247525212&idx=3&sn=f404e544ebea8b60f0a90a0cc0702a0b&chksm=97cba864a0bc2172ff4b2824ad472f1068026b5fb347ec775ea2863e6bfc81922cd6bcc82ebe&scene=27#wechat_redirect)

随着国内对于电信诈骗的重拳打击，国内的电信诈骗团伙现在大多已经转移到国外，尤其是东南亚地区。

最初，转移到国外的电信诈骗会盘踞在菲律宾、马来西亚等东南亚地区，但近五年来，缅甸北部逐渐成为这些诈骗团伙首选的栖身之地。 

缅甸北部主要指中缅边境上的四个自治区，他们分别由独立于中央政府的四个军事武装力量所控制，处于“军阀割据”的状态。这些地方受中国文化影响颇深，和云南省在生活习惯和语言上非常相似。

官方语言里包括汉语，可以流通人民币，也可以使用到移动、联通、电信三大运营商的信号。

出于经济发展需求，当地军政府甚至成立了专门的经济开发区，为这些从事电信诈骗的犯罪分子提供专门的营业许可证，来支持电信诈骗的发展。正是因为这种种原因，缅北成为了诈骗犯罪分子的天堂。 

![](img/7d559f2b925f0b37a9796f1cfa329186.png)

今天来说说，被骗到缅北了做诈骗或者狗推了，怎么花最小代价逃离那里。花少一点的，离开这里回国。

首先你要知道你所在的位置是缅北哪里，克钦邦 迈扎央 棒赛 木姐 果敢老街 佤邦 小勐拉还是金三角的大其力或者妙瓦底。知道大地方名字了，再问问具体位置，这个一定要知道，不知道就问么？嘴巴用来干嘛的，说话的 问一起上班的人，多问几个总会有人知道。

知道位置以后，就要看看你上班地方是不是封闭式管理，不是封闭式管理的 最好下班后，了解附近情况，问问国门在哪个位置。去国门要多久，打车多少钱去，心里有个底。有空自己打车去去看看，提前了解一下情况。

如果是封闭式管理的。那就要看看自己能不能跑出来，并且切记跑出来不会被抓到，你要明白这一点，万一被抓到，那你基本上就是凶多吉少。被打残废事情经常发生的。

要不然就是问问老板要赔付他多少 w，他才能让你走，大部分这些老板都是求财，你只要把该给的 w 给了，他就会放你出来。还你自由，但是按照目前行情至少要好几个 w，有的黑心老板要你十几个也会存在，或者说给钱不放人的也有。真要是这样你也没办法，只能在想其他办法。

另外你实在没钱，.这些地方基本上这些挨着中国云南的边境地区，不过国家现在严厉打击电信诈骗，从 2019 年开始，只要你从国内来到中缅边境地区，手机号被公安封号的概率就是百分之五十，有的人一来直接被封号，不能打电话也不能接电话，也不能上网。有的人又不会这样，或者来一段时间以后过几天手机号码自动被公安封停了，需要回国以后才能解封，不被封号的也有，不过很少。

如果你手机没封，那就万幸，中缅边境地区都可以用中国电话，网络也是用的中国网，也没有什么漫游费，国际费之内的。 

你有空就拨打报警电话报警，把自己的情况告诉他们，说被骗过来了，现在人生自由受到影响，叫他们解救你，把的具体位置尽可能的告诉警察叔叔。

实在不知道你就说不知道，警察叔叔他们会有办法找到你的，但是最好要知道大概位置。直接拨打 110 就行，如果你在金三角大其力地区或者妙瓦底 需要加国家编号 0086。 

万一你的手机号码被封号了，也打不了电话或者无法上网，也是封闭式管理的，你们上班的地方也有小卖部啊，叫商店老板给你花钱买一张流量卡上网，或者电话卡，现在行情也是贵的离谱，运气不好买过了就只能用几天，被封号了，运气好一点就可以多用几天。

在报警的时候一定要切记，不能让你上班地方的领导，老板知道，一定要找个没人地方报警，聊天记录记得随时删除，被他们知道了。后果就太严重了。这个你一定要知道。

疫情期间话你能自己跑出来，回国自首也要花费一笔不小的开支。到边境国门自首，需要你去当地的医院办理核酸检测证明，这个你不用担心不会说缅甸话。基本上挨着云南边境地区的缅甸这边，都说是中国话，也用人民币，
也可以微信或者支付宝交易，你需要用现金，也可以去随便找个手机店铺，只要的账号钱是干净的，是本人的。他们收取手续费都给你换，现在手续费比较贵要一万人民币五百手续费了。一般医院都是收现金的不支持转账，核酸检测证明要么当天拿到，要么第二天拿到。拿到核酸检测证明就可以去国门自首。

金三角大其力跟妙瓦底这两个地方用缅币或者泰铢多一点 ，手机号码也是缅甸号码，上网也是用缅甸号码，但是只要你有钱基本上都买的到或者花钱叫人去给你买。这两个地方 要先到小勐拉，在从国门自首回去比较合适或者去老挝金三角特区从那里去云南磨憨口岸自首回头（但是老挝最近好像也被封了）。

你要明白，现在自首回国，你一定要说你是自首回来的，才不会拘留你，但是隔离费用几千块自理，隔离半个月或者 21 天，罚款 5000 以内，基本上就是这个情况。

千万不要小路回来，小路如果抓到了现在，隔离罚款加拘留，拘留都是好几个月起步，现在疫情期间非常严格，切记不要挑战权威，通过合法途径回去。

另外打算逃离你上班的地方，不要跟你上班地方任何人说，不要过于相信任何人。万一被人卖了，后果很严重，你一定要明白！！ 

万一你跑不出来，或者不敢跑，赔付给老板又没钱。自己选择的路就要先扛着，首先活着才是硬道理，活着生存下去才有机会，在慢慢的寻找机会。
