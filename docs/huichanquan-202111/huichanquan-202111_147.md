# 电影《亲爱的》原型

苦寻儿子 14 年的孙海洋 

终于见到了儿子孙卓

他紧紧抱住儿子，

迟迟未放开，失声痛哭。

**![](img/1e74297bb18da88c75842c5cf028f6df.png)**

同日 16 时许

孙海洋夫妻和儿子一起吃了第一顿团圆饭

孙卓坐在孙海洋夫妻俩的中间

三人一边吃饭，一边交流，

有说有笑，场面较为温馨。

![](img/92944a9a082b1343a42836c6519669bb.png)

认亲后，孙卓和父母跟奶奶视频通话

7 日，

孙海洋夫妻带着孙卓回湖北老家

孙卓和奶奶紧紧相拥

孙卓的爷爷看到孙子回家，眼含泪光

老人说，他都没想到孙子能找回来

听到孙子喊爷爷，他很开心。

![](img/d5b6c57d53c3e1c83e4a42a8d57f47cd.png)

不少主播到场直播

入夜后也仍未离去

视频中，一男主播不断询问孙海洋

“我们没有打搅到你吧？”

孙海洋回答“没有”。
