# 女主播喝药自杀后，骨灰被掉包配阴婚？这世界对她太残忍

落叶归根是对死者最起码的尊重，不过就有一些人为了谋求利益而剥夺了死者落叶归根的权利，对骨灰动起了歪心思。

还记得前段时间服农药自杀的美女主播“罗小猫猫子”吗？

![](img/2268cb67cbf9943b48aefc4fd9dccb49.png)

**这姑娘，估计打死也想不到，你这边自杀，那边还有人觊觎着你的骨灰——以后可不敢随便自杀了。**

“罗小猫猫子”是一名坐拥几十万粉丝的网红，经常在网上分享一些自己的穿搭视频，深受网友们的喜爱。

![](img/c3d13c4387a70da64c0aa9fe76a9d984.png)

**女主播家属表示，遭受抑郁症困扰的她是因为网友在直播间起哄，最后才在直播间喝下农药抢救无效去世的。**

![](img/8a370db07c95251653c6a4d7177e6e9c.png)

网传用户“豆豆容易饿”是“罗小猫猫子”的好友。现在，“豆豆容易饿”的个人签名是“逝者已逝，让她安心走吧”。

她最新发布的视频里写道：**“那天小猫并没有想过真的自杀，只是想以这种方式给现任看，敌草快是她稀释兑了饮料，喝下去也是因为直播间的人起哄逞强喝下去的，后来也是自己打的 120。”**

![](img/f3d1addf4d7dec607ec5fa337dbe8cb8.png)

**“罗小猫猫子”的一个视频中，用户“你最珍贵”曾评论：“哈哈哈真喝了。”**后该评论被删除，平台也无法搜索到该用户。

![](img/48f8eb4e66d58f393ebf40555e5bd961.png)

查阅该直播平台官方微博看到，网友“你最珍贵”已被封禁账号，理由是打击网络暴力言论。

![](img/35cabf78b13db16832203a2669bec420.png)

这位女网红真的太惨了，在其喝药自杀后，近日又被曝出，**她的骨灰被当地殡仪馆的人员掉包，用来配阴婚。**

![](img/4ca38224a8d973efc3a2077cd303c14a.png)

何为配阴婚，可能很多人不太懂，其实就是给死人结婚，他们所花的代价不比娶一个活人当媳妇小，甚至还有过之。

![](img/f9b18610d7bf49c764bddab67856724e.png)

如果某家一个未婚男子去世了，他们是不能够进入祖坟安葬地，如果要想破解唯一的办法就是找个人来配阴婚。

![](img/2479555e074816acc76e848a21eb12d5.png)

根据网传的消息显示，犯罪嫌疑人共 3 名，一位是汶上县殡仪馆火化工邵某，还有两位是当地殡葬业从业者张某和雷某。

![](img/d973efc637409f22c71bc084cb987a4b.png)

**邵某利用在殡仪馆作为火化工的职务之便，将死者的骨灰偷出，而张某和雷某则负责寻找买家和运输骨灰。**

![](img/4f93ded8b02ce8ac7e0b42fb38219605.png)

而爆料人表示，在当地如果配成功，则一般可以有 5-7 万元左右的报酬。

![](img/e0d3a9e2afaa4779eff40c78426e7170.png)

犯罪嫌疑人张某的妻子表示因为买家没有相中“罗小猫猫子”的骨灰，所以阴婚没有配成功。

![](img/ce100616072fe015bb91b44c82cbbd6a.png)

他这个人本事不大，却一心想着要发横财，本来这事也不大，可是刚好赶在风口浪尖上，他们就出事了。

![](img/286b4d5ad1fed4699eea7ef841704631.png)

从主犯张某妻子的描述中，她表面上说丈夫本事不大，暗地里似乎却不拿这当回事，还说是赶在风口浪尖。

**从她的言语中说明一个信息，是不是以前他们就做过这种事，只是亲属并未发现而已，具体可能就要等到警方去查，才能够水落石出了。**

![](img/626d9abe96dd0540a9dc1c330e60580c.png)

目前 3 名犯罪嫌疑人也是被警方刑事拘留，而“罗小猫猫子”的骨灰也已经被警方追回，目前已经移交给家属。

![](img/621dbb07269d8bfc7d4d8ff77af28cb1.png)

****落后没有底线，野蛮没有天花板。****

**生前被起哄自杀，死后还要被人卖骨灰，世界对这个女孩真的太残忍了。**

“配阴婚”是封建流毒、性别歧视，是对女性的最后一次物化，臭不可闻。

希望“罗小猫猫子”得到安息，也希望法律能及时亮剑，向愚昧贪婪亮剑，向封建残余亮剑，向网络无良“看客”亮剑。

**愿天堂里没有痛苦，没有这所有不好的一切。**
