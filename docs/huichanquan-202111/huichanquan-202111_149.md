# **并且愿意掏钱上当受骗呢？**

我们可以从骗子和受害方两个维度来分析 

**首先，骗子十分专业**

一般的诈骗团伙

起码会有**推广、客服和财务**三个组构成

推广组负责养猪跟人培养感情

客服组负责杀猪

从网名的设计

到对「人设」中的兴趣爱好做了解

再到「早期不透露成长往事」

每一个步骤都做得缜密细致

![](img/d5ee8ef61823cd86aac994e9830e0828.png)

有很多的被害者都会深陷在感情中无法自拔

因为骗子对你说的情话

从来都不是独一无二的

**而是在无数人身上演练过无数次**

**从而形成的肌肉记忆和条件反射**

 ![](img/c588311ba5a8761bfe171825aff4dac9.png)

这样『条件反射』式的情话

当然不是一朝一夕就能够练成的

事实上

杀猪盘团伙内部还会组织定期的交流

将开单的聊天记录反复揣摩、互相讨论

你在热恋中每一句头脑发热的情话

都有可能被全组甚至全公司几十上百人一起传阅并点评

![](img/6a6300ac52b30a9f495133272e5d2c56.png)

因此

“杀猪盘”的受害者们遇到的

都不仅仅是一个人设

而是一个完整的“人”

更不单纯只是一个骗子

**而是一整个集团极具针对性的全套解决方案**

而且“受众”十分精准

![](img/fd5bc232266fdbdc32778896d6b2fd44.png)

**那从受害的一方来说**

每个人其实都面临这样那样的生活压力

每个人都希望有一个情感出口

**特别是单身女性**

**在情感上有缺失，因而更加渴求**

**便使“屠夫集团”有了生存空间**

他们在虚拟世界里扮演伸手的角色

给“猪仔”们暂时逃离现实压迫的时间

让“猪仔”们对“依靠”有了期待

![](img/403fe0faf5420aa09b685ee838e305ff.png)

看似生活在社会边缘的乡镇青年

高学历的都市白领......

**"屠夫集团"针对每个人设计了适合他们的那双手**

只等着告诉他们
