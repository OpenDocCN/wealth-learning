# 裸聊骗局是如何一步步榨干你的

这天，你在网上认识了一位女孩儿，她热情似火，邀请你视频裸聊，你春心荡漾，以为 20 多年的单身生活能就此终结，却不料早已掉进了对方的陷阱，成为了待宰的羔羊。

色情裸聊究竟是如何一步步榨干你的？

[`v.qq.com/iframe/preview.html?width=500&height=375&auto=0&vid=wxv_2129656675555950595`](https://v.qq.com/iframe/preview.html?width=500&height=375&auto=0&vid=wxv_2129656675555950595)

和你一样，被裸聊诈骗的人不在少数，他们上至博士，下至小学，平均聊天不到两个半小时，诈骗金额高达 5 万元。

![](img/5cadc618e9003807b881ef71511eef31.png)

为什么裸聊诈骗能如此轻易得手？

实际上，敲诈人员多为团伙作案。一次裸聊敲诈的实施，至少需要四组人马。

首先，是吸粉组。他们负责在网上添加公司高管、公务员、医生等重点目标。作为猎人，他们往往以猎物的形式出现，例如，这名叫小茉莉的热辣美女，背后实际是个精神小伙。

![](img/927ef008e274663b4a0fa976e9692dbd.png)

等猎物上钩后，聊天组就上线了。

他们先是进行甜言蜜语的猛攻，再是发出裸聊邀请，蛊惑你下载视频聊天软件。

![](img/15d8833b4a5f5706936456fff29975f1.png)

当你迫不及待地点开这些软件时，你一丝不挂的形象以及手机的通讯录、个人短信、定位等信息，就都被截取了下来。

![](img/c4344192782f8c9ef0c90854c1c477e6.png)

它们都将成为敲诈组威胁你的筹码。为了逼你束手就擒，这些敲诈人员还研究出了一套专门的话术，俗称“打枪”。

第一枪叫做摊牌枪。敲诈组会拿着裸聊的视频向你要钱。如果你想报警，他们会告诉你：“没事，兄弟，我建议你直接去报警吧，这件事闹得越大越好，我跟你明说，我在国外，你拿我没办法。”

如果你打算冷处理，那么他们就会以发不雅视频给通讯录好友为由，对你进行威胁。

![](img/030ebbf70afffe8b9345dafa8f9505f0.png)

你以为花一次钱就能消灾，但现实是诈骗人员还会继续第二轮敲诈，此时他们会向你索要删除通讯录的费用。

![](img/d192697c2d10b44eb6b98955eb99993f.png)

随后，还有自称是“老板”的人物出马，告诉你只要给钱，就能一次性解决。

如果你信以为真，那就太天真了。敲诈组还会继续以删除聊天记录、号码回收费用、支付保证金、安抚裸聊小妹甚至是加班费等理由连续勒索你。

![](img/50aa530ae444819df3cd8a846c917919.png)

即使你已经被掏空，这些诈骗团伙也会想方设法让你开通网贷，榨干你的一点一滴。

那么，这些被诈骗的钱又流向了哪里？

为了掩人耳目，裸聊诈骗团伙专门设置了洗钱组，他们先是购入他人的银行卡账户、收款二维码，收取你的钱款，随后将赃款回流到团伙组织者手中。

![](img/75f0ea8fde356f091b2391b0adb57d8c.png)

从吸粉到实施敲诈再到转移赃款，你可能会以为这就是一个诈骗团伙的完整组织架构。

但事实是，这些人只是团伙中最底层的业务员，在他们的上游，还有管理组以及高层领导，控制着整个团伙的运转。

![](img/120d876af169a332b648d07a3c9dc647.png)

以这个盘踞缅甸的诈骗团伙为例。几名管理员中有负责对账、统计业务员业绩的，有负责包装并提供虚假 QQ 号给业务员的，也有负责团伙生活起居的。

![](img/54715baa68f33cfbf923bd32f57e8ffa.png)

而更上游的股东周某，则为视频软件提供技术支持，并购入大量 QQ 号。

面对组织分明的诈骗团伙，如何才能实现反诈骗呢？

其实，总结起来就四个字：守身如玉。

![](img/9e427431d44dcdc81df4493b6a4f88cf.png)

天降的美女和帅哥可能千年都难遇，一上来就邀你裸聊的不是精神小伙就是抠脚大汉。即使你把持不住上当受骗，也应该及时报警，“社死”是一时的，但幸福的可是下半辈子。
