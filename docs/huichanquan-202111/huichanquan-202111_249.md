**【涉嫌诈骗男子被抓装晕 3 天 3 夜被民警饭香馋醒】**近日，湖北孝感一男子因涉嫌诈骗被警方抓获。为逃避调查，该男子装晕 3 天 3 夜，期间医生想尽办法，插尿管、上心电、掐人中等，都没有反应。最后，因忍受不了民警在床前吃咖喱牛肉饭，该男子被“唤醒”。目前，男子已被刑拘。
