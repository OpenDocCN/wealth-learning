什么是 freestyle？其实 freestyle 有很多含义，一般指即兴的，随性的随意的发挥

很多观众包括我看完一期节目下来，一个选手都没记住，就记住了 freestyle。

当然就像吴亦凡说的，之所以如此强调，是因为他觉得 freestyle 很重要，能够看出选手们的词汇量。

但不管怎么说，才第一期节目，“freestyle”的创始人吴亦凡，火了。

所以，今天你有 freestyle 吗？
