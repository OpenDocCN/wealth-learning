# “结婚 3 个月，出轨 400 人！”又一“时间管理大师”遭曝光：他的放纵，扯开婚姻遮羞布...

在现在的社会中，有越来越多的年轻人，他们不能坚守自己的感情。在一段婚姻中，他们不能从始至终的都爱着自己的伴侣。

**![](img/29e969110fac2fcc2af535f7739761c5.png)**

**出轨 400 人，收藏 N 条女性衣物？**

爆出大料的人，就是女当事人网红周小磕。

她是一个小有名气的美妆博主，经常向粉丝分享化妆技巧，偶尔会聊起自己的情感经历。

![](img/04d67a8df0b1ec89a9678147f040ea68.png)

**没想到，周小磕突然有一天红着眼，讲述了自己老公的出轨经历，现在已经离婚。**

![](img/9509693d25b944aded81bf4e99e14961.png)

其实没离婚之前，周小磕和她老公的关系一直很好。

在平时相处，她视前夫为完美伴侣，因为对方实在是太温柔、太体贴。

所以，出于信任，她并没有随时拿对方手机查岗。

**好景不长，周小磕的闺蜜在国外的社交软件上，刷到了周小磕的老公。**

![](img/61d2971740be9dca79e2dbdcc7afe769.png)

在半夜趁老公睡着后，周小磕偷偷检查他的手机。

**一看吓一跳，一堆恶心的聊天记录瞬间呈现出来。**

![](img/9de48d37003168a4d0d369f4406601d6.png)

更会坐在周小磕旁边，偷偷用手机和对方隔空“偷情”。

![](img/b96fdb6e128e93610c7eb263483248d5.png)

**不仅如此，他并不满足单身女性，还会约一些有家庭、有孩子的女性。**

![](img/0b99c457432b7f753888e4a94c1a7c46.png)

周小磕在他手机上，发现了 400 多个人的聊天记录…

这种不堪入目的画面，让她感到崩溃，6 年的感情，终究就是一场笑话。

会不会是自己的魅力不够呢？周小磕苦想，她也舍不得这段感情，回想起以前的甜蜜，她原谅了。

![](img/27a3b58ddf665be570ee1265ad5d049d.png)

**周小磕觉得老公只是一时糊涂，更有把一些过错放在自己“失去魅力”的身上。**

![](img/53cf74bbe6a5c59f0373a0bad933abb2.png)

事实上，出轨过的男人，真的不会再犯吗？

![](img/b377fcf649c55e91f3d3cf0a4222cddf.png)

**出轨只有 0 次与无数次！**

事情败露后，男方主动去庙里待一个月，结束后再与老婆复合。

很快，周小磕就被自己的“原谅”疯狂打脸。

**要知道，出轨只有 0 次与无数次！**

趁老婆出差后，这男人又偷偷地把之前删去的微信，一个个加回来。

![](img/7c362f68b275c0a1fa84ce1fe2729f15.png)

**“吃素”这么久，男方也忍不住了，决定要大干“几天几夜”。**

![](img/8073148c799b8b77e56e0f79c3619c69.png)

周小磕回来发现后，才知道自己复合的举动，就是一个大笑话！

**这次周小磕坚决离婚，但在收拾行李的时候，居然在两人的柜子里发现了一堆女性的内裤……**

![](img/45ecc7683dbf598d8e68070552ff6672.png)

看到这里，周小磕回想起男方写的情书，有这么一句话：

**“如果人只听从身体的荷尔蒙，那何必进化出发达的大脑和丰富的情感？”**

现在回想起这句话，真的是太可笑。

![](img/fe30219b810bd2609e94fa07adcc11f1.png)

周小磕面对男方的多次出轨，除了感到愤怒之外，居然还会反思自己是不是年老色衰，毫无魅力？

![](img/4b5ff771c0723728da0c77d2cabffc36.png)

**及时止损，免于伤害**

****周小磕第一次得知老公的背叛，她还说过：“只要你不离开我，怎样都行。”****

**婚姻里，双方都是平等的，一方突然变得如此卑微，怎么还能获得对方的尊重呢？**

**像这种渣男，表面看上去帅气多金，那也只是他包装的一种人设，内心的道德观念却让人感到无比恶心。**

**其实出轨的初衷都是一样的，不甘平淡，想要寻求刺激，最终毁了自己的家庭。但是说到底，这些人还是责任心不够，同时也没有身为妻子或丈夫的自觉，才会去追求那些虚无缥缈的东西。**

**婚姻是能给人带来充实感和幸福感的，而那些体会不到的，往往自己就没有进入妻子或丈夫的角色，所以才会选择在婚姻之外去满足自己的心理。但他们这样的人，这种选择，根本就是害人害己，也毁了别人对婚姻的向往。**
