# 脚踩酸菜后，"口水蚕豆"又来了……

[**脚踩酸菜**](http://mp.weixin.qq.com/s?__biz=MzIyMDYwMTk0Mw==&mid=2247531669&idx=2&sn=9445b30a26ea8a09383deba893a95b02&chksm=97cbb7ada0bc3ebb9be3f2aa864fff6f1b1a0ff406e0129fa5a79efc561f1049f436c48ce554&scene=21#wechat_redirect)**的阴影还没有过，** 

**现在又来了嘴剥蚕豆。**

小作坊的食品再暴雷！ 

大姨们用嘴剥皮的“口水蚕豆”，

引发网友热议

近日，在云南昆明云山村。

据一段执法工作人员拍摄的视频， 

一农副产品加工坊工人们

竟然用嘴剥蚕豆皮

![](img/18005dcb8964860e9fa4511ff116538c.png)

现场已经剥好的蚕豆上， 

还可以看到**疑似牙齿咬过的痕迹**

![](img/b829e53b2d634d4da3d0fae71f553864.png)

现场调查后了解，

工人们是按剥壳称重算工资的，

**为了多点收入所以用嘴来剥壳**

![](img/8721185f1a2f55dc50b2b0af07956b59.png)

执法人员痛批恶心，

**现场所有工人及负责人**

**均无法出示健康证。**

现场联系到加工作坊的老板，

工作人员问及情况

![](img/1b33cebe4de506519892282041814511.png)

对此竟然辩称**“生意太好遭同行嫉妒”** 

不但没认识到错，

还声称她都掌握了其他几家

也都是这样（用嘴剥豆），

监管局回应，欢迎举报。

![](img/7c4fdec2863b4b1f64811d0f5d5fda5a.png)

[`v.qq.com/iframe/preview.html?width=500&height=375&auto=0&vid=g33296ucj6r`](https://v.qq.com/iframe/preview.html?width=500&height=375&auto=0&vid=g33296ucj6r)

执法人员**共查获用嘴剥的蚕豆 198.4 公斤**，

执法人员现场对蚕豆进行了销毁，

防止流入市场。

同时对老板进行批评教育，

要求其写下保证书，杜绝口水蚕豆。

这一轮“口水蚕豆”，

让不少吃货的世界又塌了房

![](img/fc59f5f3543c539c6715206a7a1aa00a.png)

**真是如检查人员所说**

**＂太恶心了＂！**
