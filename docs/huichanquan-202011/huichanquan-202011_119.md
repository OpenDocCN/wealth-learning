# OK 出金被冻卡并关押 37 天，Ok 还随意冻结他人千万美金账户！！！

一些做电信诈骗、资金盘、博彩以及杀猪盘的犯罪分子利用加密货币进行违规洗钱操作，所以这个出金被冻卡的问题一直都存在，这也和中彩票一样，中了麻烦还真不小？所以出金还是要时刻注意。 

今天，又发生了一件被冻卡的事情，据该受害者称：自己在 OK 出金，然后收到黑钱，被带走调查并且关了 37 天。该受害者者表示自己没有买过币，都是出金，那么收到黑钱根本不是自己的问题，但是 OK 却配合将其账户冻结，这是让受害者最气愤的。

其中一点：收到黑钱银行卡被冻结这是正常，或者受害者的其他资产被冻结也正常，因为可能涉及到的案子比较大。这也可以理解。但是受害者的账户被冻结这就不应该了，因为账户资金不是拿黑钱买的，除非证明账户资金是资金盘或者来路不明的资产 OK 有权冻结用户账户。 

还有一点就是，该受害者称没有买过币，那么账户资产怎么来的？其实这个也需要说明白。当时有人说，这个人好像是个土狗项目方，割了无数散户 ，被维权了。所以警方跨省抓捕，然后 OK 配合冻结。(暂未证实这个说法)

那么，如果用户资产是清白的，那么 OK 冻结账户实属不应该，OK 作为知名交易所该赔偿的赔偿。如果资产来路不明或者是割韭菜的，那么 OK 做的是没问题。 

受害者被关了 37 天放出来了，那么说明受害者没问题。那 OK 就要给他说法了，收到的钱是黑钱，那为何还要冻结受害者千万美金账户。黑钱和账户里的 U 扯能扯上什么关系，除非证明 U 有问题。

这个事不说明白，以后谁敢在 OK 出金，而且也不敢将资产放在 OK。说不准哪天说你都到黑 U 直接冻结你，你也没办法，毕竟控制权在 OK。你们说是不是！！

每个人都担心被冻卡，因为冻卡确实很麻烦。虽说在币圈炒币不违法，但是风险自担。如果出金碰到黑钱，那么这也是风险。警方把你资产扣掉，那么这钱交易所如果不赔付，那么只有你自己承担。 

如果一旦你真被冻卡，有可能会接到警方电话，需要你去配合调查，记住，一定要去，不要不当回事。因为这里面不仅仅涉及到钱的问题，可能还会是犯罪的事情，如果不去，被定性网逃那后果是非常严重的。

还有就是你之前被冻过卡，后来解封了。奉劝你这张卡就不要用了，因为只要被冻结过一次，即使只有几天，你的卡相关信息都会在警方和银行系统中留下，下次如果有一点正常，都会被列为风险账户。熊大一朋友，前两年冻过卡，然后前两个月因为出入金的问题就被列为风险账户，网上转账限制，只能柜台操作。

那么如何规避只是建议：1）出金后立马 ATM 机变现，转至亲戚卡或者拿着现金，不过就是有点麻烦。2）出金提现完尽早转到其他银行卡、股票账户、理财账户里。3）选择蓝盾商家，交易次数多的卖家。4）线下熟人操作 4）严打期间尽量不要出金。

最重要的一点就是：不要犯法，不要犯法，不要犯法！
