# 直播又被封了！狗头萝莉：到底哪里低俗了？

“狗头萝莉”虽然自身流量很高，但是由于自身风格以及各种黑料，空有人气却很难变现（**此前报道：**[**打擦边球的网红“狗头萝莉”自曝成长经历，她生命中的坏人太多了！**](http://mp.weixin.qq.com/s?__biz=MzIyMDYwMTk0Mw==&mid=2247527853&idx=3&sn=fd13fade3e1b07985e4ef0bfcafbd9a5&chksm=97cba695a0bc2f83f31332b1f8296a29506bf368a934107c6d2de6bec7740c803499accc90ec&scene=21#wechat_redirect)）。 

而且她的直播间经常被封，严重时她的号都处于搜不到的状态。

![](img/5cf44447137fe1707b9424fba677f76d.png)

虽然她经常开小号来直播，不过还是避免不了被封禁的命运。

![](img/61d8b2d21ea22df52bb3dffd924e58e2.png)

因为“狗头萝莉”发布的短视频老被平台判定为低俗不适宜公开。 

这也让她开始研究起了平台的审核规则，不过还是没用，她的视频还是经常被下架，各位绅士想看都看不到啊。

![](img/aeb69c11677cdfe8dd75e3cb696633d2.png)

“狗头萝莉”直言并没有觉得自己的作品有低俗的内容，总而言之就是她的作品在达到一定播放量后都会被判定违规。

[`mp.weixin.qq.com/mp/readtemplate?t=pages/video_player_tmpl&action=mpvideo&auto=0&vid=wxv_2293850087107854337`](https://mp.weixin.qq.com/mp/readtemplate?t=pages/video_player_tmpl&action=mpvideo&auto=0&vid=wxv_2293850087107854337)

不少网友也是直言“狗头萝莉”真的也是太难了，让她坚持做自己，往这条路走下去就行，管那么多干嘛呢。

![](img/54ee3cd35510a4cdf4d01ca36f35517c.png)

“狗头萝莉”在视频中注明自己穿了打底裤，还向审核员问好，解释自己的衣着无暴露和动作并没有暗示。

网友调侃，这几个字着实是看着有些“可怜”了，并且还有一些心酸。

![](img/fe016ae91b1cc04d366a456de9abc010.png)

“狗头萝莉”表示是有人故意在整她，还透露自己的直播权限已经被永久封号了。

就算是换小号戴口罩直播也是一样，因为现在的“瞳孔识别”能够识别出来。

![](img/e3d2cb392206acfb4205b3c7d12f971e.png)

对于自己男友的爆料事件，“狗头萝莉”也是感慨事情都过去这么久了，男方都已经“出来”了，但是自己身上的这个标签似乎永远洗不掉了。
